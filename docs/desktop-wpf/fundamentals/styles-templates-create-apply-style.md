<span data-ttu-id="05660-159">Как было показано выше, присвоив свойству <xref:System.Windows.Style.TargetType%2A> значения `TextBlock` без назначения стиля `x:Key`, вы примените стиль ко всем элементам <xref:System.Windows.Controls.TextBlock>.</span><span class="sxs-lookup"><span data-stu-id="05660-159">As previously shown, setting the <xref:System.Windows.Style.TargetType%2A> property to `TextBlock` without assigning the style an `x:Key` causes the style to be applied to all <xref:System.Windows.Controls.TextBlock> elements.</span></span> <span data-ttu-id="05660-160">В этом случае для `x:Key` неявно устанавливается значение `{x:Type TextBlock}`.</span><span class="sxs-lookup"><span data-stu-id="05660-160">In this case, the `x:Key` is implicitly set to `{x:Type TextBlock}`.</span></span> <span data-ttu-id="05660-161">Это означает, что при явном присвоении `x:Key` значения, отличающегося от `{x:Type TextBlock}`, <xref:System.Windows.Style> не применяется ко всем элементам `TextBlock` автоматически.</span><span class="sxs-lookup"><span data-stu-id="05660-161">This means that if you explicitly set the `x:Key` value to anything other than `{x:Type TextBlock}`, the <xref:System.Windows.Style> isn't applied to all `TextBlock` elements automatically.</span></span> <span data-ttu-id="05660-162">Вместо этого необходимо явным образом применить стиль (используя значение `x:Key`) ко всем элементам `TextBlock`.</span><span class="sxs-lookup"><span data-stu-id="05660-162">Instead, you must apply the style (by using the `x:Key` value) to the `TextBlock` elements explicitly.</span></span> <span data-ttu-id="05660-163">Если стиль находится в разделе ресурсов и вы не задаете для стиля свойство `TargetType`, задайте атрибут `x:Key`.</span><span class="sxs-lookup"><span data-stu-id="05660-163">If your style is in the resources section and you don't set the `TargetType` property on your style, then you must set the `x:Key` attribute.</span></span>

<span data-ttu-id="05660-164">Свойство `TargetType` не только предоставляет значение по умолчанию для `x:Key`, но и указывает тип, к которому применяются свойства метода задания.</span><span class="sxs-lookup"><span data-stu-id="05660-164">In addition to providing a default value for the `x:Key`, the `TargetType` property specifies the type to which setter properties apply.</span></span> <span data-ttu-id="05660-165">Если вы не указываете `TargetType`, необходимо определите свойства в объектах <xref:System.Windows.Setter>, используя имя класса и синтаксис `Property="ClassName.Property"`.</span><span class="sxs-lookup"><span data-stu-id="05660-165">If you don't specify a `TargetType`, you must qualify the properties in your <xref:System.Windows.Setter> objects with a class name by using the syntax `Property="ClassName.Property"`.</span></span> <span data-ttu-id="05660-166">Например, вместо установки `Property="FontSize"` необходимо задать для <xref:System.Windows.Setter.Property%2A> значение `"TextBlock.FontSize"` или `"Control.FontSize"`.</span><span class="sxs-lookup"><span data-stu-id="05660-166">For example, instead of setting `Property="FontSize"`, you must set <xref:System.Windows.Setter.Property%2A> to `"TextBlock.FontSize"` or `"Control.FontSize"`.</span></span>
