### <a name="to-create-a-variable-that-does-not-change-in-value"></a><span data-ttu-id="6cb12-110">Для создание переменной, которая не изменяет значение</span><span class="sxs-lookup"><span data-stu-id="6cb12-110">To create a variable that does not change in value</span></span>  
  
1. <span data-ttu-id="6cb12-111">На уровне модуля, объявите переменную-член с [оператор Dim](../../../../visual-basic/language-reference/statements/dim-statement.md)и включают [ReadOnly](../../../../visual-basic/language-reference/modifiers/readonly.md) ключевое слово.</span><span class="sxs-lookup"><span data-stu-id="6cb12-111">At module level, declare a member variable with the [Dim Statement](../../../../visual-basic/language-reference/statements/dim-statement.md), and include the [ReadOnly](../../../../visual-basic/language-reference/modifiers/readonly.md) keyword.</span></span>  
  
    ```  
    Dim ReadOnly timeStarted  
    ```  
  
     <span data-ttu-id="6cb12-112">Можно указать `ReadOnly` только на переменную-член.</span><span class="sxs-lookup"><span data-stu-id="6cb12-112">You can specify `ReadOnly` only on a member variable.</span></span> <span data-ttu-id="6cb12-113">Это означает, что необходимо определить переменную уровня модуля, вне любой процедуры.</span><span class="sxs-lookup"><span data-stu-id="6cb12-113">This means you must define the variable at module level, outside of any procedure.</span></span>  
  
2. <span data-ttu-id="6cb12-114">Если вы можете вычислить значение в одной инструкции во время компиляции, используйте предложение инициализации в `Dim` инструкции.</span><span class="sxs-lookup"><span data-stu-id="6cb12-114">If you can compute the value in a single statement at compile time, use an initialization clause in the `Dim` statement.</span></span> <span data-ttu-id="6cb12-115">Выполните [как](../../../../visual-basic/language-reference/statements/as-clause.md) предложение со знака равенства (`=`), за которым следует выражение.</span><span class="sxs-lookup"><span data-stu-id="6cb12-115">Follow the [As](../../../../visual-basic/language-reference/statements/as-clause.md) clause with an equal sign (`=`), followed by an expression.</span></span> <span data-ttu-id="6cb12-116">Убедитесь, что компилятор может вычислить это выражение с постоянным значением.</span><span class="sxs-lookup"><span data-stu-id="6cb12-116">Be sure the compiler can evaluate this expression to a constant value.</span></span>  
  
    ```

    ```  
  
     <span data-ttu-id="6cb12-117">Можно присвоить значение `ReadOnly` переменной только один раз.</span><span class="sxs-lookup"><span data-stu-id="6cb12-117">You can assign a value to a `ReadOnly` variable only once.</span></span> <span data-ttu-id="6cb12-118">После этого код не может когда-либо измените его значение.</span><span class="sxs-lookup"><span data-stu-id="6cb12-118">Once you do so, no code can ever change its value.</span></span>
