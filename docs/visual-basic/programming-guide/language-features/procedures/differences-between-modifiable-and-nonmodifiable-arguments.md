 <span data-ttu-id="287fb-122">Объект *неизменяемого аргумента* базовой неизменяемым или он передается [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md).</span><span class="sxs-lookup"><span data-stu-id="287fb-122">A *nonmodifiable argument* either has a nonmodifiable underlying element or is passed [ByVal](../../../../visual-basic/language-reference/modifiers/byval.md).</span></span> <span data-ttu-id="287fb-123">Процедура не может изменить элемент в вызывающем коде, даже если он является изменяемым.</span><span class="sxs-lookup"><span data-stu-id="287fb-123">The procedure cannot modify the underlying element in the calling code, even if it is a modifiable element.</span></span> <span data-ttu-id="287fb-124">Если он является неизменяемым, в вызывающем коде нельзя изменять.</span><span class="sxs-lookup"><span data-stu-id="287fb-124">If it is a nonmodifiable element, the calling code itself cannot modify it.</span></span>
