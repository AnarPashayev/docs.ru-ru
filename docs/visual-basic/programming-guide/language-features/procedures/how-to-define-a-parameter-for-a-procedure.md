1. <span data-ttu-id="686b5-108">В объявлении процедуры добавьте имя параметра в список параметров процедуры, отделив его от других параметров запятыми.</span><span class="sxs-lookup"><span data-stu-id="686b5-108">In the procedure declaration, add the parameter name to the procedure's parameter list, separating it from other parameters by commas.</span></span>
