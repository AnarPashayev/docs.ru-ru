### <a name="to-create-a-get-procedure-that-retrieves-a-property-value"></a><span data-ttu-id="96a16-116">Чтобы создать процедуру Get, которая извлекает значение свойства</span><span class="sxs-lookup"><span data-stu-id="96a16-116">To create a Get procedure that retrieves a property value</span></span>  
  
1. <span data-ttu-id="96a16-117">Между `Property` и `End Property` операторы, записывают [оператор Get](../../../../visual-basic/language-reference/statements/get-statement.md), за которым следует `End Get` инструкции.</span><span class="sxs-lookup"><span data-stu-id="96a16-117">Between the `Property` and `End Property` statements, write a [Get Statement](../../../../visual-basic/language-reference/statements/get-statement.md), followed by an `End Get` statement.</span></span> <span data-ttu-id="96a16-118">Необходимо определить все параметры `Get` процедуры.</span><span class="sxs-lookup"><span data-stu-id="96a16-118">You do not need to define any parameters for the `Get` procedure.</span></span>

 <span data-ttu-id="96a16-122">Необходимо написать `Get` процедуры для свойства чтения и записи и для свойства только для чтения.</span><span class="sxs-lookup"><span data-stu-id="96a16-122">You must write a `Get` procedure for a read-write property and for a read-only property.</span></span> <span data-ttu-id="96a16-123">Определять не нужно `Get` процедуры для свойства только для записи.</span><span class="sxs-lookup"><span data-stu-id="96a16-123">You must not define a `Get` procedure for a write-only property.</span></span>  
  
### <a name="to-create-a-set-procedure-that-writes-a-propertys-value"></a><span data-ttu-id="96a16-124">Чтобы создать процедуру набора, который записывает значение свойства</span><span class="sxs-lookup"><span data-stu-id="96a16-124">To create a Set procedure that writes a property's value</span></span>  
  
1. <span data-ttu-id="96a16-125">Между `Property` и `End Property` операторы, записывают [инструкция Set](../../../../visual-basic/language-reference/statements/set-statement.md), за которым следует `End Set` инструкции.</span><span class="sxs-lookup"><span data-stu-id="96a16-125">Between the `Property` and `End Property` statements, write a [Set Statement](../../../../visual-basic/language-reference/statements/set-statement.md), followed by an `End Set` statement.</span></span>

4. <span data-ttu-id="96a16-132">Чтобы принять значение, указанное в вызывающем коде, используйте параметр значение.</span><span class="sxs-lookup"><span data-stu-id="96a16-132">Use the value parameter to accept the value supplied by the calling code.</span></span> <span data-ttu-id="96a16-133">Можно хранить это значение непосредственно в операторе присваивания или использовать его в выражении для вычисления внутреннего значения для сохранения.</span><span class="sxs-lookup"><span data-stu-id="96a16-133">You can either store this value directly in an assignment statement, or use it in an expression to calculate the internal value to be stored.</span></span>
