# <a name="overload-resolution-visual-basic"></a><span data-ttu-id="557f6-102">Разрешение перегрузки (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="557f6-102">Overload Resolution (Visual Basic)</span></span>
<span data-ttu-id="557f6-103">Когда компилятор Visual Basic сталкивается с вызовом процедуры, которая определена в нескольких перегруженных версиях, компилятор должен решить, какую из перегрузок вызывать.</span><span class="sxs-lookup"><span data-stu-id="557f6-103">When the Visual Basic compiler encounters a call to a procedure that is defined in several overloaded versions, the compiler must decide which of the overloads to call.</span></span> <span data-ttu-id="557f6-104">Он делает это, выполняя следующие шаги:</span><span class="sxs-lookup"><span data-stu-id="557f6-104">It does this by performing the following steps:</span></span>  
  
1. <span data-ttu-id="557f6-105">**Доступность.**</span><span class="sxs-lookup"><span data-stu-id="557f6-105">**Accessibility.**</span></span> <span data-ttu-id="557f6-106">Он устраняет любые перегрузки с уровнем доступа, который предотвращает вызов кода вызова от вызова.</span><span class="sxs-lookup"><span data-stu-id="557f6-106">It eliminates any overload with an access level that prevents the calling code from calling it.</span></span>  
  
2. <span data-ttu-id="557f6-107">**Количество параметров.**</span><span class="sxs-lookup"><span data-stu-id="557f6-107">**Number of Parameters.**</span></span> <span data-ttu-id="557f6-108">Он устраняет любые перегрузки, определяющие другое количество параметров, чем в вызове.</span><span class="sxs-lookup"><span data-stu-id="557f6-108">It eliminates any overload that defines a different number of parameters than are supplied in the call.</span></span>  
  
3. <span data-ttu-id="557f6-109">**Типы параметров данных.**</span><span class="sxs-lookup"><span data-stu-id="557f6-109">**Parameter Data Types.**</span></span> <span data-ttu-id="557f6-110">Компилятор дает приоритеты, отдавая предпочтение методам расширения.</span><span class="sxs-lookup"><span data-stu-id="557f6-110">The compiler gives instance methods preference over extension methods.</span></span> <span data-ttu-id="557f6-111">Если найден какой-либо метод экземпляра, который требует только расширения конверсий в соответствии с вызовом процедуры, все методы расширения отбрасываются, а компилятор продолжает только с кандидатами метода экземпляра.</span><span class="sxs-lookup"><span data-stu-id="557f6-111">If any instance method is found that requires only widening conversions to match the procedure call, all extension methods are dropped and the compiler continues with only the instance method candidates.</span></span> <span data-ttu-id="557f6-112">Если такой метод экземпляра не найден, он продолжается как с экземпляром, так и с методами расширения.</span><span class="sxs-lookup"><span data-stu-id="557f6-112">If no such instance method is found, it continues with both instance and extension methods.</span></span>  
  
     <span data-ttu-id="557f6-113">На этом этапе он устраняет любую перегрузку, для которой типы данных вызывающих аргументов не могут быть преобразованы в типы параметров, определенные в перегрузке.</span><span class="sxs-lookup"><span data-stu-id="557f6-113">In this step, it eliminates any overload for which the data types of the calling arguments cannot be converted to the parameter types defined in the overload.</span></span>  
  
4. <span data-ttu-id="557f6-114">**Сужение конверсий.**</span><span class="sxs-lookup"><span data-stu-id="557f6-114">**Narrowing Conversions.**</span></span> <span data-ttu-id="557f6-115">Он устраняет любые перегрузки, требующие сужения преобразования от типов вызывающего аргумента к определенным типам параметров.</span><span class="sxs-lookup"><span data-stu-id="557f6-115">It eliminates any overload that requires a narrowing conversion from the calling argument types to the defined parameter types.</span></span> <span data-ttu-id="557f6-116">Это верно ли тип проверки переключатель `On` ( `Off`Option[Strict Statement](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) является или .</span><span class="sxs-lookup"><span data-stu-id="557f6-116">This is true whether the type checking switch ([Option Strict Statement](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) is `On` or `Off`.</span></span>  
  
5. <span data-ttu-id="557f6-117">**Наименьшее расширение.**</span><span class="sxs-lookup"><span data-stu-id="557f6-117">**Least Widening.**</span></span> <span data-ttu-id="557f6-118">Компилятор рассматривает оставшиеся перегрузки парами.</span><span class="sxs-lookup"><span data-stu-id="557f6-118">The compiler considers the remaining overloads in pairs.</span></span> <span data-ttu-id="557f6-119">Для каждой пары сравниваются типы данных определенных параметров.</span><span class="sxs-lookup"><span data-stu-id="557f6-119">For each pair, it compares the data types of the defined parameters.</span></span> <span data-ttu-id="557f6-120">Если типы в одной из перегрузок расширяются до соответствующих типов в другом, компилятор устраняет последнее.</span><span class="sxs-lookup"><span data-stu-id="557f6-120">If the types in one of the overloads all widen to the corresponding types in the other, the compiler eliminates the latter.</span></span> <span data-ttu-id="557f6-121">То есть он сохраняет перегрузку, которая требует наименьшего количества расширения.</span><span class="sxs-lookup"><span data-stu-id="557f6-121">That is, it retains the overload that requires the least amount of widening.</span></span>  
  
6. <span data-ttu-id="557f6-122">**Единый кандидат.**</span><span class="sxs-lookup"><span data-stu-id="557f6-122">**Single Candidate.**</span></span> <span data-ttu-id="557f6-123">Он продолжает рассматривать перегрузки в парах до тех пор, пока не останется только одна перегрузка, и не разрешит вызов этой перегрузки.</span><span class="sxs-lookup"><span data-stu-id="557f6-123">It continues considering overloads in pairs until only one overload remains, and it resolves the call to that overload.</span></span> <span data-ttu-id="557f6-124">Если компилятор не может уменьшить перегрузки до одного кандидата, он генерирует ошибку.</span><span class="sxs-lookup"><span data-stu-id="557f6-124">If the compiler cannot reduce the overloads to a single candidate, it generates an error.</span></span>
