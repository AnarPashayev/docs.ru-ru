- <span data-ttu-id="05170-118">Один объект <xref:System.Xml.Linq.XElement>.</span><span class="sxs-lookup"><span data-stu-id="05170-118">One <xref:System.Xml.Linq.XElement> object.</span></span> <span data-ttu-id="05170-119">Это корневой узел XML-документа.</span><span class="sxs-lookup"><span data-stu-id="05170-119">This is the root node of the XML document.</span></span>

- <span data-ttu-id="05170-125">Один тип документа <xref:System.Xml.Linq.XDocumentType> для DTD.</span><span class="sxs-lookup"><span data-stu-id="05170-125">One <xref:System.Xml.Linq.XDocumentType> for the DTD.</span></span>

## <a name="using-xdocument"></a><span data-ttu-id="05170-132">Использование XDocument</span><span class="sxs-lookup"><span data-stu-id="05170-132">Using XDocument</span></span>  
 <span data-ttu-id="05170-133">При создании объектов <xref:System.Xml.Linq.XDocument> используется функциональное построение, так же как и при создании объектов <xref:System.Xml.Linq.XElement>.</span><span class="sxs-lookup"><span data-stu-id="05170-133">To construct an <xref:System.Xml.Linq.XDocument>, use functional construction, just like you do to construct <xref:System.Xml.Linq.XElement> objects.</span></span>  
  
 <span data-ttu-id="05170-134">Следующий фрагмент кода создает объект <xref:System.Xml.Linq.XDocument> и ассоциированные с ним вложенные объекты.</span><span class="sxs-lookup"><span data-stu-id="05170-134">The following code creates an <xref:System.Xml.Linq.XDocument> object and its associated contained objects.</span></span>  
  
```vb  
Dim doc As XDocument = <?xml version="1.0" encoding="utf-8"?>  
                       <!--This is a comment.-->  
                       <?xml-stylesheet href='mystyle.css' title='Compact' type='text/css'?>  
                       <Pubs>  
                           <Book>  
                               <Title>Artifacts of Roman Civilization</Title>  
                               <Author>Moreno, Jordao</Author>  
                           </Book>  
                           <Book>  
                               <Title>Midieval Tools and Implements</Title>  
                               <Author>Gazit, Inbar</Author>  
                           </Book>  
                       </Pubs>  
                       <!--This is another comment.-->  
doc.Save("test.xml")  
```
