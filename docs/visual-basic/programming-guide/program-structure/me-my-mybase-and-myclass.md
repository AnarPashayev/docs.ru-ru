## <a name="me"></a><span data-ttu-id="9d87f-105">Me</span><span class="sxs-lookup"><span data-stu-id="9d87f-105">Me</span></span>  
 <span data-ttu-id="9d87f-106">`Me` Ключевое слово предоставляет возможность ссылаться на конкретный экземпляр класса или структуры, в котором в данный момент выполняется код.</span><span class="sxs-lookup"><span data-stu-id="9d87f-106">The `Me` keyword provides a way to refer to the specific instance of a class or structure in which the code is currently executing.</span></span> <span data-ttu-id="9d87f-107">`Me` ведет себя как объектную переменную или переменную структуры, ссылка на текущий экземпляр.</span><span class="sxs-lookup"><span data-stu-id="9d87f-107">`Me` behaves like either an object variable or a structure variable referring to the current instance.</span></span> <span data-ttu-id="9d87f-108">С помощью `Me` особенно полезна для передачи сведений о текущий выполняемый экземпляр класса или структуры в процедуру в другой класс, структуру или модуля.</span><span class="sxs-lookup"><span data-stu-id="9d87f-108">Using `Me` is particularly useful for passing information about the currently executing instance of a class or structure to a procedure in another class, structure, or module.</span></span>

   FormName.BackColor = Color.FromArgb(Rnd() * 256, Rnd() * 256, Rnd() * 256)  
End Sub  
```  
  
 <span data-ttu-id="9d87f-110">Можно вызвать эту процедуру и передать текущий экземпляр <xref:System.Windows.Forms.Form> класса как аргумент с помощью следующей инструкции.</span><span class="sxs-lookup"><span data-stu-id="9d87f-110">You can call this procedure and pass the current instance of the <xref:System.Windows.Forms.Form> class as an argument by using the following statement.</span></span>

```
