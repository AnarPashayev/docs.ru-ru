# <a name="program-structure-and-code-conventions-visual-basic"></a>Соглашения о структуре программы и коде (Visual Basic)
В этом разделе представлены типичная структура программы Visual Basic, предоставляет простой программы на Visual Basic, «Hello, World» и описывает соглашения о написании кода Visual Basic. Соглашения о написании кода, рекомендации, которые касаются не логики программы а ее внешней структуры и внешнего вида. Удовлетворяющий этим соглашениям код легче читать, понимать и поддерживать. Соглашения о коде могут включать, помимо прочего:  
  
- Стандартизованные форматы использования меток и комментирования кода.  
  
- Рекомендации для интервала, форматирование и выделение отступами кода.  
  
- Соглашения об именовании для объектов, переменных и процедур.  
  
 Ниже содержится набор правил программирования для программ на Visual Basic, а также примеры применения этих рекомендаций.
