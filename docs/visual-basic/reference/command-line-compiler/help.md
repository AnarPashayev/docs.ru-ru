# <a name="-help---visual-basic"></a>-help, -? (Visual Basic)
Отображает параметры компилятора.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
-help  
' -or-  
-?  
```  
  
## <a name="remarks"></a>Примечания  
 Если включить этот параметр при компиляции, выходной файл не создается и компиляция не происходит.  
  
> [!NOTE]
>  `-help` Не доступна из среды разработки Visual Studio; она доступна только при компиляции из командной строки.  
  
## <a name="example"></a>Пример  
 Следующий код отображает справку из командной строки.  
  
```  
vbc -help  
```
