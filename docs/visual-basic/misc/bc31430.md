# <a name="expression-of-type-typename1-can-never-be-of-type-typename2"></a><span data-ttu-id="01482-102">Выражение типа " \<typename1> " не может быть типа " \<typename2> "</span><span class="sxs-lookup"><span data-stu-id="01482-102">Expression of type '\<typename1>' can never be of type '\<typename2>'</span></span>
<span data-ttu-id="01482-103">Выражение `TypeOf`...`Is` проверяет переменную, ссылающуюся на объект, на тип данных, который она не может содержать.</span><span class="sxs-lookup"><span data-stu-id="01482-103">A `TypeOf`...`Is` expression tests an object reference variable to a data type that it cannot hold.</span></span>  
  
 <span data-ttu-id="01482-104">В некоторых случаях компилятор может определить, что проверка `TypeOf`...`Is` может завершиться только ошибкой, например, если отсутствует отношение наследования между двумя классами.</span><span class="sxs-lookup"><span data-stu-id="01482-104">In some cases the compiler can determine that a `TypeOf`...`Is` test can only fail, for example if there is no inheritance relationship between two classes.</span></span>  
  
 <span data-ttu-id="01482-105">Приведенный ниже код может вызвать эту ошибку.</span><span class="sxs-lookup"><span data-stu-id="01482-105">The following code can generate this error.</span></span>  
  
 `Dim refVar as System.Windows.Forms.Form`  
  
 `If TypeOf refVar Is System.Array`  
  
 `End If`  
  
 <span data-ttu-id="01482-106">Так как <xref:System.Windows.Forms.Form> и <xref:System.Array> являются полностью несвязанными типами, компилятор может определить, что выражение `TypeOf`...`Is` возвращает `False` для любого значения `refVar`.</span><span class="sxs-lookup"><span data-stu-id="01482-106">Because <xref:System.Windows.Forms.Form> and <xref:System.Array> are totally unrelated types, the compiler can determine that the `TypeOf`...`Is` expression returns `False` for any value of `refVar`.</span></span>  
  
 <span data-ttu-id="01482-107">**Идентификатор ошибки:** BC31430</span><span class="sxs-lookup"><span data-stu-id="01482-107">**Error ID:** BC31430</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="01482-108">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="01482-108">To correct this error</span></span>  
  
- <span data-ttu-id="01482-109">Проверяйте переменную на реалистичный тип данных или вообще удалите проверку `TypeOf`...`Is` .</span><span class="sxs-lookup"><span data-stu-id="01482-109">Test the variable for a realistic data type, or remove the `TypeOf`...`Is` test altogether.</span></span>
