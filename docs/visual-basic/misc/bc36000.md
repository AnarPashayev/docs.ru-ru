 <span data-ttu-id="28782-104">Ключевое слово `Global` позволяет коду получать доступ к пространству имен корневого уровня из пространства имен, где такой доступ заблокирован.</span><span class="sxs-lookup"><span data-stu-id="28782-104">The purpose of `Global` is to allow your code to access a root-level namespace from inside a namespace structure that has blocked the root-level namespace.</span></span>
