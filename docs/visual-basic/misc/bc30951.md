# <a name="wrong-number-of-type-arguments-passed-to-method-genericprocedurename"></a>Неправильное число аргументов типа, передаваемого методу '\<имя_универсальной_процедуры >'
Универсальная процедура вызвана с количеством аргументов типа, не соответствующим количеству параметров типа, с которым она была определена.  
  
 **Идентификатор ошибки:** BC30951  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Укажите аргумент типа для каждого параметра типа, определенного для универсальной процедуры.  
  
     -или-  
  
- Вызовите универсальную процедуру вообще без аргументов типа и позвольте компилятору попытаться определить аргументы типа.
