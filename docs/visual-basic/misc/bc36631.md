 <span data-ttu-id="9c85e-104">**Идентификатор ошибки:** BC36631</span><span class="sxs-lookup"><span data-stu-id="9c85e-104">**Error ID:** BC36631</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="9c85e-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="9c85e-105">To correct this error</span></span>  
  
1. <span data-ttu-id="9c85e-106">Добавьте ключевое слово `Join` в предложение `Group Join` , как показано в следующем примере:</span><span class="sxs-lookup"><span data-stu-id="9c85e-106">Add the `Join` keyword to the `Group Join` clause, as shown in the following example:</span></span>
