# <a name="name-cannot-expose-type-type1-in-specifier-type2-typename-through-name"></a>'\<имя >' не может представлять тип \<тип1 > в \<описатель > \<тип2 > '\<typename >' через \<имя >
Предпринята попытка предоставить закрытый тип за пределами открытого класса `MyClass`.  
  
 **Идентификатор ошибки:** BC30508  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Объявите тип `Public`.
