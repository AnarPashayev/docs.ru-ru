<span data-ttu-id="a9927-103">Нельзя использовать символ типа для имени свойства при объявлении экземпляра анонимного типа.</span><span class="sxs-lookup"><span data-stu-id="a9927-103">You cannot use a type character on a property name when you declare an instance of an anonymous type.</span></span> <span data-ttu-id="a9927-104">Тип данных свойства выводится из присвоенного значения.</span><span class="sxs-lookup"><span data-stu-id="a9927-104">The data type of the property is inferred from the value assigned to it.</span></span> <span data-ttu-id="a9927-105">Например, ниже приведены недопустимые объявления.</span><span class="sxs-lookup"><span data-stu-id="a9927-105">For example, the following declarations are not valid.</span></span>  
  
```vb  
'' Not valid.  
'Dim anon1 = New With {.ID$ = "abc"}  
'Dim anon2 = New With {.ID$ = 42}  
```  
  
 <span data-ttu-id="a9927-106">**Идентификатор ошибки:** BC36560</span><span class="sxs-lookup"><span data-stu-id="a9927-106">**Error ID:** BC36560</span></span>
