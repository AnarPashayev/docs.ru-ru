# <a name="maximum-number-of-warnings-has-been-exceeded"></a>Превышено максимальное количество предупреждений.

Максимальное число предупреждений компиляции — 100. Компилятор налагает это ограничение, чтобы уменьшить размер отчетов об ошибках и предупреждениях.  
  
 По умолчанию данное сообщение является предупреждением. Дополнительные сведения о скрытии предупреждений или обработке предупреждений как ошибок см. в разделе [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **Идентификатор ошибки:** BC42206  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Просмотрите другие предупреждения, созданные этой компиляцией, и устраните как можно больше из них. Если исходный код содержит более 100 предупреждений, при следующей компиляции может возникнуть еще больше предупреждений.
