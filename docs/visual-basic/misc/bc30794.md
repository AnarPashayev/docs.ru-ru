 <span data-ttu-id="2088b-104">Когда код использует адрес процедуры, которая определена в нескольких перегруженных версиях, компилятор должен решить, какую из перегрузок следует использовать.</span><span class="sxs-lookup"><span data-stu-id="2088b-104">When code uses the address of a procedure that is defined in several overloaded versions, the compiler must decide which of the overloads to use.</span></span> <span data-ttu-id="2088b-105">Он пытается найти единственную версию со списком параметров, который соответствует списку параметров делегатов.</span><span class="sxs-lookup"><span data-stu-id="2088b-105">It tries to find a single version with a parameter list that matches the delegate parameter list.</span></span> <span data-ttu-id="2088b-106">Дополнительные сведения см. в разделе [Overload Resolution](../programming-guide/language-features/procedures/overload-resolution.md).</span><span class="sxs-lookup"><span data-stu-id="2088b-106">For more information, see [Overload Resolution](../programming-guide/language-features/procedures/overload-resolution.md).</span></span>
