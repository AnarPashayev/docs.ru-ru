# <a name="end-operator-must-be-the-first-statement-on-a-line"></a>End Operator должен быть первым оператором в строке

Оператор `End Operator` следует за разделителем операторов в виде двоеточия (`:`). `End Operator` должен быть единственным оператором в своей строке исходного кода.  
  
 **Идентификатор ошибки:** BC33006  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Разместите несколько операторов в отдельных строках.
