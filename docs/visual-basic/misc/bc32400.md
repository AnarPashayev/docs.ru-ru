# <a name="class-classname-could-not-be-created-error"></a>Класс\<имя_класса > 'не удалось создать: \<ошибка >
Ошибка инициализации класса. Одной из возможных причин является повреждение установки [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] .  
  
 **Идентификатор ошибки:** BC32400  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1. Проверьте указанное сообщение об ошибке и выполните соответствующее действие.  
  
2. Переустановите [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)] и выполните повторную компиляцию программы.  
  
3. Если ошибка повторяется, переустановите компилятор Visual Basic.  
  
4. Если ошибка повторяется после переустановки, соберите сведения об условиях ее возникновения и уведомите службу технической поддержки Майкрософт.
