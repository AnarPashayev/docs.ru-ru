<span data-ttu-id="84233-103">Универсальный тип объявлен с конфликтующими ограничениями из-за сочетания косвенных ограничений.</span><span class="sxs-lookup"><span data-stu-id="84233-103">A generic type is declared with conflicting constraints due to a combination of indirect constraints.</span></span>
