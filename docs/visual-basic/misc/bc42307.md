# <a name="xml-comment-parameter-parameter-does-not-match-a-parameter-on-the-corresponding-keyword-statement"></a>Параметр комментария XML '\<параметр >' не соответствует параметру соответствующего \<ключевое слово > инструкции
Параметр комментария XML \<параметр > не соответствует параметру соответствующего \<ключевое слово > инструкции. Комментарий XML будет проигнорирован.  
  
 Тег XML-комментария указан для некоторых, но не для всех параметров в методе.  
  
 **Идентификатор ошибки:** BC42307  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Укажите теги XML-комментария для всех параметров в методе.
