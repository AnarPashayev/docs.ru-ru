# <a name="-expected"></a>Требуется ">"
Указанный код имеет неверный синтаксис. Эта ошибка обычно указывает на то, что закрывающая угловая скобка атрибута или литерала XML-элемента отсутствует.  
  
 **Идентификатор ошибки:** BC30636  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Проверьте синтаксис используемого кода.
