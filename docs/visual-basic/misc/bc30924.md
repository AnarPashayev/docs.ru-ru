# <a name="name-cannot-expose-type-type1-used-in-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-outside-the-project-through-specifier-type2"></a>"\<имя >" не может представлять тип "\<тип1 >" используется в базовом делегированном типе "\<тип_делегата >" события, он реализует вне проекта посредством \<описатель > "\<тип2 > "
Код представляет закрытый тип за пределами открытого класса.  
  
 **Идентификатор ошибки:** BC30924  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Объявите тип `Public` или используйте другой тип.
