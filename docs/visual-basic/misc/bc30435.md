# <a name="members-in-a-structure-cannot-be-declared-specifier"></a>Члены структуры не могут объявляться как " \<specifier> "
Вы использовали спецификатор, который недопустим для члена в операторе `Structure` .  
  
 **Идентификатор ошибки:** BC30435  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Удалите спецификатор.
