# <a name="raiseevent-statements-are-not-valid-in-the-immediate-window"></a>Операторы RaiseEvent недопустимы в окне интерпретации
`RaiseEvent`операторы нельзя использовать в окне **интерпретации** .  
  
 **Идентификатор ошибки:** BC30739  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Протестируйте `RaiseEvent` операторы из кода в **редакторе кода**.
