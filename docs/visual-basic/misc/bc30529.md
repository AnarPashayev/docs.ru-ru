# <a name="all-parameters-must-be-explicitly-typed-if-any-are"></a>Все параметры должны быть явно типизированы, если они имеются
Смешаны разные типы объявлений типов в функции.  
  
 **Идентификатор ошибки:** BC30529  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Убедитесь, что все параметры явно типизированы.
