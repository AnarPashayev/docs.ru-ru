# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-the-array-types-have-different-numbers-of-dimensions"></a>Значение типа '\<тип1 >' нельзя преобразовать в '\<тип2 >', так как в типах массивов указано различное количество измерений
Вы попытались преобразовать массив в другой массив с иным числом измерений.  
  
 **Идентификатор ошибки:** BC30414  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Измените один или оба массива таким образом, чтобы число измерений в них совпадало.
