# <a name="type-characters-cannot-be-used-in-range-variable-declarations"></a>Символы типа не могут использоваться в объявлениях переменных диапазона
Символ типа был включен в конец объявления имени переменной диапазона в запросе LINQ.  
  
 **Идентификатор ошибки:** BC36601  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1. Удалите символ типа из объявления переменной диапазона.
