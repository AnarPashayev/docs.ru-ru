# <a name="a-nullable-type-cannot-be-inferred-for-variable-variablename"></a>Недействительный тип не может быть\<выведен для переменной "> переменного имени"
Недопустимый тип значения не может быть выведен из типов ссылки, `String`например массива, класса или . Значение, на основе которого определяется тип данных, должно иметь тип значения. Приведенный ниже код иллюстрирует эту ошибку.  
  
```vb  
'' Not valid.
'Dim arrList? = New ArrayList  
'Dim except? = New Exception  
'Dim obj? = New Object  
'Dim stringVar? = "Open the application."  
  
' Valid.  
Dim intVar? = 10  
```  
  
 **Идентификатор ошибки:** BC36628  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Удалите назначение, допускающее значения NULL.
