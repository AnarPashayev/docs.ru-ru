# <a name="new-cannot-be-used-on-a-type-parameter-that-does-not-have-a-new-constraint"></a><span data-ttu-id="2f2ac-102">"New" нельзя использовать для параметра типа, у которого нет ограничения "New"</span><span class="sxs-lookup"><span data-stu-id="2f2ac-102">'New' cannot be used on a type parameter that does not have a 'New' constraint</span></span>
<span data-ttu-id="2f2ac-103">В операторе объявления используется [оператор New](../../visual-basic/language-reference/operators/new-operator.md) предложение, указав параметр типа в качестве типа должен быть создан и параметр типа объявляется без `New` ограничение.</span><span class="sxs-lookup"><span data-stu-id="2f2ac-103">A declaration statement uses a [New Operator](../../visual-basic/language-reference/operators/new-operator.md) clause specifying a type parameter as the type to be created, and the type parameter is declared without a `New` constraint.</span></span>  
  
 <span data-ttu-id="2f2ac-104">*Ограничение* для параметра типа накладывает требование на любой аргумент типа, передаваемый параметру типа при создании универсального типа.</span><span class="sxs-lookup"><span data-stu-id="2f2ac-104">A *constraint* on a type parameter imposes a requirement on any type argument passed to that type parameter when the generic type is created.</span></span> <span data-ttu-id="2f2ac-105">Ограничение `New` указывает, что аргумент типа должен предоставлять конструктор без параметров, к которому код создания может получить доступ.</span><span class="sxs-lookup"><span data-stu-id="2f2ac-105">The `New` constraint specifies that the type argument must expose a parameterless constructor that the creating code can access.</span></span> <span data-ttu-id="2f2ac-106">Это позволяет использовать предложение `New` в операторе объявления для создания экземпляра этого типа.</span><span class="sxs-lookup"><span data-stu-id="2f2ac-106">This is what allows a `New` clause in a declaration statement to create an instance of that type.</span></span>  
  
 <span data-ttu-id="2f2ac-107">**Идентификатор ошибки:** BC32046</span><span class="sxs-lookup"><span data-stu-id="2f2ac-107">**Error ID:** BC32046</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="2f2ac-108">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="2f2ac-108">To correct this error</span></span>  
  
- <span data-ttu-id="2f2ac-109">Если можно потребовать, чтобы аргумент типа предоставлял доступ к конструктору без параметров, добавьте к объявлению параметра типа ограничение `New` .</span><span class="sxs-lookup"><span data-stu-id="2f2ac-109">If you can require the type argument to expose an accessible parameterless constructor, add the `New` constraint to the declaration of the type parameter.</span></span>
