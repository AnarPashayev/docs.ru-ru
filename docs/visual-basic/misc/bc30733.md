# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>'\<имя_члена >' уже объявлен в '\<имя_процедуры >', созданный для данного \<имя_контейнера >

Объявленный член, который имеет имя, совпадающее с именем члена, объявленном вызовом среда CLR.

**Идентификатор ошибки:** BC30733

## <a name="to-correct-this-error"></a>Исправление ошибки

- Измените имя члена, чтобы устранить конфликт.
