# <a name="typename-in-assembly-assemblyname-has-been-forwarded-to-itself-and-so-is-an-unsupported-type"></a><span data-ttu-id="e1562-102">'\<typename >' в сборке '\<имя_сборки >' переадресован самому себе и поэтому является неподдерживаемым типом</span><span class="sxs-lookup"><span data-stu-id="e1562-102">'\<typename>' in assembly '\<assemblyname>' has been forwarded to itself and so is an unsupported type</span></span>
<span data-ttu-id="e1562-103">Сборка использует <xref:System.Runtime.CompilerServices.TypeForwardedToAttribute> для переадресации одного из ее типов в другую сборку, но указывает тот же тип в той же сборке.</span><span class="sxs-lookup"><span data-stu-id="e1562-103">An assembly uses the <xref:System.Runtime.CompilerServices.TypeForwardedToAttribute> to forward one of its types to another assembly, but specifies the same type in the same assembly.</span></span>  
  
 <span data-ttu-id="e1562-104">*Переадресация типа* означает переназначение определения класса, структуры, интерфейса, делегата или перечисления сборки, отличной от той, в которой находилось исходное определение.</span><span class="sxs-lookup"><span data-stu-id="e1562-104">*Type forwarding* means reassigning the definition of a class, structure, interface, delegate, or enumeration to an assembly other than the one in which it was originally defined.</span></span> <span data-ttu-id="e1562-105">Она часто используется вместе с *рефакторингом кода*, с помощью чего вы можете разбить сборку на несколько сборок или переместить код из одной сборки в другую.</span><span class="sxs-lookup"><span data-stu-id="e1562-105">It is often used in conjunction with *code refactoring*, by which you split an assembly into two or more assemblies or move code from one assembly to another.</span></span>  
  
 <span data-ttu-id="e1562-106">Переадресация типа самому себе приводит к циклической переадресации.</span><span class="sxs-lookup"><span data-stu-id="e1562-106">Forwarding a type to itself results in circular forwarding.</span></span> <span data-ttu-id="e1562-107">Если другая сборка пытается получить доступ к переадресованному типу, то происходит бесконечная переадресация, никогда не достигающая типа, который не был переадресован.</span><span class="sxs-lookup"><span data-stu-id="e1562-107">If another assembly attempted to access the forwarded type, it would go through endless forwarding without ever arriving at a type that had not been forwarded.</span></span>  
  
 <span data-ttu-id="e1562-108">**Идентификатор ошибки:** BC31425</span><span class="sxs-lookup"><span data-stu-id="e1562-108">**Error ID:** BC31425</span></span>
