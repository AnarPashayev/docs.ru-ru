## <a name="to-correct-this-error"></a><span data-ttu-id="35c05-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="35c05-106">To correct this error</span></span>

    Dim fv = filterValue  
    Dim queryResult = From num In collection _  
                      Where num < fv
