# <a name="get-statements-are-no-longer-supported"></a>Операторы "Get" больше не поддерживаются
`Get` больше не поддерживаются. Функциональность файлового ввода-вывода доступна в пространстве имен `Microsoft.VisualBasic` .  
  
 `Get` не поддерживается для операций с файлами и может использоваться только в синтаксисе процедуры свойства.  
  
 **Идентификатор ошибки:** BC30829  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1. Проводит операции с файлами с помощью элементов `System.IO`, `FileSystemObject`и функции времени выполнения Visual Basic.
