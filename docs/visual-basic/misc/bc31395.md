# <a name="type-characters-are-not-allowed-in-label-identifiers"></a>Символы типа недопустимы в идентификаторах надписей

Метка оператора содержит по крайней мере один символ типа идентификатора.  
  
 Метка оператора должна иметь допустимое в Visual Basic имя. В разрешенные символы не входят символы типа идентификатора (`%`, `&`, `@`, `!`, `#`и `$`). См. раздел [Declared Element Names](../programming-guide/language-features/declared-elements/declared-element-names.md).  
  
 **Идентификатор ошибки:** BC31395  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Удалите символ или символы типа идентификатора из метки оператора.
