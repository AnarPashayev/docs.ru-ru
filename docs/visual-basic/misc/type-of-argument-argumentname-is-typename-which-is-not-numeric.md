# <a name="type-of-argument-argumentname-is-typename-which-is-not-numeric"></a>Типом аргумента "\<argumentname>" является "\<typename>", то есть не числовой тип

Вместо ожидаемого числового аргумента используется нечисловой.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Убедитесь в том, что аргумент является числовым.
