# <a name="property-propertyname-cannot-be-set-to-an-empty-string-or-nothing"></a>Свойство \<имя_свойства > нельзя установить равным пустой строки или Nothing
Для свойства требуется значение, отличное от пустой строки ("") или `Nothing`. Это может быть результатом неправильного вычисления значения.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Укажите допустимое значение.
