# <a name="method-methodname-is-already-declared-in-interface-interfacename"></a>Метод '\<имя_метода >' уже объявлен в интерфейсе '\<имя_интерфейса >'
Вы объявили метод интерфейса второй раз, дублируя его.  
  
 **Идентификатор ошибки:** BC30377  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Удалите повторяющееся объявление.
