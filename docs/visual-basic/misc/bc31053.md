<span data-ttu-id="ee019-103">Оператор `Implements` обнаружен в недопустимом положении.</span><span class="sxs-lookup"><span data-stu-id="ee019-103">An `Implements` statement was encountered in an invalid location.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="ee019-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="ee019-105">To correct this error</span></span>  
  
- <span data-ttu-id="ee019-106">Поместите операторы `Implements` сразу после операторов `Inherits` , но перед любыми объявлениями.</span><span class="sxs-lookup"><span data-stu-id="ee019-106">Place `Implements` statements immediately following any `Inherits` statements, but before any declarations.</span></span> <span data-ttu-id="ee019-107">Пример:</span><span class="sxs-lookup"><span data-stu-id="ee019-107">For example:</span></span>

    ```vb  
       Inherits Base

          ' Add code to implement the method.  
       End Sub  
    End Class  
    ```
