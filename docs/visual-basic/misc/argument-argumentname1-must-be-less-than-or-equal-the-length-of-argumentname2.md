# <a name="argument-argumentname1-must-be-less-than-or-equal-the-length-of-argument-argumentname2"></a>Аргумент '\<аргумент1 >' должно быть меньше или равен длине аргумента '\<аргумент2 >'
Длина предоставленного аргумента превышает длину аргумента, который он замещает.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Сократите длину первого аргумента, чтобы она была не больше длины второго аргумента.
