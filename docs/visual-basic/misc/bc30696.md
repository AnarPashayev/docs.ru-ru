# <a name="membername1-and-membername2-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>'\<имя_члена1 >' и '\<имя_члена2 >' не могут перегружать друг друга, так как они отличаются только типами необязательных параметров
Перегруженные члены должны иметь разные типы данных для одного или нескольких обязательных параметров.  
  
 **Идентификатор ошибки:** BC30696  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Определите по крайней мере один обязательный параметр с другим типом данных.
