# <a name="let-and-set-assignment-statements-are-no-longer-supported"></a>Операторы присваивания "Let" и "Set" больше не поддерживаются
Операторы`Let` и `Set` нельзя использовать в присваиваниях.  
  
 **Идентификатор ошибки:** BC30807  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Удалите `Let` и `Set` из операторов присваивания, например:  
  
     `X = 2`
