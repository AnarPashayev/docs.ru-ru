# <a name="type-typename-cannot-inherit-from-a-type-nested-within-it"></a><span data-ttu-id="a50ef-102">Тип "\<typename>" не могут наследовать от вложенного в него типа</span><span class="sxs-lookup"><span data-stu-id="a50ef-102">Type '\<typename>' cannot inherit from a type nested within it</span></span>
<span data-ttu-id="a50ef-103">Определение класса или интерфейса включает инструкцию [Inherits Statement](../language-reference/statements/inherits-statement.md) , указывающую вложенный в него тип.</span><span class="sxs-lookup"><span data-stu-id="a50ef-103">A class or interface definition includes an [Inherits Statement](../language-reference/statements/inherits-statement.md) that specifies a type nested within it.</span></span>  
  
 <span data-ttu-id="a50ef-104">Наследование должно быть линейным, не циклическим.</span><span class="sxs-lookup"><span data-stu-id="a50ef-104">Inheritance must be linear, not circular.</span></span> <span data-ttu-id="a50ef-105">Тип не может наследовать от типа, который наследует от него.</span><span class="sxs-lookup"><span data-stu-id="a50ef-105">A type cannot inherit from a type that inherits from it.</span></span>  
  
 <span data-ttu-id="a50ef-106">Связанное ограничение — тип не может наследовать от типа, который еще не определен.</span><span class="sxs-lookup"><span data-stu-id="a50ef-106">A related restriction is that a type cannot inherit from a type that is not yet defined.</span></span> <span data-ttu-id="a50ef-107">Наследование включает возможность повторно использовать элементы базового класса, а для этого необходимо, чтобы эти элементы были определены.</span><span class="sxs-lookup"><span data-stu-id="a50ef-107">Inheritance involves the ability to reuse members of the base class, which in turn requires that these members be defined.</span></span> <span data-ttu-id="a50ef-108">Поэтому Visual Basic не может скомпилировать код, как в следующем примере.</span><span class="sxs-lookup"><span data-stu-id="a50ef-108">Therefore, Visual Basic cannot compile code such as the following example.</span></span>  
  
```vb  
Public Class outerClass  
    ' The following statement is INVALID because innerClass is not defined.  
    Inherits innerClass  
    Public Class innerClass  
        Public Sub doSomething()  
        End Sub  
    End Class  
End Class  
```
