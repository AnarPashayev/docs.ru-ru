# <a name="runtime-errors-might-occur-when-converting-type1-to-type2"></a>Возможно возникновение ошибок выполнения при преобразовании '\<тип1 >' для '\<тип2 >'.

При попытке преобразования типа интерфейса в тип класса может возникнуть ошибка времени выполнения.

**Идентификатор ошибки:** BC42322
