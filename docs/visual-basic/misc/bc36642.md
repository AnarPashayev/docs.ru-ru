# <a name="option-strict-on-requires-each-lambda-expression-parameter-to-be-declared-with-an-as-clause-if-its-type-cannot-be-inferred"></a><span data-ttu-id="e7514-102">Для Option Strict On необходимо, чтобы каждый параметр лямбда-выражения объявлялся с конструкцией As, если его тип не удается вывести.</span><span class="sxs-lookup"><span data-stu-id="e7514-102">Option Strict On requires each lambda expression parameter to be declared with an 'As' clause if its type cannot be inferred</span></span>

<span data-ttu-id="e7514-103">Параметр в лямбда-выражении был объявлен без использования предложения `As` , с `Option Strict` on.</span><span class="sxs-lookup"><span data-stu-id="e7514-103">You have declared a parameter in a lambda expression without using an `As` clause, with `Option Strict` on.</span></span>  
  
```vb  
' Not valid when Option Strict is on.  
' Dim increment1 = Function (n) n + 1  
```  
  
 <span data-ttu-id="e7514-104">Предыдущее объявление допустимо, если тип `n` может быть определен.</span><span class="sxs-lookup"><span data-stu-id="e7514-104">The previous declaration is valid if the type of `n` can be inferred.</span></span> <span data-ttu-id="e7514-105">Например, если предыдущее лямбда-выражение назначается делегату функции `Del`:</span><span class="sxs-lookup"><span data-stu-id="e7514-105">For example, if you are assigning the previous lambda expression to a function delegate, `Del`:</span></span>  
  
```vb  
Delegate Function Del(ByVal p As Integer) As Integer  
```  
  
 <span data-ttu-id="e7514-106">Теперь тип `n` может быть выведен из параметра `p`:</span><span class="sxs-lookup"><span data-stu-id="e7514-106">Now the type of `n` can be inferred from parameter `p`:</span></span>  
  
```vb  
Dim increment2 as Del = Function(n) n + 1  
```  
  
 <span data-ttu-id="e7514-107">**Идентификатор ошибки:** BC36642</span><span class="sxs-lookup"><span data-stu-id="e7514-107">**Error ID:** BC36642</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e7514-108">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="e7514-108">To correct this error</span></span>  
  
- <span data-ttu-id="e7514-109">Добавьте предложение `As` в объявление параметра:</span><span class="sxs-lookup"><span data-stu-id="e7514-109">Add an `As` clause to the parameter declaration:</span></span>  
  
    ```vb  
    Dim increment3 = Function (n As Integer) n + 1  
    ```
