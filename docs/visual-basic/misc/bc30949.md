# <a name="array-initializer-cannot-be-specified-for-a-non-constant-dimension-use-the-empty-initializer-"></a><span data-ttu-id="2765b-102">Инициализатор массива не может быть указан для измерения, не являющегося константой; использовать пустой инициализатор "{}"</span><span class="sxs-lookup"><span data-stu-id="2765b-102">Array initializer cannot be specified for a non constant dimension; use the empty initializer '{}'</span></span>
<span data-ttu-id="2765b-103">Массив инициализирует размерность, неизвестную во время компиляции.</span><span class="sxs-lookup"><span data-stu-id="2765b-103">An array initializes a dimension that is not known at compile time.</span></span>  
  
 <span data-ttu-id="2765b-104">Следующий код вызывает эту ошибку.</span><span class="sxs-lookup"><span data-stu-id="2765b-104">The following code generates this error.</span></span>  
  
```vb  
Dim j As Integer  
Dim intArray As Integer = New Integer(1, j) {{0, 100}, {1,101}}  
```  
  
 <span data-ttu-id="2765b-105">Следующий код позволяет избежать этой ошибки.</span><span class="sxs-lookup"><span data-stu-id="2765b-105">The following code avoids the error.</span></span>  
  
```vb  
Dim intArray As Integer = New Integer(1, j) {}  
For i As Integer = 0 To j  
    intArray(0, i) = i  
    intArray(1, i) = 100 + i  
Next i  
```  
  
 <span data-ttu-id="2765b-106">**Идентификатор ошибки:** BC30949</span><span class="sxs-lookup"><span data-stu-id="2765b-106">**Error ID:** BC30949</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="2765b-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="2765b-107">To correct this error</span></span>  
  
- <span data-ttu-id="2765b-108">Если это возможно, укажите постоянную размерность в объявлении массива.</span><span class="sxs-lookup"><span data-stu-id="2765b-108">If possible, specify a constant dimension in the array declaration.</span></span>  
  
- <span data-ttu-id="2765b-109">Если вы не можете указать постоянную размерность, необходимо инициализировать массив с помощью цикла, когда переменная размерность становится известной.</span><span class="sxs-lookup"><span data-stu-id="2765b-109">If you cannot specify a constant dimension, then you must initialize the array using a loop when the nonconstant dimension becomes known.</span></span>
