# <a name="argument-argumentname-cannot-be-converted-to-a-numeric-value"></a>Невозможно преобразовать аргумент "\<argumentname>" в числовое значение
Предпринята попытка преобразовать переменную, например строку, в числовое значение.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1. Найдите преобразование, имеющее смысл.
