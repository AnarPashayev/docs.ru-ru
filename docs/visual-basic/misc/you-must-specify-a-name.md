# <a name="you-must-specify-a-name"></a>Необходимо указать имя.
При выполнении операций с файлами, таких как переименование файла, необходимо указать имя.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Укажите имя.
