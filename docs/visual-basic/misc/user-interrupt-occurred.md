# <a name="user-interrupt-occurred"></a>Прерывание со стороны пользователя

Пользователь нажал клавиши CTRL+BREAK или другую клавишу прерывания.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Повторите операцию.
