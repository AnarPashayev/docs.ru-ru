# <a name="option-option-cannot-be-followed-by--or---ignored"></a>за режимом "\<option>" не могут следовать символы "+" или "-"; пропускается

Параметр компилятора был указан со значением `+` или `-`; эти значения не поддерживаются.  
  
 Дополнительные сведения о параметрах компилятора и значениях, которые они могут принимать, см. в разделе [Visual Basic Compiler Options Listed Alphabetically](../reference/command-line-compiler/compiler-options-listed-alphabetically.md).  
  
 **Идентификатор ошибки:** BC2028  
  
## <a name="to-correct-this-error-and-use-the-option"></a>Исправление ошибки и использование параметра  
  
- Укажите параметр компилятора без значения.  
  
## <a name="to-correct-this-error-and-not-use-the-option"></a>Исправление ошибки без использования параметра  
  
- Удалите параметр компилятора.
