# <a name="instance-members-and-me-cannot-be-used-in-a-query-expression"></a><span data-ttu-id="c0db7-102">Нельзя использовать члены экземпляра и "Me" в выражении запроса</span><span class="sxs-lookup"><span data-stu-id="c0db7-102">Instance members and 'Me' cannot be used in a query expression</span></span>
<span data-ttu-id="c0db7-103">Запрос LINQ в `Structure` содержит ссылку на `Me` или на член экземпляра структуры.</span><span class="sxs-lookup"><span data-stu-id="c0db7-103">A LINQ query in a `Structure` includes a reference to `Me` or to an instance member of the structure.</span></span> <span data-ttu-id="c0db7-104">Ссылки на `Me` или на члены экземпляра не допускаются в выражениях запроса в `Structure`.</span><span class="sxs-lookup"><span data-stu-id="c0db7-104">References to `Me` or instance members are not allowed in query expressions within a `Structure`.</span></span>  
  
 <span data-ttu-id="c0db7-105">**Идентификатор ошибки:** BC36535</span><span class="sxs-lookup"><span data-stu-id="c0db7-105">**Error ID:** BC36535</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="c0db7-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="c0db7-106">To correct this error</span></span>  
  
1. <span data-ttu-id="c0db7-107">Создайте копию члена экземпляра или значения, возвращаемого по ссылке на `Me` , и используйте эту копию в выражении запроса, как показано в следующем примере.</span><span class="sxs-lookup"><span data-stu-id="c0db7-107">Create a copy of the instance member or the value returned by the reference to `Me` and use the copy in the query expression, as shown in the following example.</span></span>  
  
```vb  
Structure SampleStructure  
    Public SearchValue As Integer  
  
    Public Sub SetSearchValue(ByVal number As Integer)  
      SearchValue = number  
    End Sub  
  
    Public Sub GetData()  
        Dim sv = SearchValue  
        Dim SampleData = New Integer() {1, 2, 3, 4}  
        Dim query = From number In SampleData _  
                    Where number < sv  
    End Sub  
End Structure  
```
