# <a name="the-default-assembly-reference-reference-has-already-been-automatically-added-ignored"></a>Ссылка на сборку по умолчанию "\<ссылка >" уже была добавлена автоматически; игнорируется
Предпринята попытка добавить ссылку на сборку по умолчанию, но сборка уже добавлена, поэтому попытка игнорируется.  
  
 **Идентификатор ошибки:** BC2024  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Удалите лишний код.
