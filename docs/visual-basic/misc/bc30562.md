# <a name="name-is-ambiguous-between-declarations-in-modules-modulename1-and-modulename2"></a>" \<name> " является неоднозначным между объявлениями в модулях " \<modulename1> " и " \<modulename2> "

Вы указали неоднозначное имя, которое конфликтует с другим именем. Компилятор Visual Basic не имеет правил разрешения конфликтов; необходимо самостоятельно устранить неоднозначность имен.  
  
 **Идентификатор ошибки:** BC30562  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Укажите полное имя.
