 <span data-ttu-id="ae3cc-104">Вы можете инициализировать общую ссылочную переменную в структуре, и можете иметь не являющуюся общей ссылочную переменную без инициализации, как показано в следующих строках кода.</span><span class="sxs-lookup"><span data-stu-id="ae3cc-104">You can initialize a shared reference variable in a structure, and you can have a nonshared reference variable without initialization, as the following code lines show.</span></span>
