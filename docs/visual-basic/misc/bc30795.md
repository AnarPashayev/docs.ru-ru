# <a name="non-shared-members-in-a-structure-cannot-be-declared-new"></a><span data-ttu-id="48a75-102">Члены структуры, не являющиеся общими, не могут быть объявлены как "New"</span><span class="sxs-lookup"><span data-stu-id="48a75-102">Non-shared members in a Structure cannot be declared 'New'</span></span>

<span data-ttu-id="48a75-103">Не являющаяся общей переменная в структуре объявлена с использованием предложения `New` .</span><span class="sxs-lookup"><span data-stu-id="48a75-103">A nonshared variable in a structure is declared with a `New` clause.</span></span>  
  
 <span data-ttu-id="48a75-104">Вы можете инициализировать общую ссылочную переменную в структуре, и можете иметь не являющуюся общей ссылочную переменную без инициализации, как показано в следующих строках кода.</span><span class="sxs-lookup"><span data-stu-id="48a75-104">You can initialize a shared reference variable in a structure, and you can have a nonshared reference variable without initialization, as the following code lines show.</span></span>

 `Dim structVar2 As System.ApplicationException`  
  
 <span data-ttu-id="48a75-105">Однако вы не можете инициализировать не являющуюся общей ссылочную переменную в структуре.</span><span class="sxs-lookup"><span data-stu-id="48a75-105">However, you cannot initialize a nonshared reference variable in a structure.</span></span> <span data-ttu-id="48a75-106">Следующий код является неправильным.</span><span class="sxs-lookup"><span data-stu-id="48a75-106">The following code line is invalid.</span></span>
