# <a name="xml-comment-parameter-must-have-a-name-attribute"></a>Параметр комментария XML должен содержать атрибут name

Параметр комментария XML должен содержать атрибут "name". Комментарий XML будет проигнорирован.  
  
 В XML-теге отсутствует обязательный атрибут `name` .  
  
 **Идентификатор ошибки:** BC42308  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Укажите обязательный атрибут `name` для тега.
