<span data-ttu-id="ebfad-103">Параметр типа используется в качестве операнда для [IsNot Operator](../language-reference/operators/isnot-operator.md) , если параметр типа определен без ключевого слова [Class (Visual Basic)](../language-reference/statements/class-statement.md) , или конкретного имени класса в его списке ограничений.</span><span class="sxs-lookup"><span data-stu-id="ebfad-103">A type parameter is used as an operand for the [IsNot Operator](../language-reference/operators/isnot-operator.md) when the type parameter is defined without either the [Class (Visual Basic)](../language-reference/statements/class-statement.md) keyword or a specific class name in its constraint list.</span></span>  
  
 <span data-ttu-id="ebfad-104">Оператор`IsNot` сравнивает два ссылочных типа, чтобы определить, указывают ли они на разные экземпляры объекта в памяти.</span><span class="sxs-lookup"><span data-stu-id="ebfad-104">`IsNot` compares two reference types to determine whether they point to different object instances in memory.</span></span> <span data-ttu-id="ebfad-105">Он не может принимать операнд, который не является ссылочным типом, если только другой операнд не равен [Nothing](../language-reference/nothing.md).</span><span class="sxs-lookup"><span data-stu-id="ebfad-105">It cannot take an operand that is not a reference type unless the other operand is [Nothing](../language-reference/nothing.md).</span></span>  
  
 <span data-ttu-id="ebfad-106">**Идентификатор ошибки:** BC32097</span><span class="sxs-lookup"><span data-stu-id="ebfad-106">**Error ID:** BC32097</span></span>
