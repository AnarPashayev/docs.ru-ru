# <a name="type-of-variablename-cannot-be-inferred-from-an-expression-containing-variablename"></a><span data-ttu-id="5f3d5-102">Тип " \<variablename> " не может быть определен из выражения, содержащего " \<variablename> "</span><span class="sxs-lookup"><span data-stu-id="5f3d5-102">Type of '\<variablename>' cannot be inferred from an expression containing '\<variablename>'</span></span>

<span data-ttu-id="5f3d5-103">Компилятор не может получить тип данных переменной, если переменная используется в установке ее начального значения в объявлении.</span><span class="sxs-lookup"><span data-stu-id="5f3d5-103">The compiler cannot infer the data type of a variable if the variable is used in establishing its initial value in the declaration.</span></span>  
  
 <span data-ttu-id="5f3d5-104">Например, если параметр `Option Infer` имеет значение `On`, не компилируются следующие примеры:</span><span class="sxs-lookup"><span data-stu-id="5f3d5-104">For example, with `Option Infer` set to `On`, the following examples do not compile:</span></span>  
  
- <span data-ttu-id="5f3d5-105">Объявления</span><span class="sxs-lookup"><span data-stu-id="5f3d5-105">Declarations</span></span>  
  
    ```vb  
    ' Does not compile with Option Infer on.  
    Dim m = m  
    Dim d = someFunction(d)  
    ```  
  
- <span data-ttu-id="5f3d5-106">Цикл`For`</span><span class="sxs-lookup"><span data-stu-id="5f3d5-106">`For` loop</span></span>  
  
    ```vb  
    ' Does not compile with Option Infer on.  
    For j = 1 To j  
    Next  
    ```  
  
- <span data-ttu-id="5f3d5-107">Цикл`For Each`</span><span class="sxs-lookup"><span data-stu-id="5f3d5-107">`For Each` loop</span></span>  
  
    ```vb  
    ' Does not compile with Option Infer on.  
    For Each customer In customer.Orders  
    Next  
    ```  
  
 <span data-ttu-id="5f3d5-108">**Идентификатор ошибки:** BC30980</span><span class="sxs-lookup"><span data-stu-id="5f3d5-108">**Error ID:** BC30980</span></span>
