# <a name="unable-to-access-member"></a>Не удается получить доступ к члену
Указанный член недоступен.  
  
 **Идентификатор ошибки:** BC30749  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1. Убедитесь, что этот член определен, а его имя указано правильно.  
  
2. Попробуйте обратиться к одному из членов, объявленных в модуле. В некоторых случаях среда отладки не может найти члены, потому что модули, в которых они объявлены, еще не были загружены.
