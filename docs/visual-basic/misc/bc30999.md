<span data-ttu-id="c0c9b-103">Вы можете использовать символ продолжения строки (знак подчеркивания (_)), чтобы разделить одну длинную строку кода на несколько строк в файле.</span><span class="sxs-lookup"><span data-stu-id="c0c9b-103">You can use the line-continuation character, which is an underscore (_), to break a long line of code over several lines in your file.</span></span> <span data-ttu-id="c0c9b-104">Непосредственно перед знаком подчеркивания должен стоять пробел, а сразу за ним должен быть признак конца строки (возврат каретки).</span><span class="sxs-lookup"><span data-stu-id="c0c9b-104">The underscore must be immediately preceded by a space and immediately followed by a line terminator (carriage return).</span></span> <span data-ttu-id="c0c9b-105">Пример:</span><span class="sxs-lookup"><span data-stu-id="c0c9b-105">For example:</span></span>

```vb  
    XDocument.Load(My.Application.Info.DirectoryPath & _  
                 "\..\..\Data\books.xml")  
```  
  
 <span data-ttu-id="c0c9b-106">**Идентификатор ошибки:** BC30999</span><span class="sxs-lookup"><span data-stu-id="c0c9b-106">**Error ID:** BC30999</span></span>
