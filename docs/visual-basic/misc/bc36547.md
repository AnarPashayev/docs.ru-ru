# <a name="anonymous-type-member-or-property-propertyname-is-already-declared"></a><span data-ttu-id="35824-102">Анонимный член типа или свойство "\<propertyname>" уже объявлены</span><span class="sxs-lookup"><span data-stu-id="35824-102">Anonymous type member or property '\<propertyname>' is already declared</span></span>
<span data-ttu-id="35824-103">Имя свойства может быть установлено только один раз в объявлении анонимного типа.</span><span class="sxs-lookup"><span data-stu-id="35824-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="35824-104">Например, следующие объявления неправильны:</span><span class="sxs-lookup"><span data-stu-id="35824-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="35824-105">**Идентификатор ошибки:** BC36547</span><span class="sxs-lookup"><span data-stu-id="35824-105">**Error ID:** BC36547</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="35824-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="35824-106">To correct this error</span></span>  
  
- <span data-ttu-id="35824-107">Используйте другое имя для одного из свойств.</span><span class="sxs-lookup"><span data-stu-id="35824-107">Choose a different name for one of the properties.</span></span>  
  
```vb  
' Valid.  
Dim anonType3 = New With {.Name = "Debra Garcia", .Label = .Name & ", President"}  
```  
  
- <span data-ttu-id="35824-108">Укажите новые имена для переменных или имен свойств, из которых выводятся имена и значения.</span><span class="sxs-lookup"><span data-stu-id="35824-108">Provide new names for the variables or property names from which you are inferring names and values.</span></span>  
  
```vb  
' Valid.  
Dim anonType4 = New With {Key .ProductName = product.Name, Key .CarName = car1.Name}  
```
