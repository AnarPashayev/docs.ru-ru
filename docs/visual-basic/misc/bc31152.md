# <a name="xml-namespace-prefix-prefix-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>Префикс пространства имен XML "префикс" зарезервирован для использования в XML, поэтому не удается изменить URI пространства имен.
В XML-литерале использовался зарезервированный префикс пространства имен XML, такой как `xmlns`.  
  
 **Идентификатор ошибки:** BC31152  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Измените префикс пространства имен XML на не зарезервированное значение.
