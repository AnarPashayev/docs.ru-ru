# <a name="of-required-when-specifying-type-arguments-for-a-generic-type-or-method"></a>При задании аргументов типов для универсального типа или метода необходимо использовать Of
Оператор пытается создать тип из универсального типа или вызвать универсальный метод, не используя предложение [Of](../../visual-basic/language-reference/statements/of-clause.md) .  
  
 Вызывает синтаксис Visual Basic для универсальных типов для параметров типа и аргументов типа для указываемых `Of` ключевое слово. Кроме того, список параметров типов или список аргументов типов должен быть заключен в скобки.  
  
 **Идентификатор ошибки:** BC32093  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Включите ключевое слово `Of` в начало списка аргументов типов и заключите весь список в скобки.
