# <a name="cannot-save-file-to-temp"></a><span data-ttu-id="05b92-102">Невозможно сохранить файл в TEMP</span><span class="sxs-lookup"><span data-stu-id="05b92-102">Cannot save file to TEMP</span></span>

<span data-ttu-id="05b92-103">Не удается найти каталог с именем TEMP, либо на диске или в разделе, содержащем каталог TEMP, недостаточно места для сохранения информации.</span><span class="sxs-lookup"><span data-stu-id="05b92-103">Either a component cannot find a directory named TEMP, or the drive or partition containing the TEMP directory lacks sufficient space to save information.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="05b92-104">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="05b92-104">To correct this error</span></span>  
  
1. <span data-ttu-id="05b92-105">Создайте каталог с именем TEMP и задайте в качестве значения переменной среды TEMP путь к каталогу.</span><span class="sxs-lookup"><span data-stu-id="05b92-105">Create a directory named "TEMP" and set the TEMP environment variable equal to its path.</span></span>  
  
2. <span data-ttu-id="05b92-106">Освободите место на диске, удалив ненужные файлы, или создайте каталог TEMP в другом разделе и задайте в качестве значения переменной среды TEMP путь к каталогу.</span><span class="sxs-lookup"><span data-stu-id="05b92-106">Make space on the drive by erasing unnecessary files, or create a TEMP directory on another partition and set the TEMP environment variable equal to its path.</span></span>
