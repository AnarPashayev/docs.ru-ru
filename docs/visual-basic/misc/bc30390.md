# <a name="qualifiedmethod-is-not-accessible-in-this-context-because-it-is-modifier"></a>'\<уточненный_метод >' недоступен в этом контексте, так как он является '\<модификатор >'
Предпринята попытка обращения к методу, который недоступен в данном контексте из-за примененного к нему модификатора.  
  
 **Идентификатор ошибки:** BC30390  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Удалите этот модификатор.
