# <a name="field-fieldname-is-of-an-unsupported-type"></a>Поле '\<имя_поля >' имеет неподдерживаемый тип
Поле объявляется в качестве типа, который не может быть представлен Visual Studio.  
  
 **Идентификатор ошибки:** BC30656  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1. Укажите в определении поля поддерживаемый тип.  
  
2. Разработайте новый тип, описывающий данные поля.
