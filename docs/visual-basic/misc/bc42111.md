<span data-ttu-id="0fade-103">Для статических локальных переменных компилятор не выводит тип данных.</span><span class="sxs-lookup"><span data-stu-id="0fade-103">The compiler does not infer the data type of static local variables.</span></span> <span data-ttu-id="0fade-104">В следующем примере (для параметра `Option Strict` задано значение `Off`) тип `m` будет `Object`независимо от того, задано ли для `Option Infer` значение `On` или `Off`.</span><span class="sxs-lookup"><span data-stu-id="0fade-104">In the following example, with `Option Strict` set to `Off`, the type of `m` will be `Object`, regardless of whether `Option Infer` is set to `On` or `Off`.</span></span> <span data-ttu-id="0fade-105">Локальный вывод типов не производится.</span><span class="sxs-lookup"><span data-stu-id="0fade-105">Local type inference does not apply.</span></span>
