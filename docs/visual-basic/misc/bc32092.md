# <a name="type-or-new-expected"></a><span data-ttu-id="9d92e-102">Ожидается тип или New</span><span class="sxs-lookup"><span data-stu-id="9d92e-102">Type or 'New' expected</span></span>
<span data-ttu-id="9d92e-103">Параметр типа в объявлении универсального типа представляет список ограничений с ключевым словом `As` , но не задает допустимое ограничение.</span><span class="sxs-lookup"><span data-stu-id="9d92e-103">A type parameter in the declaration of a generic type introduces a constraint list with the `As` keyword but does not specify a valid constraint.</span></span>  
  
 <span data-ttu-id="9d92e-104">Ограничение в параметре типа должно быть допустимым классом или интерфейсом или одним из ключевых слов `Class`, `Structure`или `New`.</span><span class="sxs-lookup"><span data-stu-id="9d92e-104">A constraint on a type parameter must be a valid class or interface, or one of the keywords `Class`, `Structure`, or `New`.</span></span> <span data-ttu-id="9d92e-105">Если задано недопустимое ограничение или вообще не заданы никакие ограничения, компилятор создает эту ошибку.</span><span class="sxs-lookup"><span data-stu-id="9d92e-105">If you specify an invalid constraint or none at all, the compiler generates this error.</span></span>  
  
 <span data-ttu-id="9d92e-106">**Идентификатор ошибки:** BC32092</span><span class="sxs-lookup"><span data-stu-id="9d92e-106">**Error ID:** BC32092</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="9d92e-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="9d92e-107">To correct this error</span></span>  
  
1. <span data-ttu-id="9d92e-108">Определите, как параметр типа должен быть ограничен, и задайте соответствующее ограничение в списке ограничений.</span><span class="sxs-lookup"><span data-stu-id="9d92e-108">Determine how the type parameter should be constrained and specify the appropriate constraint in the constraint list.</span></span>  
  
2. <span data-ttu-id="9d92e-109">Если вы хотите ограничить параметр типа с помощью класса или интерфейса, убедитесь, что в ограничении он записан правильно.</span><span class="sxs-lookup"><span data-stu-id="9d92e-109">If you intend to constrain the type parameter by a class or interface, make sure the constraint spells it correctly.</span></span>  
  
3. <span data-ttu-id="9d92e-110">Не забудьте разделять несколько ограничений для одного параметра типа запятыми и заключать список ограничений в фигурные скобки (`{ }`).</span><span class="sxs-lookup"><span data-stu-id="9d92e-110">Remember to separate multiple constraints on a single type parameter with commas, and to enclose the constraint list in braces (`{ }`).</span></span>
