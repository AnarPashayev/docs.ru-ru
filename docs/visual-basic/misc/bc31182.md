# <a name="attribute-value-is-not-valid-expecting-value1-or-value2"></a>Недопустимое значение атрибута; требуется "значение1" или "значение2"
В литерале XML-документа указано недопустимое значение атрибута. Допустимые значения атрибутов для литерала XML-документа указаны в спецификации XML 1.0.  
  
 **Идентификатор ошибки:** BC31182  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Измените значение атрибута на значение, являющееся допустимым в соответствии со спецификацией XML 1.0.
