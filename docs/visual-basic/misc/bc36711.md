# <a name="argument-matching-parameter-parametername-narrows-to-typename"></a>Аргумент, соответствующий параметру " \<parametername> ", является узким до " \<typename> "

Выполнен вызов перегруженного метода, но компилятор не может найти метод, который можно вызывать без преобразования сужения. *Преобразование сужения* изменяет тип данных значения на тип, который не сможет содержать некоторые возможные значения.  
  
 **Идентификатор ошибки:** BC36711
