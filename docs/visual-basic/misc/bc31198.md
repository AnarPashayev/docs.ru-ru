# <a name="xml-literal-cannot-appear-here-unless-it-is-enclosed-in-parentheses"></a><span data-ttu-id="4fb1f-102">XML-литерал можно использовать здесь, только если заключить его в скобки</span><span class="sxs-lookup"><span data-stu-id="4fb1f-102">XML literal cannot appear here unless it is enclosed in parentheses</span></span>
<span data-ttu-id="4fb1f-103">Объявление XML-литерала используется в выражении в расположении, которое неоднозначно для компилятора Visual Basic.</span><span class="sxs-lookup"><span data-stu-id="4fb1f-103">An XML literal declaration is used in an expression in a location that is ambiguous to the Visual Basic compiler.</span></span> <span data-ttu-id="4fb1f-104">То есть компилятор Visual Basic не может определить, предназначен ли символ "меньше" (<) в качестве оператора сравнения или начала XML-литерала.</span><span class="sxs-lookup"><span data-stu-id="4fb1f-104">That is, the Visual Basic compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="4fb1f-105">Примером является следующий код:</span><span class="sxs-lookup"><span data-stu-id="4fb1f-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _  
                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```  
  
 <span data-ttu-id="4fb1f-106">**Идентификатор ошибки:** BC31198</span><span class="sxs-lookup"><span data-stu-id="4fb1f-106">**Error ID:** BC31198</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4fb1f-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="4fb1f-107">To correct this error</span></span>  
  
- <span data-ttu-id="4fb1f-108">Заключите объявление XML-литерала в скобки, как показано в следующем примере:</span><span class="sxs-lookup"><span data-stu-id="4fb1f-108">Enclose the XML literal declaration in parentheses, as shown in the following example:</span></span>  
  
```vb  
Dim queryResult = From element In elements _  
                  Where (<sample> Value</sample>) = "Value" _  
                  Select element  
```  
  
- <span data-ttu-id="4fb1f-109">Измените код, чтобы он ссылался на существующий XML-литерал, как показано в следующем примере:</span><span class="sxs-lookup"><span data-stu-id="4fb1f-109">Modify your code to refer to an existing XML literal, as shown in the following example:</span></span>  
  
```vb  
Dim queryResult = From element In elements _  
                  Where e.<sample>.Value = "Value" _  
                  Select element  
```
