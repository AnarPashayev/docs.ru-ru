# <a name="cannot-infer-a-common-type-for-the-first-and-second-operands-of-the-binary-if-operator"></a><span data-ttu-id="54c5f-102">Не удается определить общий тип первого и второго операндов бинарного оператора If</span><span class="sxs-lookup"><span data-stu-id="54c5f-102">Cannot infer a common type for the first and second operands of the binary 'If' operator</span></span>

<span data-ttu-id="54c5f-103">Не удается определить общий тип первого и второго операндов бинарного оператора If.</span><span class="sxs-lookup"><span data-stu-id="54c5f-103">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="54c5f-104">Один из них должен иметь расширяющее преобразование в другой.</span><span class="sxs-lookup"><span data-stu-id="54c5f-104">One must have a widening conversion to the other.</span></span>

<span data-ttu-id="54c5f-105">Бинарный оператор `If` требует наличия расширяющего преобразования между одним из аргументов и другим аргументом.</span><span class="sxs-lookup"><span data-stu-id="54c5f-105">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="54c5f-106">Например, из-за отсутствия расширяющего преобразования в любом направлении между `Integer` и `String`приведенный ниже код вызывает эту ошибку.</span><span class="sxs-lookup"><span data-stu-id="54c5f-106">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>

```vb
Dim first? As Integer
Dim second As String = "First is Nothing"
'' Not valid.
' Console.WriteLine(If(first, second))
```

<span data-ttu-id="54c5f-107">**Идентификатор ошибки:** BC33110</span><span class="sxs-lookup"><span data-stu-id="54c5f-107">**Error ID:** BC33110</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="54c5f-108">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="54c5f-108">To correct this error</span></span>

- <span data-ttu-id="54c5f-109">Предоставьте явное преобразование для одного из операндов в коде, если это возможно.</span><span class="sxs-lookup"><span data-stu-id="54c5f-109">Provide an explicit conversion for one of the operands, if that is possible in your code:</span></span>

  ```vb
  Console.WriteLine(If(first, CInt(second)))
  ```

- <span data-ttu-id="54c5f-110">Перепишите код, используя другую условную конструкцию.</span><span class="sxs-lookup"><span data-stu-id="54c5f-110">Rewrite the code by using a different conditional construction.</span></span>

  ```vb
  If first IsNot Nothing Then
      Console.WriteLine(first)
  Else
      Console.WriteLine(second)
  End If
  ```
