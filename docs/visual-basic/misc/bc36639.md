# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="ef0e2-102">Параметр «ByRef» '\<имя_параметра >' не может использоваться в лямбда-выражения</span><span class="sxs-lookup"><span data-stu-id="ef0e2-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>
<span data-ttu-id="ef0e2-103">Лямбда-выражение, объявленное в `Sub` или функции, не может использовать любые параметры `ByRef` этого `Sub` или функции.</span><span class="sxs-lookup"><span data-stu-id="ef0e2-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="ef0e2-104">Например, следующий код вызовет эту ошибку, поскольку параметр `ByRef` `n` используется в лямбда-выражении.</span><span class="sxs-lookup"><span data-stu-id="ef0e2-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```  
'' Not valid.   
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="ef0e2-105">**Идентификатор ошибки:** BC36639</span><span class="sxs-lookup"><span data-stu-id="ef0e2-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ef0e2-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="ef0e2-106">To correct this error</span></span>  
  
- <span data-ttu-id="ef0e2-107">Назначьте параметр `ByRef` локальной переменной и используйте эту локальную переменную в лямбда-выражении, как показано в следующем коде.</span><span class="sxs-lookup"><span data-stu-id="ef0e2-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>  
  
    ```  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
