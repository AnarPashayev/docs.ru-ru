# <a name="method-methodname-cannot-be-declared-partial-because-only-one-method-methodname-can-be-marked-partial"></a>Метод '\<имя_метода >' не может быть объявлен «Partial», так как только один метод '\<имя_метода >' может быть помечен как «Partial»
Не допускается иметь два объявления разделяемых методов с одним именем и сигнатурой.  
  
 **Идентификатор ошибки:** BC31433  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1. Измените имя или сигнатуру одного из объявлений разделяемого метода.
