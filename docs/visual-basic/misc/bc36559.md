# <a name="anonymous-type-member-property-propertyname-cannot-be-used-to-infer-the-type-of-another-member-property-because-the-type-of-propertyname-is-not-yet-established"></a><span data-ttu-id="e9591-102">Свойство члена анонимного типа "\<propertyname>" не может использоваться для выведения типа другого свойства члена, так как тип для "\<propertyname>" еще не установлен</span><span class="sxs-lookup"><span data-stu-id="e9591-102">Anonymous type member property '\<propertyname>' cannot be used to infer the type of another member property because the type of '\<propertyname>' is not yet established</span></span>

<span data-ttu-id="e9591-103">Пока тип свойства анонимного типа не установлен, он не может использоваться для установки типа другого свойства.</span><span class="sxs-lookup"><span data-stu-id="e9591-103">Until the type of an anonymous type property is established, it cannot be used to establish the type of another property.</span></span> <span data-ttu-id="e9591-104">Например, в следующем объявлении `.IDName = .LastName` недопустимо, поскольку еще не выполнена инициализация `.LastName` .</span><span class="sxs-lookup"><span data-stu-id="e9591-104">For example, in the following declaration `.IDName = .LastName` is not valid because `.LastName` has not yet been initialized.</span></span>

```vb
' Not valid.
Dim anon1 = New With {Key .IDName = .LastName, Key .LastName = "Jones"}
```

<span data-ttu-id="e9591-105">**Идентификатор ошибки:** BC36559</span><span class="sxs-lookup"><span data-stu-id="e9591-105">**Error ID:** BC36559</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="e9591-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="e9591-106">To correct this error</span></span>

- <span data-ttu-id="e9591-107">Установите тип свойства перед его использованием для инициализации другого свойства.</span><span class="sxs-lookup"><span data-stu-id="e9591-107">Establish the type of the property before using it to initialize another property.</span></span>

  ```vb
  Dim anon2 = New With {Key .LastName = "Jones", Key .IDName = .LastName}
  ```
