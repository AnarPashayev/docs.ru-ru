# <a name="anonymous-type-member-property-propertyname-cannot-be-used-to-infer-the-type-of-another-member-property-because-the-type-of-propertyname-is-not-yet-established"></a><span data-ttu-id="6ac89-102">Свойство члена анонимного типа '\<имя_свойства >' не может использоваться для определения типа другого свойства члена, так как тип '\<имя_свойства >' еще не установлен</span><span class="sxs-lookup"><span data-stu-id="6ac89-102">Anonymous type member property '\<propertyname>' cannot be used to infer the type of another member property because the type of '\<propertyname>' is not yet established</span></span>

<span data-ttu-id="6ac89-103">Пока тип свойства анонимного типа не установлен, он не может использоваться для установки типа другого свойства.</span><span class="sxs-lookup"><span data-stu-id="6ac89-103">Until the type of an anonymous type property is established, it cannot be used to establish the type of another property.</span></span> <span data-ttu-id="6ac89-104">Например, в следующем объявлении `.IDName = .LastName` недопустимо, поскольку еще не выполнена инициализация `.LastName` .</span><span class="sxs-lookup"><span data-stu-id="6ac89-104">For example, in the following declaration `.IDName = .LastName` is not valid because `.LastName` has not yet been initialized.</span></span>

```vb
' Not valid.
Dim anon1 = New With {Key .IDName = .LastName, Key .LastName = "Jones"}
```

<span data-ttu-id="6ac89-105">**Идентификатор ошибки:** BC36559</span><span class="sxs-lookup"><span data-stu-id="6ac89-105">**Error ID:** BC36559</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="6ac89-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="6ac89-106">To correct this error</span></span>

- <span data-ttu-id="6ac89-107">Установите тип свойства перед его использованием для инициализации другого свойства.</span><span class="sxs-lookup"><span data-stu-id="6ac89-107">Establish the type of the property before using it to initialize another property.</span></span>

