# <a name="method-methodname-does-not-have-a-signature-compatible-with-delegate-delegatename"></a><span data-ttu-id="36e97-102">Метод '\<имя_метода >' не имеет сигнатуры, совместимой с делегатом \<«имя_делегата» ></span><span class="sxs-lookup"><span data-stu-id="36e97-102">Method '\<methodname>' does not have a signature compatible with delegate \<'delegatename'></span></span>
<span data-ttu-id="36e97-103">Эта ошибка возникает, когда требуется преобразование между методом и делегатом, которое невозможно.</span><span class="sxs-lookup"><span data-stu-id="36e97-103">This error occurs when a conversion is required between a method and a delegate that is not possible.</span></span> <span data-ttu-id="36e97-104">Причиной ошибки может быть преобразование между параметрами или, когда метод и делегат являются функциями, преобразование в возвращаемые значения.</span><span class="sxs-lookup"><span data-stu-id="36e97-104">The cause of the error can be conversion between parameters or, when the method and delegate are functions, conversion in the return values.</span></span>  
  
 <span data-ttu-id="36e97-105">Следующий код иллюстрирует неудачные преобразования.</span><span class="sxs-lookup"><span data-stu-id="36e97-105">The following code illustrates failed conversions.</span></span> <span data-ttu-id="36e97-106">Делегат — `FunDel`.</span><span class="sxs-lookup"><span data-stu-id="36e97-106">The delegate is `FunDel`.</span></span>  
  
```vb  
Delegate Function FunDel(ByVal i As Integer, ByVal d As Double) As Integer  
```  
  
 <span data-ttu-id="36e97-107">Следующие функции отличаются от `FunDel` способом, который приведет к этой ошибке.</span><span class="sxs-lookup"><span data-stu-id="36e97-107">The following functions each differ from `FunDel` in a way that will cause this error.</span></span>  
  
```vb  
Function ExampleMethod1(ByVal m As Integer, ByVal aDate As Date) As Integer  
End Function  
  
Function ExampleMethod2(ByVal m As Integer, ByVal aDouble As Double) As Date  
End Function  
```  
  
 <span data-ttu-id="36e97-108">Каждый из следующих операторов назначения вызывает ошибку.</span><span class="sxs-lookup"><span data-stu-id="36e97-108">Each of the following assignment statements causes the error.</span></span>  
  
```vb  
Sub Main()  
    ' The second parameters of FunDel and ExampleMethod1, Double and Date,  
    ' are not compatible.  
    'Dim d1 As FunDel = AddressOf ExampleMethod1  
  
    ' The return types of FunDel and ExampleMethod2, Integer and Date,  
    ' are not compatible.  
    'Dim d2 As FunDel = AddressOf ExampleMethod2  
  
End Sub  
```  
  
 <span data-ttu-id="36e97-109">**Идентификатор ошибки:** BC31143</span><span class="sxs-lookup"><span data-stu-id="36e97-109">**Error ID:** BC31143</span></span>

- <span data-ttu-id="36e97-111">Проверьте соответствующие параметры и типы возврата (при их наличии), чтобы определить, какая пара несовместима.</span><span class="sxs-lookup"><span data-stu-id="36e97-111">Examine the corresponding parameters and, if they are present, return types to determine which pair is not compatible.</span></span>
