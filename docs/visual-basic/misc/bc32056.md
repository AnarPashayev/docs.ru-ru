<span data-ttu-id="b3967-103">Предложение `Implements` в универсальном типе определяет параметр типа как реализуемый член.</span><span class="sxs-lookup"><span data-stu-id="b3967-103">An `Implements` clause in a generic type specifies a type parameter as the member to be implemented.</span></span>
