# <a name="type-parameter-not-allowed-in-implements-clause"></a><span data-ttu-id="b1a9e-102">Параметр типа нельзя использовать в предложении Implements</span><span class="sxs-lookup"><span data-stu-id="b1a9e-102">Type parameter not allowed in 'Implements' clause</span></span>
<span data-ttu-id="b1a9e-103">Предложение `Implements` в универсальном типе определяет параметр типа как реализуемый член.</span><span class="sxs-lookup"><span data-stu-id="b1a9e-103">An `Implements` clause in a generic type specifies a type parameter as the member to be implemented.</span></span>

```

    Implements i1(Of t)

End Class  
```  
  
 <span data-ttu-id="b1a9e-107">**Идентификатор ошибки:** BC32056</span><span class="sxs-lookup"><span data-stu-id="b1a9e-107">**Error ID:** BC32056</span></span>
