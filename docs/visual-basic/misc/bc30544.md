# <a name="method-cannot-contain-both-a-try-statement-and-an-on-error-or-resume-statement"></a>Метод не может одновременно содержать оператор Try и оператор On Error или Resume.

Вы объединили оператор `Try` с `On Error` или `Resume`.  
  
 **Идентификатор ошибки:** BC30544  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1. Удалите `On Error` или `Resume`.
