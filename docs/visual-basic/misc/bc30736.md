# <a name="expression-cannot-be-evaluated-at-this-time"></a>В данный момент вычисление выражения невозможно
Указанное выражение не удалось вычислить в текущем контексте.  
  
 **Идентификатор ошибки:** BC30736  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Измените синтаксис выражения.
