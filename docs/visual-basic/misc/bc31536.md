# <a name="friend-access-was-denied-to-assembly-if-you-want-the-compiler-to-consider-friend-types-in-assembly-add-the-public-key-information-in-the-project-settings-or-through-a-command-line-switch-to-the-compiler-for-the-project-project"></a>Дружественный доступ был запрещен для \<assembly >. Если необходимо, чтобы компилятор рассматривал дружественные типы в \<assembly >, добавьте сведения об открытом ключе в параметры проекта или с помощью параметра командной строки в компилятор для проекта \<project >.

**Идентификатор ошибки:** BC31536
