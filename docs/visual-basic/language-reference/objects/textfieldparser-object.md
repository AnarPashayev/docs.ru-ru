## <a name="syntax"></a>Синтаксис  
  
```vb  
Public Class TextFieldParser  
```  
  
## <a name="remarks"></a>Заметки  
 Сведения о методах и свойствах объекта `TextFieldParser` см. в разделе <xref:Microsoft.VisualBasic.FileIO.TextFieldParser>.
