# <a name="mycomputerclipboard-object"></a>Объект My.Computer.Clipboard
Предоставляет методы для управления буфером обмена.  
  
## <a name="remarks"></a>Примечания  
 Сведения о методах и свойствах объекта `My.Computer.Clipboard` см. в разделе <xref:Microsoft.VisualBasic.MyServices.ClipboardProxy>.  
  
 Дополнительные сведения см. в разделе [Сохранение данных в буфер обмена и чтение из него](../../../visual-basic/developing-apps/programming/computer-resources/storing-data-to-and-reading-from-the-clipboard.md).  
  
> [!NOTE]
> Для управления буфером обмена можно также <xref:System.Windows.Forms.Clipboard?displayProperty=nameWithType> использовать методы класса.  
  
## <a name="requirements"></a>Требования  
 **Пространство имен:** <xref:Microsoft.VisualBasic.MyServices>  
  
 **Класс:** (предоставляет доступ к <xref:System.Windows.Forms.Clipboard>) <xref:Microsoft.VisualBasic.MyServices.ClipboardProxy>  
  
 **Сборок** Visual Basic (библиотека времени выполнения, в Microsoft.VisualBasic.dll)
