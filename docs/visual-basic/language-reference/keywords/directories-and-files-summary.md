# <a name="directories-and-files-summary-visual-basic"></a>Сводка по каталогам и файлам (Visual Basic)
Ключевые слова языка Visual Basic и члены библиотек времени выполнения упорядочены по назначению и использованию.  
  
 С помощью `My` возможность предоставляет большую производительность и быстродействие в операциях файлового ввода-вывода, чем использование этих элементов. Дополнительные сведения см. в разделе <xref:Microsoft.VisualBasic.FileIO.FileSystem>.  
  
|**Действие**|**Элемент языка**|  
|----------------|--------------------------|  
|Измените каталог или папку.|<xref:Microsoft.VisualBasic.FileSystem.ChDir%2A>|  
|Изменение диска.|<xref:Microsoft.VisualBasic.FileSystem.ChDrive%2A>|  
|Скопируйте файл.|<xref:Microsoft.VisualBasic.FileSystem.FileCopy%2A>|  
|Сделайте каталогом или папкой.|<xref:Microsoft.VisualBasic.FileSystem.MkDir%2A>|  
|Удалите каталог или папку.|<xref:Microsoft.VisualBasic.FileSystem.RmDir%2A>|  
|Переименование файла, каталога или папки.|<xref:Microsoft.VisualBasic.FileSystem.Rename%2A>|  
|Возвращает текущий путь.|<xref:Microsoft.VisualBasic.FileSystem.CurDir%2A>|  
|Возвращает метку даты и времени файла.|<xref:Microsoft.VisualBasic.FileSystem.FileDateTime%2A>|  
|Возврат файла, каталога или атрибуты метки.|<xref:Microsoft.VisualBasic.FileSystem.GetAttr%2A>|  
|Вернуть длину файла.|<xref:Microsoft.VisualBasic.FileSystem.FileLen%2A>|  
|Возврат файла имя или метки тома.|<xref:Microsoft.VisualBasic.FileSystem.Dir%2A>|  
|Задайте сведения об атрибутах для файла.|<xref:Microsoft.VisualBasic.FileSystem.SetAttr%2A>|
