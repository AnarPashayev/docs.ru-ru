### <a name="element-names"></a><span data-ttu-id="f2d12-139">Имена элементов</span><span class="sxs-lookup"><span data-stu-id="f2d12-139">Element Names</span></span>  
 <span data-ttu-id="f2d12-140">Если вы указали `element`, он должен представлять *элемент-контейнер*, то есть программный элемент, который может содержать другие элементы.</span><span class="sxs-lookup"><span data-stu-id="f2d12-140">If you supply `element`, it must represent a *container element*, that is, a programming element that can contain other elements.</span></span> <span data-ttu-id="f2d12-141">Элементы контейнера включают классы, структуры, модули, интерфейсы и перечисления.</span><span class="sxs-lookup"><span data-stu-id="f2d12-141">Container elements include classes, structures, modules, interfaces, and enumerations.</span></span>
