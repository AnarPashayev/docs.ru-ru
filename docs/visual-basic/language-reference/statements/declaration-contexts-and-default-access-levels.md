 <span data-ttu-id="b8f9c-105">*Контекст объявления* элемента программирования является область кода, в котором она объявлена.</span><span class="sxs-lookup"><span data-stu-id="b8f9c-105">The *declaration context* of a programming element is the region of code in which it is declared.</span></span> <span data-ttu-id="b8f9c-106">Это часто другой программный элемент, который затем вызывается *содержащий элемент*.</span><span class="sxs-lookup"><span data-stu-id="b8f9c-106">This is often another programming element, which is then called the *containing element*.</span></span>
