## <a name="using-the-of-keyword"></a><span data-ttu-id="5ed61-105">Использование ключевого слова of</span><span class="sxs-lookup"><span data-stu-id="5ed61-105">Using the Of Keyword</span></span>  
 <span data-ttu-id="5ed61-106">В следующем примере кода ключевое слово `Of` используется для определения структуры класса, принимающего два параметра типа.</span><span class="sxs-lookup"><span data-stu-id="5ed61-106">The following code example uses the `Of` keyword to define the outline of a class that takes two type parameters.</span></span> <span data-ttu-id="5ed61-107">Он *ограничивает* параметр `keyType` интерфейсом <xref:System.IComparable>, что означает, что в используемом коде должен быть указан аргумент типа, реализующий <xref:System.IComparable>.</span><span class="sxs-lookup"><span data-stu-id="5ed61-107">It *constrains* the `keyType` parameter by the <xref:System.IComparable> interface, which means the consuming code must supply a type argument that implements <xref:System.IComparable>.</span></span> <span data-ttu-id="5ed61-108">Это необходимо для того, чтобы процедура `add` могла вызвать метод <xref:System.IComparable.CompareTo%2A?displayProperty=nameWithType>.</span><span class="sxs-lookup"><span data-stu-id="5ed61-108">This is necessary so that the `add` procedure can call the <xref:System.IComparable.CompareTo%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="5ed61-109">Дополнительные сведения об ограничениях см. в разделе [Type List](../../../visual-basic/language-reference/statements/type-list.md).</span><span class="sxs-lookup"><span data-stu-id="5ed61-109">For more information on constraints, see [Type List](../../../visual-basic/language-reference/statements/type-list.md).</span></span>  
  
```vb  
Public Class Dictionary(Of entryType, keyType As IComparable)  
    Public Sub add(ByVal e As entryType, ByVal k As keyType)  
        Dim dk As keyType  
        If k.CompareTo(dk) = 0 Then  
        End If  
    End Sub  
    Public Function find(ByVal k As keyType) As entryType  
    End Function  
End Class  
```  
  
 <span data-ttu-id="5ed61-110">Если вы закончите предыдущее определение класса, можно создать из него разнообразные `dictionary` классы.</span><span class="sxs-lookup"><span data-stu-id="5ed61-110">If you complete the preceding class definition, you can construct a variety of `dictionary` classes from it.</span></span> <span data-ttu-id="5ed61-111">Типы, предоставляемые для `entryType` и `keyType` определяют, какой тип записи принадлежит классу и какой тип ключа он связывает с каждой записью.</span><span class="sxs-lookup"><span data-stu-id="5ed61-111">The types you supply to `entryType` and `keyType` determine what type of entry the class holds and what type of key it associates with each entry.</span></span> <span data-ttu-id="5ed61-112">Из-за ограничения необходимо указать, чтобы `keyType` тип, реализующий <xref:System.IComparable>.</span><span class="sxs-lookup"><span data-stu-id="5ed61-112">Because of the constraint, you must supply to `keyType` a type that implements <xref:System.IComparable>.</span></span>  
  
 <span data-ttu-id="5ed61-113">В следующем примере кода создается объект, содержащий `String` записи и связывающий ключ `Integer` с каждым из них.</span><span class="sxs-lookup"><span data-stu-id="5ed61-113">The following code example creates an object that holds `String` entries and associates an `Integer` key with each one.</span></span> <span data-ttu-id="5ed61-114">`Integer` реализует <xref:System.IComparable> и, следовательно, удовлетворяет ограничениям на `keyType`.</span><span class="sxs-lookup"><span data-stu-id="5ed61-114">`Integer` implements <xref:System.IComparable> and therefore satisfies the constraint on `keyType`.</span></span>
