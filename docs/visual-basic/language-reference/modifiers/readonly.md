> [!IMPORTANT]
> <span data-ttu-id="b7f99-120">Если тип данных переменной является ссылочным типом, например массивом или экземпляром класса, его члены можно изменить, даже если сама переменная `ReadOnly`.</span><span class="sxs-lookup"><span data-stu-id="b7f99-120">If the data type of the variable is a reference type, such as an array or a class instance, its members can be changed even if the variable itself is `ReadOnly`.</span></span> <span data-ttu-id="b7f99-121">Это показано в следующем примере.</span><span class="sxs-lookup"><span data-stu-id="b7f99-121">The following example illustrates this.</span></span>

<span data-ttu-id="b7f99-122">При инициализации массив, на который указывает `characterArray()`, содержит "x", "y" и "z".</span><span class="sxs-lookup"><span data-stu-id="b7f99-122">When initialized, the array pointed to by `characterArray()` holds "x", "y", and "z".</span></span> <span data-ttu-id="b7f99-123">Так как переменная `characterArray` `ReadOnly`, ее значение нельзя изменить после инициализации. то есть нельзя присвоить ему новый массив.</span><span class="sxs-lookup"><span data-stu-id="b7f99-123">Because the variable `characterArray` is `ReadOnly`, you cannot change its value once it is initialized; that is, you cannot assign a new array to it.</span></span> <span data-ttu-id="b7f99-124">Однако можно изменить значения одного или нескольких элементов массива.</span><span class="sxs-lookup"><span data-stu-id="b7f99-124">However, you can change the values of one or more of the array members.</span></span> <span data-ttu-id="b7f99-125">После вызова процедуры `ChangeArrayElement`, массив, на который указывает `characterArray()`, содержит "x", "M" и "z".</span><span class="sxs-lookup"><span data-stu-id="b7f99-125">Following a call to the procedure `ChangeArrayElement`, the array pointed to by `characterArray()` holds "x", "M", and "z".</span></span>

<span data-ttu-id="b7f99-126">Обратите внимание, что это похоже на объявление параметра процедуры как [ByVal](byval.md), что не позволяет процедуре изменять сам аргумент вызова, но позволяет ему изменять его члены.</span><span class="sxs-lookup"><span data-stu-id="b7f99-126">Note that this is similar to declaring a procedure parameter to be [ByVal](byval.md), which prevents the procedure from changing the calling argument itself but allows it to change its members.</span></span>
