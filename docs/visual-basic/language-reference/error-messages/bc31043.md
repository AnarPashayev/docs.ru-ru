# <a name="bc31043-arrays-declared-as-structure-members-cannot-be-declared-with-an-initial-size"></a><span data-ttu-id="7f254-102">BC31043: массивы, объявленные как члены структуры, не могут объявляться с начальным размером</span><span class="sxs-lookup"><span data-stu-id="7f254-102">BC31043: Arrays declared as structure members cannot be declared with an initial size</span></span>

<span data-ttu-id="7f254-103">Массив в структуре объявляется с начальным размером.</span><span class="sxs-lookup"><span data-stu-id="7f254-103">An array in a structure is declared with an initial size.</span></span> <span data-ttu-id="7f254-104">Нельзя инициализировать какой-либо элемент структуры, и объявление размера массива является одной из форм инициализации.</span><span class="sxs-lookup"><span data-stu-id="7f254-104">You cannot initialize any structure element, and declaring an array size is one form of initialization.</span></span>

<span data-ttu-id="7f254-105">**Идентификатор ошибки:** BC31043</span><span class="sxs-lookup"><span data-stu-id="7f254-105">**Error ID:** BC31043</span></span>

## <a name="example"></a><span data-ttu-id="7f254-106">Пример</span><span class="sxs-lookup"><span data-stu-id="7f254-106">Example</span></span>

<span data-ttu-id="7f254-107">Следующий пример приводит к возникновению ошибки BC31043:</span><span class="sxs-lookup"><span data-stu-id="7f254-107">The following example generates BC31043:</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="7f254-108">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="7f254-108">To correct this error</span></span>

1. <span data-ttu-id="7f254-109">Определите массив в структуре как динамический (без исходного размера).</span><span class="sxs-lookup"><span data-stu-id="7f254-109">Define the array in your structure as dynamic (no initial size).</span></span>

2. <span data-ttu-id="7f254-110">Если требуется определенный размер массива, можно изменить размер динамического массива с помощью [оператора ReDim](../statements/redim-statement.md) при выполнении кода.</span><span class="sxs-lookup"><span data-stu-id="7f254-110">If you require a certain size of array, you can redimension a dynamic array with a [ReDim Statement](../statements/redim-statement.md) when your code is running.</span></span> <span data-ttu-id="7f254-111">Проиллюстрируем это на примере.</span><span class="sxs-lookup"><span data-stu-id="7f254-111">The following example illustrates this:</span></span>

    ```vb
    Structure DemoStruct
        Public demoArray() As Integer
    End Structure
    Sub UseStruct()
        Dim struct As DemoStruct
        ReDim struct.demoArray(9)
        Struct.demoArray(2) = 777
    End Sub
    ```
