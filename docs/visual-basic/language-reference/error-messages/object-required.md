# <a name="object-required-visual-basic"></a>Требуется объект (Visual Basic)
Ссылки на свойства и методы часто требуют явного квалификатора объекта. Это такой случай.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1. Убедитесь, что ссылки на свойство или метод квалификатор допустимый объект. Укажите квалификатор объекта, если вы не укажете другое значение.  
  
2. Проверьте правильность написания квалификатора объекта и убедитесь, что объект является видимым в части программы, в котором содержится ссылка на него.  
  
3. Если путь указано в ведущее приложение **Открытие файла** команды, проверьте правильность аргументов в нем.  
  
4. Просмотрите документацию объекта и убедитесь, что действие является допустимым.
