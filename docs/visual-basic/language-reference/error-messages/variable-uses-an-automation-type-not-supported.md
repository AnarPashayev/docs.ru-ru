# <a name="variable-uses-an-automation-type-not-supported-in-visual-basic"></a>Переменная использует тип автоматизации, не поддерживаемый в Visual Basic

Вы попытались использовать переменную, определенную в библиотеке типов или библиотекой объектов, с типом данных, не поддерживается в Visual Basic.

## <a name="to-correct-this-error"></a>Исправление ошибки

- Используйте переменную типа распознаются Visual Basic.

     -или-

- Если эта ошибка возникает при использовании `FileGet` или `FileGetObject`, убедитесь, что вы пытаетесь использовать записи в файл с `FilePut` или `FilePutObject`.
