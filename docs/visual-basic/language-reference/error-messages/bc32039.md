# <a name="bc32039-array-declared-as-for-loop-control-variable-cannot-be-declared-with-an-initial-size"></a><span data-ttu-id="182f3-102">BC32039: массив, объявленный как управляющая переменная цикла for, не может быть объявлен с начальным размером</span><span class="sxs-lookup"><span data-stu-id="182f3-102">BC32039: Array declared as for loop control variable cannot be declared with an initial size</span></span>

<span data-ttu-id="182f3-103">`For Each`Цикл использует массив в качестве переменной итерации, но инициализирует этот массив.</span><span class="sxs-lookup"><span data-stu-id="182f3-103">A `For Each` loop uses an array as its iteration variable but initializes that array.</span></span>

<span data-ttu-id="182f3-104">**Идентификатор ошибки:** BC32039</span><span class="sxs-lookup"><span data-stu-id="182f3-104">**Error ID:** BC32039</span></span>

## <a name="example"></a><span data-ttu-id="182f3-105">Пример</span><span class="sxs-lookup"><span data-stu-id="182f3-105">Example</span></span>

<span data-ttu-id="182f3-106">Следующий пример приводит к возникновению ошибки BC32039:</span><span class="sxs-lookup"><span data-stu-id="182f3-106">The following example generates bc32039:</span></span>

```vb

Next
```

## <a name="to-correct-this-error"></a><span data-ttu-id="182f3-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="182f3-107">To correct this error</span></span>


<span data-ttu-id="182f3-109">также можно использовать вывод типа:</span><span class="sxs-lookup"><span data-stu-id="182f3-109">or you can use type inference:</span></span>
