# <a name="array-declared-as-for-loop-control-variable-cannot-be-declared-with-an-initial-size"></a><span data-ttu-id="755c0-102">Массив, объявленный как переменная управления циклом, не может быть объявлен с исходным размером</span><span class="sxs-lookup"><span data-stu-id="755c0-102">Array declared as for loop control variable cannot be declared with an initial size</span></span>

<span data-ttu-id="755c0-103">Цикл `For Each` использует массив в качестве переменной итерации, но инициализирует этот массив.</span><span class="sxs-lookup"><span data-stu-id="755c0-103">A `For Each` loop uses an array as its iteration variable but initializes that array.</span></span>

<span data-ttu-id="755c0-104">**Идентификатор ошибки:** BC32039</span><span class="sxs-lookup"><span data-stu-id="755c0-104">**Error ID:** BC32039</span></span>

## <a name="example"></a><span data-ttu-id="755c0-105">Пример</span><span class="sxs-lookup"><span data-stu-id="755c0-105">Example</span></span>

<span data-ttu-id="755c0-106">Следующий пример приводит к возникновению ошибки BC32039:</span><span class="sxs-lookup"><span data-stu-id="755c0-106">The following example generates bc32039:</span></span>

```vb  
Dim arrayList As New List(Of Integer())
For Each listElement(1) As Integer In arrayList

Next
```  

## <a name="to-correct-this-error"></a><span data-ttu-id="755c0-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="755c0-107">To correct this error</span></span>

Next
```

<span data-ttu-id="755c0-109">также можно использовать вывод типа:</span><span class="sxs-lookup"><span data-stu-id="755c0-109">or you can use type inference:</span></span>

```
