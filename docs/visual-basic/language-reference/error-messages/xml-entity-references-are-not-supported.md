# <a name="xml-entity-references-are-not-supported"></a>Ссылки на XML-сущности не поддерживаются
Ссылка на сущность (например, `©`), не определенный в XML 1.0 спецификация включен в качестве значения для XML-литерала. Только `&`, `"`, `<`, `>`, и `'` ссылки на сущности XML поддерживаются в XML-литералов.  
  
 **Идентификатор ошибки:** BC31180  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Удалите ссылки на сущности не поддерживается.
