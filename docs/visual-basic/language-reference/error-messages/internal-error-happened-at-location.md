# <a name="internal-error-happened-at-location"></a>Произошла внутренняя ошибка в расположении \<
Произошла внутренняя ошибка. Строка, в которой она была выполнена, содержится в сообщении об ошибке.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Убедитесь, что эта ошибка не была создана оператором `Error` или методом `Raise`. Если это не так, обратитесь в службу технической поддержки Майкрософт, чтобы сообщить о требованиях, при которых оно появилось.
