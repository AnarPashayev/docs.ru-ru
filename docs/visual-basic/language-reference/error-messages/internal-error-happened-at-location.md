# <a name="internal-error-happened-at-location"></a>Произошла внутренняя ошибка в \<расположение >
Произошла внутренняя ошибка. В сообщении об ошибке содержится строке, в котором она произошла.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Убедитесь, что эта ошибка не была создана `Error` инструкции или `Raise` метод; Если не удалось, обратитесь в службу технической поддержки Майкрософт, чтобы сообщить об условиях, при которых появилось это сообщение.
