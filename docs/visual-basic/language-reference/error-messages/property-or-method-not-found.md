# <a name="property-or-method-not-found"></a>Свойство или метод не найдены
Указанный метод или свойство объекта не определено.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Проверьте правильность написания имени объекта. Чтобы узнать, какие свойства и методы, определенные для объекта, откройте обозреватель объектов. Выберите библиотеку, подходящий объект для просмотра списка доступных свойств и методов.
