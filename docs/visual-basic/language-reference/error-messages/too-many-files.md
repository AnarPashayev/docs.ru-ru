# <a name="too-many-files"></a>Слишком много файлов
В корневом каталоге создано либо больше файлов, чем разрешено операционной системой, либо несколько файлов были открыты, чем указано в файле конфигурации в параметре **Files =** . SYS.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1. Если программа открывает, закрывает или сохраняет файлы в корневом каталоге, измените программу так, чтобы она использовала подкаталог.  
  
2. Увеличьте число файлов, указанных в параметре **Files =** , в файле конфигурации. SYS и перезагрузить компьютер.
