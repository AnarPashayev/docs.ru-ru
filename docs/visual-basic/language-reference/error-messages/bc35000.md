# <a name="bc35000-requested-operation-is-not-available-because-the-runtime-library-function-function-is-not-defined"></a><span data-ttu-id="053e8-102">BC35000: запрошенная операция недоступна, так как функция библиотеки времени выполнения " \<function> " не определена.</span><span class="sxs-lookup"><span data-stu-id="053e8-102">BC35000: Requested operation is not available because the runtime library function '\<function>' is not defined.</span></span>

<span data-ttu-id="053e8-103">Visual Basic пытается выполнить внутренний вызов функции в среде выполнения Visual Basic (*Microsoft.VisualBasic.dll*), которая не может быть найдена.</span><span class="sxs-lookup"><span data-stu-id="053e8-103">Visual Basic is attempting to make an internal call to a function in the Visual Basic Runtime (*Microsoft.VisualBasic.dll*) that cannot be found.</span></span>

<span data-ttu-id="053e8-104">**Идентификатор ошибки:** BC35000</span><span class="sxs-lookup"><span data-stu-id="053e8-104">**Error ID:** BC35000</span></span>

<span data-ttu-id="053e8-105">Эта ошибка возникает в проектах в стиле пакета SDK (в проектах с расширением \* \* VBPROJ\* , которые начинаются со строки `<Project Sdk="Microsoft.NET.Sdk">` ).</span><span class="sxs-lookup"><span data-stu-id="053e8-105">This error occurs in SDK-style projects (projects with a *\*.vbproj* file that begins with the line `<Project Sdk="Microsoft.NET.Sdk">`).</span></span> <span data-ttu-id="053e8-106">По умолчанию в сборку приложения внедряется только подмножество *Microsoft.VisualBasic.dll* сборки и *\<function>* не включается в это подмножество.</span><span class="sxs-lookup"><span data-stu-id="053e8-106">By default, only a subset of the *Microsoft.VisualBasic.dll* assembly is embedded in the application assembly, and *\<function>* is not included in that subset.</span></span>
