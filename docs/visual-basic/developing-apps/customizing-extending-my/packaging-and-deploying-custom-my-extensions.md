### <a name="add-the-customdatasignature-element-to-the-vstemplate-file"></a><span data-ttu-id="61afb-144">Добавить \<CustomDataSignature > элемент VSTEMPLATE-файл</span><span class="sxs-lookup"><span data-stu-id="61afb-144">Add the \<CustomDataSignature> element to the .vstemplate file</span></span>

<span data-ttu-id="61afb-145">Для идентификации элемента шаблона Visual Studio как `My` расширение пространства имен, необходимо также изменить файл VSTEMPLATE для шаблона элемента.</span><span class="sxs-lookup"><span data-stu-id="61afb-145">To identify your Visual Studio item template as a `My` namespace extension, you must also modify the .vstemplate file for your item template.</span></span> <span data-ttu-id="61afb-146">Необходимо добавить `<CustomDataSignature>` элемент `<TemplateData>` элемент.</span><span class="sxs-lookup"><span data-stu-id="61afb-146">You must add a `<CustomDataSignature>` element to the `<TemplateData>` element.</span></span> <span data-ttu-id="61afb-147">`<CustomDataSignature>` Элемент должен содержать текст `Microsoft.VisualBasic.MyExtension`, как показано в следующем примере.</span><span class="sxs-lookup"><span data-stu-id="61afb-147">The `<CustomDataSignature>` element must contain the text `Microsoft.VisualBasic.MyExtension`, as shown in the following example.</span></span>

```xml
<CustomDataSignature>Microsoft.VisualBasic.MyExtension</CustomDataSignature>
```

<span data-ttu-id="61afb-148">Нельзя напрямую изменять файлы в сжатую папку (ZIP-файл).</span><span class="sxs-lookup"><span data-stu-id="61afb-148">You cannot modify files in a compressed folder (.zip file) directly.</span></span> <span data-ttu-id="61afb-149">Необходимо скопировать файл VSTEMPLATE из сжатой папки, измените его и затем заменить файл VSTEMPLATE в сжатой папке обновленной версией.</span><span class="sxs-lookup"><span data-stu-id="61afb-149">You must copy the .vstemplate file from the compressed folder, modify it, and then replace the .vstemplate file in the compressed folder with your updated copy.</span></span>

<span data-ttu-id="61afb-150">В следующем примере показано содержимое VSTEMPLATE-файл, имеющий `<CustomDataSignature>` добавлен указанный элемент.</span><span class="sxs-lookup"><span data-stu-id="61afb-150">The following example shows the contents of a .vstemplate file that has the `<CustomDataSignature>` element added.</span></span>

```xml

  <TemplateData>
    <DefaultName>MyCustomExtensionModule.vb</DefaultName>

    <ProjectItem SubType="Code"
                 TargetFileName="$fileinputname$.vb"
                 ReplaceParameters="true"
     >MyCustomExtensionModule.vb</ProjectItem>
  </TemplateContent>
```
