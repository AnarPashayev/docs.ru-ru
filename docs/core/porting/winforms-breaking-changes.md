# <a name="breaking-changes-in-windows-forms-net-framework-to-net-core"></a>Критические изменения в Windows Forms (миграция с .NET Framework на .NET Core)

Поддержка WPF и Windows Forms была добавлена в .NET Core в версии 3.0. Если вы переносите приложение Windows Forms или Windows Presentation Foundation с .NET Framework на .NET Core, критические изменения, перечисленные в этой статье, могут повлиять на работу приложения.

Критические изменения группируются по версии .NET Core, в которой они появились.
