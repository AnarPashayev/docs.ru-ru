# <a name="download-and-install-net-core"></a>Загрузка и установка .NET Core

В этой статье содержатся сведения о загрузке и установке .NET Core. .NET Core состоит из двух компонентов: **среды выполнения**, которая используется для запуска приложений, и **пакета SDK**, использующегося для создания приложений. Пакет SDK включает среду выполнения.

- Если вы — пользователь, и .NET Core требуется вам для запуска приложений, см. статью [Установка среды выполнения .NET Core](runtime.md).

- Если вы — разработчик, и .NET Core требуется вам для создания приложений, см. статью [Установка пакета SDK .NET Core](sdk.md).

> [!NOTE]
> Пакет SDK .NET Core включает среду выполнения.

## <a name="dependencies"></a>Зависимости

.NET Core поддерживается в Windows, Linux и macOS. Полный список требований см. в разделе [Поддерживаемые операционные системы](dependencies.md).
