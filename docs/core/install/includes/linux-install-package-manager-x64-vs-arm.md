_Установка диспетчера пакетов поддерживается только в архитектуре **x64**_. Для других архитектур, например **ARM**, нужно вручную установить пакет SDK для .NET Core или среду выполнения .NET Core. Дополнительные сведения см. в разделе [Установка вручную](#manual-install) ниже.
