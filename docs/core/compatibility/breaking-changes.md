# <a name="net-core-breaking-changes"></a>Критические изменения .NET Core

Следующее средство выбора версий содержит отфильтрованный список критических изменений для .NET Core, ASP.NET Core и EF Core. Целевые версии включают не только основные выпуски, такие как .NET Core 2.2 и .NET Core 3.0, но также выпуски предварительной версии, например .NET Core 3.0 (предварительная версия 7).

Вы также можете просмотреть отдельные вопросы, касающиеся критических изменений, в следующих репозиториях GitHub:

- Для .NET Core — репозиторий [dotnet/docs](https://github.com/dotnet/docs/issues?q=is%3Aopen+is%3Aissue+label%3Abreaking-change).
- Для ASP.NET Core — репозиторий [aspnet/AspNetCore](https://github.com/aspnet/AspNetCore/issues?q=is%3Aopen+is%3Aissue+label%3Abreaking-change).
- Для Entity Framework Core — репозиторий [aspnet/EntityFrameworkCore](https://github.com/aspnet/EntityFrameworkCore/issues?q=is%3Aopen+is%3Aissue+label%3Abreaking-change).
