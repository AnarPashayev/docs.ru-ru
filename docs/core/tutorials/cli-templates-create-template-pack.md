<span data-ttu-id="10ef1-137">Затем откройте файл _templatepack.csproj_ в редакторе и замените содержимое следующим кодом XML:</span><span class="sxs-lookup"><span data-stu-id="10ef1-137">Next, open the _templatepack.csproj_ file in your favorite editor and replace the content with the following XML:</span></span>
