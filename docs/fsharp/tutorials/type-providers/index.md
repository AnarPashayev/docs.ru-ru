# <a name="type-providers"></a><span data-ttu-id="ef643-103">Поставщики типов</span><span class="sxs-lookup"><span data-stu-id="ef643-103">Type Providers</span></span>

<span data-ttu-id="ef643-104">Поставщик типов F# — это компонент, предоставляющий типы, свойства и методы для использования в программе.</span><span class="sxs-lookup"><span data-stu-id="ef643-104">An F# type provider is a component that provides types, properties, and methods for use in your program.</span></span> <span data-ttu-id="ef643-105">Поставщики типов создавать так называемые **типы, предоставляемые**, которой создаются F# компилятора того на них в зависимости от внешнего источника данных.</span><span class="sxs-lookup"><span data-stu-id="ef643-105">Type Providers generate what are known as **Provided Types**, which are generated by the F# compiler and are based on an external data source.</span></span>

<span data-ttu-id="ef643-106">Например F# тип поставщика для SQL можно создать типы, представляющие таблицы и столбцы в реляционной базе данных.</span><span class="sxs-lookup"><span data-stu-id="ef643-106">For example, an F# Type Provider for SQL can generate types representing tables and columns in a relational database.</span></span> <span data-ttu-id="ef643-107">На самом деле, что это [SQLProvider](https://fsprojects.github.io/SQLProvider/) и поставщик типа.</span><span class="sxs-lookup"><span data-stu-id="ef643-107">In fact, this is what the [SQLProvider](https://fsprojects.github.io/SQLProvider/) Type Provider does.</span></span>

<span data-ttu-id="ef643-108">При условии, что типы зависят от входных параметров для поставщика типов.</span><span class="sxs-lookup"><span data-stu-id="ef643-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="ef643-109">Такие входные данные могут быть образец источника данных (например, JSON файл схемы), URL-адрес, указывающий непосредственно на внешней службы или строку подключения к источнику данных.</span><span class="sxs-lookup"><span data-stu-id="ef643-109">Such input can be a sample data source (such as a JSON schema file), a URL pointing directly to an external service, or a connection string to a data source.</span></span> <span data-ttu-id="ef643-110">Поставщик типов также можно обеспечить, что групп типов только расширяются по запросу; то есть они имеют расширенные типы фактически ссылки в программе.</span><span class="sxs-lookup"><span data-stu-id="ef643-110">A Type Provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="ef643-111">Это позволяет применять прямую интеграцию по запросу широкомасштабных информационных пространств (например, рынков оперативных данных) в строго типизированном виде.</span><span class="sxs-lookup"><span data-stu-id="ef643-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

## <a name="generative-and-erased-type-providers"></a><span data-ttu-id="ef643-112">Поставщики генерирующих и удаленных типов</span><span class="sxs-lookup"><span data-stu-id="ef643-112">Generative and Erased Type Providers</span></span>

<span data-ttu-id="ef643-113">Поставщики типов имеют две формы: Генерирующих и удаленных.</span><span class="sxs-lookup"><span data-stu-id="ef643-113">Type Providers come in two forms: Generative and Erased.</span></span>

<span data-ttu-id="ef643-114">Генерирующих поставщиков типов создавать типы, которые могут быть написаны как типы .NET в сборку, в котором они созданы.</span><span class="sxs-lookup"><span data-stu-id="ef643-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="ef643-115">Это позволяет им для использования из кода в других сборках.</span><span class="sxs-lookup"><span data-stu-id="ef643-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="ef643-116">Это означает, что типизированное представление источника данных, обычно необходимо, возможно, для представления с типами .NET.</span><span class="sxs-lookup"><span data-stu-id="ef643-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>

<span data-ttu-id="ef643-117">Стирания поставщиков типов создавать типы, которые могут использоваться только в сборке, или они формируются из проекта.</span><span class="sxs-lookup"><span data-stu-id="ef643-117">Erasing Type Providers produce types that can only be consumed in the assembly or project they are generated from.</span></span> <span data-ttu-id="ef643-118">Типы являются временными; то есть они не записываются в сборку и не может использоваться в коде в других сборках.</span><span class="sxs-lookup"><span data-stu-id="ef643-118">The types are ephemeral; that is, they are not written into an assembly and cannot be consumed by code in other assemblies.</span></span> <span data-ttu-id="ef643-119">Они могут содержать *отложенный* члены, позволяя использовать типы из пространства потенциально бесконечный сведения.</span><span class="sxs-lookup"><span data-stu-id="ef643-119">They can contain *delayed* members, allowing you to use provided types from a potentially infinite information space.</span></span> <span data-ttu-id="ef643-120">Они полезны для использования небольшое подмножество источником большой и взаимосвязанных данных.</span><span class="sxs-lookup"><span data-stu-id="ef643-120">They are useful for using a small subset of a large and interconnected data source.</span></span>
