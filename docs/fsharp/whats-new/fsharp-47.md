# <a name="whats-new-in-f-47"></a>Что нового в ФЗ 4.7

ФЗ 4.7 добавляет несколько улучшений в язык F..

## <a name="get-started"></a>Начало работы

ФЗ 4.7 доступен во всех дистрибутивах .NET Core и инструментарии Visual Studio. [Начало работы с ФЗ,](../get-started/index.md) чтобы узнать больше.

## <a name="language-version"></a>Версия языка

Компилятор F'4.7 вводит возможность настройки эффективной языковой версии через свойство в файле проекта:

```xml
<PropertyGroup>
    <LangVersion>preview</LangVersion>
</PropertyGroup>
```

Вы можете установить его `4.6` `4.7`на `latest`значения, и `preview`. Значение по умолчанию — `latest`.

Если вы установите `preview`его, ваш компилятор активирует все функции предварительного просмотра F, которые реализованы в компиляторе.

## <a name="implicit-yields"></a>Неявные урожаи

Вам больше не нужно `yield` применять ключевое слово в массивах, списках, последовательностях или выражениях вычислений, в которых можно сделать вывод о типе. В следующем примере оба выражения `yield` требовали оператора для каждой записи до 4.7 F.7:

```fsharp
let s = seq { 1; 2; 3; 4; 5 }

let daysOfWeek includeWeekend =
    [
        "Monday"
        "Tuesday"
        "Wednesday"
        "Thursday"
        "Friday"
        if includeWeekend then
            "Saturday"
            "Sunday"
    ]
```

Если вы введете одно `yield` ключевое слово, каждый другой элемент должен также `yield` применяться к нему.

Неявные урожаи не активируются `yield!` при использовании в выражении, которое также использует для сглаживания последовательности. Вы должны продолжать `yield` использовать сегодня в этих случаях.

## <a name="wildcard-identifiers"></a>Идентификаторы Wildcard

В коде ФЗ, включающем классы, самоисждатель должен всегда быть явным в декларациях членов. Но в тех случаях, когда самоижденитель никогда не используется, традиционно было условно использовать двойной показатель для обозначения безымянных самоиспоминовемых. Теперь вы можете использовать один подчеркнуть:

```fsharp
type C() =
    member _.M() = ()
```

Это также относится к `for` циклам:

```fsharp
for _ in 1..10 do printfn "Hello!"
```

## <a name="indentation-relaxations"></a>Отклонения отступов

До 4,7 ФЗ требования к отступам для первичного конструктора и статических аргументов членов требовали чрезмерного отступа. Теперь они требуют только одной области отступов:

```fsharp
type OffsideCheck(a:int,
    b:int, c:int,
    d:int) = class end

type C() =
    static member M(a:int,
        b:int, c:int,
        d:int) = 1
```
