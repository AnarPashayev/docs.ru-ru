<span data-ttu-id="cbb34-106">*Элементы* являются компонентами, которые входят в состав определения типа и объявляются с помощью ключевого слова `member`.</span><span class="sxs-lookup"><span data-stu-id="cbb34-106">*Members* are features that are part of a type definition and are declared with the `member` keyword.</span></span> <span data-ttu-id="cbb34-107">Типы объектов F#, такие как записи, классы, размеченные объединения, интерфейсы и структуры, поддерживают элементы.</span><span class="sxs-lookup"><span data-stu-id="cbb34-107">F# object types such as records, classes, discriminated unions, interfaces, and structures support members.</span></span> <span data-ttu-id="cbb34-108">Дополнительные сведения см. в статьях [Записи](../records.md), [Классы](../classes.md), [Размеченные объединения](../discriminated-Unions.md), [Интерфейсы](../interfaces.md) и [Структуры](../structures.md).</span><span class="sxs-lookup"><span data-stu-id="cbb34-108">For more information, see [Records](../records.md), [Classes](../classes.md), [Discriminated Unions](../discriminated-Unions.md), [Interfaces](../interfaces.md), and [Structures](../structures.md).</span></span>

<span data-ttu-id="cbb34-109">Обычно элементы составляют открытый интерфейс для типа, поэтому они считаются открытыми, если не указано иное.</span><span class="sxs-lookup"><span data-stu-id="cbb34-109">Members typically make up the public interface for a type, which is why they are public unless otherwise specified.</span></span> <span data-ttu-id="cbb34-110">Элементы также можно объявлять как закрытые или внутренние.</span><span class="sxs-lookup"><span data-stu-id="cbb34-110">Members can also be declared private or internal.</span></span> <span data-ttu-id="cbb34-111">Дополнительные сведения см. в статье [Управление доступом](../access-Control.md).</span><span class="sxs-lookup"><span data-stu-id="cbb34-111">For more information, see [Access Control](../access-Control.md).</span></span> <span data-ttu-id="cbb34-112">Сигнатуры для типов также можно использовать, чтобы предоставлять или не предоставлять определенные элементы типа.</span><span class="sxs-lookup"><span data-stu-id="cbb34-112">Signatures for types can also be used to expose or not expose certain members of a type.</span></span> <span data-ttu-id="cbb34-113">Дополнительные сведения см. в статье [Сигнатуры](../signatures.md).</span><span class="sxs-lookup"><span data-stu-id="cbb34-113">For more information, see [Signatures](../signatures.md).</span></span>
