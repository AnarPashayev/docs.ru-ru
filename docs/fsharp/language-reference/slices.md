<span data-ttu-id="062de-110">Наиболее распространенными типами данных, которые разделены являются F# списками и массивами.</span><span class="sxs-lookup"><span data-stu-id="062de-110">The most common data types that are sliced are F# lists and arrays.</span></span> <span data-ttu-id="062de-111">В следующем примере показано, как это сделать с помощью списков:</span><span class="sxs-lookup"><span data-stu-id="062de-111">The following example demonstrates how to do this with lists:</span></span>

```fsharp
// Generate a list of 100 integers
let fullList = [ 1 .. 100 ]

// Create a slice from indices 1-5 (inclusive)
let smallSlice = fullList.[1..5]
printfn "Small slice: %A" smallSlice

// Create a slice from the beginning to index 5 (inclusive)
let unboundedBeginning = fullList.[..5]
printfn "Unbounded beginning slice: %A" unboundedBeginning

// Create a slice from an index to the end of the list
let unboundedEnd = fullList.[94..]
printfn "Unbounded end slice: %A" unboundedEnd
```

## <a name="slicing-multidimensional-arrays"></a><span data-ttu-id="062de-113">Создание срезов многомерные массивы</span><span class="sxs-lookup"><span data-stu-id="062de-113">Slicing multidimensional arrays</span></span>

<span data-ttu-id="062de-117">Следующие примеры демонстрируют, как выполнить срез двумерного массива:</span><span class="sxs-lookup"><span data-stu-id="062de-117">The following examples demonstrate how to slice a 2D array:</span></span>

```fsharp

let A = array2D [[1;2;3];[4;5;6];[7;8;9]]
printfn "Full matrix:\n %A" A

// Take the first row
let row0 = A.[0,*]
printfn "Row 0: %A" row0

// Take the first column

printfn "Column 0: %A" col0

// Take all rows but only two columns

printfn "%A" subA

// Take two rows and all columns
let subA' = A.[0..1,*]
printfn "%A" subA'

// Slice a 2x2 matrix out of the full 3x3 matrix
let twoByTwo = A.[0..1,0..1]
printfn "%A" twoByTwo
```

## <a name="defining-slices-for-other-data-structures"></a><span data-ttu-id="062de-120">Определение срезы для других структур данных</span><span class="sxs-lookup"><span data-stu-id="062de-120">Defining slices for other data structures</span></span>

<span data-ttu-id="062de-121">F# Библиотека ядра определяет фрагменты для ограниченного набора типов.</span><span class="sxs-lookup"><span data-stu-id="062de-121">The F# core library defines slices for a limited set of types.</span></span> <span data-ttu-id="062de-122">Если вы хотите определить срезы для типов данных, это можно сделать само определение типа или в расширение типа.</span><span class="sxs-lookup"><span data-stu-id="062de-122">If you wish to define slices for more data types, you can do so either in the type definition itself or in a type extension.</span></span>

<span data-ttu-id="062de-123">Например, вот как можно определить срезы для <xref:System.ArraySegment%601> класс, позволяющий для удобное управление данными:</span><span class="sxs-lookup"><span data-stu-id="062de-123">For example, here's how you might define slices for the <xref:System.ArraySegment%601> class to allow for convenient data manipulation:</span></span>

```fsharp
open System

type ArraySegment<'TItem> with
    member segment.GetSlice(?start, ?finish) =
        let start = defaultArg start 0
        let finish = defaultArg finish segment.Count
        ArraySegment(segment.Array, segment.Offset + start, finish - start)

let arr = ArraySegment [| 1 .. 10 |]
let slice = arr.[2..5] //[ 3; 4; 5]
```

### <a name="use-inlining-to-avoid-boxing-if-it-is-necessary"></a><span data-ttu-id="062de-124">Встраивание, чтобы избежать упаковки-преобразования, если это необходимо для использования</span><span class="sxs-lookup"><span data-stu-id="062de-124">Use inlining to avoid boxing if it is necessary</span></span>

<span data-ttu-id="062de-125">При определении срезы для типа, который является фактически структурой, мы рекомендуем вам `inline` `GetSlice` член.</span><span class="sxs-lookup"><span data-stu-id="062de-125">If you are defining slices for a type that is actually a struct, we recommend that you `inline` the `GetSlice` member.</span></span> <span data-ttu-id="062de-126">F# Компилятор оптимизирует код, удаляя необязательные аргументы, как избежать выделений кучи, в результате создания срезов.</span><span class="sxs-lookup"><span data-stu-id="062de-126">The F# compiler optimizes away the optional arguments, avoiding any heap allocations as a result of slicing.</span></span> <span data-ttu-id="062de-127">Это критически важно для среза конструкции, такие как <xref:System.Span%601> , не удалось выделить в куче.</span><span class="sxs-lookup"><span data-stu-id="062de-127">This is critically important for slicing constructs such as <xref:System.Span%601> that cannot be allocated on the heap.</span></span>

```fsharp
open System

type Span<'T> with
    // Note the 'inline' in the member definition
    member inline sp.GetSlice(startIdx, endIdx) =
        let s = defaultArg startIdx 0
        let e = defaultArg endIdx sp.Length
        sp.Slice(s, e)

let printSpan (sp: Span<int>) =
    let arr = sp.ToArray()
    printfn "%A" arr

let sp = [| 1; 2; 3; 4; 5 |].AsSpan()
printSpan sp.[0..] // [|1; 2; 3; 4; 5|]
printSpan sp.[..5] // [|1; 2; 3; 4; 5|]
printSpan sp.[0..3] // [|1; 2; 3|]
printSpan sp.[1..2] // |2; 3|]
```
