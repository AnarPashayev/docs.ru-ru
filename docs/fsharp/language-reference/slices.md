## <a name="basic-slicing-with-f-lists-and-arrays"></a><span data-ttu-id="3fcd0-109">Базовые срезы с F# списками и массивами</span><span class="sxs-lookup"><span data-stu-id="3fcd0-109">Basic slicing with F# lists and arrays</span></span>

<span data-ttu-id="3fcd0-110">Наиболее распространенными типами данных, которые являются срезами F# , являются списки и массивы.</span><span class="sxs-lookup"><span data-stu-id="3fcd0-110">The most common data types that are sliced are F# lists and arrays.</span></span> <span data-ttu-id="3fcd0-111">В следующем примере показано, как это сделать с помощью списков.</span><span class="sxs-lookup"><span data-stu-id="3fcd0-111">The following example demonstrates how to do this with lists:</span></span>

```fsharp
// Generate a list of 100 integers
let fullList = [ 1 .. 100 ]

// Create a slice from indices 1-5 (inclusive)
let smallSlice = fullList.[1..5]
printfn "Small slice: %A" smallSlice

// Create a slice from the beginning to index 5 (inclusive)
let unboundedBeginning = fullList.[..5]
printfn "Unbounded beginning slice: %A" unboundedBeginning

// Create a slice from an index to the end of the list
let unboundedEnd = fullList.[94..]
printfn "Unbounded end slice: %A" unboundedEnd
```

<span data-ttu-id="3fcd0-112">Массивы срезов так же, как и списки срезов:</span><span class="sxs-lookup"><span data-stu-id="3fcd0-112">Slicing arrays is just like slicing lists:</span></span>

```fsharp

// Create a slice from indices 1-5 (inclusive)
let smallSlice = fullArray.[1..5]
printfn "Small slice: %A" smallSlice

// Create a slice from the beginning to index 5 (inclusive)
let unboundedBeginning = fullArray.[..5]
printfn "Unbounded beginning slice: %A" unboundedBeginning

// Create a slice from an index to the end of the list

printfn "Unbounded end slice: %A" unboundedEnd
```

## <a name="slicing-multidimensional-arrays"></a><span data-ttu-id="3fcd0-113">Многомерные массивы с срезами</span><span class="sxs-lookup"><span data-stu-id="3fcd0-113">Slicing multidimensional arrays</span></span>

<span data-ttu-id="3fcd0-114">F#поддерживает многомерные массивы в F# основной библиотеке.</span><span class="sxs-lookup"><span data-stu-id="3fcd0-114">F# supports multidimensional arrays in the F# core library.</span></span> <span data-ttu-id="3fcd0-115">Как и в случае с одномерным массивами, можно также использовать срезы многомерных массивов.</span><span class="sxs-lookup"><span data-stu-id="3fcd0-115">As with one-dimensional arrays, slices of multidimensional arrays can also be useful.</span></span> <span data-ttu-id="3fcd0-116">Однако введение дополнительных измерений требует немного другого синтаксиса, чтобы можно было создавать срезы конкретных строк и столбцов.</span><span class="sxs-lookup"><span data-stu-id="3fcd0-116">However, the introduction of additional dimensions mandates a slightly different syntax so that you can take slices of specific rows and columns.</span></span>

<span data-ttu-id="3fcd0-117">В следующих примерах показано, как выполнить срез 2D-массива:</span><span class="sxs-lookup"><span data-stu-id="3fcd0-117">The following examples demonstrate how to slice a 2D array:</span></span>

```fsharp
// Generate a 3x3 2D matrix
let A = array2D [[1;2;3];[4;5;6];[7;8;9]]
printfn "Full matrix:\n %A" A

// Take the first row
let row0 = A.[0,*]
printfn "Row 0: %A" row0

// Take the first column
let col0 = A.[*,0]
printfn "Column 0: %A" col0

// Take all rows but only two columns
let subA = A.[*,0..1]
printfn "%A" subA

// Take two rows and all columns
let subA' = A.[0..1,*]
printfn "%A" subA'

// Slice a 2x2 matrix out of the full 3x3 matrix
let twoByTwo = A.[0..1,0..1]
printfn "%A" twoByTwo
```

<span data-ttu-id="3fcd0-118">F# Базовая библиотека не определена `GetSlice`для трехмерных массивов.</span><span class="sxs-lookup"><span data-stu-id="3fcd0-118">The F# core library does not define `GetSlice`for 3D arrays.</span></span> <span data-ttu-id="3fcd0-119">Если вы хотите выполнить срез этих или других массивов большего размера, необходимо определить `GetSlice` члена самостоятельно.</span><span class="sxs-lookup"><span data-stu-id="3fcd0-119">If you wish to slice those or other arrays of more dimensions, you must define the `GetSlice` member yourself.</span></span>

## <a name="defining-slices-for-other-data-structures"></a><span data-ttu-id="3fcd0-120">Определение срезов для других структур данных</span><span class="sxs-lookup"><span data-stu-id="3fcd0-120">Defining slices for other data structures</span></span>

<span data-ttu-id="3fcd0-121">F# Основная библиотека определяет срезы для ограниченного набора типов.</span><span class="sxs-lookup"><span data-stu-id="3fcd0-121">The F# core library defines slices for a limited set of types.</span></span> <span data-ttu-id="3fcd0-122">Если вы хотите определить срезы для большего числа типов данных, это можно сделать либо в самом определении типа, либо в расширении типа.</span><span class="sxs-lookup"><span data-stu-id="3fcd0-122">If you wish to define slices for more data types, you can do so either in the type definition itself or in a type extension.</span></span>

<span data-ttu-id="3fcd0-123">Например, ниже показано, как можно определить срезы для <xref:System.ArraySegment%601> класса, чтобы обеспечить удобную обработку данных:</span><span class="sxs-lookup"><span data-stu-id="3fcd0-123">For example, here's how you might define slices for the <xref:System.ArraySegment%601> class to allow for convenient data manipulation:</span></span>

```fsharp
open System

type ArraySegment<'TItem> with
    member segment.GetSlice(?start, ?finish) =
        let start = defaultArg start 0
        let finish = defaultArg finish segment.Count
        ArraySegment(segment.Array, segment.Offset + start, finish - start)

let arr = ArraySegment [| 1 .. 10 |]
let slice = arr.[2..5] //[ 3; 4; 5]
```

### <a name="use-inlining-to-avoid-boxing-if-it-is-necessary"></a><span data-ttu-id="3fcd0-124">Используйте встраивание, чтобы избежать упаковки-преобразования в случае необходимости</span><span class="sxs-lookup"><span data-stu-id="3fcd0-124">Use inlining to avoid boxing if it is necessary</span></span>

<span data-ttu-id="3fcd0-125">Если вы определяете срезы для типа, который действительно является структурой, мы рекомендуем использовать `inline` `GetSlice` элемент.</span><span class="sxs-lookup"><span data-stu-id="3fcd0-125">If you are defining slices for a type that is actually a struct, we recommend that you `inline` the `GetSlice` member.</span></span> <span data-ttu-id="3fcd0-126">F# Компилятор оптимизирует необязательные аргументы, избегая выделения кучи в результате среза.</span><span class="sxs-lookup"><span data-stu-id="3fcd0-126">The F# compiler optimizes away the optional arguments, avoiding any heap allocations as a result of slicing.</span></span> <span data-ttu-id="3fcd0-127">Это критически важно для конструкций среза, таких как <xref:System.Span%601> , которые не могут быть выделены в куче.</span><span class="sxs-lookup"><span data-stu-id="3fcd0-127">This is critically important for slicing constructs such as <xref:System.Span%601> that cannot be allocated on the heap.</span></span>

```fsharp
open System

type Span<'T> with
    // Note the 'inline' in the member definition
    member inline sp.GetSlice(startIdx, endIdx) =
        let s = defaultArg startIdx 0
        let e = defaultArg endIdx sp.Length
        sp.Slice(s, e)

let printSpan (sp: Span<int>) =
    let arr = sp.ToArray()
    printfn "%A" arr

let sp = [| 1; 2; 3; 4; 5 |].AsSpan()
printSpan sp.[0..] // [|1; 2; 3; 4; 5|]
printSpan sp.[..5] // [|1; 2; 3; 4; 5|]
printSpan sp.[0..3] // [|1; 2; 3|]
printSpan sp.[1..2] // |2; 3|]
```
