# <a name="f-types"></a>Типы языка F#

В этом разделе описываются типы, которые используются в F# и как F# типы их именования и описания.

## <a name="summary-of-f-types"></a>Сводка по F# типы
Некоторые типы считаются *типов-примитивов*, такие как тип данных Boolean `bool` и типы точек целочисленные и с плавающей запятой различных размеров, в том числе типы для байт и символов. Эти типы описаны в [типов-примитивов](primitive-types.md).

Другие типы, встроенные в язык включают кортежи, списки, массивы, последовательности, записи и размеченные объединения. Если у вас опыт работы с другими языками .NET и являются обучения F#, ознакомьтесь со статьями для каждого из этих типов. Ссылки на дополнительные сведения об этих типах включаются в [связанные разделы](https://msdn.microsoft.com/library/#rel) этого раздела. Эти F#-определенных типов поддерживают разные стили программирования, которые являются общими для языков функционального программирования. Многие из этих типов имеют связанные модули в F# библиотеки, которые поддерживают распространенные операции с этими типами.

Тип функции включает в себя сведения о типах параметров и тип возвращаемого значения.

.NET Framework является источником типов объектов, типы интерфейсов, типы делегатов и другим пользователям. Можно определить собственные типы объектов, так же, как и в любом другом языке .NET.

Кроме того F# кода можно определить псевдонимы, которые называются *аббревиатуры типов*, которые являются альтернативными именами типов. Аббревиатуры типов можно использовать, когда тип может измениться в будущем, и вы хотите избежать изменения кода, который зависит от типа. Кроме того, может использовать сокращенная форма типа как понятное имя для типа, который может сделать код проще читать и понимать.

F#предоставляет полезные коллекции типов, которые разработаны с функционального программирования в виду. Использование этих типов коллекции позволяет писать код, который является более широкими возможностями в стиле. Дополнительные сведения см. в разделе [ F# типы коллекций](fsharp-collection-types.md).

## <a name="syntax-for-types"></a>Синтаксис для типов
В F# кода, часто приходится писать имена типов. Каждый тип имеет синтаксическую форму, и вы используется в аннотации типов, объявления абстрактных методов, объявлений делегатов, подписей и других конструкций. Каждый раз, когда вы объявляете новую конструкцию программу в интерпретаторе, интерпретатор напечатает имя конструкции и синтаксис для его типа. Этот синтаксис может быть только идентификатор для определяемого пользователем типа или встроенным идентификатором, например, для `int` или `string`, но для более сложных типов, более сложный синтаксис.
