
<span data-ttu-id="472dc-111">Если класс наследуется от другого класса, методы и члены базового класса становятся доступными для пользователей производного класса, как если бы они были прямыми членами производного класса.</span><span class="sxs-lookup"><span data-stu-id="472dc-111">If a class inherits from another class, the methods and members of the base class are available to users of the derived class as if they were direct members of the derived class.</span></span>

<span data-ttu-id="472dc-112">Все привязки let и параметры конструктора являются закрытыми для класса, поэтому к ним нельзя получить доступ из производных классов.</span><span class="sxs-lookup"><span data-stu-id="472dc-112">Any let bindings and constructor parameters are private to a class and, therefore, cannot be accessed from derived classes.</span></span>
