# <a name="f-collection-types"></a>Типы коллекций F#

Просмотрев в этом разделе, можно определить, какие F# тип коллекции наиболее соответствует особой необходимости. Эти типы коллекции отличаются от типов коллекций в .NET Framework, например, в `System.Collections.Generic` пространство имен, в который F# типы коллекций изначально разработаны с функциональной точки зрения программирования, а не объектно ориентированный Перспектива. В частности только коллекции массива имеет изменяемых элементов. Таким образом при изменении коллекции, создайте экземпляр измененную коллекцию вместо изменения исходной коллекции.

Типы коллекций также отличаются в типе структуры данных, в которой хранятся объекты. Структуры данных, таких как хэш-таблицы, связанные списки и массивы имеют разные характеристики производительности и другой набор доступных операций.
