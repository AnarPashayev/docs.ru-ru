## <a name="passing-by-reference"></a><span data-ttu-id="eb8fb-167">Передача по ссылке</span><span class="sxs-lookup"><span data-stu-id="eb8fb-167">Passing by Reference</span></span>

<span data-ttu-id="eb8fb-168">Передача F# значения по ссылке включает в себя [ByRef](byrefs.md), которые являются типами управляемых указателей.</span><span class="sxs-lookup"><span data-stu-id="eb8fb-168">Passing an F# value by reference involves [byrefs](byrefs.md), which are managed pointer types.</span></span> <span data-ttu-id="eb8fb-169">Ниже приведены рекомендации по использованию типа.</span><span class="sxs-lookup"><span data-stu-id="eb8fb-169">Guidance for which type to use is as follows:</span></span>
