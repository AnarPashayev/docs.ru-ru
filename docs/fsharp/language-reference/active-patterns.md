# <a name="active-patterns"></a><span data-ttu-id="68f84-103">Активные шаблоны</span><span class="sxs-lookup"><span data-stu-id="68f84-103">Active Patterns</span></span>

<span data-ttu-id="68f84-104">*Активные шаблоны* позволяют определить именованные секции, которые подделят входные данные, чтобы эти имена можно было использовать в выражении сопоставления шаблонов точно так же, как для размеченного объединения.</span><span class="sxs-lookup"><span data-stu-id="68f84-104">*Active patterns* enable you to define named partitions that subdivide input data, so that you can use these names in a pattern matching expression just as you would for a discriminated union.</span></span> <span data-ttu-id="68f84-105">Активные шаблоны можно использовать для разложения данных в настраиваемом порядке для каждого раздела.</span><span class="sxs-lookup"><span data-stu-id="68f84-105">You can use active patterns to decompose data in a customized manner for each partition.</span></span>

## <a name="syntax"></a><span data-ttu-id="68f84-106">Синтаксис</span><span class="sxs-lookup"><span data-stu-id="68f84-106">Syntax</span></span>

```fsharp
// Active pattern of one choice.
let (|identifier|) [arguments] valueToMatch= expression

// Active Pattern with multiple choices.
// Uses a FSharp.Core.Choice<_,...,_> based on the number of case names. In F#, the limitation n <= 7 applies.
let (|identifer1|identifier2|...|) valueToMatch = expression

// Partial active pattern definition.
// Uses a FSharp.Core.option<_> to represent if the type is satisfied at the call site.
let (|identifier|_|) [arguments ] valueToMatch = expression
```
