# <a name="what-is-net-for-apache-spark"></a>Что такое .NET для Apache Spark?

[Apache Spark](https://spark.apache.org/) является механизмом распределенной обработки общего назначения для аналитики по большим наборам данных — обычно терабайт или петабайтов данных. .NET для Apache Spark делает Apache Spark доступным для разработчиков, предоставляя возможности Spark в приложениях с поддержкой уже знакомых языков.

С C# помощью F#и можно получить доступ к следующим:

* Кадр данных и SparkSQL для работы с структурированными данными.
* Структурированная потоковая передача Spark для работы с потоковой передачей данных.

.NET для Apache Spark соответствует .NET Standard, формальной спецификации API-интерфейсов .NET, которые являются общими для различных реализаций .NET. Это означает, что вы можете использовать .NET для Apache Spark в любом месте, где вы пишете код .NET, что позволяет использовать все знания, навыки, код и библиотеки, которые уже есть у разработчика .NET.

.NET для Apache Spark работает в Windows, Linux и macOS с помощью .NET Core. Он также работает в Windows с помощью .NET Framework. Приложения можно развертывать во всех основных поставщиках облачных служб, включая Azure HDInsight Spark, Amazon EMR Spark, Azure Databricks и кирпичи в AWS.

## <a name="net-foundation"></a>.NET Foundation

Проект .NET для Apache Spark является частью [.NET Foundation](https://www.dotnetfoundation.org/).
