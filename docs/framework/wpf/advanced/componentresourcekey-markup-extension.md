# <a name="componentresourcekey-markup-extension"></a><span data-ttu-id="9dcf0-102">Расширение разметки ComponentResourceKey</span><span class="sxs-lookup"><span data-stu-id="9dcf0-102">ComponentResourceKey Markup Extension</span></span>

