# <a name="mcignorable-attribute"></a><span data-ttu-id="e4146-102">Атрибут mc: Ignorable</span><span class="sxs-lookup"><span data-stu-id="e4146-102">mc:Ignorable Attribute</span></span>
<span data-ttu-id="e4146-103">Указывает, какие [!INCLUDE[TLA2#tla_xml](../../../../includes/tla2sharptla-xml-md.md)] префиксы пространства имен в файле разметки могут игнорироваться [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] процессора.</span><span class="sxs-lookup"><span data-stu-id="e4146-103">Specifies which [!INCLUDE[TLA2#tla_xml](../../../../includes/tla2sharptla-xml-md.md)] namespace prefixes encountered in a markup file may be ignored by a [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] processor.</span></span> <span data-ttu-id="e4146-104">`mc:Ignorable` Атрибут поддерживает совместимость разметки для пользовательского сопоставления пространства имен и [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] управления версиями.</span><span class="sxs-lookup"><span data-stu-id="e4146-104">The `mc:Ignorable` attribute supports markup compatibility both for custom namespace mapping and for [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] versioning.</span></span>  
  
## <a name="xaml-attribute-usage-single-prefix"></a><span data-ttu-id="e4146-105">Использование атрибута XAML (один префикс)</span><span class="sxs-lookup"><span data-stu-id="e4146-105">XAML Attribute Usage (Single Prefix)</span></span>  
  
```

  xmlns:ignorablePrefix="ignorableUri"

  mc:Ignorable="ignorablePrefix"...>  
    <ignorablePrefix1:ThisElementCanBeIgnored/>  
</object>  
```
