<span data-ttu-id="3ef3f-215">Графическая система WPF использует аппаратно-независимые единицы, чтобы обеспечить независимость от разрешения и устройства.</span><span class="sxs-lookup"><span data-stu-id="3ef3f-215">The WPF graphics system uses device-independent units to enable resolution and device independence.</span></span> <span data-ttu-id="3ef3f-216">Каждое устройство, независимое от устройства, автоматически масштабируется с помощью параметра системы в точках на дюйм (DPI).</span><span class="sxs-lookup"><span data-stu-id="3ef3f-216">Each device independent pixel automatically scales with the system's dots per inch (dpi) setting.</span></span> <span data-ttu-id="3ef3f-217">Это обеспечивает правильное масштабирование приложений WPF для различных параметров dpi и позволяет приложению автоматически учитывать dpi.</span><span class="sxs-lookup"><span data-stu-id="3ef3f-217">This provides WPF applications proper scaling for different dpi settings and makes the application automatically dpi-aware.</span></span>
