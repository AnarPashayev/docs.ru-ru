 <span data-ttu-id="f3190-105">Ниже перечислены возможности графического оборудования, которые сильнее всего влияют на уровни отрисовки.</span><span class="sxs-lookup"><span data-stu-id="f3190-105">The features of the graphics hardware that most impact the rendering tier levels are:</span></span>  
  
- <span data-ttu-id="f3190-106">**Видеопамять**.
