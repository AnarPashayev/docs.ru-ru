## <a name="graphics-hardware"></a><span data-ttu-id="16357-104">Графическое оборудование</span><span class="sxs-lookup"><span data-stu-id="16357-104">Graphics Hardware</span></span>  
 <span data-ttu-id="16357-105">Ниже перечислены возможности графического оборудования, которые сильнее всего влияют на уровни отрисовки.</span><span class="sxs-lookup"><span data-stu-id="16357-105">The features of the graphics hardware that most impact the rendering tier levels are:</span></span>  
  
-   <span data-ttu-id="16357-106">**Видеопамять**. -   <span data-ttu-id="16357-107">**Построитель текстуры**.
