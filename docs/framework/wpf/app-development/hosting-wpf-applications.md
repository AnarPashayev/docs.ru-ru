# <a name="hosting-wpf-applications"></a>Размещение WPF-приложений
Приложения браузера WPF XAML (XBAP) — это многофункциональные клиентские приложения, которые можно развернуть на веб-сервере и запустить в браузере. Узел WPF (PresentationHost. exe) регистрируется в качестве оболочки и обработчика MIME для файлов XBAP и XAML. Таким образом, Internet Explorer знает, как запустить узел WPF при запуске XBAP. Пользователи Firefox могут установить надстройки Firefox, которые позволят Firefox размещать XBAP. XBAP может размещаться в других браузерах или автономных приложениях с помощью собственных API размещения браузера, предоставляемых WPF.
