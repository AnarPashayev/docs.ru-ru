 <span data-ttu-id="40008-120">В этой политике запрошенные ресурсы из кэша никогда не используются, а ресурсы никогда не помещаются в кэш.</span><span class="sxs-lookup"><span data-stu-id="40008-120">A requested resource is never used from any cache and is never placed in any cache.</span></span> <span data-ttu-id="40008-121">Если запрошенный ресурс присутствует в локальном кэше, он удаляется.</span><span class="sxs-lookup"><span data-stu-id="40008-121">If a requested resource is present in the local cache, it is removed.</span></span> <span data-ttu-id="40008-122">При использовании этого уровня политики также удаляется ресурс в промежуточном кэше.</span><span class="sxs-lookup"><span data-stu-id="40008-122">This policy level indicates to intermediate caches that they should also remove the resource.</span></span> <span data-ttu-id="40008-123">В протоколе кэширования HTTP для этого используется директива управления кэшем "Не сохранять в кэше" (no-store).</span><span class="sxs-lookup"><span data-stu-id="40008-123">In the HTTP caching protocol, this is achieved using the no-store cache control directive.</span></span>  
  
## <a name="refresh-policy"></a><span data-ttu-id="40008-124">Политика "Обновление"</span><span class="sxs-lookup"><span data-stu-id="40008-124">Refresh Policy</span></span>
