### <a name="to-set-a-policy-that-takes-requested-resources-from-a-cache"></a><span data-ttu-id="7a7bf-109">Установка политики, которая принимает запрошенные ресурсы из кэша</span><span class="sxs-lookup"><span data-stu-id="7a7bf-109">To set a policy that takes requested resources from a cache</span></span>  
  
- <span data-ttu-id="7a7bf-110">Создайте политику, которая принимает запрошенные ресурсы из кэша, если они есть в кэше.              (HttpRequestCacheLevel.CacheIfAvailable)

