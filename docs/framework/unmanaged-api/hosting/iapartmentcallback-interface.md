# <a name="iapartmentcallback-interface"></a>Интерфейс IApartmentCallback
Предоставляет методы для осуществления обратных вызовов в подразделении. *Подразделения* — это логический контейнер, в рамках процесса для объектов, которые совместно используют те же требования доступа потоке.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[Метод DoCallback](../../../../docs/framework/unmanaged-api/hosting/iapartmentcallback-docallback-method.md)|Выполняет указанную функцию в подразделении.|
