# <a name="iappdomainbinding-interface"></a>Интерфейс IAppDomainBinding
Предоставляет метод, вызываемый средой CLR для уведомления ведущего приложения о создании домена приложения.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[Метод OnAppDomain](../../../../docs/framework/unmanaged-api/hosting/iappdomainbinding-onappdomain-method.md)|Вызывается средой CLR для уведомления узла о создании домена приложения.|
