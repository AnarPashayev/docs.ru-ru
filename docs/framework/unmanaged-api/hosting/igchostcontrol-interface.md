# <a name="igchostcontrol-interface"></a>Интерфейс IGCHostControl
Предоставляет метод, позволяющий сборщику мусора запрашивать изменение ограничений виртуальной памяти на узле.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[Метод RequestVirtualMemLimit](../../../../docs/framework/unmanaged-api/hosting/igchostcontrol-requestvirtualmemlimit-method.md)|Запрашивает у узла изменение ограничений виртуальной памяти.|
