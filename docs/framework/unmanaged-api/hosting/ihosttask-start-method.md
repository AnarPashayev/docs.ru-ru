# <a name="ihosttaskstart-method"></a>Метод IHostTask::Start
Запрашивает, что узел перемещает задачу, представленную текущим экземпляром [IHostTask](../../../../docs/framework/unmanaged-api/hosting/ihosttask-interface.md) , из приостановленного в активное состояние, в котором можно выполнить код.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT Start ();  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
  
|HRESULT|Описание|  
|-------------|-----------------|  
|S_OK|Успешное начало возврата.|  
|E_FAIL|Произошла неизвестная фатальная ошибка. Когда метод возвращает значение E_FAIL, среда CLR больше не может использоваться в процессе. Последующие вызовы методов размещения возвращают HOST_E_CLRNOTAVAILABLE.|  
  
## <a name="remarks"></a>Заметки  
 `Start` всегда возвращает значение HRESULT, равное S_OK, за исключением случаев, когда произошла разрушительная ошибка.
