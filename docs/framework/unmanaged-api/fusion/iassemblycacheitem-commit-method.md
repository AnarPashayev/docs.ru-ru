# <a name="iassemblycacheitemcommit-method"></a>Метод IAssemblyCacheItem::Commit
Фиксирует ссылку кэшированной сборки на память.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT Commit (  
    [in] DWORD dwFlags,
    [out, optional] ULONG *pulDisposition  
);  
```  
  
## <a name="parameters"></a>Параметры  
 `dwFlags`  
 (в) Флаги, определенные в Fusion.idl.  
  
 `pulDisposition`  
 (вне, необязательно) Значение, указывававание результата операции.
