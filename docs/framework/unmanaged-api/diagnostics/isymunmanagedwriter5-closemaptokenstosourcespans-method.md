# <a name="isymunmanagedwriter5closemaptokenstosourcespans-method"></a>Метод ISymUnmanagedWriter5::CloseMapTokensToSourceSpans
Закройте Специальный раздел настраиваемых данных, чтобы получить сведения о сопоставлении диапазона "токен-источник". После закрытия не удается добавить дополнительные сведения о сопоставлении.  
  
## <a name="syntax"></a>Синтаксис  
  
```idl  
HRESULT CloseMapTokensToSourceSpans();  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Возвращает `HRESULT`.  
  
## <a name="requirements"></a>Требования  
 **Заголовок:** Корсим. idl, Корсим. h
