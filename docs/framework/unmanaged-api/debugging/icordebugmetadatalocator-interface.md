# <a name="icordebugmetadatalocator-interface"></a>Интерфейс ICorDebugMetaDataLocator

Предоставляет сведения о метаданных для отладчика.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[Метод GetMetaData](icordebugmetadatalocator-getmetadata-method.md)|Запрашивает у отладчика возврат полного пути к модулю, метаданные которого необходимы для завершения запрошенной отладчиком операции.|  
  
## <a name="remarks"></a>Комментарии  
  
> [!NOTE]
> Этот интерфейс не поддерживает удаленные вызовы между компьютерами или между процессами.
