# <a name="icordebugstepperdeactivate-method"></a>Метод ICorDebugStepper::Deactivate
Принуждает ICorDebugStepper для отмены последней команды шага, оно получено.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT Deactivate ();  
```  
  
## <a name="remarks"></a>Примечания  
 Новая команда пошагового выполнения могут быть выданы после отмены самой последней полученной команда этапа.
