# <a name="icordebugmoduleenum-interface"></a>Интерфейс ICorDebugModuleEnum

Реализует методы ICorDebugEnum и перечисляет ICorDebugModule массивы.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[Метод Next](../../../../docs/framework/unmanaged-api/debugging/icordebugmoduleenum-next-method.md)|Возвращает указанное число экземпляров `ICorDebugModule` из перечисления, начиная с текущей позиции.|  
  
## <a name="remarks"></a>Заметки  
  
> [!NOTE]
> Этот интерфейс не поддерживает удаленные вызовы между компьютерами или между процессами.
