 <span data-ttu-id="2ffca-161">Все обратные вызовы сериализуются, вызываются в том же потоке и вызываются с процессом в состоянии SYNCHRONIZED.</span><span class="sxs-lookup"><span data-stu-id="2ffca-161">All callbacks are serialized, called in the same thread, and called with the process in the synchronized state.</span></span>
