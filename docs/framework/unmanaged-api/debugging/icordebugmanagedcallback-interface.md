## <a name="remarks"></a><span data-ttu-id="cba64-160">Заметки</span><span class="sxs-lookup"><span data-stu-id="cba64-160">Remarks</span></span>  
 <span data-ttu-id="cba64-161">Все обратные вызовы сериализуются, вызываются в том же потоке и вызываются с процессом в состоянии SYNCHRONIZED.</span><span class="sxs-lookup"><span data-stu-id="cba64-161">All callbacks are serialized, called in the same thread, and called with the process in the synchronized state.</span></span>  
  
 <span data-ttu-id="cba64-162">Каждая реализация обратного вызова должна вызывать [ICorDebugController:: Continue](icordebugcontroller-continue-method.md) для возобновления выполнения.</span><span class="sxs-lookup"><span data-stu-id="cba64-162">Each callback implementation must call [ICorDebugController::Continue](icordebugcontroller-continue-method.md) to resume execution.</span></span> <span data-ttu-id="cba64-163">Если `ICorDebugController::Continue` не вызывается до возврата обратного вызова, процесс останется остановленным, а обратные вызовы событий не будут выполняться до вызова `ICorDebugController::Continue`.</span><span class="sxs-lookup"><span data-stu-id="cba64-163">If `ICorDebugController::Continue` is not called before the callback returns, the process will remain stopped and no more event callbacks will occur until `ICorDebugController::Continue` is called.</span></span>  
  
 <span data-ttu-id="cba64-164">Отладчик должен реализовать [ICorDebugManagedCallback2](icordebugmanagedcallback2-interface.md) при отладке приложений .NET Framework версии 2,0.</span><span class="sxs-lookup"><span data-stu-id="cba64-164">A debugger must implement [ICorDebugManagedCallback2](icordebugmanagedcallback2-interface.md) if it is debugging .NET Framework version 2.0 applications.</span></span> <span data-ttu-id="cba64-165">Экземпляр `ICorDebugManagedCallback` или `ICorDebugManagedCallback2` передается в качестве объекта обратного вызова в метод [ICorDebug:: SetManagedHandler](icordebug-setmanagedhandler-method.md).</span><span class="sxs-lookup"><span data-stu-id="cba64-165">An instance of `ICorDebugManagedCallback` or `ICorDebugManagedCallback2` is passed as the callback object to [ICorDebug::SetManagedHandler](icordebug-setmanagedhandler-method.md).</span></span>  
  
> [!NOTE]
> <span data-ttu-id="cba64-166">Этот интерфейс не поддерживает удаленные вызовы между компьютерами или между процессами.</span><span class="sxs-lookup"><span data-stu-id="cba64-166">This interface does not support being called remotely, either cross-machine or cross-process.</span></span>
