# <a name="icordebugprocess8-interface"></a>Интерфейс ICorDebugProcess8
[Поддерживается в .NET Framework 4.6 и более поздних версиях]  
  
 Логически расширяет интерфейс ICorDebugProcess для включения или отключения определенных типов [ICorDebugManagedCallback2](../../../../docs/framework/unmanaged-api/debugging/icordebugmanagedcallback2-interface.md) обратных вызовов исключения.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[Метод EnableExceptionCallbacksOutsideOfMyCode](../../../../docs/framework/unmanaged-api/debugging/icordebugprocess8-enableexceptioncallbacksoutsideofmycode-method.md)|Включает или отключает определенные виды [ICorDebugManagedCallback2](../../../../docs/framework/unmanaged-api/debugging/icordebugmanagedcallback2-interface.md) обратных вызовов исключения.|
