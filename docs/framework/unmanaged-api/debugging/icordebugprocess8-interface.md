# <a name="icordebugprocess8-interface"></a>Интерфейс ICorDebugProcess8
[Поддерживается в .NET Framework 4,6 и более поздних версиях]  
  
 Логически расширяет интерфейс ICorDebugProcess, чтобы включать или отключать определенные типы обратных вызовов исключений [ICorDebugManagedCallback2](../../../../docs/framework/unmanaged-api/debugging/icordebugmanagedcallback2-interface.md) .  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[Метод EnableExceptionCallbacksOutsideOfMyCode](../../../../docs/framework/unmanaged-api/debugging/icordebugprocess8-enableexceptioncallbacksoutsideofmycode-method.md)|Включает или отключает определенные типы обратных вызовов исключений [ICorDebugManagedCallback2](../../../../docs/framework/unmanaged-api/debugging/icordebugmanagedcallback2-interface.md) .|
