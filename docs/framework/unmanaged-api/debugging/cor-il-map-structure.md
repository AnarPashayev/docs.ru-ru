# <a name="cor_il_map-structure"></a>Структура COR_IL_MAP
Указывает изменения в относительном смещении функции.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
typedef struct _COR_IL_MAP {  
    ULONG32 oldOffset;
    ULONG32 newOffset;
    BOOL    fAccurate;  
} COR_IL_MAP;  
```  
  
## <a name="members"></a>Члены  
  
|Участник|Описание|  
|------------|-----------------|  
|`oldOffset`|Старый промежуточный язык Microsoft (MSIL) смещен по отношению к началу функции.|  
|`newOffset`|Новый MSIL смещен иссяк по отношению к началу функции.|  
|`fAccurate`|`true`если отображение известно, что является точным; в `false`противном случае, .|  
  
## <a name="remarks"></a>Remarks  
 Формат карты заключается в следующем: отладчик `oldOffset` будет считать, что относится к СМЕЩЕНию MSIL в рамках оригинального, неизмененного кода MSIL. Параметр `newOffset` относится к соответствующему смещению MSIL в рамках нового инструментируемого кода.  
  
 Для правильной работы необходимо выполнить следующие требования:  
  
- Карта должна быть отсортирована в порядке возрастания.  
  
- Инструментальный код MSIL не должен быть переупорядочен.  
  
- Исходный код MSIL не должен быть удален.  
  
- Карта должна включать записи для картирования всех точек последовательности из файла базы данных программы (PDB).  
  
 Карта не интерполирует недостающие записи. В следующем примере показана карта и ее результаты.  
  
 Карта:  
  
- 0 старых смещений, 0 новых смещений  
  
- 5 старых смещений, 10 новых смещений  
  
- 9 старых смещений, 20 новых смещений  
  
 Результат:  
  
- Старое смещение 0, 1, 2, 3 или 4 будет отображено на новое смещение 0.  
  
- Старый смещение 5, 6, 7 или 8 будет отображено на новые смещения 10.  
  
- Старое смещение 9 или выше будет отображено на новые смещения 20.  
  
- Новое смещение 0, 1, 2, 3, 4, 5, 6, 7, 8 или 9 будет отображено на старые смещения 0.  
  
- Новое смещение 10, 11, 12, 13, 14, 15, 16, 17, 18 или 19 будет отображено на старые смещения 5.  
  
- Новое смещение 20 или выше будет отображено на старый смещение 9.
