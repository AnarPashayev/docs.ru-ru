# <a name="icordebugtypegetrank-method"></a>Метод ICorDebugType::GetRank
Возвращает число измерений в типе массива.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT GetRank (  
    [out] ULONG32   *pnRank  
);  
```  
  
## <a name="parameters"></a>Параметры  
 `pnRank`  
 заполняет Указатель на число измерений.
