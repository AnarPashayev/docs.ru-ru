# <a name="icordebugtypegetrank-method"></a>Метод ICorDebugType::GetRank
Получает число измерений в тип массива.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
HRESULT GetRank (  
    [out] ULONG32   *pnRank  
);  
```  
  
## <a name="parameters"></a>Параметры  
 `pnRank`  
 [out] Указатель на число измерений.
