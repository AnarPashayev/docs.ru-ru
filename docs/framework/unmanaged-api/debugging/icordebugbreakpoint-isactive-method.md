# <a name="icordebugbreakpointisactive-method"></a>Метод ICorDebugBreakpoint::IsActive
Возвращает значение, указывающее, активна ли эта `ICorDebugBreakpoint`.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT IsActive (  
    [out] BOOL *pbActive  
);  
```  
  
## <a name="parameters"></a>Параметры  
 `pbActive`  
 [out] `true`, если эта точка останова активна; в противном случае `false`.
