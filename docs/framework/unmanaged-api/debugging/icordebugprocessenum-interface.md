# <a name="icordebugprocessenum-interface"></a>Интерфейс ICorDebugProcessEnum
Реализует методы ICorDebugEnum и перечисляет ICorDebugProcess массивы.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[Метод Next](../../../../docs/framework/unmanaged-api/debugging/icordebugprocessenum-next-method.md)|Возвращает указанное число экземпляров `ICorDebugProcess` из перечисления, начиная с текущей позиции.|  
  
## <a name="remarks"></a>Заметки  
  
> [!NOTE]
> Этот интерфейс не поддерживает удаленные вызовы между компьютерами или между процессами.
