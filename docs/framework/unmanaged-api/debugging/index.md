# <a name="debugging-unmanaged-api-reference"></a>Отладка (ссылка неуправляемого API)

API отладки позволяет отладчику производить отладку кода, который выполняется в среде CLR. Отлаживаемым кодом может быть любой тип кода, поддерживаемый средой CLR.  
  
## <a name="in-this-section"></a>в этом разделе  

 [Компонентные классы отладки](debugging-coclasses.md)  
 Описываются неуправляемые коклассы, которые использует API отладки.  
  
 [Интерфейсы отладки](debugging-interfaces.md)  
 Описываются неуправляемые интерфейсы, позволяющие выполнять отладку программы, выполняющейся в среде CLR.  
  
 [Глобальные статические функции отладки](debugging-global-static-functions.md)  
 Описываются неуправляемые глобальные статистические функции, которые использует API отладки.  
  
 [Перечисления отладки](debugging-enumerations.md)  
 Описываются неуправляемые перечисления, которые использует API отладки.  
  
 [Структуры отладки](debugging-structures.md)  
 Описываются неуправляемые структуры, которые использует API отладки.
