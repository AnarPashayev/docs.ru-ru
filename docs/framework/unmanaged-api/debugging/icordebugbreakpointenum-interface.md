# <a name="icordebugbreakpointenum-interface"></a>Интерфейс ICorDebugBreakpointEnum

Реализует методы ICorDebugEnum и перечисляет Икордебугбреакпоинт массивы.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[Метод Next](../../../../docs/framework/unmanaged-api/debugging/icordebugbreakpointenum-next-method.md)|Возвращает указанное количество `ICorDebugBreakpoint` экземпляров из перечисления, начиная с текущей позиции.|  
  
## <a name="remarks"></a>Примечания  
  
> [!NOTE]
> Этот интерфейс не поддерживает удаленные вызовы между компьютерами или между процессами.
