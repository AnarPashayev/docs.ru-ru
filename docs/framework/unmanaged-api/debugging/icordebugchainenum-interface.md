# <a name="icordebugchainenum-interface"></a>Интерфейс ICorDebugChainEnum

Реализует методы ICorDebugEnum и перечисляет ICorDebugChain массивы.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[Метод Next](../../../../docs/framework/unmanaged-api/debugging/icordebugchainenum-next-method.md)|Возвращает указанное количество `ICorDebugChain` экземпляров из перечисления, начиная с текущей позиции.|  
  
## <a name="remarks"></a>Примечания  
  
> [!NOTE]
> Этот интерфейс не поддерживает удаленные вызовы между компьютерами или между процессами.
