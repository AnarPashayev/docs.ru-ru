# <a name="icordebugchainenum-interface"></a>Интерфейс ICorDebugChainEnum

Реализует методы ICorDebugEnum и перечисляет ICorDebugChain массивы.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[Метод Next](icordebugchainenum-next-method.md)|Возвращает указанное число экземпляров `ICorDebugChain` из перечисления, начиная с текущей позиции.|  
  
## <a name="remarks"></a>Заметки  
  
> [!NOTE]
> Этот интерфейс не поддерживает удаленные вызовы между компьютерами или между процессами.
