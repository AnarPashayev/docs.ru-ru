# <a name="icorpublishenumreset-method"></a>Метод ICorPublishEnum::Reset
Перемещает курсор в начало перечисления.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT Reset ();  
```  
  
## <a name="parameters"></a>Параметры  
 `Reset` не принимает никаких параметров.
