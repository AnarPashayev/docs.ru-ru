# <a name="icordebugstepperenum-interface"></a>Интерфейс ICorDebugStepperEnum
Реализует методы ICorDebugEnum и перечисляет массивы ICorDebugStepper.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[Метод Next](icordebugstepperenum-next-method.md)|Возвращает указанное число экземпляров `ICorDebugStepper` из перечисления, начиная с текущей позиции.|  
  
## <a name="remarks"></a>Заметки  
  
> [!NOTE]
> Этот интерфейс не поддерживает удаленные вызовы между компьютерами или между процессами.
