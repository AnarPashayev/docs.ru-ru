# <a name="icorprofilermoduleenumreset-method"></a>Метод ICorProfilerModuleEnum::Reset
Moves this enumerator's cursor to the starting position of the sequence.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT Reset();  
```
