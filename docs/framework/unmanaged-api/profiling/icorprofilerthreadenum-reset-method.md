# <a name="icorprofilerthreadenumreset-method"></a>Метод ICorProfilerThreadEnum::Reset
Перемещает курсор перечислителя в начальную позицию последовательности.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT Reset();  
```
