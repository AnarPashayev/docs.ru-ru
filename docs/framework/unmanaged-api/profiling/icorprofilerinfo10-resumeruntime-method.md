# <a name="icorprofilerinfo10resumeruntime-method"></a>Метод ICorProfilerInfo10:: Ресумерунтиме
  
Возобновляет работу среды выполнения без выполнения сборки мусора.   
  
## <a name="syntax"></a>Синтаксис  
  
```cpp
HRESULT ResumeRuntime();
```
