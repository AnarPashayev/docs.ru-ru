# <a name="put-function"></a>Функция размещения

Задает новое значение для именованного свойства.

[!INCLUDE[internalonly-unmanaged](../../../../includes/internalonly-unmanaged.md)]

## <a name="syntax"></a>Синтаксис

```cpp
HRESULT Put (
   [in] int               vFunc,
   [in] IWbemClassObject* ptr,
   [in] LPCWSTR           wszName,
   [in] LONG              lFlags,
   [in] VARIANT*          pVal,
   [in] CIMTYPE           vtType
);
```

## <a name="parameters"></a>Параметры

`vFunc`\
окне Этот параметр не используется.

`ptr`\
окне Указатель на экземпляр [ивбемклассобжект](/windows/desktop/api/wbemcli/nn-wbemcli-iwbemclassobject) .

`wszName`\
окне Имя свойства. Этот параметр не может `null`быть.

`lFlags`\
[in] Зарезервировано. Этот параметр должен иметь значение 0.

`pVal`\
окне Указатель на допустимое `VARIANT` значение, которое станет новым значением свойства. Если `pVal` параметр `null` имеет значение или указывает `VARIANT` на тип `VT_NULL`, свойство устанавливается в `null`значение.

`vtType`\
окне Тип `VARIANT` ,`pVal`на который указывает. Дополнительные сведения см. в разделе ["Примечания"](#remarks) .

## <a name="return-value"></a>Возвращаемое значение

Следующие значения, возвращаемые этой функцией, определены в файле заголовка *вбемкли. h* , или их можно определить как константы в коде:

|Константа  |Значение  |Описание  |
|---------|---------|---------|
|`WBEM_E_FAILED` | 0x80041001 | Общий сбой. |
|`WBEM_E_INVALID_PARAMETER` | 0x80041008 | Один или несколько параметров недопустимы. |
|`WBEM_E_INVALID_PROPERTY_TYPE` | 0x8004102a | Тип свойства не распознан. Это значение возвращается при создании экземпляров класса, если класс уже существует. |
|`WBEM_E_OUT_OF_MEMORY` | 0x80041006 | Недостаточно памяти для завершения операции. |
| `WBEM_E_TYPE_MISMATCH` | 0x80041005 | Для экземпляров: Указывает, `pVal` что указывает `VARIANT` на недопустимый тип для свойства. <br/> Для определений классов: Свойство уже существует в родительском классе, а новый COM-тип отличается от старого типа COM. |
|`WBEM_S_NO_ERROR` | 0 | Вызов функции выполнен успешно. |

## <a name="remarks"></a>Примечания

Эта функция заключает вызов метода [ивбемклассобжект::P UT](/windows/desktop/api/wbemcli/nf-wbemcli-iwbemclassobject-put) .

Эта функция всегда перезаписывает текущее значение свойства новым. Если [ивбемклассобжект](/windows/desktop/api/wbemcli/nn-wbemcli-iwbemclassobject) указывает на определение класса, `Put` создает или обновляет значение свойства. Если [ивбемклассобжект](/windows/desktop/api/wbemcli/nn-wbemcli-iwbemclassobject) указывает на экземпляр CIM, `Put` обновляет только значение свойства; `Put` не удается создать значение свойства.

`__CLASS` Системное свойство доступно для записи только во время создания класса, если оно не может быть пустым. Все остальные системные свойства доступны только для чтения.

Пользователь не может создавать свойства с именами, которые начинаются или заканчиваются символом подчеркивания ("_"). Он зарезервирован для системных классов и свойств.

Если свойство, заданное `Put` функцией, существует в родительском классе, значение свойства по умолчанию изменяется, если тип свойства не совпадает с типом родительского класса. Если свойство не существует и не является несоответствием типов, создается свойство.

`VT_NULL` `VARIANT` `null` `pVal` Используйте параметр только при создании новых свойств в определении класса CIM и указывает на тип или. `vtType` В этом случае `vType` параметр указывает тип CIM для свойства. В любом другом случае `vtType` значение должно быть равно 0. `vtType`также должно быть равно 0, если базовый объект является экземпляром (даже `Val` если `null`имеет значение), так как тип свойства является фиксированным и не может быть изменен.
