# <a name="getnames-function"></a>Функция GetNames
Получает подмножество имен или все имена свойств объекта.

[!INCLUDE[internalonly-unmanaged](../../../../includes/internalonly-unmanaged.md)]

## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT GetNames (
   [in] int                 vFunc,
   [in] IWbemClassObject*   ptr,
   [in] LPCWSTR             wszQualifierName,
   [in] LONG                lFlags,
   [in] VARIANT*            pQualifierValue,
   [out] SAFEARRAY (BSTR)** pstrNames
);
```  

## <a name="parameters"></a>Параметры

`vFunc`  
(в) Этот параметр не используется.

`ptr`  
(в) Указатель на экземпляр [IWbemClassObject.](/windows/desktop/api/wbemcli/nn-wbemcli-iwbemclassobject)

`wszQualifierName`  
(в) Указатель на действительное, `LPCWSTR` которое определяет имя квалификатора, работающего как часть фильтра. Для получения дополнительной информации смотрите раздел [Замечания.](#remarks) Этот параметр может иметь значение `null`.

`lFlags`  
(в) Комбинация битовых полей. Для получения дополнительной информации смотрите раздел [Замечания.](#remarks)

`pQualifierValue`(в) Указатель на действительную `VARIANT` структуру, инициализированную к значению фильтра. Этот параметр может иметь значение `null`.

`pstrNames`  
(ваут) Структура, `SAFEARRAY` содержащая имена свойств. При входе этот параметр всегда `null`должен быть указателем на . Дополнительную информацию можно узнать в разделе [«Замечания».](#remarks)

## <a name="return-value"></a>Возвращаемое значение

Следующие значения, возвращенные этой функцией, определяются в файле заголовка *WbemCli.h* или вы можете определить их как константы в коде:

|Постоянно  |Значение  |Описание  |
|---------|---------|---------|
|`WBEM_E_FAILED` | 0x80041001 | Произошел общий сбой. |
|`WBEM_E_INVALID_PARAMETER` | 0x80041008 | Один или несколько параметров недействительны, или было указано неправильное сочетание флагов и параметров. |
|`WBEM_E_OUT_OF_MEMORY` | 0x80041006 | Недостаточно памяти для выполнения операции. |
|`WBEM_S_NO_ERROR` | 0 | Вызов функции был успешным.  |
  
## <a name="remarks"></a>Remarks

Эта функция завершает вызов методом [IWbemClassObject::GetNames.](/windows/desktop/api/wbemcli/nf-wbemcli-iwbemclassobject-getnames)

Имя возвращается контролируется сочетанием флагов и параметров. Например, функция может возвращать имена всех свойств или только имена ключевых свойств.  Первичный фильтр указан `lFlags` в параметре, а остальные параметры варьируются в зависимости от него.

Значения флага `lFlags` в битовых полях

Флаги, которые могут `lEnumFlags` быть переданы как аргумент, являются битовыми полями, которые определяются в файле заголовка *WbemCli.h,* или вы можете определить их как константы в коде.  Вы можете объединить один флаг из каждой группы с любым флагом из любой другой группы. Тем не менее, флаги из той же группы являются взаимоисключающими.

| Флаги группы 1 |Значение  |Описание  |
|---------|---------|---------|
| `WBEM_FLAG_ALWAYS` | 0 | Верните все имена недвижимости. `strQualifierName`и `pQualifierVal` не используются. |
| `WBEM_FLAG_ONLY_IF_TRUE` | 1 | Возврат только свойств, которые имеют квалификатор имени, указанного параметром. `strQualifierName` Если этот флаг используется, `strQualifierName`необходимо указать. |
|`WBEM_FLAG_ONLY_IF_FALSE` | 2 |  Возврат только свойств, не имеющие квалификатора имени, указанного параметром. `strQualifierName` Если этот флаг используется, `strQualifierName`необходимо указать. |
|`WBEM_FLAG_ONLY_IF_IDENTICAL` | 3 | Возврат только свойств, которые имеют квалификатор имени, указанного `wszQualifierName` параметром, `pQualifierVal` а также имеют значение, идентичное тому, которое указывает структура. Если используется этот флаг, необходимо `wszQualifierName` указать `pQualifierValue`как a, так и a. |

| Флаги группы 2 |Значение  |Описание  |
|---------|---------|---------|
|`WBEM_FLAG_KEYS_ONLY` | 0x4 | Верните только имена свойств, определяющих ключи. |
|`WBEM_FLAG_REFS_ONLY` | 0x8 | Верните только имена свойств, которые являются ссылками объектов. |

| Флаги группы 3 |Значение  |Описание  |
|---------|---------|---------|
| `WBEM_FLAG_LOCAL_ONLY` | 0x10 | Верните только имена свойств, которые принадлежат к наиболее производным классам. Исключить свойства из родительских классов. |
| `WBEM_FLAG_PROPAGATED_ONLY` |  0x20 | Верните только имена свойств, которые принадлежат к родительским классам. |
|`WBEM_FLAG_SYSTEM_ONLY` | 0x30 | Верните только названия системных свойств. |
|`WBEM_FLAG_NONSYSTEM_ONLY` | 0x40 | Верните только названия несистемных свойств. |

Функция всегда выделяет новый, `SAFEARRAY` `WBEM_S_NO_ERROR`если `pstrNames` он возвращается, и всегда настроен ателье на него. Возвращаемый массив может иметь 0 элементов, если свойства не соответствуют указанным фильтрам. Если функция возвращает значение, `WBM_S_NO_ERROR`кроме, `SAFEARRAY` новая структура не возвращается.
