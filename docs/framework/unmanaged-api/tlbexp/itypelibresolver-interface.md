# <a name="itypelibresolver-interface"></a>Интерфейс ITypeLibResolver
Предоставляет [метод ресолветипелиб](resolvetypelib-method.md), который разрешает путь к файлу библиотеки типов.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[Метод ResolveTypeLib](resolvetypelib-method.md)|Разрешает простое имя библиотеки типов, возвращая полный путь.|
