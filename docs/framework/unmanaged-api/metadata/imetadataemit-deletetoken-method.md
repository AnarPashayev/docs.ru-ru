# <a name="imetadataemitdeletetoken-method"></a>Метод IMetaDataEmit::DeleteToken
Удаляет указанный токен из текущей области метаданных.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT DeleteToken (   
    [in]  mdToken     tkObj   
);  
```  
  
## <a name="parameters"></a>Параметры  
 `tkObj`  
 окне Удаляемый токен.
