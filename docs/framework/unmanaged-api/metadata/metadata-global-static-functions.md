# <a name="metadata-global-static-functions"></a>Глобальные статические функции метаданных
Начиная с .NET Framework 4, в документации по `_Cor*`, `CoInitialize*`, `CoUninitialize*`, и `CoEEShutDownCOM` функций, которые относятся к размещению, находится в [устаревшей функции размещения CLR](../../../../docs/framework/unmanaged-api/hosting/deprecated-clr-hosting-functions.md). Не существует глобальных статических функций, относящихся к метаданным.
