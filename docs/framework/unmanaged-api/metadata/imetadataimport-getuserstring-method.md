# <a name="imetadataimportgetuserstring-method"></a>Метод IMetaDataImport::GetUserString
Получает строку литералов, представленную указанным токеном метаданных.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT GetUserString (  
   [in]   mdString    stk,  
   [out]  LPWSTR      szString,  
   [in]   ULONG       cchString,  
   [out]  ULONG       *pchString  
);  
```  
  
## <a name="parameters"></a>Параметры  
 `stk`  
 [in] The String token to return the associated string for.  
  
 `szString`  
 [out] A copy of the requested string.  
  
 `cchString`  
 [in] The maximum size in wide characters of the requested `szString`.  
  
 `pchString`  
 [out] The size in wide characters of the returned `szString`.
