# <a name="imetadataemitdefinemethodimpl-method"></a>Метод IMetaDataEmit::DefineMethodImpl
Создает определение для реализации метода, унаследованного от интерфейса, и возвращает маркер к определению реализации этого метода.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT DefineMethodImpl (
    [in]  mdTypeDef         td,
    [in]  mdToken           tkBody,
    [in]  mdToken           tkDecl  
);  
```  
  
## <a name="parameters"></a>Параметры  
 `td`  
 (в) Токен `mdTypedef` класса реализации.  
  
 `tkBody`  
 (в) `mdMemberRef` Или `mdMethodDef` маркер корпуса кода.  
  
 `tkDecl`  
 (в) `mdMemberRef` Или `mdMethodDef` маркер внедренного метода интерфейса.
