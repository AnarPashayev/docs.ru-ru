# <a name="imetadataimportgetsigfromtoken-method"></a>Метод IMetaDataImport::GetSigFromToken
Возвращает двоичную подпись метаданных, связанную с указанным токеном.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp  
HRESULT GetSigFromToken (
   [in]   mdSignature        mdSig,
   [out]  PCCOR_SIGNATURE    *ppvSig,
   [out]  ULONG              *pcbSig
);  
```  
  
## <a name="parameters"></a>Параметры  
 `mdSig`  
 (в) Токен для возврата двоичной подписи метаданных для.  
  
 `ppvSig`  
 (ваут) Указатель на возвращенную подпись метаданных.  
  
 `pcbSig`  
 (ваут) Размер байтов двоичной подписи метаданных.
