# <a name="httpwebrequest_httpresponse-field"></a>HttpWebRequest. \_ Поле HttpResponse

`HttpWebRequest._HttpResponse`<xref:System.Net.HttpWebResponse>содержит сведения об HTTP-ответе из HTTP-запроса. Это может быть `null` до получения ответа HTTP.

## <a name="syntax"></a>Синтаксис
  
```csharp  
internal HttpWebResponse _HttpResponse
```

> [!WARNING]
> `HttpWebRequest._HttpResponse`Поле является внутренним и не предназначено для непосредственного использования в коде.
>
> Корпорация Майкрософт не поддерживает использование этого поля в рабочем приложении при каких-либо обстоятельствах.

**Пространство имен:** <xref:System.Net>
