# <a name="datamemberfieldeditor-class"></a>Класс DataMemberFieldEditor

Предоставляет пользовательский интерфейс для редактирования свойств объектов, привязанных к данным, путем перечисления всех свойств объекта `DataSource`, из которого выбирается источник значения.  
  
## <a name="syntax"></a>Синтаксис
  
```csharp
internal class DataMemberFieldEditor : UITypeEditor
```

> [!WARNING]
> Класс `DataMemberFieldEditor` является внутренним и не предназначен для непосредственного использования в коде.
> 
> Корпорация Майкрософт не поддерживает использование этого класса в рабочем приложении при каких-либо обстоятельствах.
