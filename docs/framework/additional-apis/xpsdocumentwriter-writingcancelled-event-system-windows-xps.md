# <a name="xpsdocumentwriter_writingcancelled-event"></a>XpsDocumentWriter. \_WritingCancelled событие

Происходит при отмене операции <xref:System.Windows.Xps.XpsDocumentWriter.Write%2A> или <xref:System.Windows.Xps.XpsDocumentWriter.WriteAsync%2A>. 

## <a name="syntax"></a>Синтаксис

``` csharp
internal event WritingCancelledEventHandler _WritingCancelled
```

## <a name="requirements"></a>Требования

**Пространство имен:** <xref:System.Windows.Xps>

**Сборка:** System. Printing (в System. Printing. dll)

**.NET Framework версии:** Доступно с 3,0
