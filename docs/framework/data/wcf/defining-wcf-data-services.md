# <a name="defining-wcf-data-services"></a>Определение служб данных WCF

В этом разделе описывается создание и Настройка WCF Data Services для предоставления данных в виде веб-канала Open Data Protocol (OData). Дополнительные сведения о базовых шагах, необходимых для создания службы данных, см. в разделе [предоставление данных в виде службы](exposing-your-data-as-a-service-wcf-data-services.md).
