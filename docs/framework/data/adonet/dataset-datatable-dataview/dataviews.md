# <a name="dataviews"></a><span data-ttu-id="e0e18-102">Объекты DataView</span><span class="sxs-lookup"><span data-stu-id="e0e18-102">DataViews</span></span>
<span data-ttu-id="e0e18-103"><xref:System.Data.DataView> позволяет создавать различные представления данных, которые хранятся в <xref:System.Data.DataTable>. Эта возможность часто используется в приложениях связывания данных.</span><span class="sxs-lookup"><span data-stu-id="e0e18-103">A <xref:System.Data.DataView> enables you to create different views of the data stored in a <xref:System.Data.DataTable>, a capability that is often used in data-binding applications.</span></span> <span data-ttu-id="e0e18-104">С помощью **DataView**, можно представлять данные в таблице с различными порядками сортировки, и можно фильтровать данные по состоянию строки или на основе критерия фильтра.</span><span class="sxs-lookup"><span data-stu-id="e0e18-104">Using a **DataView**, you can expose the data in a table with different sort orders, and you can filter the data by row state or based on a filter expression.</span></span>  
  
 <span data-ttu-id="e0e18-105">Объект **DataView** обеспечивает динамическое представление данных в базовом **DataTable**: содержимое, упорядочение и членство отображают изменения по мере их появления.</span><span class="sxs-lookup"><span data-stu-id="e0e18-105">A **DataView** provides a dynamic view of data in the underlying **DataTable**: the content, ordering, and membership reflect changes as they occur.</span></span> <span data-ttu-id="e0e18-106">Это поведение отличается от **выберите** метод **DataTable**, который возвращает <xref:System.Data.DataRow> массива из таблицы на основании определенного фильтра или порядка сортировки: это содержимое отражает изменения базовые таблицы, но при этом членство и упорядочение остаются в статический.</span><span class="sxs-lookup"><span data-stu-id="e0e18-106">This behavior differs from the **Select** method of the **DataTable**, which returns a <xref:System.Data.DataRow> array from a table based on a particular filter and/or sort order: this content reflects changes to the underlying table, but its membership and ordering remain static.</span></span> <span data-ttu-id="e0e18-107">Динамические возможности **DataView** идеально подходит для приложений привязки данных.</span><span class="sxs-lookup"><span data-stu-id="e0e18-107">The dynamic capabilities of the **DataView** make it ideal for data-binding applications.</span></span>  
  
 <span data-ttu-id="e0e18-108">Объект **DataView** обеспечивает динамическое представление единого набора данных, как представление базы данных, к которому можно применить различные критерии сортировки и фильтрации.</span><span class="sxs-lookup"><span data-stu-id="e0e18-108">A **DataView** provides you with a dynamic view of a single set of data, much like a database view, to which you can apply different sorting and filtering criteria.</span></span> <span data-ttu-id="e0e18-109">В отличие от представления базы данных, однако **DataView** не может рассматриваться как таблицы и не может обеспечить представление соединяемых таблиц.</span><span class="sxs-lookup"><span data-stu-id="e0e18-109">Unlike a database view, however, a **DataView** cannot be treated as a table and cannot provide a view of joined tables.</span></span> <span data-ttu-id="e0e18-110">Кроме того, нельзя исключать столбцы, существующие в исходной таблице, добавлять столбцы (например, вычисляемые), которых нет в исходной таблице.</span><span class="sxs-lookup"><span data-stu-id="e0e18-110">You also cannot exclude columns that exist in the source table, nor can you append columns, such as computational columns, that do not exist in the source table.</span></span>  
  
 <span data-ttu-id="e0e18-111">Можно использовать <xref:System.Data.DataView.DataViewManager%2A> управлять настройками представления для всех таблиц в **набора данных**.</span><span class="sxs-lookup"><span data-stu-id="e0e18-111">You can use a <xref:System.Data.DataView.DataViewManager%2A> to manage view settings for all the tables in a **DataSet**.</span></span> <span data-ttu-id="e0e18-112">**DataViewManager** предоставляет удобный способ управления настройками представления по умолчанию для каждой таблицы.</span><span class="sxs-lookup"><span data-stu-id="e0e18-112">The **DataViewManager** provides you with a convenient way to manage default view settings for each table.</span></span> <span data-ttu-id="e0e18-113">При привязке элемента управления к более чем одну таблицу **набора данных**привязка **DataViewManager** является идеальным выбором.</span><span class="sxs-lookup"><span data-stu-id="e0e18-113">When binding a control to more than one table of a **DataSet**, binding to a **DataViewManager** is the ideal choice.</span></span>
