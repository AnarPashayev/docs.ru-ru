 <span data-ttu-id="8dfee-110">Кроме того, строго типизированные **набора данных** предоставляет доступ к значениям как правильный тип во время компиляции.</span><span class="sxs-lookup"><span data-stu-id="8dfee-110">Additionally, the strongly typed **DataSet** provides access to values as the correct type at compile time.</span></span> <span data-ttu-id="8dfee-111">С помощью строго типизированного **набора данных**, перехватываются ошибки несоответствия типов, когда код компилируется а не во время выполнения.</span><span class="sxs-lookup"><span data-stu-id="8dfee-111">With a strongly typed **DataSet**, type mismatch errors are caught when the code is compiled rather than at run time.</span></span>
