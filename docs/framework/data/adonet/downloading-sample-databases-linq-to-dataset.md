# <a name="download-sample-databases-linq-to-dataset"></a>Загрузка выборочных баз данных (LIN) к DataSet)

Образцы и пошаговые работы в документации ДЛЯ DataSet с использованием выборочной базы данных AdventureWorks. Вы можете скачать эту выборочную базу данных бесплатно. Просмотр [выборочных баз данных AdventureWorks](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks) для загрузки базы данных. Затем следуйте инструкциям на этой странице для присоединения или восстановления базы данных.
  
## <a name="sql-server-express"></a>SQL Server Express

Образцы и пошаговые листы в разделе LIN's to DataSet используют s'L Server 2005 в качестве хранилища данных, но их можно изменить, чтобы использовать [s'L Server Express.](https://go.microsoft.com/fwlink/?linkid=866658) Серверэкспресс s'L является бесплатным, и вы можете перераспределить его с приложениями.
