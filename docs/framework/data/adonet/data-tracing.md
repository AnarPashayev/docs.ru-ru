<span data-ttu-id="98e82-131">Ниже приведен пример, использующий Transact-SQL для запуска сеанса расширенных событий, который будет храниться в кольцевом буфере, и будет записывать идентификатор действия, отправленный клиентом по протоколу RPC и пакетным операциям.</span><span class="sxs-lookup"><span data-stu-id="98e82-131">The following is a sample that uses Transact-SQL to start an extended events session that will be stored in a ring buffer and will record the activity ID sent from a client on RPC and batch operations.</span></span>
