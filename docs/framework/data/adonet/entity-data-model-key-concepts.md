бизнес-приложении.</span><span class="sxs-lookup"><span data-stu-id="bc419-107">In a conceptual model, entity types are constructed from [properties](property.md) and describe the structure of  <span data-ttu-id="bc419-127">[Типы сущностей](entity-type.md) содержат [свойства](property.md) , определяющие их структуру и характеристики.</span><span class="sxs-lookup"><span data-stu-id="bc419-127">[Entity types](entity-type.md) contain [properties](property.md) that define their structure and characteristics.</span></span> <span data-ttu-id="bc419-128">Например, тип сущности «Клиент» может иметь свойства, такие как идентификатор клиента, имя и адрес.</span><span class="sxs-lookup"><span data-stu-id="bc419-128">For example, a Customer entity type may have properties such as CustomerId, Name, and Address.</span></span>  
  
 <span data-ttu-id="bc419-129">Свойства в концептуальной модели аналогичны свойствам, которые определены применительно к классу в компьютерной программе.</span><span class="sxs-lookup"><span data-stu-id="bc419-129">Properties in a conceptual model are analogous to properties defined on a class in a computer program.</span></span> <span data-ttu-id="bc419-130">По такому же принципу, как свойства, относящиеся к классу, определяют форму класса и несут информацию об объектах, свойства в концептуальной модели определяют форму типа сущности и несут информацию об экземплярах типа сущности.</span><span class="sxs-lookup"><span data-stu-id="bc419-130">In the same way that properties on a class define the shape of the class and carry information about objects, properties in a conceptual model define the shape of an entity type and carry information about entity type instances.</span></span>  
  
 <span data-ttu-id="bc419-131">Свойство может содержать примитивные данные (такие как строка, целое число, логическое значение) или структурированные данные (такие как сложный тип).</span><span class="sxs-lookup"><span data-stu-id="bc419-131">A property can contain primitive data (such as a string, an integer, or a Boolean value), or structured data (such as a complex type).</span></span> <span data-ttu-id="bc419-132">Дополнительные сведения см. в разделе [EDM: примитивные типы данных](entity-data-model-primitive-data-types.md).</span><span class="sxs-lookup"><span data-stu-id="bc419-132">For more information, see [Entity Data Model: Primitive Data Types](entity-data-model-primitive-data-types.md).</span></span>
