<span data-ttu-id="8e159-103">*Тип ассоциации* (называемый также Ассоциацией) — это фундаментальный Стандартный блок для описания связей в EDM (EDM).</span><span class="sxs-lookup"><span data-stu-id="8e159-103">An *association type* (also called an association) is the fundamental building block for describing relationships in the Entity Data Model (EDM).</span></span> <span data-ttu-id="8e159-104">В концептуальной модели Ассоциация представляет связь между двумя [типами сущностей](entity-type.md) (например `Customer` , и `Order` ).</span><span class="sxs-lookup"><span data-stu-id="8e159-104">In a conceptual model, an association represents a relationship between two [entity types](entity-type.md) (such as `Customer` and `Order`).</span></span> <span data-ttu-id="8e159-105">В приложении экземпляр ассоциации представляет собой специфическую ассоциацию (такую как ассоциация между экземпляром `Customer` и экземпляром `Order`).</span><span class="sxs-lookup"><span data-stu-id="8e159-105">In an application, an instance of an association represents a specific association (such as an association between an instance of `Customer` and an instance of `Order`).</span></span> <span data-ttu-id="8e159-106">Экземпляры ассоциаций логически группируются в [набор ассоциаций](association-set.md).</span><span class="sxs-lookup"><span data-stu-id="8e159-106">Association instances are logically grouped in an [association set](association-set.md).</span></span>

    > [!NOTE]

 <span data-ttu-id="8e159-116">Каждый элемент ассоциации должен указывать количество [элементов ассоциации](association-end-multiplicity.md) , которое указывает число экземпляров типа сущности, которые могут находиться на одном конце ассоциации.</span><span class="sxs-lookup"><span data-stu-id="8e159-116">Each association end must specify an [association end multiplicity](association-end-multiplicity.md) that indicates the number of entity type instances that can be at one end of the association.</span></span> <span data-ttu-id="8e159-117">Кратность окончания ассоциации может иметь значение один (1), ноль или один (0.. 1) или многие ( \* ).</span><span class="sxs-lookup"><span data-stu-id="8e159-117">An association end multiplicity can have a value of one (1), zero or one (0..1), or many (\*).</span></span> <span data-ttu-id="8e159-118">Доступ к экземплярам типа сущности на одном конце ассоциации возможен через [Свойства навигации](navigation-property.md) или внешние ключи, если они представлены в типе сущности.</span><span class="sxs-lookup"><span data-stu-id="8e159-118">Entity type instances at one end of an association can be accessed through [navigation properties](navigation-property.md) or foreign keys if they are exposed on an entity type.</span></span> <span data-ttu-id="8e159-119">Дополнительные сведения см. в разделе [EDM: внешние ключи](foreign-key-property.md).</span><span class="sxs-lookup"><span data-stu-id="8e159-119">For more information, see [Entity Data Model: Foreign Keys](foreign-key-property.md).</span></span>
