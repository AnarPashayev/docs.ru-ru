### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="92a59-116">Создание модели School для проекта Visual Basic с помощью программы EdmGen.exe</span><span class="sxs-lookup"><span data-stu-id="92a59-116">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>  
  
1. <span data-ttu-id="92a59-117">Создайте базу данных School.</span><span class="sxs-lookup"><span data-stu-id="92a59-117">Create the School database.</span></span> <span data-ttu-id="92a59-118">Дополнительные сведения см. в разделе [Создание образца базы данных School](https://docs.microsoft.com/previous-versions/dotnet/netframework-4.0/bb399731(v=vs.100)).</span><span class="sxs-lookup"><span data-stu-id="92a59-118">For more information, see [Creating the School Sample Database](https://docs.microsoft.com/previous-versions/dotnet/netframework-4.0/bb399731(v=vs.100)).</span></span>  
  
2. <span data-ttu-id="92a59-119">Выполните в командной строке следующую команду (введя ее без разрывов строк):</span><span class="sxs-lookup"><span data-stu-id="92a59-119">At the command prompt, execute the following command without line breaks:</span></span>  
  
    ```  
    "%windir%\Microsoft.NET\Framework\v4.0.30319\edmgen.exe" /mode:fullgeneration   
    /c:"Data Source=%datasourceserver%; Initial Catalog=School; Integrated Security=SSPI"   
    /project:School /entitycontainer:SchoolEntities /namespace:SchoolModel /language:VB  
    ```
