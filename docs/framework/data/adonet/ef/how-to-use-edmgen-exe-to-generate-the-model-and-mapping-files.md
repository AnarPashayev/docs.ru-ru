### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="56338-116">Создание модели School для проекта Visual Basic с помощью программы EdmGen.exe</span><span class="sxs-lookup"><span data-stu-id="56338-116">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>

2. <span data-ttu-id="56338-119">Выполните в командной строке следующую команду (введя ее без разрывов строк):</span><span class="sxs-lookup"><span data-stu-id="56338-119">At the command prompt, execute the following command without line breaks:</span></span>

    ```console  
    /c:"Data Source=%datasourceserver%; Initial Catalog=School; Integrated Security=SSPI"
    /project:School /entitycontainer:SchoolEntities /namespace:SchoolModel /language:VB  
    ```
