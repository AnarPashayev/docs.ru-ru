 <span data-ttu-id="5a999-108">Операция запроса LINQ состоит из трех действий: получение одного или нескольких источников данных, создание запроса и его выполнение.</span><span class="sxs-lookup"><span data-stu-id="5a999-108">A LINQ query operation consists of three actions: obtain the data source or sources, create the query, and execute the query.</span></span>  
  
 <span data-ttu-id="5a999-109">К источникам данных, реализующим общий интерфейс <xref:System.Collections.Generic.IEnumerable%601> или <xref:System.Linq.IQueryable%601>, можно выполнять запрос с помощью LINQ.</span><span class="sxs-lookup"><span data-stu-id="5a999-109">Data sources that implement the <xref:System.Collections.Generic.IEnumerable%601> generic interface or the <xref:System.Linq.IQueryable%601> generic interface can be queried through LINQ.</span></span> <span data-ttu-id="5a999-110">Экземпляры общего <xref:System.Data.Objects.ObjectQuery%601> класса, который реализует <xref:System.Linq.IQueryable%601> общий интерфейс, служат источником данных для запросов ДЛЯ ЛИНЗ для сущностей.</span><span class="sxs-lookup"><span data-stu-id="5a999-110">Instances of the generic <xref:System.Data.Objects.ObjectQuery%601> class, which implements the generic <xref:System.Linq.IQueryable%601> interface, serve as the data source for LINQ to Entities queries.</span></span> <span data-ttu-id="5a999-111">Универсальный класс <xref:System.Data.Objects.ObjectQuery%601> представляет запрос, который возвращает коллекцию, включающую ноль или более типизированных объектов.</span><span class="sxs-lookup"><span data-stu-id="5a999-111">The <xref:System.Data.Objects.ObjectQuery%601> generic class represents a query that returns a collection of zero or more typed objects.</span></span> <span data-ttu-id="5a999-112">Вы также можете позволить компилятору сделать вывод о `var` типе сущности, используя ключевое слово C's (Dim in Visual Basic).</span><span class="sxs-lookup"><span data-stu-id="5a999-112">You can also let the compiler infer the type of an entity by using the C# keyword `var` (Dim in Visual Basic).</span></span>  
  
 <span data-ttu-id="5a999-113">В запросе указываются данные, которые необходимо получить из источника данных.</span><span class="sxs-lookup"><span data-stu-id="5a999-113">In the query, you specify exactly the information that you want to retrieve from the data source.</span></span> <span data-ttu-id="5a999-114">В запросе можно также указать, как следует сортировать, группировать и формировать возвращаемую информацию.</span><span class="sxs-lookup"><span data-stu-id="5a999-114">A query can also specify how that information should be sorted, grouped, and shaped before it is returned.</span></span> <span data-ttu-id="5a999-115">В LINQ запрос хранится в переменной.</span><span class="sxs-lookup"><span data-stu-id="5a999-115">In LINQ, a query is stored in a variable.</span></span> <span data-ttu-id="5a999-116">Если запрос возвращает последовательность значений, то переменная запроса должна иметь тип данных, который может быть запрошен.</span><span class="sxs-lookup"><span data-stu-id="5a999-116">If the query returns a sequence of values, the query variable itself must be a queryable type.</span></span> <span data-ttu-id="5a999-117">Эта переменная запроса не выполняет никаких действий и не возвращает данные. ## <a name="query-syntax"></a><span data-ttu-id="5a999-119">Синтаксис запроса</span><span class="sxs-lookup"><span data-stu-id="5a999-119">Query Syntax</span></span>  
 <span data-ttu-id="5a999-120">Создание запросов LINQ to Entities может производиться с использованием синтаксиса выражений запросов или синтаксиса запросов на основе методов.</span><span class="sxs-lookup"><span data-stu-id="5a999-120">LINQ to Entities queries can be composed in two different syntaxes: query expression syntax and method-based query syntax.</span></span> <span data-ttu-id="5a999-121">Синтаксис выражения запроса появился в языках C# 3.0 и Visual Basic 9.0 и состоит из набора предложений, написанных в декларативном стиле, как в языках Transact-SQL или XQuery.</span><span class="sxs-lookup"><span data-stu-id="5a999-121">Query expression syntax is new in C# 3.0 and Visual Basic 9.0, and it consists of a set of clauses written in a declarative syntax similar to Transact-SQL or XQuery.</span></span> <span data-ttu-id="5a999-122">Однако время выполнения общего языка .NET Framework (CLR) не может прочитать синтаксис выражения запроса.</span><span class="sxs-lookup"><span data-stu-id="5a999-122">However, the .NET Framework common language runtime (CLR) cannot read the query expression syntax itself.</span></span> <span data-ttu-id="5a999-123">Таким образом, во время компиляции выражения запроса преобразуются в то, что понятно CLR -
