 <span data-ttu-id="59cdd-108">Операция запроса LINQ состоит из трех действий: получение одного или нескольких источников данных, создание запроса и его выполнение.</span><span class="sxs-lookup"><span data-stu-id="59cdd-108">A LINQ query operation consists of three actions: obtain the data source or sources, create the query, and execute the query.</span></span>  
  
 <span data-ttu-id="59cdd-109">К источникам данных, реализующим общий интерфейс <xref:System.Collections.Generic.IEnumerable%601> или <xref:System.Linq.IQueryable%601>, можно выполнять запрос с помощью LINQ.</span><span class="sxs-lookup"><span data-stu-id="59cdd-109">Data sources that implement the <xref:System.Collections.Generic.IEnumerable%601> generic interface or the <xref:System.Linq.IQueryable%601> generic interface can be queried through LINQ.</span></span> <span data-ttu-id="59cdd-110">Экземпляры универсального <xref:System.Data.Objects.ObjectQuery%601> класса, реализующего универсальный <xref:System.Linq.IQueryable%601> интерфейс, служат источником данных для запросов LINQ to Entities.</span><span class="sxs-lookup"><span data-stu-id="59cdd-110">Instances of the generic <xref:System.Data.Objects.ObjectQuery%601> class, which implements the generic <xref:System.Linq.IQueryable%601> interface, serve as the data source for LINQ to Entities queries.</span></span> <span data-ttu-id="59cdd-111">Универсальный класс <xref:System.Data.Objects.ObjectQuery%601> представляет запрос, который возвращает коллекцию, включающую ноль или более типизированных объектов.</span><span class="sxs-lookup"><span data-stu-id="59cdd-111">The <xref:System.Data.Objects.ObjectQuery%601> generic class represents a query that returns a collection of zero or more typed objects.</span></span> <span data-ttu-id="59cdd-112">Можно также разрешить компилятору выводить тип сущности с помощью C# ключевого слова `var` (Dim в Visual Basic).</span><span class="sxs-lookup"><span data-stu-id="59cdd-112">You can also let the compiler infer the type of an entity by using the C# keyword `var` (Dim in Visual Basic).</span></span>  
  
 <span data-ttu-id="59cdd-113">В запросе указываются данные, которые необходимо получить из источника данных.</span><span class="sxs-lookup"><span data-stu-id="59cdd-113">In the query, you specify exactly the information that you want to retrieve from the data source.</span></span> <span data-ttu-id="59cdd-114">В запросе можно также указать, как следует сортировать, группировать и формировать возвращаемую информацию.</span><span class="sxs-lookup"><span data-stu-id="59cdd-114">A query can also specify how that information should be sorted, grouped, and shaped before it is returned.</span></span> <span data-ttu-id="59cdd-115">В LINQ запрос хранится в переменной.</span><span class="sxs-lookup"><span data-stu-id="59cdd-115">In LINQ, a query is stored in a variable.</span></span> <span data-ttu-id="59cdd-116">Если запрос возвращает последовательность значений, то переменная запроса должна иметь тип данных, который может быть запрошен.</span><span class="sxs-lookup"><span data-stu-id="59cdd-116">If the query returns a sequence of values, the query variable itself must be a queryable type.</span></span> <span data-ttu-id="59cdd-117">Эта переменная запроса не выполняет никаких действий и не возвращает данные. Она только хранит информацию о запросе.</span><span class="sxs-lookup"><span data-stu-id="59cdd-117">This query variable takes no action and returns no data; it only stores the query information.</span></span> <span data-ttu-id="59cdd-118">После создания запроса его необходимо выполнить, чтобы получить данные.</span><span class="sxs-lookup"><span data-stu-id="59cdd-118">After you create a query you must execute that query to retrieve any data.</span></span>  
  
## <a name="query-syntax"></a><span data-ttu-id="59cdd-119">Синтаксис запроса</span><span class="sxs-lookup"><span data-stu-id="59cdd-119">Query Syntax</span></span>  
 <span data-ttu-id="59cdd-120">Создание запросов LINQ to Entities может производиться с использованием синтаксиса выражений запросов или синтаксиса запросов на основе методов.</span><span class="sxs-lookup"><span data-stu-id="59cdd-120">LINQ to Entities queries can be composed in two different syntaxes: query expression syntax and method-based query syntax.</span></span> <span data-ttu-id="59cdd-121">Синтаксис выражений запросов впервые пойдет C# в 3,0 и Visual Basic 9,0.
