<span data-ttu-id="408cf-104">Запрос представляет собой выражение, извлекающее данные из источника данных.</span><span class="sxs-lookup"><span data-stu-id="408cf-104">A query is an expression that retrieves data from a data source.</span></span> <span data-ttu-id="408cf-105">Запросы обычно выражаются на специализированном языке запросов, например SQL для реляционных баз данных и XQuery для XML.</span><span class="sxs-lookup"><span data-stu-id="408cf-105">Queries are usually expressed in a specialized query language, such as SQL for relational databases and XQuery for XML.</span></span> <span data-ttu-id="408cf-106">Поэтому разработчикам приходится учить новый язык запросов для каждого типа источника данных и формата данных, для которых выполняется запрос.</span><span class="sxs-lookup"><span data-stu-id="408cf-106">Therefore, developers have had to learn a new query language for each type of data source or data format that they query.</span></span> <span data-ttu-id="408cf-107">Интегрированный в язык запрос (LINQ) предлагает упрощенную согласованную модель работы с данными для различных типов источников данных и различных форматов данных.</span><span class="sxs-lookup"><span data-stu-id="408cf-107">Language-Integrated Query (LINQ) offers a simpler, consistent model for working with data across various kinds of data sources and formats.</span></span> <span data-ttu-id="408cf-108">Запросы LINQ всегда работают с программируемыми объектами.</span><span class="sxs-lookup"><span data-stu-id="408cf-108">In a LINQ query, you always work with programming objects.</span></span>  
  
 <span data-ttu-id="408cf-109">Операция запроса LINQ состоит из трех действий: получение одного или нескольких источников данных, создание запроса и его выполнение.</span><span class="sxs-lookup"><span data-stu-id="408cf-109">A LINQ query operation consists of three actions: obtain the data source or sources, create the query, and execute the query.</span></span>  
  
 <span data-ttu-id="408cf-110">К источникам данных, реализующим общий интерфейс <xref:System.Collections.Generic.IEnumerable%601> или <xref:System.Linq.IQueryable%601>, можно выполнять запрос с помощью LINQ.</span><span class="sxs-lookup"><span data-stu-id="408cf-110">Data sources that implement the <xref:System.Collections.Generic.IEnumerable%601> generic interface or the <xref:System.Linq.IQueryable%601> generic interface can be queried through LINQ.</span></span> <span data-ttu-id="408cf-111">Экземпляры универсального <xref:System.Data.Objects.ObjectQuery%601> класса, реализующего универсальный <xref:System.Linq.IQueryable%601> интерфейс, служат источником данных для запросов LINQ to Entities.</span><span class="sxs-lookup"><span data-stu-id="408cf-111">Instances of the generic <xref:System.Data.Objects.ObjectQuery%601> class, which implements the generic <xref:System.Linq.IQueryable%601> interface, serve as the data source for LINQ to Entities queries.</span></span> <span data-ttu-id="408cf-112">Универсальный класс <xref:System.Data.Objects.ObjectQuery%601> представляет запрос, который возвращает коллекцию, включающую ноль или более типизированных объектов.</span><span class="sxs-lookup"><span data-stu-id="408cf-112">The <xref:System.Data.Objects.ObjectQuery%601> generic class represents a query that returns a collection of zero or more typed objects.</span></span> <span data-ttu-id="408cf-113">Можно также разрешить компилятору выводить тип сущности с помощью ключевого слова C# `var` (Dim в Visual Basic).</span><span class="sxs-lookup"><span data-stu-id="408cf-113">You can also let the compiler infer the type of an entity by using the C# keyword `var` (Dim in Visual Basic).</span></span>  
  
 <span data-ttu-id="408cf-114">В запросе указываются данные, которые необходимо получить из источника данных.</span><span class="sxs-lookup"><span data-stu-id="408cf-114">In the query, you specify exactly the information that you want to retrieve from the data source.</span></span> <span data-ttu-id="408cf-115">В запросе можно также указать, как следует сортировать, группировать и формировать возвращаемую информацию.</span><span class="sxs-lookup"><span data-stu-id="408cf-115">A query can also specify how that information should be sorted, grouped, and shaped before it is returned.</span></span> <span data-ttu-id="408cf-116">В LINQ запрос хранится в переменной.</span><span class="sxs-lookup"><span data-stu-id="408cf-116">In LINQ, a query is stored in a variable.</span></span> <span data-ttu-id="408cf-117">Если запрос возвращает последовательность значений, то переменная запроса должна иметь тип данных, который может быть запрошен.</span><span class="sxs-lookup"><span data-stu-id="408cf-117">If the query returns a sequence of values, the query variable itself must be a queryable type.</span></span> <span data-ttu-id="408cf-118">Эта переменная запроса не выполняет никаких действий и не возвращает данные. Она только хранит информацию о запросе.</span><span class="sxs-lookup"><span data-stu-id="408cf-118">This query variable takes no action and returns no data; it only stores the query information.</span></span> <span data-ttu-id="408cf-119">После создания запроса его необходимо выполнить, чтобы получить данные.</span><span class="sxs-lookup"><span data-stu-id="408cf-119">After you create a query you must execute that query to retrieve any data.</span></span>  
  
## <a name="query-syntax"></a><span data-ttu-id="408cf-120">Синтаксис запроса</span><span class="sxs-lookup"><span data-stu-id="408cf-120">Query Syntax</span></span>  
 <span data-ttu-id="408cf-121">Создание запросов LINQ to Entities может производиться с использованием синтаксиса выражений запросов или синтаксиса запросов на основе методов.</span><span class="sxs-lookup"><span data-stu-id="408cf-121">LINQ to Entities queries can be composed in two different syntaxes: query expression syntax and method-based query syntax.</span></span> <span data-ttu-id="408cf-122">Синтаксис выражения запроса появился в языках C# 3.0 и Visual Basic 9.0 и состоит из набора предложений, написанных в декларативном стиле, как в языках Transact-SQL или XQuery.</span><span class="sxs-lookup"><span data-stu-id="408cf-122">Query expression syntax is new in C# 3.0 and Visual Basic 9.0, and it consists of a set of clauses written in a declarative syntax similar to Transact-SQL or XQuery.</span></span> <span data-ttu-id="408cf-123">Однако .NET Framework среды CLR не может считать сам синтаксис выражения запроса.</span><span class="sxs-lookup"><span data-stu-id="408cf-123">However, the .NET Framework common language runtime (CLR) cannot read the query expression syntax itself.</span></span> <span data-ttu-id="408cf-124">Таким образом, во время компиляции выражения запроса преобразуются в то, что понятно CLR -
