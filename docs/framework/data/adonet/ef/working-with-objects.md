# <a name="working-with-objects"></a>Работа с объектами
[!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] позволяет отправлять запрос, вставлять, обновлять и обновлять данные, которые выражаются в виде объектов среды CLR, которые являются экземплярами типов сущности. Типы сущности представляют сущности, определенные в концептуальной модели. Платформа [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] сопоставляет сущности и связи, которые определены в концептуальной модели, с источником данных. [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] Предоставляет средства для следующих целей: материализовать данные, возвращенные из источника данных, как объекты; отслеживать изменения, внесенные в объекты; обработки параллелизма; распространения изменений в источнике данных; и привязки объектов к элементам управления.  
  
 Дополнительные сведения о работе с объектами в последней версии платформы Entity Framework см [работа с объектами](https://go.microsoft.com/fwlink/?LinkId=235289).
