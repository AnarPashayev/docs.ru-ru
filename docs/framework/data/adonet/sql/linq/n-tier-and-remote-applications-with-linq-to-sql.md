 <span data-ttu-id="1643f-108">На среднем уровне сущности создаются при помощи контекста данных, который отслеживает их состояние и управляет отложенной загрузкой из базы данных и передачей изменений в базу данных.</span><span class="sxs-lookup"><span data-stu-id="1643f-108">On the middle tier, entities are created by the data context, which tracks their state, and manages deferred loading from class="sxs-lookup"><span data-stu-id="1643f-111">Entities that the client sends back for updates must be reattached to the data context before providing original values and/or timestamps back to the middle tier if those are required for optimistic concurrency checks.</span></span>

 <span data-ttu-id="1643f-113">В приложениях ASP.NET <xref:System.Web.UI.WebControls.LinqDataSource> управляет большей частью данного сложного процесса.</span><span class="sxs-lookup"><span data-stu-id="1643f-113">In ASP.NET applications, the <xref:System.Web.UI.WebControls.LinqDataSource> manages most of this complexity.</span></span> <span data-ttu-id="1643f-114">Дополнительные сведения см. в статье [Общие сведения о серверном веб-элементе управления LinqDataSource](https://docs.microsoft.com/previous-versions/aspnet/bb547113(v=vs.100)).</span><span class="sxs-lookup"><span data-stu-id="1643f-114">For more information, see [LinqDataSource Web Server Control Overview](https://docs.microsoft.com/previous-versions/aspnet/bb547113(v=vs.100)).</span></span>
