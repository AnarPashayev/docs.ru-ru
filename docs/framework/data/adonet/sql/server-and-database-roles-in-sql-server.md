 <span data-ttu-id="c9a55-138">`dbo`, или владелец базы данных, представляет собой учетную запись пользователя, которая имеет неявно заданные разрешения на выполнение любых действий с базой данных.</span><span class="sxs-lookup"><span data-stu-id="c9a55-138">The `dbo`, or database owner, is a user account that has implied permissions to perform all activities in the database.</span></span> <span data-ttu-id="c9a55-139">Члены предопределенной роли сервера `sysadmin` автоматически сопоставляются с `dbo`.</span><span class="sxs-lookup"><span data-stu-id="c9a55-139">Members of the `sysadmin` fixed server role are automatically mapped to `dbo`.</span></span>  
  
> [!NOTE]
