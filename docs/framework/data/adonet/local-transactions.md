## <a name="determining-the-transaction-type"></a><span data-ttu-id="c1def-108">Определение типа транзакции</span><span class="sxs-lookup"><span data-stu-id="c1def-108">Determining the Transaction Type</span></span>  

 <span data-ttu-id="c1def-109">Транзакция считается локальной транзакцией, если она является однофазной, и непосредственно обрабатывается базой данных.</span><span class="sxs-lookup"><span data-stu-id="c1def-109">A transaction is considered to be a local transaction when it is a single-phase transaction and is handled by the database directly.</span></span> <span data-ttu-id="c1def-110">Транзакция считается распределенной транзакцией, когда она координируется монитором транзакций и использует отказоустойчивые механизмы (например, двухфазной фиксация) для разрешения транзакции.</span><span class="sxs-lookup"><span data-stu-id="c1def-110">A transaction is considered to be a distributed transaction when it is coordinated by a transaction monitor and uses fail-safe mechanisms (such as two-phase commit) for transaction resolution.</span></span>
