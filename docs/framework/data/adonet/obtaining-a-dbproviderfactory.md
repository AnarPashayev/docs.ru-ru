## <a name="registering-dbproviderfactories"></a><span data-ttu-id="2ccac-110">Регистрация фабрик DbProviderFactory</span><span class="sxs-lookup"><span data-stu-id="2ccac-110">Registering DbProviderFactories</span></span>

```xml  
<system.data>  
  <DbProviderFactories>  
    <add name="SqlClient Data Provider"  
     invariant="System.Data.SqlClient"   
     description=".Net Framework Data Provider for SqlServer"   
     type="System.Data.SqlClient.SqlClientFactory, System.Data,   
     Version=2.0.0.0, Culture=neutral, PublicKeyToken=b77a5c561934e089"  
    />  
  </DbProviderFactories>  
</system.data>  
```
