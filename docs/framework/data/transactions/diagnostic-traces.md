## <a name="enabling-tracing"></a><span data-ttu-id="83845-116">Включение трассировки</span><span class="sxs-lookup"><span data-stu-id="83845-116">Enabling Tracing</span></span>  
 <span data-ttu-id="83845-117">Чтобы включить трассировку во время обработки транзакции, необходимо отредактировать файл конфигурации приложения.</span><span class="sxs-lookup"><span data-stu-id="83845-117">To enable traces during transaction processing, you should edit your application’s configuration file.</span></span> <span data-ttu-id="83845-118">Пример.</span><span class="sxs-lookup"><span data-stu-id="83845-118">The following is an example.</span></span>  
  
```xml

<system.diagnostics>  
     <sources>  
          <source name="System.Transactions" switchValue="Warning">  
               <listeners>  
                    <add name="tx"   
                     type="System.Diagnostics.XmlWriterTraceListener"   
                     initializeData= "tx.log" />  
               </listeners>  
          </source>  
     </sources>  
</system.diagnostics>  
</configuration>  
```
