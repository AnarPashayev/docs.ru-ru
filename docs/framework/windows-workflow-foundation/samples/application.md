# <a name="application"></a>Приложение
Этот раздел содержит образцы, связанные с приложениями рабочих процессов в Windows Workflow Foundation (WF).  
  
## <a name="in-this-section"></a>В этом разделе  
 [Процесс утверждения документа](document-approval-process.md)  
 Демонстрируется использование рабочих процессов в бизнес-приложениях.  
  
 [Процесс корпоративных закупок](corporate-purchase-process.md)  
 Демонстрируется использование рабочих процессов в windows-приложениях и приложениях, размещенных на веб-сервере.  
  
 [Процесс найма](hiring-process.md)  
 Демонстрируется, как реализовать бизнес-процесс, используя действия обмена сообщениями и два рабочих процесса, размещенные как службы рабочих процессов.  
  
 [Визуальное отслеживание рабочего процесса](visual-workflow-tracking.md)  
 Демонстрируется, как реализовать бизнес-процесс, используя действия обмена сообщениями и два рабочих процесса, размещенные как службы рабочих процессов.  
  
 [Управление приостановленным экземпляром](suspended-instance-management.md)  
 Показано, как управлять экземплярами рабочего процесса, которые были приостановлены.
