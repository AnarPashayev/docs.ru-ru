# <a name="connection-string-and-connection-string-name"></a>Строка соединения и имя строки соединения
**Строка подключения** свойства указывает строку подключения, Store экземпляра рабочего процесса SQL следует использовать для подключения к базе данных сохраняемости. Этот параметр является необязательным. **Имя строки подключения** свойство указывает имя именованной строки соединения, Store экземпляра рабочего процесса SQL следует использовать для подключения к базе данных сохраняемости. Этот параметр является необязательным. Необходимо задать значение для свойства имя строки подключения или значения свойства строки подключения, если вы не хотите Store экземпляра рабочего процесса SQL для использования по умолчанию, именованную строку соединения **DefaultSqlWorkflowInstanceStoreConnectionString** .
