# <a name="migration-activity-in-wf"></a>Действие «Migration» в WF
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)]предоставляет действие для исполнения действий, которые являются производными от действия в рабочем процессе, основанном на <xref:System.Activities.Activity>. <xref:System.Activities.Statements.Interop> Дополнительные сведения см. в разделе [руководство по миграции](migration-guidance.md) .  
  
> [!NOTE]
> Действие не отображается на панели элементов конструктора рабочих процессов, если для проекта рабочего процесса не задан параметр **целевой платформы** **.NET Framework 4** или более поздней версии. <xref:System.Activities.Statements.Interop>
