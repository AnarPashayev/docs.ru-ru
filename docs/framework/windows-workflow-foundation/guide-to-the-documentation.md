# <a name="guide-to-the-windows-workflow-documentation"></a>Руководство по документации по рабочим процессам Windows
В этом разделе описывается использование документации по Windows Workflow Foundation. Документы, на которые указывают ссылки - это рекомендуемые отправные точки, сгруппированные по интересам и уровням подготовки.  
  
## <a name="new-to-windows-workflow-foundation-programming"></a>Знакомство с программированием Windows Workflow Foundation  
  
- Если вы не знакомы с программированием Windows Workflow Foundation, и вы просто хотите узнать некоторые образцы приложений, которые работают, см. в разделах, перечисленных в разделе [образцы Windows Workflow](./samples/index.md).  
  
- Описание различных моделей управления потоками Windows Workflow Foundation, см. в разделе [разработке рабочих процессов](designing-workflows.md).  
  
- Учебник, в котором рассматриваются основные шаги создания приложения Windows Workflow Foundation, см. в разделе [Приступая к работе](getting-started-tutorial.md).  
  
- Если вы заинтересованы в основные понятия Windows Workflow Foundation, см. в пункте [концептуальный обзор рабочего процесса Windows](conceptual-overview.md) раздел.  
  
- Список терминов, используемых в документации по Windows Workflow Foundation, см. в разделе [глоссарий Windows Workflow Foundation, для .NET Framework 4.5](glossary.md).  
  
- Список новых концепций и функций, см. в разделе [новые возможности в Windows Workflow Foundation](whats-new.md).
