# <a name="windows-workflow-conceptual-overview"></a>Общие сведения о рабочих процессах Windows
Этот раздел содержит набор разделов, в которых обсуждаются более крупные понятия Windows Workflow Foundation (WF).  
  
## <a name="in-this-section"></a>В этом разделе  
 [Обзор Windows Workflow Foundation](overview.md)  
 Описывает основу, позволяющую пользователям создавать системные или пользовательские рабочие процессы в приложениях, написанных для операционных систем Windows Vista, Windows XP, Windows Server 2003 и Windows Server 2008.  
  
 [Основные понятия Windows Workflow Foundation](fundamental-concepts.md)  
 Описание фундаментальных концепций, используемых в Windows Workflow Foundation разработки, которые могут быть новыми для некоторых разработчиков.  
  
 [Архитектура Windows Workflow Foundation](architecture.md)  
 Описывает компоненты, используемые при разработке Windows Workflow Foundation.
