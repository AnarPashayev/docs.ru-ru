 <span data-ttu-id="996d7-105">Следует отметить, что, хотя такие действия, как `Parallel` и `ParallelForEach`, планируют одновременное выполнение нескольких дочерних действий, в рабочем процессе результате эти действия больше всего полезны в приложениях, в которых несколько потенциально блокирующих друг друга действий должны выполняться поочередно.</span><span class="sxs-lookup"><span asynchronous work to be completed.</span></span>
