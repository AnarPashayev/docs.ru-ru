## <a name="example"></a><span data-ttu-id="3c9c1-151">Пример</span><span class="sxs-lookup"><span data-stu-id="3c9c1-151">Example</span></span>  
 <span data-ttu-id="3c9c1-152">Ниже приведен пример кода, который может активировать MDA `releaseHandleFailed`.</span><span class="sxs-lookup"><span data-stu-id="3c9c1-152">The following is a code example that can activate the `releaseHandleFailed` MDA.</span></span>

bool ReleaseHandle()  
{  
    // Calling the Win32 CloseHandle function to release the   
    // native handle wrapped by this SafeHandle. This method returns   
    // false on failure, but should only fail if the input is invalid   
    // (which should not happen here). The method specifically must not   
    // fail simply because of lack of resources or other transient
