> <span data-ttu-id="6f947-106">Процедура настройки и инструкции по построению для данного образца приведены в конце этого раздела.</span><span class="sxs-lookup"><span data-stu-id="6f947-106">The setup procedure and build instructions for this sample are located at the end of this topic.</span></span>  
  
 <span data-ttu-id="6f947-107">Свойство `Endpoint` клиента обеспечивает доступ к информации о конечной точке службы, с которой взаимодействует клиент - включая адрес, привязку и информацию о контракте.</span><span class="sxs-lookup"><span data-stu-id="6f947-107">The client's `Endpoint` property allows access to information about the service endpoint that the client is communicating with—including address, binding and contract information.</span></span> <span data-ttu-id="6f947-108">Свойство `InnerChannel` клиента является экземпляром класса <xref:System.ServiceModel.IClientChannel>, который обеспечивает доступ к информации об используемом канале, такой как его состояние и идентификатор сеанса.</span><span class="sxs-lookup"><span data-stu-id="6f947-108">The client's `InnerChannel` property is an instance of <xref:System.ServiceModel.IClientChannel> that allows access to information about the underlying channel, such as its state and session identifier.</span></span>  
  
```csharp   
// Create a client.  
CalculatorClient client = new CalculatorClient();  
...  
Console.WriteLine("Client - endpoint:  " + client.Endpoint.Address);  
Console.WriteLine("Client - binding:  " + client.Endpoint.Binding.Name);  
Console.WriteLine("Client - contract: " + client.Endpoint.Contract.Name);  
  
IClientChannel channel = client.InnerChannel;  
Console.WriteLine("Client channel - state: " + channel.State);  
Console.WriteLine("Client channel - session identifier: " + channel.SessionId);  
  
//Closing the client gracefully closes the connection and cleans up resources.  
client.Close();  
```
