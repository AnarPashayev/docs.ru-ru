 <span data-ttu-id="4590b-120">WCF включает набор привязок, предоставляемых системой и предназначенных для большинства требований приложений и сценариев.</span><span class="sxs-lookup"><span data-stu-id="4590b-120">WCF includes a set of system-provided bindings that are designed to cover most application requirements and scenarios.</span></span> <span data-ttu-id="4590b-121">В следующих классах представлены некоторые примеры привязок, предоставляемых системой.</span><span class="sxs-lookup"><span data-stu-id="4590b-121">The following classes represent some examples of system-provided bindings:</span></span>

- <span data-ttu-id="4590b-125"><xref:System.ServiceModel.NetMsmqBinding>: Использует технологии двоичной кодировки и кадрирования .NET в сочетании с очередью сообщений (также называемой MSMQ) для создания соединений с сообщениями в очереди с другими конечными точками WCF.</span><span class="sxs-lookup"><span data-stu-id="4590b-125"><xref:System.ServiceModel.NetMsmqBinding>: Uses the .NET binary encoding and framing technologies in conjunction with the Message Queuing (also known as MSMQ) to create queued message connections with other WCF endpoints.</span></span>  
  
 <span data-ttu-id="4590b-126">Полный список привязок, предоставляемых системой, с описаниями см. в разделе [привязки, предоставляемые системой](system-provided-bindings.md).</span><span class="sxs-lookup"><span data-stu-id="4590b-126">For a complete list of system-provided bindings, with descriptions, see [System-Provided Bindings](system-provided-bindings.md).</span></span>
