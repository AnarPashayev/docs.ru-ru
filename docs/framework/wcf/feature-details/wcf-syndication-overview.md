# <a name="wcf-syndication-overview"></a><span data-ttu-id="874a4-102">Общие сведения о синдикации WCF</span><span class="sxs-lookup"><span data-stu-id="874a4-102">WCF Syndication Overview</span></span>
<span data-ttu-id="874a4-103">Windows Communication Foundation (WCF) обеспечивает поддержку для предоставления веб-каналов синдикации из службы WCF.</span><span class="sxs-lookup"><span data-stu-id="874a4-103">Windows Communication Foundation (WCF) provides support for exposing syndication feeds from a WCF service.</span></span> <span data-ttu-id="874a4-104">Синдикация - это механизм интеграции приложения, суть которого в том, что сервер предоставляет некоторые данные приложения в формате с возможностью взаимодействия, называемом веб-канал.</span><span class="sxs-lookup"><span data-stu-id="874a4-104">Syndication is a mechanism of application integration in which a server exposes some application data in an interoperable format known as a feed.</span></span> <span data-ttu-id="874a4-105">Веб-канал - это коллекция данных приложения, которая состоит из метаданных уровня веб-канала (название, автор, URL-адрес и другие метаданные) и серии элементов веб-канала.</span><span class="sxs-lookup"><span data-stu-id="874a4-105">A feed is a collection of application data that consists of some feed-level metadata (title, author, URL, and other metadata) and a series of feed items.</span></span> <span data-ttu-id="874a4-106">В пределах веб-канала эти элементы, как правило, упорядочены по времени (в обратном хронологическом порядке).</span><span class="sxs-lookup"><span data-stu-id="874a4-106">Within the feed, the feed items are usually time-ordered in reverse chronological order.</span></span> <span data-ttu-id="874a4-107">Элемент веб-канала состоит из стандартного набора метаданных уровня элемента (название, URL-адрес, дата создания, категория и другие метаданные уровня элемента) и данных приложения произвольного размера.</span><span class="sxs-lookup"><span data-stu-id="874a4-107">A feed item consists of a standard set of item-level metadata (title, URL, creation date, category, and other item-level metadata) and an arbitrary amount of application specific data.</span></span> <span data-ttu-id="874a4-108">Два наиболее распространенных типов веб-каналов синдикации являются синдикации RSS (Really Simple) 2.0 и Atom 1.0, которые поддерживаются платформой WCF.</span><span class="sxs-lookup"><span data-stu-id="874a4-108">The two most common types of syndication feeds are Really Simple Syndication (RSS) 2.0 and Atom 1.0, both of which are supported by WCF.</span></span>

 <span data-ttu-id="874a4-110">WCF определяет набор классов синдикации, которые позволяют работать с каналами, элементами веб-каналов и связанными с ними метаданными в независимо от формата: <xref:System.ServiceModel.Syndication.SyndicationFeed>, <xref:System.ServiceModel.Syndication.SyndicationItem>, <xref:System.ServiceModel.Syndication.SyndicationPerson>, <xref:System.ServiceModel.Syndication.SyndicationLink>и другие классы синдикации.</span><span class="sxs-lookup"><span data-stu-id="874a4-110">WCF defines a set of syndication-specific classes that allow you to work with feeds, feed items, and the related metadata in a format-independent way: <xref:System.ServiceModel.Syndication.SyndicationFeed>, <xref:System.ServiceModel.Syndication.SyndicationItem>, <xref:System.ServiceModel.Syndication.SyndicationPerson>, <xref:System.ServiceModel.Syndication.SyndicationLink>, and other syndication-specific classes.</span></span> <span data-ttu-id="874a4-111">WCF также определяет классы инфраструктуры на основе модели программирования REST WCF для предоставления поддержки синдикации, включая: <xref:System.ServiceModel.Syndication.Atom10FeedFormatter>, и <xref:System.ServiceModel.Syndication.Rss20FeedFormatter>.</span><span class="sxs-lookup"><span data-stu-id="874a4-111">WCF also defines infrastructure classes that build on the WCF REST Programming Model to provide syndication support including: <xref:System.ServiceModel.Syndication.Atom10FeedFormatter>, and  <xref:System.ServiceModel.Syndication.Rss20FeedFormatter>.</span></span> <span data-ttu-id="874a4-112">Классы модуля форматирования веб-каналов поддерживают сериализацию объектной модели в RSS 2.0 и Atom 1.0 и из них.</span><span class="sxs-lookup"><span data-stu-id="874a4-112">The feed formatter classes support serializing the object model to and from RSS 2.0 and Atom 1.0.</span></span>
