## <a name="asynchronous-communication-using-queues"></a><span data-ttu-id="8121c-169">Асинхронное взаимодействие с использованием очередей</span><span class="sxs-lookup"><span data-stu-id="8121c-169">Asynchronous Communication Using Queues</span></span>  

 <span data-ttu-id="8121c-170">Очереди предоставляют асинхронные средства взаимодействия.</span><span class="sxs-lookup"><span data-stu-id="8121c-170">Queues provide an asynchronous means of communication.</span></span> <span data-ttu-id="8121c-171">Приложения, отправляющие сообщения с использованием очередей, не могут ожидать получения и обработки сообщения принимающим приложением из-за высокой задержки, введенной диспетчером очередей.</span><span class="sxs-lookup"><span data-stu-id="8121c-171">Applications that send messages using queues cannot wait for the message to be received and processed by the receiver because of high latency introduced by the queue manager.</span></span> <span data-ttu-id="8121c-172">Сообщения могут оставаться в очереди гораздо дольше, чем предполагалось приложением.</span><span class="sxs-lookup"><span data-stu-id="8121c-172">Messages can remain in the queue for a far longer time than the application intended.</span></span> <span data-ttu-id="8121c-173">Чтобы избежать этого, приложение может задать сообщению значение срока жизни.</span><span class="sxs-lookup"><span data-stu-id="8121c-173">To avoid this, the application can specify a Time-To-Live value on the message.</span></span> <span data-ttu-id="8121c-174">Это значение задает, как долго сообщение должно оставаться в очереди передачи.</span><span class="sxs-lookup"><span data-stu-id="8121c-174">This value specifies how long the message should remain in the transmission queue.</span></span> <span data-ttu-id="8121c-175">Если данное временное значение превышено, а сообщение не отправлено в целевую очередь, оно может быть передано в очередь недоставленных сообщений.</span><span class="sxs-lookup"><span data-stu-id="8121c-175">If this time value is exceeded, and the message still has not been sent to the target queue, the message can be transferred to a dead-letter queue.</span></span>
