# <a name="wcf-discovery"></a>Обнаружение WCF
Windows Communication Foundation (WCF) предоставляет поддержку для включения служб для обнаружения во время выполнения совместимым способом с помощью протокола WS-Discovery. Службы WCF можно объявлять свою доступность сети, с помощью многоадресной рассылки сообщения или на прокси-сервере обнаружения. Клиентские приложения могут осуществлять поиск служб, отвечающих набору указанных критериев, в сети или на прокси-сервере обнаружения. В подразделах этого раздела представлены общие сведения и описание модели программирования данной функции.
