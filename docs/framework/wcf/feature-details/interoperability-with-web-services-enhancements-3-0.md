# <a name="interoperability-with-web-services-enhancements-30"></a>Взаимодействие с расширениями веб-служб 3.0
Клиенты и службы Windows Communication Foundation (WCF) являются совместимыми с расширениями для веб-служб 3,0 для клиентов и служб Microsoft .NET (WSE), когда клиенты и службы WCF настроены для использования версии WS-Addressing в августе 2004.
