# <a name="metadata"></a>Метаданные
Windows Communication Foundation (WCF) предоставляет инфраструктуру для экспорта, публикации, извлечения и импорта метаданных службы. Службы WCF используют метаданные, чтобы описать, как взаимодействовать с конечными точками службы, чтобы такие средства, как Svcutil. exe, могли автоматически создавать код клиента для доступа к службе.
