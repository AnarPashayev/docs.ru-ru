<span data-ttu-id="69317-103">При написании приложения Windows Communication Foundation (WCF), использующего сертификат X. 509 для проверки подлинности, часто бывает необходимо указать утверждения, найденные в сертификате.</span><span class="sxs-lookup"><span data-stu-id="69317-103">When writing a Windows Communication Foundation (WCF) application that uses an X.509 certificate for authentication, it is often necessary to specify claims found in the certificate.</span></span> <span data-ttu-id="69317-104">Например, при использовании перечисления <xref:System.Security.Cryptography.X509Certificates.X509FindType.FindByThumbprint> в методе <xref:System.ServiceModel.Security.X509CertificateInitiatorClientCredential.SetCertificate%2A> необходимо указать утверждение отпечатка.</span><span class="sxs-lookup"><span data-stu-id="69317-104">For example, you must supply a thumbprint claim when using the <xref:System.Security.Cryptography.X509Certificates.X509FindType.FindByThumbprint> enumeration in the <xref:System.ServiceModel.Security.X509CertificateInitiatorClientCredential.SetCertificate%2A> method.</span></span> <span data-ttu-id="69317-105">Чтобы найти значение утверждения, необходимо выполнить два действия.</span><span class="sxs-lookup"><span data-stu-id="69317-105">Finding the claim value requires two steps.</span></span> <span data-ttu-id="69317-106">Сначала необходимо открыть оснастку сертификатов консоли управления (MMC).</span><span class="sxs-lookup"><span data-stu-id="69317-106">First, open the Microsoft Management Console (MMC) snap-in for certificates.</span></span> <span data-ttu-id="69317-107">(См. раздел [как просмотреть сертификаты с помощью оснастки MMC](how-to-view-certificates-with-the-mmc-snap-in.md).)  <span data-ttu-id="69317-108">Если сертификат используется для проверки подлинности службы, важно запомнить значение столбца **Кому выдан** (первый столбец консоли).</span><span class="sxs-lookup"><span data-stu-id="69317-108">If you are using a certificate for service authentication, it is important to note the value of the **Issued To** column (the first column in the console).</span></span> <span data-ttu-id="69317-109">При использовании для защиты транспорта протокола SSL одним из первых шагов является сравнение базового адреса универсального кода ресурса (URI) службы со значением поля **Кому выдан** .</span><span class="sxs-lookup"><span data-stu-id="69317-109">When using Secure Sockets Layer (SSL) as a transport security, one of the first checks done is to compare the base address Uniform Resource Identifier (URI) of a service to the **Issued To** value.</span></span> <span data-ttu-id="69317-110">Значения должны совпадать, в противном случае процесс проверки подлинности будет прерван.</span><span class="sxs-lookup"><span data-stu-id="69317-110">The values must match or the authentication process is halted.</span></span>  
  
 <span data-ttu-id="69317-111">Вы также можете использовать командлет PowerShell New-SelfSignedCertificate, чтобы создать временные сертификаты для использования только во время разработки.</span><span class="sxs-lookup"><span data-stu-id="69317-111">You can also use the Powershell New-SelfSignedCertificate cmdlet to create temporary certificates for use only during development.</span></span> <span data-ttu-id="69317-112">Однако по умолчанию такой сертификат не выдается центром сертификации и не может использоваться в производственных целях.</span><span class="sxs-lookup"><span data-stu-id="69317-112">By default, however, such a certificate is not issued by a certification authority and is unusable for production purposes.</span></span> <span data-ttu-id="69317-113">Дополнительные сведения см. ### <a name="to-retrieve-a-certificates-thumbprint"></a><span data-ttu-id="69317-114">Извлечение отпечатка сертификата</span><span class="sxs-lookup"><span data-stu-id="69317-114">To retrieve a certificate's thumbprint</span></span>  
  
1. <span data-ttu-id="69317-115">Откройте оснастку "Сертификаты" консоли управления (MMC).</span><span class="sxs-lookup"><span data-stu-id="69317-115">Open the Microsoft Management Console (MMC) snap-in for certificates.</span></span> <span data-ttu-id="69317-116">(См. раздел [How to: View Certificates with the MMC Snap-in](how-to-view-certificates-with-the-mmc-snap-in.md)).</span><span class="sxs-lookup"><span data-stu-id="69317-116">(See [How to: View Certificates with the MMC Snap-in](how-to-view-certificates-with-the-mmc-snap-in.md).)</span></span>  
  
2. <span data-ttu-id="69317-117">В левой области окна **Корень консоли** щелкните узел **Сертификаты (локальный компьютер)**.</span><span class="sxs-lookup"><span data-stu-id="69317-117">In the **Console Root** window's left pane, click **Certificates (Local Computer)**.</span></span>  
  
3. <span data-ttu-id="69317-118">Щелкните папку **Личные** , чтобы развернуть ее.</span><span class="sxs-lookup"><span data-stu-id="69317-118">Click the **Personal** folder to expand it.</span></span>  
  
4. <span data-ttu-id="69317-119">Щелкните папку **Сертификаты** , чтобы развернуть ее.</span><span class="sxs-lookup"><span data-stu-id="69317-119">Click the **Certificates** folder to expand it.</span></span>  
  
5. <span data-ttu-id="69317-120">В списке сертификатов найдите заголовок **Назначения** .</span><span class="sxs-lookup"><span data-stu-id="69317-120">In the list of certificates, note the **Intended Purposes** heading.</span></span> <span data-ttu-id="69317-121">Найдите сертификат, назначением которого является **Проверка подлинности клиента** .</span><span class="sxs-lookup"><span data-stu-id="69317-121">Find a certificate that lists **Client Authentication** as an intended purpose.</span></span>  
  
6. <span data-ttu-id="69317-122">Дважды щелкните сертификат.</span><span class="sxs-lookup"><span data-stu-id="69317-122">Double-click the certificate.</span></span>  
  
7. <span data-ttu-id="69317-123">В диалоговом окне **Сертификат** перейдите на вкладку **Состав** .</span><span class="sxs-lookup"><span data-stu-id="69317-123">In the **Certificate** dialog box, click the **Details** tab.</span></span>  
  
8. <span data-ttu-id="69317-124">Найдите в списке поле **Отпечаток**и щелкните его.</span><span class="sxs-lookup"><span data-stu-id="69317-124">Scroll through the list of fields and click **Thumbprint**.</span></span>  
  
9. <span data-ttu-id="69317-125">Скопируйте шестнадцатеричные значения из текстового поля.</span><span class="sxs-lookup"><span data-stu-id="69317-125">Copy the hexadecimal characters from the box.</span></span> <span data-ttu-id="69317-126">Если этот отпечаток используется в коде `X509FindType`, удалите пробелы между шестнадцатеричными значениями.</span><span class="sxs-lookup"><span data-stu-id="69317-126">If this thumbprint is used in code for the `X509FindType`, remove the spaces between the hexadecimal numbers.</span></span> <span data-ttu-id="69317-127">Например, отпечаток "a9 09 50 2d d8 2a e4 14 33 e6 f8 38 86 b0 0d 42 77 a3 2a 7b" необходимо задавать в коде в виде "a909502dd82ae41433e6f83886b00d4277a32a7b".</span><span class="sxs-lookup"><span data-stu-id="69317-127">For example, the thumbprint "a9 09 50 2d d8 2a e4 14 33 e6 f8 38 86 b0 0d 42 77 a3 2a 7b" should be specified as "a909502dd82ae41433e6f83886b00d4277a32a7b" in code.</span></span>
