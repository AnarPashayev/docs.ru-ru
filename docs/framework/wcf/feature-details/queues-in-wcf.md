- <span data-ttu-id="65514-117">Операции при отсутствии подключения к сети.</span><span class="sxs-lookup"><span data-stu-id="65514-117">Disconnected operations.</span></span> <span data-ttu-id="65514-118">Операции отправки, получения и обработки могут прерваться во время передачи данных по сетям с высокой задержкой или сетям с ограниченным доступом, например в случае с мобильными устройствами.</span><span class="sxs-lookup"><span data-stu-id="65514-118">Sending, receiving, and processing operations can become disconnected when communicating over high-latency networks or limited-availability networks, such as in the case of mobile devices.</span></span> <span data-ttu-id="65514-119">В очереди выполнение этих операций продолжится даже при отключении конечных точек.</span><span class="sxs-lookup"><span data-stu-id="65514-119">Queues allow these operations to continue, even when the endpoints are disconnected.</span></span> <span data-ttu-id="65514-120">При повторном подключении сообщения из очереди пересылаются в принимающее приложение.</span><span class="sxs-lookup"><span data-stu-id="65514-120">When the connection is reestablished, the queue forwards messages to the receiving application.</span></span>  
  
 <span data-ttu-id="65514-121">Чтобы использовать возможность очередей в приложении WCF, можно использовать один из стандартных привязок, или можно создать пользовательскую привязку, если один из стандартных привязок не удовлетворяет вашим требованиям.</span><span class="sxs-lookup"><span data-stu-id="65514-121">To use the queues feature in a WCF application, you can use one of the standard bindings, or you can create a custom binding if one of the standard bindings does not satisfy your requirements.</span></span> <span data-ttu-id="65514-122">Дополнительные сведения о соответствующих стандартных привязках и способах выбора, см. в разделе [как: Обмен сообщениями с конечными точками WCF и приложениями очереди сообщений](../../../../docs/framework/wcf/feature-details/how-to-exchange-messages-with-wcf-endpoints-and-message-queuing-applications.md).</span><span class="sxs-lookup"><span data-stu-id="65514-122">For more information about relevant standard bindings and how to choose one, see [How to: Exchange Messages with WCF Endpoints and Message Queuing Applications](../../../../docs/framework/wcf/feature-details/how-to-exchange-messages-with-wcf-endpoints-and-message-queuing-applications.md).</span></span> <span data-ttu-id="65514-123">Дополнительные сведения о создании настраиваемых привязок см. в разделе [Пользовательские привязки](../../../../docs/framework/wcf/extending/custom-bindings.md).</span><span class="sxs-lookup"><span data-stu-id="65514-123">For more information about creating custom bindings, see [Custom Bindings](../../../../docs/framework/wcf/extending/custom-bindings.md).</span></span>
