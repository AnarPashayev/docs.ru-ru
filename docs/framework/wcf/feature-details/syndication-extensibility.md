# <a name="syndication-extensibility"></a><span data-ttu-id="421b5-102">Расширяемость синдикации</span><span class="sxs-lookup"><span data-stu-id="421b5-102">Syndication Extensibility</span></span>
<span data-ttu-id="421b5-103">API синдикации предназначен для обеспечения независимой от формата модели программирования, которая позволяет передавать сводный контент по каналам связи в различных форматах.</span><span class="sxs-lookup"><span data-stu-id="421b5-103">The Syndication API is designed to provide a format-neutral programming model that allows syndicated content to be written to the wire in a variety of formats.</span></span> <span data-ttu-id="421b5-104">Абстрактная модель данных состоит из следующих классов:</span><span class="sxs-lookup"><span data-stu-id="421b5-104">The abstract data model consists of the following classes:</span></span>

- <xref:System.ServiceModel.Syndication.SyndicationItem>  
  
- <xref:System.ServiceModel.Syndication.SyndicationLink>  
  
- <xref:System.ServiceModel.Syndication.SyndicationPerson>  
  
 <span data-ttu-id="421b5-105">Эти классы тесно сопоставлены с конструкциями, определенными в спецификации Atom 1.0, хотя некоторые имена различаются.</span><span class="sxs-lookup"><span data-stu-id="421b5-105">These classes map closely to the constructs defined in the Atom 1.0 specification, although some of the names are different.</span></span>
