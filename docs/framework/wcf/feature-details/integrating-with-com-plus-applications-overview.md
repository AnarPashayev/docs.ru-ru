 <span data-ttu-id="986fb-166">При использовании незащищенной привязки возможно искажение передаваемых данных или раскрытие информации.</span><span class="sxs-lookup"><span data-stu-id="986fb-166">The use of a non-secured binding can leave communication open to tampering or information disclosure.</span></span> <span data-ttu-id="986fb-167">Во избежание этого рекомендуется использовать защищенную привязку.</span><span class="sxs-lookup"><span data-stu-id="986fb-167">To prevent this, it is recommended that you use a secured binding.</span></span>  
  
 <span data-ttu-id="986fb-168">При размещении в служба COM+и на веб-сервере клиентское приложение должно разрешать серверному процессу олицетворять клиентского пользователя.</span><span class="sxs-lookup"><span data-stu-id="986fb-168">For the COM+-hosted and Web-hosted modes, client applications must permit the server process to impersonate the client user.</span></span> <span data-ttu-id="986fb-169">Это можно сделать в клиентах WCF, задав для <xref:System.Security.Principal.TokenImpersonationLevel.Impersonation>уровня олицетворения значение.</span><span class="sxs-lookup"><span data-stu-id="986fb-169">This can be done in WCF clients by setting the impersonation level to <xref:System.Security.Principal.TokenImpersonationLevel.Impersonation>.</span></span>

 <span data-ttu-id="986fb-174">При предоставлении приложения COM+ с настроенными ролями COM+ в качестве размещенной в Интернете службы необходимо добавить в одну из ролей приложения учетную запись запуска IIS Process.</span><span class="sxs-lookup"><span data-stu-id="986fb-174">When exposing a COM+ application with configured COM+ roles as a Web-hosted service, the "Launch IIS Process Account" must be added to one of the application’s roles.</span></span> <span data-ttu-id="986fb-175">Эта учетная запись (обычно она имеет имя IWAM_имя_компьютера) должна добавляться, чтобы можно было безопасно отключить объекты после использования.</span><span class="sxs-lookup"><span data-stu-id="986fb-175">This account, typically with the name IWAM_machinename, must be added to enable clean shutdown of objects after use.</span></span> <span data-ttu-id="986fb-176">Не следует предоставлять этой учетной записи какие-либо дополнительные разрешения.</span><span class="sxs-lookup"><span data-stu-id="986fb-176">The account should not be granted any additional permissions.</span></span>
