# <a name="custom-tokens"></a>Пользовательские маркеры
Несмотря на то, что Windows Communication Foundation (WCF) изначально поддерживает сертификаты X.509, маркеры контекста безопасности, билетов Kerberos и маркеров имени пользователя в качестве учетных данных, является достаточно гибким, чтобы можно было использовать собственные пользовательские учетные данные.  
  
## <a name="in-this-section"></a>В этом разделе

 Создание пользовательского маркера безопасности с помощью класса <xref:System.IdentityModel.Tokens.SecurityToken> и его интеграция с поставщиком пользовательских маркеров безопасности и структурой проверки подлинности.
