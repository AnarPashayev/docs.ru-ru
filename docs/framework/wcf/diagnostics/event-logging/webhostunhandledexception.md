# <a name="webhostunhandledexception"></a>WebHostUnhandledException

Идентификатор: 109  
  
 Важность: ошибка  
  
 Категория: ServiceModel  
  
## <a name="description"></a>Описание  

 Данное событие указывает на необработанное исключение WebHost. В событии указаны информация об отправителе, исключение, имя и идентификатор процесса.
