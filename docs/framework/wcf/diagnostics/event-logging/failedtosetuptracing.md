# <a name="failedtosetuptracing"></a>FailedToSetupTracing
ИД: 100  
  
 "Уровень серьезности" — Ошибка  
  
 Категори Трассировка  
  
## <a name="description"></a>Описание  
 Попытка настроить трассировку не удалась. Трассировка отключена.  
  
 В событии указаны исключение, имя процесса и идентификатор процесса.
