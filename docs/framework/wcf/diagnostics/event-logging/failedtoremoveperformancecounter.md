# <a name="failedtoremoveperformancecounter"></a>FailedToRemovePerformanceCounter
ИД: 119  
  
 "Уровень серьезности" — Ошибка  
  
 Категори ServiceModel  
  
## <a name="description"></a>Описание  
 Данное событие указывает, что счетчик производительности не был удален. В событии указаны имя категории, имя счетчика, имя экземпляра, исключение, имя процесса и ИД процесса.
