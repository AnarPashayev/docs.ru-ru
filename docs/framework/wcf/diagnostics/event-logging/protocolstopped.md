# <a name="protocolstopped"></a>ProtocolStopped
ИД: 151  
  
 Уровень серьезности: Error  
  
 Категория: TransactionBridge  
  
## <a name="description"></a>Описание  
 Это событие показывает, что служба протокола WS-Atomic Transaction была остановлена. В событии указаны идентификатор и имя протокола, идентификатор и имя процесса. .
