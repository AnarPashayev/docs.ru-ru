# <a name="wmipropertymissing"></a>WmiPropertyMissing

Идентификатор: 129  
  
 Важность: ошибка  
  
 Категория: ServiceModel  
  
## <a name="description"></a>Описание  

 Это событие указывает, что имело место несоответствие типов. У заданного класса CIM нет указанного свойства конкретного типа. В событии указаны имя и ИД процесса.
