# <a name="failfastexception"></a>FailFastException

Идентификатор: 103  
  
 Важность: ошибка  
  
 Категория: Tracing  
  
## <a name="description"></a>Описание  

 В режиме FailFast возникло исключение. В событии указаны исключение, имя процесса и идентификатор процесса.
