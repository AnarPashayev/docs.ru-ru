# <a name="transportauthenticationsuccess"></a>TransportAuthenticationSuccess

Идентификатор: 173  
  
 Важность: информационное сообщение  
  
 Категория: SecurityAudit  
  
## <a name="description"></a>Описание  

 Это событие показывает, что проверка подлинности транспорта выполнена успешно. В событии указаны служба, учетная запись клиента и идентификатор действия.
