# <a name="protocolrecoverybeginningfailure"></a>ProtocolRecoveryBeginningFailure
ИД: 144  
  
 "Уровень серьезности" — Ошибка  
  
 Категори трансактионбридже  
  
## <a name="description"></a>Описание  
 Это событие означает, что в начале восстановления произошел сбой протокола MSDTC WS-Atomic Transaction. В результате функции протокола WS-Atomic Transaction отключены. В событии указаны идентификатор и имя протокола, исключение, имя и идентификатор процесса.
