# <a name="failedtotraceeventwithexception"></a>FailedToTraceEventWithException

Идентификатор: 105  
  
 Важность: ошибка  
  
 Категория: Tracing  
  
## <a name="description"></a>Описание  

 Одно или несколько событий не трассировались.  
  
 В событии указаны исходная строка события, исключение, имя и идентификатор процесса.
