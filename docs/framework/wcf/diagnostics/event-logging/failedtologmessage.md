# <a name="failedtologmessage"></a>FailedToLogMessage
ИД: 113  
  
 Уровень серьезности: Error  
  
 Категория: ServiceModel  
  
## <a name="description"></a>Описание  
 Это событие показывает, что сообщение не было зарегистрировано в журнале. В событии указаны исключение, имя процесса и ИД процесса.
