# <a name="failfast"></a>FailFast
ИД: 102  
  
 Уровень серьезности: Error  
  
 Категория: Трассировка  
  
## <a name="description"></a>Описание  
 Запущен режим FailFast. В событии указаны сообщение, трассировка стека, имя процесса и ИД процесса.
