# <a name="unknownlisteneradaptererror"></a>UnknownListenerAdapterError
ИД: 161  
  
 "Уровень серьезности" — Ошибка  
  
 Категори SMSvcHost  
  
## <a name="description"></a>Описание  
 Это событие означает, что в адаптере прослушивателя произошла непредвиденная ошибка во время обработки уведомления службы активации Windows. Процесс завершен. В событии указываются протокол, исключение, имя и идентификатор процесса.
