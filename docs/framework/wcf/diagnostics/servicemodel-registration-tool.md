# <a name="servicemodel-registration-tool"></a>Средство регистрации ServiceModel
Программа командной строки ServiceModel Registration Tool позволяет управлять регистрацией ServiceModel на одном компьютере. Дополнительные сведения об этом инструменте см. в разделе [средство регистрации ServiceModel (ServiceModelReg.exe)](../../../../docs/framework/wcf/servicemodelreg-exe.md).
