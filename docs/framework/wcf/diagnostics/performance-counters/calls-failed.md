# <a name="calls-failed"></a>Сбои вызовов
Имя счетчика: Сбои вызовов  
  
## <a name="description"></a>Описание  
 Количество вызовов с необработанными исключениями в данной операции. Этого счетчика увеличивается при каждом обнаружении необработанного исключения в этой операции.
