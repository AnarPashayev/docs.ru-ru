# <a name="calls-duration"></a>Длительность обработки вызовов
Имя счетчика: Длительность обработки вызовов  
  
## <a name="description"></a>Описание  
 Средняя длительность вызовов этой операции. Средняя длительность вычисляется на основе этого уравнения: (N1-N0)/(D1-D0).  
  
> [!WARNING]
> При использовании в асинхронной службе WCF счетчик вызовов будет всегда возвращать значение-1.
