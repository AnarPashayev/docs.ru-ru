# <a name="windowsstreamsecuritybindingelement"></a>WindowsStreamSecurityBindingElement
WindowsStreamSecurityBindingElement  
  
## <a name="syntax"></a>Синтаксис  
  
```csharp
class WindowsStreamSecurityBindingElement : BindingElement  
{  
  string ProtectionLevel;  
};  
```  
  
## <a name="methods"></a>Методы  
 Класс WindowsStreamSecurityBindingElement не определяет никаких методов.  
  
## <a name="properties"></a>Свойства  
 Класс WindowsStreamSecurityBindingElement имеет следующее свойство:  
  
### <a name="protectionlevel"></a>ProtectionLevel  
 Тип данных: string  
  
 Тип доступа: Только чтение  
  
 Уровень защиты для потока TCP.

|---------|-----------------------------------|  
|Пространство имен|Определено в root\ServiceModel.|
