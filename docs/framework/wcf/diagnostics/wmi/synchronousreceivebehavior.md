# <a name="synchronousreceivebehavior"></a>SynchronousReceiveBehavior
SynchronousReceiveBehavior  
  
## <a name="syntax"></a>Синтаксис  
  
```csharp
class SynchronousReceiveBehavior : Behavior  
{  
};  
```  
  
## <a name="methods"></a>Методы
