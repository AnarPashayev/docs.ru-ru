# <a name="4812---discoverymessagewithnullreplyto"></a>4812 - DiscoveryMessageWithNullReplyTo
## <a name="properties"></a>Свойства  
  
|||  
|-|-|  
|ID|4812|  
|Ключевые слова|Обнаружение|  
|Уровень|Предупреждение|  
|Канал|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Описание  
 Это событие создается при удалении клиентом DiscoveryClient сообщения запроса обнаружения из-за отсутствия в нем адреса ReplyTo.  
  
## <a name="message"></a>Сообщение  
 Сообщение запроса обнаружения с messageId="%1" удалено, поскольку отсутствует адрес ReplyTo.
