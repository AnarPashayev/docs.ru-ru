## <a name="description"></a>Описание  

 Это событие отправляется, когда служба маршрутизации передает сообщение.  
  
## <a name="message"></a>Сообщение  

 Служба маршрутизации передает сообщение с идентификатором «%1» [операция %2] в «%3».
