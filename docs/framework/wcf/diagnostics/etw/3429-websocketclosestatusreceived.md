|Level|Подробный|

## <a name="description"></a>Описание  

 Это событие создается при получении состояния закрытия WebSocket.  
  
## <a name="message"></a>Сообщение  

 Получено сообщение о закрытии соединения WebSocketId:%1 с состоянием «%2».
