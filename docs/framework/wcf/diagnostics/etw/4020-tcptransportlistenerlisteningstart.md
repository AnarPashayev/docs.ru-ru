## <a name="description"></a>Описание  

 Это событие создается, когда прослушиватель транспорта TCP начинает прослушивание.  
  
## <a name="message"></a>Сообщение  

 Прослушиватель транспорта TCP начинает прослушивание по URI: «%1».
