|||  
|-|-|  
|ID|3352|  
|Keywords|TCP|  
|Level|Ошибка|

## <a name="description"></a>Описание  

 Это событие создается при истечении времени ожидания TCP-соединения.  
  
## <a name="message"></a>Сообщение  

 Истекло время ожидания SocketId:%1 с удаленным адресом %2.
