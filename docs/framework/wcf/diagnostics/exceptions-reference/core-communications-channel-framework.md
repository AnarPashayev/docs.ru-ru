# <a name="core-communications-channel-framework"></a>Основное взаимодействие: структура каналов
В этом разделе перечислены все исключения, вызываемые инфраструктуры канала Windows Communication Foundation (WCF).  
  
## <a name="exception-list"></a>Список исключений  
  
|Код ресурса|Строка ресурса|  
|-------------------|---------------------|  
|ReceiveRequestTimedOut|Время ожидания полученного запроса определенного локального адреса истекло по прошествии заданного времени. Возможно, выделенное для этой операции время было частью более длительного времени ожидания.|  
|ReceiveTimedOutNoLocalAddress|Срок ожидания операции получения истек по прошествии заданного времени. Возможно, выделенное для этой операции время было частью более длительного времени ожидания.|  
|UnableToDemuxChannel|Нет доступного канала для получения сообщения с заданным действием.|
