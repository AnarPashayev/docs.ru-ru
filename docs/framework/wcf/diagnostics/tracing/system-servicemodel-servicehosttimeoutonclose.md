# <a name="systemservicemodelservicehosttimeoutonclose"></a>System.ServiceModel.ServiceHostTimeoutOnClose
System.ServiceModel.ServiceHostTimeoutOnClose  
  
## <a name="description"></a>Описание  
 Истекло время ожидания для операции закрытия узла ServiceHost. Это говорит о том, что некоторые каналы, связанные с сеансами, не закрыты клиентом надлежащим образом и, возможно, прерваны. Измените время по умолчанию, чтобы отрегулировать время ожидания закрытия.
