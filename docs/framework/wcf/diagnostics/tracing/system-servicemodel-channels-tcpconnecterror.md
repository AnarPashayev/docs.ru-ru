# <a name="systemservicemodelchannelstcpconnecterror"></a>System.ServiceModel.Channels.TcpConnectError
Ошибка операции TCP-соединения.  
  
## <a name="description"></a>Описание  
 Данная запись-предупреждение указывает на сбой соединения с конечной точкой TCP. Это может произойти, если удаленная конечная точка по данному IP-адресу и порту не отвечает. Эту запись можно игнорировать, если последующие попытки соединения с другими допустимыми IP-адресами (такими как IPv4- или IPv6-адреса или другие адреса, представляющие данное имя узла) будут успешными. Исключение в этой записи может содержать дополнительные сведения об ошибке.
