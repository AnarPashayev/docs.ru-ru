# <a name="wcf-system-requirements"></a>Требования WCF к системе

Windows Communication Foundation (WCF) — это инфраструктура связи, которая используется для создания распределенных приложений. Ниже перечислены требования, которые позволяют запускать WCF.

## <a name="system-requirements"></a>Требования к системе

По умолчанию WCF устанавливается в [!INCLUDE[wv](../../../includes/wv-md.md)].

WCF также может быть установлена в [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)]или [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)].

> [!NOTE]
> Функции очереди сообщений (MSMQ) [!INCLUDE[wv](../../../includes/wv-md.md)]WCF поддерживаются только в, [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)]и [!INCLUDE[wxppro](../../../includes/wxppro-md.md)].
