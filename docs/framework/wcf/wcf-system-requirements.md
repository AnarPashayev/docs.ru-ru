# <a name="wcf-system-requirements"></a>Требования WCF к системе

Windows Communication Foundation (WCF) — это инфраструктура связи, которая используется для создания распределенных приложений. Ниже перечислены требования, которые позволяют запускать WCF.

## <a name="system-requirements"></a>Требования к системе

По умолчанию WCF устанавливается на [!INCLUDE[wv](../../../includes/wv-md.md)].

WCF также может быть установлена на [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], Windows Server 2003 R2 или [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)].

> [!NOTE]
> Функции очереди сообщений (MSMQ) WCF поддерживаются только в [!INCLUDE[wv](../../../includes/wv-md.md)], Windows Server 2003 R2, [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)] и Windows XP Professional.
