<xref:System.Windows.Automation.ScrollPatternIdentifiers.NoScroll>.</span><span class="sxs-lookup"><span data-stu-id="dae88-120">Likewise, if <xref:System.Windows.Automation.ScrollPatternIdentifiers.NoScroll>.</span></span> <span data-ttu-id="dae88-121">Это позволяет клиенту автоматизации пользовательского интерфейса использовать эти не заинтересован в активации прокрутки.</span><span class="sxs-lookup"><span data-stu-id="dae88-121">This allows a UI Automation client to use these property values within the - <span data-ttu-id="dae88-122"><xref:System.Windows.Automation.Provider.IScrollProvider.HorizontalScrollPercent%2A> зависит от конкретного языка.</span><span class="sxs-lookup"><span data-stu-id="dae88-122"><xref:System.Windows.Automation.Provider.IScrollProvider.HorizontalScrollPercent%2A> is locale-specific.</span></span> <span data-ttu-id="dae88-123">Установка HorizontalScrollPercent = 100.0 должна задавать расположение прокрутки элемента управления в крайней правой позиции для таких языков, как английский, где чтение выполняется слева направо.</span><span class="sxs-lookup"><span data-stu-id="dae88-123">Setting HorizontalScrollPercent = 100.0 must set the scrolling location of the control to the equivalent of its rightmost position for languages such as English that read left to right.</span></span> <span data-ttu-id="dae88-124">И наоборот, для таких языков, как арабский, где чтение выполняется справа налево, установка HorizontalScrollPercent = 100.0 должен задавать расположение прокрутки в крайней левой позиции.</span><span class="sxs-lookup"><span data-stu-id="dae88-124">Alternately, for languages such as Arabic that read right to left, setting HorizontalScrollPercent = 100.0 must set the scroll location to the leftmost position.</span></span>
