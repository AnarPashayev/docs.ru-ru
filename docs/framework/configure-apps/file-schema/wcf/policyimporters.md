# <a name="policyimporters"></a>\<policyImporters >
Этот элемент конфигурации задает все средства импорта политики, контролирующие импорт утверждений пользовательской политики о привязках. Каждый дочерний элемент является <`policyImporter`>, используемый для поиска утверждений пользовательской политики о свойствах привязки, а также для подключения элемента пользовательской привязки, реализующей свойства, необходимые для утверждения.
