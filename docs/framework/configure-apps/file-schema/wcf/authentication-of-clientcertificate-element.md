## <a name="example"></a><span data-ttu-id="eb052-192">Пример</span><span class="sxs-lookup"><span data-stu-id="eb052-192">Example</span></span>  
 <span data-ttu-id="eb052-193">В следующем примере кода задается сертификат X.509 и пользовательский тип проверки в элементе `<authentication>`.</span><span class="sxs-lookup"><span data-stu-id="eb052-193">The following code specifies an X.509 certificate and a custom validation type in the `<authentication>` element.</span></span>

  <behavior name="myServiceBehavior">
    <clientCertificate>
      <certificate findValue="www.cohowinery.com"
                   storeLocation="CurrentUser"

                      certificateValidationMode="Custom"
                      revocationMode="Offline"
                      includeWindowsGroups="false"
                      mapClientCertificateToWindowsAccount="true" />
