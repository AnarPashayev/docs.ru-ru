# <a name="transports"></a>Транспорты
Транспортный уровень является самым нижним уровнем стека каналов. Основными транспортами, используемыми в Windows Communication Foundation (WCF), являются HTTP, HTTPS, TCP и именованные каналы. В этом разделе представлены элементы конфигурации, которые позволяют настраивать различные типы транспорта.
