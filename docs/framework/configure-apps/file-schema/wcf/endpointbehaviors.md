Дополнительные сведения о поведениях клиентской конфигурации см. в разделе [указание поведения клиента во время выполнения](../../../../../docs/framework/wcf/specifying-client-run-time-behavior.md) и [Настройка поведений клиента](../../../../../docs/framework/wcf/configuring-client-behaviors.md).
