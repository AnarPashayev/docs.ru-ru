# <a name="tokenrequestparameters"></a>\<tokenRequestParameters>
Данный элемент конфигурации содержит дочерние элементы, которые могут использоваться для указания дополнительных элементов XML, которые будут отправляться в теле сообщения в службу маркеров безопасности (STS) при запросе маркера.
