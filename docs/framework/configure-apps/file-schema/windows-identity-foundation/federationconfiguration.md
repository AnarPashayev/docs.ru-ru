## <a name="example"></a><span data-ttu-id="03984-161">Пример</span><span class="sxs-lookup"><span data-stu-id="03984-161">Example</span></span>  
 <span data-ttu-id="03984-162">В следующем XML-коде `<federationConfiguration>` показан элемент, указывающий параметры для всфам и указывающий, что SAM использует обработчик файлов cookie по <xref:System.IdentityModel.Services.ChunkedCookieHandler> умолчанию (экземпляр класса).</span><span class="sxs-lookup"><span data-stu-id="03984-162">The following XML shows a `<federationConfiguration>` element that specifies settings for the WSFAM and specifies that the default cookie handler (an instance of the <xref:System.IdentityModel.Services.ChunkedCookieHandler> class) be used by the SAM.</span></span>  
  
> [!WARNING]
>  <span data-ttu-id="03984-163">В этом примере ни обработчику файлов cookie, ни ВСФАМ не требуются для использования HTTPS.</span><span class="sxs-lookup"><span data-stu-id="03984-163">In this example, neither the cookie handler nor WSFAM are required to use HTTPS.</span></span> <span data-ttu-id="03984-164">Это обусловлено тем `requireHttps` , что атрибут `<wsFederation>` элемента `<cookieHandlerElement>` и `requireSsl` атрибут в имеют `false`значение.</span><span class="sxs-lookup"><span data-stu-id="03984-164">This is because the `requireHttps` attribute on the `<wsFederation>` element and the `requireSsl` attribute on the `<cookieHandlerElement>` are `false`.</span></span> <span data-ttu-id="03984-165">Эти параметры не рекомендуются для большинства рабочих сред, так как они могут представлять угрозу безопасности.</span><span class="sxs-lookup"><span data-stu-id="03984-165">These settings are not recommended for most production environments as they may present a security risk.</span></span>
