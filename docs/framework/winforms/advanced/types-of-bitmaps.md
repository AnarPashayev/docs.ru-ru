 <span data-ttu-id="44e56-177">EXIF — это формат файлов, используемый для фотографий, захваченных цифровыми камерами.</span><span class="sxs-lookup"><span data-stu-id="44e56-177">EXIF is a file format used for photographs captured by digital cameras.</span></span> <span data-ttu-id="44e56-178">Файл EXIF содержит изображение, сжатое в соответствии со спецификацией JPEG.</span><span class="sxs-lookup"><span data-stu-id="44e56-178">An EXIF file contains an image that is compressed according to the JPEG specification.</span></span> <span data-ttu-id="44e56-179">Файл EXIF также содержит сведения о фотографии (Дата съемки, скорость затвора, время экспозиции и т. д.) и сведения о камере (изготовитель, модель и т. д.).</span><span class="sxs-lookup"><span data-stu-id="44e56-179">An EXIF file also contains information about the photograph (date taken, shutter speed, exposure time, and so on) and information about the camera (manufacturer, model, and so on).</span></span>  
  
### <a name="portable-network-graphics-png"></a><span data-ttu-id="44e56-180">Формат PNG (Portable Network Graphics)</span><span class="sxs-lookup"><span data-stu-id="44e56-180">Portable Network Graphics (PNG)</span></span>  
 <span data-ttu-id="44e56-181">Формат PNG позволяет сохранять многие преимущества формата GIF, но также предоставляет возможности, которые выходят за рамки GIF.</span><span class="sxs-lookup"><span data-stu-id="44e56-181">The PNG format retains many of the advantages of the GIF format but also provides capabilities beyond those of GIF.</span></span> <span data-ttu-id="44e56-182">Как и GIF-файлы, PNG-файлы сжимаются без потери информации.</span><span class="sxs-lookup"><span data-stu-id="44e56-182">Like GIF files, PNG files are compressed with no loss of information.</span></span> <span data-ttu-id="44e56-183">PNG-файлы могут хранить цвета с 8, 24 или 48 бит на пиксель, а градации серого — с 1, 2, 4, 8 или 16 битами на пиксель.</span><span class="sxs-lookup"><span data-stu-id="44e56-183">PNG files can store colors with 8, 24, or 48 bits per pixel and grayscales with 1, 2, 4, 8, or 16 bits per pixel.</span></span> <span data-ttu-id="44e56-184">В отличие от этого, GIF-файлы могут использовать только 1, 2, 4 или 8 бит на пиксель.</span><span class="sxs-lookup"><span data-stu-id="44e56-184">In contrast, GIF files can use only 1, 2, 4, or 8 bits per pixel.</span></span> <span data-ttu-id="44e56-185">PNG-файл может также хранить альфа-значение для каждого пикселя, определяющее степень смешения цвета этого пикселя с цветом фона.</span><span class="sxs-lookup"><span data-stu-id="44e56-185">A PNG file can also store an alpha value for each pixel, which specifies the degree to which the color of that pixel is blended with the background color.</span></span>  
  
 <span data-ttu-id="44e56-186">PNG улучшает возможность поэтапного вывода изображения в формате GIF (т. е. для просмотра лучшей и лучшей аппроксимации изображения по мере его поступления через сетевое подключение).</span><span class="sxs-lookup"><span data-stu-id="44e56-186">PNG improves on GIF in its ability to progressively display an image (that is, to display better and better approximations of the image as it arrives over a network connection).</span></span> <span data-ttu-id="44e56-187">PNG-файлы могут содержать гамма-коррекцию и информацию о коррекции цвета, чтобы изображения можно было точно представить на различных устройствах отображения.</span><span class="sxs-lookup"><span data-stu-id="44e56-187">PNG files can contain gamma correction and color correction information so that the images can be accurately rendered on a variety of display devices.</span></span>  
  
### <a name="tag-image-file-format-tiff"></a><span data-ttu-id="44e56-188">Формат файла изображения тегов (TIFF)</span><span class="sxs-lookup"><span data-stu-id="44e56-188">Tag Image File Format (TIFF)</span></span>  
 <span data-ttu-id="44e56-189">TIFF — это гибкий и расширяемый формат, поддерживаемый многими платформами и приложениями обработки изображений.</span><span class="sxs-lookup"><span data-stu-id="44e56-189">TIFF is a flexible and extendable format that is supported by a wide variety of platforms and image-processing applications.</span></span> <span data-ttu-id="44e56-190">TIFF-файлы могут хранить изображения с произвольным количеством битов на пиксель, а также использовать разнообразные алгоритмы сжатия.</span><span class="sxs-lookup"><span data-stu-id="44e56-190">TIFF files can store images with an arbitrary number of bits per pixel and can employ a variety of compression algorithms.</span></span> <span data-ttu-id="44e56-191">Несколько образов могут храниться в одном многостраничном файле TIFF.</span><span class="sxs-lookup"><span data-stu-id="44e56-191">Several images can be stored in a single, multiple-page TIFF file.</span></span> <span data-ttu-id="44e56-192">Сведения, относящиеся к изображению (создание сканера, главный компьютер, тип сжатия, ориентация, выборка на пиксель и т. д.), можно сохранить в файле и упорядочить с помощью тегов.</span><span class="sxs-lookup"><span data-stu-id="44e56-192">Information related to the image (scanner make, host computer, type of compression, orientation, samples per pixel, and so on) can be stored in the file and arranged through the use of tags.</span></span> <span data-ttu-id="44e56-193">Формат TIFF может быть расширен по мере необходимости утверждением и добавлением новых тегов.</span><span class="sxs-lookup"><span data-stu-id="44e56-193">The TIFF format can be extended as needed by the approval and addition of new tags.</span></span>
