 <span data-ttu-id="9eaac-177">EXIF — это формат файла, используемый для хранения фотографий, полученных с помощью цифровой камеры.</span><span class="sxs-lookup"><span data-stu-id="9eaac-177">EXIF is a file format used for photographs captured by digital cameras.</span></span> <span data-ttu-id="9eaac-178">Файл EXIF содержит изображение, сжимается в соответствии со спецификацией JPEG.</span><span class="sxs-lookup"><span data-stu-id="9eaac-178">An EXIF file contains an image that is compressed according to the JPEG specification.</span></span> <span data-ttu-id="9eaac-179">Файл EXIF также содержит данные о фотографии (дата съемки, скорость затвора, ограниченное время и т. д.) и информацию о камеры (производитель, модель и т. д.).</span><span class="sxs-lookup"><span data-stu-id="9eaac-179">An EXIF file also contains information about the photograph (date taken, shutter speed, exposure time, and so on) and information about the camera (manufacturer, model, and so on).</span></span>  
  
### <a name="portable-network-graphics-png"></a><span data-ttu-id="9eaac-180">Формат PNG (Portable Network Graphics)</span><span class="sxs-lookup"><span data-stu-id="9eaac-180">Portable Network Graphics (PNG)</span></span>  
 <span data-ttu-id="9eaac-181">Формат PNG сохраняет многие преимущества формата GIF, но также предоставляет возможности, недоступные в формате GIF.</span><span class="sxs-lookup"><span data-stu-id="9eaac-181">The PNG format retains many of the advantages of the GIF format but also provides capabilities beyond those of GIF.</span></span> <span data-ttu-id="9eaac-182">Подобно файлам GIF PNG-файлы будут сжаты без потери информации.</span><span class="sxs-lookup"><span data-stu-id="9eaac-182">Like GIF files, PNG files are compressed with no loss of information.</span></span> <span data-ttu-id="9eaac-183">PNG-файлы могут содержать цвета 8, 24 или 48 бит на пиксел и оттенках серого с 1, 2, 4, 8 или 16 бит на пиксель.</span><span class="sxs-lookup"><span data-stu-id="9eaac-183">PNG files can store colors with 8, 24, or 48 bits per pixel and grayscales with 1, 2, 4, 8, or 16 bits per pixel.</span></span> <span data-ttu-id="9eaac-184">Напротив GIF-файлы можно использовать только 1, 2, 4 или 8 бит на пиксель.</span><span class="sxs-lookup"><span data-stu-id="9eaac-184">In contrast, GIF files can use only 1, 2, 4, or 8 bits per pixel.</span></span> <span data-ttu-id="9eaac-185">PNG-файл также можно хранить альфа-значение для каждого пикселя, который определяет степень, в которую цвета точки смешивается с цветом фона.</span><span class="sxs-lookup"><span data-stu-id="9eaac-185">A PNG file can also store an alpha value for each pixel, which specifies the degree to which the color of that pixel is blended with the background color.</span></span>  
  
 <span data-ttu-id="9eaac-186">Улучшает PNG предсказуемый последовательно выводит изображение GIF (то есть для отображения более и более подробное приближение изображения, так как он поступает через сетевое подключение).</span><span class="sxs-lookup"><span data-stu-id="9eaac-186">PNG improves on GIF in its ability to progressively display an image (that is, to display better and better approximations of the image as it arrives over a network connection).</span></span> <span data-ttu-id="9eaac-187">PNG-файлы могут содержать данные о гамма коррекции и о цветовой коррекции, что позволяет точно изображение на различных устройствах отображения.</span><span class="sxs-lookup"><span data-stu-id="9eaac-187">PNG files can contain gamma correction and color correction information so that the images can be accurately rendered on a variety of display devices.</span></span>  
  
### <a name="tag-image-file-format-tiff"></a><span data-ttu-id="9eaac-188">Формат файла тег изображения (TIFF)</span><span class="sxs-lookup"><span data-stu-id="9eaac-188">Tag Image File Format (TIFF)</span></span>  
 <span data-ttu-id="9eaac-189">TIFF — это гибкий и расширяемый формат, поддерживаемый для разнообразных платформ и приложений для обработки изображений.</span><span class="sxs-lookup"><span data-stu-id="9eaac-189">TIFF is a flexible and extendable format that is supported by a wide variety of platforms and image-processing applications.</span></span> <span data-ttu-id="9eaac-190">TIFF-файлы можно хранить изображения с произвольное число бит на пиксель и использовать различные алгоритмы сжатия.</span><span class="sxs-lookup"><span data-stu-id="9eaac-190">TIFF files can store images with an arbitrary number of bits per pixel and can employ a variety of compression algorithms.</span></span> <span data-ttu-id="9eaac-191">Несколько образов могут храниться в одной, нескольких страниц TIFF-файл.</span><span class="sxs-lookup"><span data-stu-id="9eaac-191">Several images can be stored in a single, multiple-page TIFF file.</span></span> <span data-ttu-id="9eaac-192">Сведения, связанные с изображением (модель сканера, главного компьютера, тип сжатия, ориентация, примеры на пиксел и т. д.) можно хранить в файле и упорядочены с помощью тегов.</span><span class="sxs-lookup"><span data-stu-id="9eaac-192">Information related to the image (scanner make, host computer, type of compression, orientation, samples per pixel, and so on) can be stored in the file and arranged through the use of tags.</span></span> <span data-ttu-id="9eaac-193">Формат TIFF может быть расширена, при необходимости путем добавления новых тегов и утверждения.</span><span class="sxs-lookup"><span data-stu-id="9eaac-193">The TIFF format can be extended as needed by the approval and addition of new tags.</span></span>
