# <a name="coordinate-systems-and-transformations"></a>Системы координат и преобразования
GDI + предоставляет мировое преобразование и преобразование страницы таким образом, чтобы их можно преобразовать (поворот, масштабирование, преобразования и так далее) элементов, будет нарисована. Два преобразования также позволяют работать в различных систем координат.
