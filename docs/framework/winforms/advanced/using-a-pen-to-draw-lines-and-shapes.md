# <a name="using-a-pen-to-draw-lines-and-shapes"></a>Рисование линий и фигур с помощью пера
Использование GDI + `Pen` объектов, чтобы нарисовать сегменты линии, кривые и контуры фигуры. В этом разделе *строки* ссылается на любой из них, если не указано для обозначения только сегмент линии. Задание свойств пера для управления цвет, ширину, выравнивание и стиль линий, нарисованных при помощи этого пера.
