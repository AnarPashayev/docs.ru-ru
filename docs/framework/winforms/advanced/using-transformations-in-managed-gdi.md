# <a name="using-transformations-in-managed-gdi"></a>Использование преобразований в управляемом GDI+
Аффинные преобразования включают поворот, масштабирование, отражение, наклон и преобразования. В GDI + <xref:System.Drawing.Drawing2D.Matrix> класс предоставляет основу для осуществления аффинных преобразований векторные рисунки, изображения и текста.
