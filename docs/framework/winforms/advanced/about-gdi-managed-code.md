# <a name="about-gdi-managed-code"></a>Управляемый код GDI+

GDI+ — это часть операционной системы Windows, которая предоставляет двухмерные векторную графику, изображения и типографии. GDI+ расширяет возможности GDI (интерфейс графических устройств, включенные в более ранние версии Windows), добавляя новые функции и оптимизируя существующие.

Интерфейс управляемого класса GDI+ (набор оболочек) является частью .NET Framework, среды для сборки, развертывания и выполнения веб-служб XML и других приложений.

В этом разделе содержатся сведения об API GDI+ для программистов, использующих управляемый код.
