# <a name="integrating-user-help-in-windows-forms"></a>Интеграция справки пользователя в формы Windows Forms
Важным, но часто с незнакомым аспектом создания приложений на основе Windows является справочная система, так как в этом случае пользователи могут не обращаться за помощью к времени путаницы. Windows Forms поддерживают два разных типа справки, каждая из которых предоставляется [компонентом HelpProvider](../controls/helpprovider-component-windows-forms.md). Первый включает в себя указание пользователя на файл справки HTML или HTML Help 1. формат *x* или более. Во втором может отображаться краткий обзор справки по типам отдельных элементов управления. Это особенно полезно в диалоговых окнах. Оба типа справки можно использовать в одной и той же форме.  
  
 Кроме того, [компонент ToolTip](../controls/tooltip-component-windows-forms.md) можно использовать для предоставления отдельной справки по элементам управления Windows Forms.
