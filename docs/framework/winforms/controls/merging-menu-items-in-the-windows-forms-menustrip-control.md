## <a name="general-concepts"></a><span data-ttu-id="30706-105">Общие понятия</span><span class="sxs-lookup"><span data-stu-id="30706-105">General Concepts</span></span>

- <span data-ttu-id="30706-107">Целевым элементом является <xref:System.Windows.Forms.MenuStrip> элемент управления основной родительской формы или родительского интерфейса MDI, в который объединяются пункты меню.</span><span class="sxs-lookup"><span data-stu-id="30706-107">The target is the <xref:System.Windows.Forms.MenuStrip> control on the main or MDI parent form into which you are merging menu items.</span></span>

 <span data-ttu-id="30706-109">Свойство <xref:System.Windows.Forms.MenuStrip.MdiWindowListItem%2A> определяет пункт меню, раскрывающийся список которого будет заполнен заголовками дочерних MDI-форм родительской формы MDI.</span><span class="sxs-lookup"><span data-stu-id="30706-109">The <xref:System.Windows.Forms.MenuStrip.MdiWindowListItem%2A> property identifies the menu item whose drop-down list you will populate with the titles of the current MDI parent form's MDI children.</span></span> <span data-ttu-id="30706-110">Например, вы обычно перечислите дочерние элементы MDI, которые в данный момент открыты в меню **окно** .</span><span class="sxs-lookup"><span data-stu-id="30706-110">For example, you typically list MDI children that are currently open on the **Window** menu.</span></span>  
  
 <span data-ttu-id="30706-111">Свойство <xref:System.Windows.Forms.ToolStripMenuItem.IsMdiWindowListEntry%2A> определяет, какие пункты меню берутся из <xref:System.Windows.Forms.MenuStrip> в дочерней форме MDI.</span><span class="sxs-lookup"><span data-stu-id="30706-111">The <xref:System.Windows.Forms.ToolStripMenuItem.IsMdiWindowListEntry%2A> property identifies which menu items come from a <xref:System.Windows.Forms.MenuStrip> on an MDI child form.</span></span>  
  
 <span data-ttu-id="30706-112">Элементы меню можно объединять вручную или автоматически.</span><span class="sxs-lookup"><span data-stu-id="30706-112">You can merge menu items manually or automatically.</span></span> <span data-ttu-id="30706-113">Элементы меню объединяются одинаково для обоих методов, но слияние активируется по-разному, как описано в разделах "слияние вручную" и "Автоматическое слияние" Далее в этом разделе.</span><span class="sxs-lookup"><span data-stu-id="30706-113">The menu items merge in the same way for both methods, but the merge is activated differently, as discussed in the "Manual Merging" and "Automatic Merging" sections later in this topic.</span></span> <span data-ttu-id="30706-114">Как в ручном, так и в автоматическом слиянии каждое действие слияния влияет на следующее действие слияния.</span><span class="sxs-lookup"><span data-stu-id="30706-114">In both manual and automatic merging, each merge action affects the next merge action.</span></span>
