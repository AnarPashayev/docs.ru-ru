# <a name="how-to-set-the-image-displayed-by-a-windows-forms-control-using-the-designer"></a>Практическое руководство. Задание изображения, отображаемого Windows Forms элементом управления, с помощью конструктора

Несколько элементов управления Windows Forms могут отображать изображения. Изображение может быть значком, который обозначает назначение элемента управления, например значок диска на кнопке, указывающий на команду **сохранить** . Или же значок может быть фоновым изображением, чтобы придать элементу управления нужный внешний вид.

## <a name="to-set-the-image-displayed-by-a-control"></a>Задание изображения, отображаемого элементом управления

1. В окне **Свойства** выберите свойство **Image** или **BackgroundImage** элемента управления, а затем нажмите кнопку с многоточием (![кнопка с многоточием в Visual Studio](./media/visual-studio-ellipsis-button.png)), чтобы отобразить выбранный **ресурс** . диалоговое окно.

2. Выберите изображение, которое требуется отобразить.
