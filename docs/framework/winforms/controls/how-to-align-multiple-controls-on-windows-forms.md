# <a name="how-to-align-multiple-controls-on-windows-forms"></a>Практическое руководство. Выравнивание нескольких элементов управления в формах Windows Forms

Чтобы стандартизировать макет пользовательского интерфейса приложения Windows, можно разместить группы элементов управления с помощью одной команды.

## <a name="to-align-multiple-controls-on-a-form"></a>Выровняйте несколько элементов управления в форме

1. В Visual Studio откройте форму, содержащую элементы управления, которые требуется разместить в **конструктор Windows Forms**.

2. Выберите элементы управления, которые необходимо выстроить. Первый выбранный элемент управления является основным элементом управления, к которому будут согласовываться другие.

3. В меню **Формат** выберите выровняйте, а затем выберите один из семи доступных вариантов.
