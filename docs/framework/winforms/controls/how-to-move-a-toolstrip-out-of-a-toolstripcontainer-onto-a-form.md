# <a name="how-to-move-a-toolstrip-out-of-a-toolstripcontainer-onto-a-form"></a>Практическое руководство. Перемещение элемента ToolStrip из контейнера ToolStripContainer в форму
Используйте следующую процедуру, чтобы переместить элемент <xref:System.Windows.Forms.ToolStrip> <xref:System.Windows.Forms.ToolStripContainer> из элемента на форму.

## <a name="to-move-a-toolstrip-out-of-a-toolstripcontainer-onto-a-form"></a>Перемещение элемента ToolStrip из ToolStripContainer на форму

1. Выберите <xref:System.Windows.Forms.ToolStrip>.

2. Вырежьте нажатием клавиш CTRL + X или щелкните правой кнопкой мыши <xref:System.Windows.Forms.ToolStrip> и выберите **Вырезать** в контекстном меню. <xref:System.Windows.Forms.ToolStrip>

3. Выберите форму.

4. Вставьте, нажав сочетание клавиш CTRL + V или кнопку **Вставить** в меню **Правка.** <xref:System.Windows.Forms.ToolStrip>

5. Задайте для <xref:System.Windows.Forms.ToolStrip>свойствазначение Top. <xref:System.Windows.Forms.ToolStrip.Dock%2A>
