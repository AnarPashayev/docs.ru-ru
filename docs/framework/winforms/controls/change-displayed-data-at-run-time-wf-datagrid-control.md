    >  <span data-ttu-id="d6285-110">Для указания первой таблицы <xref:System.Data.DataSet> или первую строку таблицы, укажите 0.</span><span class="sxs-lookup"><span data-stu-id="d6285-110">To specify the first table of the <xref:System.Data.DataSet> or the first row of the table, use 0.</span></span>

### <a name="to-change-the-datasource-programmatically"></a><span data-ttu-id="d6285-116">Чтобы изменить источник данных программными средствами</span><span class="sxs-lookup"><span data-stu-id="d6285-116">To change the DataSource programmatically</span></span>  
  
1. <span data-ttu-id="d6285-117">Задайте <xref:System.Windows.Forms.DataGrid.SetDataBinding%2A> метод имя источника данных и вы хотите выполнить привязку к таблице.</span><span class="sxs-lookup"><span data-stu-id="d6285-117">Set the <xref:System.Windows.Forms.DataGrid.SetDataBinding%2A> method to the name of the data source and table you want to bind to.</span></span>  
  
     <span data-ttu-id="d6285-118">Приведенный ниже показано, как изменить дату источника с помощью <xref:System.Windows.Forms.DataGrid.SetDataBinding%2A> метода к элементу управления данных ADO.NET (adoPubsAuthors), который подключен к таблицы Authors базы данных Pubs.</span><span class="sxs-lookup"><span data-stu-id="d6285-118">The following example shows how to change the date source using the <xref:System.Windows.Forms.DataGrid.SetDataBinding%2A> method to an ADO.NET data control (adoPubsAuthors) that is connected to the Authors table in the Pubs database.</span></span>
