### <a name="to-change-the-datasource-programmatically"></a><span data-ttu-id="3db21-116">Чтобы изменить источник данных программными средствами</span><span class="sxs-lookup"><span data-stu-id="3db21-116">To change the DataSource programmatically</span></span>  
  
1. <span data-ttu-id="3db21-117">Задайте <xref:System.Windows.Forms.DataGrid.SetDataBinding%2A> метод имя источника данных и вы хотите выполнить привязку к таблице.</span><span class="sxs-lookup"><span data-stu-id="3db21-117">Set the <xref:System.Windows.Forms.DataGrid.SetDataBinding%2A> method to the name of the data source and table you want to bind to.</span></span>  
  
     <span data-ttu-id="3db21-118">Приведенный ниже показано, как изменить дату источника с помощью <xref:System.Windows.Forms.DataGrid.SetDataBinding%2A> метод [!INCLUDE[vstecado](../../../../includes/vstecado-md.md)] управления данных (adoPubsAuthors), который подключен к таблицы Authors базы данных Pubs.</span><span class="sxs-lookup"><span data-stu-id="3db21-118">The following example shows how to change the date source using the <xref:System.Windows.Forms.DataGrid.SetDataBinding%2A> method to an [!INCLUDE[vstecado](../../../../includes/vstecado-md.md)] data control (adoPubsAuthors) that is connected to the Authors table in the Pubs database.</span></span>
