### <a name="to-add-a-table-to-the-datagrid-control-in-the-designer"></a><span data-ttu-id="b5b6f-113">Добавление таблицы в элемент управления DataGrid в конструкторе</span><span class="sxs-lookup"><span data-stu-id="b5b6f-113">To add a table to the DataGrid control in the designer</span></span>

1. <span data-ttu-id="b5b6f-114">Чтобы отобразить данные в таблице, необходимо сначала привязать элемент управления <xref:System.Windows.Forms.DataGrid> к набору данных.</span><span class="sxs-lookup"><span data-stu-id="b5b6f-114">In order to display data in the table, you must first bind the <xref:System.Windows.Forms.DataGrid> control to a dataset.</span></span> <span data-ttu-id="b5b6f-115">Дополнительные сведения см. в разделе [руководство. привязка Windows Forms элемента управления DataGrid к источнику данных с помощью конструктора](bind-wf-datagrid-control-to-a-data-source-using-the-designer.md).</span><span class="sxs-lookup"><span data-stu-id="b5b6f-115">For more information, see [How to: Bind the Windows Forms DataGrid Control to a Data Source Using the Designer](bind-wf-datagrid-control-to-a-data-source-using-the-designer.md).</span></span>

3. <span data-ttu-id="b5b6f-117">В редакторе коллекции нажмите кнопку **Добавить** , чтобы вставить стиль таблицы.</span><span class="sxs-lookup"><span data-stu-id="b5b6f-117">In the collection editor, click **Add** to insert a table style.</span></span>

5. <span data-ttu-id="b5b6f-120">В поле **члены** редактора коллекции выберите стиль таблицы.</span><span class="sxs-lookup"><span data-stu-id="b5b6f-120">In the **Members** box of the collection editor, click the table style.</span></span>

6. <span data-ttu-id="b5b6f-121">В диалоговом окне **Свойства** редактора коллекции выберите <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> значение для таблицы, которую требуется отобразить.</span><span class="sxs-lookup"><span data-stu-id="b5b6f-121">In the **Properties** box of the collection editor, select the <xref:System.Windows.Forms.DataGridTableStyle.MappingName%2A> value for the table you want to display.</span></span>
