# <a name="linklabel-control-windows-forms"></a>Элемент управления LinkLabel (Windows Forms)
Элемент управления `LinkLabel` Windows Forms позволяет добавлять ссылки в веб-стиле в приложения Windows Forms. Вы можете `LinkLabel` использовать элемент управления для <xref:System.Windows.Forms.Label> всего, что вы можете использовать элемент управления для; вы также можете установить часть текста в виде ссылки на объект или веб-страницу.
