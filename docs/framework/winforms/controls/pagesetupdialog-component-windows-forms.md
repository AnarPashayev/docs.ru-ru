# <a name="pagesetupdialog-component-windows-forms"></a>Компонент PageSetupDialog (Windows Forms)
Компонент Windows <xref:System.Windows.Forms.PageSetupDialog> Forms — это предварительно настроенное диалоговое окно, используемое для настройки деталей страницы для печати в приложениях на базе Windows. Используйте его в приложении на базе Windows в качестве простого решения для пользователей, чтобы установить настройки страницы вместо настройки собственного диалогового окна. Вы можете позволить пользователям устанавливать корректировки границ и границ, заголовки и колонтитулы, а также портрет против ландшафтной ориентации. Использование стандартных диалоговых окон Windows помогает создавать приложения, основные функциональные возможности которых хорошо знакомы пользователям.
