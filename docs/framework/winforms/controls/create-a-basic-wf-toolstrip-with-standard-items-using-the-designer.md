# <a name="how-to-create-a-basic-windows-forms-toolstrip-with-standard-items-using-the-designer"></a>Практическое руководство. Создание базового Windows Forms ToolStrip со стандартными элементами с помощью конструктора
В следующей процедуре показано, как создать <xref:System.Windows.Forms.ToolStrip> и добавить семь <xref:System.Windows.Forms.ToolStripButton> элементов управления, представляющих типичные задачи.

## <a name="to-add-standard-items-in-the-designer"></a>Добавление стандартных элементов в конструктор

1. <xref:System.Windows.Forms.ToolStrip> Создайте элемент управления.

2. В правом верхнем углу <xref:System.Windows.Forms.ToolStrip>щелкните стрелку смарт-задачи, чтобы отобразить панель **Задачи ToolStrip** .

3. В области **Задачи ToolStrip** выберите пункт **Вставить стандартные элементы**.
