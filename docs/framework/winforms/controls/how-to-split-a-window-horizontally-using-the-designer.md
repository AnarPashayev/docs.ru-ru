# <a name="how-to-split-a-window-horizontally-using-the-designer"></a>Практическое руководство. Разделение окна по горизонтали с помощью конструктора

Этот пример делает разделитель, разделяющий <xref:System.Windows.Forms.SplitContainer> элемент управления по горизонтали.

> [!NOTE]
> <xref:System.Windows.Forms.SplitContainer.Orientation%2A> Свойство<xref:System.Windows.Forms.SplitContainer> элемента управления определяет направление разделителя, а не сам элемент управления.

## <a name="to-split-a-window-horizontally"></a>Разбиение окна по горизонтали

В окне **Свойства** задайте <xref:System.Windows.Forms.SplitContainer.Orientation%2A> <xref:System.Windows.Forms.SplitContainer> для<xref:System.Windows.Forms.Orientation.Horizontal>свойства элемента управления значение.
