1. <span data-ttu-id="73296-105">Щелкните элемент управления, размер которого нужно изменить, и перетащите один из восьми маркеров изменения размера.</span><span class="sxs-lookup"><span data-stu-id="73296-105">Click the control to be resized and drag one of the eight sizing handles.</span></span>  
  
    > [!NOTE]
    > <span data-ttu-id="73296-106">Выберите элемент управления и нажмите клавиши со СТРЕЛКАми, удерживая нажатой клавишу SHIFT, чтобы изменить размер элемента управления на один пиксель за раз.</span><span class="sxs-lookup"><span data-stu-id="73296-106">Select the control and press the ARROW keys while holding down the SHIFT key to resize the control one pixel at a time.</span></span> <span data-ttu-id="73296-107">Нажмите клавиши со стрелками вниз или вправо, удерживая нажатыми клавиши SHIFT и CTRL, чтобы изменить размер элемента управления с большим шагом.</span><span class="sxs-lookup"><span data-stu-id="73296-107">Press the DOWN or RIGHT arrow keys while holding down the SHIFT and CTRL keys to resize the control in large increments.</span></span>  
  
### <a name="to-resize-multiple-controls-on-a-form"></a><span data-ttu-id="73296-108">Изменение размера нескольких элементов управления в форме</span><span class="sxs-lookup"><span data-stu-id="73296-108">To resize multiple controls on a form</span></span>  
  
1. <span data-ttu-id="73296-109">Удерживайте нажатой клавишу CTRL или SHIFT и выберите элементы управления, размер которых необходимо изменить.</span><span class="sxs-lookup"><span data-stu-id="73296-109">Hold down the CTRL or SHIFT key and select the controls you want to resize.</span></span> <span data-ttu-id="73296-110">Размер первого выбранного элемента управления используется для других элементов управления.</span><span class="sxs-lookup"><span data-stu-id="73296-110">The size of the first control you select is used for the other controls.</span></span>  
  
2. <span data-ttu-id="73296-111">В меню **Формат** выберите пункт **сделать тот же размер**и выберите один из четырех параметров.</span><span class="sxs-lookup"><span data-stu-id="73296-111">On the **Format** menu, choose **Make Same Size**, and select one of the four options.</span></span> <span data-ttu-id="73296-112">Первые три команды изменяют размеры элементов управления в соответствии с первым выбранным элементом управления.</span><span class="sxs-lookup"><span data-stu-id="73296-112">The first three commands change the dimensions of the controls to match the first-selected control.</span></span>
