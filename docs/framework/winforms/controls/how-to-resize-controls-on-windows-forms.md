1. <span data-ttu-id="85383-109">В Visual Studio удерживайте нажатой клавишу **CTRL** или **SHIFT** и выберите элементы управления, размер которых необходимо изменить.</span><span class="sxs-lookup"><span data-stu-id="85383-109">In Visual Studio, hold down the **Ctrl** or **Shift** key and select the controls you want to resize.</span></span> <span data-ttu-id="85383-110">Размер первого выбранного элемента управления используется для других элементов управления.</span><span class="sxs-lookup"><span data-stu-id="85383-110">The size of the first control you select is used for the other controls.</span></span>
