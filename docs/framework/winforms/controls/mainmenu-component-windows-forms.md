# <a name="mainmenu-component-windows-forms"></a>Компонент MainMenu (Windows Forms)
> [!NOTE]
> Хотя `MenuStrip` и `ContextMenuStrip` заменить и добавить функции в `MainMenu` и <xref:System.Windows.Forms.ContextMenu> элементы управления предыдущих версий, `MainMenu` и <xref:System.Windows.Forms.ContextMenu> сохраняются для обратной совместимости и использования в будущем при выборе.  
  
 Компонент `MainMenu` Windows Forms отображает меню во время выполнения.
