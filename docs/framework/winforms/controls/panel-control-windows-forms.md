# <a name="panel-control-windows-forms"></a>Элемент управления Panel (Windows Forms)
`Panel`Элементы управления Windows Forms используются для предоставления идентифицируемого группирования для других элементов управления. Как правило, панели используются для разделения формы на функции. `Panel`Элемент управления аналогичен <xref:System.Windows.Forms.GroupBox> элементу управления, но только `Panel` элемент управления может иметь полосы прокрутки, а только <xref:System.Windows.Forms.GroupBox> элемент управления отображает заголовок.
