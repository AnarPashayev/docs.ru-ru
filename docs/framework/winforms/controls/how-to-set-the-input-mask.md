# <a name="how-to-set-the-input-mask"></a>Практическое руководство. Определение маски ввода
Элемент управления "текстовое поле с маской" — Это Улучшенный элемент управления "текстовое поле", поддерживающий декларативный синтаксис для принятия или отклонения вводимых пользователем данных. Установив свойство Mask, можно указать допустимый пользовательский ввод без написания какой бы то ни было пользовательской логики проверки в приложении. Дополнительные сведения см. в разделе <xref:System.Windows.Forms.MaskedTextBox> "Примечания" класса.  
  
## <a name="setting-the-mask-property-manually"></a>Задание свойства маски вручную  
 Если вы знакомы с символами, которые поддерживает свойство Mask, можно ввести его вручную. Общие сведения о символах, поддерживаемых свойством Mask, см. в разделе <xref:System.Windows.Forms.MaskedTextBox.Mask%2A> "Примечания" Свойства.  
  
### <a name="to-set-the-mask-property-manually"></a>Задание свойства маски вручную  
  
1. В режиме **конструктора** выберите <xref:System.Windows.Forms.MaskedTextBox>.  
  
2. В <xref:System.Windows.Forms.MaskedTextBox.Mask%2A> окне **Свойства** выберите свойство.  
  
3. Введите нужную маску. Например, введите `###`.  
  
## <a name="using-the-input-mask-dialog-box"></a>Использование диалогового окна «маска ввода»  
 Диалоговое окно «Маска ввода» предоставляет некоторые предопределенные маски ввода. Можно также изменить стандартные маски или ввести собственную маску вручную.  
  
### <a name="to-open-the-input-mask-dialog-box"></a>Открытие диалогового окна «маска ввода»  
  
1. В режиме **конструктора** выберите <xref:System.Windows.Forms.MaskedTextBox>.  
  
    1. Щелкните смарт-тег, чтобы открыть панель **задачи MaskedTextBox** .  
  
    2. Щелкните **задать маску**.  
  
     \- или -  
  
    1. В окне **Свойства** выберите <xref:System.Windows.Forms.MaskedTextBox.Mask%2A> свойство.  
  
    2. Нажмите кнопку с многоточием в столбце значение свойства.  
  
     Откроется диалоговое окно **Маска ввода** .  
  
### <a name="to-use-the-input-mask-dialog-box"></a>Использование диалогового окна «маска ввода»  
  
1. Используемых Выберите одну из стандартных масок в списке.  
  
2. Используемых Измените стандартную маску в поле **Маска** .  
  
3. Используемых Введите новую маску в поле **Маска** . То есть вам не нужно использовать одну из стандартных масок.  
  
    > [!NOTE]
    > В поле Предварительный просмотр отображаются символы, которые видит пользователь в <xref:System.Windows.Forms.MaskedTextBox>. Эти символы являются руководством, помогающим пользователю правильно ввести данные.  
  
4. Установите или снимите флажок **использовать валидатингтипе** . Флажок **использовать валидатингтипе** указывает, используется ли тип данных для проверки данных, вводимых пользователем. Дополнительные сведения см. в описании свойства <xref:System.Windows.Forms.MaskedTextBox.ValidatingType%2A>.  
  
5. Нажмите кнопку **ОК**.  
  
     Маска указывается в свойстве **Mask** в окне **свойства** .
