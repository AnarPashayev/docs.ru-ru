# <a name="how-to-add-and-remove-tabs-with-the-windows-forms-tabcontrol-using-the-designer"></a>Практическое руководство. Добавление и удаление вкладок с использованием элемента управления TabControl в формах Windows Forms с помощью конструктора
При помещении <xref:System.Windows.Forms.TabControl> элемента управления в форму он по умолчанию содержит две вкладки. Можно добавлять или удалять вкладки с помощью конструктора.

 Для следующей процедуры требуется проект **приложения Windows** с формой, содержащей <xref:System.Windows.Forms.TabControl> элемент управления. Сведения о настройке такого проекта см. в статьях [как создать проект приложения Windows Forms](/visualstudio/ide/step-1-create-a-windows-forms-application-project) и [как добавить элементы управления в Windows Forms](how-to-add-controls-to-windows-forms.md).

## <a name="to-add-or-remove-a-tab-using-the-designer"></a>Добавление или удаление вкладки с помощью конструктора

- На смарт-теге элемента управления щелкните **Добавить** вкладку или **Удалить вкладку** .

     -или-

     В окне **Свойства** нажмите кнопку **с многоточием** ( ![ кнопку с многоточием (...) в окно свойств Visual Studio. ](./media/visual-studio-ellipsis-button.png) ) рядом со <xref:System.Windows.Forms.TabControl.TabPages%2A> свойством, чтобы открыть **Редактор коллекции TabPage**. Нажмите кнопку **Добавить** или **Удалить** .
