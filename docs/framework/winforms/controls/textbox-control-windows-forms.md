# <a name="textbox-control-windows-forms"></a>Элемент управления TextBox (Windows Forms)
Windows Forms текстовые поля используются для получения входных данных от пользователя или для вывода текста. `TextBox`Элемент управления обычно используется для редактируемого текста, хотя его также можно сделать доступным только для чтения. Текстовые поля могут отображать несколько строк, переносить текст в размер элемента управления и добавлять базовое форматирование. `TextBox`Элемент управления допускает один формат текста, отображаемого или вводимых в элементе управления.
