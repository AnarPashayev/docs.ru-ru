# <a name="label-control-windows-forms"></a>Элемент управления Label (Windows Forms)
> [!IMPORTANT]
> Элемент управления <xref:System.Windows.Forms.ToolStripLabel> заменяет и расширяет функциональные возможности элемента управления <xref:System.Windows.Forms.Label>. Можно использовать <xref:System.Windows.Forms.ToolStripLabel> с другими новыми элементами управления, такими как <xref:System.Windows.Forms.ToolStripDropDown>. Однако элемент управления <xref:System.Windows.Forms.Label> сохраняется как для обратной совместимости, так и для будущего использования, если вы решили.  
  
 Windows Forms элементы управления <xref:System.Windows.Forms.Label> используются для вывода текста или изображений, которые не могут быть изменены пользователем. Они используются для обнаружения объектов в форме — для предоставления описания того, что может делать определенный элемент управления, например, или для вывода сведений в ответ на событие или процесс во время выполнения в приложении. Поскольку элемент управления <xref:System.Windows.Forms.Label> не может получать фокус, он также может использоваться для создания ключей доступа для других элементов управления.
