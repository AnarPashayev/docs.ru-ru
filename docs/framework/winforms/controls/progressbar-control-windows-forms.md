# <a name="progressbar-control-windows-forms"></a>Элемент управления ProgressBar (Windows Forms)
> [!IMPORTANT]
> Элемент управления <xref:System.Windows.Forms.ToolStripProgressBar> заменяет элемент управления <xref:System.Windows.Forms.ProgressBar> и расширяет его функциональные возможности; однако при необходимости элемент управления <xref:System.Windows.Forms.ProgressBar> можно сохранить для обратной совместимости и использования в будущем.  
  
 Элемент управления Windows Forms <xref:System.Windows.Forms.ProgressBar> показывает ход выполнения действия, отображая соответствующее количество прямоугольников, расположенных на горизонтальной полосе. После завершения действия Панель заполняется. Индикаторы выполнения обычно используются, чтобы дать пользователю указание о том, как долго ожидать завершения завершенного действия — например, при загрузке большого файла.
