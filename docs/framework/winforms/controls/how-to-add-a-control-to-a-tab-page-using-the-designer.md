# <a name="how-to-add-a-control-to-a-tab-page-using-the-designer"></a>Практическое руководство. Добавление элемента управления на вкладку с помощью конструктора
Использование Windows Forms <xref:System.Windows.Forms.TabControl> заключается в том, чтобы отображать другие элементы управления организованным способом. Эти инструкции можно использовать для вывода изображения в основной части страницы вкладки. Сведения о добавлении значка в часть метки на странице вкладки см. в разделе [как Измените внешний вид Windows Forms TabControl](how-to-change-the-appearance-of-the-windows-forms-tabcontrol.md).

 Для следующей процедуры требуется проект **приложения Windows** с формой, содержащей <xref:System.Windows.Forms.TabControl> элемент управления. Сведения о настройке такого проекта см. в разделе [как Создайте проект](/visualstudio/ide/step-1-create-a-windows-forms-application-project) приложения Windows Forms и [выполните следующие действия. Добавьте элементы управления в](how-to-add-controls-to-windows-forms.md)Windows Forms.

## <a name="to-add-a-control-using-the-designer"></a>Добавление элемента управления с помощью конструктора

1. Щелкните соответствующую страницу вкладки, чтобы она появилась сверху.

2. Нарисуйте элемент управления на странице вкладки.
