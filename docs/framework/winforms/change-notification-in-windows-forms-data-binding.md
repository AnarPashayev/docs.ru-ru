<span data-ttu-id="7cbab-103">Одним из наиболее важных концепций привязки данных Windows Forms является *уведомление об изменениях*.</span><span class="sxs-lookup"><span data-stu-id="7cbab-103">One of the most important concepts of Windows Forms data binding is *change notification*.</span></span> <span data-ttu-id="7cbab-104">Чтобы гарантировать, что источник данных и привязанные элементы управления всегда будут иметь самые последние данные, необходимо добавить уведомление об изменении для привязки данных.</span><span class="sxs-lookup"><span data-stu-id="7cbab-104">To ensure that your data source and bound controls always have the most recent data, you must add change notification for data binding.</span></span> <span data-ttu-id="7cbab-105">В частности, необходимо убедиться, что привязанные элементы управления получают уведомления об изменениях, внесенных в источник данных, а источник данных уведомляет об изменениях, внесенных в связанные свойства элемента управления.</span><span class="sxs-lookup"><span data-stu-id="7cbab-105">Specifically, you want to ensure that bound controls are notified of changes that were made to their data source, and the data source is notified of changes that were made to the bound properties of a control.</span></span>  
  
 <span data-ttu-id="7cbab-106">Существуют различные виды уведомлений об изменениях в зависимости от типа привязки данных.</span><span class="sxs-lookup"><span data-stu-id="7cbab-106">There are different kinds of change notification, depending on the kind of data binding:</span></span>  
  
- <span data-ttu-id="7cbab-107">Простая привязка, в которой одно свойство элемента управления привязано к одному экземпляру объекта.</span><span class="sxs-lookup"><span data-stu-id="7cbab-107">Simple binding, in which a single control property is bound to a single instance of an object.</span></span>  
  
- <span data-ttu-id="7cbab-108">Привязка на основе списка, которая может включать одно свойство элемента управления, привязанное к свойству элемента в списке, или свойство элемента управления, привязанное к списку объектов.</span><span class="sxs-lookup"><span data-stu-id="7cbab-108">List-based binding, which can include a single control property bound to the property of an item in a list or a control property bound to a list of objects.</span></span>  
  
 <span data-ttu-id="7cbab-109">Кроме того, при создании Windows Forms элементов управления, которые будут использоваться для привязки данных, необходимо применить шаблон " *PropertyName*Changed" к элементам управления, чтобы изменения привязанного свойства элемента управления распространялись на источник данных.</span><span class="sxs-lookup"><span data-stu-id="7cbab-109">Additionally, if you are creating Windows Forms controls that you want to use for data binding, you must apply the *PropertyName*Changed pattern to the controls, so that changes to the bound property of a control are propagated to the data source.</span></span>
