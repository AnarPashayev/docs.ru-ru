# <a name="keyboard-input-in-a-windows-forms-application"></a>Ввод с клавиатуры в приложении Windows Forms
Windows Forms включает стандартные события клавиатуры, которые позволяют реагировать на определенные нажатия клавиш, а также предоставляет способы перехвата, изменения и использования нажатий клавиш на уровне приложения, формы и элемента управления.

 [Использование событий клавиатуры](using-keyboard-events.md)  
 Содержит сведения о типах событий клавиатуры и сведения, получаемые обработчиками событий клавиатуры.  
  
 [Практическое руководство. Изменение ввода с клавиатуры в стандартном элементе управления](how-to-modify-keyboard-input-to-a-standard-control.md)  
 Представляет пример кода, демонстрирующий, как изменить значения ключа до того, как они достигают элемента управления.  
  
 [Практическое руководство. Определение нажатой управляющей клавиши](how-to-determine-which-modifier-key-was-pressed.md)  
 Демонстрируется, как определить, была ли нажата клавиша SHIFT, ALT или CTRL в дополнение к другому ключу.  
  
 [Практическое руководство. Обработка ввода с клавиатуры на уровне формы](how-to-handle-keyboard-input-at-the-form-level.md)  
 Представляет пример кода, демонстрирующий перехват ключей перед достижением элемента управления.
