 <span data-ttu-id="c0581-191">Следующая команда создает библиотеку типов с таким же именем, что и у сборки, найденной в `myTest.dll`.</span><span class="sxs-lookup"><span data-stu-id="c0581-191">The following command generates a type library with the same name as the assembly found in `myTest.dll`.</span></span>
