## <a name="remarks"></a><span data-ttu-id="1dc01-178">Заметки</span><span class="sxs-lookup"><span data-stu-id="1dc01-178">Remarks</span></span>  
 <span data-ttu-id="1dc01-179">Элемент `<Assembly>` определяет политику среды выполнения для всех типов в сборке.</span><span class="sxs-lookup"><span data-stu-id="1dc01-179">The `<Assembly>` element defines runtime policy for all the types in an assembly.</span></span> <span data-ttu-id="1dc01-180">Он отличается от элемента [\<Library>](library-element-net-native.md), который указывает библиотеку, но зависит от его дочерних элементов для определения политики отражения среды выполнения.</span><span class="sxs-lookup"><span data-stu-id="1dc01-180">It differs from the [\<Library>](library-element-net-native.md) element, which specifies a library but depends on its child elements to define runtime reflection policy.</span></span> <span data-ttu-id="1dc01-181">Элемент `<Assembly>` применяется ко всем типам в сборке, если они не переопределены дочерним элементом.</span><span class="sxs-lookup"><span data-stu-id="1dc01-181">The `<Assembly>` element applies to all the types in an assembly unless they are overridden by a child element.</span></span>  
  
 <span data-ttu-id="1dc01-182">В следующем примере показано, как применять политику среды выполнения для всех типов сборки в пакете приложения путем назначения атрибуту `Name` значения "\*Application\*".</span><span class="sxs-lookup"><span data-stu-id="1dc01-182">The following example shows how you can apply runtime policy to all the types in assemblies within your app package by assigning the `Name` attribute a value of "\*Application\*".</span></span> <span data-ttu-id="1dc01-183">Элемент `<Assembly>` должен быть дочерним для элемента [\<Application>](application-element-net-native.md).</span><span class="sxs-lookup"><span data-stu-id="1dc01-183">The `<Assembly>` element must be a child of the [\<Application>](application-element-net-native.md) element.</span></span>  
  
```xml

  <Application>   
     <Assembly Name="*Application*" Dynamic="Required All" />   
  </Application>   
</Directives>  
```  
  
 <span data-ttu-id="1dc01-184">Атрибуты `Activate`, `Browse`, `Dynamic` и `Serialize` необязательны.</span><span class="sxs-lookup"><span data-stu-id="1dc01-184">The `Activate`, `Browse`, `Dynamic`, and `Serialize` attributes are all optional.</span></span> <span data-ttu-id="1dc01-185">Тем не менее, элемент `<Assembly>` должен содержать хотя бы один из этих атрибутов.</span><span class="sxs-lookup"><span data-stu-id="1dc01-185">However, the `<Assembly>` element must contain at least one of these attributes.</span></span>
