4. <span data-ttu-id="5c401-121">Создать экземпляр <xref:System.TimeSpan> объект, который определяет смещение часового пояса от времени UTC.</span><span class="sxs-lookup"><span data-stu-id="5c401-121">Instantiate a <xref:System.TimeSpan> object that defines the time zone's offset from UTC.</span></span> <span data-ttu-id="5c401-122">Часовые пояса со временем, которые прибыли позже, чем UTC, имеют положительное смещение.</span><span class="sxs-lookup"><span data-stu-id="5c401-122">Time zones with times that are later than UTC have a positive offset.</span></span> <span data-ttu-id="5c401-123">Часовые пояса со временем, предшествующих UTC, имеют отрицательное смещение.</span><span class="sxs-lookup"><span data-stu-id="5c401-123">Time zones with times that are earlier than UTC have a negative offset.</span></span>
