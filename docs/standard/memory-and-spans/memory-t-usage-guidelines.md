## <a name="owners-consumers-and-lifetime-management"></a><span data-ttu-id="349a5-112">Владельцы, объекты-получатели и управление жизненным циклом</span><span class="sxs-lookup"><span data-stu-id="349a5-112">Owners, consumers, and lifetime management</span></span>

- <span data-ttu-id="349a5-120">**Использование**.</span><span class="sxs-lookup"><span data-stu-id="349a5-120">**Consumption**.</span></span> <span data-ttu-id="349a5-121">Объект-получатель экземпляра буфера может выполнять чтение и запись в буфер.</span><span class="sxs-lookup"><span data-stu-id="349a5-121">The consumer of a buffer instance is allowed to use the buffer instance by reading from it and possibly writing to it.</span></span> <span data-ttu-id="349a5-122">У буфера одновременно может быть только один объект-получатель, если не обеспечен какой-либо внешний механизм синхронизации.</span><span class="sxs-lookup"><span data-stu-id="349a5-122">Buffers can have one consumer at a time unless some external synchronization mechanism is provided.</span></span> <span data-ttu-id="349a5-123">Обратите внимание, что объектом-получателя буфера не обязательно является владелец буфера.</span><span class="sxs-lookup"><span data-stu-id="349a5-123">Note that the active consumer of a buffer isn't necessarily the buffer's owner.</span></span>

```csharp
using System;
```

<span data-ttu-id="349a5-132">У буфера два объекта-получателя — `WriteInt32ToBuffer` и `DisplayBufferToConsole`.</span><span class="sxs-lookup"><span data-stu-id="349a5-132">The buffer has two consumers, `WriteInt32ToBuffer` and `DisplayBufferToConsole`.</span></span> <span data-ttu-id="349a5-133">Одновременно у буфера может быть только один объект-получатель (сначала `WriteInt32ToBuffer`, а затем `DisplayBufferToConsole`), и ни один из них не является владельцем буфера.</span><span class="sxs-lookup"><span data-stu-id="349a5-133">There is only one consumer at a time (first `WriteInt32ToBuffer`, then `DisplayBufferToConsole`), and neither of the consumers owns the buffer.</span></span> <span data-ttu-id="349a5-134">Обратите внимание, что объект-получатель здесь имеет не только возможность чтения из буфера.
