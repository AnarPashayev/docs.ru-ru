- <span data-ttu-id="f5c14-106">`A` — это один буквенный символ, который называют *описателем формата*.</span><span class="sxs-lookup"><span data-stu-id="f5c14-106">`A` is a single alphabetic character called the *format specifier*.</span></span> <span data-ttu-id="f5c14-107">Любая строка числового формата, содержащая более одной буквы, включая пробелы, интерпретируется как строка настраиваемого числового формата.</span><span class="sxs-lookup"><span data-stu-id="f5c14-107">Any numeric format string that contains more than one alphabetic character, including white space, is interpreted as a custom numeric format string.</span></span> <span data-ttu-id="f5c14-108">Дополнительные сведения см. в разделе [Строки настраиваемых числовых форматов](custom-numeric-format-strings.md).</span><span class="sxs-lookup"><span data-stu-id="f5c14-108">For more information, see [Custom Numeric Format Strings](custom-numeric-format-strings.md).</span></span>

  <span data-ttu-id="f5c14-114">Когда *указатель точности* ограничивает число цифр дробной части в итоговой строке, числа в итоговых строках округляются до представимого результата, ближайшего к бесконечно точному.</span><span class="sxs-lookup"><span data-stu-id="f5c14-114">When *precision specifier* controls the number of fractional digits in the result string, the result string reflects a number that is rounded to a representable result nearest to the infinitely precise result.</span></span> <span data-ttu-id="f5c14-115">Если имеются два одинаково близких представимых результата:</span><span class="sxs-lookup"><span data-stu-id="f5c14-115">If there are two equally near representable results:</span></span>
