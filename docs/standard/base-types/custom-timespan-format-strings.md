<span data-ttu-id="48e0c-105">Строка формата <xref:System.TimeSpan> определяет строковое представление значения <xref:System.TimeSpan>, получаемого после операции форматирования.</span><span class="sxs-lookup"><span data-stu-id="48e0c-105">A <xref:System.TimeSpan> format string defines the string representation of a <xref:System.TimeSpan> value that results from a formatting operation.</span></span> <span data-ttu-id="48e0c-106">Строка настраиваемого формата содержит один или несколько описателей настраиваемого формата <xref:System.TimeSpan> и любое число литеральных символов.</span><span class="sxs-lookup"><span data-stu-id="48e0c-106">A custom format string consists of one or more custom <xref:System.TimeSpan> format specifiers along with any number of literal characters.</span></span> <span data-ttu-id="48e0c-107">Любая строка, не являющаяся [строкой стандартного формата TimeSpan](standard-timespan-format-strings.md), воспринимается как строка пользовательского формата <xref:System.TimeSpan>.</span><span class="sxs-lookup"><span data-stu-id="48e0c-107">Any string that isn't a [Standard TimeSpan format string](standard-timespan-format-strings.md) is interpreted as a custom <xref:System.TimeSpan> format string.</span></span>

> [!IMPORTANT]
> <span data-ttu-id="48e0c-108">Описатели настраиваемого формата <xref:System.TimeSpan> не включают разделяющие символы, то есть символы, которые отделяют дни от часов, часы от минут или секунды от долей секунд.</span><span class="sxs-lookup"><span data-stu-id="48e0c-108">The custom <xref:System.TimeSpan> format specifiers don't include placeholder separator symbols, such as the symbols that separate days from hours, hours from minutes, or seconds from fractional seconds.</span></span> <span data-ttu-id="48e0c-109">Вместо этого эти символы должны быть включены в строку настраиваемого формата как строковые литералы.</span><span class="sxs-lookup"><span data-stu-id="48e0c-109">Instead, these symbols must be included in the custom format string as string literals.</span></span> <span data-ttu-id="48e0c-110">Например, строка `"dd\.hh\:mm"` определяет точку (.) как разделитель дней и часов и двоеточие (:) как разделитель часов и минут.</span><span class="sxs-lookup"><span data-stu-id="48e0c-110">For example, `"dd\.hh\:mm"` defines a period (.) as the separator between days and hours, and a colon (:) as the separator between hours and minutes.</span></span>
>
> <span data-ttu-id="48e0c-111">Настраиваемые спецификаторы формата <xref:System.TimeSpan> также не содержат символ знака, позволяющий различать положительные и отрицательные временные интервалы.</span><span class="sxs-lookup"><span data-stu-id="48e0c-111">Custom <xref:System.TimeSpan> format specifiers also don't include a sign symbol that enables you to differentiate between negative and positive time intervals.</span></span> <span data-ttu-id="48e0c-112">Чтобы включить символ знака, необходимо построить строку формата с помощью условной логики.</span><span class="sxs-lookup"><span data-stu-id="48e0c-112">To include a sign symbol, you have to construct a format string by using conditional logic.</span></span> <span data-ttu-id="48e0c-113">См. пример в разделе [Остальные символы](#other-characters).</span><span class="sxs-lookup"><span data-stu-id="48e0c-113">The [Other characters](#other-characters) section includes an example.</span></span>
