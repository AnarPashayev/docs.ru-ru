# <a name="security-in-net"></a>Безопасность в .NET
Среда CLR и .NET предоставляют много полезных классов и служб, которые позволяют разработчикам написание безопасного кода и позволяют системным администраторам настраивать разрешения для кода можно получить доступ к защищенным ресурсам. Кроме того среда выполнения и .NET предоставляют полезные классы и службами, которые облегчают использование криптографии и безопасности на основе ролей.
