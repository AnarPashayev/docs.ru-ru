# <a name="azure-serverless-platform"></a>Платформа Azure без сервера

Платформа Azure без сервера включает в себя функции Azure, Logic Apps и "Сетка событий". Эти службы совместной работы и подключение с помощью множество других ресурсов. Бессерверная платформа работает с все, что из баз данных и хранения для аналитики и машинного обучения и искусственного интеллекта.

Можно также использовать Application Insights, бессерверная платформа для записи трассировки диагностики и телеметрии. Application Insights доступны для приложений всех типов (классических, мобильных, или веб-) и реализации. Платформа визуализируются на следующей схеме:
