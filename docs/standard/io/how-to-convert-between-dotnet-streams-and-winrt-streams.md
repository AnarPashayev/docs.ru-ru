## <a name="convert-from-a-windows-runtime-to-a-net-framework-stream"></a><span data-ttu-id="2bb54-110">Преобразование потока среды выполнения Windows в поток .NET Framework</span><span class="sxs-lookup"><span data-stu-id="2bb54-110">Convert from a Windows Runtime to a .NET Framework stream</span></span>
