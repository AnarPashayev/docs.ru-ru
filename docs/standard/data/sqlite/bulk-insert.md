# <a name="bulk-insert"></a>Массовая вставка

В SQLite нет специального способа выполнения операций массовой вставки данных. Чтобы обеспечить оптимальную производительность при вставке или обновлении данных, необходимо сделать следующее:

- Использовать транзакцию.
- Повторно использовать одну и ту же параметризованную команду. При последующих выполнениях повторно использовать компиляцию первого выполнения.
