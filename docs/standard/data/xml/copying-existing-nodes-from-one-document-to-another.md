<span data-ttu-id="80639-103">Метод **ImportNode** позволяет скопировать узел или целое поддерево узлов из одного документа **XmlDocument** в другой.</span><span class="sxs-lookup"><span data-stu-id="80639-103">The **ImportNode** method is the mechanism by which a node or entire node subtree is copied from one **XmlDocument** to another.</span></span> <span data-ttu-id="80639-104">Узел, возвращенный после вызова, является копией узла из исходного документа, включая значения атрибутов, имя узла, тип узла и все атрибуты, связанные с пространством имен, такие как префикс, локальное имя и URI пространства имен.</span><span class="sxs-lookup"><span data-stu-id="80639-104">The node returned from the call is a copy of the node from the source document, including attribute values, the node name, node type, and all namespace-related attributes such as the prefix, local name, and namespace Uniform Resource Identifier (URI).</span></span> <span data-ttu-id="80639-105">Исходный документ не изменяется.</span><span class="sxs-lookup"><span data-stu-id="80639-105">The source document is not changed.</span></span> <span data-ttu-id="80639-106">После импорта узла его необходимо добавить к дереву с помощью одного из методов, используемых для вставки узлов.</span><span class="sxs-lookup"><span data-stu-id="80639-106">After you have imported the node, you still have to add it to the tree using one of the methods used to insert nodes.</span></span>  
  
 <span data-ttu-id="80639-107">При присоединении узла к новому документу, узел принадлежит этому документу.</span><span class="sxs-lookup"><span data-stu-id="80639-107">When the node is attached to its new document, the new document owns the node.</span></span> <span data-ttu-id="80639-108">Причина в том, что каждый узел при создании принадлежит какому-нибудь документу, даже если узлы создаются в разных фрагментах документа.</span><span class="sxs-lookup"><span data-stu-id="80639-108">The reason is that each node, when created, has an owning document, even if the nodes are created in separate document fragments.</span></span> <span data-ttu-id="80639-109">Это требование объектной модели DOM XML-документа, которое обусловлено структурой создания фабрики в классе **XmlDocument**.</span><span class="sxs-lookup"><span data-stu-id="80639-109">This is a requirement of the XML Document Object Model (DOM) and is enforced by the factory creation design on the **XmlDocument** class.</span></span> <span data-ttu-id="80639-110">Например, метод **CreateElement** является единственным способом создания новых узлов.</span><span class="sxs-lookup"><span data-stu-id="80639-110">For example, **CreateElement**, is the only way to create new nodes.</span></span>
