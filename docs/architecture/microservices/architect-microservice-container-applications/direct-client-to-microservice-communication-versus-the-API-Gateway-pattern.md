## <a name="drawbacks-of-the-api-gateway-pattern"></a><span data-ttu-id="548f2-251">Недостатки шаблона шлюза API</span><span class="sxs-lookup"><span data-stu-id="548f2-251">Drawbacks of the API Gateway pattern</span></span>
