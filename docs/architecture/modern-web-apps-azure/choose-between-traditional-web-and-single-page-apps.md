<span data-ttu-id="5d105-116">Кроме того, для работы с платформой одностраничных приложений требуется гораздо больший опыт в построении архитектуры и обеспечении безопасности.</span><span class="sxs-lookup"><span data-stu-id="5d105-116">Additionally, SPA frameworks require greater architectural and security expertise.</span></span> <span data-ttu-id="5d105-117">Они характеризуются большей частотой обновления и появления новых платформ по сравнению с традиционными веб-приложениями.</span><span class="sxs-lookup"><span data-stu-id="5d105-117">They experience greater churn due to frequent updates and new frameworks than traditional web applications.</span></span> <span data-ttu-id="5d105-118">Кроме того, по сравнению с традиционными веб-приложениями при работе с одностраничными приложениями может усложнятся настройка процессов автоматизированного построения и развертывания, а также использование таких вариантов развертывания, как контейнеры.</span><span class="sxs-lookup"><span data-stu-id="5d105-118">Configuring automated build and deployment processes and utilizing deployment options like containers may be more difficult with SPA applications than traditional web apps.</span></span>

## <a name="when-to-choose-traditional-web-apps"></a><span data-ttu-id="5d105-132">Когда следует выбирать традиционные веб-приложения</span><span class="sxs-lookup"><span data-stu-id="5d105-132">When to choose traditional web apps</span></span>
