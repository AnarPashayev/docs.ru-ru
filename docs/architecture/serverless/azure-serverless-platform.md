# <a name="azure-serverless-platform"></a>Несерверная Платформа Azure

Бессерверная Платформа Azure включает функции Azure, Logic Apps и службу "Сетка событий". Эти службы работают вместе и соединяются с множеством других ресурсов. Бессерверная платформа работает со всеми базами данных и хранилищами для аналитики, машинного обучения и искусственного интеллекта.

Вы также можете использовать Application Insights, бессерверную платформу для сбора диагностических трассировок и телеметрии. Application Insights доступны для приложений всех типов (настольные системы, мобильные приложения и веб-приложения), а также бессерверных реализаций.
