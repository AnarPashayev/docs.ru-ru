# <a name="azure-serverless-platform"></a>Бессерверная платформа Azure

В состав бессерверной платформы Azure входят Функции Azure, Logic Apps и Сетка событий. Эти службы работают вместе и могут подключаться к множеству других ресурсов. Бессерверная платформа работает с любыми решениями — от баз данных и хранилищ до средств аналитики, машинного обучения и искусственного интеллекта.

Вы также можете использовать Application Insights — бессерверную службу для сбора диагностических данных трассировки и телеметрии. Служба Application Insights доступна для приложений всех типов (классических, мобильных и веб-приложений), а также бессерверных реализаций. Платформа показана на следующей схеме:
