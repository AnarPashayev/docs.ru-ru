## <a name="introduction"></a><span data-ttu-id="1ff64-129">Вступление</span><span class="sxs-lookup"><span data-stu-id="1ff64-129">Introduction</span></span>

<span data-ttu-id="1ff64-130">.NET уже давно поддерживает разработку веб-приложений через ASP.NET — комплексный набор платформ и средств для создания любых типов приложений.</span><span class="sxs-lookup"><span data-stu-id="1ff64-130">.NET has long supported web app development through ASP.NET, a comprehensive set of frameworks and tools for building any kind of web app.</span></span> <span data-ttu-id="1ff64-131">ASP.NET имеет собственную линейку веб-платформ и технологий, начиная с классического ASP.</span><span class="sxs-lookup"><span data-stu-id="1ff64-131">ASP.NET has its own lineage of web frameworks and technologies starting all the way back with classic Active Server Pages (ASP).</span></span> <span data-ttu-id="1ff64-132">Такие платформы, как ASP.NET Web Forms, ASP.NET MVC, веб-страницы ASP.NET и более поздняя ASP.NET Core, предоставляют эффективный и функциональный способ создания веб-приложений, *подготавливаемых к просмотру на сервере*, в которых содержимое пользовательского интерфейса динамически формируется на сервере в ответ на HTTP-запросы.</span><span class="sxs-lookup"><span data-stu-id="1ff64-132">Frameworks like ASP.NET Web Forms, ASP.NET MVC, ASP.NET Web Pages, and more recently ASP.NET Core, provide a productive and powerful way to build определенную аудиторию и подход к созданию приложений.</span><span class="sxs-lookup"><span data-stu-id="1ff64-133">Each ASP.NET framework caters to a different audience and app building разработчикам классических приложений, таких как многократно используемые элементы управления пользовательского интерфейса с простой обработкой событий.</span><span class="sxs-lookup"><span data-stu-id="1ff64-134">ASP.NET Web Forms shipped with the original release of the .NET Framework and enabled web development using many of the patterns familiar to desktop developers, like reusable UI controls with simple event handling.</span></span> <span data-ttu-id="1ff64-135">Однако ни одно из предложений ASP.NET не предоставляет способа для выполнения кода, выполняемого в браузере пользователя.</span><span class="sxs-lookup"><span data-stu-id="1ff64-135">However, none of the ASP.NET offerings provide a way to run code that executed in the user's browser.</span></span> <span jQuery, Knockout, Angular, React и т. д.</span><span class="sxs-lookup"><span data-stu-id="1ff64-136">To do that requires writing JavaScript and using any of the many JavaScript frameworks and tools <span data-ttu-id="1ff64-137">[Blazor](https://blazor.net) — это новая веб-платформа, которая расширяет возможности создания веб-приложений с помощью .NET.</span><span class="sxs-lookup"><span web standard called WebAssembly.</span></span> <span data-ttu-id="1ff64-140">Кроме того, Blazor позволяет запускать интерфейсные компоненты .NET на сервере и гибко управлять всеми взаимодействиями с интерфейсом через подключение к браузеру в режиме реального времени.</span><span class="sxs-lookup"><span data-stu-id="1ff64-140">Or alternatively, Blazor can run your .NET UI components on the server <span data-ttu-id="1ff64-143">Эта книга знакомит разработчиков ASP.NET Web Forms с Blazor в привычном и удобном для них формате.</span><span class="sxs-lookup"><span data-stu-id="1ff64-143">This book introduces ASP.NET Web Forms developers to Blazor in a way that is familiar and convenient.</span></span> <span data-ttu-id="1ff64-144">Понятия Blazor вводятся здесь в сравнении с аналогичными понятиями из ASP.NET Web Forms, и дается пояснение новых понятий, которые могут оказаться менее знакомыми.</span><span class="sxs-lookup"><span data-stu-id="1ff64-144">It introduces Blazor concepts in parallel with analogous concepts in ASP.NET Web Forms while also explaining new concepts that may be less familiar.</span></span> <span data-ttu-id="1ff64-145">Книга охватывает обширный спектр вопросов и проблем, включая создание компонентов, маршрутизацию, макеты, настройку и безопасность.</span><span class="sxs-lookup"><span data-stu-id="1ff64-145">It covers a broad range of topics and concerns including component authoring, routing, layout, configuration, and security.</span></span> <span data-ttu-id="1ff64-146">И хотя она в первую очередь помогает делать новые разработки, в ней также приведены рекомендации и стратегии по модернизации и переносу на Blazor уже существующих приложений на базе ASP.NET Web Forms.</span><span class="sxs-lookup"><span data-stu-id="1ff64-146">And while the content of this book is primarily for enabling new development, it also covers guidelines and strategies for migrating existing ASP.NET Web Forms to Blazor for when you want to modernize an existing app.</span></span>
