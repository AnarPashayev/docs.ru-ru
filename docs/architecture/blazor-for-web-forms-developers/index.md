## <a name="introduction"></a><span data-ttu-id="3d1eb-126">Вступление</span><span class="sxs-lookup"><span data-stu-id="3d1eb-126">Introduction</span></span>

<span data-ttu-id="3d1eb-127">.NET уже давно поддерживает разработку веб-приложений через ASP.NET — комплексный набор платформ и средств для создания любых типов приложений.</span><span class="sxs-lookup"><span data-stu-id="3d1eb-127">.NET has long supported web app development through ASP.NET, a comprehensive set of frameworks and tools for building any kind of web app.</span></span> <span data-ttu-id="3d1eb-128">ASP.NET имеет собственную линейку веб-платформ и технологий, начиная с классического ASP.</span><span class="sxs-lookup"><span data-stu-id="3d1eb-128">ASP.NET has its own lineage of web frameworks and technologies starting all the way back with classic Active Server Pages (ASP).</span></span> <span data-ttu-id="3d1eb-129">Такие платформы, как ASP.NET Web Forms, ASP.NET MVC, веб-страницы ASP.NET и более поздняя ASP.NET Core, предоставляют эффективный и функциональный способ создания веб-приложений, *подготавливаемых к просмотру на сервере*, в которых содержимое пользовательского интерфейса динамически формируется на сервере в ответ на HTTP-запросы.</span><span class="sxs-lookup"><span data-stu-id="3d1eb-129">Frameworks like ASP.NET Web Forms, ASP.NET MVC, ASP.NET Web Pages, and more recently ASP.NET Core, provide a productive and powerful way to build *server-rendered* web apps, where UI content is dynamically generated on the server in response to HTTP requests.</span></span> <span data-ttu-id="3d1eb-130">Каждая платформа ASP.NET ориентирована на определенную аудиторию и подход к созданию приложений.</span><span class="sxs-lookup"><span data-stu-id="3d1eb-130">Each ASP.NET framework caters to a different audience and app building philosophy.</span></span> <span data-ttu-id="3d1eb-131">ASP.NET Web Forms поставляется с исходным выпуском .NET Framework и поддержкой веб-разработки с использованием множества шаблонов, знакомых разработчикам классических приложений, таких как многократно используемые элементы управления пользовательского интерфейса с простой обработкой событий.</span><span class="sxs-lookup"><span data-stu-id="3d1eb-131">ASP.NET Web Forms shipped with the original release of the .NET Framework and enabled web development using many of the patterns familiar to desktop developers, like reusable UI controls with simple event handling.</span></span> <span data-ttu-id="3d1eb-132">Однако ни одно из предложений ASP.NET не предоставляет способа для выполнения кода, выполняемого в браузере пользователя.</span><span class="sxs-lookup"><span data-stu-id="3d1eb-132">However, none of the ASP.NET offerings provide a way to run code that executed in the user's browser.</span></span> <span jQuery, Knockout, Angular, React и т. д.</span><span class="sxs-lookup"><span data-stu-id="3d1eb-133">To do that requires writing JavaScript and using any of the many JavaScript frameworks and tools <span data-ttu-id="3d1eb-134">[Blazor](https://blazor.net) — это новая веб-платформа, которая расширяет возможности создания веб-приложений с помощью .NET.</span><span class="sxs-lookup"><span WebAssembly.</span></span> <span data-ttu-id="3d1eb-137">Кроме того, Blazor позволяет запускать компоненты пользовательского интерфейса .NET на сервере и гибко управлять всеми взаимодействиями пользовательского интерфейса через подключение в режиме реального времени с помощью браузера.</span><span class="sxs-lookup"><span data-stu-id="3d1eb-137">Or alternatively, Blazor can run your .NET UI class="sxs-lookup"><span data-stu-id="3d1eb-139">While Blazor shares many commonalities with ASP.NET Web Forms, like having a reusable component model and a simple way to handle user events, it also <span data-ttu-id="3d1eb-140">Эта книга привычным и удобным образом знакомит разработчиков ASP.NET Web Forms с Blazor.</span><span class="sxs-lookup"><span data-stu-id="3d1eb-140">This book introduces ASP.NET Web Forms developers to Blazor in a way that is familiar and convenient.</span></span> <span data-ttu-id="3d1eb-141">В ней представлены концепции Blazor вместе с аналогичными концепциями в ASP.NET Web Forms, а также разобраны новые концепции, которые могут оказаться непривычными.</span><span class="sxs-lookup"><span data-stu-id="3d1eb-141">It introduces Blazor concepts in parallel with analogous concepts in ASP.NET Web Forms while also explaining new concepts that may be less familiar.</span></span> <span data-ttu-id="3d1eb-142">Она охватывает обширный спектр вопросов и проблем, включая создание компонентов, маршрутизацию, макеты, настройку и безопасность.</span><span class="sxs-lookup"><span data-stu-id="3d1eb-142">It covers a broad range of topics and concerns including component authoring, routing, layout, configuration, and security.</span></span> <span data-ttu-id="3d1eb-143">И хотя эта книга в основном посвящена формированию новой процедуры разработки, в ней также описаны рекомендации и стратегии переноса существующих решений ASP.NET Web Forms в Blazor, когда требуется модернизировать существующее приложение.</span><span class="sxs-lookup"><span data-stu-id="3d1eb-143">And while the content of this book is primarily for enabling new development, it also covers guidelines and strategies for migrating existing ASP.NET Web Forms to Blazor for when you want to modernize an existing app.</span></span>
