# <a name="scaling-cloud-native-applications"></a>Масштабирование облачных приложений

[!INCLUDE [book-preview](../../../includes/book-preview.md)]

Одним из наиболее часто похвалу преимуществ перехода на облачную среду размещения является масштабируемость. Масштабируемость или возможность приложения принимать дополнительную нагрузку, не нарушая производительность каждого пользователя. Чаще всего это достигается путем разбиения приложения на небольшие части, которые могут предоставлять все необходимые ресурсы. Поставщики облачных решений обеспечивают широкие возможности масштабирования в любое время и в любой точке мира.

 В этой главе обсуждаются технологии, позволяющие выполнять масштабирование собственных приложений в облаке в соответствии с потребностями пользователей. Ниже приведены эти технологии.

- Контейнеры
- Оркестраторы
- Бессерверные вычисления
