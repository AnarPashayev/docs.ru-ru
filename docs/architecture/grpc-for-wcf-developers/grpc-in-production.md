# <a name="grpc-in-production"></a>gRPC в рабочей среде

ASP.NET Core 3,0 приложений, включая службы gRPC Services, могут выполняться в Windows, Linux и в контейнерах с помощью современных платформ, таких как DOCKER и Kubernetes. В этой главе рассматриваются различные варианты запуска служб gRPC в рабочей среде и рассмотрены параметры мониторинга и ведения журнала, позволяющие обеспечить оптимальную работу систем.
