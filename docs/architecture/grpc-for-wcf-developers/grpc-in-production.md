# <a name="grpc-in-production"></a>gRPC в рабочей среде

Вы можете запускать приложения ASP.NET Core 3,0, в том числе службы gRPC Services, в Windows, в Linux и в контейнерах с помощью современных платформ, таких как DOCKER и Kubernetes. В этой главе рассматриваются различные варианты запуска служб gRPC в рабочей среде и рассматриваются параметры мониторинга и ведения журнала, позволяющие обеспечить оптимальную работу систем.
