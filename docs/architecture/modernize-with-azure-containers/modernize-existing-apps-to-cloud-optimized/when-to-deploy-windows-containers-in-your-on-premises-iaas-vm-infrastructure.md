# <a name="when-to-deploy-windows-containers-in-your-on-premises-iaas-vm-infrastructure"></a>Когда следует развертывать контейнеры Windows в локальной инфраструктуре виртуальных машин IaaS

- Возможно, ваша организация не готова к переходу в облако или не может его выполнить из коммерческих соображений. Но вы по-прежнему можете получать преимущества от использования контейнеров Windows в собственных центрах обработки данных.

- У вас могут быть другие артефакты, которые используются в локальной среде, и это может замедлить работу при попытке перехода в облако. К ним могут относиться, например, зависимости безопасности или проверки подлинности с локальной службой Windows Server Active Directory либо любой другой локальный ресурс.

- Если вы начнете использовать контейнеры Windows сегодня, завтра вы сможете гораздо эффективнее выполнить поэтапную миграцию в облако. Контейнеры Windows становятся единицей облачного развертывания без привязки к конкретному облаку.
