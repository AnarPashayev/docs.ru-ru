# <a name="when-to-deploy-windows-containers-in-your-on-premises-iaas-vm-infrastructure"></a>Когда следует развертывать контейнеры Windows в локальной инфраструктуре виртуальных машин IaaS

- Возможно, ваша организация не готова к переходу в облако или не может перейти в облако по бизнес-причине. Но вы по-прежнему можете получить преимущества использования контейнеров Windows в собственных центрах обработки данных.

- У вас могут быть другие артефакты, которые используются локально, и это может замедлить работу при попытке перехода в облако. Например, зависимости безопасности или проверки подлинности с локальным Active Directory Windows Server или любым другим локальным ресурсом.

- Если вы начинаете использовать контейнеры Windows уже сегодня, можно выполнить поэтапную миграцию в облако завтра из гораздо более высокого качества. Контейнеры Windows становятся единицами развертывания для любого облака без блокировки.
