# <a name="endregion-c-reference"></a>#endregion (справочник по C#)
`#endregion` отмечает конец блока [#region](./preprocessor-region.md). Например:  
  
```csharp
#region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
