# <a name="compiler-warning-level-1-cs0465"></a><span data-ttu-id="adc74-103">Предупреждение компилятора (уровень 1) CS0465</span><span class="sxs-lookup"><span data-stu-id="adc74-103">Compiler Warning (level 1) CS0465</span></span>

<span data-ttu-id="adc74-104">Введение метода "Finalize" может конфликтовать с вызовом деструктора.</span><span class="sxs-lookup"><span data-stu-id="adc74-104">Introducing a 'Finalize' method can interfere with destructor invocation.</span></span> <span data-ttu-id="adc74-105">Деструктор был объявлен намеренно?</span><span class="sxs-lookup"><span data-stu-id="adc74-105">Did you intend to declare a destructor?</span></span>  
  
 <span data-ttu-id="adc74-106">Это предупреждение возникает при создании класса, содержащего метод с сигнатурой `public virtual void Finalize`.</span><span class="sxs-lookup"><span data-stu-id="adc74-106">This warning occurs when you create a class with a method whose signature is `public virtual void Finalize`.</span></span>  
  
 <span data-ttu-id="adc74-107">Если такой класс используется в качестве базового класса и в производном классе определяется деструктор, этот деструктор переопределит метод базового класса `Finalize`, а не <xref:System.Object.Finalize%2A>.</span><span class="sxs-lookup"><span data-stu-id="adc74-107">If such a class is used as a base class and if the deriving class defines a destructor, the destructor will override the base class `Finalize` method, not <xref:System.Object.Finalize%2A>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="adc74-108">Пример</span><span class="sxs-lookup"><span data-stu-id="adc74-108">Example</span></span>  

 <span data-ttu-id="adc74-109">В следующем примере возникает предупреждение CS0465.</span><span class="sxs-lookup"><span data-stu-id="adc74-109">The following sample generates CS0465.</span></span>  
  
```csharp  
// CS0465.cs  
// compile with: /target:library  
class A  
{  
   public virtual void Finalize() {}   // CS0465  
}  
  
// OK  
class B  
{  
   ~B() {}  
}  
```
