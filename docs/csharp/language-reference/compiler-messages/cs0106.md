## <a name="example"></a><span data-ttu-id="2edf9-115">Пример</span><span class="sxs-lookup"><span data-stu-id="2edf9-115">Example</span></span>

```csharp

{
   interface I
   {
      void m();

   public class MyClass
   {
      public void I.m() {}   // CS0106
      public static void Main() {}
   }
}
```
