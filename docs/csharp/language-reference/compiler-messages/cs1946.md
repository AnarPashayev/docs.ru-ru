# <a name="compiler-error-cs1946"></a><span data-ttu-id="e4ce3-102">Ошибка компилятора CS1946</span><span class="sxs-lookup"><span data-stu-id="e4ce3-102">Compiler Error CS1946</span></span>

<span data-ttu-id="e4ce3-103">Выражение анонимного метода не может быть преобразовано в дерево выражения.</span><span class="sxs-lookup"><span data-stu-id="e4ce3-103">An anonymous method expression cannot be converted to an expression tree.</span></span>

<span data-ttu-id="e4ce3-104">[Анонимный метод](../operators/delegate-operator.md) — это набор инструкций, но дерево выражения не должно содержать инструкции.</span><span class="sxs-lookup"><span data-stu-id="e4ce3-104">An [anonymous method](../operators/delegate-operator.md) represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="e4ce3-105">Таким образом, анонимный метод не может быть представлен в виде дерева выражения.</span><span class="sxs-lookup"><span data-stu-id="e4ce3-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>

<span data-ttu-id="e4ce3-106">Чтобы устранить эту ошибку, замените анонимный метод [лямбда-выражением](../operators/lambda-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="e4ce3-106">To correct this error, change the anonymous method to a [lambda expression](../operators/lambda-expressions.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="e4ce3-107">Пример</span><span class="sxs-lookup"><span data-stu-id="e4ce3-107">Example</span></span>

<span data-ttu-id="e4ce3-108">В следующем примере возникает ошибка CS1946:</span><span class="sxs-lookup"><span data-stu-id="e4ce3-108">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
