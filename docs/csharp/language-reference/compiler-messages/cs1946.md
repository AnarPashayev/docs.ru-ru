# <a name="compiler-error-cs1946"></a><span data-ttu-id="0e92f-103">Ошибка компилятора CS1946</span><span class="sxs-lookup"><span data-stu-id="0e92f-103">Compiler Error CS1946</span></span>

<span data-ttu-id="0e92f-104">Выражение анонимного метода не может быть преобразовано в дерево выражения.</span><span class="sxs-lookup"><span data-stu-id="0e92f-104">An anonymous method expression cannot be converted to an expression tree.</span></span>

<span data-ttu-id="0e92f-105">[Анонимный метод](../operators/delegate-operator.md) — это набор инструкций, но дерево выражения не должно содержать инструкции.</span><span class="sxs-lookup"><span data-stu-id="0e92f-105">An [anonymous method](../operators/delegate-operator.md) represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="0e92f-106">Таким образом, анонимный метод не может быть представлен в виде дерева выражения.</span><span class="sxs-lookup"><span data-stu-id="0e92f-106">Therefore an anonymous method cannot be represented by an expression tree.</span></span>

<span data-ttu-id="0e92f-107">Чтобы устранить эту ошибку, замените анонимный метод [лямбда-выражением](../operators/lambda-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="0e92f-107">To correct this error, change the anonymous method to a [lambda expression](../operators/lambda-expressions.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="0e92f-108">Пример</span><span class="sxs-lookup"><span data-stu-id="0e92f-108">Example</span></span>

<span data-ttu-id="0e92f-109">В следующем примере возникает ошибка CS1946:</span><span class="sxs-lookup"><span data-stu-id="0e92f-109">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
