# <a name="compiler-error-cs1567"></a><span data-ttu-id="45079-103">Ошибка компилятора CS1567</span><span class="sxs-lookup"><span data-stu-id="45079-103">Compiler Error CS1567</span></span>
<span data-ttu-id="45079-104">Ошибка при создании ресурса Win32: "файл"</span><span class="sxs-lookup"><span data-stu-id="45079-104">Error generating Win32 resource: 'file'</span></span>  
  
 <span data-ttu-id="45079-105">При компиляции использовался параметр компилятора [-win32icon](../compiler-options/win32icon-compiler-option.md) или не использовался параметр [-win32res](../compiler-options/win32res-compiler-option.md). В результате этого компилятору не удается создать необходимый файл со сведениями о ресурсах из-за нехватки пространства на диске или другой ошибки.</span><span class="sxs-lookup"><span data-stu-id="45079-105">Your compilation either used the [-win32icon](../compiler-options/win32icon-compiler-option.md) compiler option or did not use [-win32res](../compiler-options/win32res-compiler-option.md), which causes the compiler to generate a file that contains resource information, but the compiler was unable to create the file due to insufficient disk space or some other error.</span></span>  
  
 <span data-ttu-id="45079-106">Если вам не удается решить проблему с созданием файла, попробуйте использовать параметр [-win32res](../compiler-options/win32res-compiler-option.md), при выборе которого файл со сведениями о ресурсах не создается.</span><span class="sxs-lookup"><span data-stu-id="45079-106">If you are unable to resolve the file-generation problem, you could use [-win32res](../compiler-options/win32res-compiler-option.md), which does not generate a file that contains resource information.</span></span>
