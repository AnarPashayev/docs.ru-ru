## <a name="example"></a><span data-ttu-id="776b7-109">Пример</span><span class="sxs-lookup"><span data-stu-id="776b7-109">Example</span></span>  
 <span data-ttu-id="776b7-110">Этот пример кода включает CPP-файл, который компилируется в DLL-файл, а также CS-файл, который использует полученный DLL-файл.</span><span class="sxs-lookup"><span data-stu-id="776b7-110">This code sample consists of a .cpp file, which compiles to a .dll, and a .cs file, which uses that .dll.</span></span> <span data-ttu-id="776b7-111">Следующий код предназначен для DLL-файла и определяет свойство, к которому будет обращаться код в CS-файле.</span><span class="sxs-lookup"><span data-stu-id="776b7-111">The following code is for the .dll file and defines a property to be accessed by the code in the .cs file.</span></span>  
  
```cpp  
// CPP1546.cpp  
// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```
