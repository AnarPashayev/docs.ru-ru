# <a name="compiler-error-cs0019"></a><span data-ttu-id="31876-102">Ошибка компилятора CS0019</span><span class="sxs-lookup"><span data-stu-id="31876-102">Compiler Error CS0019</span></span>

 <span data-ttu-id="31876-104">Бинарный оператор применяется к типам данных, которые не поддерживают его.</span><span class="sxs-lookup"><span data-stu-id="31876-104">A binary operator is applied to data types that do not support it.</span></span> <span data-ttu-id="31876-105">Например, оператор [&#124;&#124;](../operators/boolean-logical-operators.md#conditional-logical-or-operator-) невозможно применять к строкам, операторы [+](../operators/addition-operator.md), [-](../operators/subtraction-operator.md), [\<](../operators/comparison-operators.md#less-than-operator-) или [>](../operators/comparison-operators.md#greater-than-operator-) — к переменным типа [bool](../keywords/bool.md), а оператор [==](../operators/equality-operators.md#equality-operator-) — с типом `struct`, если этот тип не перегружает оператор явно.</span><span class="sxs-lookup"><span data-stu-id="31876-105">For example, you cannot use the [&#124;&#124;](../operators/boolean-logical-operators.md#conditional-logical-or-operator-) operator on strings, you cannot use [+](../operators/addition-operator.md), [-](../operators/subtraction-operator.md), [\<](../operators/comparison-operators.md#less-than-operator-), or [>](../operators/comparison-operators.md#greater-than-operator-) operators on [bool](../keywords/bool.md) variables, and you cannot use the [==](../operators/equality-operators.md#equality-operator-) operator with a `struct` type unless the type explicitly overloads that operator.</span></span>

## <a name="example"></a><span data-ttu-id="31876-108">Пример</span><span class="sxs-lookup"><span data-stu-id="31876-108">Example</span></span>

 <span data-ttu-id="31876-109">В следующем примере ошибка CS0019 возникает в двух местах, поскольку тип [bool](../keywords/bool.md) в C# не допускает преобразование в [int](../builtin-types/integral-numeric-types.md). Кроме того, ошибка CS0019 возникает при попытке применить оператор вычитания к строке.</span><span class="sxs-lookup"><span data-stu-id="31876-109">In the following example, CS0019 is generated in two places because [bool](../keywords/bool.md) in C# is not convertible to [int](../builtin-types/integral-numeric-types.md). CS0019 also is generated when the subtraction operator is applied to a string.</span></span> <span data-ttu-id="31876-110">Оператор сложения (+) можно использовать со строковыми операндами, поскольку он перегружен в классе `String`, где реализует объединение строк.</span><span class="sxs-lookup"><span data-stu-id="31876-110">The addition operator (+) can be used with string operands because that operator is overloaded by the `String` class to perform string concatenation.</span></span>

```csharp
static void Main()
{
    bool result = true;
    if (result > 0) //CS0019
    {
        // Do something.
    }

    int i = 1;
    // You cannot compare an integer and a boolean value.
    if (i == true) //CS0019
    {
        //Do something...
    }
    
    string s = "Just try to subtract me.";
    float f = 100 - s; // CS0019
}
```
