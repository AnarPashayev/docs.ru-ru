## <a name="example"></a><span data-ttu-id="12eae-109">Пример</span><span class="sxs-lookup"><span data-stu-id="12eae-109">Example</span></span>

<span data-ttu-id="12eae-110">В следующем примере ошибка CS0019 возникает в трех местах, поскольку тип [bool](../builtin-types/bool.md) в C# не допускает преобразование в [int](../builtin-types/integral-numeric-types.md). Кроме того, ошибка CS0019 возникает при попытке применить оператор вычитания `-` к строке.</span><span class="sxs-lookup"><span data-stu-id="12eae-110">In the following example, CS0019 is generated in three places because [bool](../builtin-types/bool.md) in C# is not convertible to [int](../builtin-types/integral-numeric-types.md). CS0019 is also generated when the subtraction operator `-` is applied to a string.</span></span> <span data-ttu-id="12eae-111">Оператор сложения `+` можно использовать со строковыми операндами, поскольку он перегружен в классе `String`, где реализует объединение строк.</span><span class="sxs-lookup"><span data-stu-id="12eae-111">The addition operator `+` can be used with string operands because that operator is overloaded by the `String` class to perform string concatenation.</span></span>

```csharp
static void Main()
{
    bool result = true;
    if (result > 0) //CS0019
    {
        // Do something.
    }

    int i = 1;
    // You cannot compare an integer and a boolean value.
    if (i == true) //CS0019
    {
        //Do something...
    }

    string s = "Just try to subtract me.";
    float f = 100 - s; // CS0019
}
```
