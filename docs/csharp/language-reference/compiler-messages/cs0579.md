# <a name="compiler-error-cs0579"></a>Ошибка компилятора CS0579
Повторяющийся атрибут "attribute"  
  
 Один атрибут нельзя использовать несколько раз, если для этого атрибута не установлен параметр **AllowMultiple = true** в его атрибуте [AttributeUsage](../../language-reference/attributes/general.md).  
  
## <a name="example"></a>Пример  
 Следующий пример приводит к возникновению ошибки CS0579.  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
