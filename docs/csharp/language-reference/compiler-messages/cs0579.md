# <a name="compiler-error-cs0579"></a><span data-ttu-id="64e7c-102">Ошибка компилятора CS0579</span><span class="sxs-lookup"><span data-stu-id="64e7c-102">Compiler Error CS0579</span></span>
<span data-ttu-id="64e7c-103">Повторяющийся атрибут "attribute"</span><span class="sxs-lookup"><span data-stu-id="64e7c-103">Duplicate 'attribute' attribute</span></span>  
  
 <span data-ttu-id="64e7c-104">Один атрибут нельзя использовать несколько раз, если для этого атрибута не установлен параметр **AllowMultiple = true** в его атрибуте [AttributeUsage](../../programming-guide/concepts/attributes/attributeusage.md).</span><span class="sxs-lookup"><span data-stu-id="64e7c-104">It is not possible to specify the same attribute more than once unless the attribute specifies **AllowMultiple=true** in its [AttributeUsage](../../programming-guide/concepts/attributes/attributeusage.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="64e7c-105">Пример</span><span class="sxs-lookup"><span data-stu-id="64e7c-105">Example</span></span>  
 <span data-ttu-id="64e7c-106">Следующий пример приводит к возникновению ошибки CS0579.</span><span class="sxs-lookup"><span data-stu-id="64e7c-106">The following example generates CS0579.</span></span>  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
