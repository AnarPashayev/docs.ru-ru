# <a name="compiler-error-cs1943"></a><span data-ttu-id="2605e-102">Ошибка компилятора CS1943</span><span class="sxs-lookup"><span data-stu-id="2605e-102">Compiler Error CS1943</span></span>
<span data-ttu-id="2605e-103">Выражение типа "тип" недопустимо в последующем предложении from в выражении запроса с исходным типом "тип".</span><span class="sxs-lookup"><span data-stu-id="2605e-103">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="2605e-104">Ошибка определения типа при вызове в "метод".</span><span class="sxs-lookup"><span data-stu-id="2605e-104">Type inference failed in the call to 'method'.</span></span>  
  
 <span data-ttu-id="2605e-105">Все переменные диапазона должны представлять запрашиваемые типы.</span><span class="sxs-lookup"><span data-stu-id="2605e-105">All range variables must represent queryable types.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="2605e-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="2605e-106">To correct this error</span></span>  
  
1. <span data-ttu-id="2605e-107">Убедитесь, что тип является запрашиваемым типом, который реализует `IEnumerable`, `IEnumerable<T>` или производный интерфейс, либо любым другим типом, для которого определен шаблон запроса.</span><span class="sxs-lookup"><span data-stu-id="2605e-107">Make sure that the type is a queryable type that implements `IEnumerable`, `IEnumerable<T>` or a derived interface, or any other type which has a query pattern defined for it.</span></span>  
  
2. <span data-ttu-id="2605e-108">Если тип является неуниверсальным типом `IEnumerable`, укажите явный тип переменной диапазона.</span><span class="sxs-lookup"><span data-stu-id="2605e-108">If the type is a non-generic `IEnumerable`, provide an explicit type on the range variable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2605e-109">Пример</span><span class="sxs-lookup"><span data-stu-id="2605e-109">Example</span></span>  
 <span data-ttu-id="2605e-110">В следующем коде возникает ошибка CS1943:</span><span class="sxs-lookup"><span data-stu-id="2605e-110">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
