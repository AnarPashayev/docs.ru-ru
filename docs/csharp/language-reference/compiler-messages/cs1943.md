# <a name="compiler-error-cs1943"></a><span data-ttu-id="a0974-103">Ошибка компилятора CS1943</span><span class="sxs-lookup"><span data-stu-id="a0974-103">Compiler Error CS1943</span></span>
<span data-ttu-id="a0974-104">Выражение типа "тип" недопустимо в последующем предложении from в выражении запроса с исходным типом "тип".</span><span class="sxs-lookup"><span data-stu-id="a0974-104">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="a0974-105">Ошибка определения типа при вызове в "метод".</span><span class="sxs-lookup"><span data-stu-id="a0974-105">Type inference failed in the call to 'method'.</span></span>  
  
 <span data-ttu-id="a0974-106">Все переменные диапазона должны представлять запрашиваемые типы.</span><span class="sxs-lookup"><span data-stu-id="a0974-106">All range variables must represent queryable types.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a0974-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="a0974-107">To correct this error</span></span>  
  
1. <span data-ttu-id="a0974-108">Убедитесь, что тип является запрашиваемым типом, который реализует `IEnumerable`, `IEnumerable<T>` или производный интерфейс, либо любым другим типом, для которого определен шаблон запроса.</span><span class="sxs-lookup"><span data-stu-id="a0974-108">Make sure that the type is a queryable type that implements `IEnumerable`, `IEnumerable<T>` or a derived interface, or any other type which has a query pattern defined for it.</span></span>  
  
2. <span data-ttu-id="a0974-109">Если тип является неуниверсальным типом `IEnumerable`, укажите явный тип переменной диапазона.</span><span class="sxs-lookup"><span data-stu-id="a0974-109">If the type is a non-generic `IEnumerable`, provide an explicit type on the range variable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a0974-110">Пример</span><span class="sxs-lookup"><span data-stu-id="a0974-110">Example</span></span>  
 <span data-ttu-id="a0974-111">В следующем коде возникает ошибка CS1943:</span><span class="sxs-lookup"><span data-stu-id="a0974-111">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
