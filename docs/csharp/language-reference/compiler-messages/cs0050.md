## <a name="example"></a><span data-ttu-id="28649-106">Пример</span><span class="sxs-lookup"><span data-stu-id="28649-106">Example</span></span>

 <span data-ttu-id="28649-107">В следующем примере возникает ошибка CS0050. Это связано с тем, что для `MyClass` не указан модификатор доступности и по умолчанию используется уровень доступности `private`:</span><span class="sxs-lookup"><span data-stu-id="28649-107">The following sample generates CS0050 because no accessibility modifier is supplied for `MyClass`, and its accessibility therefore defaults to `private`:</span></span>

```csharp
// CS0050.cs
class MyClass // Accessibility defaults to private.
// Try the following line instead.
// public class MyClass
{
}

public class MyClass2
{
    public static MyClass MyMethod()   // CS0050
    {
        return new MyClass();
    }

    public static void Main() { }
}
```
