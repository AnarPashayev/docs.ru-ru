# <a name="compiler-warning-level-1-cs0420"></a><span data-ttu-id="1c5e2-103">Предупреждение компилятора (уровень 1) CS0420</span><span class="sxs-lookup"><span data-stu-id="1c5e2-103">Compiler Warning (level 1) CS0420</span></span>

<span data-ttu-id="1c5e2-104">"идентификатор": ссылка на поле с модификатором volatile не будет считаться изменяемой</span><span class="sxs-lookup"><span data-stu-id="1c5e2-104">'identifier': a reference to a volatile field will not be treated as volatile</span></span>  
  
 <span data-ttu-id="1c5e2-105">Как правило, поле с модификатором volatile не следует передавать с помощью параметров `ref` или **out**, поскольку оно не будет обрабатываться как изменяемое в области действия функции.</span><span class="sxs-lookup"><span data-stu-id="1c5e2-105">A volatile field should not normally be passed using a `ref` or **out** parameter, since it will not be treated as volatile within the scope of the function.</span></span> <span data-ttu-id="1c5e2-106">Существуют исключения из такой реализации, например при вызове заблокированного API.</span><span class="sxs-lookup"><span data-stu-id="1c5e2-106">There are exceptions to this, such as when calling an interlocked API.</span></span> <span data-ttu-id="1c5e2-107">Как и в случае с любым другим предупреждением, можно использовать [#pragma warning](../preprocessor-directives/preprocessor-pragma-warning.md) для отключения этого предупреждения в тех редких случаях, когда поле с модификатором volatile намеренно используется в качестве ссылочного параметра.</span><span class="sxs-lookup"><span data-stu-id="1c5e2-107">As with any warning, you may use the [#pragma warning](../preprocessor-directives/preprocessor-pragma-warning.md) to disable this warning in those rare cases where you are intentionally using a volatile field as a reference parameter.</span></span>  
  
 <span data-ttu-id="1c5e2-108">В следующем примере возникает предупреждение CS0420:</span><span class="sxs-lookup"><span data-stu-id="1c5e2-108">The following sample generates CS0420:</span></span>  
  
```csharp  
// CS0420.cs  
// compile with: /W:1  
using System;  
  
class TestClass  
{  
   private volatile int i;  
  
   public void TestVolatile(ref int ii)  
   {  
   }  
  
   public static void Main()  
   {  
      TestClass x = new TestClass();  
      x.TestVolatile(ref x.i);   // CS0420
   }  
}  
```
