# <a name="compiler-error-cs1579"></a><span data-ttu-id="38019-102">Ошибка компилятора CS1579</span><span class="sxs-lookup"><span data-stu-id="38019-102">Compiler Error CS1579</span></span>
<span data-ttu-id="38019-103">Оператор foreach не работает с переменными типа "тип1", так как "тип2" не содержит открытого определения для "идентификатор"</span><span class="sxs-lookup"><span data-stu-id="38019-103">foreach statement cannot operate on variables of type 'type1' because 'type2' does not contain a public definition for 'identifier'</span></span>

<span data-ttu-id="38019-104">Чтобы иметь возможность выполнить итерацию коллекции с помощью оператора [foreach](../keywords/foreach-in.md), коллекция должна соответствовать следующим требованиям:</span><span class="sxs-lookup"><span data-stu-id="38019-104">To iterate through a collection using the [foreach](../keywords/foreach-in.md) statement, the collection must meet the following requirements:</span></span>

- <span data-ttu-id="38019-105">Его тип должен включать открытый метод `GetEnumerator` без параметров со следующим типом возвращаемого значения: класс, структура или тип интерфейса.</span><span class="sxs-lookup"><span data-stu-id="38019-105">Its type must include a public parameterless `GetEnumerator` method whose return type is either class, struct, or interface type.</span></span>
- <span data-ttu-id="38019-106">Тип возвращаемого значения метода `GetEnumerator` должен содержать открытое свойство `Current` и открытый метод `MoveNext` без параметров с типом возвращаемого значения <xref:System.Boolean>.</span><span class="sxs-lookup"><span data-stu-id="38019-106">The return type of the `GetEnumerator` method must contain a public property named `Current` and a public parameterless method named `MoveNext` whose return type is <xref:System.Boolean>.</span></span>

## <a name="example"></a><span data-ttu-id="38019-107">Пример</span><span class="sxs-lookup"><span data-stu-id="38019-107">Example</span></span>

<span data-ttu-id="38019-108">Следующий пример приводит к ошибке CS1579, так как класс `MyCollection` не содержит открытый метод `GetEnumerator`:</span><span class="sxs-lookup"><span data-stu-id="38019-108">The following sample generates CS1579 because the `MyCollection` class doesn't contain the public `GetEnumerator` method:</span></span>

```csharp  
// CS1579.cs  
using System;  
public class MyCollection
{  
   int[] items;  
   public MyCollection()
   {  
      items = new int[5] {12, 44, 33, 2, 50};  
   }  
  
   // Delete the following line to resolve.  
   MyEnumerator GetEnumerator()  
  
   // Uncomment the following line to resolve:  
   // public MyEnumerator GetEnumerator()
   {  
      return new MyEnumerator(this);  
   }  
  
   // Declare the enumerator class:  
   public class MyEnumerator
   {  
      int nIndex;  
      MyCollection collection;  
      public MyEnumerator(MyCollection coll)
      {  
         collection = coll;  
         nIndex = -1;  
      }  
  
      public bool MoveNext()
      {  
         nIndex++;  
         return (nIndex < collection.items.Length);  
      }  
  
      public int Current => collection.items[nIndex];
   }  
  
   public static void Main()
   {  
      MyCollection col = new MyCollection();  
      Console.WriteLine("Values in the collection are:");  
      foreach (int i in col)   // CS1579  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
