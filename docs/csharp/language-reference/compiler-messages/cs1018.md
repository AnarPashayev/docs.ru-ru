## <a name="example"></a><span data-ttu-id="06fcd-106">Пример</span><span class="sxs-lookup"><span data-stu-id="06fcd-106">Example</span></span>  

 <span data-ttu-id="06fcd-107">В следующем примере возникает ошибка CS1018 и предлагается несколько способов ее устранения:</span><span class="sxs-lookup"><span data-stu-id="06fcd-107">The following example generates CS1018, and suggests several ways to resolve the error:</span></span>  
  
```csharp  
// CS1018.cs  
public class C  
{  
}  
  
public class a : C  
{  
    public a(int i)  
    {  
    }  
  
    public a () :   // CS1018  
    // possible resolutions:  
    // public a () resolves by removing the colon  
    // public a () : base() calls C's parameterless constructor  
    // public a () : this(1) calls the assignment constructor of class a  
    {  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
