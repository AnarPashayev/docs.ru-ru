 <span data-ttu-id="43763-105">Если [оператор switch](../keywords/switch.md) содержит более одного раздела переключения, необходимо явно завершить все разделы, включая последний, используя одно из следующих ключевых слов:</span><span class="sxs-lookup"><span data-stu-id="43763-105">When a [switch statement](../keywords/switch.md) contains more than one switch section, you must explicitly terminate each section, including the last one, by using one of the following keywords:</span></span>  
  
- [<span data-ttu-id="43763-106">return</span><span class="sxs-lookup"><span data-stu-id="43763-106">return</span></span>](../keywords/return.md)  
  
- [<span data-ttu-id="43763-107">goto</span><span class="sxs-lookup"><span data-stu-id="43763-107">goto</span></span>](../keywords/goto.md)  
  
- [<span data-ttu-id="43763-108">break</span><span class="sxs-lookup"><span data-stu-id="43763-108">break</span></span>](../keywords/break.md)

 <span data-ttu-id="43763-111">Если необходимо реализовать поведение "проваливания" из одного раздела в следующий, используйте выражение `goto case #`.</span><span class="sxs-lookup"><span data-stu-id="43763-111">If you want to implement "fall through" behavior from one section to the next, use `goto case #`.</span></span> <span data-ttu-id="43763-112">Дополнительные сведения и примеры см. в разделе [switch](../keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="43763-112">For more information and examples, see [switch](../keywords/switch.md).</span></span>  
  
 <span data-ttu-id="43763-113">Следующий пример приводит к возникновению ошибки CS0163.</span><span class="sxs-lookup"><span data-stu-id="43763-113">The following sample generates CS0163.</span></span>  
  
```csharp  
// CS0163.cs  
public class MyClass  
{  
    public static void Main()  
    {  
        int i = 0;  
  
        switch (i)   // CS0163  
        {  
            // Compiler error CS0163 is reported on the following line.  
            case 1:  
                i++;  
            // To resolve the error, uncomment one of the following example statements.  
            // return;  
            // break;  
            // goto case 3;  
  
            case 2:  
                i++;  
                return;  
  
            case 3:  
                i = 0;  
                return;  
  
            // Compiler error CS0163 is reported on the following line.  
            default:  
                Console.WriteLine("Default");  
                // To resolve the error, uncomment the following line:  
            //break;  
        }  
    }  
```
