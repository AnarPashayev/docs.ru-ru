# <a name="compiler-error-cs1519"></a><span data-ttu-id="fbd8a-103">Ошибка компилятора CS1519</span><span class="sxs-lookup"><span data-stu-id="fbd8a-103">Compiler Error CS1519</span></span>

<span data-ttu-id="fbd8a-104">Недопустимый токен "token" в объявлении класса, структуры или члена интерфейса</span><span class="sxs-lookup"><span data-stu-id="fbd8a-104">Invalid token 'token' in class, struct, or interface member declaration</span></span>  
  
 <span data-ttu-id="fbd8a-105">Эта ошибка возникает при обнаружении токена там, где его быть не должно.</span><span class="sxs-lookup"><span data-stu-id="fbd8a-105">This error is generated whenever a token is encountered in a location where it does not belong.</span></span> <span data-ttu-id="fbd8a-106">*token* — это ключевое слово; идентификатор (имя класса, структуры, метода и т. д); строка, символ или числовое значение литерала, например 108, "Hello" или ";" либо оператор или знак препинания, такой как `==` или `;`.</span><span class="sxs-lookup"><span data-stu-id="fbd8a-106">A *token* is a keyword; an identifier (the name of a class, struct, method, and so on); a string, character, or numeric literal value such as 108, "Hello", or 'A'; or an operator or punctuator such as `==` or `;`.</span></span>  
  
 <span data-ttu-id="fbd8a-107">Любое объявление [класса](../keywords/class.md), структуры или члена интерфейса, содержащее недопустимые модификаторы перед типом, будет вызывать такую ошибку.</span><span class="sxs-lookup"><span data-stu-id="fbd8a-107">Any [class](../keywords/class.md), struct, or interface member declaration that contains invalid modifiers before the type will generate this error.</span></span> <span data-ttu-id="fbd8a-108">Чтобы устранить эту ошибку, удалите недопустимые модификаторы.</span><span class="sxs-lookup"><span data-stu-id="fbd8a-108">To fix the error, remove the invalid modifiers.</span></span>  
  
 <span data-ttu-id="fbd8a-109">В следующем примере ошибка CS1519 возникает в пяти местах, поскольку токены находятся там, где они недопустимы:</span><span class="sxs-lookup"><span data-stu-id="fbd8a-109">The following sample generates CS1519 in five places because tokens are placed in locations where they are not valid:</span></span>  
  
```csharp  
// CS1519.cs  
// Generates CS1519 because a class name cannot be a number:  
class Test 42
{  
// Generates CS1519 because of 'j' following 'I'  
// with no comma between them:  
    int i j;
// Generates CS1519 because of "checked" on void method:  
    checked void f4();
  
// Generates CS1519 because of "num":  
    void f5(int a num){}
  
// Generates CS1519 because of namespace inside class:  
    namespace;
  
}  
```
