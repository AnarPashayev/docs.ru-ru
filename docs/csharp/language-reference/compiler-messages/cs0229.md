# <a name="compiler-error-cs0229"></a>Ошибка компилятора CS0229
Неоднозначность между "член1" и "член2"  
  
 Члены разных интерфейсов имеют одинаковые имена. Если вы хотите использовать те же имена, необходимо определить их. Дополнительные сведения см. в статье [Интерфейсы](../../programming-guide/interfaces/index.md).  
  
> [!NOTE]
> В некоторых случаях неоднозначность можно устранить, явно предоставив префикс интерфейса с помощью псевдонима [using](../keywords/using-directive.md).  
  
## <a name="example"></a>Пример  
 В следующем примере возникает ошибка CS0229:  
  
```csharp  
// CS0229.cs  
  
interface IList  
{  
    int Count  
    {  
        get;  
        set;  
    }  
  
    void Counter();  
}  
  
interface ICounter  
{  
    double Count  
    {  
        get;  
        set;  
    }  
}  
  
interface IListCounter : IList, ICounter {}  
  
class MyClass  
{  
    void Test(IListCounter x)  
    {  
        x.Count = 1;  // CS0229  
        // Try one of the following lines instead:  
        // ((IList)x).Count = 1;  
        // or  
        // ((ICounter)x).Count = 1;  
    }  
  
    public static void Main() {}  
}  
```
