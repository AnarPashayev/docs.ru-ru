 <span data-ttu-id="01b48-104">Эта ошибка возникает, если вы указали для параметра [-langversion](../compiler-options/langversion-compiler-option.md) значение ISO-1, но в компилируемом коде при этом используются возможности, не поддерживаемые стандартом ISO 1.0.</span><span class="sxs-lookup"><span data-stu-id="01b48-104">This error occurs if you specified the [-langversion](../compiler-options/langversion-compiler-option.md) option ISO-1 and the code you are compiling uses features that are not part of the ISO 1.0 standard.</span></span> <span data-ttu-id="01b48-105">Чтобы устранить эту ошибку, при выборе параметра совместимости ISO-1 не используйте возможности компилятора C# 2.0, такие как универсальные шаблоны или анонимные методы.</span><span class="sxs-lookup"><span data-stu-id="01b48-105">To resolve this error, do not use any of the C# 2.0 compiler features such as generics or anonymous methods with the ISO-1 compatibility option.</span></span>
