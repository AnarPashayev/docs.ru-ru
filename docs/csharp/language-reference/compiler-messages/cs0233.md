# <a name="compiler-error-cs0233"></a><span data-ttu-id="f588a-103">Ошибка компилятора CS0233</span><span class="sxs-lookup"><span data-stu-id="f588a-103">Compiler Error CS0233</span></span>

<span data-ttu-id="f588a-104">"Идентификатор" не имеет предопределенного размера, поэтому оператор sizeof может использоваться только в небезопасном контексте.</span><span class="sxs-lookup"><span data-stu-id="f588a-104">'identifier' does not have a predefined size, therefore sizeof can only be used in an unsafe context</span></span>
  
 <span data-ttu-id="f588a-105">Вне [небезопасного](../keywords/unsafe.md) контекста оператор [sizeof](../operators/sizeof.md) может применяться только для типов, размер которых является константой, вычисляемой во время компиляции.</span><span class="sxs-lookup"><span data-stu-id="f588a-105">Without [unsafe](../keywords/unsafe.md) context, the [sizeof](../operators/sizeof.md) operator can only be used for types which size is a compile-time constant.</span></span> <span data-ttu-id="f588a-106">Если возникает эта ошибка, следует использовать небезопасный контекст.</span><span class="sxs-lookup"><span data-stu-id="f588a-106">If you are getting this error, use an unsafe context.</span></span>
  
<span data-ttu-id="f588a-107">В следующем примере возникает ошибка CS0233:</span><span class="sxs-lookup"><span data-stu-id="f588a-107">The following example generates CS0233:</span></span>
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following instead:  
        // unsafe
        // {
        //     Console.WriteLine(sizeof(S));
        // }
   }  
}  
```
