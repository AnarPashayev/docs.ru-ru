<span data-ttu-id="676d2-103">"identifier": не удается задать аргументы при создании экземпляра типа переменной</span><span class="sxs-lookup"><span data-stu-id="676d2-103">'identifier': cannot provide arguments when creating an instance of a variable type</span></span>  
  
 <span data-ttu-id="676d2-104">Эта ошибка возникает при наличии аргументов у вызова оператора `new` в параметре-типе.</span><span class="sxs-lookup"><span data-stu-id="676d2-104">This error occurs if a call to the `new` operator on a type parameter has arguments.</span></span> <span data-ttu-id="676d2-105">Единственным конструктором, который можно вызвать с помощью оператора `new` в неизвестном параметре-типе, является конструктор без аргументов.</span><span class="sxs-lookup"><span data-stu-id="676d2-105">The only constructor that can be called by using the `new` operator on an unknown parameter type is a constructor that has no arguments.</span></span> <span data-ttu-id="676d2-106">Если необходимо вызвать другой конструктор, следует рассмотреть возможность использования ограничения типа класса или ограничения интерфейса.</span><span class="sxs-lookup"><span data-stu-id="676d2-106">If you need to call another constructor, consider using a class type constraint or interface constraint.</span></span>  
  
## <a name="example"></a><span data-ttu-id="676d2-107">Пример</span><span class="sxs-lookup"><span data-stu-id="676d2-107">Example</span></span>  

 <span data-ttu-id="676d2-108">Следующий пример приводит к возникновению ошибки CS0417:</span><span class="sxs-lookup"><span data-stu-id="676d2-108">The following example generates CS0417:</span></span>  

```csharp

    // The following line causes CS0417.  
    T instance1 = new T(1);
  
    // The following line doesn't cause the error.  
    T instance2 = new T();  
}  
```
