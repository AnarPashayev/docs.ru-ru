## <a name="example"></a><span data-ttu-id="dd427-109">Пример</span><span class="sxs-lookup"><span data-stu-id="dd427-109">Example</span></span>  

 <span data-ttu-id="dd427-110">В следующем примере показано определение настраиваемого атрибута:</span><span class="sxs-lookup"><span data-stu-id="dd427-110">The following sample shows how you might define an attribute:</span></span>  
  
```csharp  
// CreateAttrib.cs  
// compile with: /target:library  
using System;  
  
[AttributeUsage(AttributeTargets.Class|AttributeTargets.Interface)]  
public class MyAttr : Attribute  
{  
   public int Name = 0;  
   public int Count = 0;  
  
   public MyAttr (int iCount, int sName)  
   {  
      Count = iCount;  
      Name = sName;  
   }  
}  
  
[MyAttr(5, 50)]  
class Class1 {}  
  
[MyAttr(6, 60)]  
interface Interface1 {}  
```
