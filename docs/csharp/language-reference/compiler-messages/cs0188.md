# <a name="compiler-error-cs0188"></a><span data-ttu-id="ca9e7-102">Ошибка компилятора CS0188</span><span class="sxs-lookup"><span data-stu-id="ca9e7-102">Compiler Error CS0188</span></span>

<span data-ttu-id="ca9e7-103">Невозможно использовать объект "this", пока не будут назначены все его поля</span><span class="sxs-lookup"><span data-stu-id="ca9e7-103">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="ca9e7-104">Все поля в `struct` должны быть присвоены конструктором до того, как конструктор сможет вызвать метод в `struct`.</span><span class="sxs-lookup"><span data-stu-id="ca9e7-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="ca9e7-105">Если эта ошибка появляется при попытке инициализировать свойство в конструкторе структуры, инициализируйте резервное поле этого свойства напрямую вместо самого свойства.</span><span class="sxs-lookup"><span data-stu-id="ca9e7-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to initialize the backing field of that property directly instead of the property itself.</span></span>

## <a name="example"></a><span data-ttu-id="ca9e7-106">Пример</span><span class="sxs-lookup"><span data-stu-id="ca9e7-106">Example</span></span>

 <span data-ttu-id="ca9e7-107">В следующем примере возникает ошибка CS0188:</span><span class="sxs-lookup"><span data-stu-id="ca9e7-107">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
