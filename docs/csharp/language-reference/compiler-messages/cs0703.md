# <a name="compiler-error-cs0703"></a><span data-ttu-id="d4a0c-102">Ошибка компилятора CS0703</span><span class="sxs-lookup"><span data-stu-id="d4a0c-102">Compiler Error CS0703</span></span>
<span data-ttu-id="d4a0c-103">Несогласованность по доступности: доступность типа ограничения "идентификатор" ниже доступности "идентификатор"</span><span class="sxs-lookup"><span data-stu-id="d4a0c-103">Inconsistent accessibility: constraint type 'identifier' is less accessible than 'identifier'</span></span>  
  
 <span data-ttu-id="d4a0c-104">Ограничение не может принудительно задавать уровень доступности для универсального параметра ниже, чем у самого универсального класса.</span><span class="sxs-lookup"><span data-stu-id="d4a0c-104">A constraint may not force the generic parameter to be less accessible than the generic class itself.</span></span> <span data-ttu-id="d4a0c-105">В следующем примере общий класс C\<T> объявлен открытым, и ограничение пытается вынудить T реализовать внутренний интерфейс.</span><span class="sxs-lookup"><span data-stu-id="d4a0c-105">In the following example, while the generic class C\<T> is declared public, the constraint attempts to force T to implement an internal interface.</span></span> <span data-ttu-id="d4a0c-106">Даже если это разрешено, создавать параметры этого класса и фактически использовать его смогут только клиенты с внутренним доступом.</span><span class="sxs-lookup"><span data-stu-id="d4a0c-106">Even if this were allowed, only clients with internal access would be able to create the parameter for the class, so in effect, the class could be used only by clients with internal access.</span></span>  
  
 <span data-ttu-id="d4a0c-107">Чтобы устранить эту ошибку, убедитесь, что уровень доступа универсального класса задан не менее строгим, чем уровень доступа для любых классов или интерфейсов, которые появляются в пределах его границ.</span><span class="sxs-lookup"><span data-stu-id="d4a0c-107">To get rid of this error, make sure the access level of the generic class is not less restrictive than any classes or interfaces that appear in the bounds.</span></span>  
  
 <span data-ttu-id="d4a0c-108">В следующем примере возникает ошибка CS0703:</span><span class="sxs-lookup"><span data-stu-id="d4a0c-108">The following sample generates CS0703:</span></span>  
  
```csharp  
// CS0703.cs  
internal interface I {}  
public class C<T> where T : I  // CS0703 – I is internal; C<T> is public  
{  
}  
```
