# <a name="compiler-warning-level-1-cs3007"></a><span data-ttu-id="1685e-103">Предупреждение компилятора (уровень 1) CS3007</span><span class="sxs-lookup"><span data-stu-id="1685e-103">Compiler Warning (level 1) CS3007</span></span>

<span data-ttu-id="1685e-104">Переопределенный метод "метод", отличающийся только типами массивов без имен, несовместим с CLS</span><span class="sxs-lookup"><span data-stu-id="1685e-104">Overloaded method 'method' differing only by unnamed array types is not CLS-compliant</span></span>  
  
 <span data-ttu-id="1685e-105">Эта ошибка возникает, если имеется перегруженный метод, который принимает массив массивов, и единственное различие между сигнатурами методов заключается в типе элементов массива.</span><span class="sxs-lookup"><span data-stu-id="1685e-105">This error occurs if you have an overloaded method that takes a jagged array and the only difference between the method signatures is the element type of the array.</span></span> <span data-ttu-id="1685e-106">Чтобы избежать возникновения этой ошибки, рекомендуется выполнить следующие действия: использовать не массив массивов, а прямоугольный массив; использовать дополнительный параметр для устранения неоднозначности при вызове функции; переименовать один из перегруженных методов; удалить атрибут <xref:System.CLSCompliantAttribute> (если совместимость со спецификацией CLS не требуется).</span><span class="sxs-lookup"><span data-stu-id="1685e-106">To avoid this error, consider using a rectangular array rather than a jagged array; use an additional parameter to disambiguate the function call; rename one or more of the overloaded methods; or, if CLS Compliance is not needed, remove the <xref:System.CLSCompliantAttribute> attribute.</span></span> <span data-ttu-id="1685e-107">Дополнительные сведения о соответствии спецификации CLS см. в разделе [Независимость от языка и независимые от языка компоненты](../../../standard/language-independence-and-language-independent-components.md).</span><span class="sxs-lookup"><span data-stu-id="1685e-107">For more information on CLS Compliance, see [Language Independence and Language-Independent Components](../../../standard/language-independence-and-language-independent-components.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="1685e-108">Пример</span><span class="sxs-lookup"><span data-stu-id="1685e-108">Example</span></span>  

 <span data-ttu-id="1685e-109">Следующий пример приводит к возникновению предупреждения CS3007:</span><span class="sxs-lookup"><span data-stu-id="1685e-109">The following example generates CS3007:</span></span>  
  
```csharp  
// CS3007.cs  
[assembly: System.CLSCompliant(true)]  
public struct S  
{  
    public void F(int[][] array) { }  
    public void F(byte[][] array) { }  // CS3007  
    // Try this instead:  
    // public void F1(int[][] array) {}  
    // public void F2(byte[][] array) {}  
    // or
    // public void F(int[,] array) {}  
    // public void F(byte[,] array) {}  
}  
```
