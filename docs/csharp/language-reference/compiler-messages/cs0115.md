## <a name="example"></a><span data-ttu-id="f6e8d-106">Пример</span><span class="sxs-lookup"><span data-stu-id="f6e8d-106">Example</span></span>  
 <span data-ttu-id="f6e8d-107">В следующем примере возникает ошибка CS0115.</span><span class="sxs-lookup"><span data-stu-id="f6e8d-107">The following sample generates CS0115.</span></span> <span data-ttu-id="f6e8d-108">Ошибку CS0115 можно разрешить двумя способами:</span><span class="sxs-lookup"><span data-stu-id="f6e8d-108">You can resolve CS0115 in one of two ways:</span></span>  
  
- <span data-ttu-id="f6e8d-109">Удалить ключевое слово `override` из метода в `MyClass2`.</span><span class="sxs-lookup"><span data-stu-id="f6e8d-109">Remove the `override` keyword from the method in `MyClass2`.</span></span>  
  
- <span data-ttu-id="f6e8d-110">Использовать `MyClass1` в качестве базового класса для `MyClass2`.</span><span class="sxs-lookup"><span data-stu-id="f6e8d-110">Use `MyClass1` as a base class for `MyClass2`.</span></span>  
  
```csharp  
// CS0115.cs  
namespace MyNamespace  
{  
    abstract public class MyClass1  
    {  
        public abstract int f();  
    }  
  
    abstract public class MyClass2  
    {  
        public override int f()   // CS0115  
        {  
            return 0;  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
