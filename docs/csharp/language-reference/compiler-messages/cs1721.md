## <a name="example"></a><span data-ttu-id="17de4-107">Пример</span><span class="sxs-lookup"><span data-stu-id="17de4-107">Example</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="17de4-109">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="17de4-109">To correct this error</span></span>

<span data-ttu-id="17de4-110">Ниже описываются различные способы исправления этой ошибки.</span><span class="sxs-lookup"><span data-stu-id="17de4-110">The following are different ways to correct this error:</span></span>

    ```csharp
    public class MyClass : A, B {}
    ```
