## <a name="to-correct-this-error"></a><span data-ttu-id="9c42f-110">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="9c42f-110">To correct this error</span></span>

<span data-ttu-id="9c42f-111">Ниже описываются различные способы исправления этой ошибки.</span><span class="sxs-lookup"><span data-stu-id="9c42f-111">The following are different ways to correct this error:</span></span>

    ```csharp

    ```

    ```csharp
    public interface B {}
    public class MyClass : A, B {}
    ```
