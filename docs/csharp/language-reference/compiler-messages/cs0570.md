<span data-ttu-id="3bf8b-103">Свойство, индексатор или событие "имя" не поддерживается в данном языке; попробуйте напрямую вызвать метод доступа "имя!"</span><span class="sxs-lookup"><span data-stu-id="3bf8b-103">Property, indexer, or event 'name' is not supported by the language; try directly calling accessor method 'name!'</span></span>  
  
 <span data-ttu-id="3bf8b-104">Эта ошибка возникает в том случае, если импортированные метаданные были созданы другим компилятором.</span><span class="sxs-lookup"><span data-stu-id="3bf8b-104">This error occurs when using imported metadata that was generated by another compiler.</span></span> <span data-ttu-id="3bf8b-105">В коде предпринимается попытка использовать член класса, который не может быть обработан компилятором.</span><span class="sxs-lookup"><span data-stu-id="3bf8b-105">Your code attempted to use a class member that the compiler cannot process.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3bf8b-106">Пример</span><span class="sxs-lookup"><span data-stu-id="3bf8b-106">Example</span></span>  
 <span data-ttu-id="3bf8b-107">В следующей программе C++ используется атрибут RequiredAttributeAttribute, который может не поддерживаться другими языками.</span><span class="sxs-lookup"><span data-stu-id="3bf8b-107">The following C++ program uses an attribute, RequiredAttributeAttribute, which may not be consumed by other languages.</span></span>  
  
```cpp
// CPP0570.cpp  
// compile with: /clr /LD  
  
using namespace System;  
using namespace System::Runtime::CompilerServices;  
  
namespace CS0570_Server {  
   [RequiredAttributeAttribute(Int32::typeid)]
   public ref struct Scenario1 {  
      int intVar;  
   };  
  
   public ref struct CS0570Class {  
      Scenario1 ^ sc1_field;  
  
      property virtual Scenario1 ^ sc1_prop {  
         Scenario1 ^ get() { return sc1_field; }  
      }  
  
      Scenario1 ^ sc1_method() { return sc1_field; }  
   };  
};  
```  
  
## <a name="example"></a><span data-ttu-id="3bf8b-108">Пример</span><span class="sxs-lookup"><span data-stu-id="3bf8b-108">Example</span></span>  
 <span data-ttu-id="3bf8b-109">В следующем примере возникает ошибка CS0570.</span><span class="sxs-lookup"><span data-stu-id="3bf8b-109">The following sample generates CS0570.</span></span>

```csharp  
// CS0570.cs

using System;  
using CS0570_Server;

      CS0570Class r = new CS0570Class();  
      r.sc1_field = null;   // CS0570  
      object o = r.sc1_prop;   // CS0570  
      r.sc1_method();   // CS0570  
   }  
}  
```
