# <a name="compiler-error-cs1019"></a><span data-ttu-id="eaabc-103">Ошибка компилятора CS1019</span><span class="sxs-lookup"><span data-stu-id="eaabc-103">Compiler Error CS1019</span></span>

<span data-ttu-id="eaabc-104">Ожидается перегружаемый унарный оператор</span><span class="sxs-lookup"><span data-stu-id="eaabc-104">Overloadable unary operator expected</span></span>  
  
 <span data-ttu-id="eaabc-105">Было объявлено нечто, похожее на перегруженный унарный оператор, однако в сигнатуре оператор отсутствует или расположен неверно.</span><span class="sxs-lookup"><span data-stu-id="eaabc-105">Something that looks like an overloaded unary operator has been declared, but the operator is missing or is in the wrong location in the signature.</span></span>  
  
 <span data-ttu-id="eaabc-106">*Унарный оператор* — это оператор, применяемый к одному операнду.</span><span class="sxs-lookup"><span data-stu-id="eaabc-106">A *unary operator* is an operator that operates on a single operand.</span></span> <span data-ttu-id="eaabc-107">Например, `++` является унарным оператором.</span><span class="sxs-lookup"><span data-stu-id="eaabc-107">For example, `++` is a unary operator.</span></span> <span data-ttu-id="eaabc-108">Некоторые унарные операторы можно перегрузить с помощью ключевого слова `operator`, указав при этом один параметр типа, к которому будет применяться данный оператор.</span><span class="sxs-lookup"><span data-stu-id="eaabc-108">You can overload some unary operators by using the `operator` keyword and specifying a single parameter of the type that the operator operates on.</span></span> <span data-ttu-id="eaabc-109">Например, если требуется перегрузить оператор `++` для определяемого пользователем класса `Temperature`, чтобы написать `Temperature++`, можно воспользоваться следующим кодом.</span><span class="sxs-lookup"><span data-stu-id="eaabc-109">For example, if you want to overload the operator `++` for a user-defined class `Temperature` so that you can write `Temperature++`, you can define it in this way:</span></span>  
  
```csharp  
public static  Temperature operator ++ (Temperature temp)  
{  
    temp.Degrees++;  
    return temp;  
}  
```  
  
 <span data-ttu-id="eaabc-110">Если возникла такая ошибка, это означает, что было объявлено нечто, похожее на перегруженный унарный оператор, однако сам оператор в сигнатуре отсутствует или расположен неверно.</span><span class="sxs-lookup"><span data-stu-id="eaabc-110">When you receive this error, you have declared something that looks like an overloaded unary operator, except that the operator itself is missing or is in the wrong location in the signature.</span></span> <span data-ttu-id="eaabc-111">Если в предыдущем примере удалить из сигнатуры оператор `++`, будет получена ошибка CS1019.</span><span class="sxs-lookup"><span data-stu-id="eaabc-111">If you remove the `++` from the signature in the previous example, you will generate CS1019.</span></span>  
  
 <span data-ttu-id="eaabc-112">Следующий код приводит к возникновению ошибки CS1019:</span><span class="sxs-lookup"><span data-stu-id="eaabc-112">The following code generates CS1019:</span></span>  
  
```csharp  
// CS1019.cs  
public class ii  
{  
   int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
}  
  
public class a  
{  
    public int i;  
// Generates CS1019: "ii" is not a unary operator.  
   public static a operator ii(a aa)
  
   // Use the following line instead:  
   //public static a operator ++(a aa)  
   {  
      aa.i++;  
      return aa;
   }  
  
   public static void Main()  
   {  
   }  
}  
```
