## <a name="example"></a><span data-ttu-id="ae7de-107">Пример</span><span class="sxs-lookup"><span data-stu-id="ae7de-107">Example</span></span>  
 <span data-ttu-id="ae7de-108">В следующем примере возникает ошибка CS0071.</span><span class="sxs-lookup"><span data-stu-id="ae7de-108">The following sample generates CS0071.</span></span>  
  
```csharp  
// CS0071.cs  
public delegate void MyEvent(object sender);  
  
interface ITest  
{  
    event MyEvent Clicked;  
}  
  
class Test : Itest  
{  
    event MyEvent ITest.Clicked;  // CS0071  
  
    // try the following code instead  
/*  
private MyEvent clicked;  
  
    event MyEvent Itest.Clicked  
    {  
        add  
        {  
            clicked += value;  
        }  
        remove  
        {  
            clicked -= value;  
        }  
    }  
*/  
    public static void Main() { }  
}  
```
