# <a name="compiler-error-cs0071"></a><span data-ttu-id="69a50-102">Ошибка компилятора CS0071</span><span class="sxs-lookup"><span data-stu-id="69a50-102">Compiler Error CS0071</span></span>

<span data-ttu-id="69a50-103">Явная реализация интерфейса события должна использовать синтаксис функции доступа к событию</span><span class="sxs-lookup"><span data-stu-id="69a50-103">An explicit interface implementation of an event must use event accessor syntax</span></span>

<span data-ttu-id="69a50-104">При явной реализации [события](../keywords/event.md), которое было объявлено как интерфейс, необходимо вручную предоставить методы доступа к событиям `add` и `remove`, которые обычно предоставляются компилятором.</span><span class="sxs-lookup"><span data-stu-id="69a50-104">When explicitly implementing an [event](../keywords/event.md) that was declared in an interface, you must manually provide the `add` and `remove` event accessors that are typically provided by the compiler.</span></span> <span data-ttu-id="69a50-105">Код метода доступа может связывать событие интерфейса с другим событием в классе (см. далее в этом разделе) или с его собственным типом делегата.</span><span class="sxs-lookup"><span data-stu-id="69a50-105">The accessor code can connect the interface event to another event in your class (shown later in this topic) or to its own delegate type.</span></span> <span data-ttu-id="69a50-106">Дополнительные сведения см. в статье [Как реализовать события интерфейса (руководство по программированию на C#)](../../programming-guide/events/how-to-implement-interface-events.md).</span><span class="sxs-lookup"><span data-stu-id="69a50-106">For more information, see [How to implement interface events](../../programming-guide/events/how-to-implement-interface-events.md).</span></span>
  
## <a name="example"></a><span data-ttu-id="69a50-107">Пример</span><span class="sxs-lookup"><span data-stu-id="69a50-107">Example</span></span>

 <span data-ttu-id="69a50-108">В следующем примере возникает ошибка CS0071.</span><span class="sxs-lookup"><span data-stu-id="69a50-108">The following sample generates CS0071.</span></span>

```csharp
// CS0071.cs
public delegate void MyEvent(object sender);

interface ITest
{
    event MyEvent Clicked;
}

class Test : ITest
{
    event MyEvent ITest.Clicked;  // CS0071

    // Try the following code instead.
    /*
    private MyEvent clicked;

    event MyEvent ITest.Clicked
    {
        add
        {
            clicked += value;
        }
        remove
        {
            clicked -= value;
        }
    }
    */
    public static void Main() { }
}
```
