# <a name="compiler-error-cs0006"></a>Ошибка компилятора CS0006

Не удалось найти файл метаданных "dll_name"
  
 Программа была скомпилирована и явно передала имя файла с метаданными, однако DLL-файл не был найден. См. сведения в разделе [-reference (параметры компилятора C#)](../compiler-options/reference-compiler-option.md).
