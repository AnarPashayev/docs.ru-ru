# <a name="compiler-error-cs0840"></a><span data-ttu-id="ce9fd-103">Ошибка компилятора CS0840</span><span class="sxs-lookup"><span data-stu-id="ce9fd-103">Compiler Error CS0840</span></span>

<span data-ttu-id="ce9fd-104">"Имя свойства" должен объявлять тело, поскольку он не помечен как abstract и extern.</span><span class="sxs-lookup"><span data-stu-id="ce9fd-104">'Property name' must declare a body because it is not marked abstract or extern.</span></span> <span data-ttu-id="ce9fd-105">Свойства, реализованные автоматически, должны определять методы доступа как get, так и set.</span><span class="sxs-lookup"><span data-stu-id="ce9fd-105">Automatically implemented properties must define both get and set accessors.</span></span>  
  
 <span data-ttu-id="ce9fd-106">Если обычное свойство не помечено как `abstract` или `extern`, а также если оно не является членом типа `partial`, это свойство должно объявлять тело.</span><span class="sxs-lookup"><span data-stu-id="ce9fd-106">Unless a regular property is marked as `abstract` or `extern`, or is a member of a `partial` type, it must supply a body.</span></span> <span data-ttu-id="ce9fd-107">Автоматически реализуемые свойства не предоставляют тела методов доступа, но при этом должны определять оба метода доступа.</span><span class="sxs-lookup"><span data-stu-id="ce9fd-107">Auto-implemented properties do not provide accessor bodies, but they must specify both accessors.</span></span> <span data-ttu-id="ce9fd-108">Чтобы создать автоматически реализуемое свойство, доступное только для чтения, присвойте методу доступа set модификатор `private`.</span><span class="sxs-lookup"><span data-stu-id="ce9fd-108">To create a read-only auto-implemented property, make the set accessor `private`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ce9fd-109">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="ce9fd-109">To correct this error</span></span>  
  
1. <span data-ttu-id="ce9fd-110">Реализуйте недостающее тело или метод доступа, либо используйте для него и (или) включающего типа модификаторы [abstract](../keywords/abstract.md), [extern](../keywords/extern.md) или [partial (тип)](../keywords/partial-type.md).</span><span class="sxs-lookup"><span data-stu-id="ce9fd-110">Supply the missing body or accessor or else use the [abstract](../keywords/abstract.md), [extern](../keywords/extern.md), or [partial (Type)](../keywords/partial-type.md) modifiers on it and/or its enclosing type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ce9fd-111">Пример</span><span class="sxs-lookup"><span data-stu-id="ce9fd-111">Example</span></span>  

 <span data-ttu-id="ce9fd-112">В следующем примере возникает ошибка CS0840:</span><span class="sxs-lookup"><span data-stu-id="ce9fd-112">The following example generates CS0840:</span></span>  
  
```csharp  
// cs0840.cs  
// Compile with /target:library  
using System;  
class Test  
{  
    public int myProp { get; } // CS0840  
  
    // to create a read-only property  
    // try the following line instead  
    public int myProp2 { get; private set; }  
  
}  
```
