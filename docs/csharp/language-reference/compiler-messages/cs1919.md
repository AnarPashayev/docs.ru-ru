# <a name="compiler-error-cs1919"></a><span data-ttu-id="944c2-103">Ошибка компилятора CS1919</span><span class="sxs-lookup"><span data-stu-id="944c2-103">Compiler Error CS1919</span></span>

<span data-ttu-id="944c2-104">Небезопасный тип "имя типа" не может применяться при создании объекта.</span><span class="sxs-lookup"><span data-stu-id="944c2-104">Unsafe type 'type name' cannot be used in object creation.</span></span>  
  
 <span data-ttu-id="944c2-105">Оператор `new` создает объекты только в управляемой куче.</span><span class="sxs-lookup"><span data-stu-id="944c2-105">The `new` operator creates objects only on the managed heap.</span></span> <span data-ttu-id="944c2-106">Тем не менее объекты можно создавать в неуправляемой памяти косвенно, используя возможности взаимодействия языка для вызова собственных методов, возвращающих указатели.</span><span class="sxs-lookup"><span data-stu-id="944c2-106">However, you can create objects in unmanaged memory indirectly by using the interoperability capabilities of the language to call native methods that return pointers.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="944c2-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="944c2-107">To correct this error</span></span>  
  
1. <span data-ttu-id="944c2-108">Используйте безопасный тип в выражении создания нового объекта.</span><span class="sxs-lookup"><span data-stu-id="944c2-108">Use a safe type in the new object creation expression.</span></span> <span data-ttu-id="944c2-109">Например, используйте `char` или `int` вместо `char*` или `int*`.</span><span class="sxs-lookup"><span data-stu-id="944c2-109">For example, use `char` or `int` instead of `char*` or `int*`.</span></span>  
  
2. <span data-ttu-id="944c2-110">Если необходимо создавать объекты в неуправляемой памяти, используйте метод Win32 или COM либо напишите собственную функцию на языке C или C++ и вызовите ее из C#.</span><span class="sxs-lookup"><span data-stu-id="944c2-110">If you must create objects in unmanaged memory, use a Win32 or COM method or else write your own function in C or C++ and call it from C#.</span></span>  
  
## <a name="example"></a><span data-ttu-id="944c2-111">Пример</span><span class="sxs-lookup"><span data-stu-id="944c2-111">Example</span></span>

 <span data-ttu-id="944c2-112">В следующем примере вызывается ошибка CS1919, поскольку тип указателя небезопасен:</span><span class="sxs-lookup"><span data-stu-id="944c2-112">The following example generates CS1919 because a pointer type is unsafe:</span></span>

```csharp
// cs1919.cs  
// Compile with: /unsafe  
unsafe public class C  
{  
    public static int Main()  
    {  
        var col1 = new int* { }; // CS1919  
        var col2 = new char* { }; // CS1919  
        return 1;  
    }  
}  
```
