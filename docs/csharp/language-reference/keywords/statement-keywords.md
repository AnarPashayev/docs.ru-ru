# <a name="statement-keywords-c-reference"></a>Ключевые слова операторов (Справочник по C#)

Операторы представляют собой инструкции для программы. За исключением случаев, которые описаны в разделах, приведенных в следующей таблице, операторы выполняются последовательно. В следующей таблице приводятся ключевые слова операторов в C#. Дополнительные сведения об операторах, которые не выражаются ключевыми словами, см. в разделе [Операторы](../../programming-guide/statements-expressions-operators/statements.md).
