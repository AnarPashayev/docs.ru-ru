# <a name="floating-point-types-table-c-reference"></a>Таблица типов с плавающей запятой (справочник по C#)

В приведенной ниже таблице представлен точный и приблизительный диапазон значений для типов с плавающей запятой.  
  
|Тип|Приблизительный диапазон значений|Точность|  
|----------|-----------------------|---------------|  
|[float](float.md)|От ±1,5 x 10<sup>−45</sup> до ±3,4 x 10<sup>38</sup>|6–9 цифр|  
|[double](double.md)|от ±5,0 × 10<sup>−324</sup> до ±1,7 × 10<sup>308</sup>|15–17 цифр|
