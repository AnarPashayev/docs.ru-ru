# <a name="default-c-reference"></a>default (справочник по C#)

Ключевое слово `default` можно использовать двумя способами:
- в качестве [инструкции `switch`](switch.md) для указания метки по умолчанию;
- в качестве [выражения значения по умолчанию](../../programming-guide/statements-expressions-operators/default-value-expressions.md) для создания значения типа по умолчанию.
