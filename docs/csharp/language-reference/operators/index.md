## <a name="operator-associativity"></a><span data-ttu-id="a6607-165">Ассоциативность операторов</span><span class="sxs-lookup"><span data-stu-id="a6607-165">Operator associativity</span></span>

<span data-ttu-id="a6607-166">Если операторы имеют одинаковый приоритет, порядок их выполнения определяется ассоциативностью операторов:</span><span class="sxs-lookup"><span data-stu-id="a6607-166">When operators have the same precedence, associativity of the operators determines the order in which the operations are performed:</span></span>
