## <a name="operator-associativity"></a><span data-ttu-id="19db7-167">Ассоциативность операторов</span><span class="sxs-lookup"><span data-stu-id="19db7-167">Operator associativity</span></span>

<span data-ttu-id="19db7-168">Если операторы имеют одинаковый приоритет, порядок их выполнения определяется ассоциативностью операторов:</span><span class="sxs-lookup"><span data-stu-id="19db7-168">When operators have the same precedence, associativity of the operators determines the order in which the operations are performed:</span></span>

- <span data-ttu-id="19db7-169">Операторы с *левой ассоциативностью* вычисляются слева направо.</span><span class="sxs-lookup"><span data-stu-id="19db7-169">*Left-associative* operators are evaluated in order from left to right.</span></span> <span data-ttu-id="19db7-170">За исключением [операторов присваивания](assignment-operator.md) и [оператора объединения со значением NULL](null-coalescing-operator.md), все бинарные операторы имеют левую ассоциативность.</span><span class="sxs-lookup"><span data-stu-id="19db7-170">Except for the [assignment operators](assignment-operator.md) and the [null-coalescing operators](null-coalescing-operator.md), all binary operators are left-associative.</span></span> <span data-ttu-id="19db7-171">Например, выражение `a + b - c` вычисляется как `(a + b) - c`.</span><span class="sxs-lookup"><span data-stu-id="19db7-171">For example, `a + b - c` is evaluated as `(a + b) - c`.</span></span>
