## <a name="example"></a><span data-ttu-id="070b6-104">Пример</span><span class="sxs-lookup"><span data-stu-id="070b6-104">Example</span></span>  
 <span data-ttu-id="070b6-105">Чтобы создать элемент или атрибут, находящийся в пространстве имен, необходимо сначала объявить и инициализировать объект <xref:System.Xml.Linq.XNamespace>.</span><span class="sxs-lookup"><span data-stu-id="070b6-105">To create an element or an attribute that is in a namespace, you first declare and initialize an <xref:System.Xml.Linq.XNamespace> object.</span></span> <span data-ttu-id="070b6-106">Затем следует использовать перегруженный оператор сложения для объединения пространства имен с локальным именем, выраженным строкой.</span><span class="sxs-lookup"><span data-stu-id="070b6-106">You then use the addition operator overload to combine the namespace with the local name, expressed as a string.</span></span>  
  
 <span data-ttu-id="070b6-107">В следующем примере создается документ с одним пространством имен.</span><span class="sxs-lookup"><span data-stu-id="070b6-107">The following example creates a document with one namespace.</span></span> <span data-ttu-id="070b6-108">По умолчанию [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] сериализует документ с использованием пространства имен по умолчанию.</span><span class="sxs-lookup"><span data-stu-id="070b6-108">By default, [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] serializes this document with a default namespace.</span></span>  
  
```csharp  
// Create an XML tree in a namespace.  
XNamespace aw = "http://www.adventure-works.com";  
XElement root = new XElement(aw + "Root",  
    new XElement(aw + "Child", "child content")  
);  
Console.WriteLine(root);  
```  
  
 <span data-ttu-id="070b6-109">В этом примере выводятся следующие данные:</span><span class="sxs-lookup"><span data-stu-id="070b6-109">This example produces the following output:</span></span>  
  
```xml  
<Root xmlns="http://www.adventure-works.com">  
  <Child>child content</Child>  
</Root>  
```  
  
## <a name="example"></a><span data-ttu-id="070b6-110">Пример</span><span class="sxs-lookup"><span data-stu-id="070b6-110">Example</span></span>  
 <span data-ttu-id="070b6-111">В следующем примере создается документ с одним пространством имен.</span><span class="sxs-lookup"><span data-stu-id="070b6-111">The following example creates a document with one namespace.</span></span> <span data-ttu-id="070b6-112">Он также создает атрибут, который объявляет пространство имен с префиксом пространства имен.</span><span class="sxs-lookup"><span data-stu-id="070b6-112">It also creates an attribute that declares the namespace with a namespace prefix.</span></span> <span data-ttu-id="070b6-113">Создать атрибут, объявляющий пространство имен с префиксом, можно, указав имя атрибута в качестве префикса пространства имен и поместив его в пространство имен <xref:System.Xml.Linq.XNamespace.Xmlns%2A>.</span><span class="sxs-lookup"><span data-stu-id="070b6-113">To create an attribute that declares a namespace with a prefix, you create an attribute where the name of the attribute is the namespace prefix, and this name is in the <xref:System.Xml.Linq.XNamespace.Xmlns%2A> namespace.</span></span> <span data-ttu-id="070b6-114">Значение этого атрибута представляет собой URI пространства имен.</span><span class="sxs-lookup"><span data-stu-id="070b6-114">The value of this attribute is the URI of the namespace.</span></span>  
  
```csharp  
// Create an XML tree in a namespace, with a specified prefix  
XNamespace aw = "http://www.adventure-works.com";  
XElement root = new XElement(aw + "Root",  
    new XAttribute(XNamespace.Xmlns + "aw", "http://www.adventure-works.com"),  
    new XElement(aw + "Child", "child content")  
);  
Console.WriteLine(root);  
```  
  
 <span data-ttu-id="070b6-115">В этом примере выводятся следующие данные:</span><span class="sxs-lookup"><span data-stu-id="070b6-115">This example produces the following output:</span></span>  
  
```xml  
<aw:Root xmlns:aw="http://www.adventure-works.com">  
  <aw:Child>child content</aw:Child>  
</aw:Root>  
```  
  
## <a name="example"></a><span data-ttu-id="070b6-116">Пример</span><span class="sxs-lookup"><span data-stu-id="070b6-116">Example</span></span>  
 <span data-ttu-id="070b6-117">Следующий пример иллюстрирует создание документа, содержащего два пространства имен.</span><span class="sxs-lookup"><span data-stu-id="070b6-117">The following example shows the creation of a document that contains two namespaces.</span></span> <span data-ttu-id="070b6-118">Одно из них - пространство имен по умолчанию.</span><span class="sxs-lookup"><span data-stu-id="070b6-118">One is the default namespace.</span></span> <span data-ttu-id="070b6-119">Другое - пространство имен с префиксом.</span><span class="sxs-lookup"><span data-stu-id="070b6-119">Another is a namespace with a prefix.</span></span>  
  
 <span data-ttu-id="070b6-120">При включении атрибутов пространств имен в корневой элемент пространства имен сериализуются, поэтому `http://www.adventure-works.com` становится пространством имен по умолчанию, а `www.fourthcoffee.com` сериализуется с префиксом "fc".</span><span class="sxs-lookup"><span data-stu-id="070b6-120">By including namespace attributes in the root element, the namespaces are serialized so that `http://www.adventure-works.com` is the default namespace, and `www.fourthcoffee.com` is serialized with a prefix of "fc".</span></span> <span data-ttu-id="070b6-121">Чтобы создать атрибут, объявляющий применяемое по умолчанию пространство имен, необходимо создать атрибут с именем «xmlns» без пространства имен.</span><span class="sxs-lookup"><span data-stu-id="070b6-121">To create an attribute that declares a default namespace, you create an attribute with the name "xmlns", without a namespace.</span></span> <span data-ttu-id="070b6-122">Значение этого атрибута является используемым по умолчанию идентификатором URI пространства имен.</span><span class="sxs-lookup"><span data-stu-id="070b6-122">The value of the attribute is the default namespace URI.</span></span>  
  
```csharp  
// The http://www.adventure-works.com namespace is forced to be the default namespace.  
XNamespace aw = "http://www.adventure-works.com";  
XNamespace fc = "www.fourthcoffee.com";  
XElement root = new XElement(aw + "Root",  
    new XAttribute("xmlns", "http://www.adventure-works.com"),  
    new XAttribute(XNamespace.Xmlns + "fc", "www.fourthcoffee.com"),  
    new XElement(fc + "Child",  
        new XElement(aw + "DifferentChild", "other content")  
    ),  
    new XElement(aw + "Child2", "c2 content"),  
    new XElement(fc + "Child3", "c3 content")  
);  
Console.WriteLine(root);  
```  
  
 <span data-ttu-id="070b6-123">В этом примере выводятся следующие данные:</span><span class="sxs-lookup"><span data-stu-id="070b6-123">This example produces the following output:</span></span>  
  
```xml  
<Root xmlns="http://www.adventure-works.com" xmlns:fc="www.fourthcoffee.com">  
  <fc:Child>  
    <DifferentChild>other content</DifferentChild>  
  </fc:Child>  
  <Child2>c2 content</Child2>  
  <fc:Child3>c3 content</fc:Child3>  
</Root>  
```
