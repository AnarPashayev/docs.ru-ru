- <xref:System.DateTime>  
  
- <xref:System.TimeSpan>

- <span data-ttu-id="65f92-110">Любой тип, реализующий `Object.ToString`.</span><span class="sxs-lookup"><span data-stu-id="65f92-110">Any type that implements `Object.ToString`.</span></span>

- <span data-ttu-id="65f92-132">Ноль или более комментариев.</span><span class="sxs-lookup"><span data-stu-id="65f92-132">Zero or more comments.</span></span>  
  
- <span data-ttu-id="65f92-133">Ноль или более инструкций по обработке.</span><span class="sxs-lookup"><span data-stu-id="65f92-133">Zero or more processing instructions.</span></span>
