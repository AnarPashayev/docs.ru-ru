<span data-ttu-id="d0889-103">*Поле* является переменной любого типа, которая объявлена непосредственно в [классе](../../language-reference/keywords/class.md) или [структуре](../../language-reference/keywords/struct.md).</span><span class="sxs-lookup"><span data-stu-id="d0889-103">A *field* is a variable of any type that is declared directly in a [class](../../language-reference/keywords/class.md) or [struct](../../language-reference/keywords/struct.md).</span></span> <span data-ttu-id="d0889-104">Поля являются *членами* содержащих их типов.</span><span class="sxs-lookup"><span data-stu-id="d0889-104">Fields are *members* of their containing type.</span></span>
