<span data-ttu-id="0cf01-103">*Поле* является переменной любого типа, которая объявлена непосредственно в [классе](../../language-reference/keywords/class.md) или [структуре](../../language-reference/keywords/struct.md).</span><span class="sxs-lookup"><span data-stu-id="0cf01-103">A *field* is a variable of any type that is declared directly in a [class](../../language-reference/keywords/class.md) or [struct](../../language-reference/keywords/struct.md).</span></span> <span data-ttu-id="0cf01-104">Поля являются *членами* содержащих их типов.</span><span class="sxs-lookup"><span data-stu-id="0cf01-104">Fields are *members* of their containing type.</span></span>

<span data-ttu-id="0cf01-105">Класс или структура может иметь поля экземпляра и (или) статические поля.</span><span class="sxs-lookup"><span data-stu-id="0cf01-105">A class or struct may have instance fields, static fields, or both.</span></span> <span data-ttu-id="0cf01-106">Поля экземпляра относятся только к экземпляру типа.</span><span class="sxs-lookup"><span data-stu-id="0cf01-106">Instance fields are specific to an instance of a type.</span></span> <span data-ttu-id="0cf01-107">Если имеется класс T с полем экземпляра F, можно создать два объекта типа T и изменить значение F в каждом объекте, не влияя на значение в другом объекте.</span><span class="sxs-lookup"><span data-stu-id="0cf01-107">If you have a class T, with an instance field F, you can create two objects of type T, and modify the value of F in each object without affecting the value in the other object.</span></span> <span data-ttu-id="0cf01-108">Напротив, статическое поле принадлежит к самому классу и является общим для всех экземпляров этого класса.</span><span class="sxs-lookup"><span data-stu-id="0cf01-108">By contrast, a static field belongs to the class itself, and is shared among all instances of that class.</span></span> <span data-ttu-id="0cf01-109">Доступ к статическому полю можно получить только с помощью имени класса.</span><span class="sxs-lookup"><span data-stu-id="0cf01-109">You can access the static field only by using the class name.</span></span> <span data-ttu-id="0cf01-110">Если вы обращаетесь к статическому полю по имени экземпляра, вы получите ошибки времени компиляции [CS0176](../../misc/cs0176.md).</span><span class="sxs-lookup"><span data-stu-id="0cf01-110">If you access the static field by an instance name, you get [CS0176](../../misc/cs0176.md) compile-time error.</span></span>
