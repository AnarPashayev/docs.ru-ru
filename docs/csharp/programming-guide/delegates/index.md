## <a name="delegates-overview"></a><span data-ttu-id="a3a7e-119">Общие сведения о делегатах</span><span class="sxs-lookup"><span data-stu-id="a3a7e-119">Delegates Overview</span></span>  
 <span data-ttu-id="a3a7e-120">Делегаты имеют следующие свойства.</span><span class="sxs-lookup"><span data-stu-id="a3a7e-120">Delegates have the following properties:</span></span>  
  
- <span data-ttu-id="a3a7e-121">Делегаты подобны указателям на функции в C++, но являются полностью объектно-ориентированными и, в отличие от указателей C++ на функции-члены, инкапсулируют экземпляр объекта вместе с методом.</span><span class="sxs-lookup"><span data-stu-id="a3a7e-121">Delegates are similar to C++ function pointers, but delegates are fully object-oriented, and unlike C++ pointers to member functions, delegates encapsulate both an object instance and a method.</span></span>
  
- <span data-ttu-id="a3a7e-122">Делегаты допускают передачу методов в качестве параметров.</span><span class="sxs-lookup"><span data-stu-id="a3a7e-122">Delegates allow methods to be passed as parameters.</span></span>  
  
- <span data-ttu-id="a3a7e-123">Делегаты можно использовать для определения методов обратного вызова.</span><span class="sxs-lookup"><span data-stu-id="a3a7e-123">Delegates can be used to define callback methods.</span></span>  
  
- <span data-ttu-id="a3a7e-124">Делегаты можно связывать друг с другом; например, при появлении одного события можно вызывать несколько методов.</span><span class="sxs-lookup"><span data-stu-id="a3a7e-124">Delegates can be chained together; for example, multiple methods can be called on a single event.</span></span>  
  
- <span data-ttu-id="a3a7e-125">Точное соответствие методов типу делегата не требуется.</span><span class="sxs-lookup"><span data-stu-id="a3a7e-125">Methods do not have to match the delegate type exactly.</span></span> <span data-ttu-id="a3a7e-126">Дополнительные сведения см. в разделе [Использование вариативности в делегатах](../../../csharp/programming-guide/concepts/covariance-contravariance/using-variance-in-delegates.md).</span><span class="sxs-lookup"><span data-stu-id="a3a7e-126">For more information, see [Using Variance in Delegates](../../../csharp/programming-guide/concepts/covariance-contravariance/using-variance-in-delegates.md).</span></span>  
  
- <span data-ttu-id="a3a7e-127">В C# версии 2.0 введена концепция [анонимных методов](../../../csharp/programming-guide/statements-expressions-operators/anonymous-methods.md), которые позволяют передавать блоки кода в виде параметров вместо использования отдельно определенного метода.</span><span class="sxs-lookup"><span data-stu-id="a3a7e-127">C# version 2.0 introduced the concept of [Anonymous Methods](../../../csharp/programming-guide/statements-expressions-operators/anonymous-methods.md), which allow code blocks to be passed as parameters in place of a separately defined method.</span></span> <span data-ttu-id="a3a7e-128">В C# 3.0 для краткой записи встроенных блоков кода введены лямбда-выражения.</span><span class="sxs-lookup"><span data-stu-id="a3a7e-128">C# 3.0 introduced lambda expressions as a more concise way of writing inline code blocks.</span></span> <span data-ttu-id="a3a7e-129">В результате компиляции как анонимных методов, так и лямбда-выражений (в определенном контексте) получаются типы делегатов.</span><span class="sxs-lookup"><span data-stu-id="a3a7e-129">Both anonymous methods and lambda expressions (in certain contexts) are compiled to delegate types.</span></span> <span data-ttu-id="a3a7e-130">В настоящее время эти возможности называются анонимными возможностями.</span><span class="sxs-lookup"><span data-stu-id="a3a7e-130">Together, these features are now known as anonymous functions.</span></span> <span data-ttu-id="a3a7e-131">Дополнительные сведения о лямбда-выражениях см. в разделе [Анонимные функции](../../../csharp/programming-guide/statements-expressions-operators/anonymous-functions.md).</span><span class="sxs-lookup"><span data-stu-id="a3a7e-131">For more information about lambda expressions, see [Anonymous Functions](../../../csharp/programming-guide/statements-expressions-operators/anonymous-functions.md).</span></span>
