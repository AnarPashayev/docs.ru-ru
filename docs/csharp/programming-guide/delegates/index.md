## <a name="delegates-overview"></a><span data-ttu-id="eccd7-119">Общие сведения о делегатах</span><span class="sxs-lookup"><span data-stu-id="eccd7-119">Delegates Overview</span></span>  
 <span data-ttu-id="eccd7-120">Делегаты имеют следующие свойства.</span><span class="sxs-lookup"><span data-stu-id="eccd7-120">Delegates have the following properties:</span></span>  
  
- <span data-ttu-id="eccd7-121">Делегаты подобны указателям на функции в C++, но являются полностью объектно-ориентированными и, в отличие от указателей C++ на функции-члены, инкапсулируют экземпляр объекта вместе с методом.</span><span class="sxs-lookup"><span data-stu-id="eccd7-121">Delegates are similar to C++ function pointers, but delegates are fully object-oriented, and unlike C++ pointers to member functions, delegates encapsulate both an object instance and a method.</span></span>
  
- <span data-ttu-id="eccd7-122">Делегаты допускают передачу методов в качестве параметров.</span><span class="sxs-lookup"><span data-stu-id="eccd7-122">Delegates allow methods to be passed as parameters.</span></span>  
  
- <span data-ttu-id="eccd7-123">Делегаты можно использовать для определения методов обратного вызова.</span><span class="sxs-lookup"><span data-stu-id="eccd7-123">Delegates can be used to define callback methods.</span></span>  
  
- <span data-ttu-id="eccd7-124">Делегаты можно связывать друг с другом; например, при появлении одного события можно вызывать несколько методов.</span><span class="sxs-lookup"><span data-stu-id="eccd7-124">Delegates can be chained together; for example, multiple methods can be called on a single event.</span></span>

- <span data-ttu-id="eccd7-127">В C# версии 2.0 введена концепция [анонимных методов](../../language-reference/operators/delegate-operator.md), которые позволяют передавать блоки кода в виде параметров вместо использования отдельно определенного метода.</span><span class="sxs-lookup"><span data-stu-id="eccd7-127">C# version 2.0 introduced the concept of [anonymous methods](../../language-reference/operators/delegate-operator.md), which allow code blocks to be passed as parameters in place of a separately defined method.</span></span> <span data-ttu-id="eccd7-128">В C# 3.0 для краткой записи встроенных блоков кода введены лямбда-выражения.</span><span class="sxs-lookup"><span data-stu-id="eccd7-128">C# 3.0 introduced lambda expressions as a more concise way of writing inline code blocks.</span></span> <span data-ttu-id="eccd7-129">В результате компиляции как анонимных методов, так и лямбда-выражений (в определенном контексте) получаются типы делегатов.</span><span class="sxs-lookup"><span data-stu-id="eccd7-129">Both anonymous methods and lambda expressions (in certain contexts) are compiled to delegate types.</span></span> <span data-ttu-id="eccd7-130">В настоящее время эти возможности называются анонимными возможностями.</span><span class="sxs-lookup"><span data-stu-id="eccd7-130">Together, these features are now known as anonymous functions.</span></span> <span data-ttu-id="eccd7-131">Дополнительные сведения о лямбда-выражениях см. в разделе [Лямбда-выражения](../statements-expressions-operators/lambda-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="eccd7-131">For more information about lambda expressions, see [Lambda expressions](../statements-expressions-operators/lambda-expressions.md).</span></span>
