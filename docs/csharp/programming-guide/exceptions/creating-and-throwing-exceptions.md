 <span data-ttu-id="9f403-130">Ниже приводятся рекомендации по тому, чего следует избегать при вызове исключений.</span><span class="sxs-lookup"><span data-stu-id="9f403-130">The following list identifies practices to avoid when throwing exceptions:</span></span>

- <span data-ttu-id="9f403-135">Не рекомендуется создавать исключения, которые могут вызываться в режиме отладки, а не в режиме выпуска.</span><span class="sxs-lookup"><span data-stu-id="9f403-135">Do not create exceptions that can be thrown in debug mode but not release mode.</span></span> <span data-ttu-id="9f403-136">Чтобы определить ошибки времени выполнения на этапе разработки, используйте Debug Assert.</span><span class="sxs-lookup"><span data-stu-id="9f403-136">To identify run-time errors during the development phase, use Debug Assert instead.</span></span>
