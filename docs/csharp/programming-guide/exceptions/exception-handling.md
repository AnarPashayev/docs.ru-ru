## <a name="finally-blocks"></a><span data-ttu-id="10d94-133">Блоки "Finally"</span><span class="sxs-lookup"><span data-stu-id="10d94-133">Finally Blocks</span></span>

<span data-ttu-id="10d94-134">Блок `finally` позволяет удалить действия, выполненные в блоке `try`.</span><span class="sxs-lookup"><span data-stu-id="10d94-134">A `finally` block enables you to clean up actions that are performed in a `try` block.</span></span> <span data-ttu-id="10d94-135">При наличии блока `finally` он выполняется последним, после блока `try` и всех выполняемых блоков `catch`.</span><span class="sxs-lookup"><span data-stu-id="10d94-135">If present, the `finally` block executes last, after the `try` block and any matched `catch` block.</span></span> <span data-ttu-id="10d94-136">Блок `finally` выполняется всегда, независимо от того, возникло ли исключение, или найден ли блок `catch`, соответствующий типу исключения.</span><span class="sxs-lookup"><span data-stu-id="10d94-136">A `finally` block always runs, whether an exception is thrown or a `catch` block matching the exception type is found.</span></span>
