# <a name="compiler-error-cs0736"></a><span data-ttu-id="fd1f5-103">Ошибка компилятора CS0736</span><span class="sxs-lookup"><span data-stu-id="fd1f5-103">Compiler Error CS0736</span></span>

<span data-ttu-id="fd1f5-104">"имя типа" не реализует член интерфейса "имя члена".</span><span class="sxs-lookup"><span data-stu-id="fd1f5-104">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="fd1f5-105">Не удается реализовать член интерфейса "имя метода", поскольку он является статическим.</span><span class="sxs-lookup"><span data-stu-id="fd1f5-105">'method name' cannot implement an interface member because it is static.</span></span>  
  
 <span data-ttu-id="fd1f5-106">Эта ошибка возникает в том случае, если статический метод неявно или явно объявлен в качестве реализации члена интерфейса.</span><span class="sxs-lookup"><span data-stu-id="fd1f5-106">This error is generated when a static method is either implicitly or explicitly declared as an implementation of an interface member.</span></span>

- <span data-ttu-id="fd1f5-110">Переопределите содержащий метод тип, чтобы он не наследовал от этого интерфейса.</span><span class="sxs-lookup"><span data-stu-id="fd1f5-110">Redefine the containing type so that it does not inherit from the interface.</span></span>  
  
## <a name="example"></a><span data-ttu-id="fd1f5-111">Пример</span><span class="sxs-lookup"><span data-stu-id="fd1f5-111">Example</span></span>  

 <span data-ttu-id="fd1f5-112">Следующий код приводит к возникновению ошибки CS0736, потому что метод `Program.testMethod` объявлен с модификатором static:</span><span class="sxs-lookup"><span data-stu-id="fd1f5-112">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }
}  
```
