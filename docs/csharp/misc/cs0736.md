# <a name="compiler-error-cs0736"></a><span data-ttu-id="855ed-102">Ошибка компилятора CS0736</span><span class="sxs-lookup"><span data-stu-id="855ed-102">Compiler Error CS0736</span></span>
<span data-ttu-id="855ed-103">"имя типа" не реализует член интерфейса "имя члена".</span><span class="sxs-lookup"><span data-stu-id="855ed-103">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="855ed-104">Не удается реализовать член интерфейса "имя метода", поскольку он является статическим.</span><span class="sxs-lookup"><span data-stu-id="855ed-104">'method name' cannot implement an interface member because it is static.</span></span>  
  
 <span data-ttu-id="855ed-105">Эта ошибка возникает в том случае, если статический метод неявно или явно объявлен в качестве реализации члена интерфейса.</span><span class="sxs-lookup"><span data-stu-id="855ed-105">This error is generated when a static method is either implicitly or explicitly declared as an implementation of an interface member.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="855ed-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="855ed-106">To correct this error</span></span>  
  
- <span data-ttu-id="855ed-107">Удалите модификатор [static](../language-reference/keywords/static.md) из объявления метода.</span><span class="sxs-lookup"><span data-stu-id="855ed-107">Remove the [static](../language-reference/keywords/static.md) modifier from the method declaration.</span></span>  
  
- <span data-ttu-id="855ed-108">Измените имя метода интерфейса.</span><span class="sxs-lookup"><span data-stu-id="855ed-108">Change the name of the interface method.</span></span>  
  
- <span data-ttu-id="855ed-109">Переопределите содержащий метод тип, чтобы он не наследовал от этого интерфейса.</span><span class="sxs-lookup"><span data-stu-id="855ed-109">Redefine the containing type so that it does not inherit from the interface.</span></span>  
  
## <a name="example"></a><span data-ttu-id="855ed-110">Пример</span><span class="sxs-lookup"><span data-stu-id="855ed-110">Example</span></span>  
 <span data-ttu-id="855ed-111">Следующий код приводит к возникновению ошибки CS0736, потому что метод `Program.testMethod` объявлен с модификатором static:</span><span class="sxs-lookup"><span data-stu-id="855ed-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp  
// cs0736.cs  
namespace CS0736  
{
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }
}  
```
