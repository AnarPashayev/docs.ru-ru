## <a name="example"></a><span data-ttu-id="323b2-106">Пример</span><span class="sxs-lookup"><span data-stu-id="323b2-106">Example</span></span>  
 <span data-ttu-id="323b2-107">Следующий пример приводит к возникновению предупреждения CS3002:</span><span class="sxs-lookup"><span data-stu-id="323b2-107">The following example generates CS3002:</span></span>  
  
```csharp  
// CS3002.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
