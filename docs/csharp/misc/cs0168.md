# <a name="compiler-warning-level-3-cs0168"></a><span data-ttu-id="2e690-103">Предупреждение компилятора (уровень 3) CS0168</span><span class="sxs-lookup"><span data-stu-id="2e690-103">Compiler Warning (level 3) CS0168</span></span>
<span data-ttu-id="2e690-104">Переменная "var" объявлена, но не используется</span><span class="sxs-lookup"><span data-stu-id="2e690-104">The variable 'var' is declared but never used</span></span>

<span data-ttu-id="2e690-105">Компилятор выдает предупреждение об уровне 3 при объявлении переменной, но не использует ее.</span><span class="sxs-lookup"><span data-stu-id="2e690-105">The compiler issues a level-three warning when you declare a variable, but do not use it.</span></span>
  
<span data-ttu-id="2e690-106">В следующем примере создается предупреждение CS0168:</span><span class="sxs-lookup"><span data-stu-id="2e690-106">The following sample generates a CS0168 warning:</span></span>  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
