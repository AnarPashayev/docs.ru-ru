 <span data-ttu-id="930e1-104">[Статический](../../csharp/language-reference/keywords/static.md) конструктор не может объявляться с одним или несколькими параметрами.</span><span class="sxs-lookup"><span data-stu-id="930e1-104">A [static](../../csharp/language-reference/keywords/static.md) constructor cannot be declared with one or more parameters.</span></span> <span data-ttu-id="930e1-105">Дополнительные сведения см. в разделе [Конструкторы](../../csharp/programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="930e1-105">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>

```csharp

      public MyClass(int i)

      }

```
