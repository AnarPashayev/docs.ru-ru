## <a name="example"></a><span data-ttu-id="aa287-105">Пример</span><span class="sxs-lookup"><span data-stu-id="aa287-105">Example</span></span>  
 <span data-ttu-id="aa287-106">В следующем примере возникает ошибка CS0135:</span><span class="sxs-lookup"><span data-stu-id="aa287-106">The following sample generates CS0135:</span></span>  
  
```csharp

{  
   public static int i = 0;  
  
   public static void Main()  
   {  
      {  
         int i = 4;  
         i++;  
      }  
      i = 0;   // CS0135  
   }  
}  
```
