# <a name="compiler-error-cs0416"></a><span data-ttu-id="ed486-102">Ошибка компилятора CS0416</span><span class="sxs-lookup"><span data-stu-id="ed486-102">Compiler Error CS0416</span></span>
<span data-ttu-id="ed486-103">"параметр_типа": аргумент атрибута не может использовать параметры типа</span><span class="sxs-lookup"><span data-stu-id="ed486-103">'type parameter': an attribute argument cannot use type parameters</span></span>  
  
 <span data-ttu-id="ed486-104">Параметр типа использовался как аргумент атрибута, что запрещено.</span><span class="sxs-lookup"><span data-stu-id="ed486-104">A type parameter was used as an attribute argument, which is not allowed.</span></span> <span data-ttu-id="ed486-105">Используйте тип, не являющийся универсальным.</span><span class="sxs-lookup"><span data-stu-id="ed486-105">Use a non-generic type.</span></span>  
  
 <span data-ttu-id="ed486-106">Следующий пример приводит к возникновению ошибки CS0416:</span><span class="sxs-lookup"><span data-stu-id="ed486-106">The following sample generates CS0416:</span></span>  
  
```csharp  
// CS0416.cs  
public class MyAttribute : System.Attribute  
{  
   public MyAttribute(System.Type t)  
   {  
   }  
}  
  
class G<T>  
{  
  
   [MyAttribute(typeof(G<T>))]  // CS0416  
   public void F()  
   {  
   }  
  
}  
```
