# <a name="compiler-error-cs0176"></a><span data-ttu-id="38945-103">Ошибка компилятора CS0176</span><span class="sxs-lookup"><span data-stu-id="38945-103">Compiler Error CS0176</span></span>

<span data-ttu-id="38945-104">Доступ к статическому члену "член" по ссылке на экземпляр невозможен. Уточните имя, указав тип.</span><span class="sxs-lookup"><span data-stu-id="38945-104">Static member 'member' cannot be accessed with an instance reference; qualify it with a type name instead</span></span>  
  
 <span data-ttu-id="38945-105">Для уточнения [статической](../language-reference/keywords/static.md) переменной можно использовать только имя класса; имя экземпляра не может быть квалификатором.</span><span class="sxs-lookup"><span data-stu-id="38945-105">Only a class name can be used to qualify a [static](../language-reference/keywords/static.md) variable; an instance name cannot be a qualifier.</span></span> <span data-ttu-id="38945-106">Дополнительные сведения см. в статье [Статические классы и члены статических классов](../programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span><span class="sxs-lookup"><span data-stu-id="38945-106">For more information, see [Static Classes and Static Class Members](../programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span></span>  
  
 <span data-ttu-id="38945-107">При компиляции следующего примера возникнет ошибка CS0176:</span><span class="sxs-lookup"><span data-stu-id="38945-107">The following sample generates CS0176:</span></span>  
  
```csharp  
// CS0176.cs  
public class MyClass2  
{  
    public static int num;  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        MyClass2 mc2 = new MyClass2();  
        int i = mc2.num;   // CS0176  
        // try the following line instead  
        // int i = MyClass2.num;  
    }  
}  
```
