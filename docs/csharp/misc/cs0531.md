# <a name="compiler-error-cs0531"></a><span data-ttu-id="10976-103">Ошибка компилятора CS0531</span><span class="sxs-lookup"><span data-stu-id="10976-103">Compiler Error CS0531</span></span>
<span data-ttu-id="10976-104">"член": члены интерфейса не могут иметь определения</span><span class="sxs-lookup"><span data-stu-id="10976-104">'member' : interface members cannot have a definition</span></span>  
  
 <span data-ttu-id="10976-105">Методы, объявленные в [интерфейсе](../language-reference/keywords/interface.md) , должны быть реализованы в классе, который наследует от него, а не в самом интерфейсе.</span><span class="sxs-lookup"><span data-stu-id="10976-105">Methods that are declared in an [interface](../language-reference/keywords/interface.md) must be implemented in a class that inherits from it and not in the interface itself.</span></span>  
  
 <span data-ttu-id="10976-106">В следующем примере возникает ошибка CS0531:</span><span class="sxs-lookup"><span data-stu-id="10976-106">The following sample generates CS0531:</span></span>  
  
```csharp  
// CS0531.cs  
namespace x  
{  
   public interface clx  
   {  
      int xclx()   // CS0531, cannot define xclx  
      // Try the following declaration instead:  
      // int xclx();  
      {  
         return 0;  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
