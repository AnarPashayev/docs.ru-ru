# <a name="compiler-error-cs0177"></a>Ошибка компилятора CS0177
До передачи управления из текущего метода выходному параметру "параметр" должно быть присвоено значение  
  
 Параметру, помеченному ключевым словом [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) , не было присвоено значение в теле метода. Дополнительные сведения см. в разделе [передача параметров](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)  
  
 Следующий пример приводит к возникновению ошибки CS0177:  
  
```csharp  
// CS0177.cs  
public class MyClass  
{  
   public static void Foo(out int i)   // CS0177  
   {  
   // uncomment the following line to resolve this error  
   //   i = 0;  
   }  
  
   public static void Main()  
   {  
       int x = -1;  
       Foo(out x);  
   }  
}  
```
