# <a name="compiler-error-cs0758"></a><span data-ttu-id="f5e94-103">Ошибка компилятора CS0758</span><span class="sxs-lookup"><span data-stu-id="f5e94-103">Compiler Error CS0758</span></span>

<span data-ttu-id="f5e94-104">Параметр params должен использоваться в обоих объявлениях разделяемого метода или не должен использоваться ни в одном из них.</span><span class="sxs-lookup"><span data-stu-id="f5e94-104">Both partial method declarations must use a params parameter or neither may use a params parameter</span></span>  
  
 <span data-ttu-id="f5e94-105">Если в одной части разделяемого метода указан параметр `params` , то он должен быть указан и в другой части.</span><span class="sxs-lookup"><span data-stu-id="f5e94-105">If one part of a partial method specifies a `params` parameter, the other part must specify one also.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f5e94-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="f5e94-106">To correct this error</span></span>  
  
1. <span data-ttu-id="f5e94-107">Либо добавьте модификатор `params` в одной части метода, либо удалите его из другой.</span><span class="sxs-lookup"><span data-stu-id="f5e94-107">Either add the `params` modifier in one part of the method, or remove it in the other.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f5e94-108">Пример</span><span class="sxs-lookup"><span data-stu-id="f5e94-108">Example</span></span>  

 <span data-ttu-id="f5e94-109">При компиляции следующего кода возникнет ошибка CS0758:</span><span class="sxs-lookup"><span data-stu-id="f5e94-109">The following code generates CS0758:</span></span>  
  
```csharp  
using System;  
  
    public partial class C  
    {  
        partial void Part(int i, params char[] array);  
        partial void Part(int i, char[] array) // CS0758  
        {  
        }  
  
        public static int Main()  
        {  
            return 1;  
        }  
  
    }  
```
