# <a name="compiler-error-cs0167"></a>Ошибка компилятора CS0167
В делегате "делегат" отсутствует метод Invoke  
  
 Вы импортировали и использовали управляемую программу (использующую среду CLR .NET Framework), созданную с помощью другого компилятора. Этот компилятор разрешил неправильно сформированный [делегат](../language-reference/builtin-types/reference-types.md). Таким образом, метод `Invoke` был недоступен. Дополнительные сведения см. в разделе [Делегаты](../programming-guide/delegates/index.md).
