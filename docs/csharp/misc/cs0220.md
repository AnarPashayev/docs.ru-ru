# <a name="compiler-error-cs0220"></a><span data-ttu-id="0df32-103">Ошибка компилятора CS0220</span><span class="sxs-lookup"><span data-stu-id="0df32-103">Compiler Error CS0220</span></span>
<span data-ttu-id="0df32-104">Переполнение при выполнении операции во время компиляции в режиме с проверкой</span><span class="sxs-lookup"><span data-stu-id="0df32-104">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="0df32-105">Операция была обнаружена по [checked](../language-reference/keywords/checked.md), который используется по умолчанию, и привела к потере данных.</span><span class="sxs-lookup"><span data-stu-id="0df32-105">An operation was detected by [checked](../language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="0df32-106">Исправьте входные значения для назначения или используйте [unchecked](../language-reference/keywords/unchecked.md) , чтобы устранить эту ошибку.</span><span class="sxs-lookup"><span data-stu-id="0df32-106">Either correct the inputs to the assignment or use [unchecked](../language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="0df32-107">Дополнительные сведения см. в разделе [установленный и непроверенный](../language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="0df32-107">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="0df32-108">В следующем примере возникает ошибка CS0220:</span><span class="sxs-lookup"><span data-stu-id="0df32-108">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
