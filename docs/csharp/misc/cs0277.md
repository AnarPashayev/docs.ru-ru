## <a name="example"></a><span data-ttu-id="d6cb5-109">Пример</span><span class="sxs-lookup"><span data-stu-id="d6cb5-109">Example</span></span>  
 <span data-ttu-id="d6cb5-110">В следующем примере возникает ошибка CS0277.</span><span class="sxs-lookup"><span data-stu-id="d6cb5-110">The following example generates CS0277:</span></span>

```csharp  
    int Property  
    {  
        get;  
        set;  
    }

{  
    public int Property  
    {  
        get { return 0; }  
        // Try this instead:  
        //set { }  
        protected set { }  
    }  
}  
```
