 <span data-ttu-id="6d57a-106">Эта ошибка возникает при попытке реализовать свойство интерфейса, но реализация метода доступа свойства в классе не является открытой.</span><span class="sxs-lookup"><span data-stu-id="6d57a-106">This error occurs when you try to implement a property of an interface, but the implementation of the property accessor in the class is not public.</span></span> <span data-ttu-id="6d57a-107">Методы, которые реализуют члены интерфейса, должны иметь доступность уровня public.</span><span class="sxs-lookup"><span data-stu-id="6d57a-107">Methods that implement interface members need to have public accessibility.</span></span> <span data-ttu-id="6d57a-108">Для устранения ошибки удалите модификатор доступа в методе доступа свойства.</span><span class="sxs-lookup"><span data-stu-id="6d57a-108">To resolve, remove the access modifier on the property accessor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="6d57a-109">Пример</span><span class="sxs-lookup"><span data-stu-id="6d57a-109">Example</span></span>  

 <span data-ttu-id="6d57a-110">В следующем примере возникает ошибка CS0277.</span><span class="sxs-lookup"><span data-stu-id="6d57a-110">The following example generates CS0277:</span></span>  
  
```csharp

    int Property  
    {  
        get;  
        set;  
    }  
}

{  
    public int Property  
    {  
        get { return 0; }  
        // Try this instead:  
        //set { }  
        protected set { }  
    }  
}  
```
