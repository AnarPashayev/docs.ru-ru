# <a name="compiler-error-cs0752"></a><span data-ttu-id="0c70a-102">Ошибка компилятора CS0752</span><span class="sxs-lookup"><span data-stu-id="0c70a-102">Compiler Error CS0752</span></span>
<span data-ttu-id="0c70a-103">Разделяемый метод не может иметь параметры out.</span><span class="sxs-lookup"><span data-stu-id="0c70a-103">A partial method cannot have out parameters</span></span>  
  
 <span data-ttu-id="0c70a-104">Разделяемый метод не может иметь параметр [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) .</span><span class="sxs-lookup"><span data-stu-id="0c70a-104">A partial method cannot have an [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="0c70a-105">Параметры out запрещены, поскольку при удалении компилятором разделяемого метода нет никакой гарантии, что параметр out когда-либо будет назначен.</span><span class="sxs-lookup"><span data-stu-id="0c70a-105">Out parameters are not allowed because if the partial method is removed by the compiler then there is no guarantee that the out parameter is ever assigned.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="0c70a-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="0c70a-106">To correct this error</span></span>  
  
1. <span data-ttu-id="0c70a-107">Удалите модификатор out из параметра и используйте вместо него возвращаемое значение метода либо удалите модификатор partial из объявления метода.</span><span class="sxs-lookup"><span data-stu-id="0c70a-107">Remove the out modifier from the parameter and use the return value of the method instead, or else remove the partial modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0c70a-108">Пример</span><span class="sxs-lookup"><span data-stu-id="0c70a-108">Example</span></span>

    partial void Part(out int num); // CS0752

    }
