# <a name="compiler-error-cs0752"></a><span data-ttu-id="9dd22-103">Ошибка компилятора CS0752</span><span class="sxs-lookup"><span data-stu-id="9dd22-103">Compiler Error CS0752</span></span>

<span data-ttu-id="9dd22-104">Разделяемый метод не может иметь параметры out.</span><span class="sxs-lookup"><span data-stu-id="9dd22-104">A partial method cannot have out parameters</span></span>  
  
 <span data-ttu-id="9dd22-105">Разделяемый метод не может иметь параметр [out](../language-reference/keywords/out-parameter-modifier.md) .</span><span class="sxs-lookup"><span data-stu-id="9dd22-105">A partial method cannot have an [out](../language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="9dd22-106">Параметры out запрещены, поскольку при удалении компилятором разделяемого метода нет никакой гарантии, что параметр out когда-либо будет назначен.</span><span class="sxs-lookup"><span data-stu-id="9dd22-106">Out parameters are not allowed because if the partial method is removed by the compiler then there is no guarantee that the out parameter is ever assigned.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="9dd22-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="9dd22-107">To correct this error</span></span>  
  
1. <span data-ttu-id="9dd22-108">Удалите модификатор out из параметра и используйте вместо него возвращаемое значение метода либо удалите модификатор partial из объявления метода.</span><span class="sxs-lookup"><span data-stu-id="9dd22-108">Remove the out modifier from the parameter and use the return value of the method instead, or else remove the partial modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9dd22-109">Пример</span><span class="sxs-lookup"><span data-stu-id="9dd22-109">Example</span></span>  

 <span data-ttu-id="9dd22-110">В следующем коде возникает ошибка CS0752:</span><span class="sxs-lookup"><span data-stu-id="9dd22-110">The following code generates CS0752:</span></span>  
  
```csharp  
// cs0752.cs  
public partial class C  
{  
    partial void Part(out int num); // CS0752  
    // try the following line instead  
    // partial void Part(int num);  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
