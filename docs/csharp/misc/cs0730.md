## <a name="example"></a><span data-ttu-id="5e7e3-106">Пример</span><span class="sxs-lookup"><span data-stu-id="5e7e3-106">Example</span></span>  

 <span data-ttu-id="5e7e3-107">Следующий пример приводит к возникновению ошибки CS0730.</span><span class="sxs-lookup"><span data-stu-id="5e7e3-107">The following sample generates CS0730.</span></span> <span data-ttu-id="5e7e3-108">Он состоит из двух файлов исходного кода.</span><span class="sxs-lookup"><span data-stu-id="5e7e3-108">It consists of two source files.</span></span> <span data-ttu-id="5e7e3-109">Сначала скомпилируйте файл библиотеки `CS0730a.cs`, а затем скомпилируйте файл `CS0730.cs` , ссылающийся на файл библиотеки.</span><span class="sxs-lookup"><span data-stu-id="5e7e3-109">First, compile the library file `CS0730a.cs`, and the compile the file `CS0730.cs` referencing the library file.</span></span>  
  
```csharp  
// CS0730a.cs  
// compile with: /t:library  
public class Outer  
{  
   public class Nested {}  
}  
```  
  
```csharp  
// CS0730.cs  
// compile with: /t:library /r:CS0730a.dll  
using System.Runtime.CompilerServices;  
  
[assembly:TypeForwardedToAttribute(typeof(Outer.Nested))]   // CS0730  
  
[assembly:TypeForwardedToAttribute(typeof(Outer))]   // OK  
```
