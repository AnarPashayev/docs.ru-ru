# <a name="compiler-error-cs1932"></a><span data-ttu-id="87605-102">Ошибка компилятора CS1932</span><span class="sxs-lookup"><span data-stu-id="87605-102">Compiler Error CS1932</span></span>

<span data-ttu-id="87605-103">Невозможно назначить "выражение" переменной диапазона.</span><span class="sxs-lookup"><span data-stu-id="87605-103">Cannot assign 'expression' to a range variable.</span></span>  
  
 <span data-ttu-id="87605-104">Компилятор должен иметь возможность вывести тип переменной диапазона, представленной в предложении `from` или `let` .</span><span class="sxs-lookup"><span data-stu-id="87605-104">The compiler must be able to infer the type of a range variable, whether it is introduced in a `from` clause or a `let` clause.</span></span> <span data-ttu-id="87605-105">Он не может иметь значение null, поскольку значение null не является типом и не может быть назначено с помощью выражения небезопасного типа.</span><span class="sxs-lookup"><span data-stu-id="87605-105">It cannot be null because null is not a type, and it cannot be assigned with an expression of an unsafe type.</span></span>

## <a name="example"></a><span data-ttu-id="87605-109">Пример</span><span class="sxs-lookup"><span data-stu-id="87605-109">Example</span></span>

```csharp  
class Test  
{  
    static void Main()  
    {  
  
        var x = from i in Enumerable.Range(1, 100)  
                let k = null // CS1932  
                // Try the following line instead.

                select i;  
    }  
}  
```
