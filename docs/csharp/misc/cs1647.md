# <a name="compiler-error-cs1647"></a>Ошибка компилятора CS1647
Выражение слишком длинное или сложное для компиляции рядом с "код"  
  
 Произошло переполнение стека в компиляторе, обрабатывающем ваш код. Чтобы устранить эту ошибку, упростите код. Если код допустим, получите помощь на сайте [сообщества разработчиков](https://developercommunity.visualstudio.com/spaces/61/index.html) или в [StackOverflow](https://stackoverflow.com/).
