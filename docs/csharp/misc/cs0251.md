# <a name="compiler-warning-level-2-cs0251"></a><span data-ttu-id="d8bf0-103">Предупреждение компилятора (уровень 2) CS0251</span><span class="sxs-lookup"><span data-stu-id="d8bf0-103">Compiler Warning (level 2) CS0251</span></span>

<span data-ttu-id="d8bf0-104">Индексирование массива с отрицательным индексом (индексы массива всегда начинаются с нуля)</span><span class="sxs-lookup"><span data-stu-id="d8bf0-104">Indexing an array with a negative index (array indices always start at zero)</span></span>  
  
 <span data-ttu-id="d8bf0-105">Не используйте отрицательное число в качестве индекса массива.</span><span class="sxs-lookup"><span data-stu-id="d8bf0-105">Do not use a negative number to index into an array.</span></span>  
  
 <span data-ttu-id="d8bf0-106">Следующий пример приводит к возникновению предупреждения CS0251:</span><span class="sxs-lookup"><span data-stu-id="d8bf0-106">The following sample generates CS0251:</span></span>  
  
```csharp  
// CS0251.cs  
// compile with: /W:2  
class MyClass  
{  
   public static void Main()  
   {  
      int[] myarray = new int[] {1,2,3};
      try  
      {  
         myarray[-1]++;   // CS0251  
         // try the following line instead  
         // myarray[1]++;  
      }  
      catch (System.IndexOutOfRangeException e)  
      {  
         System.Console.WriteLine("{0}", e);  
      }  
   }  
}  
```
