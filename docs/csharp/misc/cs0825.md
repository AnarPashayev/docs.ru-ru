## <a name="example"></a><span data-ttu-id="1660d-107">Пример</span><span class="sxs-lookup"><span data-stu-id="1660d-107">Example</span></span>  
 <span data-ttu-id="1660d-108">В следующем коде возникает ошибка CS0825, поскольку `var` используется в поле класса:</span><span class="sxs-lookup"><span data-stu-id="1660d-108">The following code generates CS0825 because `var` is used on a class field:</span></span>

```csharp  
{  
    // Both of these declarations trigger CS0825
    private var genreName;   
    private var bookTitles = new List<string>();

    {  
        var totalBooks = 42; // var is OK here  
        return -1;  
    }  
}  
```
