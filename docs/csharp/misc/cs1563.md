# <a name="compiler-error-cs1563"></a>Ошибка компилятора CS1563
Для создания "выходной_файл" не указаны исходные файлы  
  
 Указан параметр компилятора [-out](../language-reference/compiler-options/out-compiler-option.md) , но файлы исходного кода не следуют. Не следует использовать параметр **/out**или с параметром **/out**необходимо указать файлы с исходным кодом.
