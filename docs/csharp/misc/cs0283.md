# <a name="compiler-error-cs0283"></a><span data-ttu-id="0f245-103">Ошибка компилятора CS0283</span><span class="sxs-lookup"><span data-stu-id="0f245-103">Compiler Error CS0283</span></span>

<span data-ttu-id="0f245-104">Тип "тип" не может быть объявлен как константа.</span><span class="sxs-lookup"><span data-stu-id="0f245-104">The type 'type' cannot be declared const</span></span>  
  
<span data-ttu-id="0f245-105">Тип, указанный в объявлении [константы](../language-reference/keywords/const.md) , должен быть `byte` , `sbyte` , `ushort` ,, `short` `uint` , `int` , `ulong` , `long` , `char` , `float` , `double` , `decimal` ,, `bool` `string` , типом [перечисления](../language-reference/builtin-types/enum.md) или ссылочным типом, которому присвоено значение `null` .</span><span class="sxs-lookup"><span data-stu-id="0f245-105">The type specified in a [constant](../language-reference/keywords/const.md) declaration must be `byte`, `sbyte`, `ushort`, `short`, `uint`, `int`, `ulong`, `long`, `char`, `float`, `double`, `decimal`, `bool`, `string`, an [enum](../language-reference/builtin-types/enum.md) type, or a reference type that is assigned a value of `null`.</span></span> <span data-ttu-id="0f245-106">Каждое константное выражение должно возвращать значение целевого типа или типа, который неявно преобразуется в целевой тип.</span><span class="sxs-lookup"><span data-stu-id="0f245-106">Each constant expression must yield a value of the target type or of a type that is implicitly convertible to the target type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0f245-107">Пример</span><span class="sxs-lookup"><span data-stu-id="0f245-107">Example</span></span>  

 <span data-ttu-id="0f245-108">Следующий пример приводит к возникновению ошибки CS0283.</span><span class="sxs-lookup"><span data-stu-id="0f245-108">The following example generates CS0283.</span></span>  
  
```csharp  
// CS0283.cs  
struct MyTest  
{  
}  
class MyClass
{  
    // To resolve the error but retain the "const-ness",  
    // change const to readonly.  
    const MyTest test = new MyTest();   // CS0283  
  
    public static int Main() {  
        return 1;  
    }  
}  
```
