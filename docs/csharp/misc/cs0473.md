<span data-ttu-id="21a2f-103">Реализация явного интерфейса "имя_метода" соответствует более чем одному членом интерфейса.</span><span class="sxs-lookup"><span data-stu-id="21a2f-103">Explicit interface implementation 'method name' matches more than one interface member.</span></span> <span data-ttu-id="21a2f-104">Фактический выбор конкретного члена интерфейса зависит от реализации.</span><span class="sxs-lookup"><span data-stu-id="21a2f-104">Which interface member is actually chosen is implementation-dependent.</span></span> <span data-ttu-id="21a2f-105">Попробуйте вместо явной реализации использовать неявную реализацию интерфейса.</span><span class="sxs-lookup"><span data-stu-id="21a2f-105">Consider using a non-explicit implementation instead.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="21a2f-109">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="21a2f-109">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="21a2f-111">Пример</span><span class="sxs-lookup"><span data-stu-id="21a2f-111">Example</span></span>

<span data-ttu-id="21a2f-112">Следующий код приводит к возникновению ошибки CS0473:</span><span class="sxs-lookup"><span data-stu-id="21a2f-112">The following code generates CS0473:</span></span>

```csharp
public interface ITest<T>
{
    int TestMethod(int i);
    int TestMethod(T i);
}

public class ImplementingClass : ITest<int>
{
    int ITest<int>.TestMethod(int i) // CS0473
    {
        return i + 1;
    }

    public int TestMethod(int i)
    {
        return i - 1;
    }
}

class T
{
    static int Main()
    {
        ImplementingClass a = new ImplementingClass();
        if (a.TestMethod(0) != -1)
            return -1;

        ITest<int> i_a = a;
        System.Console.WriteLine(i_a.TestMethod(0).ToString());
        if (i_a.TestMethod(0) != 1)
            return -1;

        return 0;
    }
}
```
