# <a name="compiler-error-cs1025"></a><span data-ttu-id="f9a32-103">Ошибка компилятора CS1025</span><span class="sxs-lookup"><span data-stu-id="f9a32-103">Compiler Error CS1025</span></span>
<span data-ttu-id="f9a32-104">Требуется однострочный комментарий или признак конца строки.</span><span class="sxs-lookup"><span data-stu-id="f9a32-104">Single-line comment or end-of-line expected</span></span>  
  
 <span data-ttu-id="f9a32-105">Строка с [директивой препроцессора](../language-reference/preprocessor-directives/index.md) не может содержать многострочный комментарий.</span><span class="sxs-lookup"><span data-stu-id="f9a32-105">A line with a [preprocessor directive](../language-reference/preprocessor-directives/index.md) cannot have a multiline comment.</span></span>

```csharp  
#if true /* hello  
*/   // CS1025  
#endif   // this is a good comment  
```  
  
 <span data-ttu-id="f9a32-107">Ошибка CS1025 также может возникать при попытке использования недопустимой директивы препроцессора, например:</span><span class="sxs-lookup"><span data-stu-id="f9a32-107">CS1025 could also occur if you attempt some invalid preprocessor directive, as follows:</span></span>  
  
```csharp

#define a  
  
class Sample  
{  
   static void Main()  
   {  
      #if a 1   // CS1025, invalid syntax  
         System.Console.WriteLine("Hello, World!");  
      #endif  
   }  
}  
```
