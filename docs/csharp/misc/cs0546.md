# <a name="compiler-error-cs0546"></a><span data-ttu-id="282c5-103">Ошибка компилятора CS0546</span><span class="sxs-lookup"><span data-stu-id="282c5-103">Compiler Error CS0546</span></span>
<span data-ttu-id="282c5-104">"метод доступа": переопределение невозможно, так как "свойство" не имеет функции доступа set, доступной для переопределения</span><span class="sxs-lookup"><span data-stu-id="282c5-104">'accessor' : cannot override because 'property' does not have an overridable set accessor</span></span>  
  
 <span data-ttu-id="282c5-105">Попытка переопределить один из методов доступа для свойства не удалась, так как метод доступа невозможно переопределить.</span><span class="sxs-lookup"><span data-stu-id="282c5-105">An attempt to override one of the accessor methods for a property failed because the accessor cannot be overridden.</span></span> <span data-ttu-id="282c5-106">Эта ошибка может произойти, если:</span><span class="sxs-lookup"><span data-stu-id="282c5-106">This error can occur if:</span></span>  
  
- <span data-ttu-id="282c5-107">Свойство базового класса не объявлено как [virtual](../language-reference/keywords/virtual.md).</span><span class="sxs-lookup"><span data-stu-id="282c5-107">the base class property is not declared as [virtual](../language-reference/keywords/virtual.md).</span></span>  
  
- <span data-ttu-id="282c5-108">Свойство базового класса не объявляет метод доступа [get](../language-reference/keywords/get.md) или [set](../language-reference/keywords/set.md) , который вы пытаетесь переопределить.</span><span class="sxs-lookup"><span data-stu-id="282c5-108">the base class property does not declare the [get](../language-reference/keywords/get.md) or [set](../language-reference/keywords/set.md) accessor you are trying to override.</span></span>  
  
 <span data-ttu-id="282c5-109">Если вы не хотите переопределять свойство базового класса, то можете использовать ключевое слово [new](../language-reference/keywords/new-modifier.md) перед свойством в производном классе.</span><span class="sxs-lookup"><span data-stu-id="282c5-109">If you do not want to override the base class property, you can use the [new](../language-reference/keywords/new-modifier.md) keyword before the property in derived class.</span></span>

## <a name="example"></a><span data-ttu-id="282c5-111">Пример</span><span class="sxs-lookup"><span data-stu-id="282c5-111">Example</span></span>  
 <span data-ttu-id="282c5-112">В приведенном ниже примере возникает ошибка CS0546, так как в базовом классе не объявлен метод доступа set для свойства.</span><span class="sxs-lookup"><span data-stu-id="282c5-112">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```
