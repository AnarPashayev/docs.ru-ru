# <a name="compiler-error-cs1641"></a><span data-ttu-id="967b0-103">Ошибка компилятора CS1641</span><span class="sxs-lookup"><span data-stu-id="967b0-103">Compiler Error CS1641</span></span>
<span data-ttu-id="967b0-104">Поле буфера фиксированного размера должно иметь спецификатор размера массива после имени поля.</span><span class="sxs-lookup"><span data-stu-id="967b0-104">A fixed size buffer field must have the array size specifier after the field name</span></span>  
  
 <span data-ttu-id="967b0-105">В отличие от обычных массивов буферам фиксированного размера требуется указание размера константы в момент объявления.</span><span class="sxs-lookup"><span data-stu-id="967b0-105">Unlike regular arrays, fixed size buffers require a constant size to be specified at the declaration point.</span></span> <span data-ttu-id="967b0-106">Чтобы устранить эту ошибку, добавьте положительный целочисленный литерал или постоянное положительное целое число и поместите квадратные скобки после идентификатора.</span><span class="sxs-lookup"><span data-stu-id="967b0-106">To resolve this error, add a positive integer literal or a constant positive integer and put the square brackets after the identifier.</span></span>  
  
 <span data-ttu-id="967b0-107">Следующий пример приводит к возникновению ошибки CS1641:</span><span class="sxs-lookup"><span data-stu-id="967b0-107">The following sample generates CS1641:</span></span>  
  
```csharp  
// CS1641.cs  
// compile with: /unsafe /target:library  
unsafe struct S {  
   fixed int [] a;  // CS1641  
  
   // OK  
   fixed int b [10];  
   const int c = 10;  
   fixed int d [c];  
}  
```
