# <a name="compiler-error-cs0412"></a><span data-ttu-id="b1235-103">Ошибка компилятора CS0412</span><span class="sxs-lookup"><span data-stu-id="b1235-103">Compiler Error CS0412</span></span>

<span data-ttu-id="b1235-104">"универсальный": имя параметра или локальной переменной не может совпадать с именем параметра типа метода</span><span class="sxs-lookup"><span data-stu-id="b1235-104">'generic': a parameter or local variable cannot have the same name as a method type parameter</span></span>  
  
 <span data-ttu-id="b1235-105">Существует конфликт имен между параметром типа универсального метода и локальной переменной в методе или один из параметров метода.</span><span class="sxs-lookup"><span data-stu-id="b1235-105">There is a name conflict between the type parameter of a generic method and a local variable in the method or one of the method's parameters.</span></span> <span data-ttu-id="b1235-106">Чтобы избежать этой ошибки, переименуйте все конфликтующие параметры и локальные переменные.</span><span class="sxs-lookup"><span data-stu-id="b1235-106">To avoid this error, rename any conflicting parameters or local variables.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b1235-107">Пример</span><span class="sxs-lookup"><span data-stu-id="b1235-107">Example</span></span>  

 <span data-ttu-id="b1235-108">Следующий пример приводит к возникновению ошибки CS0412:</span><span class="sxs-lookup"><span data-stu-id="b1235-108">The following sample generates CS0412:</span></span>  
  
```csharp  
// CS0412.cs  
using System;  
  
class C  
{  
    // Parameter name is the same as method type parameter name  
    public void G<T>(int T)  // CS0412  
    {  
    }  
    public void F<T>()  
    {  
        // Method local variable name is the same as method type  
        // parameter name  
        double T = 0.0;  // CS0412  
        Console.WriteLine(T);  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
