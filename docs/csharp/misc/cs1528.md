# <a name="compiler-error-cs1528"></a><span data-ttu-id="c8ae6-103">Ошибка компилятора CS1528</span><span class="sxs-lookup"><span data-stu-id="c8ae6-103">Compiler Error CS1528</span></span>

<span data-ttu-id="c8ae6-104">Требуется ";" или "=" (невозможно задать аргументы конструктора в объявлении).</span><span class="sxs-lookup"><span data-stu-id="c8ae6-104">Expected ; or = (cannot specify constructor arguments in declaration)</span></span>  
  
 <span data-ttu-id="c8ae6-105">Ссылка на класс сформирована таким образом, как если бы создавался объект класса.</span><span class="sxs-lookup"><span data-stu-id="c8ae6-105">A reference to a class was formed as if an object to the class was being created.</span></span> <span data-ttu-id="c8ae6-106">Например, была попытка передать переменную в конструктор.</span><span class="sxs-lookup"><span data-stu-id="c8ae6-106">For example, there was an attempt to pass a variable to a constructor.</span></span> <span data-ttu-id="c8ae6-107">Для создания объекта класса используйте оператор [new](../language-reference/operators/new-operator.md) .</span><span class="sxs-lookup"><span data-stu-id="c8ae6-107">Use the [new](../language-reference/operators/new-operator.md) operator to create an object of a class.</span></span>  
  
 <span data-ttu-id="c8ae6-108">Следующий пример приводит к возникновению ошибки CS1528:</span><span class="sxs-lookup"><span data-stu-id="c8ae6-108">The following sample generates CS1528:</span></span>  
  
```csharp  
// CS1528.cs  
using System;  
  
public class B  
{  
   public B(int i)  
   {  
      _i = i;  
   }  
  
   public void PrintB()  
   {  
      Console.WriteLine(_i);  
   }  
  
   private int _i;  
}  
  
public class mine  
{  
   public static void Main()  
   {  
      B b(3);   // CS1528, reference is not an object  
      // try one of the following  
      // B b;  
      // or  
      // B bb = new B(3);  
      // bb.PrintB();  
   }  
}  
```
