# <a name="compiler-error-cs0539"></a><span data-ttu-id="9c00d-102">Ошибка компилятора CS0539</span><span class="sxs-lookup"><span data-stu-id="9c00d-102">Compiler Error CS0539</span></span>
<span data-ttu-id="9c00d-103">"член" в явном объявлении интерфейса не является членом интерфейса</span><span class="sxs-lookup"><span data-stu-id="9c00d-103">'member' in explicit interface declaration is not a member of interface</span></span>  
  
 <span data-ttu-id="9c00d-104">Была предпринята попытка явного объявления члена [интерфейса](../language-reference/keywords/interface.md) , который не существует.</span><span class="sxs-lookup"><span data-stu-id="9c00d-104">An attempt was made to explicitly declare an [interface](../language-reference/keywords/interface.md) member that does not exist.</span></span> <span data-ttu-id="9c00d-105">Необходимо либо удалить объявление, либо изменить его таким образом, чтобы оно ссылалось на допустимый член интерфейса.</span><span class="sxs-lookup"><span data-stu-id="9c00d-105">You should either delete the declaration or change it so that it refers to a valid interface member.</span></span>  
  
 <span data-ttu-id="9c00d-106">Следующий пример приводит к возникновению ошибки CS0539:</span><span class="sxs-lookup"><span data-stu-id="9c00d-106">The following sample generates CS0539:</span></span>  
  
```csharp  
// CS0539.cs  
namespace x  
{  
   interface I  
   {  
      void m();  
   }  
  
   public class clx : I  
   {  
      void I.x()   // CS0539  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
