## <a name="example"></a><span data-ttu-id="47199-108">Пример</span><span class="sxs-lookup"><span data-stu-id="47199-108">Example</span></span>  
 <span data-ttu-id="47199-109">В следующем примере возникает ошибка CS0241.</span><span class="sxs-lookup"><span data-stu-id="47199-109">The following sample generates CS0241.</span></span> <span data-ttu-id="47199-110">Кроме того, пример показывает, как с помощью перегрузки метода имитировать метод с аргументами по умолчанию.</span><span class="sxs-lookup"><span data-stu-id="47199-110">In addition, the sample shows how to simulate, with overloading, a method with default arguments.</span></span>  
  
```csharp  
// CS0241.cs  
public class A  
{  
   public void Test(int i = 9) {}   // CS0241  
}  
  
public class B  
{  
   public void Test() { Test(9); }  
   public void Test(int i)  {}  
}  
  
public class C  
{  
   public static void Main()  
   {
      B x = new B();  
      x.Test();  
   }  
}  
```
