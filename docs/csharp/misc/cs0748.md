# <a name="compiler-error-cs0748"></a><span data-ttu-id="1f939-102">Ошибка компилятора CS0748</span><span class="sxs-lookup"><span data-stu-id="1f939-102">Compiler Error CS0748</span></span>

<span data-ttu-id="1f939-103">Несоответствие использования лямбда-параметров; типы параметров должны быть явными или неявно.</span><span class="sxs-lookup"><span data-stu-id="1f939-103">Inconsistent lambda parameter usage; parameter types must be all explicit or all implicit.</span></span>
  
<span data-ttu-id="1f939-104">Если лямбда-выражение имеет несколько входных параметров, не разрешается, чтобы одни параметры использовали явную типизацию, а другие неявную.</span><span class="sxs-lookup"><span data-stu-id="1f939-104">If a lambda expression has multiple input parameters, some parameters cannot use implicit typing while others use explicit typing.</span></span>

<span data-ttu-id="1f939-105">Чтобы исправить эту ошибку, либо пропустите все объявления типа параметра, либо явно укажите тип всех параметров.</span><span class="sxs-lookup"><span data-stu-id="1f939-105">To correct this error, either omit all parameter type declarations or explicitly specify the type of all parameters.</span></span>
