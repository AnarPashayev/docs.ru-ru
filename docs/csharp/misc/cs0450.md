# <a name="compiler-error-cs0450"></a><span data-ttu-id="71eeb-103">Ошибка компилятора CS0450</span><span class="sxs-lookup"><span data-stu-id="71eeb-103">Compiler Error CS0450</span></span>

<span data-ttu-id="71eeb-104">"имя_параметра_типа": невозможно одновременно задать класс ограничения и ограничение "class" или "struct"</span><span class="sxs-lookup"><span data-stu-id="71eeb-104">'Type Parameter Name': cannot specify both a constraint class and the 'class' or 'struct' constraint</span></span>  
  
 <span data-ttu-id="71eeb-105">Если параметр типа ограничен ограничением типа структуры, логически противоречиво для него будет ограничение определенным типом класса, поскольку структура и класс являются взаимоисключающими категориями.</span><span class="sxs-lookup"><span data-stu-id="71eeb-105">If a type parameter is constrained by the struct type constraint, it is logically contradictory for it to also be constrained by a specific class type, since struct and class are mutually exclusive categories.</span></span> <span data-ttu-id="71eeb-106">Если параметр типа ограничивается ограничением определенного типа класса, то он по определению ограничен ограничением типа класса, и поэтому указание ограничения типа класса является избыточным.</span><span class="sxs-lookup"><span data-stu-id="71eeb-106">If a type parameter is constrained by a specific class type constraint, then it is by definition constrained by the class type constraint, and so specifying the class type constraint is redundant.</span></span>  
  
## <a name="example"></a><span data-ttu-id="71eeb-107">Пример</span><span class="sxs-lookup"><span data-stu-id="71eeb-107">Example</span></span>  
  
```csharp  
// CS0450.cs  
// compile with: /t:library  
public class GenericsErrors
{  
    public class B { }  
    public class G3<T> where T : struct, B { } // CS0450  
// To resolve, use the following line instead:  
// public class G3<T> where T : B { }  
}  
```
