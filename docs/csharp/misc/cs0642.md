<span data-ttu-id="280ba-104">Возможно, ошибочный пустой оператор</span><span class="sxs-lookup"><span data-stu-id="280ba-104">Possible mistaken empty statement</span></span>  
  
 <span data-ttu-id="280ba-105">Точка с запятой, стоящая после условного оператора, может быть причиной изменения предполагаемого способа выполнения кода.</span><span class="sxs-lookup"><span data-stu-id="280ba-105">A semicolon after a conditional statement may cause your code to execute differently than intended.</span></span>  
  
 <span data-ttu-id="280ba-106">Можно использовать параметр компилятора **/nowarn** или `#pragmas warning` для отключения этого предупреждения; для получения дополнительных сведений см. "-не [предупреждать" (параметры компилятора C#)](../language-reference/compiler-options/nowarn-compiler-option.md) или [#pragma предупреждение](../language-reference/preprocessor-directives/preprocessor-pragma-warning.md) .</span><span class="sxs-lookup"><span data-stu-id="280ba-106">You can use **/nowarn** compiler option or `#pragmas warning` to disable this warning; see [-nowarn (C# Compiler Options)](../language-reference/compiler-options/nowarn-compiler-option.md) or [#pragma warning](../language-reference/preprocessor-directives/preprocessor-pragma-warning.md) for more information.</span></span>  
  
 <span data-ttu-id="280ba-107">При компиляции следующего примера будет выдано предупреждение CS0642:</span><span class="sxs-lookup"><span data-stu-id="280ba-107">The following sample generates CS0642:</span></span>  
  
```csharp  
// CS0642.cs  
// compile with: /W:3  
class MyClass  
{  
   public static void Main()  
   {  
      int i;  
  
      for (i = 0; i < 10; i += 1);   // CS0642 semicolon intentional?  
      {  
         System.Console.WriteLine (i);  
      }  
   }  
}  
```
