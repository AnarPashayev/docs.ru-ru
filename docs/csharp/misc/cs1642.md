# <a name="compiler-error-cs1642"></a><span data-ttu-id="0865c-103">Ошибка компилятора CS1642</span><span class="sxs-lookup"><span data-stu-id="0865c-103">Compiler Error CS1642</span></span>

<span data-ttu-id="0865c-104">Поля буферов фиксированного размера могут быть только членами структур.</span><span class="sxs-lookup"><span data-stu-id="0865c-104">Fixed size buffer fields may only be members of structs.</span></span>  
  
 <span data-ttu-id="0865c-105">Эта ошибка возникает, если поле буфера фиксированного размера используется в `class`, а не в `struct`.</span><span class="sxs-lookup"><span data-stu-id="0865c-105">This error occurs if you use a fixed size buffer field in a `class`, instead of a `struct`.</span></span> <span data-ttu-id="0865c-106">Чтобы устранить эту ошибку, измените `class` на `struct` или объявите это поле как обычный массив.</span><span class="sxs-lookup"><span data-stu-id="0865c-106">To resolve this error, change the `class` to a `struct` or declare the field as an ordinary array.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0865c-107">Пример</span><span class="sxs-lookup"><span data-stu-id="0865c-107">Example</span></span>  

 <span data-ttu-id="0865c-108">Следующий пример приводит к возникновению ошибки CS1642.</span><span class="sxs-lookup"><span data-stu-id="0865c-108">The following sample generates CS1642.</span></span>  
  
```csharp  
// CS1642.cs  
// compile with: /unsafe /target:library  
unsafe class C  
{  
   fixed int a[10];   // CS1642  
}  
  
unsafe struct D  
{  
    fixed int a[10];  
}  
  
unsafe class E  
{  
   public int[] a = null;  
}  
```
