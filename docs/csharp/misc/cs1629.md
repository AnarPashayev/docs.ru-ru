 <span data-ttu-id="50821-105">В следующем примере возникает ошибка CS1629:</span><span class="sxs-lookup"><span data-stu-id="50821-105">The following sample generates CS1629:</span></span>  
  
```csharp  
// CS1629.cs

class C   
{  
   IEnumerator<int> IteratorMeth() {  
      int i;  
      unsafe  // CS1629

      }  
   }  
}
```
