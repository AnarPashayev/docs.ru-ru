# <a name="compiler-error-cs1512"></a><span data-ttu-id="2f7e4-103">Ошибка компилятора CS1512</span><span class="sxs-lookup"><span data-stu-id="2f7e4-103">Compiler Error CS1512</span></span>

<span data-ttu-id="2f7e4-104">Ключевое слово "base" неприменимо в текущем контексте.</span><span class="sxs-lookup"><span data-stu-id="2f7e4-104">Keyword 'base' is not available in the current context</span></span>  
  
 <span data-ttu-id="2f7e4-105">Ключевое слово [base](../language-reference/keywords/base.md) было использовано за пределами метода, свойства или конструктора.</span><span class="sxs-lookup"><span data-stu-id="2f7e4-105">The [base](../language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>  
  
 <span data-ttu-id="2f7e4-106">В следующем примере возникает ошибка CS1512:</span><span class="sxs-lookup"><span data-stu-id="2f7e4-106">The following example generates CS1512:</span></span>  
  
```csharp  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
