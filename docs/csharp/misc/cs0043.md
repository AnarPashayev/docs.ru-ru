# <a name="compiler-error-cs0043"></a>Ошибка компилятора CS0043

PDB-файл "файл" имеет неверный или устаревший формат. Удалите его и создайте заново.

 Удалите PDB-файл для этой компиляции и выполните перекомпиляцию.

> [!NOTE]
> Эта ошибка компилятора больше не используется в Roslyn.
