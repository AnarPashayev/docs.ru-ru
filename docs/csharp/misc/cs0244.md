 <span data-ttu-id="8f18a-106">В следующем примере возникает ошибка CS0244:</span><span class="sxs-lookup"><span data-stu-id="8f18a-106">The following sample generates CS0244:</span></span>  
  
```csharp  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
