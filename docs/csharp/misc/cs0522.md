# <a name="compiler-error-cs0522"></a><span data-ttu-id="559d6-102">Ошибка компилятора CS0522</span><span class="sxs-lookup"><span data-stu-id="559d6-102">Compiler Error CS0522</span></span>
<span data-ttu-id="559d6-103">"конструктор": структуры не могут вызывать конструкторы базового класса</span><span class="sxs-lookup"><span data-stu-id="559d6-103">'constructor' : structs cannot call base class constructors</span></span>  
  
 <span data-ttu-id="559d6-104">[Структура](../language-reference/keywords/struct.md) не может вызвать конструктор базового класса; удалите вызов конструктора базового класса.</span><span class="sxs-lookup"><span data-stu-id="559d6-104">A [struct](../language-reference/keywords/struct.md) cannot call a base class constructor; remove the call to the base class constructor.</span></span>  
  
 <span data-ttu-id="559d6-105">Следующий пример приводит к возникновению ошибки CS0522:</span><span class="sxs-lookup"><span data-stu-id="559d6-105">The following sample generates CS0522:</span></span>  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```
