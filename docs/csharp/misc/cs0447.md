# <a name="compiler-error-cs0447"></a><span data-ttu-id="cd2e4-103">Ошибка компилятора CS0447</span><span class="sxs-lookup"><span data-stu-id="cd2e4-103">Compiler Error CS0447</span></span>

<span data-ttu-id="cd2e4-104">Невозможно использовать атрибуты для аргументов-типов, можно только для параметров-типов</span><span class="sxs-lookup"><span data-stu-id="cd2e4-104">Attributes cannot be used on type arguments, only on type parameters</span></span>

<span data-ttu-id="cd2e4-105">Эта ошибка возникает при применении атрибута к аргументу типа, который содержится в операторе вызова.</span><span class="sxs-lookup"><span data-stu-id="cd2e4-105">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="cd2e4-106">Разрешается применять атрибут к параметру типа в операторе объявления класса или метода, как показано в следующем коде:</span><span class="sxs-lookup"><span data-stu-id="cd2e4-106">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>

```csharp

```

<span data-ttu-id="cd2e4-107">В приведенной ниже строке кода возникает эта ошибка.</span><span class="sxs-lookup"><span data-stu-id="cd2e4-107">The following line of code will generate this error.</span></span> <span data-ttu-id="cd2e4-108">Предполагается, что класс `C`, определенный в предыдущей строке кода, имеет статический метод с именем `MyStaticMethod`.</span><span class="sxs-lookup"><span data-stu-id="cd2e4-108">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>

```csharp

```

## <a name="example"></a><span data-ttu-id="cd2e4-109">Пример</span><span class="sxs-lookup"><span data-stu-id="cd2e4-109">Example</span></span>

```csharp
// CS0447.cs
using System;

namespace Test41
{
    public interface I<A>
    {
        void Meth<B>();
    }
    public class B : I<int>
    {
        void I<[Test] int>.Meth<X>() { }  // CS0447
    }
}
```
