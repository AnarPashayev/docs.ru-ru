# <a name="compiler-error-cs1558"></a><span data-ttu-id="71f15-102">Ошибка компилятора CS1558</span><span class="sxs-lookup"><span data-stu-id="71f15-102">Compiler Error CS1558</span></span>
<span data-ttu-id="71f15-103">Класс "класс" не имеет подходящего статического метода Main</span><span class="sxs-lookup"><span data-stu-id="71f15-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="71f15-104">Параметр компилятора [-Main](../language-reference/compiler-options/main-compiler-option.md) указывает класс, в котором следует искать метод **Main** .</span><span class="sxs-lookup"><span data-stu-id="71f15-104">The [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="71f15-105">Однако метод [Main](../programming-guide/main-and-command-args/index.md) не был определен правильно.</span><span class="sxs-lookup"><span data-stu-id="71f15-105">However, the [Main](../programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="71f15-106">В следующем примере возникает ошибка CS1558 из-за недопустимого типа возврата.</span><span class="sxs-lookup"><span data-stu-id="71f15-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
