# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="853e0-102">Предупреждение о компиляторе (уровень 1) CS0612</span><span class="sxs-lookup"><span data-stu-id="853e0-102">Compiler warning (level 1) CS0612</span></span>

<span data-ttu-id="853e0-103">"член" устарел</span><span class="sxs-lookup"><span data-stu-id="853e0-103">'member' is obsolete</span></span>  
  
 <span data-ttu-id="853e0-104">Дизайнер класса пометил участника [устаревшим атрибутом.](../language-reference/attributes/general.md#obsolete-attribute)</span><span class="sxs-lookup"><span data-stu-id="853e0-104">The class designer marked a member with the [Obsolete attribute](../language-reference/attributes/general.md#obsolete-attribute).</span></span> <span data-ttu-id="853e0-105">Это означает, что этот член может не поддерживаться в будущих версиях данного класса.</span><span class="sxs-lookup"><span data-stu-id="853e0-105">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="853e0-106">В приведенном ниже примере показано, как при доступе к нерекомендуемому члену создается предупреждение CS0612.</span><span class="sxs-lookup"><span data-stu-id="853e0-106">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```csharp  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
