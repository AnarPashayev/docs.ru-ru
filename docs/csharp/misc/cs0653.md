 <span data-ttu-id="59ef9-105">Пользовательский класс атрибутов [abstract](../language-reference/keywords/abstract.md) нельзя использовать в качестве атрибута.</span><span class="sxs-lookup"><span data-stu-id="59ef9-105">An [abstract](../language-reference/keywords/abstract.md) custom attribute class cannot be used as an attribute.</span></span>

```csharp

{  
}  
  
public class My2Attribute : MyAttribute  
{  
}

   {  
   }  
}
```
