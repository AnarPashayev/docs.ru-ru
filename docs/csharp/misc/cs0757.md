# <a name="compiler-error-cs0757"></a><span data-ttu-id="432ad-102">Ошибка компилятора CS0757</span><span class="sxs-lookup"><span data-stu-id="432ad-102">Compiler Error CS0757</span></span>
<span data-ttu-id="432ad-103">Разделяемый метод не может иметь несколько реализующих объявлений.</span><span class="sxs-lookup"><span data-stu-id="432ad-103">A partial method may not have multiple implementing declarations.</span></span>  
  
 <span data-ttu-id="432ad-104">Разделяемый метод состоит из определяющего объявления (сигнатуры) и необязательного реализующего объявления (тела).</span><span class="sxs-lookup"><span data-stu-id="432ad-104">A partial method consists of exactly one defining declaration (signature) and one or zero implementing declarations (body).</span></span> <span data-ttu-id="432ad-105">Не допускаются несколько реализующих объявлений для одних и тех же определяющих объявлений.</span><span class="sxs-lookup"><span data-stu-id="432ad-105">Multiple implementing declarations for the same identical defining declarations are not allowed.</span></span> <span data-ttu-id="432ad-106">Разделяемые методы можно перегружать, и каждая перегруженная версия может иметь не более одного реализующего объявления.</span><span class="sxs-lookup"><span data-stu-id="432ad-106">Partial methods may be overloaded, and each overloaded version may have one or zero implementing declarations.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="432ad-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="432ad-107">To correct this error</span></span>  
  
1. <span data-ttu-id="432ad-108">Удалите все определяющие объявления для разделяемого метода, кроме одного.</span><span class="sxs-lookup"><span data-stu-id="432ad-108">Remove all except one of the implementing declarations for the partial method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="432ad-109">Пример</span><span class="sxs-lookup"><span data-stu-id="432ad-109">Example</span></span>  
 <span data-ttu-id="432ad-110">В следующем примере возникает ошибка CS0757:</span><span class="sxs-lookup"><span data-stu-id="432ad-110">The following example generates CS0757:</span></span>  
  
```csharp  
// cs0757.cs  
using System;  
  
    public partial class C  
    {  
        // Defining declaration.  
        partial void Part();  
  
        // Implementing declaration.  
        partial void Part()  
        {  
            //...Do something.  
        }  
  
        // Second implementing declaration.  
        partial void Part() // CS0757  
        {  
            //...Do something.  
        }  
  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
