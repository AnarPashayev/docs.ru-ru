# <a name="compiler-error-cs1952"></a><span data-ttu-id="07446-102">Ошибка компилятора CS1952</span><span class="sxs-lookup"><span data-stu-id="07446-102">Compiler Error CS1952</span></span>

## <a name="example"></a><span data-ttu-id="07446-107">Пример</span><span class="sxs-lookup"><span data-stu-id="07446-107">Example</span></span>  

 <span data-ttu-id="07446-108">Следующий код приводит к возникновению ошибки CS1952.</span><span class="sxs-lookup"><span data-stu-id="07446-108">The following code produces CS1952:</span></span>  
  
```csharp  
// cs1952.cs  
using System;  
using System.Linq.Expressions;  
  
class Test  
{  
    public static int M(__arglist)  
    {  
        return 1;  
    }  
  
    static int Main()  
    {  
        Expression<Func<int, int>> f = x => Test.M(__arglist(x)); // CS1952  
        return 1;  
    }  
}  
```
