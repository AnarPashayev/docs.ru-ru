 <span data-ttu-id="59edd-104">Определение бинарного [оператор](../../csharp/language-reference/operators/operator-overloading.md) должно принимать два параметра.</span><span class="sxs-lookup"><span data-stu-id="59edd-104">The definition of a binary [operator](../../csharp/language-reference/operators/operator-overloading.md) must take two parameters.</span></span>

```csharp

{

   // try the following line instead

      return new MyClass();  
   }  
  
   public static int Main()  
   {  
      return 1;  
   }  
}  
```
