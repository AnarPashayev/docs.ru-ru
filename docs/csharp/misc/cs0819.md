# <a name="compiler-error-cs0819"></a><span data-ttu-id="40739-102">Ошибка компилятора CS0819</span><span class="sxs-lookup"><span data-stu-id="40739-102">Compiler Error CS0819</span></span>
<span data-ttu-id="40739-103">Неявно типизированные переменные не могут иметь несколько деклараторов.</span><span class="sxs-lookup"><span data-stu-id="40739-103">Implicitly-typed variables cannot have multiple declarators.</span></span>

 <span data-ttu-id="40739-104">В объявлениях, в которых тип задается явным образом, можно использовать несколько операторов объявления, однако в случае неявно типизированных переменных этого делать нельзя.</span><span class="sxs-lookup"><span data-stu-id="40739-104">Multiple declarators are allowed in explicit type declarations, but not with implicitly typed variables.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="40739-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="40739-105">To correct this error</span></span>

<span data-ttu-id="40739-106">У вас есть два варианта:</span><span class="sxs-lookup"><span data-stu-id="40739-106">There are two options:</span></span>
1. <span data-ttu-id="40739-107">Если переменные имеют один и тот же тип, используйте явные объявления.</span><span class="sxs-lookup"><span data-stu-id="40739-107">If the variables are of the same type, use explicit declarations.</span></span>
1. <span data-ttu-id="40739-108">Объявите каждую неявно типизированную локальную переменную и присвойте ей значение в отдельной строке.</span><span class="sxs-lookup"><span data-stu-id="40739-108">Declare and assign a value to each implicitly typed local variable on a separate line.</span></span>

## <a name="example-1"></a><span data-ttu-id="40739-109">Пример 1</span><span class="sxs-lookup"><span data-stu-id="40739-109">Example 1</span></span>
 <span data-ttu-id="40739-110">Следующий код приводит к возникновению ошибки CS0819:</span><span class="sxs-lookup"><span data-stu-id="40739-110">The following code generates CS0819:</span></span>
  
```csharp
// cs0819.cs
class Program
{
    public static void Main()
    {
        var a = 3, b = 2; // CS0819

        // First correction option.
        //int a = 3, b = 2;

        // Second correction option.
        //var a = 3;
        //var b = 2;
    }
}
```
