# <a name="compiler-error-cs0819"></a><span data-ttu-id="17e7a-103">Ошибка компилятора CS0819</span><span class="sxs-lookup"><span data-stu-id="17e7a-103">Compiler Error CS0819</span></span>

<span data-ttu-id="17e7a-104">Неявно типизированные переменные не могут иметь несколько деклараторов.</span><span class="sxs-lookup"><span data-stu-id="17e7a-104">Implicitly-typed variables cannot have multiple declarators.</span></span>

 <span data-ttu-id="17e7a-105">В объявлениях, в которых тип задается явным образом, можно использовать несколько операторов объявления, однако в случае неявно типизированных переменных этого делать нельзя.</span><span class="sxs-lookup"><span data-stu-id="17e7a-105">Multiple declarators are allowed in explicit type declarations, but not with implicitly typed variables.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="17e7a-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="17e7a-106">To correct this error</span></span>

<span data-ttu-id="17e7a-107">Существует два варианта.</span><span class="sxs-lookup"><span data-stu-id="17e7a-107">There are two options:</span></span>

1. <span data-ttu-id="17e7a-108">Если переменные имеют один и тот же тип, используйте явные объявления.</span><span class="sxs-lookup"><span data-stu-id="17e7a-108">If the variables are of the same type, use explicit declarations.</span></span>
1. <span data-ttu-id="17e7a-109">Объявите каждую неявно типизированную локальную переменную и присвойте ей значение в отдельной строке.</span><span class="sxs-lookup"><span data-stu-id="17e7a-109">Declare and assign a value to each implicitly typed local variable on a separate line.</span></span>

## <a name="example-1"></a><span data-ttu-id="17e7a-110">Пример 1</span><span class="sxs-lookup"><span data-stu-id="17e7a-110">Example 1</span></span>

