# <a name="compiler-error-cs1625"></a><span data-ttu-id="99857-103">Ошибка компилятора CS1625</span><span class="sxs-lookup"><span data-stu-id="99857-103">Compiler Error CS1625</span></span>
<span data-ttu-id="99857-104">Нельзя использовать оператор yield в теле предложения finally.</span><span class="sxs-lookup"><span data-stu-id="99857-104">Cannot yield in the body of a finally clause</span></span>  
  
 <span data-ttu-id="99857-105">Использование оператора yield в основной части предложения finally недопустимо.</span><span class="sxs-lookup"><span data-stu-id="99857-105">A yield statement is not allowed in the body of a finally clause.</span></span> <span data-ttu-id="99857-106">Чтобы избежать этой ошибки, выведите оператор yield за пределы предложения finally.</span><span class="sxs-lookup"><span data-stu-id="99857-106">To avoid this error, move the yield statement out of the finally clause.</span></span>  
  
 <span data-ttu-id="99857-107">Следующий пример приводит к возникновению ошибки CS1625:</span><span class="sxs-lookup"><span data-stu-id="99857-107">The following sample generates CS1625:</span></span>  
  
```csharp  
// CS1625.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
      }  
      finally  
      {  
        yield return this;  // CS1625  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
