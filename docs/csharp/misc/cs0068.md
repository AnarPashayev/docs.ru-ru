 <span data-ttu-id="0f195-105">Событие в интерфейсе не может иметь инициализатора.</span><span class="sxs-lookup"><span data-stu-id="0f195-105">An event in an interface cannot have an initializer.</span></span> <span data-ttu-id="0f195-106">Дополнительные сведения см. в разделе [Интерфейсы](../programming-guide/interfaces/index.md).</span><span class="sxs-lookup"><span data-stu-id="0f195-106">For more information, see [Interfaces](../programming-guide/interfaces/index.md).</span></span>

```csharp

{  
   event MyDelegate d = new MyDelegate(M.f);   // CS0068  
   // try the following line instead  
   // event MyDelegate d2;  
}  
  
class M  
{  
   event MyDelegate d = new MyDelegate(M.f);  
  
   public static void f()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
