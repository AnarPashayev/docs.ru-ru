# <a name="compiler-error-cs1661"></a><span data-ttu-id="8494b-103">Ошибка компилятора CS1661</span><span class="sxs-lookup"><span data-stu-id="8494b-103">Compiler Error CS1661</span></span>
<span data-ttu-id="8494b-104">Невозможно преобразовать блок анонимного метода в тип делегата "тип_делегата", так как типы параметров указанного блока не соответствуют типам параметров делегата</span><span class="sxs-lookup"><span data-stu-id="8494b-104">Cannot convert anonymous method block to delegate type 'delegate type' because the specified block's parameter types do not match the delegate parameter types</span></span>  
  
 <span data-ttu-id="8494b-105">Эта ошибка возникает, если в определении анонимного метода типы параметров анонимного метода не соответствуют типам параметров делегата.</span><span class="sxs-lookup"><span data-stu-id="8494b-105">This error occurs if, in an anonymous method definition, the parameter types of the anonymous method do not match the delegate parameter types.</span></span> <span data-ttu-id="8494b-106">Проверьте число параметров, типы параметров и все параметры ref и out, а также точное соответствие.</span><span class="sxs-lookup"><span data-stu-id="8494b-106">Check the number of parameters, the parameter types, and any ref or out parameters and verify an exact match.</span></span>  
  
 <span data-ttu-id="8494b-107">Следующий пример приводит к возникновению ошибки CS1661:</span><span class="sxs-lookup"><span data-stu-id="8494b-107">The following sample generates CS1661:</span></span>  
  
```csharp  
// CS1661.cs

class C  
{  
    public static void Main()  
    {  
        MyDelegate d = delegate(string s) { };  // CS1661  
    }  
}  
```
