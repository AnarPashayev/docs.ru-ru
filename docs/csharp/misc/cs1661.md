<span data-ttu-id="1dc2f-103">Невозможно преобразовать блок анонимного метода в тип делегата "тип_делегата", так как типы параметров указанного блока не соответствуют типам параметров делегата</span><span class="sxs-lookup"><span data-stu-id="1dc2f-103">Cannot convert anonymous method block to delegate type 'delegate type' because the specified block's parameter types do not match the delegate parameter types</span></span>  
  
 <span data-ttu-id="1dc2f-104">Эта ошибка возникает, если в определении анонимного метода типы параметров анонимного метода не соответствуют типам параметров делегата.</span><span class="sxs-lookup"><span data-stu-id="1dc2f-104">This error occurs if, in an anonymous method definition, the parameter types of the anonymous method do not match the delegate parameter types.</span></span> <span data-ttu-id="1dc2f-105">Проверьте число параметров, типы параметров и все параметры ref и out, а также точное соответствие.</span><span class="sxs-lookup"><span data-stu-id="1dc2f-105">Check the number of parameters, the parameter types, and any ref or out parameters and verify an exact match.</span></span>

        MyDelegate d = delegate(string s) { };  // CS1661
