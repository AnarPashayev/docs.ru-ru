# <a name="compiler-error-cs1527"></a><span data-ttu-id="266bf-102">Ошибка компилятора CS1527</span><span class="sxs-lookup"><span data-stu-id="266bf-102">Compiler Error CS1527</span></span>

<span data-ttu-id="266bf-103">Элементы, определенные в пространстве имен, не могут явно объявляться как частные, защищенные, защищенные внутренние или частные защищенные.</span><span class="sxs-lookup"><span data-stu-id="266bf-103">Elements defined in a namespace cannot be explicitly declared as private, protected, protected internal or private protected.</span></span>
  
 <span data-ttu-id="266bf-104">Объявления типов в пространстве имен могут иметь только [открытый](../language-reference/keywords/public.md) или [внутренний](../language-reference/keywords/internal.md) доступ.</span><span class="sxs-lookup"><span data-stu-id="266bf-104">Type declarations in a namespace can have either [public](../language-reference/keywords/public.md) or [internal](../language-reference/keywords/internal.md) access.</span></span> <span data-ttu-id="266bf-105">Если модификатор доступа не указан, по умолчанию используется **внутренний** тип.</span><span class="sxs-lookup"><span data-stu-id="266bf-105">If no accessibility is specified, **internal** is the default.</span></span>  
  
 <span data-ttu-id="266bf-106">Следующий пример приводит к возникновению ошибки CS1527:</span><span class="sxs-lookup"><span data-stu-id="266bf-106">The following sample generates CS1527:</span></span>  
  
```csharp  
// CS1527.cs  
namespace Sample  
{  
   private class C1 {}             // CS1527  
   protected class C2 {}           // CS1527  
   protected internal class C3 {}  // CS1527  
   private protected class C4 {}   // CS1527
}  
```  
  
 <span data-ttu-id="266bf-107">В приведенном ниже примере возникает ошибка CS1527, так как если в коде программы пространство имен не объявлено явно, все объявления типов неявно помещаются в глобальное пространство имен.</span><span class="sxs-lookup"><span data-stu-id="266bf-107">The following example generates CS1527 because when no namespace is explicitly declared in your program code, all type declarations are located implicitly within the global namespace.</span></span>  
  
```csharp  
//cs1527_2.cs  
using System;  
  
protected class C4 {}  
private struct S1 {}  
```
