# <a name="compiler-error-cs0635"></a>Ошибка компилятора CS0635
«атрибут»: System.Interop.UnmanagedType.CustomMarshaller требуются именованные аргументы ComType и Marshal  
  
 Аргументы **ComType** и **Marshal** аргументы должны указываться, когда форматом маршалинга является **System.Interop.UnmanagedType.CustomMarshaller**.
