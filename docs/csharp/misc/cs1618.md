# <a name="compiler-error-cs1618"></a><span data-ttu-id="03aa1-103">Ошибка компилятора CS1618</span><span class="sxs-lookup"><span data-stu-id="03aa1-103">Compiler Error CS1618</span></span>
<span data-ttu-id="03aa1-104">Невозможно создать делегат с "метод", так как он имеет атрибут Conditional</span><span class="sxs-lookup"><span data-stu-id="03aa1-104">Cannot create delegate with 'method' because it has a Conditional attribute</span></span>  
  
 <span data-ttu-id="03aa1-105">Невозможно создать делегат с помощью условного метода, так как метод может отсутствовать в некоторых сборках.</span><span class="sxs-lookup"><span data-stu-id="03aa1-105">You cannot create a delegate with a conditional method because the method might not exist in some builds.</span></span>  
  
 <span data-ttu-id="03aa1-106">Следующий пример приводит к возникновению ошибки CS1618:</span><span class="sxs-lookup"><span data-stu-id="03aa1-106">The following sample generates CS1618:</span></span>  
  
```csharp  
// CS1618.cs  
using System;  
using System.Diagnostics;  
  
delegate void del();  
  
class MakeAnError {  
   public static void Main() {  
      del d = new del(ConditionalMethod);   // CS1618  
      // Invalid because on builds where DEBUG is not set,
      // there will be no "ConditionalMethod".  
   }  
   // To fix the error, remove the next line:  
   [Conditional("DEBUG")]  
   public static void ConditionalMethod()
   {  
      Console.WriteLine("Do something only in debug");  
   }  
}  
```
