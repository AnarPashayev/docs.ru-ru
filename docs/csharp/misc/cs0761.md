# <a name="compiler-error-cs0761"></a><span data-ttu-id="55c9d-102">Ошибка компилятора CS0761</span><span class="sxs-lookup"><span data-stu-id="55c9d-102">Compiler Error CS0761</span></span>
<span data-ttu-id="55c9d-103">Объявления разделяемого метода "метод\<T >" имеют несовместимые ограничения параметров типа.</span><span class="sxs-lookup"><span data-stu-id="55c9d-103">Partial method declarations of 'method\<T>' have inconsistent type parameter constraints.</span></span>  
  
 <span data-ttu-id="55c9d-104">Если разделяемый метод имеет реализацию, ограничение универсального типа должно быть идентичным ограничению, заданному в сигнатуре метода.</span><span class="sxs-lookup"><span data-stu-id="55c9d-104">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="55c9d-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="55c9d-105">To correct this error</span></span>  
  
1. <span data-ttu-id="55c9d-106">Сделайте ограничения универсального типа идентичными во всех частях разделяемого метода.</span><span class="sxs-lookup"><span data-stu-id="55c9d-106">Make the generic type constraints identical on each part of the partial method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="55c9d-107">Пример</span><span class="sxs-lookup"><span data-stu-id="55c9d-107">Example</span></span>  
 <span data-ttu-id="55c9d-108">Следующий код приводит к возникновению ошибки CS0761:</span><span class="sxs-lookup"><span data-stu-id="55c9d-108">The following code generates CS0761:</span></span>  
  
```csharp  
// cs0761.cs  
using System;  
  
public partial class C  
{  
    partial void Part<T>() where T : class;  
    partial void Part<T>() where T : struct // CS0761  
    {  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
