 <span data-ttu-id="04997-104">Компилятор обнаружил ошибку в значении, переданном в директиву [#line](../language-reference/preprocessor-directives/preprocessor-line.md) .</span><span class="sxs-lookup"><span data-stu-id="04997-104">The compiler detected an error with the value passed to the [#line](../language-reference/preprocessor-directives/preprocessor-line.md) directive.</span></span>

```csharp

{  
   static void Main()  
   {  
      #line "abc.sc"         // CS1576  
      // try the following line instead

      intt i;  // error will be reported on line 101  
   }  
}  
```
