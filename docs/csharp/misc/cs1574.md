# <a name="compiler-warning-level-1-cs1574"></a><span data-ttu-id="96206-103">Предупреждение компилятора (уровень 1) CS1574</span><span class="sxs-lookup"><span data-stu-id="96206-103">Compiler Warning (level 1) CS1574</span></span>

<span data-ttu-id="96206-104">Комментарий XML в конструкции "конструкция" имеет синтаксически недопустимый атрибут cref "имя"</span><span class="sxs-lookup"><span data-stu-id="96206-104">XML comment on 'construct' has syntactically incorrect cref attribute 'name'</span></span>  
  
 <span data-ttu-id="96206-105">Строка, передаваемая тегу cref, например внутри \<exception> тега, ссылается на элемент, недоступный в текущей среде сборки.</span><span class="sxs-lookup"><span data-stu-id="96206-105">A string passed to a cref tag, for example, within an \<exception> tag, referred to a member that is not available within the current build environment.</span></span> <span data-ttu-id="96206-106">Строка, передаваемая в тег cref, должна быть синтаксически правильным именем члена или поля.</span><span class="sxs-lookup"><span data-stu-id="96206-106">The string that you pass to a cref tag must be the syntactically correct name of a member or field.</span></span>  
  
 <span data-ttu-id="96206-107">Дополнительные сведения см. в разделе [Рекомендуемые теги для комментариев документации](../programming-guide/xmldoc/recommended-tags-for-documentation-comments.md).</span><span class="sxs-lookup"><span data-stu-id="96206-107">For more information, see [Recommended Tags for Documentation Comments](../programming-guide/xmldoc/recommended-tags-for-documentation-comments.md).</span></span>  
  
 <span data-ttu-id="96206-108">Следующий пример приводит к возникновению ошибки CS1574:</span><span class="sxs-lookup"><span data-stu-id="96206-108">The following sample generates CS1574:</span></span>  
  
```csharp  
// CS1574.cs  
// compile with: /W:1 /doc:x.xml  
using System;  
  
/// <exception cref="System.Console.WriteLin">An exception class.</exception>   // CS1574  
// instead, uncomment and try the following line  
// /// <exception cref="System.Console.WriteLine">An exception class.</exception>  
class EClass : Exception  
{  
}  
  
class TestClass  
{  
   public static void Main()  
   {  
      try  
      {  
      }  
      catch(EClass)  
      {  
      }  
   }  
}  
```
