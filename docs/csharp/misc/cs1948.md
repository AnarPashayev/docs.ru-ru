## <a name="example"></a><span data-ttu-id="b940c-107">Пример</span><span class="sxs-lookup"><span data-stu-id="b940c-107">Example</span></span>  

 <span data-ttu-id="b940c-108">Приведенный ниже пример приводит к возникновению ошибки CS1948, так как идентификатор `T` используется для переменной диапазона и для параметра типа в методе `TestMethod`.</span><span class="sxs-lookup"><span data-stu-id="b940c-108">The following example generates CS1948 because the identifier `T` is used for the range variable and for the type parameter on method `TestMethod`:</span></span>

```csharp  
{  
    public void TestMethod<T>(T t)

    }  
}  
```
