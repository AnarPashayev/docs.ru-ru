B ()" — это метод, который не является допустимым в данном контексте.</span><span class="sxs-lookup"><span data-stu-id="f8645-110">The following sample generates CS0119: 'C.B()' is a method, which is not valid in the given context.</span></span> <span data-ttu-id="f8645-111">Эту ошибку можно устранить, изменив имя метода `C.B` или указав полное имя класса, например `B` `N2.B` .</span><span class="sxs-lookup"><span data-stu-id="f8645-111">You can fix this error by changing the name of the method `C.B`, or using the fully qualified name for the class `B` like `N2.B`.</span></span>

{
    public static class B
    {
        public static void X() {}
    }
}

    public class C
    {
        void B() {}
        void M() => B.X();   // CS0119
    }
