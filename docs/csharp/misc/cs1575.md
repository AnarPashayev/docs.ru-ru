# <a name="compiler-error-cs1575"></a><span data-ttu-id="7c2a0-103">Ошибка компилятора CS1575</span><span class="sxs-lookup"><span data-stu-id="7c2a0-103">Compiler Error CS1575</span></span>

<span data-ttu-id="7c2a0-104">В выражении stackalloc после типа требуется []</span><span class="sxs-lookup"><span data-stu-id="7c2a0-104">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="7c2a0-105">Размер выделенной памяти, запрошенной с помощью оператора [stackalloc](../language-reference/operators/stackalloc.md), необходимо указывать в квадратных скобках.</span><span class="sxs-lookup"><span data-stu-id="7c2a0-105">The size of the requested allocation, with [stackalloc](../language-reference/operators/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="7c2a0-106">Следующий пример приводит к возникновению ошибки CS1575:</span><span class="sxs-lookup"><span data-stu-id="7c2a0-106">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
