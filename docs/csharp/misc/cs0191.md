<span data-ttu-id="446e3-104">Невозможно присвоить значение свойству или индексатору "имя" — доступ только для чтения</span><span class="sxs-lookup"><span data-stu-id="446e3-104">Property or indexer 'name' cannot be assigned to -- it is read only</span></span>  
  
 <span data-ttu-id="446e3-105">Поле [readonly](../language-reference/keywords/readonly.md) принимает назначение только в конструкторе или в объявлении.</span><span class="sxs-lookup"><span data-stu-id="446e3-105">A [readonly](../language-reference/keywords/readonly.md) field can only take an assignment in a constructor or at declaration.</span></span> <span data-ttu-id="446e3-106">Дополнительные сведения см. в разделе [Конструкторы](../programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="446e3-106">For more information, see [Constructors](../programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="446e3-107">Ошибка CS0191 также возникает, если поле `readonly` является [static](../language-reference/keywords/static.md) (статическим), а конструктор не помечен как `static`.</span><span class="sxs-lookup"><span data-stu-id="446e3-107">CS0191 also results if the `readonly` field is [static](../language-reference/keywords/static.md) and the constructor is not marked `static`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="446e3-108">Пример</span><span class="sxs-lookup"><span data-stu-id="446e3-108">Example</span></span>  
 <span data-ttu-id="446e3-109">В следующем примере возникает ошибка CS0191:</span><span class="sxs-lookup"><span data-stu-id="446e3-109">The following sample generates CS0191.</span></span>  
  
```csharp  
// CS0191.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  // OK to assign to readonly field in declaration  
  
    MyClass()  
    {  
        TestInt = 11; // OK to assign to readonly field in constructor  
    }  
  
    public void TestReadOnly()  
    {  
        TestInt = 19;                  // CS0191  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
