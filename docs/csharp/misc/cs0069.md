 <span data-ttu-id="bd0cd-104">Нельзя определить функции доступа к событию в [интерфейсе](../../csharp/language-reference/keywords/interface.md).</span><span class="sxs-lookup"><span data-stu-id="bd0cd-104">You cannot define an event's accessor functions in an [interface](../../csharp/language-reference/keywords/interface.md).</span></span> <span data-ttu-id="bd0cd-105">Дополнительные сведения см. в разделе [события](../../csharp/programming-guide/events/index.md) и [интерфейсы](../../csharp/programming-guide/interfaces/index.md).</span><span class="sxs-lookup"><span data-stu-id="bd0cd-105">For more information, see [Events](../../csharp/programming-guide/events/index.md) and [Interfaces](../../csharp/programming-guide/interfaces/index.md).</span></span>  
  
 <span data-ttu-id="bd0cd-106">Следующий пример приводит к возникновению ошибки CS0069:</span><span class="sxs-lookup"><span data-stu-id="bd0cd-106">The following sample generates CS0069:</span></span>  
  
```csharp  
// CS0069.cs  
// compile with: /target:library  
  
public delegate void EventHandler();  
  
public interface a  
{  
   event EventHandler Click { remove {} }   // CS0069  
   event EventHandler Click2;   // OK  
}  
```
