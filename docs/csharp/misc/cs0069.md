<span data-ttu-id="06cd9-104">Событие в интерфейсе не может иметь методы доступа add и remove.</span><span class="sxs-lookup"><span data-stu-id="06cd9-104">An event in an interface cannot have add or remove accessors</span></span>  
  
 <span data-ttu-id="06cd9-105">Нельзя определить функции доступа к событию в [интерфейсе](../language-reference/keywords/interface.md).</span><span class="sxs-lookup"><span data-stu-id="06cd9-105">You cannot define an event's accessor functions in an [interface](../language-reference/keywords/interface.md).</span></span> <span data-ttu-id="06cd9-106">Дополнительные сведения см. в разделе [события](../programming-guide/events/index.md) и [интерфейсы](../programming-guide/interfaces/index.md).</span><span class="sxs-lookup"><span data-stu-id="06cd9-106">For more information, see [Events](../programming-guide/events/index.md) and [Interfaces](../programming-guide/interfaces/index.md).</span></span>  
  
 <span data-ttu-id="06cd9-107">Следующий пример приводит к возникновению ошибки CS0069:</span><span class="sxs-lookup"><span data-stu-id="06cd9-107">The following sample generates CS0069:</span></span>  
  
```csharp

// compile with: /target:library  
  
public delegate void EventHandler();  
  
public interface a  
{  
   event EventHandler Click { remove {} }   // CS0069  
   event EventHandler Click2;   // OK  
}  
```
