# <a name="compiler-error-cs0424"></a><span data-ttu-id="532fc-103">Ошибка компилятора CS0424</span><span class="sxs-lookup"><span data-stu-id="532fc-103">Compiler Error CS0424</span></span>

<span data-ttu-id="532fc-104">"класс": класс с атрибутом ComImport не может определять базовый класс</span><span class="sxs-lookup"><span data-stu-id="532fc-104">'class': a class with the ComImport attribute cannot specify a base class</span></span>  
  
 <span data-ttu-id="532fc-105">Применение атрибута <xref:System.Runtime.InteropServices.ComImportAttribute> подразумевает, что реализация класса будет импортироваться из COM-модуля.</span><span class="sxs-lookup"><span data-stu-id="532fc-105">Specifying the <xref:System.Runtime.InteropServices.ComImportAttribute> attribute implies that the implementation for the class is to be imported from a COM module.</span></span> <span data-ttu-id="532fc-106">В определенную в COM-модуле реализацию нельзя добавлять дополнительные методы или поля, наследуемые от базового класса.</span><span class="sxs-lookup"><span data-stu-id="532fc-106">Additional methods or fields inherited from the base class are not allowed to be added to the implementation defined in the COM module.</span></span>  
  
 <span data-ttu-id="532fc-107">Следующий пример приводит к возникновению ошибки CS0424:</span><span class="sxs-lookup"><span data-stu-id="532fc-107">The following sample generates CS0424:</span></span>  
  
```csharp  
// CS0424.cs  
// compile with: /target:library  
using System.Runtime.InteropServices;  
public class A {}  
  
[ ComImport, Guid("7ab770c7-0e23-4d7a-8aa2-19bfad479829") ]  
class B : A {}   // CS0424 error  
```
