# <a name="compiler-error-cs0424"></a><span data-ttu-id="183d6-102">Ошибка компилятора CS0424</span><span class="sxs-lookup"><span data-stu-id="183d6-102">Compiler Error CS0424</span></span>

 <span data-ttu-id="183d6-104">Применение атрибута <xref:System.Runtime.InteropServices.ComImportAttribute> подразумевает, что реализация класса будет импортироваться из COM-модуля.</span><span class="sxs-lookup"><span data-stu-id="183d6-104">Specifying the <xref:System.Runtime.InteropServices.ComImportAttribute> attribute implies that the implementation for the class is to be imported from a COM module.</span></span> <span data-ttu-id="183d6-105">В определенную в COM-модуле реализацию нельзя добавлять дополнительные методы или поля, наследуемые от базового класса.</span><span class="sxs-lookup"><span data-stu-id="183d6-105">Additional methods or fields inherited from the base class are not allowed to be added to the implementation defined in the COM module.</span></span>

```csharp

public class A {}  
  
[ ComImport, Guid("7ab770c7-0e23-4d7a-8aa2-19bfad479829") ]

```
