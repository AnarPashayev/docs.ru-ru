# <a name="compiler-error-cs0425"></a><span data-ttu-id="adba7-102">Ошибка компилятора CS0425</span><span class="sxs-lookup"><span data-stu-id="adba7-102">Compiler Error CS0425</span></span>
<span data-ttu-id="adba7-103">Ограничения для параметра типа "параметр типа" метода "метод" должны соответствовать ограничениям для параметра типа "параметр типа" интерфейсного метода "метод".</span><span class="sxs-lookup"><span data-stu-id="adba7-103">The constraints for type parameter 'type parameter' of method 'method' must match the constraints for type parameter 'type parameter' of interface method 'method'.</span></span> <span data-ttu-id="adba7-104">Попробуйте использовать явную реализацию интерфейса.</span><span class="sxs-lookup"><span data-stu-id="adba7-104">Consider using an explicit interface implementation instead.</span></span>  
  
 <span data-ttu-id="adba7-105">Эта ошибка возникает, если виртуальный универсальный метод переопределяется в производном классе и ограничения метода в производном классе не совпадают с ограничениями метода в базовом классе.</span><span class="sxs-lookup"><span data-stu-id="adba7-105">This error occurs if a virtual generic method is overridden in a derived class and the constraints on the method in the derived class do not match the constraints on the method in the base class.</span></span> <span data-ttu-id="adba7-106">Чтобы избежать этой ошибки, убедитесь в том, что предложения `where` в обоих объявлениях идентичны, или реализуйте интерфейс явным образом.</span><span class="sxs-lookup"><span data-stu-id="adba7-106">To avoid this error, make sure the `where` clause is identical in both declarations, or implement the interface explicitly.</span></span>  
  
## <a name="example"></a><span data-ttu-id="adba7-107">Пример</span><span class="sxs-lookup"><span data-stu-id="adba7-107">Example</span></span>  
 <span data-ttu-id="adba7-108">Следующий пример приводит к возникновению ошибки CS0425:</span><span class="sxs-lookup"><span data-stu-id="adba7-108">The following example generates CS0425:</span></span>  
  
```csharp  
// CS0425.cs  
  
class C1  
{  
}  
  
class C2  
{  
}  
  
interface IBase  
{  
    void F<ItemType>(ItemType item) where ItemType : C1;  
}

{  
    public void F<ItemType>(ItemType item) where ItemType : C2  // CS0425  
    {  
    }  
}  
  
class CMain  
{  
    public static void Main()  
    {  
    }  
}  
```
