# <a name="compiler-error-cs0271"></a><span data-ttu-id="a0c13-102">Ошибка компилятора CS0271</span><span class="sxs-lookup"><span data-stu-id="a0c13-102">Compiler Error CS0271</span></span>
<span data-ttu-id="a0c13-103">Свойство или индексатор "свойство/индексатор" не может использоваться в данном контексте, так как метод доступа get недоступен</span><span class="sxs-lookup"><span data-stu-id="a0c13-103">The property or indexer 'property/indexer' cannot be used in this context because the get accessor is inaccessible</span></span>  
  
 <span data-ttu-id="a0c13-104">Эта ошибка возникает при попытке получить доступ к недоступному методу доступа `get` .</span><span class="sxs-lookup"><span data-stu-id="a0c13-104">This error occurs when you try to access an inaccessible `get` accessor.</span></span> <span data-ttu-id="a0c13-105">Чтобы устранить эту ошибку, повысьте уровень доступности метода доступа или измените расположение вызова.</span><span class="sxs-lookup"><span data-stu-id="a0c13-105">To resolve this error, increase the accessibility of the accessor, or change the calling location.</span></span> <span data-ttu-id="a0c13-106">Для получения дополнительной информации [Properties](../programming-guide/classes-and-structs/properties.md)см. [Accessor Accessibility](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md)</span><span class="sxs-lookup"><span data-stu-id="a0c13-106">For more information, see [Accessor Accessibility](../programming-guide/classes-and-structs/restricting-accessor-accessibility.md) and [Properties](../programming-guide/classes-and-structs/properties.md).</span></span>  
  
 <span data-ttu-id="a0c13-107">В следующем примере возникает ошибка CS0271:</span><span class="sxs-lookup"><span data-stu-id="a0c13-107">The following example generates CS0271:</span></span>  
  
```csharp  
// CS0271.cs  
public class MyClass  
{  
   public int Property  
   {  
      private get { return 0; }  
      set { }  
   }  
  
   public int Property2  
   {  
      get { return 0; }  
      set { }  
   }  
}  
  
public class Test  
{  
   public static void Main(string[] args)
   {  
      MyClass c = new MyClass();  
      int a = c.Property;   // CS0271  
      int b = c.Property2;   // OK  
   }  
}  
```
