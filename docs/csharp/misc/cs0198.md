# <a name="compiler-error-cs0198"></a><span data-ttu-id="fad4a-103">Ошибка компилятора CS0198</span><span class="sxs-lookup"><span data-stu-id="fad4a-103">Compiler Error CS0198</span></span>
<span data-ttu-id="fad4a-104">Присваивание значений полям статического поля "имя", доступного только для чтения, допускается только в статическом конструкторе или инициализаторе переменных</span><span class="sxs-lookup"><span data-stu-id="fad4a-104">Fields of static readonly field 'name' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="fad4a-105">Переменная, помеченная модификатором [readonly](../language-reference/keywords/readonly.md) , должна содержать тот же модификатор использования [static](../language-reference/keywords/static.md) , что и конструктор, в котором вы хотите ее инициализировать.</span><span class="sxs-lookup"><span data-stu-id="fad4a-105">A [readonly](../language-reference/keywords/readonly.md) variable must have the same [static](../language-reference/keywords/static.md) usage as the constructor in which you want to initialize it.</span></span> <span data-ttu-id="fad4a-106">Дополнительные сведения см. в разделе [Статические конструкторы](../programming-guide/classes-and-structs/static-constructors.md).</span><span class="sxs-lookup"><span data-stu-id="fad4a-106">For more information, see [Static Constructors](../programming-guide/classes-and-structs/static-constructors.md).</span></span>  
  
 <span data-ttu-id="fad4a-107">В следующем примере возникает ошибка CS0198:</span><span class="sxs-lookup"><span data-stu-id="fad4a-107">The following sample generates CS0198:</span></span>  
  
```csharp  
// CS0198.cs  
class MyClass  
{  
   public static readonly int TestInt = 6;  
  
   MyClass()  
   {  
      TestInt = 11;   // CS0198, constructor is not static and readonly field is  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
