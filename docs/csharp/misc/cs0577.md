# <a name="compiler-error-cs0577"></a><span data-ttu-id="a7c6d-103">Ошибка компилятора CS0577</span><span class="sxs-lookup"><span data-stu-id="a7c6d-103">Compiler Error CS0577</span></span>
<span data-ttu-id="a7c6d-104">Использование атрибута Conditional в "функция" недопустимо, так как он является конструктором, деструктором, оператором или явной реализацией интерфейса</span><span class="sxs-lookup"><span data-stu-id="a7c6d-104">The Conditional attribute is not valid on 'function' because it is a constructor, destructor, operator, or explicit interface implementation</span></span>  
  
 <span data-ttu-id="a7c6d-105">Атрибут`Conditional` нельзя применять к указанным методам.</span><span class="sxs-lookup"><span data-stu-id="a7c6d-105">`Conditional` cannot be applied to the specified methods.</span></span>  
  
 <span data-ttu-id="a7c6d-106">Например, некоторые атрибуты нельзя использовать в явном определении интерфейса.</span><span class="sxs-lookup"><span data-stu-id="a7c6d-106">For example, you cannot use some attributes on an explicit interface definition.</span></span> <span data-ttu-id="a7c6d-107">Следующий пример приводит к возникновению ошибки CS0577:</span><span class="sxs-lookup"><span data-stu-id="a7c6d-107">The following sample generates CS0577:</span></span>  
  
```csharp  
// CS0577.cs  
// compile with: /target:library  
interface I  
{  
   void m();  
}  
  
public class MyClass : I  
{  
   [System.Diagnostics.Conditional("a")]   // CS0577  
   void I.m() {}  
}  
```
