# <a name="compiler-error-cs0131"></a><span data-ttu-id="e465e-103">Ошибка компилятора CS0131</span><span class="sxs-lookup"><span data-stu-id="e465e-103">Compiler Error CS0131</span></span>
<span data-ttu-id="e465e-104">Левая часть выражения присваивания должна быть переменной, свойством или индексатором</span><span class="sxs-lookup"><span data-stu-id="e465e-104">The left-hand side of an assignment must be a variable, property or indexer</span></span>  
  
 <span data-ttu-id="e465e-105">В операторах присваивания значение правой части выражения присваивается левой части выражения.</span><span class="sxs-lookup"><span data-stu-id="e465e-105">In an assignment statement, the value of the right-hand side is assigned to the left-hand side.</span></span> <span data-ttu-id="e465e-106">Левая часть выражения должна быть переменной, свойством или индексатором.</span><span class="sxs-lookup"><span data-stu-id="e465e-106">The left-hand side must be a variable, property, or indexer.</span></span>  
  
 <span data-ttu-id="e465e-107">Чтобы устранить эту ошибку, убедитесь в том, что все операторы находятся в правой части выражения, а левая часть является переменной, свойством или индексатором.</span><span class="sxs-lookup"><span data-stu-id="e465e-107">To fix this error, make sure that all operators are on the right-hand side and that the left-hand side is a variable, property, or indexer.</span></span> <span data-ttu-id="e465e-108">Дополнительные сведения см. в разделе [Инструкции, выражения и операторы](../programming-guide/statements-expressions-operators/index.md).</span><span class="sxs-lookup"><span data-stu-id="e465e-108">For more information, see [Statements, Expressions, and Operators](../programming-guide/statements-expressions-operators/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="e465e-109">Пример</span><span class="sxs-lookup"><span data-stu-id="e465e-109">Example</span></span>  
 <span data-ttu-id="e465e-110">Следующий пример приводит к возникновению ошибки CS0131:</span><span class="sxs-lookup"><span data-stu-id="e465e-110">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="e465e-111">Пример</span><span class="sxs-lookup"><span data-stu-id="e465e-111">Example</span></span>  
 <span data-ttu-id="e465e-112">Эта ошибка может также возникать при попытке выполнить арифметические операции в левой части оператора присваивания, как показано в приведенном ниже примере.</span><span class="sxs-lookup"><span data-stu-id="e465e-112">This error can also occur if you attempt to perform arithmetic operations on the left hand side of an assignment operator, as in the following example.</span></span>  
  
```csharp  
// CS0131b.cs  
public class C  
{  
    public static int Main()  
    {  
        int a = 1, b = 2, c = 3;  
        if (a + b = c) // CS0131  
        // try this instead  
        // if (a + b == c)  
            return 0;  
        return 1;  
    }  
}  
```
