# <a name="compiler-error-cs0107"></a><span data-ttu-id="4e071-102">Ошибка компилятора CS0107</span><span class="sxs-lookup"><span data-stu-id="4e071-102">Compiler Error CS0107</span></span>

<span data-ttu-id="4e071-103">Несколько модификаторов защиты.</span><span class="sxs-lookup"><span data-stu-id="4e071-103">More than one protection modifier</span></span>

<span data-ttu-id="4e071-104">Только один модификатор доступа ([открытый](../language-reference/keywords/public.md), [частный](../language-reference/keywords/private.md), [защищенный](../language-reference/keywords/protected.md)или [внутренний](../language-reference/keywords/internal.md)) разрешен в члене класса, за исключением [**защищенных внутренних**](../language-reference/keywords/protected-internal.md) и [**закрытых защищенных**](../language-reference/keywords/private-protected.md)элементов.</span><span class="sxs-lookup"><span data-stu-id="4e071-104">Only one access modifier ([public](../language-reference/keywords/public.md), [private](../language-reference/keywords/private.md), [protected](../language-reference/keywords/protected.md), or [internal](../language-reference/keywords/internal.md)) is allowed on a class member, with the exception of [**protected internal**](../language-reference/keywords/protected-internal.md) and [**private protected**](../language-reference/keywords/private-protected.md).</span></span>

<span data-ttu-id="4e071-105">Следующий пример приводит к возникновению ошибки CS0107:</span><span class="sxs-lookup"><span data-stu-id="4e071-105">The following sample generates CS0107:</span></span>

```csharp
// CS0107.cs
public class C
{
    private internal void F()   // CS0107, delete private or internal
    {
    }
}
```
