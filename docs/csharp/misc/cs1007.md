# <a name="compiler-error-cs1007"></a><span data-ttu-id="86c84-103">Ошибка компилятора CS1007</span><span class="sxs-lookup"><span data-stu-id="86c84-103">Compiler Error CS1007</span></span>

<span data-ttu-id="86c84-104">Метод доступа к свойству уже определен.</span><span class="sxs-lookup"><span data-stu-id="86c84-104">Property accessor already defined</span></span>  
  
 <span data-ttu-id="86c84-105">При объявлении [свойства](../programming-guide/classes-and-structs/using-properties.md)также необходимо объявить его методы доступа.</span><span class="sxs-lookup"><span data-stu-id="86c84-105">When you declare a [property](../programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="86c84-106">Однако свойство не может иметь несколько методов доступа `get` или несколько методов доступа `set` .</span><span class="sxs-lookup"><span data-stu-id="86c84-106">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="86c84-107">Пример</span><span class="sxs-lookup"><span data-stu-id="86c84-107">Example</span></span>  

 <span data-ttu-id="86c84-108">В следующем примере возникает ошибка CS1007:</span><span class="sxs-lookup"><span data-stu-id="86c84-108">The following sample generates CS1007:</span></span>  
  
```csharp  
// CS1007.cs  
public class clx  
{  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method  
        {  
            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
