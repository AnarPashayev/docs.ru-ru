# <a name="compiler-error-cs1623"></a><span data-ttu-id="b7117-102">Ошибка компилятора CS1623</span><span class="sxs-lookup"><span data-stu-id="b7117-102">Compiler Error CS1623</span></span>
<span data-ttu-id="b7117-103">Итераторы не могут иметь параметры out или ref</span><span class="sxs-lookup"><span data-stu-id="b7117-103">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="b7117-104">Эта ошибка возникает, если метод итератора принимает `in`, `ref`, или `out` параметра.</span><span class="sxs-lookup"><span data-stu-id="b7117-104">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="b7117-105">Чтобы избежать этой ошибки, удалите `in`, `ref`, или `out` ключевое слово из сигнатуры метода.</span><span class="sxs-lookup"><span data-stu-id="b7117-105">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b7117-106">Пример</span><span class="sxs-lookup"><span data-stu-id="b7117-106">Example</span></span>  
 <span data-ttu-id="b7117-107">Следующий пример приводит к возникновению ошибки CS1623:</span><span class="sxs-lookup"><span data-stu-id="b7117-107">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
