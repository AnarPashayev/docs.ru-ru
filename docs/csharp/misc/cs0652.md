<span data-ttu-id="d2b81-103">Сравнение с целочисленной константой не имеет смысла; константа находится вне диапазона значений типа "тип".</span><span class="sxs-lookup"><span data-stu-id="d2b81-103">Comparison to integral constant is useless; the constant is outside the range of type 'type'</span></span>  
  
 <span data-ttu-id="d2b81-104">Компилятор обнаружил сравнение константы и переменной, в котором константа выходит за пределы диапазона переменной.</span><span class="sxs-lookup"><span data-stu-id="d2b81-104">The compiler detected a comparison between a constant and a variable where the constant is out of the range of the variable.</span></span>  
  
 <span data-ttu-id="d2b81-105">В следующем примере возникает ошибка CS0652:</span><span class="sxs-lookup"><span data-stu-id="d2b81-105">The following sample generates CS0652:</span></span>  
  
```csharp  
// CS0652.cs  
// compile with: /W:2  
public class Class1  
{  
   private static byte i = 0;  
   public static void Main()  
   {  
      short j = 256;  
      if (i == 256)   // CS0652, 256 is out of range for byte  
         i = 0;  
   }  
}  
```
