# <a name="compiler-error-cs0400"></a><span data-ttu-id="46927-102">Ошибка компилятора CS0400</span><span class="sxs-lookup"><span data-stu-id="46927-102">Compiler Error CS0400</span></span>
<span data-ttu-id="46927-103">Не удалось найти имя типа или пространства имен "идентификатор" в глобальном пространстве имен (возможно, отсутствует ссылка на сборку).</span><span class="sxs-lookup"><span data-stu-id="46927-103">The type or namespace name 'identifier' could not be found in the global namespace (are you missing an assembly reference?)</span></span>  
  
 <span data-ttu-id="46927-104">Идентификатор с областью, указанной с помощью оператора глобальной области (`::`), не найден в глобальном пространстве имен.</span><span class="sxs-lookup"><span data-stu-id="46927-104">The identifier scoped with the global scope operator (`::`) was not found in the global namespace.</span></span> <span data-ttu-id="46927-105">Возможно, отсутствует ссылка на сборку, содержащую этот идентификатор, или идентификатор объявлен в классе или в пространстве имен, отличном от глобального пространства имен.</span><span class="sxs-lookup"><span data-stu-id="46927-105">You may be missing an assembly reference that contains the identifier, or the identifier may be declared in a class or namespace other than the global namespace.</span></span> <span data-ttu-id="46927-106">Эта ошибка может также возникать, если идентификатор с глобальной областью не объявлен или написан неправильно.</span><span class="sxs-lookup"><span data-stu-id="46927-106">This error could also occur if a globally scoped identifier is not declared or is misspelled.</span></span>  
  
 <span data-ttu-id="46927-107">Чтобы исключить эту ошибку, найдите объявление идентификатора, проверьте правильность написания, и если объявление находится в отдельной сборке, убедитесь в наличии соответствующей ссылки на сборку.</span><span class="sxs-lookup"><span data-stu-id="46927-107">To avoid this error, locate the declaration of the identifier, verify the correct spelling, and if the declaration is in a separate assembly, make sure that you have the appropriate assembly reference.</span></span> <span data-ttu-id="46927-108">Если идентификатор объявлен внутри другого типа или пространства имен, используйте полное имя после "::".</span><span class="sxs-lookup"><span data-stu-id="46927-108">If the identifier is declared inside another type or namespace, use the fully-qualified name after the ::.</span></span> <span data-ttu-id="46927-109">При компиляции следующего примера возникнет ошибка CS0400:</span><span class="sxs-lookup"><span data-stu-id="46927-109">The following sample generates CS0400:</span></span>  
  
```csharp  
// CS0400.cs  
class C  
{  
    public static void Main()  
    {  
        // CS0400 - D could not be found
        // in the global namespace.  
        global::D d = new global::D();  
   }  
}  
```
