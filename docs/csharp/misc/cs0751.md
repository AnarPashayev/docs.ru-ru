# <a name="compiler-error-cs0751"></a><span data-ttu-id="9a430-103">Ошибка компилятора CS0751</span><span class="sxs-lookup"><span data-stu-id="9a430-103">Compiler Error CS0751</span></span>

<span data-ttu-id="9a430-104">Разделяемый метод должен быть объявлен в разделяемом классе или разделяемой структуре</span><span class="sxs-lookup"><span data-stu-id="9a430-104">A partial method must be declared in a partial class or partial struct</span></span>  
  
 <span data-ttu-id="9a430-105">Невозможно объявить [разделяемый](../language-reference/keywords/partial-method.md) метод, если он не инкапсулирован в разделяемом классе или разделяемой структуре.</span><span class="sxs-lookup"><span data-stu-id="9a430-105">It is not possible to declare a [partial](../language-reference/keywords/partial-method.md) method unless it is encapsulated inside a partial class or partial struct.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="9a430-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="9a430-106">To correct this error</span></span>  
  
1. <span data-ttu-id="9a430-107">Удалите модификатор `partial` из метода и предоставьте реализацию или добавьте модификатор `partial` во включающий класс или структуру.</span><span class="sxs-lookup"><span data-stu-id="9a430-107">Either remove the `partial` modifier from the method and provide an implementation, or else add the `partial` modifier to the enclosing class or struct.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9a430-108">Пример</span><span class="sxs-lookup"><span data-stu-id="9a430-108">Example</span></span>  

 <span data-ttu-id="9a430-109">В следующем примере возникает ошибка CS0751:</span><span class="sxs-lookup"><span data-stu-id="9a430-109">The following example generates CS0751:</span></span>

    partial void Part(); // CS0751

    }
