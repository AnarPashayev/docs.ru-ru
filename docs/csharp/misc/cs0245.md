 <span data-ttu-id="23215-107">Следующий пример приводит к возникновению ошибки CS0245:</span><span class="sxs-lookup"><span data-stu-id="23215-107">The following sample generates CS0245:</span></span>  
  
```csharp  
// CS0245.cs  
using System;  
using System.Collections;  
  
class MyClass // : IDisposable  
{  
   /*  
   public void Dispose()  
   {  
      // cleanup code goes here  
   }  
   */  
  
   void m()  
   {  
      this.Finalize();   // CS0245  
      // this.Dispose();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
