 <span data-ttu-id="275ac-104">Разделяемый метод должен иметь определяющее объявление, которое определяет сигнатуру метода (имя, тип возвращаемого значения и параметры).</span><span class="sxs-lookup"><span data-stu-id="275ac-104">A partial method must have a defining declaration that defines the signature (name, return type and parameters) of the method.</span></span> <span data-ttu-id="275ac-105">Реализация тела метода необязательна.</span><span class="sxs-lookup"><span data-stu-id="275ac-105">The implementation or method body is optional.</span></span>

1. <span data-ttu-id="275ac-107">Укажите определяющее объявление разделяемого метода в другой части разделяемого класса или структуры.</span><span class="sxs-lookup"><span data-stu-id="275ac-107">Provide a defining declaration for the partial method in the other part of a partial class or struct.</span></span>
