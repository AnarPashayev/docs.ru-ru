 <span data-ttu-id="aea16-105">Разделяемый метод должен иметь определяющее объявление, которое определяет сигнатуру метода (имя, тип возвращаемого значения и параметры).</span><span class="sxs-lookup"><span data-stu-id="aea16-105">A partial method must have a defining declaration that defines the signature (name, return type and parameters) of the method.</span></span> <span data-ttu-id="aea16-106">Реализация тела метода необязательна.</span><span class="sxs-lookup"><span data-stu-id="aea16-106">The implementation or method body is optional.</span></span>
