# <a name="compiler-error-cs0682"></a>Ошибка компилятора CS0682
"тип1" не удается реализовать "тип2", так как он не поддерживается в данном языке  
  
 Эта ошибка возникает при попытке реализовать интерфейс, написанный на другом языке, если компилятор не поддерживает этот интерфейс.
