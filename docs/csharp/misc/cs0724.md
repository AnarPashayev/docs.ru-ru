# <a name="compiler-error-cs0724"></a><span data-ttu-id="a0ddd-103">Ошибка компилятора CS0724</span><span class="sxs-lookup"><span data-stu-id="a0ddd-103">Compiler Error CS0724</span></span>

<span data-ttu-id="a0ddd-104">Недопустимо использовать оператор throw без аргументов в предложении finally, которая находится в ближайшем вложенном предложении catch.</span><span class="sxs-lookup"><span data-stu-id="a0ddd-104">A throw statement with no arguments is not allowed in a finally clause that is nested inside the nearest enclosing catch clause</span></span>

## <a name="example"></a><span data-ttu-id="a0ddd-105">Пример</span><span class="sxs-lookup"><span data-stu-id="a0ddd-105">Example</span></span>

<span data-ttu-id="a0ddd-106">Следующий пример приводит к возникновению ошибки CS0724 из-за `throw` оператора внутри `finally` блока предложения:</span><span class="sxs-lookup"><span data-stu-id="a0ddd-106">The following example generates CS0724 because of the `throw` statement inside the `finally` clause block:</span></span>

```csharp  
// CS0724.cs  
using System;  
  
class Program
{
    static void Test()
    {
        try
        {
            throw new Exception();
        }
        catch
        {
            try
            {
            }
            finally
            {
                throw; // CS0724
            }
        }
    }

    static void Main()
    {
    }
}
```
