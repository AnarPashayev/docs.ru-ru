# <a name="compiler-error-cs4009"></a>Ошибка компилятора CS4009

"Type. Method": точка входа не может быть помечена `async` модификатором.

Нельзя использовать `async` ключевое слово в точке входа приложения (обычно это `Main` метод).  

> [!IMPORTANT]
> Начиная с C# 7,1, `Main` `async` метод может иметь модификатор. Дополнительные сведения см. в разделе [асинхронный основной](../whats-new/csharp-7-1.md#async-main) раздел статьи [новые возможности C# в 7,1](../whats-new/csharp-7-1.md) . Сведения о том, как выбрать C# языковую версию, см. в статье [Выбор C# языковой версии](../language-reference/configure-language-version.md) .

В следующем примере создается CS4009:

```csharp
using System;
using System.Threading.Tasks;

public class Example
{
   public static async void Main()
   {
       Console.WriteLine("About to wait two seconds");
       await WaitTwoSeconds();
       Console.WriteLine("About to exit the program");
   }

   private static async Task WaitTwoSeconds()
   {
      await Task.Delay(2000);
      Console.WriteLine("Returning from an asynchronous method");
   } 
}
```

## <a name="to-correct-this-error"></a>Исправление ошибки

[Обновите C# языковую версию](../language-reference/configure-language-version.md) , используемую проектом, до 7,1 или более поздней версии.

Если используется C# 7,0 или более ранняя версия, удалите `async` ключевое слово из сигнатуры точки входа приложения.  Также удалите все `await` ключевые слова, которые использовались для ожидания асинхронных методов в точке входа приложения. 

Однако по-прежнему необходимо дождаться завершения асинхронного метода до того, как точка входа возобновит выполнение.
