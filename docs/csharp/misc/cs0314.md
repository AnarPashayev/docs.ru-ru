# <a name="compiler-error-cs0314"></a><span data-ttu-id="09170-103">Ошибка компилятора CS0314</span><span class="sxs-lookup"><span data-stu-id="09170-103">Compiler Error CS0314</span></span>

<span data-ttu-id="09170-104">Тип "тип1" не может быть использован как параметр-тип "имя" в универсальном типе или методе "имя".</span><span class="sxs-lookup"><span data-stu-id="09170-104">The type 'type1' cannot be used as type parameter 'name' in the generic type or method 'name'.</span></span> <span data-ttu-id="09170-105">Нет преобразования-упаковки или преобразования параметра-типа из "тип1" в "тип2".</span><span class="sxs-lookup"><span data-stu-id="09170-105">There is no boxing conversion or type parameter conversion from 'type1' to 'type2'.</span></span>  
  
 <span data-ttu-id="09170-106">Если универсальным типом используется ограниченный параметр типа, то новый класс также должен удовлетворять тем же ограничениям.</span><span class="sxs-lookup"><span data-stu-id="09170-106">When a generic type uses a type parameter that is constrained, the new class must also satisfy those same constraints.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="09170-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="09170-107">To correct this error</span></span>  
  
1. <span data-ttu-id="09170-108">В приведенном ниже примере добавьте `where T : ClassConstraint` в класс `B`.</span><span class="sxs-lookup"><span data-stu-id="09170-108">In the example that follows, add `where T : ClassConstraint` to class `B`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="09170-109">Пример</span><span class="sxs-lookup"><span data-stu-id="09170-109">Example</span></span>  

 <span data-ttu-id="09170-110">Следующий код приводит к возникновению ошибки CS0314:</span><span class="sxs-lookup"><span data-stu-id="09170-110">The following code generates CS0314:</span></span>  
  
```csharp  
// cs0314.cs  
// Compile with: /target:library  
public class ClassConstraint { }  
  
public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.  
public class C<T> : A<T> where T : ClassConstraint  
{ }  
```
