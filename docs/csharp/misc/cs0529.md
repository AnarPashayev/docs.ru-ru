# <a name="compiler-error-cs0529"></a><span data-ttu-id="a576d-103">Ошибка компилятора CS0529</span><span class="sxs-lookup"><span data-stu-id="a576d-103">Compiler Error CS0529</span></span>

<span data-ttu-id="a576d-104">Наследуемый интерфейс "интерфейс1" образует циклическую ссылку в иерархии интерфейсов "интерфейс2".</span><span class="sxs-lookup"><span data-stu-id="a576d-104">Inherited interface 'interface1' causes a cycle in the interface hierarchy of 'interface2'</span></span>  
  
 <span data-ttu-id="a576d-105">Список наследования для [интерфейса](../language-reference/keywords/interface.md) включает прямую или косвенную ссылку на самого себя.</span><span class="sxs-lookup"><span data-stu-id="a576d-105">The inheritance list for an [interface](../language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="a576d-106">Интерфейс не может наследовать от самого себя.</span><span class="sxs-lookup"><span data-stu-id="a576d-106">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="a576d-107">В следующем примере возникает ошибка CS0529:</span><span class="sxs-lookup"><span data-stu-id="a576d-107">The following sample generates CS0529:</span></span>  
  
```csharp  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
