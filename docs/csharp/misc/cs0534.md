# <a name="compiler-error-cs0534"></a><span data-ttu-id="77820-102">Ошибка компилятора CS0534</span><span class="sxs-lookup"><span data-stu-id="77820-102">Compiler Error CS0534</span></span>
<span data-ttu-id="77820-103">"функция1" не реализует наследуемый абстрактный член "функция2"</span><span class="sxs-lookup"><span data-stu-id="77820-103">'function1' does not implement inherited abstract member 'function2'</span></span>  
  
 <span data-ttu-id="77820-104">Если класс не является абстрактным, он должен реализовывать все [абстрактные](../language-reference/keywords/abstract.md) члены базового класса.</span><span class="sxs-lookup"><span data-stu-id="77820-104">A class is required to implement all the [abstract](../language-reference/keywords/abstract.md) members in the base class, unless the class is also abstract.</span></span>  
  
 <span data-ttu-id="77820-105">Следующий пример приводит к возникновению ошибки CS0534:</span><span class="sxs-lookup"><span data-stu-id="77820-105">The following sample generates CS0534:</span></span>  
  
```csharp  
// CS0534.cs  
namespace x  
{  
   abstract public class clx  
   {  
      public abstract void f();  
   }  
  
   public class cly : clx   // CS0534, no override for clx::f  
   {  
      // uncomment the following sample override to resolve CS0534  
      // override public void f()  
      // {  
      // }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
