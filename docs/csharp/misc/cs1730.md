# <a name="compiler-error-cs1730"></a><span data-ttu-id="2fff4-102">Ошибка компилятора CS1730</span><span class="sxs-lookup"><span data-stu-id="2fff4-102">Compiler Error CS1730</span></span>
<span data-ttu-id="2fff4-103">Атрибуты сборки и модуля должны находиться перед всеми остальными элементами в файле, кроме предложений using и описаний внешних псевдонимов.</span><span class="sxs-lookup"><span data-stu-id="2fff4-103">Assembly and module attributes must precede all other elements defined in a file except using clauses and extern alias declarations.</span></span>  
  
 <span data-ttu-id="2fff4-104">Атрибут, применяемый на уровне сборки, не может указываться после определений типов.</span><span class="sxs-lookup"><span data-stu-id="2fff4-104">An attribute applied at the assembly level cannot appear after any type definitions.</span></span>
