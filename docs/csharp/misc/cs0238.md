# <a name="compiler-error-cs0238"></a><span data-ttu-id="91518-103">Ошибка компилятора CS0238</span><span class="sxs-lookup"><span data-stu-id="91518-103">Compiler Error CS0238</span></span>

<span data-ttu-id="91518-104">"элемент" не может быть запечатанным, так как не является результатом переопределения</span><span class="sxs-lookup"><span data-stu-id="91518-104">'member' cannot be sealed because it is not an override</span></span>  
  
 <span data-ttu-id="91518-105">Значение[sealed](../language-reference/keywords/sealed.md) использовалось для "элемент", который не был помечен атрибутом [override](../language-reference/keywords/override.md).</span><span class="sxs-lookup"><span data-stu-id="91518-105">[sealed](../language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../language-reference/keywords/override.md).</span></span> <span data-ttu-id="91518-106">Дополнительные сведения см. в разделе [Наследование](../programming-guide/classes-and-structs/inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="91518-106">For more information, see [Inheritance](../programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="91518-107">При компиляции следующего примера возникнет ошибка CS0238:</span><span class="sxs-lookup"><span data-stu-id="91518-107">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
