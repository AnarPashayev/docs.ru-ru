# <a name="compiler-error-cs0077"></a><span data-ttu-id="49cd2-102">Ошибка компилятора CS0077</span><span class="sxs-lookup"><span data-stu-id="49cd2-102">Compiler Error CS0077</span></span>
<span data-ttu-id="49cd2-103">Оператор as необходимо использовать с типом ссылки или типом, допускающим значение null (int является типом, не допускающим значение null).</span><span class="sxs-lookup"><span data-stu-id="49cd2-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="49cd2-104">В оператор [as](../language-reference/operators/type-testing-and-cast.md#as-operator) был передан [тип значения](../language-reference/keywords/value-types.md).</span><span class="sxs-lookup"><span data-stu-id="49cd2-104">The [as](../language-reference/operators/type-testing-and-cast.md#as-operator) operator was passed a [value type](../language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="49cd2-105">Поскольку `as` может возвращать [значение NULL](../language-reference/keywords/null.md), ему может быть передан только [ссылочный тип](../language-reference/keywords/reference-types.md) или [тип значения, допускающий значение NULL](../language-reference/builtin-types/nullable-value-types.md).</span><span class="sxs-lookup"><span data-stu-id="49cd2-105">Because `as` can return [null](../language-reference/keywords/null.md), it can only be passed a [reference type](../language-reference/keywords/reference-types.md) or a [nullable value type](../language-reference/builtin-types/nullable-value-types.md).</span></span>
  
 <span data-ttu-id="49cd2-106">Следующий пример приводит к возникновению ошибки CS0077:</span><span class="sxs-lookup"><span data-stu-id="49cd2-106">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
