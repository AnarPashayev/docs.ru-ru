# <a name="compiler-error-cs1040"></a><span data-ttu-id="91535-103">Ошибка компилятора CS1040</span><span class="sxs-lookup"><span data-stu-id="91535-103">Compiler Error CS1040</span></span>

<span data-ttu-id="91535-104">Директивы препроцессора должны располагаться в качестве первого символа, отличного от пробела, в строке</span><span class="sxs-lookup"><span data-stu-id="91535-104">Preprocessor directives must appear as the first non-white-space character on a line</span></span>  
  
 <span data-ttu-id="91535-105">В строке обнаружена [директива препроцессора](../language-reference/preprocessor-directives/index.md) , однако эта директива не является первым токеном в строке.</span><span class="sxs-lookup"><span data-stu-id="91535-105">A [preprocessor directive](../language-reference/preprocessor-directives/index.md) was found on a line and was not the first token on the line.</span></span> <span data-ttu-id="91535-106">Директива должна быть первым токеном в строке.</span><span class="sxs-lookup"><span data-stu-id="91535-106">A directive must be the first token on the line.</span></span>  
  
 <span data-ttu-id="91535-107">Следующий пример приводит к возникновению ошибки CS1040:</span><span class="sxs-lookup"><span data-stu-id="91535-107">The following sample generates CS1040:</span></span>  
  
```csharp  
// CS1040.cs  
/* Define a symbol, X */ #define X   // CS1040  
  
// try the following two lines instead  
// /* Define a symbol, X */  
// #define X  
  
public class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
