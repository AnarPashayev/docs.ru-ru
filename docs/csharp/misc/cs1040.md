# <a name="compiler-error-cs1040"></a><span data-ttu-id="32e61-102">Ошибка компилятора CS1040</span><span class="sxs-lookup"><span data-stu-id="32e61-102">Compiler Error CS1040</span></span>
<span data-ttu-id="32e61-103">Директивы препроцессора должны располагаться в качестве первого символа, отличного от пробела, в строке</span><span class="sxs-lookup"><span data-stu-id="32e61-103">Preprocessor directives must appear as the first non-white-space character on a line</span></span>  
  
 <span data-ttu-id="32e61-104">В строке обнаружена [директива препроцессора](../language-reference/preprocessor-directives/index.md) , однако эта директива не является первым токеном в строке.</span><span class="sxs-lookup"><span data-stu-id="32e61-104">A [preprocessor directive](../language-reference/preprocessor-directives/index.md) was found on a line and was not the first token on the line.</span></span> <span data-ttu-id="32e61-105">Директива должна быть первым токеном в строке.</span><span class="sxs-lookup"><span data-stu-id="32e61-105">A directive must be the first token on the line.</span></span>  
  
 <span data-ttu-id="32e61-106">Следующий пример приводит к возникновению ошибки CS1040:</span><span class="sxs-lookup"><span data-stu-id="32e61-106">The following sample generates CS1040:</span></span>  
  
```csharp  
// CS1040.cs  
/* Define a symbol, X */ #define X   // CS1040  
  
// try the following two lines instead  
// /* Define a symbol, X */  
// #define X  
  
public class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
