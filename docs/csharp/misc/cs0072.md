# <a name="compiler-error-cs0072"></a><span data-ttu-id="9b3eb-102">Ошибка компилятора CS0072</span><span class="sxs-lookup"><span data-stu-id="9b3eb-102">Compiler Error CS0072</span></span>
<span data-ttu-id="9b3eb-103">"событие": невозможно переопределить; "метод" не является событием.</span><span class="sxs-lookup"><span data-stu-id="9b3eb-103">'event' : cannot override; 'method' is not an event</span></span>  
  
 <span data-ttu-id="9b3eb-104">[Событие](../language-reference/keywords/event.md) может переопределяться только другим событием.</span><span class="sxs-lookup"><span data-stu-id="9b3eb-104">An [event](../language-reference/keywords/event.md) can only override another event.</span></span> <span data-ttu-id="9b3eb-105">Дополнительные сведения см. в статье [Events (Visual Basic)](../programming-guide/events/index.md) (События в Visual Basic).</span><span class="sxs-lookup"><span data-stu-id="9b3eb-105">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="9b3eb-106">В следующем примере возникает ошибка CS0072.</span><span class="sxs-lookup"><span data-stu-id="9b3eb-106">The following sample generates CS0072:</span></span>  
  
```csharp  
// CS0072.cs  
delegate void MyDelegate();  
  
class Test1  
{  
   public virtual event MyDelegate MyEvent;  
   public virtual void VMeth()  
   {  
   }  
  
   public void FireAway()  
   {  
      if (MyEvent != null)  
         MyEvent();  
   }  
}  
  
class Test2 : Test1  
{  
   public override event MyDelegate VMeth   // CS0072  
   // uncomment the following lines to resolve  
   // public override event MyDelegate MyEvent  
   {  
      add  
      {  
         VMeth += value;  
         // MyEvent += value;  
      }  
      remove  
      {  
         VMeth -= value;  
         // MyEvent -= value;  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
