## <a name="example"></a><span data-ttu-id="6a8c3-106">Пример</span><span class="sxs-lookup"><span data-stu-id="6a8c3-106">Example</span></span>

 <span data-ttu-id="6a8c3-107">В следующем примере возникает ошибка CS0072.</span><span class="sxs-lookup"><span data-stu-id="6a8c3-107">The following sample generates CS0072:</span></span>

```csharp
// CS0072.cs
delegate void MyDelegate();

class Test1
{
    public virtual event MyDelegate MyEvent;
    public virtual void VMeth()
    {
    }

    public void FireAway()
    {
        if (MyEvent != null)
            MyEvent();
    }
}

class Test2 : Test1
{
    public override event MyDelegate VMeth   // CS0072
    // Uncomment the following lines to resolve.
    // public override event MyDelegate MyEvent
    {
        add
        {
            VMeth += value;
            // MyEvent += value;
        }
        remove
        {
            VMeth -= value;
            // MyEvent -= value;
        }
   }

   public static void Main()
   {
   }
}
```
