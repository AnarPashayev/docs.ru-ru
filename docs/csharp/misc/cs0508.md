<span data-ttu-id="d8096-104">"Тип 1": типом возврата должен быть "Тип 2", чтобы соответствовать переопределенному члену "Имя_члена".</span><span class="sxs-lookup"><span data-stu-id="d8096-104">'Type 1': return type must be 'Type 2' to match overridden member 'Member Name'</span></span>

## <a name="example"></a><span data-ttu-id="d8096-107">Пример</span><span class="sxs-lookup"><span data-stu-id="d8096-107">Example</span></span>  

 <span data-ttu-id="d8096-108">Следующий пример приводит к возникновению ошибки CS0508.</span><span class="sxs-lookup"><span data-stu-id="d8096-108">The following sample generates CS0508.</span></span>  
  
```csharp  
// CS0508.cs  
// compile with: /target:library  
abstract public class Clx  
{  
   public int i = 0;  
   // Return type is int.  
   abstract public int F();  
}  
  
public class Cly : Clx  
{  
   public override double F()  
   {  
      return 0.0;   // CS0508  
   }  
}  
```
