# <a name="compiler-error-cs0037"></a>Ошибка компилятора CS0037
Невозможно преобразовать null в "тип", так как это тип значения, не допускающий значение null  
  
 Компилятору не удается присвоить значение `null` типу значения; `null` может быть назначен только [ссылочному типу](../language-reference/keywords/reference-types.md) или [типу значения, допускающему значение NULL](../programming-guide/nullable-types/index.md). Тип[struct](../language-reference/keywords/struct.md) является типом значения.
  
 Следующий пример приводит к возникновению ошибки CS0037:  
  
```csharp  
// CS0037.cs  
public struct s  
{  
}  
  
class a  
{  
   public static void Main()  
   {  
      int i = null;   // CS0037  
      s ss = null;    // CS0037  
   }  
}  
```
