# <a name="compiler-warning-level-2-cs0435"></a><span data-ttu-id="aee3b-103">Предупреждение компилятора (уровень 2) CS0435</span><span class="sxs-lookup"><span data-stu-id="aee3b-103">Compiler Warning (level 2) CS0435</span></span>

<span data-ttu-id="aee3b-104">Пространство имен "пространство_имен" в сборке "сборка" конфликтует с импортированным типом "тип" в сборке "сборка".</span><span class="sxs-lookup"><span data-stu-id="aee3b-104">The namespace 'namespace' in 'assembly' conflicts with the imported type 'type' in 'assembly'.</span></span> <span data-ttu-id="aee3b-105">Использование пространства имен, определенного в сборке "сборка".</span><span class="sxs-lookup"><span data-stu-id="aee3b-105">Using the namespace defined in 'assembly'.</span></span>  
  
 <span data-ttu-id="aee3b-106">Это предупреждение выдается, когда пространство имен в исходном файле (file_2) конфликтует с импортированным пространством имен в file_1.</span><span class="sxs-lookup"><span data-stu-id="aee3b-106">This warning is issued when a namespace in a source file (file_2) conflicts with an imported type in file_1.</span></span> <span data-ttu-id="aee3b-107">Компилятор использует пространство имен из исходного файла.</span><span class="sxs-lookup"><span data-stu-id="aee3b-107">The compiler uses the one in the source file.</span></span>  
  
 <span data-ttu-id="aee3b-108">Следующий пример приводит к возникновению ошибки CS0435:</span><span class="sxs-lookup"><span data-stu-id="aee3b-108">The following example generates CS0435:</span></span>  
  
 <span data-ttu-id="aee3b-109">Сначала скомпилируйте этот файл:</span><span class="sxs-lookup"><span data-stu-id="aee3b-109">Compile this file first:</span></span>  
  
```csharp  
// CS0435_1.cs  
// compile with: /t:library  
public class Util
{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="aee3b-110">Затем скомпилируйте этот файл:</span><span class="sxs-lookup"><span data-stu-id="aee3b-110">Then, compile this file:</span></span>  
  
```csharp  
// CS0435_2.cs  
// compile with: /r:CS0435_1.dll  
  
using System;  
  
namespace Util
{  
   public class A { }  
}  
  
public class Test
{  
   public static void Main()
   {  
      Console.WriteLine(typeof(Util.A)); // CS0435  
   }  
}  
```
