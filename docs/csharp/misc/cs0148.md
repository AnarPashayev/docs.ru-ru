# <a name="compiler-error-cs0148"></a>Ошибка компилятора CS0148
Делегат "делегат" не имеет допустимого конструктора  
  
 Вы импортировали и использовали управляемую программу (которая использует среду выполнения .NET), созданную с помощью другого компилятора. Этот компилятор разрешил неправильно сформированный конструктор [делегата](../language-reference/builtin-types/reference-types.md) . Дополнительные сведения см. в разделе [Делегаты](../programming-guide/delegates/index.md).
