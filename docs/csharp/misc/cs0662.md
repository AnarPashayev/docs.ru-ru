# <a name="compiler-error-cs0662"></a><span data-ttu-id="5f8fc-102">Ошибка компилятора CS0662</span><span class="sxs-lookup"><span data-stu-id="5f8fc-102">Compiler Error CS0662</span></span>

<span data-ttu-id="5f8fc-103">"метод" не может задавать для параметра ref только атрибут Out.</span><span class="sxs-lookup"><span data-stu-id="5f8fc-103">'method' cannot specify only Out attribute on a ref parameter.</span></span> <span data-ttu-id="5f8fc-104">Используйте атрибуты In и Out вместе или не используйте их.</span><span class="sxs-lookup"><span data-stu-id="5f8fc-104">Use both In and Out attributes, or neither.</span></span>  
  
 <span data-ttu-id="5f8fc-105">Метод интерфейса имеет параметр, который использует [ref](../../csharp/language-reference/keywords/ref.md) только с атрибутом [Out](xref:System.Runtime.InteropServices.OutAttribute) .</span><span class="sxs-lookup"><span data-stu-id="5f8fc-105">An interface method has a parameter that uses [ref](../../csharp/language-reference/keywords/ref.md) with just the [Out](xref:System.Runtime.InteropServices.OutAttribute) attribute.</span></span> <span data-ttu-id="5f8fc-106">Параметр `ref` , который использует атрибут **Out** , должен также использовать атрибут [In](xref:System.Runtime.InteropServices.InAttribute) .</span><span class="sxs-lookup"><span data-stu-id="5f8fc-106">A `ref` parameter that uses the **Out** attribute must also use the [In](xref:System.Runtime.InteropServices.InAttribute) attribute.</span></span>  
  
 <span data-ttu-id="5f8fc-107">Следующий пример приводит к возникновению ошибки CS0662:</span><span class="sxs-lookup"><span data-stu-id="5f8fc-107">The following sample generates CS0662:</span></span>  
  
```csharp
// CS0662.cs  
using System.Runtime.InteropServices;  
  
interface I  
{  
   void method([Out] ref int i);   // CS0662  
   // try one of the following lines instead  
   // void method(ref int i);  
   // void method([Out, In]ref int i);  
}  
  
class test  
{  
   public static void Main()  
   {  
   }  
}  
```
