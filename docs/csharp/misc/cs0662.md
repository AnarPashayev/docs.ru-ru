# <a name="compiler-error-cs0662"></a><span data-ttu-id="a4a5c-103">Ошибка компилятора CS0662</span><span class="sxs-lookup"><span data-stu-id="a4a5c-103">Compiler Error CS0662</span></span>

<span data-ttu-id="a4a5c-104">"метод" не может задавать для параметра ref только атрибут Out.</span><span class="sxs-lookup"><span data-stu-id="a4a5c-104">'method' cannot specify only Out attribute on a ref parameter.</span></span> <span data-ttu-id="a4a5c-105">Используйте атрибуты In и Out вместе или не используйте их.</span><span class="sxs-lookup"><span data-stu-id="a4a5c-105">Use both In and Out attributes, or neither.</span></span>  
  
 <span data-ttu-id="a4a5c-106">Метод интерфейса имеет параметр, который использует [ref](../language-reference/keywords/ref.md) только с атрибутом [Out](xref:System.Runtime.InteropServices.OutAttribute) .</span><span class="sxs-lookup"><span data-stu-id="a4a5c-106">An interface method has a parameter that uses [ref](../language-reference/keywords/ref.md) with just the [Out](xref:System.Runtime.InteropServices.OutAttribute) attribute.</span></span> <span data-ttu-id="a4a5c-107">Параметр `ref` , который использует атрибут **Out** , должен также использовать атрибут [In](xref:System.Runtime.InteropServices.InAttribute) .</span><span class="sxs-lookup"><span data-stu-id="a4a5c-107">A `ref` parameter that uses the **Out** attribute must also use the [In](xref:System.Runtime.InteropServices.InAttribute) attribute.</span></span>  
  
 <span data-ttu-id="a4a5c-108">Следующий пример приводит к возникновению ошибки CS0662:</span><span class="sxs-lookup"><span data-stu-id="a4a5c-108">The following sample generates CS0662:</span></span>  
  
```csharp
// CS0662.cs  
using System.Runtime.InteropServices;  
  
interface I  
{  
   void method([Out] ref int i);   // CS0662  
   // try one of the following lines instead  
   // void method(ref int i);  
   // void method([Out, In]ref int i);  
}  
  
class test  
{  
   public static void Main()  
   {  
   }  
}  
```
