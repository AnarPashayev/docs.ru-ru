<span data-ttu-id="0950e-104">Модификаторы partial могут появляться только перед "class", "struct" или "interface"</span><span class="sxs-lookup"><span data-stu-id="0950e-104">The partial modifier can only appear immediately before 'class', 'struct', or 'interface'</span></span>  
  
 <span data-ttu-id="0950e-105">В объявлении класса, структуры или интерфейса неверно расположен модификатор **partial** .</span><span class="sxs-lookup"><span data-stu-id="0950e-105">The placement of the **partial** modifier was incorrect in the declaration of the class, struct or interface.</span></span> <span data-ttu-id="0950e-106">Для устранения этой ошибки измените порядок модификаторов.</span><span class="sxs-lookup"><span data-stu-id="0950e-106">To fix the error, reorder the modifiers.</span></span> <span data-ttu-id="0950e-107">Дополнительные сведения см. в разделе [Разделяемые классы и методы](../programming-guide/classes-and-structs/partial-classes-and-methods.md).</span><span class="sxs-lookup"><span data-stu-id="0950e-107">For more information, see [Partial Classes and Methods](../programming-guide/classes-and-structs/partial-classes-and-methods.md).</span></span>  
  
 <span data-ttu-id="0950e-108">Следующий пример приводит к возникновению ошибки CS0267:</span><span class="sxs-lookup"><span data-stu-id="0950e-108">The following sample generates CS0267:</span></span>  
  
```csharp  
// CS0267.cs  
public partial class MyClass  
{  
   public MyClass()  
   {  
   }  
}  
  
partial public class MyClass  // CS0267  
// Try this line instead:  
// public partial class MyClass  
{  
   public void Foo()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
