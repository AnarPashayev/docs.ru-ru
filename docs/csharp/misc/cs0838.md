 <span data-ttu-id="6cfe6-104">Многомерные массивы в деревьях выражений нельзя инициализировать с помощью инициализатора массива.</span><span class="sxs-lookup"><span data-stu-id="6cfe6-104">Multidimensional arrays in expression trees cannot be initialized by using an array initializer.</span></span>

## <a name="example"></a><span data-ttu-id="6cfe6-107">Пример</span><span class="sxs-lookup"><span data-stu-id="6cfe6-107">Example</span></span>  
 <span data-ttu-id="6cfe6-108">В следующем примере возникает ошибка CS0838:</span><span class="sxs-lookup"><span data-stu-id="6cfe6-108">The following example generates CS0838:</span></span>  
  
```csharp  
// cs0838.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace TestNamespace  
{  
    class Test  
    {  
        static int Main()  
        {  
  
            Expression<Func<int[,]>> expr =  
                () => new int[2, 2] { { 1, 2 }, { 3, 4 } }; // CS0838  
  
            // try the following 2 lines instead  
            int[,] nums = new int[2, 2] { { 1, 2 }, { 3, 4 } };  
            Expression<Func<int[,]>> expr2 = () => nums;
  
            return 1;  
        }  
    }  
}  
```
