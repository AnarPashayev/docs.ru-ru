# <a name="compiler-error-cs0838"></a><span data-ttu-id="7bfea-103">Ошибка компилятора CS0838</span><span class="sxs-lookup"><span data-stu-id="7bfea-103">Compiler Error CS0838</span></span>
<span data-ttu-id="7bfea-104">Дерево выражения не может содержать инициализатор многомерного массива.</span><span class="sxs-lookup"><span data-stu-id="7bfea-104">An expression tree may not contain a multidimensional array initializer.</span></span>  
  
 <span data-ttu-id="7bfea-105">Многомерные массивы в деревьях выражений нельзя инициализировать с помощью инициализатора массива.</span><span class="sxs-lookup"><span data-stu-id="7bfea-105">Multidimensional arrays in expression trees cannot be initialized by using an array initializer.</span></span>

## <a name="example"></a><span data-ttu-id="7bfea-108">Пример</span><span class="sxs-lookup"><span data-stu-id="7bfea-108">Example</span></span>  
 <span data-ttu-id="7bfea-109">В следующем примере возникает ошибка CS0838:</span><span class="sxs-lookup"><span data-stu-id="7bfea-109">The following example generates CS0838:</span></span>  
  
```csharp  
// cs0838.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace TestNamespace  
{  
    class Test  
    {  
        static int Main()  
        {  
  
            Expression<Func<int[,]>> expr =  
                () => new int[2, 2] { { 1, 2 }, { 3, 4 } }; // CS0838  
  
            // try the following 2 lines instead  
            int[,] nums = new int[2, 2] { { 1, 2 }, { 3, 4 } };  
            Expression<Func<int[,]>> expr2 = () => nums;
  
            return 1;  
        }  
    }  
}  
```
