## <a name="example"></a><span data-ttu-id="00c67-107">Пример</span><span class="sxs-lookup"><span data-stu-id="00c67-107">Example</span></span>

```csharp

namespace x  
{  
    public class b : Exception  
    {  
    }  
  
    public class a  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
  
            catch   // CS1017, must be last catch  
            {  
            }  
  
            catch(b)  
            {  
                throw;  
            }  
        }  
    }  
}  
```
