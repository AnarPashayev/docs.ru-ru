# <a name="compiler-error-cs1627"></a><span data-ttu-id="cd196-103">Ошибка компилятора CS1627</span><span class="sxs-lookup"><span data-stu-id="cd196-103">Compiler Error CS1627</span></span>
<span data-ttu-id="cd196-104">Требуется выражение после оператора yield return.</span><span class="sxs-lookup"><span data-stu-id="cd196-104">Expression expected after yield return</span></span>  
  
 <span data-ttu-id="cd196-105">Эта ошибка возникает, если `yield` используется без выражения.</span><span class="sxs-lookup"><span data-stu-id="cd196-105">This error occurs if `yield` is used without an expression.</span></span> <span data-ttu-id="cd196-106">Чтобы избежать этой ошибки, вставьте в оператор соответствующее выражение.</span><span class="sxs-lookup"><span data-stu-id="cd196-106">To avoid this error, insert the appropriate expression in the statement.</span></span>  
  
 <span data-ttu-id="cd196-107">Следующий пример приводит к возникновению ошибки CS1627:</span><span class="sxs-lookup"><span data-stu-id="cd196-107">The following sample generates CS1627:</span></span>  
  
```csharp  
// CS1627.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      yield return;   // CS1627  
      // To resolve, add the following line:  
      // yield return 0;  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
