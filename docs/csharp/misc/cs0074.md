 <span data-ttu-id="1b524-104">Если [событие](../language-reference/keywords/event.md) помечено как **абстрактное**, оно не может быть инициализировано.</span><span class="sxs-lookup"><span data-stu-id="1b524-104">If an [event](../language-reference/keywords/event.md) is marked as **abstract**, it cannot be initialized.</span></span> <span data-ttu-id="1b524-105">Дополнительные сведения см. в статье [Events (Visual Basic)](../programming-guide/events/index.md) (События в Visual Basic).</span><span class="sxs-lookup"><span data-stu-id="1b524-105">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="1b524-106">В следующем примере возникает ошибка CS0074.</span><span class="sxs-lookup"><span data-stu-id="1b524-106">The following sample generates CS0074:</span></span>  
  
```csharp  
// CS0074.cs

{  
   public abstract event D e = null;   // CS0074  
   // try the following line instead  
   // public abstract event D e;  
  
   public static void Main()  
   {  
   }  
}  
```
