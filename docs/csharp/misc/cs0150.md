# <a name="compiler-error-cs0150"></a><span data-ttu-id="52c20-103">Ошибка компилятора CS0150</span><span class="sxs-lookup"><span data-stu-id="52c20-103">Compiler Error CS0150</span></span>
<span data-ttu-id="52c20-104">Требуется постоянное значение.</span><span class="sxs-lookup"><span data-stu-id="52c20-104">A constant value is expected</span></span>  
  
 <span data-ttu-id="52c20-105">Переменная найдена там, где ожидается константа.</span><span class="sxs-lookup"><span data-stu-id="52c20-105">A variable was found where a constant was expected.</span></span> <span data-ttu-id="52c20-106">Дополнительные сведения см. в разделе [выбора](../language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="52c20-106">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="52c20-107">Следующий пример приводит к возникновению ошибки CS0150.</span><span class="sxs-lookup"><span data-stu-id="52c20-107">The following sample generates CS0150:</span></span>  
  
```csharp  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="52c20-108">Эта ошибка может также возникать, если размер массива указан с помощью значения переменной и инициализирован с помощью инициализатора массива.</span><span class="sxs-lookup"><span data-stu-id="52c20-108">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="52c20-109">Чтобы устранить эту ошибку, инициализируйте массив в отдельной инструкции или инструкциях.</span><span class="sxs-lookup"><span data-stu-id="52c20-109">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```csharp  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
