# <a name="compiler-error-cs0557"></a><span data-ttu-id="9994b-103">Ошибка компилятора CS0557</span><span class="sxs-lookup"><span data-stu-id="9994b-103">Compiler Error CS0557</span></span>

<span data-ttu-id="9994b-104">Повторяющееся пользовательское преобразование в типе "класс"</span><span class="sxs-lookup"><span data-stu-id="9994b-104">Duplicate user-defined conversion in type 'class'</span></span>  
  
 <span data-ttu-id="9994b-105">Повторяющиеся процедуры преобразования не разрешены в классе.</span><span class="sxs-lookup"><span data-stu-id="9994b-105">Duplicate conversion routines are not allowed in a class.</span></span>  
  
 <span data-ttu-id="9994b-106">В следующем примере возникает ошибка CS0557:</span><span class="sxs-lookup"><span data-stu-id="9994b-106">The following example generates CS0557:</span></span>  
  
```csharp  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
