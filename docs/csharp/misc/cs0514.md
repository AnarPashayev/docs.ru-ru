# <a name="compiler-error-cs0514"></a><span data-ttu-id="34caf-103">Ошибка компилятора CS0514</span><span class="sxs-lookup"><span data-stu-id="34caf-103">Compiler Error CS0514</span></span>
<span data-ttu-id="34caf-104">"конструктор": статический конструктор не может иметь явный вызов конструктора с this или base</span><span class="sxs-lookup"><span data-stu-id="34caf-104">'constructor' : static constructor cannot have an explicit 'this' or 'base' constructor call</span></span>  
  
 <span data-ttu-id="34caf-105">Не допускается вызывать `this` в статическом конструкторе, так как статический конструктор вызывается автоматически до создания любого экземпляра класса.</span><span class="sxs-lookup"><span data-stu-id="34caf-105">Calling `this` in the static constructor is not allowed because the static constructor is called automatically before creating any instance of the class.</span></span> <span data-ttu-id="34caf-106">Кроме того, статические конструкторы не наследуются и не могут вызываться непосредственно.</span><span class="sxs-lookup"><span data-stu-id="34caf-106">Also, static constructors are not inherited, and cannot be called directly.</span></span>  
  
 <span data-ttu-id="34caf-107">Дополнительные сведения см. в статье об [этом](../language-reference/keywords/this.md) и [базовом](../language-reference/keywords/base.md).</span><span class="sxs-lookup"><span data-stu-id="34caf-107">For more information, see [this](../language-reference/keywords/this.md) and [base](../language-reference/keywords/base.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="34caf-108">Пример</span><span class="sxs-lookup"><span data-stu-id="34caf-108">Example</span></span>

```csharp

{  
    static A() : base(0) // CS0514  
    {  
    }  
  
    public A(object o)  
    {  
    }  
}

{  
    static B() : this(null) // CS0514  
    {  
    }  
  
    public B(object o)  
    {  
    }  
}  
```
