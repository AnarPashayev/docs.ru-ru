# <a name="compiler-error-cs0514"></a><span data-ttu-id="46a71-102">Ошибка компилятора CS0514</span><span class="sxs-lookup"><span data-stu-id="46a71-102">Compiler Error CS0514</span></span>
<span data-ttu-id="46a71-103">"конструктор": статический конструктор не может иметь явный вызов конструктора с this или base</span><span class="sxs-lookup"><span data-stu-id="46a71-103">'constructor' : static constructor cannot have an explicit 'this' or 'base' constructor call</span></span>  
  
 <span data-ttu-id="46a71-104">Не допускается вызывать `this` в статическом конструкторе, так как статический конструктор вызывается автоматически до создания любого экземпляра класса.</span><span class="sxs-lookup"><span data-stu-id="46a71-104">Calling `this` in the static constructor is not allowed because the static constructor is called automatically before creating any instance of the class.</span></span> <span data-ttu-id="46a71-105">Кроме того, статические конструкторы не наследуются и не могут вызываться непосредственно.</span><span class="sxs-lookup"><span data-stu-id="46a71-105">Also, static constructors are not inherited, and cannot be called directly.</span></span>  
  
 <span data-ttu-id="46a71-106">Дополнительные сведения см. в [статье](../language-reference/keywords/base.md)об [этом](../language-reference/keywords/this.md) и базовом.</span><span class="sxs-lookup"><span data-stu-id="46a71-106">For more information, see [this](../language-reference/keywords/this.md) and [base](../language-reference/keywords/base.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="46a71-107">Пример</span><span class="sxs-lookup"><span data-stu-id="46a71-107">Example</span></span>

```csharp

    {  
    }  
  
    public A(object o)

    }  
}

{  
    static B() : this(null) // CS0514  
    {  
    }  
  
    public B(object o)  
    {  
    }  
}  
```
