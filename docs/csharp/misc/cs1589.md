 <span data-ttu-id="bd1d9-105">Синтаксис (*фрагмент*) [\<include>](../programming-guide/xmldoc/include.md) тега, который ссылался на файл ( `file` ), был неверным по указанной ***причине***.</span><span class="sxs-lookup"><span data-stu-id="bd1d9-105">The syntax (*fragment*) of a [\<include>](../programming-guide/xmldoc/include.md) tag, which referenced a file (`file`), was incorrect for the specified ***reason***.</span></span>

```csharp  
   public static void Main()  
   {  
   }  
}  
```
