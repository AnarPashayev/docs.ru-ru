## <a name="example"></a><span data-ttu-id="8f758-107">Пример</span><span class="sxs-lookup"><span data-stu-id="8f758-107">Example</span></span>  
 <span data-ttu-id="8f758-108">Следующий пример приводит к возникновению ошибки CS0026:</span><span class="sxs-lookup"><span data-stu-id="8f758-108">The following example generates CS0026:</span></span>  
  
```csharp  
// CS0026.cs  
public class A  
{  
   public static int i = 0;  
  
   public static void Main()  
   {  
// CS0026  
      this.i = this.i + 1;
      // Try the following line instead:  
      // i = i + 1;  
   }  
}  
```
