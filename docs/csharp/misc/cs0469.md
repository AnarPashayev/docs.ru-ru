# <a name="compiler-warning-level-2-cs0469"></a><span data-ttu-id="d1ac1-103">Предупреждение компилятора (уровень 2) CS0469</span><span class="sxs-lookup"><span data-stu-id="d1ac1-103">Compiler Warning (level 2) CS0469</span></span>
<span data-ttu-id="d1ac1-104">Значение "goto case" невозможно неявно преобразовать в тип "тип"</span><span class="sxs-lookup"><span data-stu-id="d1ac1-104">The 'goto case' value is not implicitly convertible to type 'type'</span></span>  
  
 <span data-ttu-id="d1ac1-105">При использовании оператора `goto case`должно выполняться неявное преобразование из значения goto case в тип параметра.</span><span class="sxs-lookup"><span data-stu-id="d1ac1-105">When you use `goto case`, there must be an implicit conversion from the value of the goto case to the type of the switch.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d1ac1-106">Пример</span><span class="sxs-lookup"><span data-stu-id="d1ac1-106">Example</span></span>  
 <span data-ttu-id="d1ac1-107">При компиляции следующего примера будет выдано предупреждение CS0469:</span><span class="sxs-lookup"><span data-stu-id="d1ac1-107">The following sample generates CS0469.</span></span>  
  
```csharp  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```
