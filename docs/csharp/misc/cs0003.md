# <a name="compiler-error-cs0003"></a>Ошибка компилятора CS0003
Нет памяти  
  
 Компилятору не удалось выделить достаточно виртуальной памяти для завершения компиляции. Закройте все ненужные приложения и повторите компиляцию.  
  
 Вы также можете увеличить размер файла подкачки и убедиться в наличии свободного места на диске.  
  
 Эта ошибка также может возникнуть в результате несовпадения версий Windows SDK, C# компилятора или одного или нескольких поврежденных файлов, поддерживающих C# компилятор; Переустановите Visual Studio.
