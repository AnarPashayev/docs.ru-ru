# <a name="compiler-error-cs0231"></a><span data-ttu-id="04bd9-103">Ошибка компилятора CS0231</span><span class="sxs-lookup"><span data-stu-id="04bd9-103">Compiler Error CS0231</span></span>

<span data-ttu-id="04bd9-104">Параметр params должен быть указан последним в списке формальных параметров.</span><span class="sxs-lookup"><span data-stu-id="04bd9-104">A params parameter must be the last parameter in a formal parameter list.</span></span>  
  
 <span data-ttu-id="04bd9-105">Параметр [params](../language-reference/keywords/params.md) поддерживает переменное число аргументов и должен указываться после всех остальных параметров.</span><span class="sxs-lookup"><span data-stu-id="04bd9-105">The [params](../language-reference/keywords/params.md) parameter supports a variable numbers of arguments and must be after all other parameters.</span></span> <span data-ttu-id="04bd9-106">Дополнительные сведения см. в статье [Методы](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="04bd9-106">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="04bd9-107">Следующий пример приводит к возникновению ошибки CS0231:</span><span class="sxs-lookup"><span data-stu-id="04bd9-107">The following sample generates CS0231:</span></span>  
  
```csharp  
// CS0231.cs  
class Test  
{  
   public void TestMethod(params int[] p, int i) {} // CS0231  
   // To resolve the error, use the following line instead:  
   // public void TestMethod(int i, params int[] p) {}
  
   static void Main()
   {  
   }  
}  
```
