# <a name="compiler-error-cs0535"></a>Ошибка компилятора CS0535

Класс "класс" не реализует член интерфейса "член"  
  
 [Класс](../language-reference/keywords/class.md) является производным от [интерфейса](../language-reference/keywords/interface.md), но не реализует один или несколько членов интерфейса. Класс должен реализовывать все члены интерфейсов, от которых он является производным, или должен объявляться как `abstract`.  
  
## <a name="example"></a>Пример  

 Следующий пример приводит к возникновению ошибки CS0535.  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a>Пример  

 Следующий пример приводит к возникновению ошибки CS0535.  
  
```csharp  
// CS0535_b.cs  
using System;  
class C : IDisposable {}   // CS0535  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
