 <span data-ttu-id="0b429-105">При определении типа в частичных объявлениях укажите одни и те же базовые типы в каждом частичном объявлении.</span><span class="sxs-lookup"><span data-stu-id="0b429-105">When defining a type in partial declarations, specify the same base types in each of the partial declarations.</span></span> <span data-ttu-id="0b429-106">Дополнительные сведения см. в разделе [Разделяемые классы и методы](../programming-guide/classes-and-structs/partial-classes-and-methods.md).</span><span class="sxs-lookup"><span data-stu-id="0b429-106">For more information, see [Partial Classes and Methods](../programming-guide/classes-and-structs/partial-classes-and-methods.md).</span></span>
