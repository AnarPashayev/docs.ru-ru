<span data-ttu-id="29b5f-105">Предпринята попытка присвоить значение [свойству](../programming-guide/classes-and-structs/using-properties.md), но свойство не имеет метода доступа set, или присваивание не находилось вне конструктора.</span><span class="sxs-lookup"><span data-stu-id="29b5f-105">An attempt was made to assign a value to a [property](../programming-guide/classes-and-structs/using-properties.md), but the property does not have a set accessor or the assignment was outside of the constructor.</span></span> <span data-ttu-id="29b5f-106">Устраните ошибку, добавив метод доступа set.</span><span class="sxs-lookup"><span data-stu-id="29b5f-106">Resolve the error by adding a set accessor.</span></span> <span data-ttu-id="29b5f-107">Дополнительные сведения см. [в разделе объявление и использование свойств для чтения и записи](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md).</span><span class="sxs-lookup"><span data-stu-id="29b5f-107">For more information, see [How to declare and use read-write properties](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md).</span></span>

## <a name="examples"></a><span data-ttu-id="29b5f-108">Примеры</span><span class="sxs-lookup"><span data-stu-id="29b5f-108">Examples</span></span>

<span data-ttu-id="29b5f-109">Следующий пример приводит к возникновению ошибки CS0200:</span><span class="sxs-lookup"><span data-stu-id="29b5f-109">The following sample generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

<span data-ttu-id="29b5f-110">В следующем примере используются [Автоматические реализуемые свойства](../programming-guide/classes-and-structs/auto-implemented-properties.md) и [инициализаторы объектов](../programming-guide/classes-and-structs/object-and-collection-initializers.md) , которые по-прежнему создают CS0200:</span><span class="sxs-lookup"><span data-stu-id="29b5f-110">The following sample uses [auto-implemented properties](../programming-guide/classes-and-structs/auto-implemented-properties.md) and [object initializers](../programming-guide/classes-and-structs/object-and-collection-initializers.md) and still generates CS0200:</span></span>

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

<span data-ttu-id="29b5f-111">Чтобы назначить свойство или индексатор "свойство", доступное только для чтения, добавьте метод доступа set или назначьте значение в конструкторе объекта.</span><span class="sxs-lookup"><span data-stu-id="29b5f-111">To assign to a property or indexer 'property' that's read-only, add a set accessor or assign the value in the object's constructor.</span></span>

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
