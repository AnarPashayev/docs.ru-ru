# <a name="compiler-error-cs0200"></a>Ошибка компилятора CS0200
Свойство или индексер 'имущество' не может быть назначено - это читается только

Была предпринята попытка присвоить значение [свойству,](../programming-guide/classes-and-structs/using-properties.md)но у свойства нет набора доступа или назначение находилось за пределами конструктора. Устраните ошибку, добавив метод доступа set. Для получения дополнительной информации см. [Как декларировать и использовать свойства записи.](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md)

## <a name="examples"></a>Примеры
Следующий пример приводит к возникновению ошибки CS0200:

```csharp
// CS0200.cs
public class Example
{
    private int _mi;
    int I
    {
        get
        {
            return _mi;
        }
        // uncomment the set accessor and declaration for _mi
        /*
        set
        {
            _mi = value;
        }
        */
    }

    public static void Main()
    {  
        Example example = new Example();
        example.I = 9;   // CS0200
    }
}  
```

В следующем примере используются [свойства автоматического реализации,](../programming-guide/classes-and-structs/auto-implemented-properties.md) [инициализаторы объектов,](../programming-guide/classes-and-structs/object-and-collection-initializers.md)и по-прежнему генерирует CS0200:

```csharp
// CS0200.cs
public class Example
{
    int I
    {
        get;
        // uncomment the set accessor and declaration
        //set;
    }

    public static void Main()
    {  
        var example = new Example
        {
            I = 9   // CS0200
        };
    }
}
```

Назначение на свойство или индексатор "имущество", которое читается только, может быть достигнуто путем добавления набора аксессуара или путем присвоения свойства в конструкторобъекта объекта.

```csharp
public class Example
{
    int I { get; }

    public Example()
    {
        I = -7;
    }
}
```
