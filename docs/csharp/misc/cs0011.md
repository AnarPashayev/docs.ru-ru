# <a name="compiler-error-cs0011"></a><span data-ttu-id="60a41-103">Ошибка компилятора CS0011</span><span class="sxs-lookup"><span data-stu-id="60a41-103">Compiler Error CS0011</span></span>
<span data-ttu-id="60a41-104">Не удается разрешить базовый класс или интерфейс "класс" в сборке "сборка", на которую ссылается тип "тип"</span><span class="sxs-lookup"><span data-stu-id="60a41-104">The base class or interface 'class' in assembly 'assembly' referenced by type 'type' could not be resolved</span></span>  
  
 <span data-ttu-id="60a41-105">Класс, который был импортирован из файла с помощью **/reference**, является производным от не найденного класса или реализует не найденный интерфейс.</span><span class="sxs-lookup"><span data-stu-id="60a41-105">A class that was imported from a file with **/reference**, is derived from a class or implements an interface that is not found.</span></span> <span data-ttu-id="60a41-106">Это может произойти, если требуемая библиотека DLL не включена в компиляцию с помощью **/reference**.</span><span class="sxs-lookup"><span data-stu-id="60a41-106">This can occur if a required DLL is not also included in the compilation with **/reference**.</span></span>  
  
 <span data-ttu-id="60a41-107">Дополнительные сведения см. в разделе [диалоговое окно "Добавление ссылки"](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) и [-Reference (параметры компилятора C#)](../language-reference/compiler-options/reference-compiler-option.md).</span><span class="sxs-lookup"><span data-stu-id="60a41-107">For more information, see [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) and [-reference (C# Compiler Options)](../language-reference/compiler-options/reference-compiler-option.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="60a41-108">Пример</span><span class="sxs-lookup"><span data-stu-id="60a41-108">Example</span></span>  
  
```csharp  
// CS0011_1.cs  
// compile with: /target:library  
  
public class Outer
{  
   public class B { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="60a41-109">Пример</span><span class="sxs-lookup"><span data-stu-id="60a41-109">Example</span></span>  
 <span data-ttu-id="60a41-110">Второй файл создает библиотеку DLL, которая определяет класс `C` , производный от класса `B` , созданного в предыдущем примере.</span><span class="sxs-lookup"><span data-stu-id="60a41-110">The second file creates a DLL that defines a class `C` that is derived from the class `B` that was created in the previous example.</span></span>  
  
```csharp  
// CS0011_2.cs  
// compile with: /target:library /reference:CS0011_1.dll  
// post-build command: del /f CS0011_1.dll  
public class C : Outer.B {}  
```  
  
## <a name="example"></a><span data-ttu-id="60a41-111">Пример</span><span class="sxs-lookup"><span data-stu-id="60a41-111">Example</span></span>  
 <span data-ttu-id="60a41-112">Третий файл заменяет библиотеку DLL, созданную на первом шаге, и пропускает определение внутреннего класса `B`.</span><span class="sxs-lookup"><span data-stu-id="60a41-112">The third file replaces the DLL created by the first step, and omits the definition of the inner class `B`.</span></span>  
  
```csharp  
// CS0011_3.cs  
// compile with: /target:library /out:cs0011_1.dll  
public class Outer {}  
```  
  
## <a name="example"></a><span data-ttu-id="60a41-113">Пример</span><span class="sxs-lookup"><span data-stu-id="60a41-113">Example</span></span>  
 <span data-ttu-id="60a41-114">Наконец, четвертый файл ссылается на класс `C` , определенный во втором примере, являющийся производным от класса `B`, который в данный момент отсутствует.</span><span class="sxs-lookup"><span data-stu-id="60a41-114">Finally, the fourth file references the class `C` defined in the second example, which is derived from class `B`, and which is now missing.</span></span>  
  
 <span data-ttu-id="60a41-115">Следующий пример приводит к возникновению ошибки CS0011.</span><span class="sxs-lookup"><span data-stu-id="60a41-115">The following sample generates CS0011.</span></span>  
  
```csharp

// CS0011 expected  
  
class M  
{  
   public static void Main()  
   {  
      C c = new C();  
   }  
}  
```
