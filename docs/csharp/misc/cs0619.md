<span data-ttu-id="b1ba8-105">Член класса помечен атрибутом [Obsolete](../language-reference/attributes/general.md#obsolete-attribute) , чтобы при обращении к этому члену возникала ошибка.</span><span class="sxs-lookup"><span data-stu-id="b1ba8-105">A class member was marked with the [Obsolete](../language-reference/attributes/general.md#obsolete-attribute) attribute, such that an error will be issued when the class member is referenced.</span></span>

## <a name="example"></a><span data-ttu-id="b1ba8-106">Пример</span><span class="sxs-lookup"><span data-stu-id="b1ba8-106">Example</span></span>

<span data-ttu-id="b1ba8-107">Следующий пример приводит к возникновению ошибки CS0619:</span><span class="sxs-lookup"><span data-stu-id="b1ba8-107">The following sample generates CS0619:</span></span>  
  
```csharp
using System;

public class C
{
    [Obsolete("Use NewMethod instead", true)] // generates an error on use
    public static void OldMethod()
    {
    }

    // this is the method you should be using
    public static void NewMethod()
    {
    }  
}

class MyClass
{
   public static void Main()
   {
      C.OldMethod();   // CS0619
   }
}
```
