# <a name="compiler-error-cs0763"></a><span data-ttu-id="58972-102">Ошибка компилятора CS0763</span><span class="sxs-lookup"><span data-stu-id="58972-102">Compiler Error CS0763</span></span>
<span data-ttu-id="58972-103">Оба объявления разделяемого метода должны быть статическими, или ни одно из объявлений не должно быть статическим.</span><span class="sxs-lookup"><span data-stu-id="58972-103">Both partial method declarations must be static or neither may be static.</span></span>  
  
 <span data-ttu-id="58972-104">Объявление разделяемого метода не может иметь одну часть статическую, а другую часть не статическую.</span><span class="sxs-lookup"><span data-stu-id="58972-104">A partial method declaration cannot have one part static and the other part not static.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="58972-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="58972-105">To correct this error</span></span>  
  
1. <span data-ttu-id="58972-106">Сделайте обе части либо статическими, либо не статическими.</span><span class="sxs-lookup"><span data-stu-id="58972-106">Make both parts either static or non-static.</span></span>  
  
## <a name="example"></a><span data-ttu-id="58972-107">Пример</span><span class="sxs-lookup"><span data-stu-id="58972-107">Example</span></span>  
 <span data-ttu-id="58972-108">В следующем коде возникает ошибка CS0763:</span><span class="sxs-lookup"><span data-stu-id="58972-108">The following code generates CS0763:</span></span>  
  
```csharp  
// cs0763.cs  
using System;  
  
    public partial class C  
    {  
        static partial void Part();  
        partial void Part() // CS0763  
        {  
        }

        {  
            return 1;  
        }  
  
    }  
```
