# <a name="compiler-error-cs0763"></a><span data-ttu-id="830d1-103">Ошибка компилятора CS0763</span><span class="sxs-lookup"><span data-stu-id="830d1-103">Compiler Error CS0763</span></span>
<span data-ttu-id="830d1-104">Оба объявления разделяемого метода должны быть статическими, или ни одно из объявлений не должно быть статическим.</span><span class="sxs-lookup"><span data-stu-id="830d1-104">Both partial method declarations must be static or neither may be static.</span></span>  
  
 <span data-ttu-id="830d1-105">Объявление разделяемого метода не может иметь одну часть статическую, а другую часть не статическую.</span><span class="sxs-lookup"><span data-stu-id="830d1-105">A partial method declaration cannot have one part static and the other part not static.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="830d1-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="830d1-106">To correct this error</span></span>  
  
1. <span data-ttu-id="830d1-107">Сделайте обе части либо статическими, либо не статическими.</span><span class="sxs-lookup"><span data-stu-id="830d1-107">Make both parts either static or non-static.</span></span>  
  
## <a name="example"></a><span data-ttu-id="830d1-108">Пример</span><span class="sxs-lookup"><span data-stu-id="830d1-108">Example</span></span>  
 <span data-ttu-id="830d1-109">В следующем коде возникает ошибка CS0763:</span><span class="sxs-lookup"><span data-stu-id="830d1-109">The following code generates CS0763:</span></span>

```csharp  
    public partial class C  
    {  
        static partial void Part();  
        partial void Part() // CS0763  
        {  
        }

        {  
            return 1;  
        }  
  
    }  
```
