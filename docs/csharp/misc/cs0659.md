# <a name="compiler-warning-level-3-cs0659"></a><span data-ttu-id="581be-103">Предупреждение компилятора (уровень 3) CS0659</span><span class="sxs-lookup"><span data-stu-id="581be-103">Compiler Warning (level 3) CS0659</span></span>

<span data-ttu-id="581be-104">Класс "класс" переопределяет Object.Equals(object o), но не переопределяет Object.GetHashCode()</span><span class="sxs-lookup"><span data-stu-id="581be-104">'class' overrides Object.Equals(object o) but does not override Object.GetHashCode()</span></span>  

<span data-ttu-id="581be-105">Компилятор обнаружил переопределение <xref:System.Object.Equals%2A?displayProperty=nameWithType> метода, но не переопределяет <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> метод.</span><span class="sxs-lookup"><span data-stu-id="581be-105">The compiler detected an override of the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method but no override of the <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="581be-106">Переопределение <xref:System.Object.Equals%2A> подразумевает также необходимость переопределения <xref:System.Object.GetHashCode%2A> .</span><span class="sxs-lookup"><span data-stu-id="581be-106">An override of <xref:System.Object.Equals%2A> implies that you also want to override <xref:System.Object.GetHashCode%2A>.</span></span>  

<span data-ttu-id="581be-107">Следующий код создает CS0659:</span><span class="sxs-lookup"><span data-stu-id="581be-107">The following code generates CS0659:</span></span>  

```csharp
// CS0659.cs  
// compile with: /W:3 /target:library  
class Test
{  
   public override bool Equals(object o) { return true; }   // CS0659  
}  
  
// OK  
class Test2  
{  
   public override bool Equals(object o) { return true; }  
   public override int GetHashCode() { return 0; }  
}  
```
