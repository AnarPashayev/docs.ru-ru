# <a name="compiler-error-cs0076"></a>Ошибка компилятора CS0076
Имя перечислителя "value__" зарезервировано и не может использоваться  
  
 Элемент в [перечислении](../language-reference/builtin-types/enum.md) не может иметь идентификатор с именем **value__**.
