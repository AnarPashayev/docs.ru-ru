## <a name="example"></a><span data-ttu-id="6bd03-111">Пример</span><span class="sxs-lookup"><span data-stu-id="6bd03-111">Example</span></span>

```csharp

    static void MTyVar<T, U>() where T : U { }

    {  
        MTyVar<int?, int>(); // CS0312  
        return 1;  
    }  
}  
```
