# <a name="compiler-error-cs0312"></a><span data-ttu-id="be84a-102">Ошибка компилятора CS0312</span><span class="sxs-lookup"><span data-stu-id="be84a-102">Compiler Error CS0312</span></span>
<span data-ttu-id="be84a-103">Тип "тип1" не может быть использован как параметр типа "имя" в универсальном типе или методе "имя".</span><span class="sxs-lookup"><span data-stu-id="be84a-103">The type 'type1' cannot be used as type parameter 'name' in the generic type or method 'name'.</span></span> <span data-ttu-id="be84a-104">Допускающий значение null тип "тип1" не удовлетворяет ограничению типа "тип2".</span><span class="sxs-lookup"><span data-stu-id="be84a-104">The nullable type 'type1' does not satisfy the constraint of 'type2'.</span></span>  
  
 <span data-ttu-id="be84a-105">Допускающий значение null тип отличается от своего аналога, не допускающего значение null; между ними не существует неявное преобразование ссылок или преобразование удостоверений.</span><span class="sxs-lookup"><span data-stu-id="be84a-105">A nullable type is distinct from its non-nullable counterpart; no implicit reference conversion or identify conversion exists between them.</span></span> <span data-ttu-id="be84a-106">Допускающая значения null упаковка-преобразование не удовлетворяет ограничению универсального типа.</span><span class="sxs-lookup"><span data-stu-id="be84a-106">A nullable boxing conversion does not satisfy a generic type constraint.</span></span> <span data-ttu-id="be84a-107">В следующем примере первый параметр типа является `Nullable<int>` , а второй параметр типа является `System.Int32`.</span><span class="sxs-lookup"><span data-stu-id="be84a-107">In the example that follows, the first type parameter is a `Nullable<int>` and the second type parameter is a `System.Int32`.</span></span>

## <a name="example"></a><span data-ttu-id="be84a-111">Пример</span><span class="sxs-lookup"><span data-stu-id="be84a-111">Example</span></span>

```csharp

class Program  
{  
    static void MTyVar<T, U>() where T : U { }  
  
    static int Main()  
    {  
        MTyVar<int?, int>(); // CS0312  
        return 1;  
    }  
}  
```  
  
 <span data-ttu-id="be84a-113">Хотя допускающий значения null тип отличается от типа, не допускающего значения null, между ними разрешены различные виды преобразований.</span><span class="sxs-lookup"><span data-stu-id="be84a-113">Although a nullable type is distinct from a non-nullable type, various kinds of conversions are allowed between nullable and non-nullable values.</span></span>
