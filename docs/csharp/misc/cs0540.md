## <a name="example"></a><span data-ttu-id="5b8e8-106">Пример</span><span class="sxs-lookup"><span data-stu-id="5b8e8-106">Example</span></span>

```csharp

   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}

{  
   void I.m() {}

}  
```
