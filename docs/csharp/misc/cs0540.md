## <a name="example"></a><span data-ttu-id="f16ff-106">Пример</span><span class="sxs-lookup"><span data-stu-id="f16ff-106">Example</span></span>  
 <span data-ttu-id="f16ff-107">При компиляции следующего примера возникнет ошибка CS0540.</span><span class="sxs-lookup"><span data-stu-id="f16ff-107">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
