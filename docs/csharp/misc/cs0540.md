## <a name="example"></a><span data-ttu-id="c927a-107">Пример</span><span class="sxs-lookup"><span data-stu-id="c927a-107">Example</span></span>  

 <span data-ttu-id="c927a-108">При компиляции следующего примера возникнет ошибка CS0540.</span><span class="sxs-lookup"><span data-stu-id="c927a-108">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
