# <a name="compiler-error-cs0025"></a>Ошибка компилятора CS0025
Не удалось найти файл стандартной библиотеки "файл"  
  
 Отсутствует файл, необходимый для компилятора. Проверьте правильность пути и существование файла.  
  
 Если файл является системным файлом Visual Studio, может потребоваться восстановить установку Visual Studio или полностью установить его.
