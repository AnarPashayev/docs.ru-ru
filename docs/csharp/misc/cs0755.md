<span data-ttu-id="0f67c-103">Разделяемый метод должен быть либо оба раза объявлен как метод расширения, либо нигде не объявлен как метод расширения.</span><span class="sxs-lookup"><span data-stu-id="0f67c-103">Both partial method declarations must be extension methods or neither may be an extension method.</span></span>  
  
 <span data-ttu-id="0f67c-104">Разделяемый метод состоит из определяющего объявления (сигнатуры) и необязательного реализующего объявления (тела).</span><span class="sxs-lookup"><span data-stu-id="0f67c-104">A partial method consists of a defining declaration (signature) and an optional implementing declaration (body).</span></span> <span data-ttu-id="0f67c-105">Если определяющее объявление является методом расширения, то при определении реализующего объявления это объявление также должно быть методом расширения.</span><span class="sxs-lookup"><span data-stu-id="0f67c-105">If the defining declaration is an extension method, the implementing declaration, if one is defined, must also be an extension method.</span></span> <span data-ttu-id="0f67c-106">Если определяющее объявление не является методом расширения, реализующее объявление также не должно быть таким методом.</span><span class="sxs-lookup"><span data-stu-id="0f67c-106">And if the defining method is not an extension method, the implementing must not be one either.</span></span>

## <a name="example"></a><span data-ttu-id="0f67c-109">Пример</span><span class="sxs-lookup"><span data-stu-id="0f67c-109">Example</span></span>  
 <span data-ttu-id="0f67c-110">Следующий пример приводит к возникновению ошибки CS0755:</span><span class="sxs-lookup"><span data-stu-id="0f67c-110">The following example generates CS0755:</span></span>  
  
```csharp  
// cs0755.cs  
    public static partial class Ext  
    {  
        static partial void Part(this C c); //Extension method  
  
        // Typically the implementing declaration is in a separate file.

        {  
        }  
    }

    {  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
