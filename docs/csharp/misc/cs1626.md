# <a name="compiler-error-cs1626"></a><span data-ttu-id="f1884-103">Ошибка компилятора CS1626</span><span class="sxs-lookup"><span data-stu-id="f1884-103">Compiler Error CS1626</span></span>
<span data-ttu-id="f1884-104">Нельзя использовать оператор yield в теле блока try, имеющего предложение catch.</span><span class="sxs-lookup"><span data-stu-id="f1884-104">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="f1884-105">Оператор yield нельзя использовать в блоке try, если имеется связанное с этим блоком предложение catch.</span><span class="sxs-lookup"><span data-stu-id="f1884-105">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="f1884-106">Чтобы избежать этой ошибки, либо переместите оператор yield из блока try/catch/finally, либо удалите блок catch.</span><span class="sxs-lookup"><span data-stu-id="f1884-106">To avoid this error, either move the yield statement out of the try/catch/finally block, or remove the catch block.</span></span>
  
 <span data-ttu-id="f1884-107">При компиляции следующего примера возникнет ошибка CS1626:</span><span class="sxs-lookup"><span data-stu-id="f1884-107">The following sample generates CS1626:</span></span>  
  
```csharp  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
      finally
      {

      }
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
