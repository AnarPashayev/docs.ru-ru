# <a name="compiler-error-cs1104"></a><span data-ttu-id="f40ed-102">Ошибка компилятора CS1104</span><span class="sxs-lookup"><span data-stu-id="f40ed-102">Compiler Error CS1104</span></span>
<span data-ttu-id="f40ed-103">Массив параметров не может быть использован с модификатором " this" в методе расширения</span><span class="sxs-lookup"><span data-stu-id="f40ed-103">A parameter array cannot be used with 'this' modifier on an extension method.</span></span>  
  
 <span data-ttu-id="f40ed-104">Первый параметр метода расширения не может быть массивом параметров.</span><span class="sxs-lookup"><span data-stu-id="f40ed-104">The first parameter of an extension method cannot be a params array.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f40ed-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="f40ed-105">To correct this error</span></span>  
  
1. <span data-ttu-id="f40ed-106">Обратите внимание, что первый параметр определения метода расширения указывает, какой тип метод будет "расширять".</span><span class="sxs-lookup"><span data-stu-id="f40ed-106">Remember that the first parameter of an extension method definition specifies which type the method will "extend".</span></span> <span data-ttu-id="f40ed-107">Это не входной параметр.</span><span class="sxs-lookup"><span data-stu-id="f40ed-107">It is not an input parameter.</span></span> <span data-ttu-id="f40ed-108">Таким образом, в этом месте массив параметров не имеет смысла.</span><span class="sxs-lookup"><span data-stu-id="f40ed-108">Therefore, it makes no sense to have a params array in this location.</span></span> <span data-ttu-id="f40ed-109">Если нужно передать массив параметров, передайте его вторым параметром.</span><span class="sxs-lookup"><span data-stu-id="f40ed-109">If you do have to pass in a params array, make it the second parameter.</span></span>
