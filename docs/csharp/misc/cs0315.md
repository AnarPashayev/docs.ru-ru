# <a name="compiler-error-cs0315"></a><span data-ttu-id="8d142-103">Ошибка компилятора CS0315</span><span class="sxs-lookup"><span data-stu-id="8d142-103">Compiler Error CS0315</span></span>

<span data-ttu-id="8d142-104">Тип "valueType" не может использоваться в качестве параметра типа "t" в универсальном типе или методе "Типеормесод \<T> ".</span><span class="sxs-lookup"><span data-stu-id="8d142-104">The type 'valueType' cannot be used as type parameter 'T' in the generic type or method 'TypeorMethod\<T>'.</span></span> <span data-ttu-id="8d142-105">Нет преобразования-упаковки из "тип_значения" в "тип_ссылки".</span><span class="sxs-lookup"><span data-stu-id="8d142-105">There is no boxing conversion from 'valueType' to 'referenceType'.</span></span>  
  
 <span data-ttu-id="8d142-106">Эта ошибка происходит, когда вы ограничиваете универсальный тип конкретным классом и пытаетесь создать экземпляр этого класса, используя тип значения, который не может быть неявно упакован.</span><span class="sxs-lookup"><span data-stu-id="8d142-106">This error occurs when you constrain a generic type to a particular class, and try to construct an instance of that class by using a value type that cannot be implicitly boxed to it.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="8d142-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="8d142-107">To correct this error</span></span>  
  
1. <span data-ttu-id="8d142-108">Одно из решений — переопределить структуру как класс.</span><span class="sxs-lookup"><span data-stu-id="8d142-108">One solution is to redefine the struct as a class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8d142-109">Пример</span><span class="sxs-lookup"><span data-stu-id="8d142-109">Example</span></span>  

 <span data-ttu-id="8d142-110">При компиляции следующего примера возникнет ошибка CS0315:</span><span class="sxs-lookup"><span data-stu-id="8d142-110">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{
}  
public class Test  
{  
    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315  
        return 1;  
    }  
}  
```
