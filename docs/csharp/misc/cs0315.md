# <a name="compiler-error-cs0315"></a><span data-ttu-id="5295c-102">Ошибка компилятора CS0315</span><span class="sxs-lookup"><span data-stu-id="5295c-102">Compiler Error CS0315</span></span>
<span data-ttu-id="5295c-103">Тип 'valueType' не может быть использован в качестве параметра типа 'T' в общем типе или методе 'TypeorMethod\<T>'.</span><span class="sxs-lookup"><span data-stu-id="5295c-103">The type 'valueType' cannot be used as type parameter 'T' in the generic type or method 'TypeorMethod\<T>'.</span></span> <span data-ttu-id="5295c-104">Нет преобразования-упаковки из "тип_значения" в "тип_ссылки".</span><span class="sxs-lookup"><span data-stu-id="5295c-104">There is no boxing conversion from 'valueType' to 'referenceType'.</span></span>  
  
 <span data-ttu-id="5295c-105">Эта ошибка происходит, когда вы ограничиваете универсальный тип конкретным классом и пытаетесь создать экземпляр этого класса, используя тип значения, который не может быть неявно упакован.</span><span class="sxs-lookup"><span data-stu-id="5295c-105">This error occurs when you constrain a generic type to a particular class, and try to construct an instance of that class by using a value type that cannot be implicitly boxed to it.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5295c-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="5295c-106">To correct this error</span></span>  
  
1. <span data-ttu-id="5295c-107">Одно из решений — переопределить структуру как класс.</span><span class="sxs-lookup"><span data-stu-id="5295c-107">One solution is to redefine the struct as a class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5295c-108">Пример</span><span class="sxs-lookup"><span data-stu-id="5295c-108">Example</span></span>  
 <span data-ttu-id="5295c-109">При компиляции следующего примера возникнет ошибка CS0315:</span><span class="sxs-lookup"><span data-stu-id="5295c-109">The following example generates CS0315:</span></span>  
  
```csharp  
// cs0315.cs  
public class ClassConstraint { }  
public struct ViolateClassConstraint { }  
  
public class Gen<T> where T : ClassConstraint  
{
}  
public class Test  
{  
    public static int Main()  
    {  
        Gen<ViolateClassConstraint> g = new Gen<ViolateClassConstraint>(); //CS0315  
        return 1;  
    }  
}  
```
