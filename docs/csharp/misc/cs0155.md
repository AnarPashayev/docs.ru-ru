# <a name="compiler-error-cs0155"></a><span data-ttu-id="e36f5-103">Ошибка компилятора CS0155</span><span class="sxs-lookup"><span data-stu-id="e36f5-103">Compiler Error CS0155</span></span>

<span data-ttu-id="e36f5-104">Тип в операторах caught или thrown должен быть производным от System.Exception.</span><span class="sxs-lookup"><span data-stu-id="e36f5-104">The type caught or thrown must be derived from System.Exception</span></span>  
  
 <span data-ttu-id="e36f5-105">Выполнена попытка передачи типа данных, который не является производным от **System.Exception** , в блок [catch](../language-reference/keywords/try-catch.md) .</span><span class="sxs-lookup"><span data-stu-id="e36f5-105">An attempt was made to pass a data type that does not derive from **System.Exception** into a [catch](../language-reference/keywords/try-catch.md) block.</span></span> <span data-ttu-id="e36f5-106">В блок **catch** можно передавать только типы данных, которые являются производными от **System.Exception** .</span><span class="sxs-lookup"><span data-stu-id="e36f5-106">Only data types that derive from **System.Exception** can be passed into a **catch** block.</span></span> <span data-ttu-id="e36f5-107">Дополнительные сведения см. в разделе [исключения и обработка исключений](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="e36f5-107">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="e36f5-108">Следующий пример приводит к возникновению ошибки CS0155:</span><span class="sxs-lookup"><span data-stu-id="e36f5-108">The following sample generates CS0155:</span></span>  
  
```csharp  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
