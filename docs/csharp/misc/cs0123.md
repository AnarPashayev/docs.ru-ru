# <a name="compiler-error-cs0123"></a><span data-ttu-id="8025f-102">Ошибка компилятора CS0123</span><span class="sxs-lookup"><span data-stu-id="8025f-102">Compiler Error CS0123</span></span>
<span data-ttu-id="8025f-103">Ни одна перегрузка для метода "метод" не соответствует делегату "делегат".</span><span class="sxs-lookup"><span data-stu-id="8025f-103">No overload for 'method' matches delegate 'delegate'</span></span>  
  
 <span data-ttu-id="8025f-104">Не удалось создать делегат, поскольку не использовалась правильная сигнатура.</span><span class="sxs-lookup"><span data-stu-id="8025f-104">An attempt to create a delegate failed because the correct signature was not used.</span></span> <span data-ttu-id="8025f-105">Экземпляры делегата должны объявляться с такой же сигнатурой, как и в объявлении делегата.</span><span class="sxs-lookup"><span data-stu-id="8025f-105">Instances of a delegate must be declared with the same signature as the delegate declaration.</span></span>  
  
 <span data-ttu-id="8025f-106">Чтобы устранить эту ошибку, исправьте либо метод, либо сигнатуру делегата.</span><span class="sxs-lookup"><span data-stu-id="8025f-106">You can resolve this error by adjusting either the method or delegate signature.</span></span> <span data-ttu-id="8025f-107">Дополнительные сведения см. в разделе [Делегаты](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="8025f-107">For more information, see [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="8025f-108">Следующий пример приводит к возникновению ошибки CS0123.</span><span class="sxs-lookup"><span data-stu-id="8025f-108">The following sample generates CS0123.</span></span>  
  
```csharp  
// CS0123.cs  
delegate void D();  
delegate void D2(int i);  
  
public class C  
{  
   public static void f(int i) {}  
  
   public static void Main()  
   {  
      D d = new D(f);   // CS0123  
      D2 d2 = new D2(f);   // OK  
   }  
}  
```
