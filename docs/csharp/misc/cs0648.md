# <a name="compiler-error-cs0648"></a>Ошибка компилятора CS0648

"тип" является типом, который не поддерживается в данном языке  
  
 Метаданные, созданные из другого языка, возможно C++, содержат тип, не помеченный как управляемый. Сведения об этом типе включаются в метаданные, но сам тип недоступен для программ, написанных на языке C#.
