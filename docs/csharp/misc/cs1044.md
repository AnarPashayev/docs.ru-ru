# <a name="compiler-error-cs1044"></a><span data-ttu-id="73aac-102">Ошибка компилятора CS1044</span><span class="sxs-lookup"><span data-stu-id="73aac-102">Compiler Error CS1044</span></span>
<span data-ttu-id="73aac-103">В операторах for, using, fixed и операторах объявления не может использоваться более одного типа.</span><span class="sxs-lookup"><span data-stu-id="73aac-103">Cannot use more than one type in a for, using, fixed, or declaration statement</span></span>  
  
 <span data-ttu-id="73aac-104">Компилятор обнаружил недопустимую инструкцию.</span><span class="sxs-lookup"><span data-stu-id="73aac-104">The compiler found an invalid statement.</span></span>  
  
 <span data-ttu-id="73aac-105">В следующем примере возникает ошибка CS1044:</span><span class="sxs-lookup"><span data-stu-id="73aac-105">The following sample generates CS1044:</span></span>  
  
```csharp  
// CS1044.cs  
using System;  
  
public class MyClass : IDisposable  
{  
   public void Dispose()  
   {  
      Console.WriteLine("Res1.Dispose()");  
   }  
  
   public static void Main()  
   {  
      using (MyClass mc1 = new MyClass(),  
             MyClass mc2 = new MyClass())   // CS1044, remove an instantiation  
      {  
      }  
   }  
}  
```
