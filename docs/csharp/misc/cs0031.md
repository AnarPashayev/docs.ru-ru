# <a name="compiler-error-cs0031"></a><span data-ttu-id="4b1ff-103">Ошибка компилятора CS0031</span><span class="sxs-lookup"><span data-stu-id="4b1ff-103">Compiler Error CS0031</span></span>

<span data-ttu-id="4b1ff-104">Постоянное значение "значение" не может быть преобразовано в "тип"</span><span class="sxs-lookup"><span data-stu-id="4b1ff-104">Constant value 'value' cannot be converted to a 'type'.</span></span>

<span data-ttu-id="4b1ff-105">Предпринята попытка назначить значение переменной, тип которой не позволяет хранить значения.</span><span class="sxs-lookup"><span data-stu-id="4b1ff-105">An attempt was made to assign a value to a variable whose type cannot store the value.</span></span> <span data-ttu-id="4b1ff-106">Дополнительные сведения см. в разделе [Типы](../programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="4b1ff-106">For more information, see [Types](../programming-guide/types/index.md).</span></span>

<span data-ttu-id="4b1ff-107">Следующий пример приводит к возникновению ошибки CS0031 в контекстах checked и unchecked:</span><span class="sxs-lookup"><span data-stu-id="4b1ff-107">The following sample generates CS0031 in both checked and unchecked contexts:</span></span>

```csharp
// CS0031.cs
namespace CS0031
{
    public class Program
    {
        public static void Main()
        {
            int num = (int)2147483648M; //CS0031
            // Try using a larger numeric type instead.
            // long num = (long)2147483648M; //CS0031

            const decimal d = -10M; // Decimal literal
            unchecked
            {
                const byte b = (byte)d; // CS0031
                // For small values try using a signed byte instead.
                // const sbyte b = (sbyte)d;
            }
        }
    }
}
```
