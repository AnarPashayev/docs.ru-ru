# <a name="compiler-error-cs0544"></a><span data-ttu-id="dfd25-103">Ошибка компилятора CS0544</span><span class="sxs-lookup"><span data-stu-id="dfd25-103">Compiler Error CS0544</span></span>
<span data-ttu-id="dfd25-104">"переопределение_свойства": невозможно переопределить, поскольку "не_свойство" не является свойством</span><span class="sxs-lookup"><span data-stu-id="dfd25-104">'property override': cannot override because 'non-property' is not a property</span></span>  
  
 <span data-ttu-id="dfd25-105">Предпринята попытка переопределения типа данных, не являющегося свойством, как [свойства](../programming-guide/classes-and-structs/properties.md), что недопустимо.</span><span class="sxs-lookup"><span data-stu-id="dfd25-105">An attempt was made to override a nonproperty data type as a [property](../programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>  
  
 <span data-ttu-id="dfd25-106">В следующем примере возникает ошибка CS0544:</span><span class="sxs-lookup"><span data-stu-id="dfd25-106">The following sample generates CS0544:</span></span>  
  
```csharp  
// CS0544.cs  
// compile with: /target:library  
public class a  
{  
   public int i;  
}  
  
public class b : a  
{  
   public override int i {   // CS0544  
   // try the following line instead  
   // public new int i {  
      get  
      {  
         return 0;  
      }  
   }  
}  
```
