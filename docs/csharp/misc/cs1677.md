# <a name="compiler-error-cs1677"></a><span data-ttu-id="f5dab-102">Ошибка компилятора CS1677</span><span class="sxs-lookup"><span data-stu-id="f5dab-102">Compiler Error CS1677</span></span>
<span data-ttu-id="f5dab-103">Параметр "число" не должен объявляться с ключевым словом "ключевое_слово"</span><span class="sxs-lookup"><span data-stu-id="f5dab-103">Parameter 'number' should not be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="f5dab-104">Эта ошибка возникает, если модификатор типа параметра в анонимном методе не соответствует модификатору, используемому в объявлении делегата, к которому выполняется приведение метода.</span><span class="sxs-lookup"><span data-stu-id="f5dab-104">This error occurs when the parameter type modifier in an anonymous method does not match that used in the declaration of the delegate, to which you are casting the method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f5dab-105">Пример</span><span class="sxs-lookup"><span data-stu-id="f5dab-105">Example</span></span>  
 <span data-ttu-id="f5dab-106">Следующий пример приводит к возникновению ошибки CS1677:</span><span class="sxs-lookup"><span data-stu-id="f5dab-106">The following sample generates CS1677:</span></span>  
  
```csharp  
// CS1677.cs  
delegate void D(int i);  
class Errors  
{  
    static void Main()
    {  
        D d = delegate(out int i) { };   // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int i) { };  
  
        D d = delegate(ref int j){}; // CS1677  
        // To resolve, use the following line instead:  
        // D d = delegate(int j){};  
    }  
}  
```
