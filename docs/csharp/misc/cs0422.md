# <a name="compiler-warning-level-4-cs0422"></a>Предупреждение компилятора (уровень 4) CS0422

Параметр /incremental больше не поддерживается

 Инкрементная компиляция **(/incr** или **/инкрементный)** не поддерживается в C'2.0.

> [!NOTE]
> Это предупреждение компилятора больше не используется в Roslyn.
