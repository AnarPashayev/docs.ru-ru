# <a name="compiler-error-cs0738"></a><span data-ttu-id="696de-103">Ошибка компилятора CS0738</span><span class="sxs-lookup"><span data-stu-id="696de-103">Compiler Error CS0738</span></span>

<span data-ttu-id="696de-104">"имя типа" не реализует член интерфейса "имя члена".</span><span class="sxs-lookup"><span data-stu-id="696de-104">'type name' does not implement interface member 'member name'.</span></span> <span data-ttu-id="696de-105">"имя_метода" не может реализовать "член_интерфейса", потому что не имеет соответствующего возвращаемого типа "имя_типа".</span><span class="sxs-lookup"><span data-stu-id="696de-105">'method name' cannot implement 'interface member' because it does not have the matching return type of ' type name'.</span></span>  
  
 <span data-ttu-id="696de-106">Возвращаемое значение реализующего метода в классе должно соответствовать возвращаемому значению члена интерфейса, который он реализует.</span><span class="sxs-lookup"><span data-stu-id="696de-106">The return value of an implementing method in a class must match the return value of the interface member that it implements.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="696de-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="696de-107">To correct this error</span></span>  
  
1. <span data-ttu-id="696de-108">Измените возвращаемый тип метода в соответствии с типом члена интерфейса.</span><span class="sxs-lookup"><span data-stu-id="696de-108">Change the return type of the method to match that of the interface member.</span></span>  
  
## <a name="example"></a><span data-ttu-id="696de-109">Пример</span><span class="sxs-lookup"><span data-stu-id="696de-109">Example</span></span>  

 <span data-ttu-id="696de-110">При компиляции следующего кода возникнет ошибка CS0738, так как метод класса возвращает `void` , а член интерфейса с тем же именем возвращает `int`:</span><span class="sxs-lookup"><span data-stu-id="696de-110">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>  
  
```csharp  
using System;  
  
interface ITest  
{  
    int TestMethod();  
}  
public class Test: ITest  
{  
    public void TestMethod() { } // CS0738  
    // Try the following line instead.  
    // public int TestMethod();  
}  
```
