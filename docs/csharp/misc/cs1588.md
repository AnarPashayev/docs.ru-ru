# <a name="compiler-error-cs1588"></a>Ошибка компилятора CS1588
Не удается определить каталог среды CLR — "причина"  
  
 Компилятору не удается получить каталог, в котором установлена среда выполнения .NET. Эта ошибка указывает, что среда CLR установлена неправильно.
