# <a name="compiler-error-cs1588"></a>Ошибка компилятора CS1588
Не удается определить каталог среды CLR — "причина"  
  
 Компилятор не может получить каталог, в котором установлена среда CLR .NET Framework. Эта ошибка указывает, что среда CLR установлена неправильно.
