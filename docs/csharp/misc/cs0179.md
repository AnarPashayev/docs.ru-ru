# <a name="compiler-error-cs0179"></a><span data-ttu-id="6f30a-103">Ошибка компилятора CS0179</span><span class="sxs-lookup"><span data-stu-id="6f30a-103">Compiler Error CS0179</span></span>

<span data-ttu-id="6f30a-104">Член "член" не может иметь модификатор extern и объявлять тело.</span><span class="sxs-lookup"><span data-stu-id="6f30a-104">'member' cannot be extern and declare a body</span></span>  
  
 <span data-ttu-id="6f30a-105">Если член класса помечен как [extern](../language-reference/keywords/extern.md), это означает, что определение этого члена находится в другом файле.</span><span class="sxs-lookup"><span data-stu-id="6f30a-105">When a class member is marked [extern](../language-reference/keywords/extern.md), it means that the member's definition is located in another file.</span></span> <span data-ttu-id="6f30a-106">Таким образом член класса, помеченный как **extern** , не может быть определен в классе.</span><span class="sxs-lookup"><span data-stu-id="6f30a-106">Therefore, a class member marked as **extern** cannot be defined in the class.</span></span> <span data-ttu-id="6f30a-107">Удалите ключевое слово `extern` или удалите определение.</span><span class="sxs-lookup"><span data-stu-id="6f30a-107">Either remove the `extern` keyword or delete the definition.</span></span> <span data-ttu-id="6f30a-108">Дополнительные сведения см. в статье [Методы](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="6f30a-108">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="6f30a-109">Следующий пример приводит к возникновению ошибки CS0179:</span><span class="sxs-lookup"><span data-stu-id="6f30a-109">The following sample generates CS0179:</span></span>  
  
```csharp  
// CS0179.cs  
public class MyClass  
{  
   public extern int ExternMethod(int aa)   // CS0179  
   {  
      return 0;  
   }  
   // try the following line instead  
   // public extern int ExternMethod(int aa);  
  
   public static void Main()  
   {  
   }  
}  
```
