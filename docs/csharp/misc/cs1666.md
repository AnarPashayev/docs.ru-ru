# <a name="compiler-error-cs1666"></a><span data-ttu-id="74dac-102">Ошибка компилятора CS1666</span><span class="sxs-lookup"><span data-stu-id="74dac-102">Compiler Error CS1666</span></span>
<span data-ttu-id="74dac-103">Нельзя использовать буферы фиксированного размера в нефиксированных выражениях.</span><span class="sxs-lookup"><span data-stu-id="74dac-103">You cannot use fixed size buffers contained in unfixed expressions.</span></span> <span data-ttu-id="74dac-104">Попробуйте использовать оператор fixed.</span><span class="sxs-lookup"><span data-stu-id="74dac-104">Try using the fixed statement.</span></span>  
  
 <span data-ttu-id="74dac-105">Эта ошибка возникает при использовании буфера фиксированного размера в выражении, включающем класс, который не является фиксированным.</span><span class="sxs-lookup"><span data-stu-id="74dac-105">This error occurs if you use the fixed sized buffer in an expression involving a class that is not itself fixed.</span></span> <span data-ttu-id="74dac-106">Среда выполнения может свободно перемещать нефиксированный класс для оптимизации доступа к памяти, что может привести к ошибкам при использовании буфера фиксированного размера.</span><span class="sxs-lookup"><span data-stu-id="74dac-106">The runtime is free to move the unfixed class around to optimize memory access, which could lead to errors when using the fixed sized buffer.</span></span> <span data-ttu-id="74dac-107">Чтобы избежать этой ошибки, используйте в операторе ключевое слово `fixed` .</span><span class="sxs-lookup"><span data-stu-id="74dac-107">To avoid this error, use the `fixed` keyword on the statement.</span></span>  
  
## <a name="example"></a><span data-ttu-id="74dac-108">Пример</span><span class="sxs-lookup"><span data-stu-id="74dac-108">Example</span></span>  
 <span data-ttu-id="74dac-109">Следующий пример приводит к возникновению ошибки CS1666.</span><span class="sxs-lookup"><span data-stu-id="74dac-109">The following sample generates CS1666.</span></span>  
  
```csharp  
// CS1666.cs  
// compile with: /unsafe /target:library  
unsafe struct S  
{  
   public fixed int buffer[1];  
}  
  
unsafe class Test  
{  
   S field = new S();  
  
   private bool example1()  
   {  
      return (field.buffer[0] == 0);   // CS1666 error  
   }  
  
   private bool example2()  
   {  
      // OK  
      fixed (S* p = &field)  
      {  
         return (p->buffer[0] == 0);  
      }  
   }  
  
   private bool example3()  
   {  
      S local = new S();  
      return (local.buffer[0] == 0);   
   }   
}  
```
