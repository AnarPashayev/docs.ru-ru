# <a name="compiler-warning-level-4-cs0078"></a><span data-ttu-id="5d2b0-103">Предупреждение компилятора (уровень 4) CS0078</span><span class="sxs-lookup"><span data-stu-id="5d2b0-103">Compiler Warning (level 4) CS0078</span></span>
<span data-ttu-id="5d2b0-104">Суффикс "l" легко спутать с цифрой "1" -- для ясности используйте "L"</span><span class="sxs-lookup"><span data-stu-id="5d2b0-104">The 'l' suffix is easily confused with the digit '1' -- use 'L' for clarity</span></span>  
  
 <span data-ttu-id="5d2b0-105">Компилятор выдает предупреждение при обнаружении приведения к типу long с использованием строчной l вместо прописной L.</span><span class="sxs-lookup"><span data-stu-id="5d2b0-105">The compiler warns when it detects a cast to long using a lowercase l instead of an uppercase L.</span></span>  
  
 <span data-ttu-id="5d2b0-106">Следующий пример приводит к возникновению ошибки CS0078:</span><span class="sxs-lookup"><span data-stu-id="5d2b0-106">The following sample generates CS0078:</span></span>  
  
```csharp  
// CS0078.cs  
// compile with: /W:4  
class test {  
   public static void TestL(long i)  
   {  
   }  
  
   public static void TestL(int i)  
   {  
   }  
  
   public static void Main()  
   {  
      TestL(25l);   // CS0078  
      // try the following line instead  
      // TestL(25L);  
   }  
}  
```
