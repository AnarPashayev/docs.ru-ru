# <a name="compiler-warning-level-1-cs0183"></a><span data-ttu-id="a236e-103">Предупреждение компилятора (уровень 1) CS0183</span><span class="sxs-lookup"><span data-stu-id="a236e-103">Compiler Warning (level 1) CS0183</span></span>

<span data-ttu-id="a236e-104">Указанное выражение всегда имеет указанный тип ("тип")</span><span class="sxs-lookup"><span data-stu-id="a236e-104">The given expression is always of the provided ('type') type</span></span>  
  
 <span data-ttu-id="a236e-105">Если условное выражение всегда равно **true**, вам не нужен условный оператор.</span><span class="sxs-lookup"><span data-stu-id="a236e-105">If a conditional statement always evaluates to **true**, then you do not need a conditional statement.</span></span> <span data-ttu-id="a236e-106">Это предупреждение возникает, когда вы пытаетесь проверить тип с помощью оператора **is** .</span><span class="sxs-lookup"><span data-stu-id="a236e-106">This warning occurs when you try to evaluate a type using the **is** operator.</span></span> <span data-ttu-id="a236e-107">Если вычисление является типом значения, такая проверка не требуется.</span><span class="sxs-lookup"><span data-stu-id="a236e-107">If the evaluation is a value type, then the check is unnecessary.</span></span>  
  
 <span data-ttu-id="a236e-108">При компиляции следующего примера будет выдано предупреждение CS0183:</span><span class="sxs-lookup"><span data-stu-id="a236e-108">The following sample generates CS0183:</span></span>  
  
```csharp  
// CS0183.cs  
// compile with: /W:1  
using System;  
public class Test  
{  
   public static void F(Int32 i32, String str)  
   {  
      if (str is Object)          // OK  
         Console.WriteLine( "str is an object" );  
      else  
         Console.WriteLine( "str is not an object" );  
  
      if (i32 is Object)   // CS0183  
         Console.WriteLine( "i32 is an object" );  
      else  
         Console.WriteLine( "i32 is not an object" ); // never reached  
   }  
  
   public static void Main()  
   {  
  
      F(0, "CS0183");  
      F(120, null);
   }  
}  
```
