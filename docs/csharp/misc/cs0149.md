# <a name="compiler-error-cs0149"></a><span data-ttu-id="a0bd1-103">Ошибка компилятора CS0149</span><span class="sxs-lookup"><span data-stu-id="a0bd1-103">Compiler Error CS0149</span></span>
<span data-ttu-id="a0bd1-104">Требуется имя метода.</span><span class="sxs-lookup"><span data-stu-id="a0bd1-104">Method name expected</span></span>  
  
 <span data-ttu-id="a0bd1-105">При создании [делегата](../language-reference/builtin-types/reference-types.md)укажите метод.</span><span class="sxs-lookup"><span data-stu-id="a0bd1-105">When creating a [delegate](../language-reference/builtin-types/reference-types.md), specify a method.</span></span> <span data-ttu-id="a0bd1-106">Дополнительные сведения см. в разделе [Делегаты](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="a0bd1-106">For more information, see [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="a0bd1-107">В следующем примере возникает ошибка CS0149:</span><span class="sxs-lookup"><span data-stu-id="a0bd1-107">The following sample generates CS0149:</span></span>  
  
```csharp  
// CS0149.cs  
using System;  
  
delegate string MyDelegate(int i);  
  
class MyClass  
{  
   // class member-field of the declared delegate type  
   static MyDelegate dt;
  
   public static void Main()  
   {  
      dt = new MyDelegate(17.45);   // CS0149  
      // try the following line instead  
      // dt = new MyDelegate(Func2);  
      F(dt);  
   }  
  
   public static string Func2(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void F(MyDelegate myFunc)  
   {  
      myFunc(8);  
   }  
}  
```
