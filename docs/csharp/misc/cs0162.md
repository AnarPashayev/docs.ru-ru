# <a name="compiler-warning-level-2-cs0162"></a><span data-ttu-id="8a061-102">Предупреждение компилятора (уровень 2) CS0162</span><span class="sxs-lookup"><span data-stu-id="8a061-102">Compiler Warning (level 2) CS0162</span></span>
<span data-ttu-id="8a061-103">Обнаружен недостижимый код</span><span class="sxs-lookup"><span data-stu-id="8a061-103">Unreachable code detected</span></span>  
  
 <span data-ttu-id="8a061-104">Компилятор обнаружил код, который никогда не будет выполняться.</span><span class="sxs-lookup"><span data-stu-id="8a061-104">The compiler detected code that will never be executed.</span></span>  
  
 <span data-ttu-id="8a061-105">Следующий пример приводит к возникновению предупреждения CS0162:</span><span class="sxs-lookup"><span data-stu-id="8a061-105">The following sample generates CS0162:</span></span>  
  
```csharp  
// CS0162.cs  
// compile with: /W:2  
public class A  
{  
   public static void Main()  
   {  
      goto lab1;  
      {  
         // The following statements cannot be reached:  
         int i = 9;   // CS0162   
         i++;  
      }  
      lab1:  
      {  
      }  
   }  
}  
```
