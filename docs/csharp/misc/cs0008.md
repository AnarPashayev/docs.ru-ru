# <a name="compiler-error-cs0008"></a>Ошибка компилятора CS0008

Непредвиденная ошибка чтения метаданных из файла "файл" — "описание"  
  
 Библиотека DLL была успешно открыта для извлечения метаданных, но она повреждена и не удается прочитать из нее данные. См. сведения в разделе [-reference (параметры компилятора C#)](../language-reference/compiler-options/reference-compiler-option.md).
