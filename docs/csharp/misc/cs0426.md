# <a name="compiler-error-cs0426"></a><span data-ttu-id="a8610-102">Ошибка компилятора CS0426</span><span class="sxs-lookup"><span data-stu-id="a8610-102">Compiler Error CS0426</span></span>
<span data-ttu-id="a8610-103">Имя типа "идентификатор" отсутствует в типе "тип"</span><span class="sxs-lookup"><span data-stu-id="a8610-103">The type name 'identifier' does not exist in the type 'type'</span></span>  
  
 <span data-ttu-id="a8610-104">Эта ошибка возникает при ссылке на тип, вложенный в другой тип, но такой вложенный тип не существует.</span><span class="sxs-lookup"><span data-stu-id="a8610-104">This error occurs when you reference a type nested within another type, but no such nested type exists.</span></span> <span data-ttu-id="a8610-105">Это может произойти, если имя вложенного типа указано с ошибкой.</span><span class="sxs-lookup"><span data-stu-id="a8610-105">This can occur if you mistype the name of the nested type.</span></span> <span data-ttu-id="a8610-106">Проверьте правильность написания имен и убедитесь, что во вмещающем типе имеется ожидаемый член.</span><span class="sxs-lookup"><span data-stu-id="a8610-106">Check the spelling of the names used, and verify that the enclosing type has the expected member.</span></span>  
  
 <span data-ttu-id="a8610-107">Следующий пример приводит к возникновению ошибки CS0426, потому что в классе C отсутствует вложенный тип A:</span><span class="sxs-lookup"><span data-stu-id="a8610-107">The following sample generates CS0426 because class C has no nested type A:</span></span>  
  
```csharp  
// CS0426.cs  
  
class C  
{  
    // No nested types are declared.     
}  
  
class D  
{  
   public static void Main()  
   {  
       C c = new C();  
       // Attempt to reference a nested type A:  
       C.A a; // CS0426 because no such type C.A  
   }  
}  
```
