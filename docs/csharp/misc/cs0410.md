 <span data-ttu-id="14f74-104">Эта ошибка возникает при попытке создать делегат с помощью функции, которая имеет неправильные типы параметров.</span><span class="sxs-lookup"><span data-stu-id="14f74-104">This error occurs if you try to instantiate a delegate with a function that has the wrong parameter types.</span></span> <span data-ttu-id="14f74-105">Типы параметров делегата должны совпадать с типами параметров функции, которая присваивается делегату.</span><span class="sxs-lookup"><span data-stu-id="14f74-105">The parameter types of the delegate must match the function that you are assigning to the delegate.</span></span>

## <a name="example"></a><span data-ttu-id="14f74-106">Пример</span><span class="sxs-lookup"><span data-stu-id="14f74-106">Example</span></span>

 <span data-ttu-id="14f74-107">Следующий пример приводит к возникновению ошибок CS0410:</span><span class="sxs-lookup"><span data-stu-id="14f74-107">The following example generates CS0410:</span></span>

```csharp
// CS0410.cs
// compile with: /langversion:ISO-1

class Test
{
    delegate void D(double d );
    static void F(int i) { }

    static void Main()
    {
        D d = new D(F);  // CS0410
    }
}
```
