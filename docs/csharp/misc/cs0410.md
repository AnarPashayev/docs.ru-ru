 <span data-ttu-id="8b579-105">Эта ошибка возникает при попытке создать делегат с помощью функции, которая имеет неправильные типы параметров.</span><span class="sxs-lookup"><span data-stu-id="8b579-105">This error occurs if you try to instantiate a delegate with a function that has the wrong parameter types.</span></span> <span data-ttu-id="8b579-106">Типы параметров делегата должны совпадать с типами параметров функции, которая присваивается делегату.</span><span class="sxs-lookup"><span data-stu-id="8b579-106">The parameter types of the delegate must match the function that you are assigning to the delegate.</span></span>

## <a name="example"></a><span data-ttu-id="8b579-107">Пример</span><span class="sxs-lookup"><span data-stu-id="8b579-107">Example</span></span>

 <span data-ttu-id="8b579-108">Следующий пример приводит к возникновению ошибок CS0410:</span><span class="sxs-lookup"><span data-stu-id="8b579-108">The following example generates CS0410:</span></span>

```csharp
// CS0410.cs
// compile with: /langversion:ISO-1

class Test
{
    delegate void D(double d );
    static void F(int i) { }

    static void Main()
    {
        D d = new D(F);  // CS0410
    }
}
```
