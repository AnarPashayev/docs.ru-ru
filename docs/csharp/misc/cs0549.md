# <a name="compiler-error-cs0549"></a><span data-ttu-id="fd47f-102">Ошибка компилятора CS0549</span><span class="sxs-lookup"><span data-stu-id="fd47f-102">Compiler Error CS0549</span></span>
<span data-ttu-id="fd47f-103">"функция" является новым виртуальным членом в запечатанном (sealed) классе "класс"</span><span class="sxs-lookup"><span data-stu-id="fd47f-103">'function' is a new virtual member in sealed class 'class'</span></span>  
  
 <span data-ttu-id="fd47f-104">[Запечатанный](../language-reference/keywords/sealed.md)[класс](../language-reference/keywords/class.md) не может использоваться в качестве базового класса.</span><span class="sxs-lookup"><span data-stu-id="fd47f-104">A [sealed](../language-reference/keywords/sealed.md)[class](../language-reference/keywords/class.md) cannot be used as a base class.</span></span>  <span data-ttu-id="fd47f-105">Таким образом, бесполезно определять виртуальный метод в запечатанном классе.</span><span class="sxs-lookup"><span data-stu-id="fd47f-105">Therefore, it is useless to have a virtual method in sealed class.</span></span>  
  
 <span data-ttu-id="fd47f-106">В следующем примере возникает ошибка CS0549:</span><span class="sxs-lookup"><span data-stu-id="fd47f-106">The following sample generates CS0549:</span></span>  
  
```csharp  
// CS0549.cs  
// compile with: /target:library  
sealed public class MyClass  
{  
   virtual public void TestMethod() {}   // CS0549  
   public void TestMethod2() {}   // OK  
}  
```
