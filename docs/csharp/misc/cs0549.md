# <a name="compiler-error-cs0549"></a><span data-ttu-id="3e827-103">Ошибка компилятора CS0549</span><span class="sxs-lookup"><span data-stu-id="3e827-103">Compiler Error CS0549</span></span>

<span data-ttu-id="3e827-104">"функция" является новым виртуальным членом в запечатанном (sealed) классе "класс"</span><span class="sxs-lookup"><span data-stu-id="3e827-104">'function' is a new virtual member in sealed class 'class'</span></span>  
  
 <span data-ttu-id="3e827-105">[Запечатанный](../language-reference/keywords/sealed.md)[класс](../language-reference/keywords/class.md) не может использоваться в качестве базового класса.</span><span class="sxs-lookup"><span data-stu-id="3e827-105">A [sealed](../language-reference/keywords/sealed.md)[class](../language-reference/keywords/class.md) cannot be used as a base class.</span></span>  <span data-ttu-id="3e827-106">Таким образом, бесполезно определять виртуальный метод в запечатанном классе.</span><span class="sxs-lookup"><span data-stu-id="3e827-106">Therefore, it is useless to have a virtual method in sealed class.</span></span>  
  
 <span data-ttu-id="3e827-107">В следующем примере возникает ошибка CS0549:</span><span class="sxs-lookup"><span data-stu-id="3e827-107">The following sample generates CS0549:</span></span>  
  
```csharp  
// CS0549.cs  
// compile with: /target:library  
sealed public class MyClass  
{  
   virtual public void TestMethod() {}   // CS0549  
   public void TestMethod2() {}   // OK  
}  
```
