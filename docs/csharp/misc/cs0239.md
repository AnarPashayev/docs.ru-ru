# <a name="compiler-error-cs0239"></a><span data-ttu-id="824b8-102">Ошибка компилятора CS0239</span><span class="sxs-lookup"><span data-stu-id="824b8-102">Compiler Error CS0239</span></span>
<span data-ttu-id="824b8-103">"член": невозможно переопределить наследуемый член "наследуемый_член", так как он запечатан</span><span class="sxs-lookup"><span data-stu-id="824b8-103">'member' : cannot override inherited member 'inherited member' because it is sealed</span></span>  
  
 <span data-ttu-id="824b8-104">Член не может [переопределить](../language-reference/keywords/override.md) [запечатанный](../language-reference/keywords/sealed.md) наследуемый член.</span><span class="sxs-lookup"><span data-stu-id="824b8-104">A member cannot [override](../language-reference/keywords/override.md) a [sealed](../language-reference/keywords/sealed.md) inherited member.</span></span> <span data-ttu-id="824b8-105">Дополнительные сведения см. в разделе [установленный и](../language-reference/keywords/checked-and-unchecked.md)непроверенный.</span><span class="sxs-lookup"><span data-stu-id="824b8-105">For more information, see [Checked and Unchecked](../language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="824b8-106">Следующий пример приводит к возникновению ошибки CS0239:</span><span class="sxs-lookup"><span data-stu-id="824b8-106">The following sample generates CS0239:</span></span>  
  
```csharp  
// CS0239.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public override sealed void f()  
   {  
   }  
}  
  
class MyClass3 : MyClass2  
{  
   public override void f()   // CS0239  
   // Try the following definition instead:  
   // public new void f()  
   {  
   }  
}  
```
