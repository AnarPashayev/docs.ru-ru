# <a name="compiler-error-cs1630"></a>Ошибка компилятора CS1630

Недопустимый параметр "параметр" для /errorreport; ожидается prompt, send, queue или none  
  
 После параметра командной строки [-errorreport](../language-reference/compiler-options/errorreport-compiler-option.md) должно следовать **приглашение**, **Отправить**, **поставить в очередь**или **нет**, указывая, какое действие следует предпринять при возникновении внутренней ошибки компилятора.
