## <a name="example"></a><span data-ttu-id="b02aa-109">Пример</span><span class="sxs-lookup"><span data-stu-id="b02aa-109">Example</span></span>  
 <span data-ttu-id="b02aa-110">В этом коде все разделяемые определения класса расположены в одном файле, но могут распространяться по нескольким файлам.</span><span class="sxs-lookup"><span data-stu-id="b02aa-110">In this code, the partial class definitions are all in a single file, but they could also be spread across multiple files.</span></span>  
  
```csharp

public class GenericsErrors
{  
    interface IFace1 { }  
    interface IFace2 { }  
    partial class PartialBadBounds<T> where T : IFace1 { } // CS0265  
    partial class PartialBadBounds<T> where T : IFace2 { }
}  
```
