# <a name="compiler-error-cs0265"></a><span data-ttu-id="86af3-103">Ошибка компилятора CS0265</span><span class="sxs-lookup"><span data-stu-id="86af3-103">Compiler Error CS0265</span></span>

<span data-ttu-id="86af3-104">Частичные объявления "тип" содержат непоследовательные ограничения для параметра типа "параметр типа"</span><span class="sxs-lookup"><span data-stu-id="86af3-104">Partial declarations of 'type' have inconsistent constraints for type parameter 'type parameter'</span></span>  
  
 <span data-ttu-id="86af3-105">Эта ошибка возникает при определении универсального класса как разделяемого класса, так что его разделяемые определения образуются в нескольких местах и ограничения универсального типа становятся несогласованными или отличаются в двух и более местах.</span><span class="sxs-lookup"><span data-stu-id="86af3-105">This error happens when you define a generic class as a partial class, so that its partial definitions occur in more than one place, and the constraints on the generic type are inconsistent or different in two or more places.</span></span> <span data-ttu-id="86af3-106">При указании ограничений в нескольких местах они должны быть одинаковыми.</span><span class="sxs-lookup"><span data-stu-id="86af3-106">If you specify the constraints in more than one place, they must all be identical.</span></span> <span data-ttu-id="86af3-107">Наиболее простым решением будет задать ограничения в одном месте и опустить их во всех остальных.</span><span class="sxs-lookup"><span data-stu-id="86af3-107">The easiest solution is to specify the constraints in one place, and omit them everywhere else.</span></span> <span data-ttu-id="86af3-108">Дополнительные сведения см. в разделе [разделяемые классы и методы](../programming-guide/classes-and-structs/partial-classes-and-methods.md) и [ограничения для параметров типа](../programming-guide/generics/constraints-on-type-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="86af3-108">For more information, see [Partial Classes and Methods](../programming-guide/classes-and-structs/partial-classes-and-methods.md) and [Constraints on Type Parameters](../programming-guide/generics/constraints-on-type-parameters.md).</span></span>  
  
 <span data-ttu-id="86af3-109">Приведенный ниже код вызывает ошибку CS0265.</span><span class="sxs-lookup"><span data-stu-id="86af3-109">The following code generates error CS0265.</span></span>  
  
## <a name="example"></a><span data-ttu-id="86af3-110">Пример</span><span class="sxs-lookup"><span data-stu-id="86af3-110">Example</span></span>  

 <span data-ttu-id="86af3-111">В этом коде все разделяемые определения класса расположены в одном файле, но могут распространяться по нескольким файлам.</span><span class="sxs-lookup"><span data-stu-id="86af3-111">In this code, the partial class definitions are all in a single file, but they could also be spread across multiple files.</span></span>  
  
```csharp  
// CS0265.cs  
public class GenericsErrors
{  
    interface IFace1 { }  
    interface IFace2 { }  
    partial class PartialBadBounds<T> where T : IFace1 { } // CS0265  
    partial class PartialBadBounds<T> where T : IFace2 { }
}  
```
