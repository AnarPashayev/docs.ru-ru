<span data-ttu-id="7bcc9-104">Передача аргумента "аргумент" в качестве параметра с ключевым словом ref или out или получение его адреса может вызвать исключение времени исполнения, поскольку она является полем класса, который упаковывается и маршалируется по ссылке.</span><span class="sxs-lookup"><span data-stu-id="7bcc9-104">Passing 'argument' as ref or out or taking its address may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="7bcc9-105">Любой класс, производный прямо или косвенно от <xref:System.MarshalByRefObject> , является классом, маршалируемым по ссылке.</span><span class="sxs-lookup"><span data-stu-id="7bcc9-105">Any class that derives, directly or indirectly, from <xref:System.MarshalByRefObject> is a marshal-by-reference class.</span></span> <span data-ttu-id="7bcc9-106">Такой класс может быть маршалирован по ссылке между процессами и компьютерами.</span><span class="sxs-lookup"><span data-stu-id="7bcc9-106">Such a class can be marshaled by reference across process and machine boundaries.</span></span> <span data-ttu-id="7bcc9-107">Таким образом, экземпляры этого класса могут быть прокси-элементами удаленного взаимодействия.</span><span class="sxs-lookup"><span data-stu-id="7bcc9-107">Thus, instances of this class could be proxies to remote objects.</span></span> <span data-ttu-id="7bcc9-108">Поле прокси-объекта нельзя передать как [ref](../language-reference/keywords/ref.md) или [out](../language-reference/keywords/out-parameter-modifier.md). Таким образом, нельзя передавать поля такого класса как `ref` или `out` , если только [это](../language-reference/keywords/this.md)не экземпляр, который не может быть прокси-объектом.</span><span class="sxs-lookup"><span data-stu-id="7bcc9-108">You cannot pass a field of a proxy object as [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md). So, you cannot pass fields of such a class as `ref` or `out`, unless the instance is [this](../language-reference/keywords/this.md), which can not be a proxy object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7bcc9-109">Пример</span><span class="sxs-lookup"><span data-stu-id="7bcc9-109">Example</span></span>  
 <span data-ttu-id="7bcc9-110">Следующий пример приводит к возникновению ошибки CS0197.</span><span class="sxs-lookup"><span data-stu-id="7bcc9-110">The following sample generates CS0197.</span></span>  
  
```csharp  
// CS0197.cs  
// compile with: /W:1  
class X : System.MarshalByRefObject  
{  
   public int i;  
}  
  
class M  
{  
   public int i;  
   static void AddSeventeen(ref int i)  
   {  
      i += 17;  
   }  
  
   static void Main()  
   {  
      X x = new X();  
      x.i = 12;  
      AddSeventeen(ref x.i);   // CS0197  
  
      // OK  
      M m = new M();  
      m.i = 12;  
      AddSeventeen(ref m.i);  
   }  
}  
```
