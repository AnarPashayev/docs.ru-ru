# <a name="compiler-warning-level-1-cs0197"></a><span data-ttu-id="b107e-102">Предупреждение компилятора (уровень 1) CS0197</span><span class="sxs-lookup"><span data-stu-id="b107e-102">Compiler Warning (level 1) CS0197</span></span>
<span data-ttu-id="b107e-103">Передача аргумента "аргумент" в качестве параметра с ключевым словом ref или out или получение его адреса может вызвать исключение времени исполнения, поскольку она является полем класса, который упаковывается и маршалируется по ссылке.</span><span class="sxs-lookup"><span data-stu-id="b107e-103">Passing 'argument' as ref or out or taking its address may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="b107e-104">Любой класс, производный прямо или косвенно от <xref:System.MarshalByRefObject> , является классом, маршалируемым по ссылке.</span><span class="sxs-lookup"><span data-stu-id="b107e-104">Any class that derives, directly or indirectly, from <xref:System.MarshalByRefObject> is a marshal-by-reference class.</span></span> <span data-ttu-id="b107e-105">Такой класс может быть маршалирован по ссылке между процессами и компьютерами.</span><span class="sxs-lookup"><span data-stu-id="b107e-105">Such a class can be marshaled by reference across process and machine boundaries.</span></span> <span data-ttu-id="b107e-106">Таким образом, экземпляры этого класса могут быть прокси-элементами удаленного взаимодействия.</span><span class="sxs-lookup"><span data-stu-id="b107e-106">Thus, instances of this class could be proxies to remote objects.</span></span> <span data-ttu-id="b107e-107">Вы не можете передавать поле прокси-элемента как [ref](../../csharp/language-reference/keywords/ref.md) или [out](../../csharp/language-reference/keywords/out-parameter-modifier.md). Таким образом, невозможно передать поля такого класса как `ref` или `out`, если экземпляр не является [this](../../csharp/language-reference/keywords/this.md), который не может быть объектом прокси.</span><span class="sxs-lookup"><span data-stu-id="b107e-107">You cannot pass a field of a proxy object as [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md). So, you cannot pass fields of such a class as `ref` or `out`, unless the instance is [this](../../csharp/language-reference/keywords/this.md), which can not be a proxy object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b107e-108">Пример</span><span class="sxs-lookup"><span data-stu-id="b107e-108">Example</span></span>  
 <span data-ttu-id="b107e-109">Следующий пример приводит к возникновению ошибки CS0197.</span><span class="sxs-lookup"><span data-stu-id="b107e-109">The following sample generates CS0197.</span></span>  
  
```csharp  
// CS0197.cs  
// compile with: /W:1  
class X : System.MarshalByRefObject  
{  
   public int i;  
}  
  
class M  
{  
   public int i;  
   static void AddSeventeen(ref int i)  
   {  
      i += 17;  
   }  
  
   static void Main()  
   {  
      X x = new X();  
      x.i = 12;  
      AddSeventeen(ref x.i);   // CS0197  
  
      // OK  
      M m = new M();  
      m.i = 12;  
      AddSeventeen(ref m.i);  
   }  
}  
```
