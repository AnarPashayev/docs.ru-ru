## <a name="example"></a><span data-ttu-id="69e12-109">Пример</span><span class="sxs-lookup"><span data-stu-id="69e12-109">Example</span></span>  
 <span data-ttu-id="69e12-110">В следующем примере происходит ошибка CS0687.</span><span class="sxs-lookup"><span data-stu-id="69e12-110">The following sample generates CS0687:</span></span>  
  
```csharp  
// CS0687.cs  
  
using M = Test;  
using System;  
  
public class Test
{  
    public static int x = 77;  
  
    public static void Main()
    {  
        Console.WriteLine(M::x); // CS0687  
        // To resolve use the following line instead:  
        // Console.WriteLine(M.x);  
    }  
}  
```
