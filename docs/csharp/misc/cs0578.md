# <a name="compiler-error-cs0578"></a><span data-ttu-id="4f868-102">Ошибка компилятора CS0578</span><span class="sxs-lookup"><span data-stu-id="4f868-102">Compiler Error CS0578</span></span>

<span data-ttu-id="4f868-103">Атрибут Conditional в функции "функция" не допускается, так как его тип возврата не void</span><span class="sxs-lookup"><span data-stu-id="4f868-103">The Conditional attribute is not valid on 'function' because its return type is not void</span></span>

<span data-ttu-id="4f868-104"><xref:System.Diagnostics.ConditionalAttribute> нельзя применить к методу, тип возврата которого отличен от `void`.</span><span class="sxs-lookup"><span data-stu-id="4f868-104"><xref:System.Diagnostics.ConditionalAttribute> cannot be applied to a method that has a return type other than `void`.</span></span> <span data-ttu-id="4f868-105">Это обусловлено тем, что любой другой тип возврата для метода может потребоваться в другой части программы.</span><span class="sxs-lookup"><span data-stu-id="4f868-105">The reason for this is that any other return type for a method may be needed by another part of your program.</span></span>

## <a name="example"></a><span data-ttu-id="4f868-106">Пример</span><span class="sxs-lookup"><span data-stu-id="4f868-106">Example</span></span>

<span data-ttu-id="4f868-107">Следующий пример приводит к возникновению ошибки CS0578.</span><span class="sxs-lookup"><span data-stu-id="4f868-107">The following sample generates CS0578.</span></span> <span data-ttu-id="4f868-108">Чтобы устранить эту ошибку, необходимо либо удалить <xref:System.Diagnostics.ConditionalAttribute>, либо изменить возвращаемое значение метода на `void`.</span><span class="sxs-lookup"><span data-stu-id="4f868-108">To resolve this error, you must either delete <xref:System.Diagnostics.ConditionalAttribute>, or you must change the return value of the method to `void`.</span></span>

```csharp
// CS0578.cs
// compile with: /target:library
public class MyClass
{
    [System.Diagnostics.ConditionalAttribute("a")]   // CS0578
    public int TestMethod()
    {
        return 0;
    }
}
```
