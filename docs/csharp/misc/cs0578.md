# <a name="compiler-error-cs0578"></a><span data-ttu-id="e09b6-103">Ошибка компилятора CS0578</span><span class="sxs-lookup"><span data-stu-id="e09b6-103">Compiler Error CS0578</span></span>

<span data-ttu-id="e09b6-104">Атрибут Conditional в функции "функция" не допускается, так как его тип возврата не void</span><span class="sxs-lookup"><span data-stu-id="e09b6-104">The Conditional attribute is not valid on 'function' because its return type is not void</span></span>

<span data-ttu-id="e09b6-105"><xref:System.Diagnostics.ConditionalAttribute> нельзя применить к методу, тип возврата которого отличен от `void`.</span><span class="sxs-lookup"><span data-stu-id="e09b6-105"><xref:System.Diagnostics.ConditionalAttribute> cannot be applied to a method that has a return type other than `void`.</span></span> <span data-ttu-id="e09b6-106">Это обусловлено тем, что любой другой тип возврата для метода может потребоваться в другой части программы.</span><span class="sxs-lookup"><span data-stu-id="e09b6-106">The reason for this is that any other return type for a method may be needed by another part of your program.</span></span>

## <a name="example"></a><span data-ttu-id="e09b6-107">Пример</span><span class="sxs-lookup"><span data-stu-id="e09b6-107">Example</span></span>

<span data-ttu-id="e09b6-108">Следующий пример приводит к возникновению ошибки CS0578.</span><span class="sxs-lookup"><span data-stu-id="e09b6-108">The following sample generates CS0578.</span></span> <span data-ttu-id="e09b6-109">Чтобы устранить эту ошибку, необходимо либо удалить <xref:System.Diagnostics.ConditionalAttribute>, либо изменить возвращаемое значение метода на `void`.</span><span class="sxs-lookup"><span data-stu-id="e09b6-109">To resolve this error, you must either delete <xref:System.Diagnostics.ConditionalAttribute>, or you must change the return value of the method to `void`.</span></span>

```csharp
// CS0578.cs
// compile with: /target:library
public class MyClass
{
    [System.Diagnostics.ConditionalAttribute("a")]   // CS0578
    public int TestMethod()
    {
        return 0;
    }
}
```
