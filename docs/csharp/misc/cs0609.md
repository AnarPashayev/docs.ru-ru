 <span data-ttu-id="fb009-105">Атрибут name (**IndexerNameAttribute**) не может применяться к индексированному свойству, которое является переопределением.</span><span class="sxs-lookup"><span data-stu-id="fb009-105">The name attribute (**IndexerNameAttribute**) cannot be applied to an indexed property that is an override.</span></span> <span data-ttu-id="fb009-106">Дополнительные сведения см. в разделе [Индексаторы](../programming-guide/indexers/index.md).</span><span class="sxs-lookup"><span data-stu-id="fb009-106">For more information, see [Indexers](../programming-guide/indexers/index.md).</span></span>

 <span data-ttu-id="fb009-107">В следующем примере возникает ошибка CS0609.</span><span class="sxs-lookup"><span data-stu-id="fb009-107">The following sample generates CS0609:</span></span>

```csharp
// CS0609.cs
using System;
using System.Runtime.CompilerServices;

public class idx
{
   public virtual int this[int iPropIndex]
   {
      get
      {
         return 0;
      }
      set
      {
      }
   }
}

public class MonthDays : idx
{
   [IndexerName("MonthInfoIndexer")]   // CS0609, delete to resolve this CS0609
   public override int this[int iPropIndex]
   {
      get
      {
         return 0;
      }
      set
      {
      }
   }
}

public class test
{
   public static void Main(string[] args)
   {
   }
}
```
