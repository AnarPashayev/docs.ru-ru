## <a name="example"></a><span data-ttu-id="6098f-105">Пример</span><span class="sxs-lookup"><span data-stu-id="6098f-105">Example</span></span>  

 <span data-ttu-id="6098f-106">Следующий пример приводит к возникновению ошибки CS1722:</span><span class="sxs-lookup"><span data-stu-id="6098f-106">The following sample generates CS1722.</span></span>  
  
```csharp

interface I {}  
  
public class MyClass : I, A {}   // CS1722

```
