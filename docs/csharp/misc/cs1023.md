# <a name="compiler-error-cs1023"></a><span data-ttu-id="e784f-103">Ошибка компилятора CS1023</span><span class="sxs-lookup"><span data-stu-id="e784f-103">Compiler Error CS1023</span></span>

<span data-ttu-id="e784f-104">Внедренный оператор не может быть объявлением или оператором с идентификатором.</span><span class="sxs-lookup"><span data-stu-id="e784f-104">Embedded statement cannot be a declaration or labeled statement</span></span>  
  
 <span data-ttu-id="e784f-105">Внедренный оператор, такой как операторы, следующие за оператором **If** , не может содержать ни объявления, ни операторы с метками.</span><span class="sxs-lookup"><span data-stu-id="e784f-105">An embedded statement, such as the statements following an **if** statement, can contain neither declarations nor labeled statements.</span></span>  
  
 <span data-ttu-id="e784f-106">Следующий пример дважды приводит к возникновению ошибки CS1023:</span><span class="sxs-lookup"><span data-stu-id="e784f-106">The following sample generates CS1023 twice:</span></span>  
  
```csharp  
// CS1023.cs  
public class a  
{  
   public static void Main()  
   {  
      if (1)  
         int i;      // CS1023, declaration is not valid here  
  
      if (1)  
         xx : i++;   // CS1023, labeled statement is not valid here  
   }  
}  
```
