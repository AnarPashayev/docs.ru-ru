# <a name="compiler-error-cs0743"></a><span data-ttu-id="58ac3-103">Ошибка компилятора CS0743</span><span class="sxs-lookup"><span data-stu-id="58ac3-103">Compiler Error CS0743</span></span>
<span data-ttu-id="58ac3-104">Требуется контекстное ключевое слово "on".</span><span class="sxs-lookup"><span data-stu-id="58ac3-104">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="58ac3-105">Для предложения `join` используется шаблон `join`...`in`...`on`...`equals`, как показано в следующем примере:</span><span class="sxs-lookup"><span data-stu-id="58ac3-105">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="58ac3-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="58ac3-106">To correct this error</span></span>  
  
1. <span data-ttu-id="58ac3-107">Добавьте ключевое слово `on` в предложение `join` .</span><span class="sxs-lookup"><span data-stu-id="58ac3-107">Add the `on` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="58ac3-108">Пример</span><span class="sxs-lookup"><span data-stu-id="58ac3-108">Example</span></span>  
 <span data-ttu-id="58ac3-109">Следующий код приводит к возникновению ошибки CS0743:</span><span class="sxs-lookup"><span data-stu-id="58ac3-109">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
