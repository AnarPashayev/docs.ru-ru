 <span data-ttu-id="175e2-104">Переменная [const](../language-reference/keywords/const.md) не может принимать в качестве своего значения выражение, которое не является константой.</span><span class="sxs-lookup"><span data-stu-id="175e2-104">A [const](../language-reference/keywords/const.md) variable cannot take as its value an expression that is not constant.</span></span> <span data-ttu-id="175e2-105">Дополнительные сведения см. в разделе [Константы](../programming-guide/classes-and-structs/constants.md).</span><span class="sxs-lookup"><span data-stu-id="175e2-105">For more information, see [Constants](../programming-guide/classes-and-structs/constants.md).</span></span>

   }
