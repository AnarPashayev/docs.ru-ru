# <a name="compiler-error-cs1628"></a><span data-ttu-id="a6fb3-103">Ошибка компилятора CS1628</span><span class="sxs-lookup"><span data-stu-id="a6fb3-103">Compiler Error CS1628</span></span>

<span data-ttu-id="a6fb3-104">Нельзя использовать параметр с ключевым словом ref или out в параметре в анонимном методе, лямбда-выражении или выражении запроса</span><span class="sxs-lookup"><span data-stu-id="a6fb3-104">Cannot use in ref or out parameter 'parameter' inside an anonymous method, lambda expression, or query expression</span></span>  
  
 <span data-ttu-id="a6fb3-105">Эта ошибка возникает при использовании `in` `ref` параметра, или `out` в блоке анонимного метода.</span><span class="sxs-lookup"><span data-stu-id="a6fb3-105">This error occurs if you use an `in`, `ref`, or `out` parameter inside an anonymous method block.</span></span> <span data-ttu-id="a6fb3-106">Чтобы избежать этой ошибки, используйте локальную переменную или другую конструкцию.</span><span class="sxs-lookup"><span data-stu-id="a6fb3-106">To avoid this error, use a local variable or some other construct.</span></span>  
  
 <span data-ttu-id="a6fb3-107">Следующий пример приводит к возникновению ошибки CS1628:</span><span class="sxs-lookup"><span data-stu-id="a6fb3-107">The following sample generates CS1628:</span></span>  
  
```csharp  
// CS1628.cs  
  
delegate int MyDelegate();  
  
class C  
{  
  public static void F(ref int i)  
  {  
      MyDelegate d = delegate { return i; };  // CS1628  
      // Try this instead:  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }  
  
  public static void Main()  
  {  
  
  }  
}  
```
