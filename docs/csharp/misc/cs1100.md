# <a name="compiler-error-cs1100"></a><span data-ttu-id="e448e-102">Ошибка компилятора CS1100</span><span class="sxs-lookup"><span data-stu-id="e448e-102">Compiler Error CS1100</span></span>
<span data-ttu-id="e448e-103">Метод "имя" имеет модификатор параметра this, который не находится в первом параметре метода.</span><span class="sxs-lookup"><span data-stu-id="e448e-103">Method 'name' has a parameter modifier 'this' which is not on the first parameter.</span></span>  
  
 <span data-ttu-id="e448e-104">Модификатор `this` разрешен только в первом параметре метода, который сообщает компилятору, что метод является методом расширения.</span><span class="sxs-lookup"><span data-stu-id="e448e-104">The `this` modifier is allowed only on the first parameter of a method, which indicates to the compiler that the method is an extension method.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e448e-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="e448e-105">To correct this error</span></span>
