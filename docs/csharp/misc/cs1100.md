# <a name="compiler-error-cs1100"></a><span data-ttu-id="ffa38-102">Ошибка компилятора CS1100</span><span class="sxs-lookup"><span data-stu-id="ffa38-102">Compiler Error CS1100</span></span>
<span data-ttu-id="ffa38-103">Метод "имя" имеет модификатор параметра this, который не находится в первом параметре метода.</span><span class="sxs-lookup"><span data-stu-id="ffa38-103">Method 'name' has a parameter modifier 'this' which is not on the first parameter.</span></span>  
  
 <span data-ttu-id="ffa38-104">Модификатор `this` разрешен только в первом параметре метода, который сообщает компилятору, что метод является методом расширения.</span><span class="sxs-lookup"><span data-stu-id="ffa38-104">The `this` modifier is allowed only on the first parameter of a method, which indicates to the compiler that the method is an extension method.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="ffa38-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="ffa38-105">To correct this error</span></span>  
  
1. <span data-ttu-id="ffa38-106">Удалите модификатор `this` из всех параметров метода, кроме первого.</span><span class="sxs-lookup"><span data-stu-id="ffa38-106">Remove the `this` modifier from all except the first parameter of the method.</span></span>
