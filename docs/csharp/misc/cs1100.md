# <a name="compiler-error-cs1100"></a><span data-ttu-id="f2d01-103">Ошибка компилятора CS1100</span><span class="sxs-lookup"><span data-stu-id="f2d01-103">Compiler Error CS1100</span></span>

<span data-ttu-id="f2d01-104">Метод "имя" имеет модификатор параметра this, который не находится в первом параметре метода.</span><span class="sxs-lookup"><span data-stu-id="f2d01-104">Method 'name' has a parameter modifier 'this' which is not on the first parameter.</span></span>  
  
 <span data-ttu-id="f2d01-105">Модификатор `this` разрешен только в первом параметре метода, который сообщает компилятору, что метод является методом расширения.</span><span class="sxs-lookup"><span data-stu-id="f2d01-105">The `this` modifier is allowed only on the first parameter of a method, which indicates to the compiler that the method is an extension method.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="f2d01-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="f2d01-106">To correct this error</span></span>
