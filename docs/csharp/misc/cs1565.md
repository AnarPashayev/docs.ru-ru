# <a name="compiler-error-cs1565"></a>Ошибка компилятора CS1565
Заданы несовместимые параметры: Файл ресурсов Win32; Значок Win32  
  
 Не допускается указывать оба параметра компилятора [/win32res](../language-reference/compiler-options/win32res-compiler-option.md) и [/win32ico](../language-reference/compiler-options/win32icon-compiler-option.md) в одной компиляции.
