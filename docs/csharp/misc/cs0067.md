<span data-ttu-id="dea58-103">Событие "событие" не используется</span><span class="sxs-lookup"><span data-stu-id="dea58-103">The event 'event' is never used</span></span>  
  
 <span data-ttu-id="dea58-104">[Событие](../language-reference/keywords/event.md) объявлено, но ни разу не используется в классе, в котором объявлено.</span><span class="sxs-lookup"><span data-stu-id="dea58-104">An [event](../language-reference/keywords/event.md) was declared but never used in the class in which it was declared.</span></span>

```csharp  
// CS0067.cs  
// compile with: /W:3

delegate void MyDelegate();  
  
class MyClass  
{  
   public event MyDelegate evt;   // CS0067  
   // uncomment TestMethod to resolve this CS0067

   private void TestMethod()  
   {  
      if (evt != null)  
         evt();  
   }  
*/  
   public static void Main()  
   {  
   }  
}  
```
