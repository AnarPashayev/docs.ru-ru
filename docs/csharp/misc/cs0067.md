 <span data-ttu-id="4609a-105">[Событие](../language-reference/keywords/event.md) объявлено, но ни разу не используется в классе, в котором объявлено.</span><span class="sxs-lookup"><span data-stu-id="4609a-105">An [event](../language-reference/keywords/event.md) was declared but never used in the class in which it was declared.</span></span>

```csharp

delegate void MyDelegate();  
  
class MyClass  
{  
   public event MyDelegate evt;   // CS0067

   {  
      if (evt != null)  
         evt();  
   }  
*/

   {  
   }  
}  
```
