# <a name="compiler-error-cs0455"></a><span data-ttu-id="2431c-103">Ошибка компилятора CS0455</span><span class="sxs-lookup"><span data-stu-id="2431c-103">Compiler Error CS0455</span></span>
<span data-ttu-id="2431c-104">Параметр типа "Имя параметра типа" наследует конфликтующие ограничения "Имя ограничения 1" и "Имя ограничения 2"</span><span class="sxs-lookup"><span data-stu-id="2431c-104">Type parameter 'Type Parameter Name' inherits conflicting constraints 'Constraint Name 1' and 'Constraint Name 2'</span></span>  
  
 <span data-ttu-id="2431c-105">Как правило, эта ошибка возникает, если ограничения устанавливаются таким образом, что параметр типа наследуется от двух несвязанных классов либо одновременно от ограничения типа класса или ссылки и ограничения типа значения или `struct` .</span><span class="sxs-lookup"><span data-stu-id="2431c-105">Two common ways to get this error are to set up constraints so that the type parameter derives from two non-related classes, or so that it derives from a class type or reference type constraint and a `struct` type or value type constraint.</span></span> <span data-ttu-id="2431c-106">Чтобы устранить эту ошибку, разрешите конфликт в иерархии наследования.</span><span class="sxs-lookup"><span data-stu-id="2431c-106">To resolve this error, remove the conflict from your inheritance hierarchy.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2431c-107">Пример</span><span class="sxs-lookup"><span data-stu-id="2431c-107">Example</span></span>  
 <span data-ttu-id="2431c-108">Приведенный ниже код вызывает ошибку CS0455.</span><span class="sxs-lookup"><span data-stu-id="2431c-108">The following code generates error CS0455.</span></span>

