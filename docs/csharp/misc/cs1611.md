# <a name="compiler-error-cs1611"></a><span data-ttu-id="8dc3c-103">Ошибка компилятора CS1611</span><span class="sxs-lookup"><span data-stu-id="8dc3c-103">Compiler Error CS1611</span></span>
<span data-ttu-id="8dc3c-104">Параметр params не может быть объявлен в ref или out</span><span class="sxs-lookup"><span data-stu-id="8dc3c-104">The params parameter cannot be declared as in ref or out</span></span>  
  
 <span data-ttu-id="8dc3c-105">Ключевые слова [в](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) и [out](../language-reference/keywords/out-parameter-modifier.md) не могут использоваться с ключевым словом [params](../language-reference/keywords/params.md) .</span><span class="sxs-lookup"><span data-stu-id="8dc3c-105">The keywords [in](../language-reference/keywords/in-parameter-modifier.md), [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) cannot be used with the [params](../language-reference/keywords/params.md) keyword.</span></span>  
  
 <span data-ttu-id="8dc3c-106">В следующем примере возникает ошибка CS1611:</span><span class="sxs-lookup"><span data-stu-id="8dc3c-106">The following sample generates CS1611:</span></span>  
  
```csharp  
// CS1611.cs  
public class MyClass  
{  
   public static void Test(params ref int[] a)   // CS1611, remove ref  
   {  
   }  
  
   public static void Main()  
   {  
      Test(1);  
   }  
}  
```
