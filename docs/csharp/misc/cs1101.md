# <a name="compiler-error-cs1101"></a><span data-ttu-id="e9c5d-103">Ошибка компилятора CS1101</span><span class="sxs-lookup"><span data-stu-id="e9c5d-103">Compiler Error CS1101</span></span>
<span data-ttu-id="e9c5d-104">Модификатор параметра "ref" не может использоваться с "this"</span><span class="sxs-lookup"><span data-stu-id="e9c5d-104">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="e9c5d-105">Изменение ключевым словом `this` первого параметра статического метода сообщает компилятору, что метод является методом расширения.</span><span class="sxs-lookup"><span data-stu-id="e9c5d-105">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="e9c5d-106">Никакие другие модификаторы не требуются и не допускаются в первом параметре метода расширения.</span><span class="sxs-lookup"><span data-stu-id="e9c5d-106">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e9c5d-107">Пример</span><span class="sxs-lookup"><span data-stu-id="e9c5d-107">Example</span></span>  
 <span data-ttu-id="e9c5d-108">Следующий пример приводит к возникновению ошибки CS1101:</span><span class="sxs-lookup"><span data-stu-id="e9c5d-108">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
