# <a name="compiler-error-cs1917"></a><span data-ttu-id="30ec0-102">Ошибка компилятора CS1917</span><span class="sxs-lookup"><span data-stu-id="30ec0-102">Compiler Error CS1917</span></span>

<span data-ttu-id="30ec0-103">Членам поля только для чтения "имя" типа "имя_структуры" не могут быть присвоены значения с помощью инициализатора объекта, так как они имеют тип значения.</span><span class="sxs-lookup"><span data-stu-id="30ec0-103">Members of read-only field 'name' of type 'struct name' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="30ec0-104">Поля, доступные только для чтения, которые являются типами значений, могут быть назначены только в конструкторе.</span><span class="sxs-lookup"><span data-stu-id="30ec0-104">Read-only fields that are value types can only be assigned in a constructor.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="30ec0-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="30ec0-105">To correct this error</span></span>  
  
- <span data-ttu-id="30ec0-106">Измените тип struct на class.</span><span class="sxs-lookup"><span data-stu-id="30ec0-106">Change the struct to a class type.</span></span>  
  
- <span data-ttu-id="30ec0-107">Инициализируйте структуру в конструкторе.</span><span class="sxs-lookup"><span data-stu-id="30ec0-107">Initialize the struct with a constructor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="30ec0-108">Пример</span><span class="sxs-lookup"><span data-stu-id="30ec0-108">Example</span></span>  

 <span data-ttu-id="30ec0-109">При компиляции следующего примера возникнет ошибка CS1917:</span><span class="sxs-lookup"><span data-stu-id="30ec0-109">The following code generates CS1917:</span></span>  
  
```csharp  
// cs1917.cs  
class CS1917  
{  
    public struct TestStruct  
    {  
        public int i;  
    }  
    public class C  
    {  
        public readonly TestStruct str = new TestStruct();  
        public static int Main()  
        {  
            C c = new C { str = { i = 1 } }; // CS1917  
            return 0;  
        }  
    }  
}  
```
