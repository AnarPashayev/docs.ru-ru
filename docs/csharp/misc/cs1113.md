 <span data-ttu-id="bd834-105">Методы расширения, определенные для типов классов, могут применяться для создания делегатов.</span><span class="sxs-lookup"><span data-stu-id="bd834-105">Extension methods that are defined for class types can be used to create delegates.</span></span> <span data-ttu-id="bd834-106">Методы расширения, определенные для типов значений, не могут.</span><span class="sxs-lookup"><span data-stu-id="bd834-106">Extension methods that are defined for value types cannot.</span></span>

## <a name="example"></a><span data-ttu-id="bd834-110">Пример</span><span class="sxs-lookup"><span data-stu-id="bd834-110">Example</span></span>  

 <span data-ttu-id="bd834-111">В следующем примере возникает ошибка CS1113:</span><span class="sxs-lookup"><span data-stu-id="bd834-111">The following example generates CS1113:</span></span>  
  
```csharp  
// cs1113.cs  
using System;  
public static class Extensions  
{  
    public static S ExtMethod(this S s)  
    {  
        return s;  
    }  
}  
  
public struct S  
{  
}  
  
public class Test  
{  
    static int Main()  
    {  
        Func<S> f = new S().ExtMethod; // CS1113  
        return 1;  
    }  
}  
```
