 <span data-ttu-id="bede6-106">Объявление класса включает ключевое слово [new](../language-reference/keywords/new-modifier.md) , хотя это объявление не переопределяет существующее определение в базовом классе.</span><span class="sxs-lookup"><span data-stu-id="bede6-106">A class declaration included the [new](../language-reference/keywords/new-modifier.md) keyword even though the declaration does not override an existing declaration in a base class.</span></span> <span data-ttu-id="bede6-107">Вы можете удалить ключевое слово **new** .</span><span class="sxs-lookup"><span data-stu-id="bede6-107">You can delete the **new** keyword.</span></span>  
  
 <span data-ttu-id="bede6-108">Следующий пример приводит к возникновению предупреждения CS0109:</span><span class="sxs-lookup"><span data-stu-id="bede6-108">The following sample generates CS0109:</span></span>  
  
```csharp  
// CS0109.cs  
// compile with: /W:4  
namespace x  
{  
   public class a  
   {  
      public int i;  
   }  
  
   public class b : a  
   {  
      public new int i;  
      public new int j;   // CS0109  
      public static void Main()  
      {  
      }  
   }  
}  
```
