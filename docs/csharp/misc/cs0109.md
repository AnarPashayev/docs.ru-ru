 <span data-ttu-id="28674-106">Объявление класса включает ключевое слово [new](../language-reference/keywords/new-modifier.md) , хотя это объявление не переопределяет существующее определение в базовом классе.</span><span class="sxs-lookup"><span data-stu-id="28674-106">A class declaration included the [new](../language-reference/keywords/new-modifier.md) keyword even though the declaration does not override an existing declaration in a base class.</span></span> <span data-ttu-id="28674-107">Вы можете удалить ключевое слово **new** .</span><span class="sxs-lookup"><span data-stu-id="28674-107">You can delete the **new** keyword.</span></span>

```csharp  
// CS0109.cs

{  
   public class a  
   {  
      public int i;  
   }  
  
   public class b : a  
   {  
      public new int i;  
      public new int j;   // CS0109  
      public static void Main()  
      {  
      }  
   }  
}  
```
