<span data-ttu-id="a06d9-104">"Конструкция" "идентификатор" не является универсальным методом.</span><span class="sxs-lookup"><span data-stu-id="a06d9-104">The 'construct' 'identifier' is not a generic method.</span></span> <span data-ttu-id="a06d9-105">Для реализации списка выражений используйте круглые скобки вокруг выражений <.</span><span class="sxs-lookup"><span data-stu-id="a06d9-105">If you intended an expression list, use parentheses around the < expression.</span></span>  
  
 <span data-ttu-id="a06d9-106">Названная конструкция не является типом или методом, единственными конструкциями, которые могут принимать универсальные аргументы.</span><span class="sxs-lookup"><span data-stu-id="a06d9-106">The construct named was not a type or a method, the only constructs that can take generic arguments.</span></span> <span data-ttu-id="a06d9-107">Удалите аргументы типа в угловых скобках.</span><span class="sxs-lookup"><span data-stu-id="a06d9-107">Remove the type arguments in angle brackets.</span></span> <span data-ttu-id="a06d9-108">Если необходим универсальный объект, объявите свою универсальную конструкцию как универсальный тип или метод.</span><span class="sxs-lookup"><span data-stu-id="a06d9-108">If a generic is needed, declare your generic construct as a generic type or method.</span></span>  
  
 <span data-ttu-id="a06d9-109">В следующем примере возникает ошибка CS0307:</span><span class="sxs-lookup"><span data-stu-id="a06d9-109">The following sample generates CS0307:</span></span>  
  
```csharp  
// CS0307.cs  
class C  
{  
   public int P { get { return 1; } }  
   public static void Main()  
   {  
      C c = new C();  
      int p = c.P<int>();  // CS0307 – C.P is a property  
      // Try this instead  
      // int p = c.P;  
   }  
}  
```
