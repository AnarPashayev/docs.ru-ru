# <a name="compiler-error-cs0756"></a><span data-ttu-id="83805-102">Ошибка компилятора CS0756</span><span class="sxs-lookup"><span data-stu-id="83805-102">Compiler Error CS0756</span></span>

 <span data-ttu-id="83805-104">В определяющем объявлении разделяемого метода определяется сигнатура метода, но не его реализация (тело метода).</span><span class="sxs-lookup"><span data-stu-id="83805-104">The defining declaration of a partial method is the part that specifies the method signature, but not the implementation (method body).</span></span> <span data-ttu-id="83805-105">Для каждой уникальной сигнатуры разделяемый метод должен содержать одно и только одно определяющее объявление.</span><span class="sxs-lookup"><span data-stu-id="83805-105">A partial method must have exactly one defining declaration for each unique signature.</span></span> <span data-ttu-id="83805-106">Каждая перегруженная версия разделяемого метода должна содержать собственное определяющее объявление.</span><span class="sxs-lookup"><span data-stu-id="83805-106">Each overloaded version of a partial method must have its own defining declaration.</span></span>
