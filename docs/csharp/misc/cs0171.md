# <a name="compiler-error-cs0171"></a><span data-ttu-id="b4927-102">Ошибка компилятора CS0171</span><span class="sxs-lookup"><span data-stu-id="b4927-102">Compiler Error CS0171</span></span>
<span data-ttu-id="b4927-103">Резервное поле для автоматически реализуемого свойства "имя" должно быть полностью назначено до возврата управления в вызывающий объект.</span><span class="sxs-lookup"><span data-stu-id="b4927-103">Backing field for automatically implemented property 'name' must be fully assigned before control is returned to the caller.</span></span> <span data-ttu-id="b4927-104">Попробуйте вызвать конструктор без параметров из инициализатора конструктора.</span><span class="sxs-lookup"><span data-stu-id="b4927-104">Consider calling the parameterless constructor from a constructor initializer.</span></span>  
  
 <span data-ttu-id="b4927-105">Конструктор в [структуре](../../csharp/language-reference/keywords/struct.md) должен инициализировать все поля структуры.</span><span class="sxs-lookup"><span data-stu-id="b4927-105">A constructor in a [struct](../../csharp/language-reference/keywords/struct.md) must initialize all fields in the struct.</span></span> <span data-ttu-id="b4927-106">Дополнительные сведения см. в разделе [Конструкторы](../../csharp/programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="b4927-106">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="b4927-107">Следующий пример приводит к возникновению ошибки CS0171:</span><span class="sxs-lookup"><span data-stu-id="b4927-107">The following sample generates CS0171:</span></span>  
  
```csharp  
// CS0171.cs  
struct MyStruct  
{  
   MyStruct(int initField)   // CS0171  
   {  
      // i = initField;      // uncomment this line to resolve this error  
   }  
   public int i;  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      MyStruct aStruct = new MyStruct();  
   }  
}  
```
