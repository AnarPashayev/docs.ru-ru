# <a name="compiler-error-cs0005"></a>Ошибка компилятора CS0005
После параметра компилятора "параметр_компилятора" должен быть указан аргумент  
  
 Для некоторых параметров компилятора требуются параметры. Если не передать аргументы, необходимые для параметра компилятора, возникнет ошибка CS0005.  
  
 Дополнительные сведения см. в разделе [Параметры компилятора C#](../language-reference/compiler-options/index.md).
