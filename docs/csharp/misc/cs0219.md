# <a name="compiler-warning-level-3-cs0219"></a><span data-ttu-id="76754-102">Предупреждение компилятора (уровень 3) CS0219</span><span class="sxs-lookup"><span data-stu-id="76754-102">Compiler Warning (level 3) CS0219</span></span>
<span data-ttu-id="76754-103">Переменной "переменная" присвоено значение, но оно ни разу не использовалось</span><span class="sxs-lookup"><span data-stu-id="76754-103">The variable 'variable' is assigned but its value is never used</span></span>

 <span data-ttu-id="76754-104">Если переменная объявлена и ей присвоено значение, но оно не используется, компилятор выдает предупреждение третьего уровня.</span><span class="sxs-lookup"><span data-stu-id="76754-104">The compiler issues a level-three warning, when you declare and assign a variable, but do not use it.</span></span>

 > [!NOTE]
 > <span data-ttu-id="76754-105">Компилятор создает это предупреждение, только если значение переменной является константой во время компиляции.</span><span class="sxs-lookup"><span data-stu-id="76754-105">The compiler generates this warning only when the variable value is a compile-time constant.</span></span> <span data-ttu-id="76754-106">Присвоение неконстантного выражения или результата метода локальной переменной упрощает просмотр этих выражений в отладчике.</span><span class="sxs-lookup"><span data-stu-id="76754-106">Assigning a non-constant expression or method result to a local variable makes it easier to observe those expressions in the debugger.</span></span> <span data-ttu-id="76754-107">Это также делает результат достижимым, предотвращая сбор мусора, пока эта переменная достижима.</span><span class="sxs-lookup"><span data-stu-id="76754-107">It also makes the result reachable, preventing garbage collection while that variable is reachable.</span></span>
 
 <span data-ttu-id="76754-108">В следующем примере показаны случаи, когда и когда не создается предупреждение:</span><span class="sxs-lookup"><span data-stu-id="76754-108">The following sample shows the cases when and when not the warning is generated:</span></span>

```csharp
// CS0219.cs
// compile with: /W:3
public class MyClass
{
   public static void Main()
   {
      var interpolated = "Interpolated";
      var a = 0; // CS0219
      int b = GetZero(); // Doesn't generate a warning.
      var c = "Regular string"; // CS0219
      var d = $"Constant interpolated string"; // Doesn't generate a warning.
      var e = $"{interpolated} string"; // Doesn't generate a warning.
   }
   
   private static int GetZero()
   {
       return 0;
   }
}  
```
