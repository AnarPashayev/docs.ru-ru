# <a name="compiler-error-cs1621"></a><span data-ttu-id="2d4fc-102">Ошибка компилятора CS1621</span><span class="sxs-lookup"><span data-stu-id="2d4fc-102">Compiler Error CS1621</span></span>
<span data-ttu-id="2d4fc-103">Внутри анонимного метода или лямбда-выражения нельзя использовать оператор yield.</span><span class="sxs-lookup"><span data-stu-id="2d4fc-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
<span data-ttu-id="2d4fc-104">Нельзя использовать оператор [yield](../../csharp/language-reference/keywords/yield.md) в [анонимном методе](../language-reference/operators/delegate-operator.md) или [лямбда-выражении](../../csharp/programming-guide/statements-expressions-operators/lambda-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="2d4fc-104">You cannot use the [yield](../../csharp/language-reference/keywords/yield.md) statement in an [anonymous method](../language-reference/operators/delegate-operator.md) or a [lambda expression](../../csharp/programming-guide/statements-expressions-operators/lambda-expressions.md).</span></span>
  
<span data-ttu-id="2d4fc-105">Следующий пример приводит к возникновению ошибки CS1621:</span><span class="sxs-lookup"><span data-stu-id="2d4fc-105">The following sample generates CS1621:</span></span>
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
