# <a name="compiler-error-cs1621"></a><span data-ttu-id="cf1b2-103">Ошибка компилятора CS1621</span><span class="sxs-lookup"><span data-stu-id="cf1b2-103">Compiler Error CS1621</span></span>
<span data-ttu-id="cf1b2-104">Внутри анонимного метода или лямбда-выражения нельзя использовать оператор yield.</span><span class="sxs-lookup"><span data-stu-id="cf1b2-104">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
<span data-ttu-id="cf1b2-105">Нельзя использовать оператор [yield](../language-reference/keywords/yield.md) в [анонимном методе](../language-reference/operators/delegate-operator.md) или [лямбда-выражении](../language-reference/operators/lambda-expressions.md).</span><span class="sxs-lookup"><span data-stu-id="cf1b2-105">You cannot use the [yield](../language-reference/keywords/yield.md) statement in an [anonymous method](../language-reference/operators/delegate-operator.md) or a [lambda expression](../language-reference/operators/lambda-expressions.md).</span></span>
  
<span data-ttu-id="cf1b2-106">Следующий пример приводит к возникновению ошибки CS1621:</span><span class="sxs-lookup"><span data-stu-id="cf1b2-106">The following sample generates CS1621:</span></span>
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
