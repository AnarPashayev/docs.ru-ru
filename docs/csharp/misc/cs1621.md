# <a name="compiler-error-cs1621"></a>Ошибка компилятора CS1621
Внутри анонимного метода или лямбда-выражения нельзя использовать оператор yield.  
  
Нельзя использовать оператор [yield](../language-reference/keywords/yield.md) в [анонимном методе](../language-reference/operators/delegate-operator.md) или [лямбда-выражении](../programming-guide/statements-expressions-operators/lambda-expressions.md).
  
Следующий пример приводит к возникновению ошибки CS1621:
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
