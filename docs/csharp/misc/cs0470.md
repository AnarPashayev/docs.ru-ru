## <a name="example"></a><span data-ttu-id="b26ea-108">Пример</span><span class="sxs-lookup"><span data-stu-id="b26ea-108">Example</span></span>  

 <span data-ttu-id="b26ea-109">В следующем примере возникает ошибка CS0470:</span><span class="sxs-lookup"><span data-stu-id="b26ea-109">The following sample generates CS0470.</span></span>  
  
```csharp  
// CS0470.cs  
// compile with: /target:library  
  
interface I  
{  
   int P { get; }  
}  
  
class MyClass : I  
{  
   public int get_P() { return 0; }   // CS0470

}  
```
