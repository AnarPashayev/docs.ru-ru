# <a name="compiler-error-cs0719"></a><span data-ttu-id="1ebef-103">Ошибка компилятора CS0719</span><span class="sxs-lookup"><span data-stu-id="1ebef-103">Compiler Error CS0719</span></span>
<span data-ttu-id="1ebef-104">"тип": элементы массива не могут иметь статический тип</span><span class="sxs-lookup"><span data-stu-id="1ebef-104">'type': array elements cannot be of static type</span></span>  
  
 <span data-ttu-id="1ebef-105">Массив статического типа не имеет смысла, поскольку элементы массива являются экземплярами, но невозможно создать экземпляры статических типов.</span><span class="sxs-lookup"><span data-stu-id="1ebef-105">An array of static type does not make sense since array elements are instances, but it is not possible to create instances of static types.</span></span>  
  
 <span data-ttu-id="1ebef-106">В следующем примере возникает ошибка CS0719:</span><span class="sxs-lookup"><span data-stu-id="1ebef-106">The following sample generates CS0719:</span></span>  
  
```csharp  
// CS0719.cs  
public static class SC  
{  
   public static void F()  
   {  
   }  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC[] sca = new SC[10];  // CS0719  
   }  
}  
```
