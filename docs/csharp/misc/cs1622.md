# <a name="compiler-error-cs1622"></a><span data-ttu-id="90c35-103">Ошибка компилятора CS1622</span><span class="sxs-lookup"><span data-stu-id="90c35-103">Compiler Error CS1622</span></span>

<span data-ttu-id="90c35-104">Итератор не может вернуть значение.</span><span class="sxs-lookup"><span data-stu-id="90c35-104">Cannot return a value from an iterator.</span></span> <span data-ttu-id="90c35-105">Используйте оператор yield return для возвращения значения или yield break для завершения итерации.</span><span class="sxs-lookup"><span data-stu-id="90c35-105">Use the yield return statement to return a value, or yield break to end the iteration.</span></span>  
  
 <span data-ttu-id="90c35-106">Итератор — это специальная функция, возвращающая значение посредством оператора yield, а не оператора return.</span><span class="sxs-lookup"><span data-stu-id="90c35-106">An iterator is a special function that returns a value via the yield statement rather than the return statement.</span></span> <span data-ttu-id="90c35-107">Дополнительные сведения см. в разделе **итераторы**.</span><span class="sxs-lookup"><span data-stu-id="90c35-107">For more information, see **iterators**.</span></span>  
  
 <span data-ttu-id="90c35-108">Следующий пример приводит к возникновению ошибки CS1622:</span><span class="sxs-lookup"><span data-stu-id="90c35-108">The following sample generates CS1622:</span></span>  
  
```csharp  
// CS1622.cs  
// compile with: /target:library  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      return (IEnumerator) this;  // CS1622  
      yield return this;   // OK  
   }  
}  
```
