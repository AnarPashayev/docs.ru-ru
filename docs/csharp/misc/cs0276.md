<span data-ttu-id="8bec2-103">"свойство или индексатор": модификаторы доступности могут использоваться для функций доступа, только если свойство или индексатор содержит обе функции доступа (get и set)</span><span class="sxs-lookup"><span data-stu-id="8bec2-103">'property/indexer': accessibility modifiers on accessors may only be used if the property or indexer has both a get and a set accessor</span></span>
