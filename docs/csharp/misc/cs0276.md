<span data-ttu-id="3a2a6-104">"свойство или индексатор": модификаторы доступности могут использоваться для функций доступа, только если свойство или индексатор содержит обе функции доступа (get и set)</span><span class="sxs-lookup"><span data-stu-id="3a2a6-104">'property/indexer': accessibility modifiers on accessors may only be used if the property or indexer has both a get and a set accessor</span></span>  
  
 <span data-ttu-id="3a2a6-105">Эта ошибка происходит в том случае, если вы объявляете свойство или индексатор только с одним методом доступа и при этом к этому методу доступа применяется модификатор доступа.</span><span class="sxs-lookup"><span data-stu-id="3a2a6-105">This error occurs when you declare a property or indexer with one accessor only, and use an access modifier on the accessor.</span></span> <span data-ttu-id="3a2a6-106">Для устранения ошибки удалите модификатор доступа или добавьте второй метод доступа.</span><span class="sxs-lookup"><span data-stu-id="3a2a6-106">To resolve, remove the access modifier or add another accessor.</span></span>
