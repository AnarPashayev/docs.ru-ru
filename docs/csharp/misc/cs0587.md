# <a name="compiler-error-cs0587"></a>Ошибка компилятора CS0587

Внутренняя ошибка компилятора: этап "этап"

 Возможно, ошибка возникает из-за сбоя компилятора при попытке проанализировать неправильный синтаксис. Если это не так, попробуйте [поговорить с нами](/visualstudio/ide/feedback-options).
 
> [!NOTE]
> Эта ошибка компилятора больше не используется в Roslyn.
