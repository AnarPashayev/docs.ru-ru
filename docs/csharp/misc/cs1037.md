# <a name="compiler-error-cs1037"></a>Ошибка компилятора CS1037
Требуется перегружаемый оператор.  
  
 При указании комментария с параметром [-doc](../language-reference/compiler-options/doc-compiler-option.md)компилятор обнаружил недопустимую ссылку.
