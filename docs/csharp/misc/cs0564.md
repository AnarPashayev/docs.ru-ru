<span data-ttu-id="67a7a-104">Тип первого операнда переопределенного оператора сдвига должен совпадать с вмещающим типом, а тип второго операнда должен быть int.</span><span class="sxs-lookup"><span data-stu-id="67a7a-104">The first operand of an overloaded shift operator must have the same type as the containing type, and the type of the second operand must be int</span></span>  
  
 <span data-ttu-id="67a7a-105">Предпринята попытка перегрузки оператора сдвига (<\< or >>) с неправильно типизированными операндами.</span><span class="sxs-lookup"><span data-stu-id="67a7a-105">You attempted to overload a shift operator (<\< or >>) with incorrectly typed operands.</span></span> <span data-ttu-id="67a7a-106">Первый операнд должен быть типом, а второй операнд должен иметь тип `int`.</span><span class="sxs-lookup"><span data-stu-id="67a7a-106">The first operand must be the type and the second operand must be of the type `int`.</span></span>  
  
 <span data-ttu-id="67a7a-107">В следующем примере возникает ошибка CS0564:</span><span class="sxs-lookup"><span data-stu-id="67a7a-107">The following sample generates CS0564:</span></span>

```csharp  
{  
   public static int operator << (C c1, C c2) // CS0564

   {  
      return 0;  
   }  
   static void Main()
   {  
   }  
}  
```
