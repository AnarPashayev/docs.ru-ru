# <a name="compiler-error-cs0744"></a><span data-ttu-id="04479-102">Ошибка компилятора CS0744</span><span class="sxs-lookup"><span data-stu-id="04479-102">Compiler Error CS0744</span></span>
<span data-ttu-id="04479-103">Требуется контекстное ключевое слово "equals".</span><span class="sxs-lookup"><span data-stu-id="04479-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="04479-104">Для предложения `join` используется шаблон `join`...`in`...`on`...`equals`, как показано в следующем примере:</span><span class="sxs-lookup"><span data-stu-id="04479-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="04479-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="04479-105">To correct this error</span></span>  
  
1. <span data-ttu-id="04479-106">Добавьте ключевое слово `equals` в предложение `join` .</span><span class="sxs-lookup"><span data-stu-id="04479-106">Add the `equals` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="04479-107">Пример</span><span class="sxs-lookup"><span data-stu-id="04479-107">Example</span></span>  
 <span data-ttu-id="04479-108">Следующий код приводит к возникновению ошибки CS0744:</span><span class="sxs-lookup"><span data-stu-id="04479-108">The following code generates CS0744:</span></span>  
  
```csharp  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```
