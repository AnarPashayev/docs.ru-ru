# <a name="compiler-error-cs0722"></a><span data-ttu-id="5641e-103">Ошибка компилятора CS0722</span><span class="sxs-lookup"><span data-stu-id="5641e-103">Compiler Error CS0722</span></span>

<span data-ttu-id="5641e-104">"тип": статические типы не могут использоваться как типы возвращаемого значения</span><span class="sxs-lookup"><span data-stu-id="5641e-104">'type': static types cannot be used as return types</span></span>  
  
 <span data-ttu-id="5641e-105">Использовать статический тип в качестве типа возвращаемого значения не имеет смысла, так как экземпляры статических типов создавать нельзя.</span><span class="sxs-lookup"><span data-stu-id="5641e-105">A static type as a return type is not meaningful since instances of static types cannot be created.</span></span>  
  
 <span data-ttu-id="5641e-106">Следующий пример приводит к возникновению ошибки CS0722:</span><span class="sxs-lookup"><span data-stu-id="5641e-106">The following sample generates CS0722:</span></span>  
  
```csharp  
// CS0722.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public SC F()  // CS0722  
   {  
      return null;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
