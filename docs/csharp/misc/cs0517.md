# <a name="compiler-error-cs0517"></a>Ошибка компилятора CS0517
"класс" не имеет базового класса и не может вызвать конструктор базового класса  
  
 CS0517 может возникать только в том случае, если среда выполнения .NET компилирует исходный код для класса Object, который является единственным классом, который не имеет базового класса.
