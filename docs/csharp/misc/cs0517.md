# <a name="compiler-error-cs0517"></a>Ошибка компилятора CS0517
"класс" не имеет базового класса и не может вызвать конструктор базового класса  
  
 Ошибка CS0517 происходит только в том случае, если среда CLR платформы .NET Framework компилирует исходный код для класса объектов, который является единственным классом, не имеющим базового класса.
