# <a name="compiler-error-cs0081"></a><span data-ttu-id="c3294-103">Ошибка компилятора CS0081</span><span class="sxs-lookup"><span data-stu-id="c3294-103">Compiler Error CS0081</span></span>
<span data-ttu-id="c3294-104">Объявление параметра-типа должно быть идентификатором, а не типом.</span><span class="sxs-lookup"><span data-stu-id="c3294-104">Type parameter declaration must be an identifier not a type</span></span>  
  
 <span data-ttu-id="c3294-105">При объявлении универсального метода или типа следует указать параметр типа как идентификатор, например Т или inputType.</span><span class="sxs-lookup"><span data-stu-id="c3294-105">When you declare a generic method or type, specify the type parameter as an identifier, for example "T" or "inputType".</span></span> <span data-ttu-id="c3294-106">При вызове метода клиентский код предоставляет тип, который заменяет каждое вхождение идентификатора в теле метода или класса.</span><span class="sxs-lookup"><span data-stu-id="c3294-106">When client code calls the method, it supplies the type, which replaces each occurrence of the identifier in the method or class body.</span></span> <span data-ttu-id="c3294-107">Дополнительные сведения см. в разделе [параметры универсального типа](../programming-guide/generics/generic-type-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="c3294-107">For more information, see [Generic Type Parameters](../programming-guide/generics/generic-type-parameters.md).</span></span>  
  
```csharp  
// CS0081.cs  
class MyClass  
{  
   public void F<int>() {}   // CS0081  
   public void F<T>(T input) {}   // OK  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
      a.F<int>(2);  
      a.F<double>(.05);  
   }  
}  
```
