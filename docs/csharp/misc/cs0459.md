# <a name="compiler-error-cs0459"></a><span data-ttu-id="63cdb-103">Ошибка компилятора CS0459</span><span class="sxs-lookup"><span data-stu-id="63cdb-103">Compiler Error CS0459</span></span>

<span data-ttu-id="63cdb-104">Невозможно получить адрес локальной переменной, доступной только для чтения</span><span class="sxs-lookup"><span data-stu-id="63cdb-104">Cannot take the address of a read-only local variable</span></span>

 <span data-ttu-id="63cdb-105">В языке C# существует три типичных сценария, формирующие локальные переменные только для чтения: `foreach`, `using`и `fixed`.</span><span class="sxs-lookup"><span data-stu-id="63cdb-105">There are three common scenarios in the C# language that generate read-only local variables: `foreach`, `using`, and `fixed`.</span></span> <span data-ttu-id="63cdb-106">В каждом из этих случаев нельзя выполнять запись в локальную переменную только для чтения или получать ее адрес.</span><span class="sxs-lookup"><span data-stu-id="63cdb-106">In each of these cases, you are not allowed to write to the read-only local variable, or to take its address.</span></span> <span data-ttu-id="63cdb-107">Эта ошибка возникает, когда компилятор определяет, что вы пытаетесь получить адрес локальной переменной, доступной только для чтения.</span><span class="sxs-lookup"><span data-stu-id="63cdb-107">This error is generated when the compiler realizes you are trying to take the address of a read-only local variable.</span></span>

## <a name="example"></a><span data-ttu-id="63cdb-108">Пример</span><span class="sxs-lookup"><span data-stu-id="63cdb-108">Example</span></span>

 <span data-ttu-id="63cdb-109">В следующем примере создается CS0459 при попытке получить адрес локальной переменной только для чтения в `foreach` цикле и в `fixed` блоке инструкций:</span><span class="sxs-lookup"><span data-stu-id="63cdb-109">The following example generates CS0459 when an attempt is made to take the address of a read-only local variable in a `foreach` loop and in a `fixed` statement block:</span></span>

```csharp
// CS0459.cs
// compile with: /unsafe

class Program
{
    public unsafe void M1()
    {
        int[] ints = new int[] { 1, 2, 3 };
        foreach (int i in ints)
        {
            int *j = &i;  // CS0459
        }

        fixed (int *i = &_i)
        {
            int **j = &i;  // CS0459
        }
    }

    private int _i = 0;
}
```
