# <a name="compiler-error-cs0633"></a><span data-ttu-id="f2574-103">Ошибка компилятора CS0633</span><span class="sxs-lookup"><span data-stu-id="f2574-103">Compiler Error CS0633</span></span>

<span data-ttu-id="f2574-104">Аргумент атрибута "атрибут" должен быть допустимым идентификатором</span><span class="sxs-lookup"><span data-stu-id="f2574-104">The argument to the 'attribute' attribute must be a valid identifier</span></span>  
  
 <span data-ttu-id="f2574-105">Любой аргумент, передаваемый атрибутам <xref:System.Diagnostics.ConditionalAttribute> и <xref:System.Runtime.CompilerServices.IndexerNameAttribute> , должен представлять собой допустимый идентификатор.</span><span class="sxs-lookup"><span data-stu-id="f2574-105">Any argument that you pass to the <xref:System.Diagnostics.ConditionalAttribute> or <xref:System.Runtime.CompilerServices.IndexerNameAttribute> attributes must be a valid identifier.</span></span> <span data-ttu-id="f2574-106">Это означает, что в нем не должны содержаться символы, использование которых не допускается в идентификаторах, например "+".</span><span class="sxs-lookup"><span data-stu-id="f2574-106">This means that it may not contain characters such as "+" that are illegal when they occur in identifiers.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f2574-107">Пример</span><span class="sxs-lookup"><span data-stu-id="f2574-107">Example</span></span>  

 <span data-ttu-id="f2574-108">В этом примере демонстрируется возникновение ошибки CS0633 при использовании атрибута <xref:System.Diagnostics.ConditionalAttribute>.</span><span class="sxs-lookup"><span data-stu-id="f2574-108">This example illustrates CS0633 using the <xref:System.Diagnostics.ConditionalAttribute>.</span></span> <span data-ttu-id="f2574-109">В примере происходит ошибка CS0633.</span><span class="sxs-lookup"><span data-stu-id="f2574-109">The following sample generates CS0633.</span></span>  
  
```csharp  
// CS0633a.cs  
#define DEBUG  
using System.Diagnostics;  
public class Test  
{  
   [Conditional("DEB+UG")]   // CS0633  
   // try the following line instead  
   // [Conditional("DEBUG")]  
   public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="f2574-110">Пример</span><span class="sxs-lookup"><span data-stu-id="f2574-110">Example</span></span>  

 <span data-ttu-id="f2574-111">В этом примере демонстрируется возникновение ошибки CS0633 при использовании атрибута <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span><span class="sxs-lookup"><span data-stu-id="f2574-111">This example illustrates CS0633 using the <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span></span>  
  
```csharp  
// CS0633b.cs  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]
   {
      get { return i; }
   }  
}  
```
