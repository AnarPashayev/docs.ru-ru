# <a name="compiler-error-cs0835"></a><span data-ttu-id="422d8-102">Ошибка компилятора CS0835</span><span class="sxs-lookup"><span data-stu-id="422d8-102">Compiler Error CS0835</span></span>
<span data-ttu-id="422d8-103">Невозможно преобразовать лямбда-выражение в дерево выражения, чей аргумент типа "тип" не является типом делегата.</span><span class="sxs-lookup"><span data-stu-id="422d8-103">Cannot convert lambda to an expression tree whose type argument 'type' is not a delegate type.</span></span>  
  
 <span data-ttu-id="422d8-104">Если лямбда-выражение преобразуется в дерево выражения, это дерево выражения должно иметь тип делегата для своего аргумента.</span><span class="sxs-lookup"><span data-stu-id="422d8-104">If a lambda expression is converted to an expression tree, the expression tree must have a delegate type for its argument.</span></span> <span data-ttu-id="422d8-105">Кроме того, лямбда-выражение должно быть преобразуемым в тип делегата.</span><span class="sxs-lookup"><span data-stu-id="422d8-105">Furthermore, the lambda expression must be convertible to the delegate type.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="422d8-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="422d8-106">To correct this error</span></span>  
  
1. <span data-ttu-id="422d8-107">Измените параметр типа с `int` на тип делегата, например `Func<int,int>`.</span><span class="sxs-lookup"><span data-stu-id="422d8-107">Change the type parameter from `int` to a delegate type, for example `Func<int,int>`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="422d8-108">Пример</span><span class="sxs-lookup"><span data-stu-id="422d8-108">Example</span></span>  
 <span data-ttu-id="422d8-109">В следующем примере возникает ошибка CS0835:</span><span class="sxs-lookup"><span data-stu-id="422d8-109">The following example generates CS0835:</span></span>  
  
```csharp  
// cs0835.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.  
       // Expression<Func<int,int>> e2 = x => x + 1;  
  
        return 1;  
    }  
}  
```
