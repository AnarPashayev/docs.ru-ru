# <a name="compiler-error-cs1577"></a>Ошибка компилятора CS1577
Сбой при создании сборки — причина  
  
 Этап компиляции, на котором происходит создание сборки, завершился ошибкой после того, как компилятор C# успешно скомпилировал исходный код. Для получения дополнительных сведений обратитесь к документации по ошибкам служебной программы ALink ([Ошибки и предупреждения средства Al.exe](../../framework/tools/al-exe-assembly-linker.md#errors-and-warnings)).
