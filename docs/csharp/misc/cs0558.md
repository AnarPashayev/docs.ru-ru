<span data-ttu-id="6d6f6-104">Определенный пользователем оператор "оператор" должен быть объявлен как статический и общий</span><span class="sxs-lookup"><span data-stu-id="6d6f6-104">User-defined operator 'operator' must be declared static and public</span></span>  
  
 <span data-ttu-id="6d6f6-105">В пользовательских операторах должны быть указаны оба **модификатора** доступа, **static** и [public](/dotnet/csharp/language-reference/keywords) .</span><span class="sxs-lookup"><span data-stu-id="6d6f6-105">Both the **static** and **public** access [modifiers](/dotnet/csharp/language-reference/keywords) must be specified on user-defined operators.</span></span>

```csharp  
// CS0558.cs

{  
   public class ii  
   {  
      public class iii  
      {  
         static implicit operator int(iii aa)   // CS0558, add public  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
