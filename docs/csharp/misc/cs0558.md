# <a name="compiler-error-cs0558"></a><span data-ttu-id="52e11-102">Ошибка компилятора CS0558</span><span class="sxs-lookup"><span data-stu-id="52e11-102">Compiler Error CS0558</span></span>

 <span data-ttu-id="52e11-104">В пользовательских операторах должны быть указаны оба **модификатора** доступа, **static** и [public](/dotnet/csharp/language-reference/keywords) .</span><span class="sxs-lookup"><span data-stu-id="52e11-104">Both the **static** and **public** access [modifiers](/dotnet/csharp/language-reference/keywords) must be specified on user-defined operators.</span></span>

```csharp  
// CS0558.cs  
namespace x  
{  
   public class ii  
   {  
      public class iii  
      {  
         static implicit operator int(iii aa)   // CS0558, add public  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
