# <a name="compiler-error-cs0216"></a><span data-ttu-id="944c1-103">Ошибка компилятора CS0216</span><span class="sxs-lookup"><span data-stu-id="944c1-103">Compiler Error CS0216</span></span>

<span data-ttu-id="944c1-104">Оператор "оператор" требует, чтобы был также определен соответствующий оператор "пропущенный_оператор"</span><span class="sxs-lookup"><span data-stu-id="944c1-104">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="944c1-105">Определяемый пользователем [==](../language-reference/operators/equality-operators.md#equality-operator-) оператор требует оператора, определяемого пользователем [! =](../language-reference/operators/equality-operators.md#inequality-operator-) , и наоборот.</span><span class="sxs-lookup"><span data-stu-id="944c1-105">A user-defined [==](../language-reference/operators/equality-operators.md#equality-operator-) operator requires a user-defined [!=](../language-reference/operators/equality-operators.md#inequality-operator-) operator, and vice versa.</span></span>  
 <span data-ttu-id="944c1-106">То же относится и к определяемому пользователем оператору [true](../language-reference/operators/true-false-operators.md) и к определенному пользователем оператору [false](../language-reference/operators/true-false-operators.md) .</span><span class="sxs-lookup"><span data-stu-id="944c1-106">The same applies also to a user-defined [true](../language-reference/operators/true-false-operators.md) operator and a user-defined [false](../language-reference/operators/true-false-operators.md) operator.</span></span>  
  
 <span data-ttu-id="944c1-107">Следующий пример приводит к возникновению ошибки CS0216:</span><span class="sxs-lookup"><span data-stu-id="944c1-107">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator == (MyClass MyIntLeft, MyClass MyIntRight)   // CS0216  
   {  
      return MyIntLeft == MyIntRight;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator != (MyClass MyIntLeft, MyClass MyIntRight)  
   {  
      return MyIntLeft != MyIntRight;  
   }  
   */  
  
   public override bool Equals (object obj)  
   {  
      return base.Equals (obj);  
   }  
  
   public override int GetHashCode()  
   {  
      return base.GetHashCode();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
