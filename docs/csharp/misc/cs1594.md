# <a name="compiler-error-cs1594"></a><span data-ttu-id="d940d-102">Ошибка компилятора CS1594</span><span class="sxs-lookup"><span data-stu-id="d940d-102">Compiler Error CS1594</span></span>
<span data-ttu-id="d940d-103">Несколько аргументов делегата "делегат" являются недопустимыми</span><span class="sxs-lookup"><span data-stu-id="d940d-103">Delegate 'delegate' has some invalid arguments</span></span>  
  
 <span data-ttu-id="d940d-104">Тип аргумента, переданного в вызов [делегата](../../csharp/language-reference/keywords/delegate.md) , не согласуется с типом параметра в объявлении делегата.</span><span class="sxs-lookup"><span data-stu-id="d940d-104">The type of an argument passed to a [delegate](../../csharp/language-reference/keywords/delegate.md) invocation does not agree with the type of the parameter in the delegate declaration.</span></span>  
  
 <span data-ttu-id="d940d-105">В следующем примере возникает ошибка CS1594:</span><span class="sxs-lookup"><span data-stu-id="d940d-105">The following sample generates CS1594:</span></span>  
  
```csharp  
// CS1594.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello("8");   // CS1594  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
