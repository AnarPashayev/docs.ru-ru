# <a name="compiler-error-cs0157"></a><span data-ttu-id="a50cb-102">Ошибка компилятора CS0157</span><span class="sxs-lookup"><span data-stu-id="a50cb-102">Compiler Error CS0157</span></span>
<span data-ttu-id="a50cb-103">Управление не может быть передано из тела предложения finally.</span><span class="sxs-lookup"><span data-stu-id="a50cb-103">Control cannot leave the body of a finally clause</span></span>  
  
 <span data-ttu-id="a50cb-104">В предложении [finally](../language-reference/keywords/try-catch-finally.md) должны быть выполнены все операторы.</span><span class="sxs-lookup"><span data-stu-id="a50cb-104">All of the statements in a [finally](../language-reference/keywords/try-catch-finally.md) clause must execute.</span></span> <span data-ttu-id="a50cb-105">Дополнительные сведения см. в разделе [исключения и обработка исключений](../programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="a50cb-105">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="a50cb-106">Следующий пример приводит к возникновению ошибки CS0157:</span><span class="sxs-lookup"><span data-stu-id="a50cb-106">The following sample generates CS0157:</span></span>  
  
```csharp  
// CS0157.cs  
using System;  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
         }  
  
         finally  
         {  
            return;   // CS0157, cannot leave finally clause  
         }  
      }  
   }  
}  
```
