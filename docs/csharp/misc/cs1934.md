# <a name="compiler-error-cs1934"></a><span data-ttu-id="6abde-102">Ошибка компилятора CS1934</span><span class="sxs-lookup"><span data-stu-id="6abde-102">Compiler Error CS1934</span></span>

 <span data-ttu-id="6abde-106">Эта ошибка возникает в том случае, если выражение запроса задает источник данных, для которого отсутствуют реализованные стандартные операторы запроса.</span><span class="sxs-lookup"><span data-stu-id="6abde-106">This error is produced if a query expression specifies a data source for which no standard query operators are implemented.</span></span> <span data-ttu-id="6abde-107">Например, эта ошибка возникает, если указать `ArrayList` без предоставления явного типа переменной диапазона.</span><span class="sxs-lookup"><span data-stu-id="6abde-107">One way to produce this error is to specify an `ArrayList` without giving an explicit type for the range variable.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="6abde-108">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="6abde-108">To correct this error</span></span>  
  
1. <span data-ttu-id="6abde-109">В следующем примере решением является просто указание типа переменной диапазона:</span><span class="sxs-lookup"><span data-stu-id="6abde-109">In the following example, the solution is to just specify the type of the range variable:</span></span>  
  
    ```csharp  
    var q = from int x in list  
    ```  
  
## <a name="example"></a><span data-ttu-id="6abde-110">Пример</span><span class="sxs-lookup"><span data-stu-id="6abde-110">Example</span></span>  
 <span data-ttu-id="6abde-111">В следующем примере показан один из возможных способов получения ошибки CS1934:</span><span class="sxs-lookup"><span data-stu-id="6abde-111">The following example shows one way to produce CS1934:</span></span>  
  
```csharp  
// cs1934.cs  
using System.Linq;  
using System.Collections;  
static class Test  
{  
    public static void Main()  
    {  
        var list = new ArrayList { 0, 1, 2, 3, 4, 5 };  
        var q = from x in list // CS1934  
                select x + 1;  
    }  
}  
```
