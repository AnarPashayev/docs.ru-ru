# <a name="compiler-error-cs0726"></a>Ошибка компилятора CS0726
"спецификатор формата" не является допустимым спецификатором формата  
  
 Эта ошибка возникает в отладчике. При вводе имени переменной в окне отладчика можно через запятую указать спецификатор формата. Например, можно указать `myInt, h` или `myString,nq`. Эта ошибка возникает, когда компилятор не распознает [описатели формата в C# ](/visualstudio/debugger/format-specifiers-in-csharp).
