<span data-ttu-id="8de06-104">Невозможно объявить локальную переменную с именем var в этой области видимости, т. к. она придаст другое значение var, которое уже используется в области видимости "родительская или текущая/дочерняя" для обозначения чего-то другого</span><span class="sxs-lookup"><span data-stu-id="8de06-104">A local variable named 'var' cannot be declared in this scope because it would give a different meaning to 'var', which is already used in a 'parent or current/child' scope to denote something else</span></span>  
  
 <span data-ttu-id="8de06-105">Объявление переменной скрывает другое объявление, которое в противном случае присутствовало бы в области.</span><span class="sxs-lookup"><span data-stu-id="8de06-105">A variable declaration hides another declaration that would otherwise be in scope.</span></span> <span data-ttu-id="8de06-106">Переименуйте переменную, объявленную в строке, которая создала ошибку CS0136.</span><span class="sxs-lookup"><span data-stu-id="8de06-106">Rename the variable that is declared on the line that generated CS0136.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8de06-107">Пример</span><span class="sxs-lookup"><span data-stu-id="8de06-107">Example</span></span>  
 <span data-ttu-id="8de06-108">Следующий пример приводит к возникновению ошибки CS0136:</span><span class="sxs-lookup"><span data-stu-id="8de06-108">The following sample generates CS0136:</span></span>  
  
```csharp
// CS0136.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         {  
            char i = 'a';   // CS0136, hides int i  
         }  
         i++;  
      }  
   }  
}  
```
