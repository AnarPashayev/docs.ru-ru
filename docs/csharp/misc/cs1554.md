# <a name="compiler-error-cs1554"></a><span data-ttu-id="650ae-103">Ошибка компилятора CS1554</span><span class="sxs-lookup"><span data-stu-id="650ae-103">Compiler Error CS1554</span></span>
<span data-ttu-id="650ae-104">Недопустимое объявление; использовать \<type> оператор "operator op (..." Используйте</span><span class="sxs-lookup"><span data-stu-id="650ae-104">Declaration is not valid; use '\<type> operator op (...' instead</span></span>  
  
<span data-ttu-id="650ae-105">Тип возвращаемого значения [перегруженного оператора](../language-reference/operators/operator-overloading.md) должен предшествовать `operator` ключевому слову.</span><span class="sxs-lookup"><span data-stu-id="650ae-105">The return type of an [overloaded operator](../language-reference/operators/operator-overloading.md) must appear before the `operator` keyword.</span></span>
  
<span data-ttu-id="650ae-106">Следующий пример приводит к возникновению ошибки CS1554:</span><span class="sxs-lookup"><span data-stu-id="650ae-106">The following sample generates CS1554:</span></span>  
  
```csharp  
// CS1554.cs  
class MyClass  
{  
   public static operator ++ MyClass (MyClass f)    // CS1554  
   // try the following line instead  
   // public static MyClass operator ++ (MyClass f)  
   {  
      return new MyClass ();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
