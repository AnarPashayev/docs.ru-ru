# <a name="compiler-error-cs0637"></a><span data-ttu-id="208ed-103">Ошибка компилятора CS0637</span><span class="sxs-lookup"><span data-stu-id="208ed-103">Compiler Error CS0637</span></span>

<span data-ttu-id="208ed-104">Атрибут FieldOffset не разрешен для статических или константных полей.</span><span class="sxs-lookup"><span data-stu-id="208ed-104">The FieldOffset attribute is not allowed on static or const fields.</span></span>
  
 <span data-ttu-id="208ed-105">Атрибут [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) нельзя использовать в полях, помеченных как [static](../language-reference/keywords/static.md) или [const](../language-reference/keywords/const.md).</span><span class="sxs-lookup"><span data-stu-id="208ed-105">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../language-reference/keywords/static.md) or [const](../language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="208ed-106">При компиляции следующего примера возникнет ошибка CS0637:</span><span class="sxs-lookup"><span data-stu-id="208ed-106">The following sample generates CS0637:</span></span>  
  
```csharp  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
