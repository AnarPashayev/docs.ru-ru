# <a name="compiler-error-cs0637"></a><span data-ttu-id="26b3a-102">Ошибка компилятора CS0637</span><span class="sxs-lookup"><span data-stu-id="26b3a-102">Compiler Error CS0637</span></span>

<span data-ttu-id="26b3a-103">Для полей static и const атрибут FieldOffset запрещено.</span><span class="sxs-lookup"><span data-stu-id="26b3a-103">The FieldOffset attribute is not allowed on static or const fields.</span></span>
  
 <span data-ttu-id="26b3a-104">Атрибут [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) нельзя использовать в полях, помеченных как [static](../../csharp/language-reference/keywords/static.md) или [const](../../csharp/language-reference/keywords/const.md).</span><span class="sxs-lookup"><span data-stu-id="26b3a-104">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../../csharp/language-reference/keywords/static.md) or [const](../../csharp/language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="26b3a-105">При компиляции следующего примера возникнет ошибка CS0637:</span><span class="sxs-lookup"><span data-stu-id="26b3a-105">The following sample generates CS0637:</span></span>  
  
```csharp  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
