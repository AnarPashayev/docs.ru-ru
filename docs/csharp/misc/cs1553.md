<span data-ttu-id="c7530-105">Возвращаемый тип [оператора преобразования](../language-reference/operators/user-defined-conversion-operators.md) должен находиться непосредственно перед списком параметров, а *Модификатор* — либо `implicit` `explicit` .</span><span class="sxs-lookup"><span data-stu-id="c7530-105">The return type for a [conversion operator](../language-reference/operators/user-defined-conversion-operators.md) must immediately precede the parameter list, and *modifier* is either `implicit` or `explicit`.</span></span>

```csharp  
   }

   }

```
