# <a name="compiler-error-cs0431"></a><span data-ttu-id="584c3-102">Ошибка компилятора CS0431</span><span class="sxs-lookup"><span data-stu-id="584c3-102">Compiler Error CS0431</span></span>
<span data-ttu-id="584c3-103">Использование псевдонима "идентификатор" с "::" невозможно, так как псевдоним ссылается на тип.</span><span class="sxs-lookup"><span data-stu-id="584c3-103">Cannot use alias 'identifier' with '::' since the alias references a type.</span></span> <span data-ttu-id="584c3-104">Вместо этого используйте ".".</span><span class="sxs-lookup"><span data-stu-id="584c3-104">Use '.' instead.</span></span>  
  
 <span data-ttu-id="584c3-105">Вы использовали "::" с псевдонимом, который ссылается на тип.</span><span class="sxs-lookup"><span data-stu-id="584c3-105">You used "::" with an alias that references a type.</span></span> <span data-ttu-id="584c3-106">Чтоб устранить эту ошибку, используйте оператор ".".</span><span class="sxs-lookup"><span data-stu-id="584c3-106">To resolve this error, use the "." operator.</span></span>  
  
 <span data-ttu-id="584c3-107">При компиляции следующего примера возникнет ошибка CS0431:</span><span class="sxs-lookup"><span data-stu-id="584c3-107">The following sample generates CS0431:</span></span>  
  
```csharp  
// CS0431.cs  
using A = Outer;  
  
public class Outer   
{  
   public class Inner   
   {  
      public static void Meth() {}  
   }  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      A::Inner.Meth();   // CS0431  
      A.Inner.Meth();   // OK  
   }  
}  
```
