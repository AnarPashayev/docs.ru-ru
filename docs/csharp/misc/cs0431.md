# <a name="compiler-error-cs0431"></a><span data-ttu-id="5b261-103">Ошибка компилятора CS0431</span><span class="sxs-lookup"><span data-stu-id="5b261-103">Compiler Error CS0431</span></span>

<span data-ttu-id="5b261-104">Использование псевдонима "идентификатор" с "::" невозможно, так как псевдоним ссылается на тип.</span><span class="sxs-lookup"><span data-stu-id="5b261-104">Cannot use alias 'identifier' with '::' since the alias references a type.</span></span> <span data-ttu-id="5b261-105">Вместо этого используйте ".".</span><span class="sxs-lookup"><span data-stu-id="5b261-105">Use '.' instead.</span></span>  
  
 <span data-ttu-id="5b261-106">Вы использовали "::" с псевдонимом, который ссылается на тип.</span><span class="sxs-lookup"><span data-stu-id="5b261-106">You used "::" with an alias that references a type.</span></span> <span data-ttu-id="5b261-107">Чтоб устранить эту ошибку, используйте оператор ".".</span><span class="sxs-lookup"><span data-stu-id="5b261-107">To resolve this error, use the "." operator.</span></span>  
  
 <span data-ttu-id="5b261-108">При компиляции следующего примера возникнет ошибка CS0431:</span><span class="sxs-lookup"><span data-stu-id="5b261-108">The following sample generates CS0431:</span></span>  
  
```csharp  
// CS0431.cs  
using A = Outer;  
  
public class Outer
{  
   public class Inner
   {  
      public static void Meth() {}  
   }  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      A::Inner.Meth();   // CS0431  
      A.Inner.Meth();   // OK  
   }  
}  
```
