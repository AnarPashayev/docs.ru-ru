# <a name="compiler-error-cs1529"></a><span data-ttu-id="41784-102">Ошибка компилятора CS1529</span><span class="sxs-lookup"><span data-stu-id="41784-102">Compiler Error CS1529</span></span>
<span data-ttu-id="41784-103">Предложение using должно предшествовать любым другим элементам пространства имен кроме объявлений внешних псевдонимов</span><span class="sxs-lookup"><span data-stu-id="41784-103">A using clause must precede all other elements defined in the namespace except extern alias declarations</span></span>  
  
 <span data-ttu-id="41784-104">Предложение [using](../language-reference/keywords/using.md) должно быть первым в пространстве имен.</span><span class="sxs-lookup"><span data-stu-id="41784-104">A [using](../language-reference/keywords/using.md) clause must appear first in a namespace.</span></span>  
  
## <a name="example"></a><span data-ttu-id="41784-105">Пример</span><span class="sxs-lookup"><span data-stu-id="41784-105">Example</span></span>  
 <span data-ttu-id="41784-106">При компиляции следующего примера возникнет ошибка CS1529:</span><span class="sxs-lookup"><span data-stu-id="41784-106">The following sample generates CS1529:</span></span>  
  
```csharp  
// CS1529.cs  
namespace X  
{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition  
    }  
  
    using System.Reflection;  // CS1529, place before namespace 'Subspace'  
}  
  
using System;                 // CS1529, place at the beginning of the file  
```
