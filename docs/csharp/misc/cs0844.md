## <a name="example"></a><span data-ttu-id="c4318-111">Пример</span><span class="sxs-lookup"><span data-stu-id="c4318-111">Example</span></span>  
 <span data-ttu-id="c4318-112">Следующий код приводит к возникновению ошибки CS0844:</span><span class="sxs-lookup"><span data-stu-id="c4318-112">The following code generates CS0844:</span></span>  
  
```csharp  
class Test  
    {  
        int num;  
        public void TestMethod()  
        {  
            num = 5; // CS0844  
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

## <a name="correct-the-error-by-using-thisnum-to-refer-to-the-class-field"></a><span data-ttu-id="c4318-113">Исправьте ошибку с помощью `this.num` для ссылки на поле класса</span><span class="sxs-lookup"><span data-stu-id="c4318-113">Correct the error by using `this.num` to refer to the class field</span></span>

```csharp  
class Test  
    {  
        int num;  
        public void TestMethod()  
        {  
            this.num = 5; // Error fixed.
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

```csharp  
class Test  
    {  
        int num;  
        public void TestMethod()  
        {  
            num = 5; // Error fixed.
            int num2 = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
