# <a name="compiler-error-cs1958"></a><span data-ttu-id="3aae7-102">Ошибка компилятора CS1958</span><span class="sxs-lookup"><span data-stu-id="3aae7-102">Compiler Error CS1958</span></span>

<span data-ttu-id="3aae7-103">Выражения инициализатора объекта и коллекции не могут применяться к выражению создания делегата</span><span class="sxs-lookup"><span data-stu-id="3aae7-103">Object and collection initializer expressions may not be applied to a delegate creation expression,</span></span>  
  
 <span data-ttu-id="3aae7-104">В отличие от класса или структуры, делегат не имеет членов, поэтому инициализатору объекта нечего инициализировать.</span><span class="sxs-lookup"><span data-stu-id="3aae7-104">A delegate has no members like a class or struct has, and so there is nothing for an object initializer to initialize.</span></span> <span data-ttu-id="3aae7-105">Если эта ошибка возникает, то возможно она вызвана фигурными скобками после выражения создания делегата.</span><span class="sxs-lookup"><span data-stu-id="3aae7-105">If you encounter this error, it is probably because there are braces after the delegate creation expression.</span></span> <span data-ttu-id="3aae7-106">Просто удалите эти фигурные скобки, и ошибка исчезнет.</span><span class="sxs-lookup"><span data-stu-id="3aae7-106">Just remove the braces and this error will disappear.</span></span>

## <a name="example"></a><span data-ttu-id="3aae7-109">Пример</span><span class="sxs-lookup"><span data-stu-id="3aae7-109">Example</span></span>  

 <span data-ttu-id="3aae7-110">В следующем коде возникает ошибка CS1958.</span><span class="sxs-lookup"><span data-stu-id="3aae7-110">The following code produces CS1958:</span></span>  
  
```csharp  
// cs1958.cs  
public class MemberInitializerTest  
{
    delegate void D<T>();  
    public static void GenericMethod<T>() { }  
    public static void Run()  
    {  
        D<int> genD = new D<int>(GenericMethod<int>) { }; // CS1958  
       // Try the following line instead  
      // D<int> genD = new D<int>(GenericMethod<int>);  
    }  
}  
```
