 <span data-ttu-id="62d30-104">В отличие от класса или структуры, делегат не имеет членов, поэтому инициализатору объекта нечего инициализировать.</span><span class="sxs-lookup"><span data-stu-id="62d30-104">A delegate has no members like a class or struct has, and so there is nothing for an object initializer to initialize.</span></span> <span data-ttu-id="62d30-105">Если эта ошибка возникает, то возможно она вызвана фигурными скобками после выражения создания делегата.</span><span class="sxs-lookup"><span data-stu-id="62d30-105">If you encounter this error, it is probably because there are braces after the delegate creation expression.</span></span> <span data-ttu-id="62d30-106">Просто удалите эти фигурные скобки, и ошибка исчезнет.</span><span class="sxs-lookup"><span data-stu-id="62d30-106">Just remove the braces and this error will disappear.</span></span>

## <a name="example"></a><span data-ttu-id="62d30-109">Пример</span><span class="sxs-lookup"><span data-stu-id="62d30-109">Example</span></span>

    delegate void D<T>();  
    public static void GenericMethod<T>() { }

    }  
}
