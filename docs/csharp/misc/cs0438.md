# <a name="compiler-error-cs0438"></a><span data-ttu-id="0c56b-102">Ошибка компилятора CS0438</span><span class="sxs-lookup"><span data-stu-id="0c56b-102">Compiler Error CS0438</span></span>
<span data-ttu-id="0c56b-103">Тип "тип" в модуле "модуль_1" конфликтует с пространством имен "пространство_имен" в "модуль_2".</span><span class="sxs-lookup"><span data-stu-id="0c56b-103">The type 'type' in 'module_1' conflicts with the namespace 'namespace' in 'module_2'.</span></span>  
  
 <span data-ttu-id="0c56b-104">Эта ошибка происходит, когда тип в файле исходного кода конфликтует с пространством имен в другом исходном файле.</span><span class="sxs-lookup"><span data-stu-id="0c56b-104">This error occurs when a type in a source file conflicts with a namespace in another source file.</span></span> <span data-ttu-id="0c56b-105">Обычно это происходит, когда один или оба типа происходят из добавленного модуля.</span><span class="sxs-lookup"><span data-stu-id="0c56b-105">This typically happens when one or both come from an added module.</span></span> <span data-ttu-id="0c56b-106">Чтобы устранить эту ошибку, переименуйте тип или пространство имен, вызвавшее конфликт.</span><span class="sxs-lookup"><span data-stu-id="0c56b-106">To resolve, rename the type or the namespace that caused the conflict.</span></span>  
  
 <span data-ttu-id="0c56b-107">В следующем примере возникает ошибка CS0438:</span><span class="sxs-lookup"><span data-stu-id="0c56b-107">The following example generates CS0438:</span></span>

```csharp  
   public class A { }

```  
  
 <span data-ttu-id="0c56b-109">Затем скомпилируйте этот файл:</span><span class="sxs-lookup"><span data-stu-id="0c56b-109">Then compile this file:</span></span>  
  
```csharp

// compile with: /target:module

{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="0c56b-110">И наконец скомпилируйте этот файл:</span><span class="sxs-lookup"><span data-stu-id="0c56b-110">And then compile this file:</span></span>  
  
```csharp

{  
   public static void Main() {  
      Console.WriteLine(typeof(Util.A));   // CS0438  
   }  
}  
```
