# <a name="compiler-error-cs0438"></a><span data-ttu-id="66e7b-103">Ошибка компилятора CS0438</span><span class="sxs-lookup"><span data-stu-id="66e7b-103">Compiler Error CS0438</span></span>
<span data-ttu-id="66e7b-104">Тип "тип" в модуле "модуль_1" конфликтует с пространством имен "пространство_имен" в "модуль_2".</span><span class="sxs-lookup"><span data-stu-id="66e7b-104">The type 'type' in 'module_1' conflicts with the namespace 'namespace' in 'module_2'.</span></span>  
  
 <span data-ttu-id="66e7b-105">Эта ошибка происходит, когда тип в файле исходного кода конфликтует с пространством имен в другом исходном файле.</span><span class="sxs-lookup"><span data-stu-id="66e7b-105">This error occurs when a type in a source file conflicts with a namespace in another source file.</span></span> <span data-ttu-id="66e7b-106">Обычно это происходит, когда один или оба типа происходят из добавленного модуля.</span><span class="sxs-lookup"><span data-stu-id="66e7b-106">This typically happens when one or both come from an added module.</span></span> <span data-ttu-id="66e7b-107">Чтобы устранить эту ошибку, переименуйте тип или пространство имен, вызвавшее конфликт.</span><span class="sxs-lookup"><span data-stu-id="66e7b-107">To resolve, rename the type or the namespace that caused the conflict.</span></span>  
  
 <span data-ttu-id="66e7b-108">В следующем примере возникает ошибка CS0438:</span><span class="sxs-lookup"><span data-stu-id="66e7b-108">The following example generates CS0438:</span></span>  
  
 <span data-ttu-id="66e7b-109">Сначала скомпилируйте этот файл:</span><span class="sxs-lookup"><span data-stu-id="66e7b-109">Compile this file first:</span></span>  
  
```csharp

{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="66e7b-110">Затем скомпилируйте этот файл:</span><span class="sxs-lookup"><span data-stu-id="66e7b-110">Then compile this file:</span></span>  
  
```csharp  
// CS0438_2.cs  
// compile with: /target:module  
namespace Util
{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="66e7b-111">И наконец скомпилируйте этот файл:</span><span class="sxs-lookup"><span data-stu-id="66e7b-111">And then compile this file:</span></span>  
  
```csharp

using System;  
public class Test  
{  
   public static void Main() {  
      Console.WriteLine(typeof(Util.A));   // CS0438  
   }  
}  
```
