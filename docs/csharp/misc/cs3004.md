# <a name="compiler-warning-level-1-cs3004"></a>Предупреждение компилятора (уровень 1) CS3004
Смешанные и разложенные знаки Юникода не соответствуют CLS  
  
 В соответствии со спецификацией CLS в идентификаторах [public](../language-reference/keywords/public.md), [protected](../language-reference/keywords/protected.md)или `protected internal` допускаются только составные символы ЮНИКОДА. Дополнительные сведения о CLS-совместимости см. в разделе [независимость от языка и независимые от языка компоненты](../../standard/language-independence.md).
