# <a name="compiler-error-cs0502"></a><span data-ttu-id="34117-102">Ошибка компилятора CS0502</span><span class="sxs-lookup"><span data-stu-id="34117-102">Compiler Error CS0502</span></span>
<span data-ttu-id="34117-103">"член" не может одновременно быть абстрактным и запечатанным</span><span class="sxs-lookup"><span data-stu-id="34117-103">'member' cannot be both abstract and sealed</span></span>  
  
 <span data-ttu-id="34117-104">Член класса не может одновременно быть [abstract](../language-reference/keywords/abstract.md) и [sealed](../language-reference/keywords/sealed.md).</span><span class="sxs-lookup"><span data-stu-id="34117-104">A class member cannot be both [abstract](../language-reference/keywords/abstract.md) and [sealed](../language-reference/keywords/sealed.md).</span></span>  
  
 <span data-ttu-id="34117-105">Следующий пример приводит к возникновению ошибки CS0502:</span><span class="sxs-lookup"><span data-stu-id="34117-105">The following sample generates CS0502:</span></span>  
  
```csharp  
// CS0502.cs  
public class B  
{  
   abstract public void F();  
}  
  
public class C : B  
{  
   abstract sealed override public void F()   // CS0502  
   {  
   }  
}  
  
public class CMain  
{  
    public static void Main()  
    { }  
}  
```
