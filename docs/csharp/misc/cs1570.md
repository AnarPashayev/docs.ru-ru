 <span data-ttu-id="8eb07-104">Если используется параметр [/doc](../../csharp/language-reference/compiler-options/doc-compiler-option.md), все комментарии в исходном коде должны быть в формате XML.</span><span class="sxs-lookup"><span data-stu-id="8eb07-104">When using [/doc](../../csharp/language-reference/compiler-options/doc-compiler-option.md), any comments in the source code must be in XML.</span></span> <span data-ttu-id="8eb07-105">Любая ошибка разметки XML вызывает предупреждение CS1570.</span><span class="sxs-lookup"><span data-stu-id="8eb07-105">Any error with your XML markup will generate CS1570.</span></span> <span data-ttu-id="8eb07-106">Пример:</span><span class="sxs-lookup"><span data-stu-id="8eb07-106">For example:</span></span>
