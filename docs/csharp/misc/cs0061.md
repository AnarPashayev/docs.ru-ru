# <a name="compiler-error-cs0061"></a><span data-ttu-id="5aad8-102">Ошибка компилятора CS0061</span><span class="sxs-lookup"><span data-stu-id="5aad8-102">Compiler Error CS0061</span></span>
<span data-ttu-id="5aad8-103">Несогласованность по доступности: доступность базового интерфейса "интерфейс_1" ниже доступности интерфейса "интерфейс_2"</span><span class="sxs-lookup"><span data-stu-id="5aad8-103">Inconsistent accessibility: base interface 'interface 1' is less accessible than interface 'interface 2'</span></span>  
  
 <span data-ttu-id="5aad8-104">[Открытая](../../csharp/language-reference/keywords/public.md) конструкция должна возвращать общедоступный объект.</span><span class="sxs-lookup"><span data-stu-id="5aad8-104">A [public](../../csharp/language-reference/keywords/public.md) construct must return a publicly accessible object.</span></span>  
  
 <span data-ttu-id="5aad8-105">Доступность интерфейса не может быть снижена в производном интерфейсе.</span><span class="sxs-lookup"><span data-stu-id="5aad8-105">Interface accessibility cannot be narrowed in a derived interface.</span></span> <span data-ttu-id="5aad8-106">Дополнительные сведения см. в разделе [интерфейсы](../../csharp/programming-guide/interfaces/index.md) и [модификаторы доступа](../../csharp/programming-guide/classes-and-structs/access-modifiers.md).</span><span class="sxs-lookup"><span data-stu-id="5aad8-106">For more information, see [Interfaces](../../csharp/programming-guide/interfaces/index.md) and [Access Modifiers](../../csharp/programming-guide/classes-and-structs/access-modifiers.md).</span></span>  
  
 <span data-ttu-id="5aad8-107">В следующем примере возникает ошибка CS0061.</span><span class="sxs-lookup"><span data-stu-id="5aad8-107">The following sample generates CS0061.</span></span>  
  
```csharp  
// CS0061.cs  
// compile with: /target:library  
internal interface A {}  
public interface AA : A {}  // CS0061  
  
// OK  
public interface B {}  
internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```
