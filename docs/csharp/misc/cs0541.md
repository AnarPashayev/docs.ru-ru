# <a name="compiler-error-cs0541"></a><span data-ttu-id="f8198-103">Ошибка компилятора CS0541</span><span class="sxs-lookup"><span data-stu-id="f8198-103">Compiler Error CS0541</span></span>

<span data-ttu-id="f8198-104">"объявление": явное объявление может содержаться только в классе или в структуре</span><span class="sxs-lookup"><span data-stu-id="f8198-104">'declaration' : explicit interface declaration can only be declared in a class or struct</span></span>  
  
 <span data-ttu-id="f8198-105">Явное объявление [интерфейса](../language-reference/keywords/interface.md) найдено вне [класса](../language-reference/keywords/class.md) или [структуры](../language-reference/builtin-types/struct.md).</span><span class="sxs-lookup"><span data-stu-id="f8198-105">An explicit [interface](../language-reference/keywords/interface.md) declaration was found outside a [class](../language-reference/keywords/class.md) or [struct](../language-reference/builtin-types/struct.md).</span></span>  
  
 <span data-ttu-id="f8198-106">Следующий пример приводит к возникновению ошибки CS0541:</span><span class="sxs-lookup"><span data-stu-id="f8198-106">The following sample generates CS0541:</span></span>  
  
```csharp  
// CS0541.cs  
namespace x  
{  
   interface IFace  
   {  
      void F();  
   }  
  
   interface IFace2 : IFace  
   {  
      void IFace.F();   // CS0541  
   }  
}  
```
