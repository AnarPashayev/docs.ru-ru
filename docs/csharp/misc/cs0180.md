 <span data-ttu-id="ba4e1-105">Ключевые слова [abstract](../language-reference/keywords/abstract.md) и [extern](../language-reference/keywords/extern.md) являются взаимоисключающими.</span><span class="sxs-lookup"><span data-stu-id="ba4e1-105">The [abstract](../language-reference/keywords/abstract.md) and [extern](../language-reference/keywords/extern.md) keywords are mutually exclusive.</span></span> <span data-ttu-id="ba4e1-106">Ключевое слово `extern` означает, что член определен за пределами файла, а ключевое слово **abstract** указывает на то, что реализация предоставлена в производном классе.</span><span class="sxs-lookup"><span data-stu-id="ba4e1-106">The `extern` keyword means that the member is defined outside the file, and **abstract** means that the implementation is provided in a derived class.</span></span> <span data-ttu-id="ba4e1-107">Дополнительные сведения см. в статье [Методы](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="ba4e1-107">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>

```csharp  
   }

```
