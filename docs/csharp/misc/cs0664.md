# <a name="compiler-error-cs0664"></a><span data-ttu-id="dd1d8-102">Ошибка компилятора CS0664</span><span class="sxs-lookup"><span data-stu-id="dd1d8-102">Compiler Error CS0664</span></span>
<span data-ttu-id="dd1d8-103">Литерал с типом double не может быть неявно преобразован к типу "тип"; для создания литерала этого типа используйте суффикс "суффикс"</span><span class="sxs-lookup"><span data-stu-id="dd1d8-103">Literal of type double cannot be implicitly converted to type 'type'; use an 'suffix' suffix to create a literal of this type</span></span>  
  
 <span data-ttu-id="dd1d8-104">Не удалось выполнить назначение; для исправления инструкции используйте суффикс.</span><span class="sxs-lookup"><span data-stu-id="dd1d8-104">An assignment could not be completed; use a suffix to correct the instruction.</span></span> <span data-ttu-id="dd1d8-105">В документации по каждому типу приводится соответствующий суффикс для типа.</span><span class="sxs-lookup"><span data-stu-id="dd1d8-105">The documentation for each type identifies the corresponding suffix for the type.</span></span> <span data-ttu-id="dd1d8-106">Дополнительные сведения о преобразованиях см. в разделе [приведение и преобразование типов](../programming-guide/types/casting-and-type-conversions.md).</span><span class="sxs-lookup"><span data-stu-id="dd1d8-106">For more information on conversions, see [Casting and Type Conversions](../programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="dd1d8-107">Следующий пример приводит к возникновению ошибки CS0664:</span><span class="sxs-lookup"><span data-stu-id="dd1d8-107">The following sample generates CS0664:</span></span>  
  
```csharp  
// CS0664.cs  
class Example  
{  
    static void Main()  
    {  
        decimal d1 = 1.0;   // CS0664, because 1.0 is interpreted  
                            // as a double.  
  
        // Try the following line instead.  
        decimal d2 = 1.0M;  // The M tells the compiler that 1.0 is a  
                            // decimal.  
        Console.WriteLine(d2);  
    }  
}  
```
