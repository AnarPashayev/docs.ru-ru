# <a name="compiler-error-cs0415"></a><span data-ttu-id="fc683-103">Ошибка компилятора CS0415</span><span class="sxs-lookup"><span data-stu-id="fc683-103">Compiler Error CS0415</span></span>

<span data-ttu-id="fc683-104">Атрибут IndexerName применим только в индексаторе, который не является явным объявлением члена интерфейса</span><span class="sxs-lookup"><span data-stu-id="fc683-104">The 'IndexerName' attribute is valid only on an indexer that is not an explicit interface member declaration</span></span>  
  
 <span data-ttu-id="fc683-105">Эта ошибка возникает, если атрибут IndexerName используется в индексаторе, который является явной реализацией интерфейса.</span><span class="sxs-lookup"><span data-stu-id="fc683-105">This error occurs if you use an IndexerName attribute on an indexer that was an explicit implementation of an interface.</span></span> <span data-ttu-id="fc683-106">Этой ошибки можно избежать, удалив имя интерфейса из объявления индексатора, если это возможно.</span><span class="sxs-lookup"><span data-stu-id="fc683-106">This error may be avoided by removing the interface name from the declaration of the indexer, if possible.</span></span> <span data-ttu-id="fc683-107">Дополнительные сведения см. в разделе [Класс IndexerNameAttribute](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="fc683-107">For more information, see the [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="fc683-108">В следующем примере возникает ошибка CS0415:</span><span class="sxs-lookup"><span data-stu-id="fc683-108">The following sample generates CS0415:</span></span>  
  
```csharp
// CS0415.cs  
using System;  
using System.Runtime.CompilerServices;  
  
public interface IA  
{  
    int this[int index]  
    {  
        get;  
        set;  
    }  
}  
  
public class A : IA  
{  
    [IndexerName("Item")]  // CS0415  
    int IA.this[int index]  
    // Try this line instead:  
    // public int this[int index]  
    {  
        get { return 0; }  
        set { }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
