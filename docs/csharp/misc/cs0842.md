 <span data-ttu-id="006b6-105">Автоматически реализуемые свойства имеют резервные поля, предоставленные компилятором, и такое поле недоступно для исходного кода.</span><span class="sxs-lookup"><span data-stu-id="006b6-105">Auto-implemented properties have their backing fields provided by the compiler and the field is not accessible to source code.</span></span> <span data-ttu-id="006b6-106">Таким образом, они не совместимы с <xref:System.Runtime.InteropServices.LayoutKind.Explicit?displayProperty=nameWithType>.</span><span class="sxs-lookup"><span data-stu-id="006b6-106">Therefore, they are not compatible with <xref:System.Runtime.InteropServices.LayoutKind.Explicit?displayProperty=nameWithType>.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="006b6-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="006b6-107">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="006b6-109">Пример</span><span class="sxs-lookup"><span data-stu-id="006b6-109">Example</span></span>  

 <span data-ttu-id="006b6-110">Следующий пример приводит к возникновению ошибки CS0842:</span><span class="sxs-lookup"><span data-stu-id="006b6-110">The following example generates CS0842:</span></span>  
  
```csharp  
// cs0842.cs  
using System;  
using System.Runtime.InteropServices;  
  
namespace TestNamespace  
{  
    [StructLayout(LayoutKind.Explicit)]  
    struct Str  
    {  
        public int Num // CS0842  
        {  
            get;  
            set;  
        }  
  
        static int Main()  
        {  
            return 1;  
        }  
    }  
}  
```
