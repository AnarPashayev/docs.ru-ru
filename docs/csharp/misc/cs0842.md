# <a name="compiler-error-cs0842"></a><span data-ttu-id="afe32-103">Ошибка компилятора CS0842</span><span class="sxs-lookup"><span data-stu-id="afe32-103">Compiler Error CS0842</span></span>

 <span data-ttu-id="afe32-105">Автоматически реализуемые свойства имеют резервные поля, предоставленные компилятором, и такое поле недоступно для исходного кода.</span><span class="sxs-lookup"><span data-stu-id="afe32-105">Auto-implemented properties have their backing fields provided by the compiler and the field is not accessible to source code.</span></span> <span data-ttu-id="afe32-106">Таким образом, они не совместимы с <xref:System.Runtime.InteropServices.LayoutKind.Explicit?displayProperty=nameWithType>.</span><span class="sxs-lookup"><span data-stu-id="afe32-106">Therefore, they are not compatible with <xref:System.Runtime.InteropServices.LayoutKind.Explicit?displayProperty=nameWithType>.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="afe32-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="afe32-107">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="afe32-109">Пример</span><span class="sxs-lookup"><span data-stu-id="afe32-109">Example</span></span>

```csharp  
// cs0842.cs

using System.Runtime.InteropServices;  
  
namespace TestNamespace  
{  
    [StructLayout(LayoutKind.Explicit)]  
    struct Str  
    {  
        public int Num // CS0842  
        {  
            get;  
            set;  
        }  
  
        static int Main()  
        {  
            return 1;  
        }  
    }  
}  
```
