# <a name="compiler-error-cs1559"></a>Ошибка компилятора CS1559
"объект" не может использоваться для метода Main, так как он импортирован  
  
 В параметре компилятора [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) указан недопустимый класс; этот класс не может использоваться как местоположение метода [Main](../../csharp/programming-guide/main-and-command-args/index.md) .
