# <a name="compiler-error-cs1559"></a>Ошибка компилятора CS1559
"объект" не может использоваться для метода Main, так как он импортирован  
  
 Для параметра компилятора [-Main](../language-reference/compiler-options/main-compiler-option.md) был указан недопустимый класс. класс не может использоваться в качестве расположения для метода [Main](../programming-guide/main-and-command-args/index.md) .
