<span data-ttu-id="f2926-103">Невозможно задать аргумент CLSCompliant в модуле, который отличается от атрибута CLSCompliant в сборке.</span><span class="sxs-lookup"><span data-stu-id="f2926-103">You cannot specify the CLSCompliant attribute on a module that differs from the CLSCompliant attribute on the assembly</span></span>

## <a name="example"></a><span data-ttu-id="f2926-108">Пример</span><span class="sxs-lookup"><span data-stu-id="f2926-108">Example</span></span>  

 <span data-ttu-id="f2926-109">В следующем примере возникает ошибка CS3017:</span><span class="sxs-lookup"><span data-stu-id="f2926-109">The following example generates CS3017:</span></span>  
  
```csharp  
// CS3017.cs  
// compile with: /target:module  
  
using System;  
  
[module: CLSCompliant(true)]  
[assembly: CLSCompliant(false)]  // CS3017  
// Try this line instead:  
// [assembly: CLSCompliant(true)]  
class C  
{  
    static void Main() {}  
}  
```
