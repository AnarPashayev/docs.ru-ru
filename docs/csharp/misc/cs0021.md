## <a name="example"></a><span data-ttu-id="8f7a4-108">Пример</span><span class="sxs-lookup"><span data-stu-id="8f7a4-108">Example</span></span>

<span data-ttu-id="8f7a4-109">В следующем C++ примере компилируется в DLL-файл.</span><span class="sxs-lookup"><span data-stu-id="8f7a4-109">The following C++ example compiles to a .dll file.</span></span> <span data-ttu-id="8f7a4-110">Обратите внимание, что для создания ошибки атрибут [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) задается комментарием.</span><span class="sxs-lookup"><span data-stu-id="8f7a4-110">Note that the [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) attribute is commented out in order to generate the error.</span></span>

```cpp
// CPP0021.cpp
// compile with: /clr /LD
using namespace System::Reflection;
// Uncomment the following line to resolve
//[DefaultMember("myItem")]
public ref class MyClassMC
{
        public:
        property int myItem[int]
        {
            int get(int i){  return 5; }
            void set(int i, int value) {}
        }
};
```

<span data-ttu-id="8f7a4-111">В следующем C# примере вызывается DLL-файл.</span><span class="sxs-lookup"><span data-stu-id="8f7a4-111">The following C# example calls the .dll file.</span></span> <span data-ttu-id="8f7a4-112">Он пытается получить доступ к классу через индексатор, но поскольку ни один член не был объявлен как индексатор по умолчанию, возникает ошибка.</span><span class="sxs-lookup"><span data-stu-id="8f7a4-112">It attempts to access the class via an indexer, but because no member was declared as the default indexer, the error is generated.</span></span> <span data-ttu-id="8f7a4-113">Чтобы устранить эту ошибку, раскомментируйте строку `[DefaultMember("myItem")]`.</span><span class="sxs-lookup"><span data-stu-id="8f7a4-113">You can address the error by uncommenting the line `[DefaultMember("myItem")]`.</span></span>

```csharp
// CS0021.cs
// compile with: /reference:CPP0021.dll
public class MyClass
{
    public static void Main()
    {
        MyClassMC myMC = new MyClassMC();
        int j = myMC[1]; // CS0021
    }
}
```
