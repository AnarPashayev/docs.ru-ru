# <a name="compiler-error-cs0021"></a><span data-ttu-id="f6efa-102">Ошибка компилятора CS0021</span><span class="sxs-lookup"><span data-stu-id="f6efa-102">Compiler Error CS0021</span></span>
<span data-ttu-id="f6efa-103">Не удается применить индексирование через [] к выражению типа "тип"</span><span class="sxs-lookup"><span data-stu-id="f6efa-103">Cannot apply indexing with [] to an expression of type 'type'</span></span>  
  
 <span data-ttu-id="f6efa-104">Попытка доступа к значению через индексатор для типа данных, который не поддерживает [Indexers](../../csharp/programming-guide/indexers/index.md).</span><span class="sxs-lookup"><span data-stu-id="f6efa-104">An attempt was made to access a value through an indexer on a data type that does not support [Indexers](../../csharp/programming-guide/indexers/index.md).</span></span>  
  
 <span data-ttu-id="f6efa-105">Ошибка CS0021 может возникнуть при попытке использовать индексатор в сборке C++.</span><span class="sxs-lookup"><span data-stu-id="f6efa-105">You may get CS0021 when trying to use an indexer in a C++ assembly.</span></span> <span data-ttu-id="f6efa-106">В этом случае оформите класс C++ с помощью атрибута `DefaultMember` , чтобы компилятор C# знал, какой из индексаторов используется по умолчанию.</span><span class="sxs-lookup"><span data-stu-id="f6efa-106">In this case, decorate the C++ class with the `DefaultMember` attribute so the C# compiler knows which indexer is the default.</span></span> <span data-ttu-id="f6efa-107">Следующий пример приводит к возникновению ошибки CS0021.</span><span class="sxs-lookup"><span data-stu-id="f6efa-107">The following sample generates CS0021.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f6efa-108">Пример</span><span class="sxs-lookup"><span data-stu-id="f6efa-108">Example</span></span>  
 <span data-ttu-id="f6efa-109">Этот файл компилируется в DLL-файл — с закомментированным атрибутом `DefaultMember` — для создания ошибки.</span><span class="sxs-lookup"><span data-stu-id="f6efa-109">This file compiles to a .dll file—with the `DefaultMember` attribute commented out—in order to generate the error.</span></span>  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  
  
## <a name="example"></a><span data-ttu-id="f6efa-110">Пример</span><span class="sxs-lookup"><span data-stu-id="f6efa-110">Example</span></span>  
 <span data-ttu-id="f6efa-111">Ниже приведен файл C#, который вызывает DLL-файл.</span><span class="sxs-lookup"><span data-stu-id="f6efa-111">The following is the C# file that calls the .dll file.</span></span> <span data-ttu-id="f6efa-112">Этот файл пытается получить доступ к классу через индексатор, но так как никакой член не был объявлен как индексатор по умолчанию, возникает ошибка.</span><span class="sxs-lookup"><span data-stu-id="f6efa-112">This file attempts to access the class via an indexer, but because no member was declared as the default indexer to be used, the error is generated.</span></span>  
  
```csharp  
// CS0021.cs  
// compile with: /reference:CPP0021.dll  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClassMC myMC = new MyClassMC();  
        int j = myMC[1]; // CS0021  
    }  
}  
```
