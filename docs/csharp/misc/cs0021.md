<span data-ttu-id="5eb69-106">Ошибка CS0021 может возникнуть при попытке использовать индексатор в сборке C++.</span><span class="sxs-lookup"><span data-stu-id="5eb69-106">You may get CS0021 when trying to use an indexer in a C++ assembly.</span></span> <span data-ttu-id="5eb69-107">В этом случае оформите класс C++ с помощью атрибута `DefaultMember` , чтобы компилятор C# знал, какой из индексаторов используется по умолчанию.</span><span class="sxs-lookup"><span data-stu-id="5eb69-107">In this case, decorate the C++ class with the `DefaultMember` attribute so the C# compiler knows which indexer is the default.</span></span> <span data-ttu-id="5eb69-108">Следующий пример приводит к возникновению ошибки CS0021.</span><span class="sxs-lookup"><span data-stu-id="5eb69-108">The following sample generates CS0021.</span></span>

## <a name="example"></a><span data-ttu-id="5eb69-109">Пример</span><span class="sxs-lookup"><span data-stu-id="5eb69-109">Example</span></span>

<span data-ttu-id="5eb69-110">Следующий пример C++ компилируется в DLL-файл.</span><span class="sxs-lookup"><span data-stu-id="5eb69-110">The following C++ example compiles to a .dll file.</span></span> <span data-ttu-id="5eb69-111">Обратите внимание, что для создания ошибки атрибут [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) задается комментарием.</span><span class="sxs-lookup"><span data-stu-id="5eb69-111">Note that the [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) attribute is commented out in order to generate the error.</span></span>

```cpp
// CPP0021.cpp
// compile with: /clr /LD
using namespace System::Reflection;
// Uncomment the following line to resolve
//[DefaultMember("myItem")]
public ref class MyClassMC
{
        public:
        property int myItem[int]
        {
            int get(int i){  return 5; }
            void set(int i, int value) {}
        }
};
```

<span data-ttu-id="5eb69-112">В следующем примере C# вызывается DLL-файл.</span><span class="sxs-lookup"><span data-stu-id="5eb69-112">The following C# example calls the .dll file.</span></span> <span data-ttu-id="5eb69-113">Он пытается получить доступ к классу через индексатор, но поскольку ни один член не был объявлен как индексатор по умолчанию, возникает ошибка.</span><span class="sxs-lookup"><span data-stu-id="5eb69-113">It attempts to access the class via an indexer, but because no member was declared as the default indexer, the error is generated.</span></span> <span data-ttu-id="5eb69-114">Чтобы устранить эту ошибку, раскомментируйте строку `[DefaultMember("myItem")]` .</span><span class="sxs-lookup"><span data-stu-id="5eb69-114">You can address the error by uncommenting the line `[DefaultMember("myItem")]`.</span></span>

```csharp
// CS0021.cs
// compile with: /reference:CPP0021.dll
public class MyClass
{
    public static void Main()
    {
        MyClassMC myMC = new MyClassMC();
        int j = myMC[1]; // CS0021
    }
}
```
