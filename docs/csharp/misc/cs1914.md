# <a name="compiler-error-cs1914"></a><span data-ttu-id="f0d71-102">Ошибка компилятора CS1914</span><span class="sxs-lookup"><span data-stu-id="f0d71-102">Compiler Error CS1914</span></span>
<span data-ttu-id="f0d71-103">Статическому полю "имя" не может быть присвоено значение в инициализаторе объекта</span><span class="sxs-lookup"><span data-stu-id="f0d71-103">Static field 'name' cannot be assigned in an object initializer</span></span>  
  
 <span data-ttu-id="f0d71-104">Инициализаторы объекта по определению инициализируют объекты или экземпляры классов.</span><span class="sxs-lookup"><span data-stu-id="f0d71-104">Object initializers by definition initialize objects, or instances, of classes.</span></span> <span data-ttu-id="f0d71-105">Они не могут использоваться для инициализации поля `static` типа.</span><span class="sxs-lookup"><span data-stu-id="f0d71-105">They cannot be used to initialize a `static` field of a type.</span></span> <span data-ttu-id="f0d71-106">Независимо от того, сколько экземпляров класса создается, существует только одна копия поля `static` .</span><span class="sxs-lookup"><span data-stu-id="f0d71-106">No matter how many instances of a class are created, there is only one copy of a `static` field.</span></span>

## <a name="example"></a><span data-ttu-id="f0d71-109">Пример</span><span class="sxs-lookup"><span data-stu-id="f0d71-109">Example</span></span>  
 <span data-ttu-id="f0d71-110">В следующем коде возникает ошибка CS1914, поскольку инициализатор пытается инициализировать поле `TestClass.Number` , которое является `static`:</span><span class="sxs-lookup"><span data-stu-id="f0d71-110">The following code generates CS1914 because the initializer tries to initialize the `TestClass.Number` field, which is `static`:</span></span>  
  
```csharp  
// cs1914.cs  
using System.Linq;  
public class TestClass  
{  
    public string Message { get; set; }  
    public static int Number { get; set; }      
}  
class Test  
{  
    static void Main()  
    {  
        TestClass b = new TestClass() { Message = "Hello", Number = "555-1212" }; // CS1914  
  
    }  
}  
```
