# <a name="compiler-error-cs1520"></a><span data-ttu-id="60eaa-103">Ошибка компилятора CS1520</span><span class="sxs-lookup"><span data-stu-id="60eaa-103">Compiler Error CS1520</span></span>

<span data-ttu-id="60eaa-104">Метод должен иметь тип возвращаемого значения</span><span class="sxs-lookup"><span data-stu-id="60eaa-104">Method must have a return type</span></span>  
  
 <span data-ttu-id="60eaa-105">Метод, объявленный в классе, структуре или интерфейсе, должен иметь явный тип возврата.</span><span class="sxs-lookup"><span data-stu-id="60eaa-105">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="60eaa-106">В следующем примере метод Square имеет возвращаемое значение [string](../language-reference/builtin-types/reference-types.md):</span><span class="sxs-lookup"><span data-stu-id="60eaa-106">In the following example, the Square method has a return value of [string](../language-reference/builtin-types/reference-types.md):</span></span>  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 <span data-ttu-id="60eaa-107">Следующий пример приводит к возникновению ошибки CS1520:</span><span class="sxs-lookup"><span data-stu-id="60eaa-107">The following sample generates CS1520:</span></span>  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="60eaa-108">Кроме того, эта ошибка может возникать, когда регистр имени конструктора отличается от регистра объявления класса или структуры, как в следующем примере.</span><span class="sxs-lookup"><span data-stu-id="60eaa-108">Alternatively, this error might be encountered when the case of a constructor's name differs from that of the class or struct declaration, as in the following sample.</span></span> <span data-ttu-id="60eaa-109">Поскольку имя не точно совпадает с именем класса, компилятор интерпретирует его как обычный метод, а не конструктор, и в результате возникает ошибка:</span><span class="sxs-lookup"><span data-stu-id="60eaa-109">Because the name is not exactly the same as the class name, the compiler interprets it as a regular method, not a constructor, and produces the error:</span></span>  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```
