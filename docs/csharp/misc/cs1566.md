# <a name="compiler-error-cs1566"></a>Ошибка компилятора CS1566
Ошибка при считывании файла ресурсов "файл" — "причина"  
  
 В компиляторе возникли проблемы с именем файла, переданного в параметр компилятора [-Resource](../language-reference/compiler-options/resource-compiler-option.md) .
