# <a name="compiler-error-cs0073"></a><span data-ttu-id="e93e0-103">Ошибка компилятора CS0073</span><span class="sxs-lookup"><span data-stu-id="e93e0-103">Compiler Error CS0073</span></span>
<span data-ttu-id="e93e0-104">Методы доступа add и remove должны иметь тело.</span><span class="sxs-lookup"><span data-stu-id="e93e0-104">An add or remove accessor must have a body</span></span>  
  
 <span data-ttu-id="e93e0-105">Ключевое слово **add** или **remove** в определении [события](../language-reference/keywords/event.md) должно иметь тело.</span><span class="sxs-lookup"><span data-stu-id="e93e0-105">An **add** or **remove** keyword in an [event](../language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="e93e0-106">Дополнительные сведения см. в статье [Events (Visual Basic)](../programming-guide/events/index.md) (События в Visual Basic).</span><span class="sxs-lookup"><span data-stu-id="e93e0-106">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="e93e0-107">В следующем примере возникает ошибка CS0073:</span><span class="sxs-lookup"><span data-stu-id="e93e0-107">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
