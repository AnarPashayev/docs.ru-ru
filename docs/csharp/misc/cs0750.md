# <a name="compiler-error-cs0750"></a><span data-ttu-id="a0b7e-102">Ошибка компилятора CS0750</span><span class="sxs-lookup"><span data-stu-id="a0b7e-102">Compiler Error CS0750</span></span>
<span data-ttu-id="a0b7e-103">Разделяемый метод не может иметь модификаторы доступа или модификаторы virtual, abstract, override, new, sealed или extern.</span><span class="sxs-lookup"><span data-stu-id="a0b7e-103">A partial method cannot have access modifiers or the virtual, abstract, override, new, sealed, or extern modifiers.</span></span>  
  
 <span data-ttu-id="a0b7e-104">Из-за особого поведения разделяемых методов на них накладываются ограничения в отношении модификаторов, которые в них можно использовать.</span><span class="sxs-lookup"><span data-stu-id="a0b7e-104">Because of their special behavior, partial methods are subject to restrictions as to the modifiers they can accept.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a0b7e-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="a0b7e-105">To correct this error</span></span>  
  
1. <span data-ttu-id="a0b7e-106">Удалите неразрешенный модификатор из объявления метода.</span><span class="sxs-lookup"><span data-stu-id="a0b7e-106">Remove the unauthorized modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a0b7e-107">Пример</span><span class="sxs-lookup"><span data-stu-id="a0b7e-107">Example</span></span>  
 <span data-ttu-id="a0b7e-108">Следующий пример приводит к возникновению ошибки CS0750:</span><span class="sxs-lookup"><span data-stu-id="a0b7e-108">The following example generates CS0750:</span></span>  
  
```csharp  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
