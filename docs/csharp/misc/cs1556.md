# <a name="compiler-error-cs1556"></a>Ошибка компилятора CS1556
"конструкция", заданная для метода Main, должна быть действительным классом или структурой  
  
 Параметру [-Main](../language-reference/compiler-options/main-compiler-option.md) компилятора был передан идентификатор, который не является именем класса.
