# <a name="compiler-error-cs0014"></a>Ошибка компилятора CS0014
Не удалось найти требуемый файл "файл"  
  
 В системе отсутствует файл, необходимый для компилятора. Проверьте правильность пути. Если файл является системным файлом Visual Studio, возможно, потребуется исправить установку или удалить и переустановить Visual Studio полностью.
