# <a name="compiler-error-cs0014"></a>Ошибка компилятора CS0014
Не удалось найти требуемый файл "файл"  
  
 В системе отсутствует файл, необходимый для компилятора. Проверьте правильность пути. Если файл является файлом системы Visual Studio, то может потребоваться восстановить установку иначе удалите и переустановите Visual Studio полностью.
