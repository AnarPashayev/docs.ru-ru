# <a name="compiler-warning-level-1-cs1635"></a><span data-ttu-id="4201b-103">Предупреждение компилятора (уровень 1) CS1635</span><span class="sxs-lookup"><span data-stu-id="4201b-103">Compiler Warning (level 1) CS1635</span></span>
<span data-ttu-id="4201b-104">Невозможно возобновить предупреждение "код предупреждения", поскольку оно было глобально отключено</span><span class="sxs-lookup"><span data-stu-id="4201b-104">Cannot restore warning 'warning code' because it was disabled globally</span></span>  
  
 <span data-ttu-id="4201b-105">Это предупреждение возникает, если используется параметр **/nowarn** командной строки или в параметрах проекта отключены предупреждения компилятора для всего блока компиляции, но при этом для попытки восстановления этого предупреждения используется директива `#pragma warning restore` .</span><span class="sxs-lookup"><span data-stu-id="4201b-105">This warning occurs if you use the **/nowarn** command line option or project setting to disable a warning for the entire compilation unit, but you use `#pragma warning restore` to attempt to restore that warning.</span></span> <span data-ttu-id="4201b-106">Для устранения этой ошибки удалите из командной строки параметр /nowarn, измените параметры проекта либо удалите директиву `#pragma warning restore` для предупреждений, отключаемых с использованием командной строки или параметров проекта.</span><span class="sxs-lookup"><span data-stu-id="4201b-106">To resolve this error, remove the /nowarn command line option or project setting, or remove the `#pragma warning restore` for any warnings you are disabling via the command line or project settings.</span></span> <span data-ttu-id="4201b-107">Дополнительные сведения см. в разделе [#pragma warning](../language-reference/preprocessor-directives/preprocessor-pragma-warning.md) .</span><span class="sxs-lookup"><span data-stu-id="4201b-107">For more information, see the [#pragma warning](../language-reference/preprocessor-directives/preprocessor-pragma-warning.md) topic.</span></span>  
  
 <span data-ttu-id="4201b-108">Следующий пример приводит к возникновению предупреждения CS1635:</span><span class="sxs-lookup"><span data-stu-id="4201b-108">The following sample generates CS1635:</span></span>  
  
```csharp  
// CS1635.cs  
// compile with: /w:1 /nowarn:162  
  
enum MyEnum {one=1,two=2,three=3};  
  
class MyClass  
{  
    public static void Main()  
    {  
#pragma warning disable 162  
  
    if (MyEnum.three == MyEnum.two)  
        System.Console.WriteLine("Duplicate");  
  
#pragma warning restore 162  
    }  
}  
```
