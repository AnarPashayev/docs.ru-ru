# <a name="compiler-error-cs1561"></a>Ошибка компилятора CS1561
Недопустимое или слишком длинное имя файла  
  
 Длина имени выходного файла не может превышать 256 символов, и в этом имени не должны содержаться недопустимые символы, такие как символ евро, знак вопроса или обратная косая черта.
