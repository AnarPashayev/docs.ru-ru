# <a name="compiler-error-cs0012"></a><span data-ttu-id="7ba92-102">Ошибка компилятора CS0012</span><span class="sxs-lookup"><span data-stu-id="7ba92-102">Compiler Error CS0012</span></span>
<span data-ttu-id="7ba92-103">Тип "тип" определен в сборке, которая не используется.</span><span class="sxs-lookup"><span data-stu-id="7ba92-103">The type 'type' is defined in an assembly that is not referenced.</span></span> <span data-ttu-id="7ba92-104">Необходимо добавить ссылку на сборку "сборка".</span><span class="sxs-lookup"><span data-stu-id="7ba92-104">You must add a reference to assembly 'assembly'.</span></span>  
  
 <span data-ttu-id="7ba92-105">Не найдено определение ссылочного типа.</span><span class="sxs-lookup"><span data-stu-id="7ba92-105">The definition for a referenced type was not found.</span></span> <span data-ttu-id="7ba92-106">Это может произойти, если требуемый DLL-файл не включен в компиляцию.</span><span class="sxs-lookup"><span data-stu-id="7ba92-106">This could occur if a required .DLL file is not included in the compilation.</span></span> <span data-ttu-id="7ba92-107">Дополнительные сведения см. в разделе [диалоговое окно "Добавление ссылки"](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) и [-Reference (C# параметры компилятора)](../language-reference/compiler-options/reference-compiler-option.md).</span><span class="sxs-lookup"><span data-stu-id="7ba92-107">For more information, see [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) and [-reference (C# Compiler Options)](../language-reference/compiler-options/reference-compiler-option.md).</span></span>  
  
 <span data-ttu-id="7ba92-108">Следующая последовательность компиляций приведет к возникновению ошибки CS0012:</span><span class="sxs-lookup"><span data-stu-id="7ba92-108">The following sequence of compilations will result in CS0012:</span></span>  
  
```csharp  
// cs0012a.cs  
// compile with: /target:library  
public class A {}  
```  
  
 <span data-ttu-id="7ba92-109">Затем:</span><span class="sxs-lookup"><span data-stu-id="7ba92-109">Then:</span></span>  
  
```csharp  
// cs0012b.cs  
// compile with: /target:library /reference:cs0012a.dll  
public class B  
{  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="7ba92-110">Затем:</span><span class="sxs-lookup"><span data-stu-id="7ba92-110">Then:</span></span>  
  
```csharp  
// cs0012c.cs  
// compile with: /reference:cs0012b.dll  
class C  
{  
   public static void Main()  
   {  
      object o = B.f();   // CS0012  
   }  
}  
```  
  
 <span data-ttu-id="7ba92-111">Вы можете устранить эту ошибку CS0012, выполнив компиляцию с `/reference:cs0012b.dll;cs0012a.dll`, или в Visual Studio, используя [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) для добавления ссылки на cs0012a.dll в дополнение к cs0012b.dll.</span><span class="sxs-lookup"><span data-stu-id="7ba92-111">You could resolve this CS0012 by compiling with `/reference:cs0012b.dll;cs0012a.dll`, or in Visual Studio by using the [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) to add a reference to cs0012a.dll in addition to cs0012b.dll.</span></span>
