# <a name="compiler-error-cs0690"></a>Ошибка компилятора CS0690
Входной файл "файл" содержит недопустимые метаданные.  
  
 Вы можете открыть файл метаданных, но из-за некоторых проблем локализации метаданные оказались поврежденными. Это похоже на ошибку [CS0009](../../csharp/misc/cs0009.md), за исключением того, что можно открыть файл метаданных.  
  
 PE-файл (Process Executable) является исполняемым файлом.
