# <a name="compiler-error-cs0170"></a>Ошибка компилятора CS0170
Использование поля "поле", которому, возможно, не присвоено значение.  
  
 Поле в структуре было использовано без предварительной инициализации. Чтобы решить эту проблему, сначала определите, какое поле не было инициализировано, а затем инициализируйте его, прежде чем пытаться получить к нему доступ. Дополнительные сведения об инициализации структур см. в разделе [структуры](../programming-guide/classes-and-structs/structs.md) и [использование структур](../programming-guide/classes-and-structs/using-structs.md).  
  
 Следующий пример приводит к возникновению ошибки CS0170:  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because   
                                         //e.i was never assigned  
   }  
}  
```
