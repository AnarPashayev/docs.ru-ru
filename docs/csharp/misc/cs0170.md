 <span data-ttu-id="bc4fa-104">Поле в структуре было использовано без предварительной инициализации.</span><span class="sxs-lookup"><span data-stu-id="bc4fa-104">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="bc4fa-105">Чтобы решить эту проблему, сначала определите, какое поле не было инициализировано, а затем инициализируйте его, прежде чем пытаться получить к нему доступ.</span><span class="sxs-lookup"><span data-stu-id="bc4fa-105">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="bc4fa-106">Дополнительные сведения об инициализации структур см. в разделе [типы структур](../language-reference/builtin-types/struct.md).</span><span class="sxs-lookup"><span data-stu-id="bc4fa-106">For more information about initializing structs, see [Structure types](../language-reference/builtin-types/struct.md).</span></span>
  
 <span data-ttu-id="bc4fa-107">Следующий пример приводит к возникновению ошибки CS0170:</span><span class="sxs-lookup"><span data-stu-id="bc4fa-107">The following sample generates CS0170:</span></span>  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because   
                                         //e.i was never assigned  
   }  
}  
```
