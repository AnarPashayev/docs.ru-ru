<span data-ttu-id="9152b-107">Ошибка CS0278 может произойти, если компилятору не удается выполнить сопоставление из-за неоднозначности.</span><span class="sxs-lookup"><span data-stu-id="9152b-107">CS0278 can occur if the compiler is unable to make the match due to ambiguities.</span></span> <span data-ttu-id="9152b-108">Например, шаблон "перечисление" требует, чтобы существовал метод с именем `MoveNext`, и ваш код может содержать два метода `MoveNext`.</span><span class="sxs-lookup"><span data-stu-id="9152b-108">For example, the "enumerable" pattern requires that there be a method called `MoveNext`, and your code might contain two methods called `MoveNext`.</span></span> <span data-ttu-id="9152b-109">Компилятор попытается найти интерфейс для использования, но рекомендуется определить и устранить причину неоднозначности.</span><span class="sxs-lookup"><span data-stu-id="9152b-109">The compiler will attempt to find an interface to use, but it is recommended that you determine and resolve the cause of the ambiguity.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9152b-110">Пример</span><span class="sxs-lookup"><span data-stu-id="9152b-110">Example</span></span>

   public static void TestForeach<W>(W w)
