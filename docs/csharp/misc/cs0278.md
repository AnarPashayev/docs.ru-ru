<span data-ttu-id="017d8-108">Ошибка CS0278 может произойти, если компилятору не удается выполнить сопоставление из-за неоднозначности.</span><span class="sxs-lookup"><span data-stu-id="017d8-108">CS0278 can occur if the compiler is unable to make the match due to ambiguities.</span></span> <span data-ttu-id="017d8-109">Например, шаблон "перечисление" требует, чтобы существовал метод с именем `MoveNext`, и ваш код может содержать два метода `MoveNext`.</span><span class="sxs-lookup"><span data-stu-id="017d8-109">For example, the "enumerable" pattern requires that there be a method called `MoveNext`, and your code might contain two methods called `MoveNext`.</span></span> <span data-ttu-id="017d8-110">Компилятор попытается найти интерфейс для использования, но рекомендуется определить и устранить причину неоднозначности.</span><span class="sxs-lookup"><span data-stu-id="017d8-110">The compiler will attempt to find an interface to use, but it is recommended that you determine and resolve the cause of the ambiguity.</span></span>
