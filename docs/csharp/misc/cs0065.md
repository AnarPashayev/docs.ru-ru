# <a name="compiler-error-cs0065"></a><span data-ttu-id="d153f-103">Ошибка компилятора CS0065</span><span class="sxs-lookup"><span data-stu-id="d153f-103">Compiler Error CS0065</span></span>

<span data-ttu-id="d153f-104">"событие": свойство события должно иметь методы доступа на добавление и удаление</span><span class="sxs-lookup"><span data-stu-id="d153f-104">'event' : event property must have both add and remove accessors</span></span>

<span data-ttu-id="d153f-105">Событие, которое не является полем, должно иметь оба метода доступа.</span><span class="sxs-lookup"><span data-stu-id="d153f-105">An event that is not a field must have both access methods.</span></span>

<span data-ttu-id="d153f-106">Следующий пример приводит к возникновению ошибки CS0065:</span><span class="sxs-lookup"><span data-stu-id="d153f-106">The following sample generates CS0065:</span></span>

```csharp
// CS0065.cs
using System;
public delegate void EventHandler(object sender, int e);
public class MyClass
{
   public event EventHandler Click   // CS0065,
   {
      // to fix, uncomment the add and remove definitions
      /*
      add
      {
         Click += value;
      }
      remove
      {
         Click -= value;
      }
      */
   }

   public static void Main()
   {
   }
}
```
