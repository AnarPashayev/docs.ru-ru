# <a name="compiler-warning-level-1-cs0602"></a>Предупреждение компилятора (уровень 1) CS0602

Функция "старая_функция" устарела. Используйте вместо нее функцию "новая_функция"

 Функция языка, используемая в коде (*старая_функция*) по-прежнему поддерживается, но в будущих выпусках эта поддержка может быть прекращена. Вместо этого следует использовать рекомендованный синтаксис (*новая_функция*).

> [!NOTE]
> Это предупреждение компилятора больше не используется в Roslyn.
