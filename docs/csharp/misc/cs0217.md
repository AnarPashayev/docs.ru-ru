# <a name="compiler-error-cs0217"></a><span data-ttu-id="206d4-102">Ошибка компилятора CS0217</span><span class="sxs-lookup"><span data-stu-id="206d4-102">Compiler Error CS0217</span></span>
<span data-ttu-id="206d4-103">При использовании определенного пользователем логического оператора ("оператор") как отладочного тип значения, возвращаемого этим оператором, должен совпадать с типом обоих его параметров.</span><span class="sxs-lookup"><span data-stu-id="206d4-103">In order to be applicable as a short circuit operator a user-defined logical operator ('operator') must have the same return type as the type of its 2 parameters.</span></span>  
  
 <span data-ttu-id="206d4-104">Если вы задаете оператор для определяемого пользователем типа, а затем пытаетесь использовать его как оператор краткой записи, определяемый пользователем оператор должен иметь параметры и возвращаемые значения того же типа.</span><span class="sxs-lookup"><span data-stu-id="206d4-104">If you define an operator for a user-defined type, and then try to use the operator as a short-circuit operator, the user-defined operator must have parameters and return values of the same type.</span></span> <span data-ttu-id="206d4-105">Дополнительные сведения об операторах сокращенной схемы см. в разделе [ `&&` оператор](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) и [ `||` оператор](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-).</span><span class="sxs-lookup"><span data-stu-id="206d4-105">For more information about short-circuit operators, see [`&&` operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) and [`||` operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-).</span></span> <span data-ttu-id="206d4-106">Дополнительные сведения об определяемых пользователем сокращенных и условных операторах см. в разделе [определяемые пользователем условные логические операторы](~/_csharplang/spec/expressions.md#user-defined-conditional-logical-operators) в [ C# спецификации языка](~/_csharplang/spec/introduction.md).</span><span class="sxs-lookup"><span data-stu-id="206d4-106">For more information about user-defined short-circuit, or conditional, operators, see the [User-defined conditional logical operators](~/_csharplang/spec/expressions.md#user-defined-conditional-logical-operators) section of the [C# language specification](~/_csharplang/spec/introduction.md).</span></span>
  
 <span data-ttu-id="206d4-107">Следующий пример приводит к возникновению ошибки CS0217:</span><span class="sxs-lookup"><span data-stu-id="206d4-107">The following sample generates CS0217:</span></span>  
  
```csharp  
// CS0217.cs  
using System;  
  
public class MyClass  
{  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static int operator & (MyClass f1, MyClass f2)   // CS0217  
   // try the following line instead  
   // public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;  
   }  
}  
```
