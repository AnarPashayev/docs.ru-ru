## <a name="example"></a><span data-ttu-id="4dcf9-108">Пример</span><span class="sxs-lookup"><span data-stu-id="4dcf9-108">Example</span></span>

<span data-ttu-id="4dcf9-109">Следующий код генерирует CS1955, потому что код пытается вызвать поле и свойство с помощью [ `()`выражения вызова. ](../language-reference/operators/member-access-operators.md#invocation-expression-)</span><span class="sxs-lookup"><span data-stu-id="4dcf9-109">The following code generates CS1955 because the code is trying to invoke a field and a property by using the [invocation expression `()`](../language-reference/operators/member-access-operators.md#invocation-expression-).</span></span> <span data-ttu-id="4dcf9-110">Вы не можете вызвать поле или свойство.</span><span class="sxs-lookup"><span data-stu-id="4dcf9-110">You cannot call a field or a property.</span></span> <span data-ttu-id="4dcf9-111">Используйте [выражение `.` доступа к членам](../language-reference/operators/member-access-operators.md#member-access-expression-) для доступа к хранимому значению.</span><span class="sxs-lookup"><span data-stu-id="4dcf9-111">Use the [member access expression `.`](../language-reference/operators/member-access-operators.md#member-access-expression-) to access the value it stores.</span></span>
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
