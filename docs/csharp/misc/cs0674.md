 <span data-ttu-id="9c6bb-106">Компилятор C# не допускает использование <xref:System.ParamArrayAttribute?displayProperty=nameWithType>. Используйте вместо этого ключевое слово [params](../language-reference/keywords/params.md) .</span><span class="sxs-lookup"><span data-stu-id="9c6bb-106">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="9c6bb-107">Следующий пример приводит к возникновению ошибки CS0674:</span><span class="sxs-lookup"><span data-stu-id="9c6bb-107">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
