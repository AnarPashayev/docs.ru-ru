## <a name="example"></a><span data-ttu-id="9822b-109">Пример</span><span class="sxs-lookup"><span data-stu-id="9822b-109">Example</span></span>

```csharp

using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };

        return 1;  
    }  
}  
```
