# <a name="compiler-error-cs0745"></a><span data-ttu-id="46242-103">Ошибка компилятора CS0745</span><span class="sxs-lookup"><span data-stu-id="46242-103">Compiler Error CS0745</span></span>
<span data-ttu-id="46242-104">Требуется контекстное ключевое слово "by".</span><span class="sxs-lookup"><span data-stu-id="46242-104">Expected contextual keyword 'by'</span></span>  
  
 <span data-ttu-id="46242-105">Шаблоном для предложения `group` является `group...by` , за которым следует необязательное ключевое слово `into`, как показано в следующем примере:</span><span class="sxs-lookup"><span data-stu-id="46242-105">The pattern for the `group` clause is `group...by` followed by an optional `into`, as shown in the following example:</span></span>  
  
```csharp  
string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
var query = from name in names  
            group name by name[0];  
```  
  
 <span data-ttu-id="46242-106">или</span><span class="sxs-lookup"><span data-stu-id="46242-106">or</span></span>  
  
```csharp  
var query2 = from name in names  
             group name by name[0] into g  
             //...additional query clauses  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="46242-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="46242-107">To correct this error</span></span>  
  
1. <span data-ttu-id="46242-108">Добавьте ключевое слово `by` в предложение `group` .</span><span class="sxs-lookup"><span data-stu-id="46242-108">Add the `by` keyword to the `group` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="46242-109">Пример</span><span class="sxs-lookup"><span data-stu-id="46242-109">Example</span></span>  
 <span data-ttu-id="46242-110">При компиляции следующего кода возникнет ошибка CS0745:</span><span class="sxs-lookup"><span data-stu-id="46242-110">The following code generates CS0745:</span></span>  
  
```csharp  
// cs0745.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
        var query = from name in names  
                    group name name[0]; // CS0745  
  
        return 1;  
    }  
}  
```
