# <a name="compiler-error-cs1650"></a><span data-ttu-id="d3444-103">Ошибка компилятора CS1650</span><span class="sxs-lookup"><span data-stu-id="d3444-103">Compiler Error CS1650</span></span>

<span data-ttu-id="d3444-104">Присваивание значений полям статического поля "идентификатор", доступного только для чтения, допускается только в статическом конструкторе или инициализаторе переменных</span><span class="sxs-lookup"><span data-stu-id="d3444-104">Fields of static readonly field 'identifier' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="d3444-105">Эта ошибка возникает при попытке изменить член поля, который доступен только для чтения и является статическим, где нельзя вносить изменения.</span><span class="sxs-lookup"><span data-stu-id="d3444-105">This error occurs when you attempt to modify a member of a field which is readonly and static where it is not allowed to be modified.</span></span> <span data-ttu-id="d3444-106">Чтобы устранить эту ошибку, при назначении полям, доступным только для чтения, ограничьтесь конструктором или инициализатором переменных или удалите ключевое слово `readonly` из объявления поля.</span><span class="sxs-lookup"><span data-stu-id="d3444-106">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the `readonly` keyword from the declaration of the field.</span></span>  
  
```csharp  
// CS1650.cs  
public struct Inner  
{  
    public int i;  
}  
  
class Outer  
{  
    public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void Main()  
    {  
        Outer.inner.i = 1;  // CS1650  
    }  
}  
```
