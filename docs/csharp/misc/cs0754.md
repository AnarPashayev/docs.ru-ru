# <a name="compiler-error-cs0754"></a><span data-ttu-id="04252-103">Ошибка компилятора CS0754</span><span class="sxs-lookup"><span data-stu-id="04252-103">Compiler Error CS0754</span></span>

<span data-ttu-id="04252-104">Разделяемый метод не может явно реализовывать метод интерфейса</span><span class="sxs-lookup"><span data-stu-id="04252-104">A partial method may not explicitly implement an interface method.</span></span>  
  
 <span data-ttu-id="04252-105">Разделяемый метод не может быть объявлен в качестве явной реализации метода, определенного в интерфейсе.</span><span class="sxs-lookup"><span data-stu-id="04252-105">A partial method cannot be declared as an explicit implementation of a method defined in an interface.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="04252-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="04252-106">To correct this error</span></span>  
  
1. <span data-ttu-id="04252-107">Удалите явную квалификацию интерфейса из объявления метода.</span><span class="sxs-lookup"><span data-stu-id="04252-107">Remove the explicit interface qualification from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="04252-108">Пример</span><span class="sxs-lookup"><span data-stu-id="04252-108">Example</span></span>  

 <span data-ttu-id="04252-109">При компиляции следующего кода возникнет ошибка CS0754:</span><span class="sxs-lookup"><span data-stu-id="04252-109">The following code generates CS0754:</span></span>  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
