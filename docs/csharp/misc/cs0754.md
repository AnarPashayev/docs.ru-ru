# <a name="compiler-error-cs0754"></a><span data-ttu-id="04e12-102">Ошибка компилятора CS0754</span><span class="sxs-lookup"><span data-stu-id="04e12-102">Compiler Error CS0754</span></span>

 <span data-ttu-id="04e12-104">Разделяемый метод не может быть объявлен в качестве явной реализации метода, определенного в интерфейсе.</span><span class="sxs-lookup"><span data-stu-id="04e12-104">A partial method cannot be declared as an explicit implementation of a method defined in an interface.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="04e12-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="04e12-105">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="04e12-107">Пример</span><span class="sxs-lookup"><span data-stu-id="04e12-107">Example</span></span>  
 <span data-ttu-id="04e12-108">При компиляции следующего кода возникнет ошибка CS0754:</span><span class="sxs-lookup"><span data-stu-id="04e12-108">The following code generates CS0754:</span></span>  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF  
    {  
        partial void IF.Part(); //CS0754  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
