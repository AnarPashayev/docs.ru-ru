# <a name="compiler-error-cs1632"></a><span data-ttu-id="f0c42-103">Ошибка компилятора CS1632</span><span class="sxs-lookup"><span data-stu-id="f0c42-103">Compiler Error CS1632</span></span>
<span data-ttu-id="f0c42-104">Невозможно передать управление из тела анонимного метода или лямбда-выражения.</span><span class="sxs-lookup"><span data-stu-id="f0c42-104">Control cannot leave the body of an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="f0c42-105">Эта ошибка возникает, если Инструкция перехода (**break**, `goto` **Continue**и т. д.) пытается переместить элемент управления из блока анонимного метода.</span><span class="sxs-lookup"><span data-stu-id="f0c42-105">This error occurs if a jump statement (**break**, `goto`, **continue**, etc.) attempts to move control out of an anonymous method block.</span></span> <span data-ttu-id="f0c42-106">Блок анонимного метода является телом функции, и из него можно выйти по оператору return или по достижении конца блока.</span><span class="sxs-lookup"><span data-stu-id="f0c42-106">An anonymous method block is a function body and can only be exited by a return statement or by reaching the end of the block.</span></span>  
  
 <span data-ttu-id="f0c42-107">Следующий пример приводит к возникновению ошибки CS1632:</span><span class="sxs-lookup"><span data-stu-id="f0c42-107">The following sample generates CS1632:</span></span>  
  
```csharp  
// CS1632.cs  
// compile with: /target:library  
delegate void MyDelegate();  
class MyClass  
{  
   public void Test()  
   {
      for (int i = 0 ; i < 5 ; i++)  
      {  
         MyDelegate d = delegate {  
            break;   // CS1632  
          };
      }  
   }  
}  
```
