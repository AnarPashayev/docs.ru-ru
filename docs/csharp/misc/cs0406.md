 <span data-ttu-id="3dbf1-105">Если для универсального типа или метода задано ограничение типа класса, это ограничение должно быть указано в списке первым.</span><span class="sxs-lookup"><span data-stu-id="3dbf1-105">When a generic type or method has a class type constraint, that constraint must be listed first.</span></span> <span data-ttu-id="3dbf1-106">Чтобы избежать этой ошибки, переместите ограничение типа класса в начало списка ограничений.</span><span class="sxs-lookup"><span data-stu-id="3dbf1-106">To avoid this error, move the class type constraint to the beginning of the constraint list.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3dbf1-107">Пример</span><span class="sxs-lookup"><span data-stu-id="3dbf1-107">Example</span></span>  
 <span data-ttu-id="3dbf1-108">Следующий пример приводит к возникновению ошибки CS0406:</span><span class="sxs-lookup"><span data-stu-id="3dbf1-108">The following sample generates CS0406.</span></span>  
  
```csharp

interface I {}  
class C {}

```
