# <a name="compiler-error-cs0243"></a><span data-ttu-id="e0590-103">Ошибка компилятора CS0243</span><span class="sxs-lookup"><span data-stu-id="e0590-103">Compiler error CS0243</span></span>

<span data-ttu-id="e0590-104">Атрибут Conditional в методе "метод" недопустим, так как он является методом переопределения</span><span class="sxs-lookup"><span data-stu-id="e0590-104">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="e0590-105"><xref:System.Diagnostics.ConditionalAttribute>Атрибут не допускается для метода, помеченного ключевым словом [override](../language-reference/keywords/override.md) .</span><span class="sxs-lookup"><span data-stu-id="e0590-105">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="e0590-106">Дополнительные сведения см. [в разделе знание времени использования переопределений и новых ключевых слов](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="e0590-106">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="e0590-107">Компилятор никогда не выполняет привязку к переопределяемым методам.</span><span class="sxs-lookup"><span data-stu-id="e0590-107">The compiler never binds to override methods.</span></span> <span data-ttu-id="e0590-108">Он привязывается только к базовому методу, а общеязыковая среда выполнения вызывает переопределение соответствующим образом.</span><span class="sxs-lookup"><span data-stu-id="e0590-108">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="e0590-109">Следующий код создает CS0243:</span><span class="sxs-lookup"><span data-stu-id="e0590-109">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
