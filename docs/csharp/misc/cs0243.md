# <a name="compiler-error-cs0243"></a><span data-ttu-id="e30b6-102">Ошибка компилятора CS0243</span><span class="sxs-lookup"><span data-stu-id="e30b6-102">Compiler error CS0243</span></span>

<span data-ttu-id="e30b6-103">Атрибут Conditional в методе "метод" недопустим, так как он является методом переопределения</span><span class="sxs-lookup"><span data-stu-id="e30b6-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
<span data-ttu-id="e30b6-104">Атрибут <xref:System.Diagnostics.ConditionalAttribute> не допускается на методе, который отмечен [переопределением](../language-reference/keywords/override.md) ключевого слова.</span><span class="sxs-lookup"><span data-stu-id="e30b6-104">The <xref:System.Diagnostics.ConditionalAttribute> attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="e30b6-105">Для получения дополнительной информации [см.](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md)</span><span class="sxs-lookup"><span data-stu-id="e30b6-105">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
<span data-ttu-id="e30b6-106">Компилятор никогда не связывается с переопределением методов.</span><span class="sxs-lookup"><span data-stu-id="e30b6-106">The compiler never binds to override methods.</span></span> <span data-ttu-id="e30b6-107">Он связывается только с базовым методом, и общее время выполнения языка вызывает переопределение, по мере необходимости.</span><span class="sxs-lookup"><span data-stu-id="e30b6-107">It only binds to the base method, and the common language runtime calls the override, as appropriate.</span></span>  
  
<span data-ttu-id="e30b6-108">Следующий код генерирует CS0243:</span><span class="sxs-lookup"><span data-stu-id="e30b6-108">The following code generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
