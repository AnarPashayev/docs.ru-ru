## <a name="example"></a><span data-ttu-id="df26a-107">Пример</span><span class="sxs-lookup"><span data-stu-id="df26a-107">Example</span></span>  
 <span data-ttu-id="df26a-108">Следующий пример приводит к возникновению ошибки CS0305:</span><span class="sxs-lookup"><span data-stu-id="df26a-108">The following sample generates CS0305.</span></span>  
  
```csharp

public class MyList<T> {}  
public class MyClass<T> {}  
  
class MyClass  
{  
   public static void Main()  
   {  
      MyList<MyClass, MyClass> list1 = new MyList<MyClass>();   // CS0305  
      MyList<MyClass> list2 = new MyList<MyClass>();   // OK  
   }  
}  
```
