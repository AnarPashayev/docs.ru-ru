# <a name="compiler-error-cs0625"></a><span data-ttu-id="6ddeb-103">Ошибка компилятора CS0625</span><span class="sxs-lookup"><span data-stu-id="6ddeb-103">Compiler Error CS0625</span></span>
<span data-ttu-id="6ddeb-104">"поле": типы полей экземпляров, помеченные StructLayout (LayoutKind. explicit), должны иметь атрибут FieldOffset.</span><span class="sxs-lookup"><span data-stu-id="6ddeb-104">'field': instance field types marked with StructLayout(LayoutKind.Explicit) must have a FieldOffset attribute.</span></span>
  
<span data-ttu-id="6ddeb-105">Если структура помечена явным атрибутом **StructLayout** , все поля структуры должны иметь атрибут [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) .</span><span class="sxs-lookup"><span data-stu-id="6ddeb-105">When a struct is marked with an explicit **StructLayout** attribute, all fields in the struct must have the [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute.</span></span> <span data-ttu-id="6ddeb-106">Дополнительные сведения см. в разделе [Класс StructLayoutAttribute](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span><span class="sxs-lookup"><span data-stu-id="6ddeb-106">For more information, see [StructLayoutAttribute Class](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span></span>

<span data-ttu-id="6ddeb-107">В следующем примере возникает ошибка CS0625:</span><span class="sxs-lookup"><span data-stu-id="6ddeb-107">The following sample generates CS0625:</span></span>  
  
```csharp  
// CS0625.cs  
// compile with: /target:library  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
struct A  
{  
   public int i;   // CS0625 not static; an instance field  
}  
  
// OK  
[StructLayout(LayoutKind.Explicit)]  
struct B  
{  
   [FieldOffset(5)]  
   public int i;  
}  
```
