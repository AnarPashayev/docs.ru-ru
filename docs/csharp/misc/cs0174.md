# <a name="compiler-error-cs0174"></a>Ошибка компилятора CS0174

Для ссылки "base" требуется базовый класс.  
  
 Эта ошибка возникает только в том случае, если среда выполнения .NET компилирует исходный код для `System.Object` класса, который является единственным классом, который не имеет базового класса.
