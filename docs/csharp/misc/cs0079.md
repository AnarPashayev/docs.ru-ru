# <a name="compiler-error-cs0079"></a><span data-ttu-id="a2e71-103">Ошибка компилятора CS0079</span><span class="sxs-lookup"><span data-stu-id="a2e71-103">Compiler Error CS0079</span></span>
<span data-ttu-id="a2e71-104">Событие "событие" может находиться только в левой части операции += или -=</span><span class="sxs-lookup"><span data-stu-id="a2e71-104">The event 'event' can only appear on the left hand side of += or -=</span></span>  
  
 <span data-ttu-id="a2e71-105">[Событие](../language-reference/keywords/event.md) было вызвано неправильно.</span><span class="sxs-lookup"><span data-stu-id="a2e71-105">An [event](../language-reference/keywords/event.md) was called incorrectly.</span></span> <span data-ttu-id="a2e71-106">Дополнительные сведения см. в разделе [события](../programming-guide/events/index.md) и [делегаты](../programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="a2e71-106">For more information, see [Events](../programming-guide/events/index.md) and [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="a2e71-107">Следующий пример приводит к возникновению ошибки CS0079:</span><span class="sxs-lookup"><span data-stu-id="a2e71-107">The following sample generates CS0079:</span></span>  
  
```csharp  
// CS0079.cs  
using System;  
  
public delegate void MyEventHandler();  
  
public class Class1  
{  
   private MyEventHandler _e;  
  
   public event MyEventHandler Pow  
   {  
      add  
      {  
         _e += value;  
         Console.WriteLine("in add accessor");  
      }  
      remove  
      {  
         _e -= value;  
         Console.WriteLine("in remove accessor");  
      }  
   }  
  
   public void Handler()  
   {  
   }  
  
   public void Fire()  
   {  
      if (_e != null)  
      {  
         Pow();   // CS0079  
         // try the following line instead  
         // _e();  
      }  
   }  
  
   public static void Main()  
   {  
      Class1 p = new Class1();  
      p.Pow += new MyEventHandler(p.Handler);  
      p._e();  
      p.Pow += new MyEventHandler(p.Handler);  
      p._e();  
      p._e -= new MyEventHandler(p.Handler);  
      if (p._e != null)  
      {  
         p._e();  
      }  
      p.Pow -= new MyEventHandler(p.Handler);  
      if (p._e != null)  
      {  
         p._e();  
      }  
   }  
}  
```
