# <a name="compiler-warning-level-1-cs0684"></a><span data-ttu-id="7a67e-103">Предупреждение компилятора (уровень 1) CS0684</span><span class="sxs-lookup"><span data-stu-id="7a67e-103">Compiler Warning (level 1) CS0684</span></span>

<span data-ttu-id="7a67e-104">Интерфейс "интерфейс" помечен с помощью "CoClassAttribute" и не помечен с помощью "ComImportAttribute"</span><span class="sxs-lookup"><span data-stu-id="7a67e-104">'interface' interface marked with 'CoClassAttribute' not marked with 'ComImportAttribute'</span></span>  
  
 <span data-ttu-id="7a67e-105">Если в интерфейсе указан атрибут **CoClassAttribute** , то следует также указать атрибут **ComImportAttribute**.</span><span class="sxs-lookup"><span data-stu-id="7a67e-105">If you specify **CoClassAttribute** on an interface, you must also specify **ComImportAttribute**.</span></span>  
  
 <span data-ttu-id="7a67e-106">Следующий пример приводит к возникновению ошибки CS0684:</span><span class="sxs-lookup"><span data-stu-id="7a67e-106">The following sample generates CS0684:</span></span>  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
