# <a name="compiler-error-cs0070"></a><span data-ttu-id="daff7-103">Ошибка компилятора CS0070</span><span class="sxs-lookup"><span data-stu-id="daff7-103">Compiler Error CS0070</span></span>

<span data-ttu-id="daff7-104">Событие "событие" может присутствовать только в левой части операций += и -= (кроме случая использования в типе "тип")</span><span class="sxs-lookup"><span data-stu-id="daff7-104">The event 'event' can only appear on the left hand side of += or -= (except when used from within the type 'type')</span></span>  
  
 <span data-ttu-id="daff7-105">За пределами класса, в котором оно определено, [событие](../language-reference/keywords/event.md) может только добавлять или убирать ссылки.</span><span class="sxs-lookup"><span data-stu-id="daff7-105">Outside of the class it is defined in, an [event](../language-reference/keywords/event.md) can only add or subtract references.</span></span> <span data-ttu-id="daff7-106">Дополнительные сведения см. в статье [Events (Visual Basic)](../programming-guide/events/index.md) (События в Visual Basic).</span><span class="sxs-lookup"><span data-stu-id="daff7-106">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="daff7-107">В следующем примере возникает ошибка CS0070:</span><span class="sxs-lookup"><span data-stu-id="daff7-107">The following sample generates CS0070:</span></span>  
  
```csharp  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```
