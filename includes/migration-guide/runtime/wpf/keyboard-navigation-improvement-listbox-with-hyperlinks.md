### <a name="keyboard-navigation-improvement-in-listbox-with-hyperlinks"></a>Улучшение навигации с помощью клавиатуры в ListBox с гиперссылками

#### <a name="details"></a>Подробнее

Исправлен неправильный результат нажатия клавиши со стрелкой, когда фокус установлен на гиперссылку в элементе, который не является выбранным элементом родительского <xref:System.Windows.Controls.ItemsControl>.
